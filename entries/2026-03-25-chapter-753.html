第753章：氷雪の入り口

朝、「氷鳥号」は氷雪の山脈の麓にある小さな港に接岸した。

港というよりは、岩場に作られた簡易的な桟橋だ。氷で補強されているが、それでも波に揺られて軋む音が聞こえる。

「ここが、最寄りの港だ」

ヴィクトル船長が告げた。

「ここから先は、船では行けない。山道を歩くしかない」

カイトは頷いた。

「ありがとうございました。船長」
「礼には及ばん。君たちのおかげで、無事に到着できたんだ。氷海蛇を倒してくれなかったら、今頃、沈んでたよ」

船長は手を振った。

「帰りも乗るなら、七日後にここで待ってる。それ以上遅れると、海が凍り始めるからな」

カイトたちは桟橋から降りた。

足元は凍った地面。一歩踏み出すたび、氷が軋む音が響く。

「寒い……」

シオンがマントを強く抱きしめた。

「山脈に入ると、もっと寒くなるよ」

エリナが警告した。

「私の氷魔法で、ある程度は防げるけど、体力を消耗する」
「温かい魔法、使おうか？」

カイトが提案した。

「お願いします」

カイトは魔法を唱えた。


// 対象温度維持魔法
function maintainBodyTemp(targets, minTemp) {
  const magic = {
    type: "fire",
    subtype: "temperature_maintenance",
    targets: targets,
    minimum_temperature: minTemp,
    duration: "永続（魔力消費型）",
    efficiency: "高"
  };
  
  return `${targets}の体温を${minTemp}度以上に維持`;
}

// 実行
maintainBodyTemp(["カイト", "エリナ", "リアム", "シオン"], 36);

四人の体が、内側から温まった。

「助かる。ありがとう、カイト」

リアムが感謝した。

桟橋から少し歩くと、小さな村が見えた。

氷と石で作られた家々。煙突から煙が上がり、住民たちが防寒着を着て歩いている。

「氷見の村だ」

エリナが説明した。

「氷雪の山脈へ向かう冒険者たちが、よく立ち寄る場所」
「フローズナさんは、この村に？」

シオンが尋ねた。

「いや、彼女は山脈の奥に住んでる。でも、村で居場所を聞けるはず」

カイトたちは村の入り口に向かった。

村の入り口には、二人の警備員が立っていた。

厚い毛皮の防寒着に、氷でできた槍を装備している。

「止まれ。氷見の村へようこそ。目的は？」

一人の警備員が尋ねた。

「氷雪の山脈に入りたいんです。氷の賢者フローズナさんに会いたい」

カイトが答えた。

「フローズナ様に？　……君たちは、冒険者か？」
「はい。それと、管理者の仕事もしています」

警備員の表情が変わった。

「管理者……！　なら、通っていい。フローズナ様は、管理者の方を歓迎してるはずだ」

警備員は道を開けた。

カイトたちは村の中に入った。

石畳の道には、氷が張っている。住民たちは、滑らないよう慎重に歩いている。

「活気がある村だね」

リアムが観察した。

「山脈で取れる氷晶石を売ってるんだ。魔法道具の材料として、需要がある」

エリナが説明した。

「あそこの店、氷晶石を売ってるみたい」

シオンが指差した。

シオンの指差した店には、様々な色の氷が並んでいた。

青、緑、紫、そして透明。どれも光を反射して輝いている。

「きれい……」

シオンが見とれた。

「買っていく？」

カイトが提案した。

「いいんですか？」
「旅の記念になるし、何かの役に立つかもしれない」

シオンは青い氷晶石を一つ選んだ。

店の店主は、老婦人だった。

白い髪に、優しそうな目。長年、この村で暮らしてきた風格がある。

「おや、見ない顔だね。旅の方かい？」
「はい。氷の賢者フローズナさんに会いに来ました」

カイトが答えた。

「フローズナ様に？　……ふむ。山脈の奥、氷の聖域に住んでるはずだよ」

老婦人は地図を取り出した。

「この道を北に進んで、二日歩く。そうすると、氷の洞窟が見えてくる。その奥が、聖域だ」
「ありがとうございます」

カイトは地図を受け取った。

カイトたちは村の宿で昼食を摂った。

温かいスープと、硬いパン。質素だが、体が温まる。

「二日かかるんだね」

シオンが確認した。

「急ぐ必要はないけど、油断はできない。山脈には、危険な魔物がいる」

リアムが警告した。

「氷狼、氷熊、氷ゴーレム……。どれも強敵だ」
「昨日、氷海蛇と戦ったばかりなのに……」

シオンがため息をついた。

「大丈夫。四人で力を合わせれば、勝てる」

カイトが励ました。

午後、カイトたちは村を出発した。

北への道は、徐々に険しくなっていく。

道幅が狭くなり、両側には氷の崖。一歩間違えれば、谷底に落ちる。

「気をつけて。足元が滑りやすい」

エリナが先頭を歩いた。

「氷魔法で、道を整えられる？」

カイトが尋ねた。

「やってみる」

エリナは氷の地面に手を触れた。


// 氷道整備魔法
function smoothIcePath(path, friction) {
  const magic = {
    type: "ice",
    subtype: "surface_control",
    target: path,
    friction_coefficient: friction,
    duration: "一時的"
  };
  
  return `${path}の摩擦係数を${friction}に調整`;
}

// 実行
smoothIcePath("登山道", 0.7);

氷の表面が変化し、歩きやすくなった。

「さすが、エリナ」

リアムが称賛した。

夕方、カイトたちは洞窟を見つけた。

氷でできた天然のシェルター。風を避けるのに最適だ。

「今日はここで野営しよう」

カイトが提案した。

「洞窟の中なら、暖を取りやすい」

四人は洞窟に入った。

洞窟の中は、意外にも暖かかった。

外の冷気と隔絶され、気温が少し上がっている。

「地熱があるのかも」

リアムが推測した。

「氷雪の山脈の地下には、温泉があるって言われてる」
「温泉！？」

シオンの目が輝いた。

「入れるかな？」
「……野営中に温泉は、ないでしょう」

カイトが苦笑した。

夜、カイトは洞窟の入り口で見張りをしていた。

空には満天の星。そして、揺れるオーロラ。

「綺麗な夜だね」

リアムが隣に来た。

「うん。こんな景色、日本じゃ見られなかった」

カイトは呟いた。

「カイトは、よく日本の話をするね」
「……懐かしいのかな。元の世界に戻れるか分からないけど、時々思い出す」

カイトは星空を見上げた。

リアムはカイトを見た。

「俺たちは、カイトの仲間だ。カイトが戻るとしても、ついていく」
「リアム……」
「エリナも、シオンも、同じ気持ちだと思う。カイトは、俺たちのリーダーだから」

リアムは微笑んだ。

「頼りにしてるよ、カイト」

カイトは胸が熱くなった。

「ありがとう。僕も、みんなを頼りにしてる」

夜中、異変が起きた。

洞窟の外から、低い唸り声が聞こえた。

「何だ？」

カイトが立ち上がった。

「カイト、起きて！　何か来る！」

エリナが叫んだ。

洞窟の入り口に、巨大な影が現れた。

白い毛皮、青い目、鋭い爪。氷狼の群れだ。

氷狼は五匹。リーダーらしき一匹は、特に大きい。

「氷狼か。夜行性で、群れで狩りをする」

リアムが剣を抜いた。

「数が多い。囲まれないように！」

シオンが魔法の準備をした。

「雷魔法で、一気に仕留めます！」
「待って。殺す必要はない」

カイトが制止した。

氷狼たちは威嚇しているが、攻撃してこない。

「様子がおかしい。ただの狩りじゃない」

カイトは観察した。

「……子狼がいる」

カイトの視線の先には、小さな氷狼がいた。

足を怪我しているようだ。

「親狼たちが、子を守ろうとしてる。僕たちを敵だと思ってるんだ」

カイトは理解した。

カイトは剣を収めた。

「敵意はない。ただの旅人だ」

カイトはゆっくり手を挙げた。

「子狼を、治療させてくれ」

カイトは氷魔法を唱えた。


// 治癒魔法（氷属性変種）
function iceHealing(target, injury) {
  const magic = {
    type: "ice",
    subtype: "regeneration",
    target: target,
    injury_type: injury,
    speed: "高速"
  };
  
  return `${target}の${injury}を治療`;
}

// 実行
iceHealing("子狼の足", "骨折");

氷の光が子狼の足を包み、怪我が治っていった。

氷狼のリーダーが、カイトに近づいた。

青い目で、カイトを見つめる。

そして、頭を下げた。

「……礼をしてる」

エリナが驚いた。

「氷狼は、知性が高い魔物だ。恩を忘れない」

カイトは氷狼の頭を撫でた。

「気をつけて。山脈には、危険がいっぱいあるから」

氷狼たちは、夜の闇に消えていった。

朝、カイトたちは再び出発した。

昨日より道が険しくなっている。氷の壁を登らなければならない場所もある。

「これが、氷雪の山脈の本気か」

リアムが息を切らした。

「でも、もうすぐだ。地図によると、氷の洞窟は近い」

カイトが地図を確認した。

昼過ぎ、巨大な洞窟が見えた。

入口は氷でできたアーチ状の門。内部から、青白い光が漏れている。

「氷の聖域……」

シオンが息を呑んだ。

「フローズナさんは、この中にいる」

エリナが告げた。

「行こう」

カイトは洞窟の入り口に足を踏み入れた。

洞窟の中は、氷の宮殿のようだった。

壁も床も天井も、すべて氷。でも、寒くない。魔法で温度が調整されている。

「美しい……」

シオンが見回した。

「氷の賢者が作ったんだ。何百年もかけて」

エリナが説明した。

洞窟の奥に、人物が立っていた。

銀髪の女性。青い瞳。氷でできた杖を持っている。

氷の賢者、フローズナだ。

「久しぶりね、カイト」

フローズナは冷ややかな声で言った。

「フローズナさん。覚えていてくれたんですね」

カイトが頭を下げた。

「管理者の顔は、忘れないわ。それで？　何しに来たの」

カイトは目的を説明した。

「管理者の塔への道を、教えてほしいんです。封印を確認して、世界を守りたい」

フローズナはカイトを見つめた。

「……本気ね」

フローズナは杖を振った。

「封印は無事よ。でも、管理者の塔への道は険しい。覚悟はある？」
「あります」

カイトは真剣に答えた。

「……いいでしょう。案内するわ」

フローズナは微笑んだ。

「氷雪の試練、乗り越えられるかしら」

カイトの旅は、続く。

第753章、了。
