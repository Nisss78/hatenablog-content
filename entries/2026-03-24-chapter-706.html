第706章：世界の異変

中央ギルド本部の朝議は、重苦しい空気に包まれていた。

カイト、エリナ、リアム、シオン、そしてアルケミアは、広い会議室の中央に座っていた。周囲には、各国の支部長たちが hologram で参加していた。ヘヴンズ島、ルミナの街、レイズ大陸、氷見の村——世界各地の代表たちが一同に会していた。

テオドール支部長が口を開いた。

「諸君、集まってくれて感謝する。緊急の報告がある」

彼は手元の資料を広げた。

「過去一週間、世界各地で異常な現象が報告されている。報告数は、すでに四十七件に達した」

カイトは身を乗り出した。

「四十七件？ そんなに？」

テオドールは頷いた。

「ああ。しかも、増え続けている。場所も、内容も、バラバラだ」

彼は hologram に地図を表示させた。世界中に赤い点が散らばっていた。

「北の氷見の村では、氷が急速に融解し始めた。数日で、氷河が半分以上消えたという報告だ」

氷見の村の支部長——ガンツ自警団長——が口を開いた。

「実際に見たが、異常だ。季節はまだ冬だというのに、まるで真夏のような気候になっている。住民たちはパニック状態だ」

テオドールは続けた。

「南のレイズ大陸では、逆に異常寒波が発生している。灼熱の砂漠に雪が降ったという報告もある」

レイズ大陸の支部長——火の賢者イグニス——が告げた。

「私の部下が確認したが、砂漠の中心部で、氷柱が形成されていた。あり得ない現象だ」

テオドールはさらに続けた。

「東の海では、巨大な渦潮が発生し、船舶が航行不能になっている。西の山脈では、地響きが止まらない。そして、ここヘヴンズ島でも——」

彼は一瞬、言葉を詰まらせた。

「重力が乱れている。特定の場所で、物が浮き上がったり、逆に異常に重くなったりする現象が報告されている」

会議室がざわめいた。

アルケミアが口を開いた。

「要素の乱れね。七つの属性——炎、水、風、地、氷、雷、光——それぞれのバランスが崩れている」

シオンが母を見た。

「お母さん、これは境界の問題？」

アルケミアは首を振った。

「いいえ、境界は修復されたはず。これは、別の原因がある」

彼女は hologram に新たなデータを表示させた。

「私が分析した結果、これらの異変には共通点がある。すべて、魔法の源流——マナ・ストリーム——が乱れている場所で発生している」

カイトは眉をひそめた。

「マナ・ストリーム？」

アルケミアは説明した。

「世界中を流れる魔力の大河よ。地上の川と同じように、世界中に網の目のように広がっている。魔法を使う時、私たちはこのストリームから魔力を引き出している」

彼女は hologram に青い線を表示させた。世界中に、青い光の川が流れていた。

「通常、マナ・ストリームは安定して流れている。しかし、現在——」

青い線が、あちこちで赤く点滅していた。

「複数の場所で、ストリームが逆流したり、途絶えたりしている」

テオドールが告げた。

「つまり、世界の魔力システムそのものが、崩壊しかけているということか？」

アルケミアは静かに頷いた。

「その通り。このままでは、世界中の魔法が使えなくなる可能性がある。さらに悪いことに——」

彼女は一息ついた。

「マナ・ストリームが完全に崩壊したら、世界そのものが崩壊する可能性がある」

会議室が沈黙に包まれた。

カイトは沈黙を破った。

「原因は分かっているのか？」

アルケミアは hologram を操作した。

「原因と思われる場所を特定した。ここよ」

地図上の一点が、赤く光った。

「地下深く——世界の中心にある『原初の泉』。そこから、マナ・ストリームは湧き出している」

カイトは地図を見つめた。

「地下深く……そこに何があるんだ？」

アルケミアは告げた。

「分からない。原初の泉は、誰も到達したことがない場所。最深部——地球の核に近い場所にあると言われている」

テオドールが口を開いた。

「つまり、調査隊を送る必要があるということだな」

アルケミアは頷いた。

「ええ。ただ、簡単な任務ではないわ。地下深くへの道は、危険で未知の領域。そして、原初の泉には強力な守護者がいる可能性が高い」

カイトは立ち上がった。

「俺たちが行く」

エリナも立ち上がった。

「私たちも一緒よ」

リアムとシオンも頷いた。

テオドールは一同を見回した。

「分かった。カイト一行に、原初の泉への調査任務を命じる。目標は、マナ・ストリームの乱れの原因を特定し、解決すること」

カイトは深く頷いた。

「了解しました」

会議が終わった後、カイトたちは準備のために宿舎に戻った。

部屋で、エリナがカイトに声をかけた。

「ねえ、大丈夫？ 内なる虚無との戦い、続いてるんでしょ？」

カイトは弱々しく笑った。

「ああ。でも、大丈夫だ。シオンがモニタリングしてくれているし、みんなが支えてくれている」

エリナはカイトの手を握った。

「無理しないでね。私たちがいるんだから」

カイトは彼女を見つめた。

「エリナ……ありがとう」

その時、シオンが部屋に入ってきた。

「カイト、スキャンするわよ」

彼女は手をカイトの胸に当てた。七属性の魔法回路が光り、カイトの内部をスキャンした。


// Daily Void Monitoring
// Status Check Protocol

class DailyVoidMonitor {

        };

シオンは結果を見た。

「四十二パーセント。先週より二パーセント増えてる」

カイトは表情を曇らせた。

「悪化してるのか」

シオンは首を振った。

「でも、まだ『注意』レベルよ。危険域には達していない。ただ、この任務中は特に気をつけて。戦闘で魔法を使えば、虚無の浸食が進む可能性がある」

カイトは頷いた。

「分かった。なるべく魔法に頼らずに戦うよ」

アルケミアが部屋に入ってきた。

「準備はできた？」

カイトは立ち上がった。

「ああ。いつでも出発できる」

アルケミアは地図を広げた。

「原初の泉への入り口は、中央大陸の地下にある。そこから、世界最深部へと降りていくことになる」

彼女は地図上のルートを指し示した。

「途中、いくつかの危険なエリアを通過しなければならない。地底湖、マグマ地帯、古代の遺跡……そして、伝説の『深淵の回廊』」

リアムが口を開いた。

「深淵の回廊？ 何だそれ」

アルケミアは説明した。

「原初の泉の直前にある、最も危険な場所。物理的な空間ではなく、魂の空間よ。そこでは、自分自身の深淵と対峙することになる」

カイトは身震いした。自分自身の深淵——それは、内なる虚無と同じものかもしれなかった。

「俺にとっては、特に危険ってことか」

アルケミアは頷いた。

「ええ。でも、避けることはできない。原初の泉に到達するためには、必ず通らなければならない場所よ」

カイトは覚悟を決めた。

「分かった。覚悟して挑む」

出発の朝、カイトたちは中央ギルドの入り口に集まった。

テオドール支部長が見送りに来ていた。

「カイト、この任務は世界の命運を左右する。君たちに期待している」

カイトは頷いた。

「必ず成功させます」

テオドールは微笑んだ。

「それと、これを渡しておく」

彼はカイトに小さな水晶を手渡した。

「通信水晶だ。最深部でも、この水晶を通じれば中央ギルドと連絡が取れる」

カイトは水晶を受け取った。

「ありがとうございます」

エリナがカイトの肩を叩いた。

「さあ、行きましょう。世界が待ってるわ」

カイトたちは出発した。

目指すは中央大陸の地下——世界の中心、原初の泉。

中央大陸への道のりは、三日かかった。

馬で草原を駆け抜け、山脈を越え、そしてついに、地下への入り口に到達した。

それは、巨大な洞窟の入り口だった。口を大きく開けた怪物のように見えた。

リアムが怖気づいたように言った。

「うわ、暗いな。本当に中に入るのか？」

シオンが光魔法を発動させた。

シオンの手から、柔らかな光の球が浮かび上がった。洞窟の中が明るく照らされた。

「これで大丈夫」

カイトたちは洞窟の中へと足を踏み入れた。

洞窟の中は、想像以上に広かった。

天井は数十メートルの高さがあり、壁には不思議な発光苔が生えていた。シオンの光魔法がなくても、薄明るかった。

エリナが壁の苔に触れた。

「きれい……自然の光ね」

カイトは周囲を警戒しながら進んだ。洞窟には、何かが潜んでいる気配があった。

「気をつけて。何かいる」

その時、闇の中から影が飛び出してきた。

それは、巨大な蝙蝠の形をしていたが、体長は三メートル以上あった。目は赤く光り、鋭い牙が見えた。

「地底蝙蝠だ！ みんな、構えて！」

リアムが剣を抜いた。

「でかいな！ こんなの、普通の蝙蝠じゃないぞ！」

カイトは魔法を発動させた。

カイトの手に、黒い光の刃が現れた。虚無の力を使った剣だ。シオンが警告した通り、魔法を使うたびに虚無の浸食が進む。しかし、戦わなければならなかった。

地底蝙蝠がカイトに向かって飛んできた。カイトは刃を振った。

「はあっ！」

刃が蝙蝠の翼を切り裂いた。蝙蝠は悲鳴を上げ、地面に墜落した。

しかし、それだけでは終わらなかった。

闇の中から、さらに多くの蝙蝠が現れた。十、二十、三十——数え切れないほどの蝙蝠が、彼らを包囲した。

エリナが水魔法を発動させた。

「水の槍！」

無数の水の槍が蝙蝠たちを貫いた。しかし、蝙蝠の数は減らなかった。

リアムが叫んだ。

「無限に湧いてくるぞ！ どうなってるんだ！？」

シオンが敵の正体を見抜いた。

「幻影だ！ 本物は一匹だけ！ あれよ！」

彼女が指差したのは、洞窟の天井に張り付いている一匹の蝙蝠だった。他の蝙蝠より一回り大きく、体から黒い霧を発していた。

「あれが本体！ 幻影を作り出してる！」

カイトは本体に狙いを定めた。

「みんな、あいつを集中攻撃するぞ！」

四人は一斉に攻撃を開始した。カイトの虚無刃、エリナの水魔法、リアムの剣技、シオンの七属性魔法——すべてが本体に向かった。

本体の蝙蝠は悲鳴を上げ、霧散した。

それと同時に、周囲の幻影も消え去った。

カイトは息を切らした。

「やったか……」

シオンがカイトに駆け寄った。

「カイト、大丈夫？ 虚無のレベルをチェックするわ」

彼女は再びスキャンを行った。

「四十五パーセント……戦闘で三パーセント増えた」

カイトは顔をしかめた。

「やっぱり、魔法を使うと悪化するのか」

シオンは頷いた。

「ええ。なるべく温存して」

カイトは拳を握った。

「分かった。できるだけ物理攻撃で戦う」

彼らはさらに洞窟の奥へと進んだ。

しばらくすると、道が二手に分かれていた。

左の道からは、冷たい風が吹いていた。右の道からは、熱気を感じた。

エリナが言った。

「どっちに行く？」

カイトは考えた。

「地底湖とマグマ地帯……どっちも危険そうだが、地底湖の方が安全かもしれない」

アルケミアの地図を確認すると、両方のルートが最終的に合流することが分かった。

リアムが提案した。

「じゃあ、二手に分かれようか？ その方が早いし」

しかし、カイトは首を振った。

「いや、分かれない方がいい。俺の状態が不安定だから、できるだけ全員で行動したい」

エリナがカイトの手を握った。

「そうね。一緒に行きましょう」

彼らは左の道——地底湖への道——を選んだ。

地底湖は、美しかった。

広大な地下空間に、青く光る湖が広がっていた。水面は鏡のように静かで、天井の発光苔が反射して、星の海のように見えた。

シオンが感嘆した。

「きれい……まるで別世界ね」

カイトも見とれた。

「本当に……こんな場所が地下にあるなんて」

しかし、美しさには隠された危険があった。

湖の中央に、何かが浮かんでいた。それは、巨大な青い球体だった。直径数十メートル。中には、何かが封印されているように見えた。

「あれは……？」

エリナが目を細めた。

「魔法の結界？ 何かを守ってるのかしら」

カイトたちは湖の岸辺に近づいた。そして、結界の中に何がいるのかを見た。

それは、一人の女性だった。

長い青い髪。透き通るような肌。閉じた目。彼女は、眠るように結界の中に浮かんでいた。

シオンが息を呑んだ。

「あの人……ミズリア？」

カイトは驚いた。

「知ってるのか？」

シオンは頷いた。

「水の賢者ミズリア。かつて、リンド大陸を守っていた偉大な賢者よ。でも、何年も前に失踪したと聞いていた」

カイトは結界の中の女性を見つめた。

「なぜ、ここに封印されているんだ？」

その時、女性の目が開いた。

青い瞳が、カイトを捉えた。

「来たのね……カイト」

カイトは驚いた。

「俺の名前を知ってるのか？」

女性——ミズリア——は静かに告げた。

「知っているわ。あなたが……私の娘と同じ日に生まれたことを」

シオンが驚いた。

「娘？ ミズリアさん、娘がいたの？」

ミズリアは微笑んだ。

「いたわ。名前は……ミーナ。リンド大陸で、あなたたちと出会ったはず」

カイトは思い出した。以前、リンド大陸で出会った少女——ミーナ。彼女もまた、魔法使いだった。

「ミーナの母親……？」

ミズリアは頷いた。

「ええ。私は、原初の泉の秘密を知りすぎたため、ここに封印された。十三使徒によって」

カイトは身を乗り出した。

「十三使徒が？ 彼らは、もう倒されたはずだ」

ミズリアは首を振った。

「いいえ、倒されたのは一部だけ。十三使徒の真のリーダー——『第一席』は、まだ存在している」

カイトは衝撃を受けた。

「第一席……？ そいつは誰だ？」

ミズリアは告げた。

「名前は——エクリプス。全ての属性を統べる、最強の使徒」

カイトの冒険は、まだ終わっていなかった。
