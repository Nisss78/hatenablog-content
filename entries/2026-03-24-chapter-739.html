第739章：西風の導き

夜明け前の静寂が街を包んでいた。カイトは窓辺に立ち、西の空を眺めていた。昨夜感じた奇妙な予感——西の方角から漂う何か——が、朝になっても消えていない。

「起きるのが早いな」

背後からエリナの声がした。振り返ると、彼女が眠そうに目をこすりながら立っていた。

「西の方角に……何かがある気がするんだ」

カイトは窓の外を指差した。空はまだ薄暗く、星々が残っていた。

「七賢者たちも同じことを言っていたわ。西の大陸で異変が起きているって」

エリナがカイトの隣に立った。二人の視線が同じ方角に向けられる。

数時間後、賢者の塔の一室に七人の賢者が集まっていた。カイトとエリナ、そして仲間たちも同席している。

「昨夜、西の空に奇妙な光が現れた」

光の賢者ルミナスが説明を始めた。彼女の手には、古びた巻物が握られていた。

「これは数百年前に記された予言書の一部よ。『西の果てに封じられしもの、星の巡りと共に目覚めん』と書かれている」
「封じられたもの……？　何が封印されているんですか？」

シオンが首を傾げて尋ねた。

「正確には分からない。ただ、その封印が解けかけていることは確かだ」

地の賢者テラドンが重々しく答えた。

「大地の脈動が乱れている。西の方角から、異質なエネルギーが流れ込んでくるのを感じる」

カイトは腕を組んで考え込んだ。

「その封印を調査する必要がありますね。僕が行きましょうか？」
「待て、カイト」

リアムが手を挙げた。

「お前だけでは危険だ。俺も同行する」
「私も！」

シオンが元気よく手を挙げた。

「七属性の魔法が使える私なら、役に立つはず！」

カイトは少し迷ったが、頷いた。シオンの力は確かに心強い。彼女の体内には七つの魔法回路が流れており、全属性の魔法を行使できるのだ。

「分かった。三人で行こう。でも、準備が必要だ」

カイトは賢者たちに向き直った。

「西の大陸について、何か情報はありますか？」
「西の大陸は『忘却の大地』と呼ばれている」

闇の賢者アンブロスが口を開いた。彼の深い声が部屋に響く。

「かつては栄えていたが、今は誰も住んでいない。古代の遺跡と化した都市が点在するのみだ」
「なぜ人は住まなくなったのですか？」

エリナが尋ねた。

「封印された『何か』の影響だと言われている。それが何なのか、我々も知らない」

風の賢者ウィンディアが翼のように軽やかな動作で立ち上がった。

「ただ一つ言えることは、風の流れが変わっているということ。西からの風に、奇妙なざわめきが混じっているのよ」

カイトは深呼吸をした。状況は複雑だが、やるべきことは明確だ。

「まずは西の大陸に渡り、調査を始めましょう」

準備を終えた三人は、港町ルミナに向かった。ここから西の大陸へ渡る船が出ている。

「久しぶりの船旅ね」

シオンが海風に髪をさらされながら言った。銀色の長い髪が風に舞う。

「前回は大変だったからな。今回は平和だといいけど」

リアムが苦笑した。彼の背中には、愛用の大剣が背負われている。

港には数隻の船が停泊していた。その中で一際目立つ船——「暁の翼号」——の前に、一人の男が立っていた。

「カイトさんですね？」

男は三十代半ばくらい。日焼けした肌に、海のように青い目をしている。

「はい、そうです」
「ヴィクトル船長から話を聞いています。私は副船長のマルコです。西の大陸までお送りします」

マルコはにこりと笑った。

「ただ、この時期の西航は危険なんです。嵐が頻発していてね」
「それは……何か関係があるのでしょうか？」

カイトが尋ねると、マルコは表情を曇らせた。

「正直、分かりません。ただ、奇妙な話があるんです。最近、西の海で船が消えるという噂が流れていてね」
「消える？」

シオンが目を丸くした。

「ああ。嵐に巻き込まれた船が、そのまま姿を消すんだ。残骸も何も見つからない」

カイトとリアムは顔を見合わせた。ただの嵐ではない。何かが起きている。

「それでも、行く必要があります」

カイトは決意を込めて言った。

「もちろんです。覚悟はおありでしょう」

マルコは頷き、船へと案内した。

「暁の翼号」は夕暮れ頃に出港した。西の空は燃えるような赤色に染まり、やがて闇に包まれていく。

カイトは甲板でコードを書いていた。魔法をプログラミングする彼の能力は、こうした静かな時間に最も発揮される。


// 西の封印調査用スキャン
class WesternSealScanner {
  constructor() {

  }

    let readings = {};
    for (let elem of this.elements) {
      readings[elem] = this.measureElement(elem);
    }
    return this.analyze(readings);
  }
  
  measureElement(element) {
    // 魔法回路を通じて元素を計測
    return manaFlow.analyze(element);
  }

    // 異常値の検出
    let anomalies = [];
    for (let [key, value] of Object.entries(readings)) {
      if (value > this.threshold) {
        anomalies.push({ element: key, level: value });
      }
    }
    return anomalies;
  }

「何か分かった？」

シオンが後ろから覗き込んだ。

「まだ分からない。でも、海上の魔力の流れが不自然なんだ」

カイトは眉をひそめた。

「水と風、二つの元素が互いに打ち消し合っている。本来なら共存できるはずなのに」
「何かが干渉しているってこと？」

リアムが割り込んだ。

「そう考えるのが妥当だね。封印が解けかけているせいで、魔力のバランスが崩れているのかもしれない」

その時、船が大きく揺れた。

「何だ！？」

リアムがバランスを崩しそうになりながら叫んだ。

海面が泡立ち始めた。まるで何かが下から押し上げているかのように。

「来るぞ！」

カイトは瞬時に戦闘態勢をとった。コードを脳内で構築し、魔法を発動させる準備をする。

海から現れたのは、巨大な影だった。正体は不明だが、複数の触手のようなものを持ち、全身が半透明の闇に覆われている。

「何なの、あれ！？」

シオンが叫んだ。

「分からない。でも、倒すしかない！」

カイトは詠唱を開始した。

金色の光の鎖が空中に現れ、影の怪物に向かって飛んでいった。鎖は怪物の触手を縛り上げ、動きを封じようとする。

だが、怪物は触手を振り回して鎖を引きちぎった。

「効かない！？」

カイトは驚愕した。光魔法は闇属性の敵に効果的であるはずなのに。

「こいつは普通の闇じゃない！」

リアムが剣を構えて前へ出た。

「俺が引き受ける。お前はシオンと共に魔法で支援してくれ！」
「分かった！」

リアムの大剣が怪物の触手を切り裂いた。だが、切られた触手はすぐに再生していく。

「再生能力があるのかよ！」

シオンが七属性の魔法を同時に発動させた。


// 七属性融合魔法：プリズム・バースト
function prismBurst() {
  const elements = ["fire", "water", "wind", "earth", "light", "dark", "thunder"];
  let combined = null;
  
  for (let elem of elements) {
    combined = manaFuse(combined, elements.summon(elem));
  }
  
  return combined.release();
}

七色の光が怪物を包み込んだ。爆発と共に、怪物の体の一部が消滅していく。

「今だ、カイト！」

シオンの声に応えて、カイトは最強の魔法を構築した。

function voidReturn(target) {
  const void = elements.summon("void");

  // 対象の存在そのものを否定

黒い穴が怪物の中心に現れた。そして、怪物は音もなく吸い込まれていった。

海面に波紋が広がり、やがて静寂が戻った。

戦いの後、三人は甲板で息を整えていた。

「あれは……何だったんだ？」

リアムが汗を拭いながら尋ねた。

「封印が解けた影響だと思う」

カイトは冷静に分析した。

「本来なら存在してはいけないものが、現世に漏れ出している。あの怪物は、その一端だろう」
「じゃあ、封印を解くのは危険なんじゃない？」

シオンが不安そうに尋ねた。

「封印が自然に解けているなら、放置する方が危険だ。正しい方法で解き、中身を制御する必要がある」

カイトの言葉に、二人は頷いた。

夜が更けていく。西の空には、微かに光る何かが見えた。それは目的地の方向を示しているようだった。

「あの光……封印の場所かな」

シオンが指差した。

「たぶん。明日には到着するはずだ」

カイトは決意を新たにした。西の果てに何があるのか、そして封印されたものの正体とは。全ての答えは、明日明らかになる。

海風が三人の髪を撫でていった。冒険の序章は、まだ始まったばかりだった。

翌朝、船は西の大陸の港に到着した。そこはかつて栄えた港町の跡だったが、今は廃墟と化している。

「ここが……忘却の大地」

エリナが呟いた。彼女も密かに同行していたのだ。

「エリナ！？　いつの間に！」

カイトが驚いて振り返った。

「ごめんなさい。どうしても心配で」

エリナは申し訳なさそうに微笑んだ。

「もう遅いから、一緒に行こう」

カイトはため息をついたが、拒否はしなかった。彼女の力は確かに心強い。

四人は港を後にし、内陸へと向かった。西の大陸——忘却の大地——そこに待つ真実へと。

カイトのプログラマーとしての能力と、仲間たちの力。それら全てを結集させなければ、この冒険を生き抜くことはできないだろう。

西風が四人の背中を押した。まるで、「行け」と促しているかのように。

第739章、了。
