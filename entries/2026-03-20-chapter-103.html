第103章：決意の楔

黄昏の光が収束し、静寂が戻ってきた廃墟の庭園。カイトは膝をつき、荒い息を吐き出していた。魔力回路が悲鳴を上げている。連続して高位の魔法を展開した代償だ。

「カイト、大丈夫？」

エリナが駆け寄り、彼の肩を支える。その表情には隠しきれない不安が滲んでいた。

「……ああ。少し魔力を使いすぎただけだ。問題ない」

カイトは立ち上がり、庭園の奥を見つめた。そこには、アークライトとの戦いで崩壊した祭壇の残骸が散らばっている。黄昏の楔と呼ばれた古代の遺物は、今は静まり返り、何の反応も示していない。

「しかし、これで終わったとは思えんな」

リアムが剣を鞘に収めながら、低い声で言った。彼の視線もまた、祭壇の方へ向けられている。

「管理者たちはまだ動いている。ヴォルターもミラベルも、そして彼らの背後にいる何かも……」

セシリアが静かに頷く。彼女の魔導書が、微かな光を放っていた。先ほどの戦いで得た情報を解析しているのだろう。

「カイト、見て。黄昏の楔の内部構造が見えてきたわ」

彼女が空中に魔法陣を展開する。複雑な幾何学模様が、次々と浮かび上がっていく。

「これは……プログラム？」

カイトは目を細めた。魔導書の解析結果が、彼のよく知る言語に似た構文として可視化されている。

STRUCT TwilightWedge {
    AnchorPoint: Coordinates;
    ManaFlow: Vector3D;
    SealCondition: Boolean;
    Trigger: Event;
}

驚くべき発見だった。この世界の魔術は、彼がいた世界のコードと根本的に同じ構造を持っていたのだ。魔法陣は関数定義、ルーン文字は変数、詠唱はコンパイル──すべてが対応している。

「やっぱりそうか。この世界の魔術は、本質的にコードなんだ」

カイトの脳内で、電撃のような閃きが走った。もしこの仮説が正しければ、彼ができることは単なる魔法の行使にとどまらない。魔術そのものを再定義し、書き換えることができるはずだ。

「カイト？ どうしたの、変よ」

エリナの声で、カイトは我に返った。彼の体から、知らず知らずのうちに魔力が溢れ出していた。

「いや、なんでもない。ただ……確信しただけだ」

彼は拳を握りしめた。掌に残る、カーネリアンの感触。彼をこの世界に召喚した触媒であり、彼の力の源でもある。

「僕がこの世界に来た理由。それは偶然じゃない。誰かが、僕を呼んだんだ」

その言葉に、全員が反応した。エリナ、リアム、セシリア──そして、柱の影から現れたアルフレッドもまた、厳しい表情を浮かべている。

「守人殿。あなたも何かご存じなのでは？」

カイトの問いかけに、アルフレッドは沈黙した後、ゆっくりと口を開いた。

「……守人の伝承には、こうある。『異界より来たりし者、世界の理を紡ぎ直さん』と」

空気が重くなる。その予言は、カイトに過剰な重圧を与えるものだった。しかし、同時に奇妙な納得感もあった。彼がこの世界で魔法をコードとして理解できたこと、古代の遺物と共鳴したこと──すべてが繋がっていたのだ。

「理を紡ぎ直す、か。……世界そのものを書き換えろってことか」

カイトは自嘲気味に笑った。プログラマーとして、バグだらけのレガシーシステムをリファクタリングすることは日常茶飯事だった。だが、世界そのものを対象にするとなると話は別だ。

「カイト、無理はしないで」

エリナが彼の手を握った。温かい。彼女の掌から、不安と、それ以上の何かが伝わってくる。

「あなた一人の肩に、全部背負わなくていいのよ。私たちがいる」

リアムが剣の柄に手を置き、静かに頷く。

「同感だ。俺たちはチームだ。お前が指揮官なら、俺たちはその指示に従うまで」

セシリアも、魔導書を閉じて微笑んだ。

「解析は私に任せて。あなたはコードを書くことに集中して」

アルフレッドは一歩前に出て、深々と頭を下げた。

「守人として、あなた方の旅路を守護することを誓います。我が命に代えても」

カイトは、仲間たちの顔を見回した。信頼。絆。この世界に来て初めて得た、かけがえのないものたちだ。

「……ありがとう。みんな」

彼は深呼吸をし、決意を新たにした。

「次の目的地は、『始原の図書館』だ。そこに、この世界の魔術システムの設計図があるはずだ」

始原の図書館。伝説によれば、世界創造の時代から存在すると言われる禁断の知識の宝庫。管理者たちでさえ、容易には立ち入れない場所だ。

「始原の図書館って……あの？」

セシリアの顔色がさっと変わった。

「無限の回廊と呼ばれる、空間迷宮よ。一度入ったら二度と出られないという……」

「出られるさ」

カイトは断言した。その瞳には、確信の光が宿っていた。

「迷宮も、所詮はシステムだ。入口と出口、ルールとロジック。それさえ解析できれば、攻略は可能だ」

エリナが呆れたように、しかし安心したように笑った。

「本当に、あなたっていう人は……」

「褒め言葉として受け取っておくよ」

カイトも笑った。重苦しい空気が、少し和らいだ気がした。

その時、カイトの懐でカーネリアンが発光した。警告のような、そして案内のような光。

// Quest Update
// Objective: Reach the Primordial Library
// Estimated Distance: 2,400 km
// Danger Level: Critical
// Recommended Level: 50+

空中に浮かび上がったステータス画面のようなものに、仲間たちは目を見張った。

「カイト、今の……？」
「僕の能力の一部だ。どうやら、クエスト管理機能も持っているらしい」

カイトは苦笑した。まるでRPGのUIのようなこの能力。彼がこの世界で「異邦人」として持つ特殊性の一部だ。

「二千四百キロか。徒歩じゃ一月はかかるな」

リアムが距離を計算しながら呟く。

「馬車を雇うか、転移魔法を使うか……」
「転移魔法なら、私がやれるわ」

セシリアが手を挙げた。

「ただし、全員まとめてとなると準備が必要ね。明日の朝出発としましょう」

カイトは頷いた。今の疲労状態で、長距離転移を試みるのは危険だ。一度休息を取る必要がある。

「わかった。今夜はここで野営しよう。明日の朝、始原の図書館へ向かう」

全員が頷き、それぞれの役割を果たし始めた。テントを張るリアム、警戒網を敷くアルフレッド、火を起こすエリナ、そして魔力回復の儀式を行うセシリア。

カイトは一人、夜空を見上げた。見慣れない星座が、彼に微笑みかけているような気がした。

（……始まったばかりだな、本当に）

彼は、自分の掌を見つめた。その中心にあるカーネリアンが、静かに脈打っている。まるで心臓のように。

「カイト、ご飯できたわよ」

エリナの声が、彼を現実に引き戻した。

「ああ、今行く」

彼は立ち上がり、仲間たちの元へと歩き出した。夜は長い。だが、明日は必ず来る。そして彼は、歩みを止めるつもりはなかった。

決意の楔（くさび）は、打ち込まれた。後は、前へ進むだけだ。

【用語解説】

  • 黄昏の楔：古代の遺物。世界中の魔力の流れを制御する機能を持つとされる。
  • 始原の図書館：世界創造の時代から存在する伝説の図書館。無限の回廊と呼ばれる迷宮。
  • カーネリアン：カイトをこの世界に召喚した触媒。彼の能力の源でもある。
  • 管理者（アドミニストレーター）：世界の秩序を監視する存在。カイトたちと敵対している。
