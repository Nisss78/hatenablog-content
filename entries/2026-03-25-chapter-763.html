第763章：水の宮殿への道

カイトたちは、波打ち際に立った。

青い海が、果てしなく広がっている。

太陽の光が、水面をキラキラと照らしていた。

「きれいな海だね」

エリナが呟いた。

「透き通るような青さ」
「でも、ただの海じゃないわ」

シオンが警戒した。

「水の賢者の領域。普通の海とは違う」
「何かが待ってるかもしれない」

カイトは海を見つめた。

静かに見える。でも、奥底には何かが潜んでいる。

水の賢者、ミズリア。

彼女に会うには、海の中に入らなければならない。

「シオン、水中呼吸の魔法、頼める？」

カイトが尋ねた。

「ええ、任せて」

シオンが頷いた。

彼女は、水の魔法を得意としていた。

七属性の魔法回路を持つシオン。水属性も扱える。


// 水中呼吸魔法
function waterBreathing(targets) {
  const spell = {
    type: "水",
    subtype: "buff",
    targets: targets,
    effect: "水中で自由に呼吸可能",
    duration: "長時間",
    range: "全員"
  };
  
  return `${targets.join("、")}の周りに水の膜ができる`;
}

// 実行
waterBreathing(["カイト", "エリナ", "リアム", "シオン"]);

シオンが魔法を唱えた。

四人の周りに、薄い青い膜ができた。

この膜があれば、水中でも呼吸ができる。

「準備完了」

シオンが言った。

「よし、行こう」

カイトが先頭に立った。

四人は、海の中へと歩き出した。

水の中に入ると、世界が変わった。

音が静かになる。

光が、水面を通して柔らかく差し込む。

魚たちが、驚いて逃げていく。

「すごい」

エリナが感嘆した。

水中でも、シオンの魔法のおかげで普通に話せる。

「海の中、別世界みたい」
「水の賢者の宮殿は、もっと深い」

リアムが言った。

「海底まで潜る必要がある」
「覚悟して」

カイトたちは、さらに深く潜った。

周りが暗くなっていく。

でも、不思議と怖くない。

水の力が、優しく包んでくれている。

しばらく潜ると、光が見えた。

海底に、青白い光が灯っている。

その光に導かれるように、進んでいく。

そして、目の前に巨大な建物が現れた。

白い大理石で作られた、美しい宮殿。

珊瑚や貝殻で飾られた、水の宮殿。

「これが、水の賢者の宮殿」

シオンが言った。

宮殿の入り口に、二人の守衛がいた。

水の精霊。半透明な体を持つ、水の守護者。

彼らが、カイトたちを止めた。

「止まれ」

左の守衛が言った。

「ここは、水の賢者ミズリアの宮殿」
「何用だ？」

カイトは、ウィンディアからもらった風の手紙を取り出した。

水の中でも、風の封筒は崩れなかった。

「風の賢者ウィンディアの紹介で来ました」

カイトが言った。

「水の賢者に会いたいんです」
「試練を受けに来た」

守衛が、風の手紙を受け取った。

封筒を開けると、ウィンディアの声が流れた。

「ミズリア、この若者を試してあげて」

ウィンディアの声が水中に響いた。

守衛たちは顔を見合わせた。

「ウィンディアの紹介か」

右の守衛が言った。

「なら、通っていい」
「ミズリア様が待っている」

カイトたちは、宮殿の中に入った。

廊下は、水で満たされている。

でも、不思議なことに乾いているように感じる。

水の魔法の応用だ。

宮殿の奥に、大きな扉があった。

扉には、波の模様が彫られている。

水属性の紋章。

カイトが扉を開けた。

中は、広い玉座の間だった。

玉座に、一人の女性が座っていた。

青い髪。青いドレス。

年齢不詳。若くも見えるし、古くも見える。

でも、美しかった。

水のように、静かで透明な美しさ。

「ようこそ、カイト」

女性が立ち上がった。

「私は、水の賢者ミズリア」
「ウィンディアから連絡を受けたわ」
「初めまして、ミズリアさん」

カイトは頭を下げた。

「僕は、カイト」
「七属性の力を完成させたくて、ここに来ました」
「七属性、ね」

ミズリアが興味深そうに見た。

「炎と風は、もう習得したのね」
「水を学びに来たというわけ」
「はい」

カイトは頷いた。

「水の試練を受けさせてください」

ミズリアは、カイトをじっと見た。

青い瞳が、カイトの心を見透かすようだ。

カイトは、緊張しながら耐えた。

「いい目をしているわ」

ミズリアが微笑んだ。

「真っ直ぐで、迷いがない」
「水の賢者に相応しい素質がある」
「ありがとうございます」

カイトは安堵した。

「でも」

ミズリアが指を立てた。

「水の試練は、簡単ではないわ」
「炎の情熱や風の自由とは違う」
「水は、もっと静かで、もっと深い」
「どんな試練なんですか？」

カイトが尋ねた。

ミズリアは、窓の方を向いた。

窓の外は、深海の風景。

魚たちが泳ぎ、珊瑚が揺れている。

「水の試練は、三つあるわ」

ミズリアが語り始めた。

「第一の試練は、『水の静寂』」
「水のように静かになり、心を澄ますの」
「第二の試練は、『水の包容』」

ミズリアが続けた。

「水は、全てを受け入れる。どんな汚れも、どんな痛みも」
「それを学ぶの」
「第三の試練は、『水の激流』」

ミズリアが振り返った。

「水は静かだけど、怒ると恐ろしい」
「水の力を戦いで使う試練よ」

カイトは、三つの試練を聞いた。

静寂、包容、激流。

風や炎とは、また違う試練だ。

「受けます」

カイトは決意した。

「三つの試練、全て受けます」
「いい返事ね」

ミズリアが満足そうに頷いた。

「じゃあ、今日は休んで」
「明日の朝、第一の試練を始めるわ」
「え、今日はもう始めないんですか？」

カイトが意外そうに尋ねた。

「水は急がないの」

ミズリアが笑った。

「ゆっくり流れる」
「あなたも、水のように学びなさい」

カイトは、ミズリアの言葉に納得した。

水は急がない。自然に流れる。

それが、水の哲学なのだ。

「案内してあげる」

ミズリアが手を叩いた。

水の精霊が現れた。

「客室へ案内して」

ミズリアが指示した。

水の精霊に案内され、カイトたちは客室に着いた。

広い部屋。ベッドが四つ。

窓からは、深海の風景が見える。

「すごい部屋」

エリナが感嘆した。

「水の中なのに、快適」
「水の魔法は、生活にも役立つ」

シオンが解説した。

「水を制御できれば、水中でも快適に過ごせる」
「カイトも、いつかできるようになるわ」

カイトは窓の外を見た。

深海の闇。でも、怖くない。

不思議な光が、所々で灯っている。

深海魚が、優雅に泳いでいく。

「水の賢者、ミズリア」

カイトは呟いた。

「静かだけど、深い人だ」
「水属性の特徴よ」

リアムが言った。

「水は、見た目より複雑」
「表面は静かでも、底には激流がある」
「明日の試練、楽しみだ」

カイトは笑った。

「水の静寂、か」
「一体どんな試練なんだろう」

夜が来た。

水中の宮殿でも、夜は夜だった。

光が弱まり、静寂が訪れる。

カイトはベッドに横たわった。

水の中で眠る。不思議な感覚。

でも、心地よかった。

夢を見た。

広い海。一人で漂っている。

水が優しく包んでくれる。

静かで、安心。

朝が来た。

カイトは目を覚ました。

体が軽い。よく眠れた。

客室のドアがノックされた。

「おはよう、カイト」

ミズリアの声だった。

「準備はいい？」
「はい」

カイトはすぐに起きた。

「いつでも行けます」

カイトたちは、ミズリアに案内された。

宮殿の中を進み、別の部屋に着いた。

そこは、円形の広間だった。

中央に、大きな水槽があった。

中には、透き通った水。

何も入っていない。

「ここで、第一の試練を行うわ」

ミズリアが言った。

「水の静寂」
「この水槽の中に入って」

ミズリアが指示した。

「そして、水と一つになるの」
「心を静めて、水の声を聞く」
「水の声？」

カイトが首を傾げた。

「水には、声があるわ」

ミズリアが微笑んだ。

「でも、静かじゃないと聞こえない」
「雑念があると、水の声は消える」

カイトは、水槽に近づいた。

中を覗く。

ただの水に見える。

でも、試練なら、何かあるはずだ。

「やります」

カイトは、水槽の中に入った。

水が、体を包む。

冷たい。でも、痛くない。

カイトは、水の中で目を閉じた。

心を静める。

何も考えない。

ただ、水を感じる。

最初は、何も聞こえなかった。

静寂だけ。

でも、徐々に何かが聞こえてきた。

水のささやき。

「カイト」

水の声が、名前を呼んだ。

「あなたは、何を求めている？」

カイトは、心の中で答えた。

「七属性の力を完成させたい」

「仲間を守りたい」

「この世界を救いたい」

水の声は、続けた。

「なぜ、それを求める？」

「力そのものが目的ではないはず」

「本当の目的は、何？」

カイトは、深く考えた。

なぜ、力を求めるのか。

強くなりたい。それは確かだ。

でも、なぜ？

「大切な人を、守りたいから」

カイトは答えた。

「エリナ、リアム、シオン」

「彼らと一緒に、生きたい」

「そのために、力が必要なんだ」

水の声は、沈黙した。

そして、再び語りかけた。

「いい答えね」

「力は、守るために使うべき」

「それが、水の真理」

水が、カイトを優しく包んだ。

温かい感覚。

水と一つになった気がした。

カイトが目を開けると、水槽の外にいた。

ミズリアが、微笑んでいた。

「合格よ」

ミズリアが言った。

「第一の試練、水の静寂」
「あなたは、水の声を聞いたわ」
「水の声……」

カイトは、不思議な感覚を思い出した。

「水が、僕に問いかけた」
「水は、心を映すの」

ミズリアが説明した。

「あなたの心が真っ直ぐだから、水も真っ直ぐに応えた」
「もし、邪心があれば、水は冷たかった」
「第一の試練、クリア」

カイトは安堵した。

「次は、第二の試練ですね」
「ええ」

ミズリアが頷いた。

「でも、今日はここまで」
「第二の試練は、明日」
「また、明日なんですか？」

カイトが驚いた。

「水は急がない」

ミズリアは再び言った。

「試練も、一日一つ」
「それが、水の流儀よ」

カイトは、ミズリアの言葉に従った。

水の試練は、ゆっくり進む。

それが、水のやり方だ。

カイトたちは、客室に戻った。

エリナが、心配そうに聞いた。

「どうだった？ 第一の試練」
「不思議だった」

カイトは振り返った。

「水と話した気がした」
「水が、僕の心を問いかけたんだ」
「水は、心を映す鏡みたいなものよ」

シオンが言った。

「水の試練は、自分自身と向き合う試練」
「カイトは、合格したんだね」
「うん」

カイトは笑った。

「水の賢者、ミズリア」
「厳しいけど、優しい人だ」

夜が来た。

二日目の夜。

カイトは、また夢を見た。

夢の中で、水の声が聞こえた。

「明日は、包容の試練」

「水は、全てを受け入れる」

「あなたに、できるかしら？」

カイトは、夢の中で頷いた。

「やってみる」

「水のように、全てを受け入れる」

朝が来た。

カイトは、決意を新たにした。

第二の試練、水の包容。

いよいよ、始まる。

第763章、了。
