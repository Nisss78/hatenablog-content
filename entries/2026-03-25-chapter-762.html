第762章：風神剣との激突

ウィンディアが風神剣を振った。

刃から見えない風の刃が飛んだ。

カイトは炎帝の剣で受けたが、衝撃が体を震わせた。

「速い……！」

カイトは後退した。

ウィンディアは追撃した。

風に乗って、瞬時にカイトの背後に回る。

「後ろよ」

ウィンディアの声が耳元で響いた。

カイトは振り向きざまに剣を振った。

でも、ウィンディアはもういない。

風だけが残った。

「どこだ？」

カイトは周囲を見回した。

「ここ」

声は上から聞こえた。

ウィンディアが空中に浮いていた。

「風は、どこにでも在れるのよ」

ウィンディアが風神剣を突き出した。

風の衝撃波が降ってきた。

カイトは炎帝の剣で防御魔法を展開した。


// 風への対抗防御
function createDefense(user) {
  const barrier = {
    type: "炎",
    subtype: "barrier",
    target: user,
    effect: "炎の障壁で風の攻撃を相殺",
    power: "high"
  };
  
  return `${user}の周りに炎の障壁が展開される`;
}

// 実行
createDefense("カイト");

炎の障壁と風の衝撃がぶつかった。

熱と風が混ざり合い、蒸気が発生する。

視界が一瞬白くなった。

「いい反応ね」

ウィンディアの声が、蒸気の中から聞こえた。

「でも、それだけじゃ足りないわよ」

風が蒸気を一瞬で吹き飛ばした。

ウィンディアが突進してきた。

風神剣が、炎帝の剣と激突した。

金属音が響き、火花が散る。

「くっ……！」

カイトは力で押し返した。

「力強いわね」

ウィンディアが評価した。

「炎帝との戦いで、腕を上げたみたい」
「でも、風は力じゃない。速さと変化よ」

ウィンディアが剣を引いた。

その瞬間、体が風のように消えた。

また背後に現れた。

「速すぎる」

カイトは回転して剣を振った。

剣がウィンディアの風を切り裂いた。

でも、彼女の体には当たらない。

ウィンディアは、風と一体化して攻撃を避けていた。

「風の賢者……やっぱり強い」

カイトは舌を巻いた。

エリナたちが、遠くから見守っていた。

「カイト、速すぎて見えない」

エリナが心配そうに言った。

「ウィンディアさんは、風そのものだ」

リアムが解説した。

「物理攻撃は、ほとんど効かない」
「カイトはどう戦うつもりなんだ？」
「カイトには、風の力もある」

シオンが言った。

「昨日の試練で、風と同調した」
「風を感じれば、ウィンディアさんの動きが読めるはず」

カイトは、シオンの言葉を聞いたわけじゃない。

でも、本能的に同じ結論に至った。

「目で追うんじゃない。風を感じるんだ」

カイトは目を閉じた。

目を閉じて、風の流れを感じる。

ウィンディアの動きが、風の乱れとして感じられる。

右から。上から。背後から。

風が教えてくれる。

「そこ！」

カイトは右に剣を振った。

ウィンディアの風神剣と、炎帝の剣が交差した。

「えっ？」

ウィンディアが驚いた。

「目を閉じていて、どうしてわかったの？」
「風が教えてくれた」

カイトは目を開けた。

「風の賢者の動きは、風を乱す。その乱れを感じたんだ」
「なるほど」

ウィンディアは微笑んだ。

「第一の試練を活かしてるわね」
「でも、本気はまだよ」

ウィンディアが両手を掲げた。

周囲の風が、巨大な竜巻になった。

「風神竜巻」

ウィンディアが技を叫んだ。

竜巻がカイトに向かった。

風の刃が、中で無数に渦巻いている。

防御しても、削り取られる。

「これをどう防ぐ？」

ウィンディアが挑んだ。

カイトは考えた。

防御では無理。回避も難しい。

なら、どうする？

「風には、風で」

カイトは風の力を込めた。


// 風の竜巻への対抗
function windCounter(user) {
  const counter = {
    type: "風",
    subtype: "counter",
    user: user,
    effect: "竜巻の流れを利用して、内部に侵入",
    risk: "high"
  };
  
  return `${user}が竜巻の流れに乗る`;
}

// 実行
windCounter("カイト");

カイトは、竜巻に向かって飛んだ。

逆らうのではなく、風の流れに乗る。

風が体を包み、竜巻の中心へと導く。

「中に入る気？」

ウィンディアが驚いた。

竜巻の中は、意外と静かだった。

風の目。嵐の中心。

そこで、カイトは剣を構えた。

「今だ！」

カイトは炎の力を込めた。

炎帝の剣が、赤く輝いた。

竜巻の中で、炎が爆発した。

「炎風剣！」

カイトが技を叫んだ。

炎と風が混ざり合い、熱風の衝撃波となった。

竜巻が内側から崩壊した。

熱風が四方に広がる。

「なっ……！」

ウィンディアは、熱風を防御した。

でも、衝撃で少し後退した。

「風と炎の融合技。私の竜巻を破ったわね」

ウィンディアは感心した。

カイトは着地した。

少し息が上がっている。

「風の試練で学んだ。風を支配するんじゃない。利用するんだ」

カイトは言った。

「成長が早いわね」

ウィンディアが再び構えた。

「でも、まだ終わらないわよ」
「風神剣、真の力」

ウィンディアの風神剣が、緑色に光った。

剣から、風の精霊が現れた。

無数の精霊が、剣の周りを舞う。

「風精霊剣」

ウィンディアが技を告げた。

風精霊が、カイトに向かった。

一つ一つが小さな風の刃。

でも、数が数百。

全てを防ぐのは不可能に見えた。

「どうする、カイト？」

リアムが叫んだ。

カイトは、昨日のことを思い出した。

風の声。風の哲学。

「風は孤独ではない。常に誰かと交わっている」

「一人で防ぐ必要はない」

カイトは七属性の力を解放した。

「七属性よ、僕を守れ」

// 七属性の総合防御
function elementalShield(user) {
  const shield = {
    type: "七属性",
    subtype: "composite",
    user: user,
    elements: ["炎", "水", "風", "地", "光", "闇", "雷"],
    effect: "全属性の障壁を展開",
    synergy: "elements amplify each other"
  };
  
  return `${user}の周りに七色の障壁が現れる`;
}

// 実行
elementalShield("カイト");

カイトの周りに、七色の障壁が現れた。

炎、水、風、地、光、闇、雷。

七つの属性が、互いを強め合い、絶対的な守りを作る。

風精霊が、障壁にぶつかった。

でも、貫けない。

七属性の共鳴が、風精霊の攻撃を無効化した。

「七属性の力……やっぱりすごい」

ウィンディアが呟いた。

カイトは障壁を解除し、反撃に転じた。

「今度はこっちの番だ」

カイトは風の力で加速した。

ウィンディアと同じように、風に乗る。

カイトがウィンディアに迫った。

炎帝の剣が、七色の光を帯びる。

「七属性剣！」

カイトが剣を振った。

七色の斬撃が、ウィンディアに向かった。

ウィンディアは風神剣で受けた。

でも、七属性の重みに耐えきれない。

「重い……！」

ウィンディアの体が、少し沈んだ。

カイトは追撃した。

連続攻撃。上から、横から、下から。

風の速さと、七属性の力。

ウィンディアは徐々に追い込まれた。

「これで、最後！」

カイトは全属性の力を剣に込めた。

炎帝の剣が、虹色に輝く。

「虹光剣！」

カイトが渾身の一撃を放った。

ウィンディアは、風神剣を横に構えた。

全力の防御。

風と七属性の衝突。

爆発的な衝撃波が、二人の間で発生した。

風の領域全体が揺れる。

木々が倒れ、岩が砕ける。

衝撃が収まると、二人の姿があった。

ウィンディアは、膝をついていた。

風神剣は、半分に折れていた。

カイトは、剣を下ろしていた。

息が切れている。

でも、立っている。

「勝負あり……ね」

ウィンディアが微笑んだ。

「私の負けよ、カイト」
「ウィンディアさん……」

カイトは手を差し出した。

「大丈夫ですか？」

ウィンディアは、カイトの手を取った。

ゆっくりと立ち上がる。

「ありがとう」

ウィンディアは微笑んだ。

「強かったわ、カイト」
「風の賢者として、嬉しい敗北よ」

エリナたちが駆け寄ってきた。

「カイト、勝ったの？」

エリナが尋ねた。

「うん、なんとか」

カイトは笑った。

「ウィンディアさん、強かったよ」
「風の試練、三つとも合格よ」

ウィンディアが宣言した。

「これで、風の賢者の認定を与えるわ」

ウィンディアが、カイトの額に手を置いた。

緑色の光が、カイトの中に流れ込む。

「風の力、完全に目覚めなさい」

ウィンディアが囁いた。

カイトは、風の力が深まったのを感じた。

以前より、風を自由に操れる。

風の声も、もっと鮮明に聞こえる。

「ありがとうございます」

カイトは頭を下げた。

「風の賢者、ウィンディアとして、あなたを認めるわ」

ウィンディアが正式に告げた。

「風属性の賢者位、授与します」

カイトの体に、緑色の光の輪が現れた。

風の賢者の証。

七属性の内、また一つが完成した。

「これで、炎と風か」

リアムが数えた。

「あと五つの属性が残ってるな」
「でも、大きな一歩よ」

シオンが言った。

「風属性は、移動と偵察に重要」
「これで、カイトの行動範囲が広がるわ」

ウィンディアが、折れた風神剣を見た。

「剣が折れちゃったわね」

ウィンディアが苦笑した。

「でも、いい折れ方よ」
「ごめんなさい」

カイトは謝った。

「壊すつもりはなかったんです」
「気にしないで」

「風神剣は、風でできている」
「また、作れるから」

ウィンディアが手を掲げると、風が集まった。

折れた剣が、風で修復されていく。

数秒で、元通りの風神剣になった。

「見て」

カイトは驚いた。

「風で剣を作れるんだ」
「賢者になると、属性そのものを形にできるの」

ウィンディアが説明した。

「あなたも、いつかできるわ」

七属性全てで、武器を作れる。

「楽しみだ」

ウィンディアが、カイトたちに告げた。

「風の試練は終わり」

ウィンディアが言った。

エリナ、リアム、シオン。

「どっちに行くか、決める必要があるな」
「カイト、どうしたい？」

巨大な風の球体が、四人を中に入れる。

森、山、川が、下を高速で過ぎ去る。

青い海と白い砂浜。心地よい潮風。
