<h3>第48章：古城の秘密</h3>

<p>朝の光が窓から差し込み、カイトは目を覚ました。見慣れない天井に一瞬戸惑ったが、すぐにここが王都の宿屋だと思い出した。昨夜遅くまで、帰還報告と今後の対策について話し合っていたのだった。</p>

<p>ベッドから起き上がり、窓の外を眺める。王都の通りはすでに人々で賑わっていた。普通の生活が続いていることが、逆に不思議に感じられた。世界の存亡をかけた戦いがあったばかりなのに。</p>

<blockquote>「起きてたの？」</blockquote>

<p>振り向くと、エリナが部屋に入ってきた。手にはパンと飲み物が入った袋を持っている。</p>

<blockquote>「朝食、買ってきたわ。みんな食堂で集まってる」</blockquote>

<p>カイトは頷き、簡単に身支度を整えた。エリナと一緒に食堂へ向かう。リアム、セシリア、そしてマグナスが既に席についていた。</p>

<blockquote>「おはよう、カイト。よく眠れたか？」</blockquote>

<p>マグナスが声をかけてきた。ギルドマスターとしての威厳ある声だが、どこか疲れているようにも聞こえた。</p>

<blockquote>「はい。それより、昨夜の続きを」</blockquote>

<p>カイトが言うと、マグナスは頷いた。</p>

<blockquote>「ああ。マルクスから受け取った資料を分析した結果だが……興味深いことがわかった」</blockquote>

<p>マグナスはテーブルに一枚の古びた地図を広げた。それは王都の北にある古城、通称「静寂城」の地図だった。</p>

<blockquote>「この城は三百年前に放棄されたとされている。しかし、マルクスの調査によれば、そこには古代魔法文明の遺産が眠っている可能性がある」</blockquote>

<blockquote>「古代魔法文明……」</blockquote>

<p>セシリアが呟いた。彼女の目に好奇心の光が宿る。</p>

<blockquote>「ヴェリディアンの時代の技術が残っているかもしれないってことですか？」</blockquote>

<blockquote>「正確には、それを管理していた『守人』たちの記録だ」</blockquote>

<p>マグナスは言葉を続けた。</p>

<blockquote>「マルクスは『収束者』としての能力で、世界各地の魔力の流れを感知できる。彼が言うには、静寂城から奇妙な魔力パターンが検出されたそうだ」</blockquote>

<p>カイトは眉をひそめた。</p>

<blockquote>「奇妙なパターン？」</blockquote>

<blockquote>「通常の魔法使いが使う魔力とは異なる。コードのような……いや、構造化された魔力だと」</blockquote>

<p>その言葉に、カイトの心臓が跳ねた。構造化された魔力。それは自分が使う魔法の方法と似ていた。</p>

<blockquote>「俺と同じタイプの魔法使いがいるってことか？」</blockquote>

<blockquote>「あるいは、かつていたのか」</blockquote>

<p>マグナスは静かに言った。</p>

<blockquote>「ヴェリディアンが残した記録によれば、古代には『構築者』と呼ばれる魔法使いがいた。彼らは魔法を言語のように扱い、世界の理を書き換えることができたという」</blockquote>

<blockquote>「構築者……」</blockquote>

<p>カイトは呟いた。もしかすると、自分がこの世界に召喚された理由と関係があるのかもしれない。</p>

<blockquote>「その古城に行けば、何かわかるかもしれませんね」</blockquote>

<p>リアムが言った。彼は既に冒険者の準備を整えている様子だった。</p>

<blockquote>「ギルドとしても正式に調査依頼を出したい。もちろん、君たちが引き受けてくれるなら最高の布陣だが」</blockquote>

<p>マグナスがカイトたちを見回した。</p>

<blockquote>「もちろん、行くわよ」</blockquote>

<p>エリナが即答した。</p>

<blockquote>「カイトの力の秘密を知る手がかりかもしれない。それに、もし誰かがそこで古代の技術を悪用しようとしていたら、放っておけないし」</blockquote>

<blockquote>「僕も賛成です」</blockquote>

<p>セシリアが手を挙げた。</p>

<blockquote>「古代魔法の研究は、魔導師として避けて通れない道ですから」</blockquote>

<blockquote>「俺も行く。カイトの力になるよ」</blockquote>

<p>リアムが微笑んだ。カイトは仲間たちの言葉に感謝しながら、頷いた。</p>

<blockquote>「ありがとう。みんな。行こう、静寂城へ」</blockquote>

<hr/>

<p>準備を終え、一行は王都の北門を出た。静寂城までは馬で半日の距離だ。</p>

<p>道中、カイトはずっと考えていた。構築者とは何なのか。そして自分はなぜこの世界に来たのか。</p>

<blockquote>「何か悩んでる？」</blockquote>

<p>馬を並べて走るエリナが声をかけてきた。</p>

<blockquote>「ああ。この世界に召喚された理由が、もしかしたら『構築者』と関係があるんじゃないかと思って」</blockquote>

<p>エリナは少し考え、言った。</p>

<blockquote>「異世界召喚って、普通は誰かが意図的に行うものよ。もしあなたを呼んだのが『構築者』の末裔か、あるいは……」</blockquote>

<p>彼女は言葉を区切った。</p>

<blockquote>「あるいは、あなた自身が『構築者』の生まれ変わりかもしれない」</blockquote>

<p>その可能性に、カイトは気づいていなかった。もしそうなら、自分の存在そのものがこの世界に深く関わっていることになる。</p>

<blockquote>「生まれ変わり……俺が？」</blockquote>

<blockquote>「確証はないけどね。でも、あなたの魔法の使い方は特殊よ。この世界の誰とも違う」</blockquote>

<p>エリナの言葉に、カイトは改めて自分の能力を考えた。プログラミングの知識を使って魔法を構築する。それは確かに、この世界の常識とは異なっていた。</p>

<blockquote>「着いたぞ」</blockquote>

<p>先頭を走っていたリアムが声を上げた。</p>

<p>森を抜けた先に、古城が姿を現した。静寂城。その名の通り、不気味なほど静かな場所だった。城の周囲には何も生えていない。ただ、枯れた木々と、ひび割れた石畳が広がっているだけだ。</p>

<blockquote>「魔力の濃度が高いわ」</blockquote>

<p>セシリアが杖を構えた。</p>

<blockquote>「それに、何か……防護魔法の痕跡もあります。かなり強力なものが」</blockquote>

<p>カイトは目を閉じ、魔力を感知した。確かに、城全体を覆うような魔法の構造を感じ取れた。</p>

<pre><code>DetectionResult = {
    type: "barrier",
    strength: "ancient",
    status: "dormant",
    pattern: "structured_code"
}</code></pre>

<p>カイトは驚いた。この防護魔法、自分が使うコード形式と酷似していたのだ。</p>

<blockquote>「どうしたの、カイト？」</blockquote>

<p>エリナが心配そうに聞いた。</p>

<blockquote>「この魔法……俺が書くコードと同じ構造だ」</blockquote>

<p>全員が息を呑んだ。</p>

<blockquote>「つまり、ここに『構築者』がいたってことか？」</blockquote>

<p>リアムが問いかけた。</p>

<blockquote>「少なくとも、俺と同じ方法で魔法を使う誰かが、この城を守っていた」</blockquote>

<p>カイトは城門に近づいた。門には古代文字が刻まれている。普通なら読めないはずの文字だが、なぜか意味がわかった。</p>

<p><em>構築者のみ、この門を開くなる</em></p>

<p>カイトは門に手を置いた。魔力を流し込む。</p>

<pre><code>Access.request({
    user: "Kaito",
    type: "Constructor",
    action: "open_gate"
});</code></pre>

<p>重い音を立てて、数百年ぶりに城門が開いた。</p>

<blockquote>「カイト、あんた……」</blockquote>

<p>エリナが驚愕の表情で見つめていた。</p>

<blockquote>「門が開いたわ。古代の封印が、あなたの魔力に反応したのよ」</blockquote>

<p>カイト自身も驚いていた。自分の力が、この城と共鳴したのだ。</p>

<blockquote>「中に入ろう。答えはきっとそこにある」</blockquote>

<p>一行は、静寂城の内部へと足を踏み入れた。</p>

<hr/>

<p>城の内部は意外にも保存状態が良かった。壁には古代の壁画が残り、廊下には彫像が並んでいる。</p>

<blockquote>「これらは……『構築者』たちの記録のようですね」</blockquote>

<p>セシリアが壁画を調べながら言った。</p>

<blockquote>「魔法をコードのように書く方法。世界の理を理解するための数学。それらがここに記されています」</blockquote>

<p>カイトは壁画に描かれた人物を見た。彼らは皆、手に何かを持っている。それは、現代で言うところの「タブレット」に見えた。</p>

<blockquote>「まさか、古代にそんな技術が？」</blockquote>

<p>リアムが驚いた。</p>

<blockquote>「魔法で作られた端末よ。『構築者』たちは、魔力を電気のように扱って、情報を記録・処理していたの」</blockquote>

<p>エリナが解説した。彼女も古代文明に詳しいのだ。</p>

<p>一行は城の奥へと進んだ。最も深い場所に、大きな扉があった。扉には複雑な魔法陣が刻まれている。</p>

<blockquote>「ここが中心部ね」</blockquote>

<p>セシリアが言った。</p>

<blockquote>「先ほど感知した魔力パターンの発生源は、この中です」</blockquote>

<p>カイトは扉に近づいた。魔法陣を観察する。これは単なる鍵ではない。認証システムだ。</p>

<pre><code>AuthenticationChallenge = {
    method: "constructor_identity",
    required: "code_proof",
    hint: "show_your_nature"
}</code></pre>

<p>カイトは理解した。この扉は、構築者としての能力を証明するコードを書くことを求めているのだ。</p>

<blockquote>「みんな、下がって」</blockquote>

<p>カイトは両手を前に出し、魔力を集中させた。</p>

<pre><code>function proveIdentity() {
    return {
        type: "Constructor",
        name: "Kaito",
        world: "Earth",
        summon: "unknown",
        ability: "code_magic",
        intention: "seek_truth"
    };
}

Authentication.verify(proveIdentity());</code></pre>

<p>扉の魔法陣が光り輝いた。そして、ゆっくりと開き始めた。</p>

<blockquote>「すごい……」</blockquote>

<p>セシリアが呟いた。</p>

<blockquote>「こんな認証システム、聞いたこともないわ」</blockquote>

<p>扉の向こうには、円形の部屋があった。部屋の中央に、一台の装置が置かれている。それは古代の「構築者」が使っていた端末に違いなかった。</p>

<p>端末の画面が、カイトが近づくと同時に点灯した。</p>

<pre><code>SYSTEM: Constructor detected.
SYSTEM: Loading message from Administrator...
SYSTEM: Playing recorded message...</code></pre>

<p> hologramのような光が部屋に投射され、一人の人物の姿が現れた。長い髪を持ち、賢者のローブを着た女性だった。</p>

<blockquote>「このメッセージを聞いているのが、新たな『構築者』であることを願う」</blockquote>

<p>女性の声が響いた。</p>

<blockquote>「私はヴェリディアン。古代魔法文明を統べる者であり、この世界に『構築者』のシステムを残した者だ」</blockquote>

<p>カイトは息を呑んだ。ヴェリディアン。その名前は、以前に出会った古代の精神体と同じだった。</p>

<blockquote>「あなたがこのメッセージを聞いているということは、世界が再び危機に瀕しているのだろう。『構築者』は、世界が最も必要とするときに現れる。異世界から召喚されることもある」</blockquote>

<p>カイトは自分が異世界から来たことを思い出した。</p>

<blockquote>「この端末には、『構築者』としての知識のすべてが記録されている。しかし、その力を使うには責任が必要だ。コードで魔法を書くということは、世界の理に干渉すること。軽率な行いは、取り返しのつかない結果を招く」</blockquote>

<p>ヴェリディアンの表情が真剣になった。</p>

<blockquote>「今、世界には『侵食』と呼ばれる現象が起こりつつあるはずだ。世界の境界が曖昧になり、異質な存在が侵入してくる。『構築者』の使命は、この侵食を止め、世界を修復すること」</blockquote>

<p>侵食。カイトたちは、最近の異常現象がその名前で呼ばれることを知っていた。</p>

<blockquote>「この端末を使い、侵食の源を見つけ出し、封印せよ。あなたの力が必要だ、新たな『構築者』よ」</blockquote>

<p>メッセージはそこで終わった。</p>

<p>カイトは端末を見つめた。これが、自分がこの世界に来た理由なのだ。答えの一つが、ここにあった。</p>

<blockquote>「カイト……」</blockquote>

<p>エリナが静かに声をかけた。</p>

<blockquote>「これでわかったわね。あなたが召喚された理由が」</blockquote>

<blockquote>「ああ。俺は、この世界を救うために呼ばれたんだ」</blockquote>

<p>カイトは仲間たちを見回した。</p>

<blockquote>「一人では無理だ。みんな、力を貸してくれ」</blockquote>

<blockquote>「言われるまでもないわ」</blockquote>

<p>エリナが微笑んだ。</p>

<blockquote>「僕たちがいれば、どんな敵にも勝てるよ」</blockquote>

<p>リアムが拳を握った。</p>

<blockquote>「古代の知識と、現代の力。最強の組み合わせですね」</blockquote>

<p>セシリアが端末のデータを解析し始めた。</p>

<p>カイトは前を向いた。これから始まる戦いは、これまで以上に困難なものになるだろう。でも、仲間がいる。そして、自分の力には意味がある。</p>

<p>物語は、まだ始まったばかりだった。</p>
