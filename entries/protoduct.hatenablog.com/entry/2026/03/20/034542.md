<h3>第23章：真実の変数</h3>

<p>約束の結晶から放たれた光が、地下空間全体を包み込んでいた。カイトは眩しさに目を細めながら、結晶の中心に浮かび上がる文字列を見つめていた。</p>

<pre><code>PROMISE_LOCKED: true
AWAITING_KEY_INPUT...</code></pre>

<p>「これは……」カイトは息を呑んだ。「結晶の中にコードが組み込まれている」</p>

<p>エリナがカイトの肩に手を置いた。「カイト、何が見えてるの？」</p>

<p>「古代のプログラムだ。この結晶は単なる宝石じゃない。何かを守るためのセキュリティシステムなんだよ」</p>

<p>リアムが剣の柄に手をかけたまま警戒を続けていた。「敵襲の可能性は？」</p>

<p>「ないと思う。でも……」カイトは結晶に近づいた。「解除するには、特定の条件を満たす必要がある」</p>

<p>セシリアが光の中から現れた。彼女の瞳にも結晶の輝きが映り込んでいる。</p>

<blockquote>「ヴェリディアンが残したものよ。彼が何を守ろうとしたのか、それを知るにはこの結晶を解読しなければならないわ」</blockquote>

<p>カイトは結晶の前に立ち、深呼吸をした。彼の瞳の中で、次々とコードが流れ始める。これは彼だけが見える世界——魔法をプログラミング言語として認識できる、彼固有の能力だった。</p>

<pre><code>// Ancient Security System v1.0
// Created by: Verdigris
// Purpose: Protect the Promise

    constructor() {
        this.locked = true;
        this.keyFragment = [];
        this.requiredFragments = 3;
    }
    
    validate(input) {

            && input.intent === 'protect';
    }

<p>「三人の鍵が必要だと？」カイトは眉をひそめた。「それに、感情と意志を検証するタイプのコード……」</p>

<p>「どういうこと？」エリナが問いかける。</p>

<p>「この結晶は、特定の言葉や行動ではなく、心の状態を鍵としているんだ。信頼と守護の意志——それが鍵になっている」</p>

<p>リアムが腕を組んだ。「具体的にはどうすればいい？」</p>

<p>「三人で、同時に結晶に触れる必要がある。そして、それぞれが心の中で『信頼』と『守りたい』という想いを抱くんだ」</p>

<p>カイトの説明に、エリナとリアムは顔を見合わせた。セシリアは静かに頷いている。</p>

<blockquote>「ヴェリディアンらしいわ。彼は技術だけじゃなく、人の心も信じていたから」</blockquote>

<p>「よし、やろう」エリナが結晶に手を伸ばした。「私が守りたいのは、カイトとの時間。そして、この世界の未来」</p>

<p>リアムも覚悟を決めたように歩み寄る。「俺が守りたいのは、仲間だ。お前たちと、この世界に生きる全ての人」</p>

<p>カイトは二人の間に立ち、結晶に手を置いた。彼の中に、明確なイメージが浮かぶ。</p>

<p>——異世界に転生した日。何も分からずに立ち尽くしていた自分を、エリナが見つけてくれたこと。リアムと出会い、共に戦う仲間を得たこと。セシリアと出会い、この世界の歴史を知ったこと。</p>

<p>そして、何より——この世界を守りたいという、純粋な願い。</p>

<p>結晶が激しく振動し始めた。光が強まり、地下空間の壁に映像が投影される。</p>

<p>それは、かつての光景だった。</p>

<p>——ヴェリディアンと呼ばれる青年が、同じ結晶の前に立っている。彼の隣には、セシリアによく似た女性がいた。いや、セシリア本人だ。ただ、今よりもずっと若い。</p>

<blockquote>「本当にこれでいいの？」女性のセシリアが問いかける。「この封印は、未来の人々への警告よ」</blockquote>

<blockquote>「ああ」ヴェリディアンは頷いた。「僕たちは、あまりにも大きな力を手に入れてしまった。このままでは、誰かがそれを悪用する」</blockquote>

<blockquote>「だからこそ、封印するのね」</blockquote>

<blockquote>「違う。封印するんじゃない——託すんだ。未来に、正しく使える誰かに」</blockquote>

<p>映像の中のヴェリディアンが、結晶に向かって何かを囁いた。その言葉は聞こえない。だが、カイトには分かった。彼が何を言ったのか。</p>

<pre><code>if (future.coder.exists()) {
    await future.coder.inherit(this.power);
    await future.coder.protect(world);
}</code></pre>

<p>——未来のコーダーが存在するなら、この力を継承せよ。そして、世界を守ってほしい。</p>

<p>映像が消え、結晶の光が収束していく。そして、結晶は小さな光の粒子となって、カイトの胸元に吸い込まれた。</p>

<p>「うっ……！」カイトは膝をついた。体の中に、膨大な情報が流れ込んでくる感覚。</p>

<p>「カイト！」エリナが駆け寄り、彼を支えた。</p>

<p>「大丈夫か？」リアムも心配そうに覗き込む。</p>

<p>カイトは荒い息を整えながら、顔を上げた。彼の瞳の中で、かつてないほど多くのコードが駆け巡っていた。</p>

<p>「ヴェリディアンの……記憶だ。彼が残した、全ての魔法の設計図」</p>

<p>セシリアがカイトの前に跪いた。彼女の瞳には、涙が浮かんでいた。</p>

<blockquote>「彼は、本当に未来を信じていたのね」</blockquote>

<p>「ああ。そして、その未来を——僕たちに託したんだ」</p>

<p>カイトは立ち上がった。体の中に、新たな力が満ちているのを感じる。だが、それと同時に、重い責任も感じていた。</p>

<p>「ヴェリディアンの記憶の中に、一つの警告があった」カイトは言った。「彼が封印した力——それを悪用しようとしている者がいる」</p>

<p>「誰だ？」リアムが鋭く問う。</p>

<p>「名前は分からない。でも、コードの中に痕跡があった。何者かが、古代の魔法を解析しようとしている。そして、ヴェリディアンの封印を破ろうとしているんだ」</p>

<p>エリナが表情を引き締めた。「つまり、私たちが結晶の封印を解いたことで——」</p>

<p>「いや、関係ない」カイトは首を振った。「ヴェリディアンは予期していた。封印が解かれる時、その力を正しい継承者に渡すために、このシステムを作ったんだ。悪用しようとする者には、力は渡らない」</p>

<p>「じゃあ、私たちは安全ってこと？」リアムが確認する。</p>

<p>「今はね。でも——」カイトは地上への出口を見上げた。「敵は諦めないだろう。僕たちが継承した力を、奪おうとしてくるはずだ」</p>

<p>セシリアが立ち上がり、カイトに近づいた。</p>

<blockquote>「ヴェリディアンの力を受け継いだあなたなら、きっと戦えるわ。でも、一人では無理よ」</blockquote>

<p>「分かってる」カイトは仲間たちを見回した。「エリナ、リアム、セシリア——君たちがいてくれてよかった」</p>

<p>エリナが微笑んだ。「私たちはチームでしょ。一人で背負い込まないで」</p>

<p>リアムも小さく笑った。「ま、俺たちに任せとけって。お前は頭脳担当、俺が物理担当だ」</p>

<p>「ありがとう」カイトは深く頷いた。「じゃあ、一旦戻ろう。この力をどう使うか、戦略を練る必要がある」</p>

<p>四人は地下空間を後にした。カイトの胸元には、結晶の温もりが残っていた。それは、ヴェリディアンからの約束——未来を託すという、確かな約束の証だった。</p>

<p>地上に出ると、空は茜色に染まり始めていた。夕暮れの光が、カイトの顔を照らす。</p>

<p>——プログラミングという形で魔法を操る僕。それが、ただの偶然じゃないことを、今日確信した。</p>

<p>ヴェリディアンもまた、魔法をコードとして理解していた。彼が作ったシステムは、僕のような「コーダー」を待っていたんだ。</p>

<p>「カイト、どうしたの？」エリナが心配そうに声をかけた。</p>

<p>「ううん、なんでもない」カイトは微笑んだ。「ただ、未来が少し見えた気がしただけ」</p>

<p>「未来？」</p>

<p>「うん。僕たちが、この世界を守る未来が」</p>

<p>エリナはカイトの手を握った。その手は温かく、力強かった。</p>

<blockquote>「一緒に見ましょう、その未来を」</blockquote>

<p>カイトは頷いた。そして、心の中で新たなコードを書き始める。これは、魔法の呪文ではない。これからの道筋——彼らの物語を紡ぐ、プログラムの設計図だった。</p>
