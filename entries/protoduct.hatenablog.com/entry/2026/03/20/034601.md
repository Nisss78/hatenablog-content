<h3>第29章：無限ループの牢獄</h3>

<p>カイトが足を踏み入れた瞬間、世界が歪んだ。</p>

<p>何かがおかしい。先ほどまで青空だった空が、灰色の霧に覆われている。城壁の形も、さっきとは微妙に違う気がする。そして何より——</p>

<blockquote>「カイト！ 無事か！？」</blockquote>

<p>エリナの声。彼女が駆け寄ってくる。でも、これって……さっきと同じじゃないか？</p>

<p>カイトは眉をひそめた。確かに、この場面を見た気がする。エリナが心配そうな表情で、まったく同じ角度から駆けてくる。リアムが後ろで杖を構えている位置も、セシリアが空を見上げている姿も、すべてが既視感だ。</p>

<blockquote>「……またか」</blockquote>

<p>カイトは呟いた。</p>

<p>三度目だ。この瞬間を、三度目に体験している。最初は混乱した。二度目は偶然だと思った。でも三度目なら——これは明確に、何らかの意図的な現象だ。</p>

<p>カイトは強く念じた。<code>Magic.analyze()</code>。</p>

<p>目の前に半透明のウィンドウが浮かび上がる。そこに表示されたのは、世界を構成するコードの断片だった。</p>

<pre><code>while (true) {
  respawnCharacters();
  resetEnvironment();
  await playerInput();
  if (breakConditionMet()) {
    break;
  }
}</code></pre>

<p>無限ループ。</p>

<p>カイトは思わず苦笑した。この世界を作った古代の魔導師は、相当意地が悪いらしい。脱出条件を満たさない限り、永遠に同じ時間を繰り返す仕組みだ。</p>

<blockquote>「カイト？ どうしたの、難しい顔して」</blockquote>

<p>エリナが心配そうに顔を覗き込んでくる。カイトは首を横に振った。</p>

<blockquote>「いや、何でもない。……エリナ、ちょっと聞きたいんだけど、私たち、ここに来てどれくらい経ったと思う？」</blockquote>

<p>エリナは首を傾げた。</p>

<blockquote>「えっと、城門を出てから……十分くらい？」</blockquote>

<p>やはり。彼女の認識では十分だ。でもカイトの意識には、三回分の記憶がある。つまり、ループの中でカイトだけが記憶を保持している——あるいは、カイトの能力が勝手にバグとして認識しているのかもしれない。</p>

<p>カイトは皆に向き直った。</p>

<blockquote>「みんな、聞いてくれ。ここは……罠だ」</blockquote>

<blockquote>「罠？」</blockquote>

<p>リアムが杖を構え直した。</p>

<blockquote>「ああ。無限ループっていう現象でね。ある条件を満たすまで、同じ時間を永遠に繰り返すんだ」</blockquote>

<blockquote>「永遠に……？」</blockquote>

<p>セシリアが不安そうに空を見上げた。</p>

<blockquote>「じゃあ、どうやって抜ければいいの？」</blockquote>

<p>カイトはコードウィンドウを見つめた。<code>breakConditionMet()</code>。この関数の中身が見えない。つまり、自分で探し出す必要がある。</p>

<blockquote>「正直、まだ分からない。でも——」</blockquote>

<p>カイトは微笑んだ。</p>

<blockquote>「プログラマーにとって、無限ループは日常茶飯事だ。何度でも挑戦できる」</blockquote>

<hr/>

<p>四度目のループ。</p>

<p>カイトは冷静に観察を始めた。まず、このエリアの境界線を探る。霧の向こう側に行こうとすると、なぜか城門の方へ戻ってしまう。つまり、移動可能範囲が限定されている。</p>

<p>次に、NPCの行動パターン。城壁の近くに立っている衛兵は、常に同じ位置にいる。話しかけても「詮索は無用だ」としか言わない。背景オブジェクトとして処理されているらしい。</p>

<p>そして、脱出の手がかり。</p>

<blockquote>「ねえ、あそこ……光ってない？」</blockquote>

<p>セシリアが指差したのは、霧の中にぼんやりと浮かぶ塔だった。以前は気づかなかったが、確かに淡い光を放っている。</p>

<p>カイトは頷いた。</p>

<blockquote>「あそこに行けば、何か分かるかもしれない」</blockquote>

<p>一行は塔に向かった。でも、どれだけ歩いても距離が縮まらない。まるで、遠近法が狂っているようだ。</p>

<p>カイトは立ち止まった。</p>

<blockquote>「……座標が固定されてる」</blockquote>

<blockquote>「え？」</blockquote>

<p>エリナが振り返る。</p>

<blockquote>「塔の座標が、プレイヤーとの相対距離で計算されてるんだ。だから、どれだけ近づこうとしても、永遠に同じ距離を保つ」</blockquote>

<p>いわゆる「到達不能な背景」だ。ゲーム開発でよく使われるテクニックだが、まさか異世界で遭遇するとは。</p>

<blockquote>「じゃあ、どうすれば……？」</blockquote>

<p>カイトは考えた。座標を直接書き換える？ でも、この世界のコードは保護されている。カイトが書けるのは、自分の魔法だけだ。</p>

<p>待てよ。</p>

<p>カイトはあることに気づいた。この無限ループ、<code>playerInput()</code>を待っている。つまり、プレイヤーの行動によって結果が変わるはずだ。</p>

<p>カイトは試しに、何もしないで立ち尽くしてみた。一分、二分、五分……。</p>

<p>すると、世界がまた歪んだ。</p>

<p>五度目のループ。</p>

<p>何もしないとリセットされる。つまり、何らかの行動を起こさないといけない。でも、塔には近づけない。他に何か……。</p>

<hr/>

<p>八度目のループ。</p>

<p>カイトは少しずつ法則を理解し始めていた。</p>

<ul>
<li>霧の中に入ると強制送還される</li>
<li>塔には近づけない</li>
<li>衛兵は会話不能</li>
<li>でも、地面の石畳の並びが毎回微妙に違う</li>
</ul>

<p>石畳？</p>

<p>カイトは地面を見た。灰色の石が、不規則なパターンで敷き詰められている。一見ランダムに見えるけれど、よく見ると……。</p>

<p><code>Magic.inspect()</code>。</p>

<p>石畳のコードが表示された。</p>

<pre><code>stones.forEach((stone, index) => {
  stone.symbol = ancientRunes[index % runes.length];
});</code></pre>

<p>古代文字？</p>

<p>カイトは地面に膝をついた。石をよく見ると、確かに小さな文字が刻まれている。でも、傷んでいて読み取れない部分も多い。</p>

<blockquote>「カイト、何か見つけたの？」</blockquote>

<p>エリナが隣にしゃがみ込んだ。</p>

<blockquote>「うん。これ、パズルかもしれない」</blockquote>

<p>カイトは石畳を指差した。</p>

<blockquote>「この文字、並べ替えると何か意味があるはずなんだ」</blockquote>

<p>リアムが杖の先で石を照らした。</p>

<blockquote>「古代語だ。でも、相当古い形式だな……読めるか？」</blockquote>

<p>カイトは首を振った。</p>

<blockquote>「まだ分からない。でも、ループごとに配置が変わるってことは、正解の並び順があるはずなんだ」</quote>

<p>九度目のループ。</p>

<p>カイトは石畳のパターンを記録し始めた。メモする道具はないけど、自分のメソッドとして保存すればいい。</p>

<pre><code>Magic.remember("loop_9", stones_pattern);</code></pre>

<p>十度目のループ。</p>

<p>十一度目のループ。</p>

<p>十二度目。</p>

<p>カイトは淡々と記録を続けた。エリナたちは最初は困惑していたけれど、何度も一緒にループを経験するうちに、カイトの意図を理解し始めた。</p>

<blockquote>「つまり、正解の並び順を見つけるまで何度でもやり直せるってこと？」</blockquote>

<p>セシリアが確認した。</p>

<blockquote>「そういうこと。パスワード総当たり攻撃みたいなもんだね」</quote>

<blockquote>「総当たり……？」</blockquote>

<p>カイトは笑った。</p>

<blockquote>「気にしないで。とにかく、地道な作業になるけど——」</blockquote>

<p>その時だった。</p>

<p>カイトの脳裏に、奇妙な感覚が走った。まるで、誰かが自分のコードに干渉しているような……。</p>

<p><code>Magic.checkIntrusion()</code>。</p>

<p>警告が表示された。</p>

<pre><code>⚠ WARNING: External entity accessing memory buffer
  Source: Unknown
  Intent: Observation
  Risk Level: Low</code></pre>

<p>誰かが見ている？</p>

<p>カイトは周囲を見渡した。霧の向こう、塔の方向に、何かが動いた気がした。</p>

<blockquote>「……誰かいる」</blockquote>

<p>エリナが剣の柄に手をかけた。</p>

<blockquote>「敵？」</blockquote>

<p>カイトは首を傾げた。</p>

<blockquote>「分からない。でも、この罠を作った本人かもしれない」</blockquote>

<hr/>

<p>十五度目のループ。</p>

<p>石畳のパターンが、少しずつ見えてきた。全部で七十二個の石があり、その中に三十六個の文字が隠されている。残りはダミーだ。</p>

<p>カイトは文字を抽出した。</p>

<pre><code>const runes = ["ᚠ","ᚢ","ᚦ","ᚨ","ᚱ","ᚲ","ᚷ","ᚹ","ᚺ","ᚾ","ᛁ","ᛃ","ᛇ","ᛈ","ᛉ","ᛊ","ᛏ","ᛒ","ᛖ","ᛗ","ᛚ","ᛜ","ᛞ","ᛟ"];
const message = extractMessage(stones, runes);
// "ᚹᚺᛖᚱᛖ_ᛁᛊ_ᛏᚺᛖ_ᚲᛟᛞᛖ"</code></pre>

<p>「WHERE IS THE CODE」</p>

<p>カイトは思わず笑ってしまった。</p>

<blockquote>「答えが答えを求めてるのかよ……」</blockquote>

<p>でも、これは重要な手がかりだ。「コードはどこにある？」——つまり、脱出の鍵となるコードが、どこかに隠されている。</p>

<blockquote>「コード……コード……」</blockquote>

<p>カイトは考えた。この世界で、コードと言えば……。</p>

<p>ふと、塔を見上げた。</p>

<p>あの塔、構造がおかしい。遠近法だけでなく、形自体が不安定だ。まるで、描画処理が正しく行われていないような……。</p>

<p><code>Magic.analyze(tower)</code>。</p>

<p>エラーが返ってきた。</p>

<pre><code>❌ ERROR: Target not found in current scope
  Possible cause: Corrupted reference
  Suggestion: Use Magic.rebuild(tower)</code></pre>

<p>参照エラー？</p>

<p>カイトは目を細めた。この塔、実体がないんじゃないか。変数として宣言されているけど、中身が空っぽ——いわゆる「未定義」の状態。</p>

<blockquote>「……分かったかも」</blockquote>

<p>カイトは皆に向き直った。</p>

<blockquote>「あの塔、実は存在していないんだ」</blockquote>>

<blockquote>「え？ でも見えてるよ？」</blockquote>

<p>セシリアが指差した。</p>

<blockquote>「見えてるだけ。変数は宣言されてるけど、中身がundefined。だから近づけないし、座標も固定されてる」</blockquote>

<p>カイトは説明した。</p>

<blockquote>「つまり、あの塔に実体を与えればいい」</blockquote>

<hr/>

<p>十八度目のループ。</p>

<p>カイトは挑戦を開始した。</p>

<p>まず、塔の変数名を特定する。<code>Magic.listVariables()</code>で表示されたリストから、それらしいものを探す。</p>

<pre><code>let tower_of_truth; // undefined
let exit_gate; // undefined
let liberation_key; // undefined</code></pre>

<p>三つとも未定義だ。この古代の魔導師、相当手間をかけている。</p>

<p>カイトは塔に向かって手をかざした。</p>

<blockquote>「<code>tower_of_truth = Magic.construct()</code>」</blockquote>

<p>何も起きない。権限がないのか、それとも——</p>

<p>その時、霧の中から声が聞こえた。</p>

<blockquote>「面白いことを考えるね、異邦人」</blockquote>

<p>カイトは振り返った。そこに立っていたのは、半透明の老人だった。ローブをまとっているけれど、体が透けていて、向こう側の霧が見える。</p>

<blockquote>「あなたが、この罠を作った人？」</blockquote>

<p>カイトが尋ねると、老人は微笑んだ。</p>

<blockquote>「作ったというか……残した、かな。私はヴェリディアン。この場所を守る者だった」</blockquote>

<p>ヴェリディアン——聞き覚えがある。以前、古代の図書館でその名前を見た。</p>

<blockquote>「どうすれば、ここから出られる？」</blockquote>

<p>カイトは単刀直入に尋ねた。ヴェリディアンは笑った。</p>

<blockquote>「答えは分かっているはずだ。コードはどこにある？」</blockquote>>

<p>カイトは考えた。石畳のメッセージ。「コードはどこにある？」</p>

<p>もしかして——</p>

<p>カイトは自分の胸に手を当てた。魔法を使う時、コードが体の中を流れる感覚がある。つまり、コードは——</p>

<blockquote>「……自分の中にある」</blockquote>

<p>カイトが答えると、ヴェリディアンは満足そうに頷いた。</p>

<blockquote>「正解だ。お前がプログラマーなら、答えは常に自分の中にある。外部のライブラリに頼るな。自分で書け」</blockquote>>

<p>カイトは深呼吸した。</p>

<p>塔に実体を与えるには、既存のコードを使うんじゃない。自分で、最初から書くんだ。</p>

<p>カイトは目を閉じた。イメージする。塔の形。材質。大きさ。色。匂い。風の音。窓からの光。</p>

<p>そして、指先で空中に文字を描いた。</p>

<pre><code>const tower_of_truth = {
  height: 100,
  material: "ancient_stone",
  windows: 12,
  contains: "exit_portal",
  accessible: true
};

Magic.render(tower_of_truth);</code></pre>

<p>カイトが目を開けると——</p>

<p>塔が、そこにあった。</p>

<p>本物の、触れられる、実体のある塔が。</p>

<blockquote>「すごい……！」</blockquote>

<p>セシリアが息を呑んだ。</p>

<p>ヴェリディアンも満足そうだった。</p>

<blockquote>「よくやった。無限ループの脱出条件は、『自分で答えを生み出すこと』。準備完了だね」</blockquote>>

<p>老人の姿が、少しずつ薄れていく。</p>

<blockquote>「塔のてっぺんに、出口がある。健闘を祈るよ、異邦人——いや、コードマスター」</blockquote>>

<p>ヴェリディアンが消えると、世界が再び歪んだ。でも、今回はリセットじゃない。霧が晴れて、空が青さを取り戻していく。</p>

<blockquote>「カイト、やったね！」</blockquote>

<p>エリナがカイトの腕をつかんだ。</p>

<p>カイトは微笑んだ。</p>

<blockquote>「うん。でも、これからが本番だ」</blockquote>>

<p>塔を見上げる。百メートルの高さ。てっぺんまでの階段を登る必要がある。その間に、どんな試練が待っているか分からない。</p>

<p>でも、カイトは怖くなかった。</p>

<p>無限ループを十八回も経験した。その間、諦めずに答えを探し続けた。そして、自分で解を見つけ出した。</p>

<p>この世界のどんな罠も、コードとして理解できる。そして、コードとして理解できるものは、書き換えられる。</p>

<blockquote>「行こう、みんな」</blockquote>>

<p>カイトは塔の入り口に向かって歩き出した。</p>

<blockquote>「次は、再帰の迷宮だ。気をつけて」</blockquote>>

<p>エリナ、リアム、セシリアが後に続く。</p>

<p>塔の扉が、重々しい音を立てて開いた。</p>

<p>その奥には、螺旋階段が続いていた。壁には無数の鏡が掛かっていて、それぞれが異なるカイトの姿を映している。</p>

<p>再帰——自分自身を呼び出す関数。</p>

<p>どうやら、次の試練は自分自身との対話になりそうだ。</p>

<p>カイトは深呼吸して、最初の段を踏み出した。</p>

<p><strong>（第29章 終）</strong></p>
