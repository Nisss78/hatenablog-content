<h3>第45章：記憶の深淵</h3>

<p>宿の部屋で、カイトはベッドに横たわっていた。身体中の傷が痛むが、それよりも心に残るものがあった。収束者の言葉。そして、モーガンが語った「根源の間」の存在。</p>

<blockquote>「完全な消滅ではない……か」</blockquote>

<p>カイトは天井を見つめながら呟いた。収束者は概念的存在。魔力の残滓がある限り、何度でも蘇る。つまり、継承者が存在する限り、収束者もまた存在し続けるのだ。</p>

<pre><code>// Fundamental Problem
// 
// const convergerParadox = {
//     existence: "概念的実体",

//     source: "継承者の存在そのもの",
//     solution: "根本的な解決が必要"
// };

<p>窓の外からは、朝の光が差し込んでいた。一夜明け、王都は日常に戻っている。だが、カイトの心は晴れない。</p>

<p>ドアがノックされた。エリナの声がした。</p>

<blockquote>「カイト、入るよ」</blockquote>

<p>エリナが部屋に入ってきた。彼女の顔にも疲れが色濃く出ている。</p>

<blockquote>「傷はどう？」</blockquote>

<blockquote>「大したことない。自然治癒で治る」</blockquote>

<p>カイトは上半身を起こした。身体が重い。収束者との戦いで、体力を消耗し尽くした。</p>

<blockquote>「無理しないで。ヴェリディアンの力を使った反動もあるはず」</blockquote>

<p>エリナがカイトの隣に座った。彼女の双剣は、部屋の隅に立てかけられている。血がついたままだった。</p>

<blockquote>「エリナこそ、大丈夫なのか？」</blockquote>

<blockquote>「慣れてる。……って言いたいところだけど、正直きつい。あの収束者ってやつ、異常だった」</blockquote>

<p>エリナは自嘲気味に笑った。</p>

<blockquote>「魔法が使えない状況で戦うなんて、初めてだった。リアムなんて、弓に魔法をかけられないからって、物理的な矢だけで戦ってたよ」</blockquote>

<blockquote>「それで勝てたんだから、良しとしよう」</blockquote>

<p>カイトは言った。だが、心の中では別のことを考えていた。魔法が使えない状況。それが、これからの戦いの予兆なのかもしれない。</p>

<h4>◆</h4>

<p>一階の食堂で、四人は朝食をとっていた。リアムとセシリアも、昨夜の戦いの疲れを引きずっている。</p>

<blockquote>「根源の間、か……どこにあるんだろう」</blockquote>

<p>リアムがスープを啜りながら言った。</p>

<blockquote>「ヴェリディアンの記憶を探れば、何か分かるかもしれない」</blockquote>

<p>カイトが答えた。彼の体内には、千年の継承者が見てきた記憶が眠っている。その中に、根源の間の手がかりがあるはずだ。</p>

<blockquote>「でも、カイトさん。記憶の探索は危険です」</blockquote>

<p>セシリアが心配そうに言った。</p>

<blockquote>「ヴェリディアン様の記憶は、千年分もあります。深く潜りすぎると、自我が溶けてしまう可能性が……」</blockquote>

<blockquote>「分かってる。でも、やるしかない」</blockquote>

<p>カイトは決意を固めていた。収束者を完全に倒すためには、根源の書が必要だ。そのためには、リスクを冒さなければならない。</p>

<pre><code>// Memory Dive Plan

//

<p>エリナがカイトの手を握った。</p>

<blockquote>「私たちがついてる。何かあったら、すぐに引き戻す」</blockquote>

<blockquote>「頼む」</blockquote>

<p>カイトは深呼吸をした。そして、目を閉じた。</p>

<h4>◆</h4>

<p>意識が沈んでいく。周囲の音が遠ざかり、代わりに別の風景が浮かび上がってくる。</p>

<p>それは、古代の都市だった。高くそびえる塔、魔法で輝く街灯、空を飛ぶ人々。かつてこの世界に存在した、魔法文明の栄華。</p>

<blockquote>「これが……千年の世界」</blockquote>

<p>カイトは呟いた。自分の姿が半透明になっている。これは記憶の中、本来なら干渉できないはずの世界だ。</p>

<p>だが、ヴェリディアンの継承者として、カイトはある程度の干渉が可能だった。記憶を観察し、情報を引き出す。</p>

<p>街の中心に、巨大な図書館があった。魔法の光で満ち溢れ、無数の本が空を飛び回っている。</p>

<blockquote>「あれが……賢者たちの図書館」</blockquote>

<p>カイトは図書館に向かった。記憶の中とはいえ、足で歩く感覚がある。リアルすぎて、現実との境界が曖昧になる。</p>

<p>図書館の中に入ると、一人の女性が立っていた。長い銀髪、青いローブ、そして智慧を宿した瞳。</p>

<blockquote>「ヴェリディアン……？」</blockquote>

<p>カイトは問いかけた。だが、女性は反応しない。これは記憶の残像、ただの映像だ。</p>

<p>女性——ヴェリディアンは、本棚の前に立っていた。手に持っているのは、一冊の古い本。</p>

<blockquote>「根源の間への道……封印されている」</blockquote>

<p>ヴェリディアンの声が響いた。記憶の音声だ。</p>

<blockquote>「マーカス、君の言う通りだ。根源の書は危険すぎる。だが、収束者を倒すには必要かもしれない」</blockquote>

<p>カイトは耳をそばだてた。マーカス——モーガンが話していた賢者だ。</p>

<p>記憶が変化した。図書館から、洞窟の中へ。ヴェリディアンと数人の賢者が、地下を歩いている。</p>

<blockquote>「ここが、古代の遺跡……」</blockquote>

<p>一人の賢者が言った。顔見は悪いが、若き日のモーガンかもしれない。</p>

<blockquote>「根源の間は、さらに奥にある。だが、封印が強すぎる」</blockquote>

<p>ヴェリディアンが答えた。彼女の手が、壁に刻まれた古代文字に触れる。</p>

<blockquote>「開け……世界の理を司る門よ」</blockquote>

<p>魔法が発動した。壁が振動し、隠し扉が現れた。だが、扉は開かない。</p>

<blockquote>「ダメだ。封印が解けない」</blockquote>

<blockquote>「ヴェリディアン様、無理してはいけません。収束者が目覚めます」</blockquote>

<p>警告したのは、別の賢者だった。老人の姿、長い髭、鋭い目つき。</p>

<blockquote>「マーカス……収束者が目覚める？どういうことだ？」</blockquote>

<p>カイトは疑問に思った。この時点では、収束者はまだ封印されていたはずだ。</p>

<p>記憶がまた変化した。今度は、崩壊した都市。炎に包まれた建物、逃げ惑う人々。</p>

<blockquote>「収束者が……暴走した」</blockquote>

<p>ヴェリディアンが叫んだ。彼女の周りで、魔法が暴発している。</p>

<blockquote>「私のせいだ……根源の間を開こうとしたせいで……！」</blockquote>

<p>カイトは息を呑んだ。かつて、ヴェリディアンは根源の間を開こうとして失敗し、それが収束者の暴走を招いたのか。</p>

<pre><code>// Historical Revelation
// 
// const ancientTragedy = {
//     event: "収束者の暴走",
//     cause: "根源の間への干渉",

//     responsible: "ヴェリディアンの過ち"
// };
// 
// // 歴史が繰り返されるのか？</code></pre>

<p>カイトは更に深く潜った。記憶の核心に近づく。</p>

<h4>◆</h4>

<p>記憶が変わった。暗闇の中、ヴェリディアンが一人で立っていた。</p>

<blockquote>「誰だ？」</blockquote>

<p>カイトが問いかけると、ヴェリディアンが振り返った。彼女の瞳が、カイトを捉えた。</p>

<blockquote>「……継承者か。ここまで来るとはな」</blockquote>

<p>ヴェリディアンが口を開いた。これは単なる記憶の残像ではない。彼女の意識の欠片だ。</p>

<blockquote>「君は……生きてるのか？」</blockquote>

<p>カイトが問うた。ヴェリディアンは微かに笑った。</p>

<blockquote>「死んでいる。だが、完全に消滅したわけではない。継承者の体内で、意識の一部が生き続けている」</blockquote>

<blockquote>「じゃあ、君と話ができるのか？」</blockquote>

<blockquote>「限定的にな。記憶の深層でしか、こうして対話できない」</blockquote>

<p>カイトは質問を変えた。</p>

<blockquote>「根源の間はどこにある？収束者を倒すために必要だ」</blockquote>

<p>ヴェリディアンの表情が曇った。</p>

<blockquote>「……知りたいのか。かつて私が犯した過ちを、繰り返すつもりか？」</blockquote>

<blockquote>「過ち？」</blockquote>

<blockquote>「根源の間を開けば、収束者が目覚める。古代文明を滅ぼしたのは、収束者の暴走だ。私は封印を解こうとして、世界を滅ぼしかけた」</blockquote>

<p>カイトは衝撃を受けた。根源の間には、収束者を倒す力があると同時に、収束者を目覚めさせる危険もあるのか。</p>

<blockquote>「じゃあ、どうすれば……」</blockquote>

<blockquote>「答えは、根源の書にある。だが、その書を読むには、根源の間に入らねばならない。矛盾しているだろう？」</blockquote>

<p>ヴェリディアンが自嘲気味に笑った。</p>

<blockquote>「千年の間、私は答えを探し続けた。だが、見つからなかった。君なら、見つけられるかもしれない」</blockquote>

<blockquote>「どうしてそう思う？」</blockquote>

<blockquote>「君はプログラマーだ。論理で物事を捉える。魔法の世界の住人とは、根本的に思考が違う」</blockquote>

<p>カイトは息を呑んだ。ヴェリディアンは、カイトの出自を知っているのか。</p>

<blockquote>「君の記憶は、私も共有している。異世界から来たプログラマー。コードで魔法を書く者。君の視点は、この世界にはない」</blockquote>

<p>ヴェリディアンがカイトに近づいた。</p>

<blockquote>「根源の間は、世界の最北にある。万年雪に覆われた山脈、その奥だ」</blockquote>

<blockquote>「最北……どれくらい遠い？」</blockquote>

<blockquote>「王都から馬で二ヶ月。魔法を使えば、半月」</blockquote>

<p>カイトは計算した。二ヶ月は長すぎる。半月でも、長い旅になる。</p>

<blockquote>「一つだけ、アドバイスをしておこう」</blockquote>

<p>ヴェリディアンが言った。</p>

<blockquote>「根源の間には、三人の守人がいる。彼らは、継承者が来るのを待っている」</blockquote>

<blockquote>「守人？」</blockquote>

<blockquote>「かつて私が任命した、根源の書の守護者たちだ。彼らを倒せば、根源の書にアクセスできる。だが、倒すには魔法が必要だ。収束者の影響下では、魔法が使えない」</blockquote>

<p>カイトは理解した。矛盾した状況だ。魔法を使えば収束者が強化される。だが、魔法を使わなければ守人を倒せない。</p>

<pre><code>// Paradox Deepens

<blockquote>「どうすればいい？」</blockquote>

<p>カイトが問うた。ヴェリディアンは首を振った。</p>

<blockquote>「それは、君が見つけるのだ。私には分からない」</blockquote>

<p>ヴェリディアンの姿が、少しずつ薄くなっていった。</p>

<blockquote>「時間だ。これ以上深層に留まると、君の自我が溶ける」</blockquote>

<blockquote>「待って、まだ聞きたいことが……」</blockquote>

<blockquote>「また会おう、継承者よ。君に幸あれ」</blockquote>

<p>ヴェリディアンの姿が消えた。カイトの意識が、急速に現実へと引き戻された。</p>

<h4>◆</h4>

<p>「カイト！カイト！」</p>

<p>エリナの声で、カイトは目を覚ました。宿の部屋、ベッドの上。現実に戻った。</p>

<blockquote>「大丈夫か？三時間も昏睡状態だったんだぞ」</blockquote>

<p>リアムが心配そうに言った。セシリアが、カイトの額に手を当てている。</p>

<blockquote>「意識は戻りました。ですが、魔力の残滓が不安定です」</blockquote>

<p>カイトは深呼吸をした。記憶の探索で、多くの情報を得た。だが、同時に新たな謎も生まれた。</p>

<blockquote>「みんな、聞いてくれ」</blockquote>

<p>カイトは、ヴェリディアンから聞いた情報を仲間たちに話した。根源の間の場所、守人の存在、そして魔法を使うことのジレンマ。</p>

<blockquote>「つまり、魔法を使わずに守人を倒せってことか？」</blockquote>

<p>リアムが困惑した表情で言った。</p>

<blockquote>「無理だろ。物理だけで強敵を倒すなんて」</blockquote>

<blockquote>「収束者の時はできた」</blockquote>

<p>エリナが反論した。</p>

<blockquote>「あれは特殊な状況だった。結界があったから……」</blockquote>

<p>カイトは考えた。魔法を使わずに守人を倒す方法。あるいは、魔法を使っても収束者を強化しない方法。</p>

<blockquote>「一つだけ、方法があるかもしれない」</blockquote>

<p>セシリアが口を開いた。</p>

<blockquote>「古代魔法の中には、『魔力の届かない領域』に干渉する術式があります。収束者の影響を受けない、純粋な魔法です」</blockquote>

<blockquote>「そんな魔法があるのか？」</blockquote>

<blockquote>「理論上は。ですが、実践されたことはありません」</blockquote>

<p>カイトの心臓が速鐘を打った。収束者の影響を受けない魔法。それがあれば、ジレンマを解決できる。</p>

<blockquote>「その魔法、どうやったら習得できる？」</blockquote>

<blockquote>「古代の遺跡に、術式が残されているかもしれません。王都の地下図書館に、手がかりがあるはずです」</blockquote>

<p>カイトは立ち上がった。身体はまだ重いが、意志は固まっていた。</p>

<blockquote>「よし。まずは王都の地下図書館だ。そこから始めよう」</blockquote>

<h4>◆</h4>

<p>カイトたちは、王都の地下図書館に向かった。王宮の許可を得て、賢者たちが集う場所へ。</p>

<p>図書館は広大だった。天井が見えないほど高く、無数の本棚が迷路のように続いている。</p>

<blockquote>「すごい……」</blockquote>

<p>リアムが感嘆した。</p>

<blockquote>「王都の知識が、ここに集まってるんだ」</blockquote>

<p>カイトは、セシリアの指示に従って、古代魔法のセクションに向かった。埃っぽい本棚、ほとんど読まれていないであろう古い本。</p>

<blockquote>「これだ」</blockquote>

<p>セシリアが一冊の本を手に取った。タイトルは、古代文字で書かれている。</p>

<blockquote>「『魔力の根源とその反転』……これは、古代の賢者が書いた論文です」</blockquote>

<p>カイトは本を開いた。難解な文字と図式が並んでいる。だが、彼の目には、それがコードのように見えた。</p>

<pre><code>// Ancient Theory
// 
// const manaInversionTheory = {

//     solution: "破壊の力を使えば干渉されない"

<p>カイトは理解した。魔力には二つの側面がある。創造の力と破壊の力。収束者が反応するのは、創造の力だけ。破壊の力を使えば、収束者の影響を受けずに魔法が使える。</p>

<blockquote>「破壊の力……それが、収束者に効く魔法だ」</blockquote>

<p>カイトが呟いた。エリナが顔をしかめた。</p>

<blockquote>「破壊の力って、危険なんじゃない？」</blockquote>

<blockquote>「だが、使わないと守人を倒せない」</blockquote>

<p>カイトは本を閉じた。リスクはある。だが、選択肢は限られている。</p>

<blockquote>「破壊の力を制御する方法を探そう。それが、次の課題だ」</blockquote>

<p>カイトの目が、決意に燃えていた。旅の目的は変わった。根源の間へ行き、守人を倒し、根源の書を手に入れる。そのためには、破壊の力を制御しなければならない。</p>

<p>冒険は、まだ始まったばかりだった。</p>
