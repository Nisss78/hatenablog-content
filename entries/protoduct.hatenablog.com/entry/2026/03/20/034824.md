<h3>第66章：虚空の来襲</h3>

<p>黒い影が空から降りてくる。その姿は不定形で、まるで生きている闇そのものだった。無数の目が空中に浮かび、すべてがカイトを凝視している。</p>

<blockquote>「これが……虚空の侵入者」</blockquote>

<p>カイトは息を呑んだ。存在するだけで空気が歪むような、異質な圧迫感。プログラマーとしての彼の直感が警鐘を鳴らしていた。これは通常のバグではない。システムそのものを脅かす、致命的なエラーだ。</p>

<p>エーテリアがカイトの隣に立った。</p>

<blockquote>「カイト、彼らを『虚空の渇望者（ヴォイド・ハンガー）』と呼びます。論理も法則も持たない、純粋な消費の存在です」</blockquote>

<blockquote>「消費って……何を？」</blockquote>

<blockquote>「すべてを。魔力も、命も、世界そのものも」</blockquote>

<p>黒い影が動いた。触手のようなものを伸ばし、地面に触れた瞬間、その部分が灰色に変色して崩れ落ちた。</p>

<blockquote>「空間そのものを侵食してるのか……！」</blockquote>

<p>カイトは素早くコードを展開した。</p>

<pre><code>

<p>カイトの周囲に、青白い光の格子が浮かび上がった。空間を固定する結界だ。</p>

<blockquote>「エーテリア、この結界で侵食を止められるか？」</blockquote>

<blockquote>「一時的には。ですが、彼らは適応します。同じ攻撃は二度と通じない」</blockquote>

<p>黒い影が再び動いた。今度は、結界に向かって無数の触手を突き出してくる。</p>

<p>触手が結界に触れた瞬間、激しい火花が散った。</p>

<blockquote>「ぐっ……！」</blockquote>

<p>カイトは膝をついた。結界を維持するために、彼の魔力が急速に消耗していく。</p>

<blockquote>「このままじゃ、持たない……」</blockquote>

<p>その時、空から光の矢が降り注いだ。</p>

<blockquote>「カイト！ 大丈夫！？」</blockquote>

<p>見上げると、エリナが空を飛んでいた。彼女の背中には、光の翼が広がっている。</p>

<blockquote>「エリナ！ どうやってここに？」</blockquote>

<blockquote>「管理者の人が呼んでくれたの。みんなも来てるよ！」</blockquote>

<p>地面から、土柱が勢いよく伸びた。リアムだ。彼の周囲には、岩石が浮遊している。</p>

<blockquote>「遅れてすまん。ここの座標が特定できなくてな」</blockquote>

<blockquote>「リアム……助かった」</blockquote>

<p>さらに、森の方から風が吹いてきた。セシリアが、無数の風の刃を纏って現れる。</p>

<blockquote>「敵の数が多いわね。包囲されないように気をつけて」</blockquote>

<p>四人の仲間が勢揃いした。しかし、黒い影は止まらない。むしろ、仲間が現れたことで活性化したかのように、その動きが激しくなった。</p>

<blockquote>「増えた……」</blockquote>

<p>エリナが空を指差した。空の亀裂から、次々と黒い影が溢れ出している。</p>

<blockquote>「あの数……どうなってるの！？」</blockquote>

<p>エーテリアが冷静に分析した。</p>

<blockquote>「裂け目が拡大しています。一つの亀裂から、百体以上の渇望者が侵入中」</blockquote>

<blockquote>「百体！？」</blockquote>

<p>カイトは絶望しかけたが、すぐに頭を振った。プログラマーとしての彼は、困難なバグに対処する時こそ冷静になると決めている。</p>

<pre><code>

<p>カイトは叫んだ。</p>

<blockquote>「みんな、聞いて！ 俺たちの目的は敵を全滅させることじゃない。あの裂け目を塞ぐことだ！」</blockquote>

<blockquote>「つまり、侵入経路を断てばいいのね？」</blockquote>

<p>セシリアが確認した。</p>

<blockquote>「そうだ。リアム、頼む。俺たちを守る壁を作ってくれ」</blockquote>

<blockquote>「任せろ」</blockquote>

<p>リアムが地面に手を置いた。すると、巨大な岩の壁が隆起し、黒い影の進行を阻害した。</p>

<blockquote>「セシリア、敵の数を減らしてくれ。範囲攻撃で！」</blockquote>

<blockquote>「了解」</blockquote>

<p>セシリアが両手を広げた。嵐のような風が巻き起こり、無数の風の刃が黒い影を切り裂いていく。</p>

<blockquote>「エリナ、高所から敵の動きを牽制して。俺が裂け目に近づくまでの時間を稼いでくれ」</blockquote>

<blockquote>「わかった！ 絶対に死なないでよ！」</blockquote>

<p>エリナが空へ舞い上がり、光の矢を連射し始めた。</p>

<p>カイトは裂け目に向かって走り出した。その背中を、仲間たちが必死に守る。</p>

<p>黒い影が次々と迫ってくるが、リアムの岩壁が道を塞ぐ。隙間から漏れた敵は、セシリアの風が吹き飛ばす。空からの奇襲は、エリナの光が迎撃した。</p>

<blockquote>「いいチームだ……」</blockquote>

<p>カイトは感心しながら、裂け目の下に到達した。空に浮かぶ亀裂は、百メートル以上の長さがある。</p>

<blockquote>「これを塞ぐのか……巨大すぎる」</blockquote>

<p>エーテリアが隣に現れた。</p>

<blockquote>「一人では不可能です。ですが、四人の力を束ねれば可能かもしれません」</blockquote>

<blockquote>「四人の力を……束ねる？」</blockquote>

<blockquote>「共鳴陣（レゾナンス・サークル）を使います。四人の魔力を一つにし、あなたのコードに流し込むのです」</blockquote>

<p>カイトは理解した。並列処理だ。複数のリソースを統合して、一つの巨大なタスクを処理する。</p>

<pre><code>
// 並列魔力処理システム
class ManaParallelProcessing {
    constructor(sources) {
        this.sources = sources; // [カイト, エリナ, リアム, セシリア]
        this.totalMana = 0;
    }
    
    synchronize() {
        // 全員の魔力を同期
        for (let source of this.sources) {
            this.totalMana += source.channelMana();
        }
        return this.totalMana;
    }
    
    executeRiftSeal(coords, mana) {
        const requiredMana = this.calculateRequired(coords);
        if (mana >= requiredMana) {
            return this.patchDimension(coords);
        }
        return 'INSUFFICIENT_MANA';
    }
    
    patchDimension(coords) {
        // 次元の裂け目を修復
        const patch = new DimensionalPatch();
        patch.setCoordinates(coords);
        patch.setSize('LARGE');
        patch.apply();
        return 'SUCCESS';
    }
}
</code></pre>

<p>カイトは叫んだ。</p>

<blockquote>「みんな！ 俺に魔力を分けてくれ！ この裂け目を塞ぐ！」</blockquote>

<blockquote>「わかった！」</blockquote>

<p>エリナが空から降りてきた。リアムとセシリアも、カイトの周囲に集まる。</p>

<blockquote>「どうすればいい？」</blockquote>

<p>リアムが尋ねた。</p>

<blockquote>「俺の手を握って。意識を集中して、魔力を流し込んでくれ」</blockquote>

<p>四人が円になり、手を取り合った。その中心に、カイトがコードを展開する。</p>

<p>光の文字が螺旋を描いて天へと昇っていく。</p>

<blockquote>「共鳴開始……！」</blockquote>

<p>四人の体から、色とりどりの光が溢れ出した。エリナの金色、リアムの茶色、セシリアの銀色、そしてカイトの青色。四つの光が一つに絡み合い、巨大なエネルギーの奔流となった。</p>

<p>黒い影たちが気づき、一斉にこちらへ向かってくる。</p>

<blockquote>「邪魔させるか！」</blockquote>

<p>エーテリアが両手を広げた。銀色の結界が、四人を守るドームとなった。</p>

<blockquote>「急いでください。私の結界も永くは持ちません」</blockquote>

<p>カイトは集中した。彼の意識が、次元の境界線へと侵入していく。</p>

<p>そこで彼が見たのは、世界の裏側だった。</p>

<p>無数の光の糸が複雑に絡み合い、現実世界を構成している。しかし、その糸の一部が断裂し、虚空へと続く穴が開いていた。</p>

<blockquote>「これが……裂け目の正体」</blockquote>

<p>カイトは震えた。あまりにも美しく、そして脆い世界の構造。</p>

<blockquote>「修復コード、実行……」</blockquote>

<pre><code>
// 次元修復プロセス
function repairDimensionalRift(rift) {
    // 断裂した光の糸を再接続
    const threads = rift.getSeveredThreads();
    
    for (let thread of threads) {
        thread.reconnect();
        thread.reinforce();
    }
    
    // 空間の穴を埋める
    const hole = rift.getVoidHole();
    hole.fill('light_mana');
    
    // 最後に、境界線を強化
    const boundary = rift.getBoundary();
    boundary.strengthen(4); // 四人の力で強化
    
    return 'RIFT_SEALED';
}
</code></pre>

<p>カイトの指先から、修復のコードが溢れ出した。それは、断裂した光の糸を一つずつ繋ぎ直していく。</p>

<p>空の裂け目が、少しずつ縮んでいった。</p>

<p>黒い影たちが悲鳴を上げた。侵入経路が消えることに気づいたのだ。</p>

<blockquote>「あと少し……！」</blockquote>

<p>カイトの額から汗が流れ落ちた。四人の魔力が、限界に近づいている。</p>

<blockquote>「みんな、もう少し耐えてくれ！」</blockquote>

<blockquote>「大丈夫……まだいける……」</blockquote>

<p>エリナが震える声で答えた。彼女の手が、カイトの手を強く握りしめた。</p>

<p>裂け目は、最後に激しく抵抗した。虚空の力が、最後の悪あがきとして四人を押し返そうとする。</p>

<blockquote>「負けるな……！」</blockquote>

<p>カイトは叫んだ。</p>

<blockquote>「俺たちは絶対に、この世界を守る！」</blockquote>

<p>その瞬間、四人の光が最大まで輝いた。</p>

<p>眩い閃光が、空の裂け目を飲み込んだ。</p>

<p>そして――</p>

<p>静寂が訪れた。</p>

<p>空を見上げると、裂け目は完全に消滅していた。二つの月だけが、静かに輝いている。</p>

<blockquote>「やった……のか？」</blockquote>

<p>リアムが膝をついた。</p>

<blockquote>「ああ……塞がった。裂け目は消えた」</blockquote>

<p>カイトもその場に座り込んだ。全身の力が抜けている。</p>

<blockquote>「みんな、ありがとう。一人じゃ絶対に無理だった」</blockquote>

<p>エリナがカイトに寄りかかった。</p>

<blockquote>「私たちがいれば、何でもできるってことね」</blockquote>

<p>セシリアが疲れた笑みを浮かべた。</p>

<blockquote>「でも、これで終わりじゃないんでしょ？」</blockquote>

<p>エーテリアが頷いた。</p>

<blockquote>「ええ。塞いだのは一つの裂け目だけです。世界中に、あと三十二カ所の異常ポイントがあります」</blockquote>

<blockquote>「三十二カ所……」</blockquote>

<p>カイトは空を見上げた。二つの月が、彼を見下ろしている。</p>

<blockquote>「長い旅になりそうだな」</blockquote>

<p>その時、遠くから誰かが歩いてくるのが見えた。黒いローブを着た、背の高い男。</p>

<blockquote>「誰だ？」</blockquote>

<p>リアムが警戒態勢を取った。</p>

<p>男は静かに近づいてきて、カイトたちの前に立った。その顔は、深いフードに隠れて見えない。</p>

<blockquote>「初めまして、カイト。私はマルクス。収束者と呼ばれている」</blockquote>

<p>低く、落ち着いた声だった。</p>

<blockquote>「エーテリアから話は聞いている。君たちの力になりたい」</blockquote>

<p>カイトは立ち上がった。新しい協力者の登場だ。</p>

<blockquote>「よろしく、マルクス。俺たちの目標は同じだ。この世界の裂け目を塞ぐこと」</blockquote>

<p>マルクスが微かに微笑んだようだった。</p>

<blockquote>「ええ。ですが、最初に警告しておかなければならないことがあります」</blockquote>

<p>彼は空を指差した。</p>

<blockquote>「君たちが今戦ったのは、渇望者の一部に過ぎません。彼らの本体は、まだ虚空の向こう側にいます。そして……彼らは君のことを知っています、カイト」</blockquote>

<blockquote>「俺のこと？」</blockquote>

<blockquote>「君が世界を書き換えた瞬間、虚空の深淵に響いたのです。彼らは君を『呼び出した者』と呼んでいる。彼らにとって、君は特別な存在です」</blockquote>

<p>カイトは背筋が凍るのを感じた。</p>

<blockquote>「特別って……何の意味だ？」</blockquote>

<blockquote>「それは、これからの旅でわかるでしょう。今は、休むことをお勧めします」</blockquote>

<p>マルクスがフードを脱いだ。その顔には、無数の傷跡が刻まれていた。</p>

<blockquote>「私は長い間、次元の境界線を守ってきました。しかし、これほど大きな脅威は初めてです。君たちの力が必要です、カイト」</blockquote>

<p>カイトは頷いた。</p>

<blockquote>「わかった。一緒に戦おう。この世界を守るために」</blockquote>

<p>五人の冒険者と一人の収束者。そして、管理者エーテリア。</p>

<p>彼らの旅は、まだ始まったばかりだった。</p>

<p>世界中に散らばる三十二の裂け目。虚空の深淵に潜む渇望者の本体。そして、カイト自身の秘密。</p>

<p>すべての謎が交錯する中、次の目的地は――北方の山脈、標高三千米の地点。</p>

<p>そこに、最も大きな裂け目が待っている。</p>

<p>第66章、終。</p>
