<h3>第77章：誓いの刻印</h3>

<p>灼熱の誓いを立てた瞬間、カイトの体を激烈な熱波が貫いた。それは物理的な炎ではない——魂そのものを焼き焦がす、存在の根源に刻まれる契約の熱だった。</p>

<p>「ぐっ……！」</p>

<p>膝をついたカイトの視界が白く染まる。意識の奥底で、何かが書き換えられていく感覚。まるでOSのカーネルが再構築されるような、存在そのもののアップデート。</p>

<pre><code>// Oath Contract Binding
function bindOath(soul, oath) {
  soul.core.contract = oath;
  soul.permissions.push('MAINTAINER_PRIME');
  soul.resilience *= 2;
  return soul.commit();
}</code></pre>

<p>目の前に浮かんだのは、見慣れたコンソールログではない。古代文字と現代的なコードが融合した、光り輝く契約の紋章だった。</p>

<blockquote>「カイト！しっかりして！」</blockquote>

<p>エリナの声が遠くから響く。だがカイトは、その声に応える余裕すらなかった。体内で暴れ回るエネルギーが、血管を、神経を、魂の回路を奔流となって駆け巡る。</p>

<blockquote>「契約が……完了したわ」</blockquote>

<p>エーテリアの静かな声が、嵐の中の灯台のように響いた。</p>

<blockquote>「彼は今、世界そのものと直接リンクしている。メンテナーとしての権限が、彼の魂に刻印されたのよ」</blockquote>

<p>カイトの背中に、熱が集まっていく。皮膚の下で何かが形作られる感覚——そして、灼熱の光が背中から噴き出した。</p>

<p>光が収まったとき、カイトの背中には複雑な紋章が刻まれていた。コードの構造をした魔法陣。世界を構成する言語そのものが、彼の肉体の一部となったのだ。</p>

<h4>■ 新たなる力</h4>

<p>カイトはゆっくりと立ち上がった。体は熱い。だが、不快な熱ではない。力が満ちている。世界と自分が繋がっている感覚が、鮮明に伝わってくる。</p>

<blockquote>「これが……メンテナーの感覚か」</blockquote>

<p>目を閉じれば、世界中の「ノイズ」が感じられる。歪んだ次元の裂け目、侵食されつつある現実の断片、そして——さらに奥にある、巨大な「何か」の存在。</p>

<pre><code>// World Status Scan
world.scan() {
  anomalies: 31,
    critical: 3,
      growing: 'VOID_NEXUS',
        status: 'STABLE_BUT_FRAGILE'
}</code></pre>

<p>三十一箇所の異常。そのうち三つが危急。そして——虚空の連結点（ヴォイド・ネクサス）と呼ばれる何かが、静かに、しかし確実に膨れ上がっている。</p>

<blockquote>「ヴォイド・ネクサス……？」</blockquote>

<p>カイトが呟くと、エーテリアが説明した。</p>

<blockquote>「全ての裂け目の中心。侵食の源泉であり、構築者が目指している最終地点よ。そこに到達されれば——世界というプログラム自体が、強制終了される」</blockquote>

<blockquote>「強制終了……つまり、世界の終わりってことか」</blockquote>

<blockquote>「ええ。でも今のあなたなら、防げるかもしれない」</blockquote>

<p>エーテリアの視線が、カイトの背中の紋章に向けられる。</p>

<blockquote>「その刻印は、世界への管理者権限そのもの。あなたは今、現実のコードに直接干渉できる。理論上は、ね」</blockquote>

<blockquote>「理論上ってのは？」</blockquote>

<blockquote>「権限があっても、使い方がわからなければ意味がない。それに——世界を書き換えるには、相応の対価が必要よ」</blockquote>

<p>カイトは自分の手を見た。掌に浮かぶ淡い光。これは単なる力ではない。世界そのものを操作する、絶対的な権限だ。</p>

<h4>■ 試練の前夜</h4>

<p>一行は、王都アークロイヤルの宿に戻っていた。カイトの覚醒を見届けた後、エーテリアが提案したのだ。</p>

<blockquote>「一度、態勢を整えなさい。これから始まる戦いは、これまでとは次元が違う。準備が必要よ」</blockquote>

<p>宿の部屋で、カイトは一人、窓の外を眺めていた。夜の街。変わらぬ日常。人々は知らない——世界が滅びの瀬戸際にあることを。</p>

<blockquote>「入るわよ」</blockquote>

<p>ノックもそこそこに、エリナが部屋に入ってきた。表情には珍しく、不安の色がある。</p>

<blockquote>「……大丈夫？」</blockquote>

<p>カイトは振り返る。エリナの紫の瞳が、心配そうに彼を見つめていた。</p>

<blockquote>「大丈夫だと思う。ただ——少し、重いな」</blockquote>

<blockquote>「重い？」</blockquote>

<blockquote>「世界の重さ、ってやつかな。背中のこの刻印……ただのマークじゃない。世界中の異常を感じ取れる。誰かが傷ついたらわかるし、どこかが壊れそうになったら警告が来る」</blockquote>

<p>カイトは苦笑した。</p>

<blockquote>「便利そうだけど、四六時中世界中の悲鳴が聞こえると思ってみろ。結構、きついぞ」</blockquote>

<p>エリナは沈黙した。そして、不意にカイトの隣に座った。</p>

<blockquote>「なら、私がそばにいるわ」</blockquote>

<blockquote>「え？」</blockquote>

<blockquote>「一人で背負う必要なんてない。あなたが世界の悲鳴を聞くなら、私はあなたの悲鳴を聞く。それでいいでしょ？」</blockquote>

<p>カイトは驚いてエリナを見た。彼女は窓の外を見ている。月光に照らされた横顔は、どこか寂しげで、でも力強かった。</p>

<blockquote>「……ありがとう、エリナ」</blockquote>

<blockquote>「礼なんていらないわ。ただ——無理して死なないでってこと。カイトがいなくなったら、誰が私の魔法のバグを直すのよ」</blockquote>

<p>その言葉に、カイトは思わず笑ってしまった。</p>

<blockquote>「バグ直しかよ」</blockquote>

<blockquote>「一番大事な仕事でしょ」</blockquote>

<p>エリナも微笑む。二人の間に、穏やかな沈黙が流れた。</p>

<h4>■ 襲撃</h4>

<p>平和な時間は、長くは続かなかった。</p>

<p>深夜、警報のような音がカイトの脳内で鳴り響いた。背中の刻印が熱を帯びる——緊急事態の合図だ。</p>

<blockquote>「来たか！」</blockquote>

<p>カイトは窓から飛び出した。夜空に、異様な影が浮かんでいた。</p>

<p>四十七体の侵食体。だが、以前の敵とは違う。より洗練された形。より攻撃的なデザイン。</p>

<blockquote>「構築者の手先だ」</blockquote>

<p>後ろから、リアムとセシリアが駆けつけてきた。</p>

<blockquote>「カイト！何が起きている！？」</blockquote>

<blockquote>「王都への襲撃だ。でも——様子がおかしい」</blockquote>

<p>カイトは目を細めた。侵食体たちは、無秩序に暴れているわけではない。整然とした陣形。明確な目的を持った動き。</p>

<blockquote>「まるで……軍隊だ」</blockquote>

<p>その時、侵食体の中央から、一際大きな影が現れた。人型。だが、その姿は歪んでいる。半分は機械的な鋭さを持ち、半分は肉体的な有機性を残す——キメラのような存在。</p>

<blockquote>「メンテナーを発見」</blockquote>

<p>影が声を発した。機械的な、感情のない声。</p>

<blockquote>「対象の排除を開始します。構築者の意志に従い、世界をリセットする」</blockquote>

<blockquote>「リセットだと？」</blockquote>

<p>カイトが問い返すと、影は答えた。</p>

<blockquote>「この世界は失敗作です。バグだらけのコード。構築者は、より完璧な現実を創造するため、一度全てを消去します。あなたはその障害。よって——排除対象として認定されました」</blockquote>

<p>カイトは背中の刻印を感じた。熱い。怒りに呼応して、力が満ちていく。</p>

<blockquote>「面白え。やってみろよ」</blockquote>

<h4>■ メンテナーの力</h4>

<p>戦闘は一瞬で始まった。</p>

<p>侵食体たちが一斉に襲いかかる。だが——カイトは動じなかった。目を閉じ、世界とリンクする。</p>

<pre><code>// Reality Overwrite

<p>カイトが掌を開くと、周囲の重力が劇的に変化した。侵食体たちが浮き上がり、制御を失う。</p>

<blockquote>「これが、メンテナーの力——！」</blockquote>

<p>リアムが驚愕の声を上げる。セシリアも目を見開いた。</p>

<blockquote>「現実そのものを書き換えている……！」</blockquote>

<p>だが、カイトは冷静だった。この力には代償がある——使うたびに、自身の生命力が削られていく感覚。</p>

<blockquote>「長くは使えない。一気に決めるぞ！」</blockquote>

<p>エリナが魔法を放つ。リアムが剣を振るう。セシリアが支援魔法を展開する。そしてカイトは——現実のコードを書き換えながら、敵を圧倒していく。</p>

<p>侵食体たちが次々と消滅していく。だが、中央の指揮官型だけは、容易には堕ちなかった。</p>

<blockquote>「優秀なコードです。ですが——不完全」</blockquote>

<p>指揮官型が腕を振ると、カイトの重力操作が打ち消された。</p>

<blockquote>「管理者権限の一部は、我々も保持しています。対抗は可能」</blockquote>

<blockquote>「くそっ——！」</blockquote>

<p>カイトは舌打ちした。構築者側も、この世界へのアクセス権を持っている。一方的な圧倒はできない。</p>

<p>なら——</p>

<pre><code>// Debug Mode: Direct Intervention

<p>カイトの体が光り輝く。限界を超えた力。六十秒間だけ、彼らは「世界の外側」から干渉できる存在となった。</p>

<blockquote>「みんな、行くぞ！一気に決める！」</blockquote>

<p>四人の連携攻撃。エリナの破壊魔法、リアムの剣技、セシリアの聖なる光——そしてカイトの現実改変が、一つの点に収束する。</p>

<p>指揮官型が悲鳴を上げた。</p>

<blockquote>「エラー——エラー——存在の論理的矛盾を検知——修復不能——」</blockquote>

<p>光の中で、指揮官型は分解されていった。コードの断片となって、夜空に散っていく。</p>

<h4>■ 刻印の代償</h4>

<p>戦闘が終わった後、カイトは膝をついた。</p>

<p>激しい疲労感。背中の刻印が、焼けるように熱い。</p>

<blockquote>「カイト！」</blockquote>

<p>エリナが駆け寄る。カイトはなんとか笑顔を作った。</p>

<blockquote>「大丈夫……ただ、ちょっと使いすぎたな」</blockquote>

<p>手を見る。震えている。メンテナーの力は強大だが、その代償は想像以上に重い。</p>

<blockquote>「無茶よ」</blockquote>

<p>エーテリアの声が、空中から響いた。</p>

<blockquote>「あなたはまだ、メンテナーになりたて。力を使いすぎれば、魂が摩耗するわ。最悪の場合——あなた自身がバグとなって、世界から排除される」</blockquote>

<blockquote>「……明るい話だな」</blockquote>

<blockquote>「現実は甘くない。でも——あなたは一人じゃない」</blockquote>

<p>エーテリアの言葉に、カイトは仲間たちを見た。エリナ、リアム、セシリア。三人とも、心配そうに彼を見つめている。</p>

<blockquote>「……そうだな。一人じゃ、背負いきれない重さだ」</blockquote>

<p>カイトはゆっくりと立ち上がった。</p>

<blockquote>「でも、誰かがやらなきゃいけない。この世界を守るのは、俺たちしかいないんだ」</blockquote>

<p>夜空を見上げる。星が綺麗だ。この景色を、守りたい。</p>

<blockquote>「行こう。ヴォイド・ネクサスへ」</blockquote>

<p>カイトの言葉に、三人が頷いた。</p>

<p>世界の運命をかけた、最後の戦いが始まろうとしていた。</p>
