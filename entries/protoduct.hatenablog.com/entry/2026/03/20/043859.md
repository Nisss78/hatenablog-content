<h3>第76章：灼熱の誓い</h3>

<p>北の氷原地帯での戦いを終えたカイトたちは、転移魔法陣を使って一時的に安全地帯へと撤退していた。第一ノードの安定化には成功したが、その代償としてカイトの魔力回路には深刻な負荷がかかっていた。</p>

<blockquote>「カイト、顔色が悪いわよ」</blockquote>

<p>エリナが心配そうにカイトの額に手を当てた。その手からは、微かに温かい治癒の魔力が感じられる。</p>

<blockquote>「大丈夫だ。ただ、管理者権限を使いすぎた」</blockquote>

<p>カイトは苦笑しながら答えた。氷原地帯での戦いでは、凍りついたノードを再起動させるために、彼は自身の魔力回路を直接接続していた。それは、プログラマーが本番環境でデバッグを行うような危険な行為だった。</p>

<pre><code>
// 警告：管理者リソース枯渇
// 現在の魔力レベル：23%
// 推定復旧時間：四十八時間
// 推奨：低負荷オペレーション継続
</code></pre>

<p>カイトの視界には、管理者としてのシステム情報が常に表示されている。それは便利ではあったが、同時に彼の精神的な負担を増やしていた。</p>

<blockquote>「残り二つのノードがあるんだろ？あんた、この状態で戦えるのか？」</blockquote>

<p>リアムの懸念はもっともだった。カイトの現在の魔力レベルでは、大規模な戦闘は避けるべきだった。</p>

<blockquote>「私がサポートするわ」</blockquote>

<p>セシリアが静かに宣言した。</p>

<blockquote>「氷原地帯では、カイトがほとんどの負荷を背負った。次は、私たちがもっと役に立つべきよ」</blockquote>

<p>カイトは仲間たちの顔を見渡した。エリナ、リアム、セシリア――彼らは、カイトが異世界に召喚されて以来、ずっと共に戦ってきた仲間だ。彼らの成長と絆は、カイトにとって何よりの支えだった。</p>

<blockquote>「頼む。次は東の砂漠遺跡だ。第二ノードがある場所へ」</blockquote>

<p>カイトは地図を表示させた。砂漠遺跡は、北の氷原地帯から南東に約三千キロメートル。転移魔法を使えば数時間で到達できるが、魔力消費を考えると慎重なルート選択が必要だった。</p>

<pre><code>
class RouteOptimizer {
    constructor(currentLocation, destinations, manaBudget) {
        this.current = currentLocation;
        this.destinations = destinations;
        this.budget = manaBudget;
    }
    
    async findOptimalRoute() {
        const routes = await this.calculateAllPaths();
        const viable = routes.filter(r => r.manaCost <= this.budget);
        return viable.sort((a, b) => a.timeCost - b.timeCost)[0];
    }
    
    // 最適化の考慮事項：
    // - 転移魔法陣の利用可能箇所
    // - 魔力充電スポット
    // - 敵対勢力の活動領域
}
</code></pre>

<p>カイトは管理者権限を使って、最適なルートを計算した。北の氷原地帯から、中継地点である「風の谷」を経由して、砂漠遺跡へ至るルートが選択された。</p>

<blockquote>「風の谷で休憩しよう。そこなら、魔力を回復できる」</blockquote>

<p>カイトの提案に、全員が同意した。</p>

---

<p>風の谷は、その名の通り、常に強風が吹き抜ける峡谷だった。だが、その風は敵意を持つものではなく、むしろ訪れる旅人を歓迎するかのように優しかった。</p>

<blockquote>「ここは……いいところね」</blockquote>

<p>エリナが峡谷の縁に座りながら呟いた。眼下には、緑豊かな谷間が広がっている。風が運ぶ花の香りが、彼女の鼻をくすぐった。</p>

<blockquote>「風の精霊が守っている場所だ」</blockquote>

<p>セシリアが説明した。</p>

<blockquote>「昔、ここに住んでいた賢者が、風の精霊と契約を結んだの。それ以来、この谷は旅人の安息の地として知られているわ」</blockquote>

<p>カイトは峡谷の風を感じながら、管理者としての情報を確認していた。風の谷には、魔力を充電できる「風力変換機」と呼ばれる古代の装置が存在した。</p>

<pre><code>
// 風力変換機ステータス
// 動作状態：正常
// 出力：1.2 MW（魔力ワット）
// 充電効率：87%
// 推定満充電時間：六時間
</code></pre>

<p>カイトは変換機の前に座り込み、目を閉じた。風のエネルギーが、彼の魔力回路に流れ込んでくる。それは、枯渇した大地に雨が降るような感覚だった。</p>

<blockquote>「カイト、少し聞いてもいいか？」</blockquote>

<p>リアムが、カイトの隣に腰を下ろした。</p>

<blockquote>「何だ？」</blockquote>

<blockquote>「あんた、ずっと気になってたことがあるんじゃないか？この世界が終わるかもしれないって、ずっと一人で抱え込んでたんじゃないかって」</blockquote>

<p>カイトは目を開けた。リアムの言葉は、彼の心の奥底にあった重荷を揺さぶった。</p>

<blockquote>「……分かってたのか」</blockquote>

<blockquote>「俺たちは戦士だ。仲間の不調くらい、感じ取れる」</blockquote>

<p>リアムは苦笑した。</p>

<blockquote>「あんたは、俺たちに迷惑をかけたくないって思ってる。だから、一番危険な役割を自分に割り当てる。でもな、カイト……俺たちは仲間だ。一人で全部背負う必要はないんだぜ」</blockquote>

<p>カイトは沈黙した。リアムの言う通りだった。彼は、プログラマーとしての習慣から、問題を一人で解決しようとする傾向があった。コードのバグは、一人でデバッグするのが最も効率的だと信じていた。だが、この世界での戦いは、一人では勝てない。</p>

<pre><code>
class Team {
    constructor(members) {
        this.members = members;
        this.trust = this.calculateTrust();
    }
    
    calculateTrust() {
        // 信頼度は、共に戦った回数と
        // 危機を乗り越えた回数に比例する
        return this.members.reduce((sum, m) => {
            return sum + m.sharedBattles * m.crisisOvercome;
        }, 0);
    }
    
    distributeLoad(task) {
        // 負荷を均等に分散する
        const loadPerMember = task.totalLoad / this.members.length;
        this.members.forEach(m => m.assignLoad(loadPerMember));
    }
}
</code></pre>

<p>カイトは、コードの例えを思い浮かべて苦笑した。彼の思考は、どうしてもプログラマー的なパターンに落ち着いてしまう。</p>

<blockquote>「ありがとう、リアム。次は、もっと頼らせてもらうよ」</blockquote>

<blockquote>「おう。それでいいんだよ」</blockquote>

<p>二人の会話を聞いていたエリナとセシリアも、自然と笑顔を浮かべていた。</p>

---

<p>六時間の充電を終えたカイトたちは、再び移動を開始した。風の谷を後にし、東の砂漠遺跡へと向かう。</p>

<p>砂漠遺跡は、かつて栄華を極めた古代文明の痕跡だった。だが現在は、灼熱の太陽と乾燥した風が支配する死の世界と化していた。</p>

<blockquote>「暑い……」</blockquote>

<p>エリナが扇風魔法を使いながら、不満を漏らした。</p>

<blockquote>「砂漠遺跡は、昼間の気温が六十度を超えるの。夜は逆に、零下まで下がるわ」</blockquote>

<p>セシリアの説明に、リアムが顔をしかめた。</p>

<blockquote>「過酷な場所だな。こんなところにノードがあるなんて、正気か？」</blockquote>

<p>カイトは砂漠の砂を踏みしめながら答えた。</p>

<blockquote>「ここは、かつて世界の魔力循環の中心だった。古代の人々が、太陽のエネルギーを魔力に変換する巨大な装置を建設したんだ」</blockquote>

<pre><code>
// 太陽魔力変換装置「ア・ソーラ」
// 建設時代：約一万年前
// 理論出力：500 MW
// 現在の状態：休止（エラー検出）
// エラー内容：コア過熱による緊急停止
</code></pre>

<p>カイトの管理者情報には、砂漠遺跡の地下に眠る「ア・ソーラ」の状態が表示されていた。この装置が、第二ノードとして機能している。</p>

<blockquote>「装置が停止しているから、魔力が滞っている。それが、ノードの不安定化を引き起こしているんだ」</blockquote>

<p>カイトの説明に、仲間たちは頷いた。</p>

<blockquote>「つまり、装置を再起動すればいいってことね？」</blockquote>

<p>エリナの確認に、カイトは首を横に振った。</p>

<blockquote>「簡単にはいかない。装置のコアが過熱している。その原因を取り除かないと、再起動してもまた停止する」</blockquote>

<p>カイトは砂漠の奥地を指差した。そこには、巨大なピラミッドのような構造物が、砂に埋もれながらもその威容を保っていた。</p>

<blockquote>「あそこが、ア・ソーラのある場所だ。行こう」</blockquote>

---

<p>ピラミッドの内部は、予想以上に複雑だった。狭い通路が迷路のように続き、至る所に古代の罠が仕掛けられていた。</p>

<blockquote>「火の罠だ！避けて！」</blockquote>

<p>セシリアの警告と同時に、壁から炎が噴き出した。カイトは瞬時に防御魔法を展開し、仲間たちを守った。</p>

<pre><code>
// 罠検出システム
// 種類：火炎噴射
// トリガー：重量感知式
// 対策：浮遊魔法または無力化
</code></pre>

<p>カイトは管理者権限を使って、罠の構造を解析した。古代の技術者たちは、侵入者を防ぐために、様々なギミックを組み込んでいた。</p>

<blockquote>「カイト、前を見て」</blockquote>

<p>リアムが指差した先には、広大な空間が広がっていた。天井は見えないほど高く、中央には巨大な水晶が浮かんでいた。</p>

<blockquote>「これが……ア・ソーラのコア」</blockquote>

<p>カイトは息を呑んだ。水晶の内部では、赤熱したエネルギーが渦巻いていた。それは、暴走したプロセスがCPUを占有するように、コア全体を支配していた。</p>

<pre><code>
// コアステータス：CRITICAL
// 温度：2,847°C
// 安定性：12%
// 推定爆発時間：四十二分
</code></pre>

<p>カイトの顔色が変わった。</p>

<blockquote>「爆発寸前だ。四十二分以内に、過熱の原因を取り除かないと……」</blockquote>

<blockquote>「原因は何なの？」</blockquote>

<p>エリナの問いに、カイトは水晶を凝視しながら答えた。</p>

<blockquote>「見てみろ。水晶の中に、何かがいる」</blockquote>

<p>仲間たちは水晶を覗き込んだ。確かに、赤熱したエネルギーの中心に、黒い影が揺らめいていた。</p>

<blockquote>「あれは……モンスター？」</blockquote>

<p>セシリアが呟いた。</p>

<blockquote>「いや、違う」</blockquote>

<p>カイトは首を振った。</p>

<blockquote>「あれは、かつてこの装置を管理していた古代の技術者の魂だ。装置の暴走を止めようとして、取り込まれたんだ」</blockquote>

<p>カイトは、管理者としての情報から、その魂の正体を特定した。</p>

<pre><code>
// 検出：魂の痕跡
// 名前：イクス・ソラリス
// 身元：ア・ソーラ主任技術者
// 状態：エネルギーに融合中
// 解放可能性：条件付きで可能
</code></pre>

<p>カイトは決意を固めた。</p>

<blockquote>「彼を解放する。それが、過熱を止める唯一の方法だ」</blockquote>

<blockquote>「どうやって？」</blockquote>

<p>リアムの問いに、カイトは答えた。</p>

<blockquote>「俺が、コアの中に入る」</blockquote>

<blockquote>「正気！？中に入ったら、あんたも蒸発するわよ！」</blockquote>

<p>エリナが叫んだ。だが、カイトは静かに首を振った。</p>

<blockquote>「管理者権限がある。コアの内部プログラムに介入して、彼を分離できるはずだ」</blockquote>

<p>カイトは仲間たちの顔を見渡した。</p>

<blockquote>「リアム、言ったよな。一人で全部背負う必要はないって。でも、これは俺にしかできない。管理者権限を持っているのは俺だけだから」</blockquote>

<p>リアムは一瞬、反論しようとしたが、言葉を飲み込んだ。彼は、カイトの決意が揺るぎないことを理解していた。</p>

<blockquote>「……分かった。だが、無理はするな。限界が来たら、即座に撤退しろ」</blockquote>

<blockquote>「約束する」</blockquote>

<p>カイトは水晶に向かって歩き出した。その背中は、仲間たちの祈りを受けて、少し大きく見えた。</p>

<pre><code>
// 管理者モード起動
// ターゲット：ア・ソーラ・コア
// ミッション：魂の分離
// 制限時間：四十二分
// 開始
</code></pre>

<p>カイトが水晶に触れた瞬間、世界が赤く染まった。灼熱の誓いが、今、試されようとしていた。</p>
