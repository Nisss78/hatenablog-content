<h3>第64章：世界のシフト</h3>

<p>夜明け前の薄暗い空の下、カイトは眠れずに窓辺に座っていた。ギルドが用意してくれた宿の一室から、静まり返った街並みを眺めている。</p>

<p>次元の境界線を修復してから三日が経っていた。表面上は何も変わらない日常が戻ってきたように見える。だが、カイトのプログラマーとしての感覚は、世界のどこかが決定的に変わったことを捉えていた。</p>

<blockquote>「コンパイルエラー……いや、構文エラーじゃない。もっと根本的な何かが違う」</blockquote>

<p>彼は空中に指を走らせ、目に見えないコードの流れを可視化した。世界を構成する基本的なパラメータ――重力、魔力濃度、時間の流れ、因果律の強度。それらが微妙に、しかし確実に変化していた。</p>

<pre><code>class WorldParameters {
    constructor() {
        this.gravity = 9.8;         // 従来値
        this.manaDensity = 1.0;     // 従来値
        this.timeFlow = 1.0;        // 従来値
        this.causalityStrength = 1.0; // 従来値
    }

    checkAnomalies() {
        const current = this.getCurrentValues();
        const deviations = [];
        
        if (Math.abs(current.gravity - this.gravity) > 0.01) {
            deviations.push("gravity fluctuation detected");
        }
        if (current.manaDensity > this.manaDensity * 1.15) {
            deviations.push("mana surge detected");
        }
        
        return deviations;
    }
}</code></pre>

<p>カイトの分析によれば、魔力密度が約18%上昇していた。時間の流れには0.3%の遅延が生じている。そして最も気になったのが、因果律の強度が低下していることだった。</p>

<blockquote>「因果律の弱体化……つまり、『ありえないこと』が『ありえること』になりやすくなっている。これは危険だ」</blockquote>

<p>ドアがノックされ、エリナが入ってきた。彼女も眠れなかったのか、軽い魔法の明かりを浮かべている。</p>

<blockquote>「カイト、起きてたの？」</blockquote>

<blockquote>「ああ。ちょっと調べたいことがあって」</blockquote>

<p>エリナはカイトの隣に座り、同じように窓の外を眺めた。</p>

<blockquote>「私も……ここ数日、何か変なの。魔法がいつもより強く感じるし、変な夢を見るし」</blockquote>

<p>カイトは興味を惹かれた。「変な夢？」</p>

<blockquote>「うん。知らない場所の夢。空に月が三つある世界とか、海が空に浮かんでいる世界とか。起きても覚えているくらい鮮明なの」</blockquote>

<p>カイトは立ち上がり、エリナに手を差し出した。</p>

<blockquote>「エリナ、ちょっと協力してくれないか？ 君の見た夢を、僕の分析用にデータ化したいんだ」</blockquote>

<blockquote>「データ化って……どうやるの？」</blockquote>

<blockquote>「精神感応の魔法を少し改造して、記憶の断片をコードとして抽出するんだ。怖くないよ、ただ見るだけだから」</blockquote>

<p>エリナは少し躊躇った後、頷いた。カイトの両手が彼女のこめかみに添えられる。柔和な光が二人を包んだ。</p>

<p>カイトの意識に、エリナの夢の映像が流れ込んでくる。三つの月が浮かぶ紫色の空。空中に浮かぶ水晶の宮殿。逆さまに流れる滝。そして、それらの場所に立っている見知らぬ人々の姿。</p>

<blockquote>「これ、単なる夢じゃない……他次元のビジョンだ」</blockquote>

<p>カイトは手を離し、眉をひそめた。</p>

<blockquote>「エリナ、君の魂が他の次元と共鳴し始めている。次元の境界線が修復された際、君の中にも何かが残ったのかもしれない」</blockquote>

<p>エリナは自分の胸に手を当てた。「私の中に……他の次元が？」</p>

<blockquote>「制御できれば強力な武器になる。でも、放っておくと君のアイデンティティが侵食されるかもしれない」</blockquote>

<p>その時、廊下から慌ただしい足音が近づいてきた。リアムがドアを開け放ち、息を切らして入ってくる。</p>

<blockquote>「おい、大変だ！ セシリアが倒れた！」</blockquote>

<p>カイトとエリナは顔を見合わせ、すぐに廊下へ飛び出した。</p>

<p>セシリアの部屋には、すでにマルクスとマグナスが集まっていた。セシリアはベッドに横たわり、顔色が悪い。だが、意識はあり、焦点の定まらない目で天井を見上げていた。</p>

<blockquote>「見える……見えます……」</blockquote>

<p>カイトが駆け寄った。「何が見えるんだ？」</p>

<p>セシリアの瞳が、異様な光を帯びていた。まるで、別の世界を映し出しているかのように。</p>

<blockquote>「世界が……重なっています。無数の世界が、この世界に押し寄せてくる。次元の境界線を修復したことで、世界間の『接続許可』が変更されたのです」</blockquote>

<p>カイトはすぐに理解した。次元の境界線を修復した際、彼が意図せず設定を変更してしまったのだ。</p>

<blockquote>「ファイアウォールを再設定したつもりが、ポートを開きっぱなしにしてしまったのか……」</blockquote>

<p>マグナスが腕を組み、深刻な表情で尋ねた。</p>

<blockquote>「状況を説明してくれ、カイト。我々一般人にもわかる言葉で」</blockquote>

<p>カイトは深呼吸をして、言葉を選んだ。</p>

<blockquote>「簡単に言えば、この世界が他の次元に対して『オープン』になったってことです。以前は厳格な認証が必要だった次元間通信が、今は緩い認証で通るようになっている」</blockquote>

<blockquote>「修正できるのか？」</blockquote>

<blockquote>「できますが、時間がかかります。それに、完全に元に戻すのは得策じゃないかもしれません」</blockquote>

<p>マルクスが口を挟んだ。「どういうことだ？」</p>

<blockquote>「今の状態は危険だけど、同時にチャンスでもあるんです。他次元との接続が緩くなったことで、僕たちは今まで以上に多くの情報やエネルギーにアクセスできる。これを制御できれば、この世界は大きく発展できる」</blockquote>

<p>リアムが呆れたように笑った。「お前、いつもそうだな。ピンチをチャンスに変えるのが得意だ」</p>

<blockquote>「プログラマーの性分さ。バグを見つけたら、それを機能に変える方法を考える」</blockquote>

<p>セシリアがゆっくりと起き上がった。まだ顔色は悪いが、目には力が戻っている。</p>

<blockquote>「カイトさん、私の見たビジョンを分析できますか？ 次元が重なる場所――そこが、今後の鍵になるはずです」</blockquote>

<blockquote>「もちろん。エリナの夢と合わせて、パターン分析をしてみるよ」</blockquote>

<p>カイトは二人のビジョンを統合し、コードとして解析を始めた。膨大なデータの中から、共通する座標、類似するイメージ、反復するシンボルを抽出していく。</p>

<p>数時間後、彼は一つの結論に到達した。</p>

<blockquote>「わかった。世界が重なるポイントは、全部で七つある。そのうち三つはこの大陸内、残りは海を越えた場所だ」</blockquote>

<p>マグナスが地図を広げた。「場所を示してくれ」</p>

<p>カイトは指で地図の上に点を打った。北の雪山、中央の森、南の火山地帯。そして、海図の上に四つの点――未知の島々、あるいは海そのもの。</p>

<blockquote>「これらの場所では、次元の境界が極端に薄くなっている。物理的な接触だけでなく、概念的な接触も起きている」</blockquote>

<blockquote>「概念的な接触？」エリナが首をかしげた。</p>

<blockquote>「例えば、北の雪山では『寒さ』という概念自体が別の次元のものに変わりつつある。ただ気温が下がるだけでなく、寒さの質が変わるんだ。ある種の『絶対零度』に近い、別次元の冷たさが混ざり始めている」</blockquote>

<p>マルクスが頷いた。「僕も収束者として、その感覚を覚えがある。次元が触れ合う場所では、法則そのものが歪むんだ」</p>

<blockquote>「正解だ。そして、その歪みを放置すれば、やがて世界全体のバランスが崩れる」</blockquote>

<p>マグナスが決断を下した。</p>

<blockquote>「では、この七つのポイントを調査し、必要なら対処する。ギルドとして正式なクエストにするぞ」</blockquote>

<p>カイトは首を振った。</p>

<blockquote>「いえ、これはギルドの仕事じゃありません。もっと個人的な、僕の責任です。次元の境界線を修復したのは僕ですから」</blockquote>

<blockquote>「バカ言うな」リアムがカイトの肩を叩いた。「お前一人で七つも回るつもりか？ 俺たちがいるだろうが」</blockquote>

<p>エリナも微笑んだ。</p>

<blockquote>「そうよ。一人で背負い込むのはプログラマーの悪い癖だね」</blockquote>

<p>セシリアが静かに立ち上がった。</p>

<blockquote>「私も行きます。私のビジョン能力は、次元の重なりを感知できるはずです」</blockquote>

<p>マルクスも決意を固めた表情で頷いた。</p>

<blockquote>「僕も力になります。収束者としての経験は、きっと役に立つ」</blockquote>

<p>カイトは仲間たちを見渡した。プログラマー、魔法使い、剣士、魔導書の継承者、そして元収束者。奇妙な組み合わせだが、確かな信頼関係があった。</p>

<blockquote>「ありがとう、みんな。――じゃあ、まずはどこから行く？」</blockquote>

<p>セシリアが地図の一点を指差した。</p>

<blockquote>「ここです。北の雪山。私のビジョンの中で、最も強い反応があった場所」</blockquote>

<p>カイトは頷いた。</p>

<blockquote>「了解。北の雪山、調査開始だ。出発は明日の朝。それまで各自準備を」</blockquote>

<p>夜が明け始めていた。窓の外で、空が白々と明るくなっていく。だが、その空の色にも微妙な違和感があった。いつもの朝の空色ではなく、どこか紫がかった、別の世界の色彩が混ざっている。</p>

<p>世界はシフトしていた。徐々に、しかし確実に。そして、カイトたちはその変化の最前線に立っていた。</p>

<blockquote>「さて、新しいバグとの戦いの始まりだ」</blockquote>

<p>カイトは小さく呟き、仲間たちと共に朝日を迎えた。</p>

<p>物語は、次のフェーズへと移ろう。次元を超える冒険の、新たな幕開けだった。</p>
