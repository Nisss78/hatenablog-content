<h3>第41章：王都潜入、そして影の支配者</h3>

<p>王都アルカディアの城壁は、夕陽を受けて黄金色に輝いていた。高さは五十メートルを超え、頂上には魔法の灯篭が並んでいる。千年の歴史を持つこの都は、まさに文明の結晶だった。</p>

<blockquote>「デカいな……」</blockquote>

<p>カイトは思わず呟いた。現代日本の高層ビルにも匹敵するスケールだ。プログラミングで例えるなら、これはスパゲッティコードではなく、設計図が完璧な大規模システムのような威容だ。</p>

<blockquote>「王国の心臓部ですからね。人口は十万人を超えると言われています」</blockquote>

<p>セシリアが説明した。魔導学院の教科書で習った知識だ。</p>

<blockquote>「十万人か。東京とは比べものにならないけど、この世界の基準なら大都市だね」</blockquote>

<p>城門には衛兵が立っており、入城する人々を検査している。カイトたちは一般の旅人に紛れることにした。</p>

<pre><code>// 潜入ミッション開始
// 
// Objective: 王都内部への侵入
// Method: 一般旅人に偽装
// Risk: 収束者の配下が警戒している可能性
// 
// Stealth.check() => 75%
// // 検査は厳しいが、通過可能</code></pre>

<p>四人は武器を目立たないように隠し、普通の冒険者パーティーのふりをして列に並んだ。</p>

<blockquote>「名前と、入城の目的を」</blockquote>

<p>衛兵が事務的な口調で尋ねた。</p>

<blockquote>「カイト。冒険者として、依頼をこなすために来ました」</blockquote>

<p>カイトは偽造した冒険者ギルドの証明書を提示した。これは、メモリの深淵を出る前にリアムが準備しておいたものだ。彼の偽造スキルは、魔導学院での研究中に培われたものだ。</p>

<blockquote>「よし、通れ」</blockquote>

<p>衛兵は簡単な確認だけで通してくれた。収束者の配下は、まだ王都内で公然と動いていないようだ。</p>

<h4>◆</h4>

<p>城門をくぐると、目の前に広がったのは活気あふれる大通りだった。石畳の道路には馬車が行き交い、両側には様々な店が並んでいる。武器屋、薬屋、食堂、そして魔道具を扱う店まで。</p>

<blockquote>「すごい……王都ってこんなに賑やかなんですね」</blockquote>

<p>セシリアが目を輝かせた。田舎の魔導学院から出たことのない彼女にとって、この光景は圧倒的だ。</p>

<blockquote>「観光してる暇はないよ。まずは情報を集めよう」</blockquote>

<p>エリナが冷静に言った。</p>

<blockquote>「地下遺跡への入り口を探すのが先決だ。そこに、召喚システムがあるはず」</blockquote>

<p>カイトは頷いた。しかし、問題がある。</p>

<blockquote>「地下遺跡は、王国の上層部しか入れない管理区域だろ？どうやって入るつもり？」</blockquote>

<blockquote>「裏口があるはずです」</blockquote>

<p>リアムが静かに言った。</p>

<blockquote>「ヴェリディアンの遺した記録によれば、地下遺跡には複数のアクセスポイントがあります。正規の入り口以外に、緊急時用の脱出口や、メンテナンス用の通路が」</blockquote>

<blockquote>「裏口か。プログラミングで言えば、バックドアみたいなものだね」</blockquote>

<p>カイトはその情報を分析した。</p>

<pre><code>// 地下遺跡へのアクセス計画
// 
// Method A: 正規ルート（王宮許可証が必要）
//   => 難易度：極高、リスク：高
// 
// Method B: 裏口（バックドア）
//   => 難易度：中、リスク：中
//   => 情報収集が必要
// 
// Method C: 強行突破
//   => 難易度：高、リスク：極高
//   => 全軍を敵に回す可能性
// 
// Selected: Method B</code></pre>

<blockquote>「裏口を探そう。僕の感覚で、おおよその位置は分かるはずだ」</blockquote>

<p>カイトは、メモリの深淵で手に入れた空間認識能力を信じていた。</p>

<h4>◆</h4>

<p>彼らはまず、酒場で情報を集めることにした。どこの世界でも、酒場は情報の宝庫だ。</p>

<p>「酔っ払いの竜」という名前の酒場に入ると、カウンターには大柄の男たちが酒を飲み交わし、テーブルでは賭け事が行われている。彼らは隅の席に座り、飲み物を頼んだ。</p>

<blockquote>「ねえ、あんたたち。旅の人だろ？」</blockquote>

<p>隣の席にいた男が話しかけてきた。四十代くらいの、特徴的な顔立ちをしている。元冒険者といった風情だ。</p>

<blockquote>「ええ、まあ」</blockquote>

<p>カイトが慎重に答えた。</p>

<blockquote>「俺はグレン。元冒険者で、今はこの街で情報屋をやってる。あんたたち、何か探してるんだろ？」</blockquote>

<p>カイトは警戒したが、同時に直感が告げた。この男は、利用できるかもしれない。</p>

<blockquote>「地下遺跡について、何か知ってる？」</blockquote>

<p>グレンの表情がわずかに変わった。</p>

<blockquote>「おっと、危ない話だな。地下遺跡は王宮の管理下だ。一般人は近づくことすら禁止されている」</blockquote>

<blockquote>「情報は買える？」</blockquote>

<p>エリナが金貨をテーブルに置いた。グレンはそれを一瞥し、ニヤリと笑った。</p>

<blockquote>「金払いがいいお客さんだ。いいだろう、教えてやる」</blockuote>

<p>グレンは声を潜めた。</p>

<blockquote>「三日前から、地下遺跡の警備が厳重になってる。表向きは『古代遺物のメンテナンス』ってことになってるが、実は違う」</blockquote>

<blockquote>「違うって？」</blockquote>

<blockquote>「何かが、起きているらしい。正確には分からないが、宮廷魔道士たちが慌てて出入りしてるって噂だ。それに———」</blockquote>

<p>グレンはさらに声を落とした。</p>

<blockquote>「『影の評議会』が動いているって話だ」</blockquote>

<p>カイトは眉をひそめた。</p>

<blockquote>「影の評議会？」</blockquote>

<blockquote>「表の政府とは別に、裏で王国を動かしてる組織だ。王族や貴族の上位層で構成されているらしいが、詳細は誰も知らない。ただ、一つだけ確かなことがある」</blockquote>

<p>グレンはカイトの目を真っ直ぐに見た。</p>

<blockquote>「お前さんたちが探してる『収束者』ってのは、その評議会のトップだって噂だ」</blockquote>

<p>カイトの心臓が跳ねた。収束者。その正体が、王国の裏の支配者なのか。</p>

<pre><code>// 重要情報取得
// 
// 1. 地下遺跡の警備が強化中
// 2. 影の評議会が存在
// 3. 収束者 = 影の評議会のトップ（推定）
// 
// Update: 収束者は王国内部の権力者
// 偽説：カイトの召喚は王国が仕組んだ</code></pre>

<h4>◆</h4>

<p>酒場を出て、彼らは路地裏で作戦会議を行った。</p>

<blockquote>「予想以上に複雑な状況ね」</blockquote>

<p>エリナが腕を組んだ。</p>

<blockquote>「王国が裏で糸を引いているとしたら、正面から入るのは自殺行為よ」</blockquote>

<blockquote>「でも、真実を知るには地下遺跡に入らないといけない」</blockquote>

<p>セシリアが言った。</p>

<blockquote>「ヴェリディアンの遺したシステムにアクセスすれば、カイトさんが召喚された理由が分かるはずです」</blockquote>

<blockquote>「裏口の場所は、もう少し探る必要があるな」</blockquote>

<p>リアムが地図を広げた。</p>

<blockquote>「王都の地下水路が、地下遺跡と繋がっている可能性があります。古代の設計では、排水システムと地下構造を一体化させるのが一般的でしたから」</blockquote>

<blockquote>「下水道か……ロマンチックじゃないけど、現実的なルートだね」</blockquote>

<p>カイトはため息をついた。</p>

<p>その時だった。カイトの空間認識が、異常を検知した。</p>

<pre><code>// WARNING: 敵性反応検知
// 
// 距離：50メートル
// 数：6体
// 配置：包囲陣形
// 
// // またしても囲まれた</code></pre>

<blockquote>「みんな、囲まれてる」</blockquote>

<p>カイトが警告した瞬間、路地裏の両端から人影が現れた。彼らは、村で遭遇した黒ローブたちとは違う。より洗練された装備、そして胸元には奇妙な紋章が刻まれている。</p>

<blockquote>「カイト。我々は『影の評議会』直属の執行部隊だ」</blockquote>

<p>リーダーらしき女が言った。フードを外したその顔は、美しくも冷徹だった。</p>

<blockquote>「収束者様がお呼びだ。大人しくついてきてもらおう」</blockquote>

<blockquote>「またそのセリフか。固定観念って知ってる？」</blockquote>

<p>カイトは軽口を叩きながら、コードを展開した。</p>

<blockquote>「コード・イグニッション」</blockquote>

<pre><code>// Combat Mode: ACTIVATED
// 
// 警告：敵の装備は高度
// 対策：空間操作で有利位置を確保
// 
// Spatial.rewrite({
//     area: currentAlley,
//     effect: "gravity_inversion",
//     targets: enemies,
//     duration: 5
// });</code></pre>

<p>カイトが指を鳴らすと、路地裏の空間が歪んだ。重力が局所的に反転し、執行部隊の足元が不安定になる。</p>

<blockquote>「なっ……！？」</blockquote>

<p>女が驚愕の声を上げた。彼らの動きが鈍った隙に、エリナが剣を抜いて突進する。</p>

<blockquote>「残念だったわね。私たちは『捕まって連れていかれる』タイプじゃないの」</blockquote>

<p>エリナの剣が、先頭の敵の武器を弾き飛ばした。</p>

<h4>◆</h4>

<p>戦闘は激化した。執行部隊は村の黒ローブたちとは段違いの実力を持っていた。彼らは連携し、魔法と剣術を組み合わせた高度な戦術を展開する。</p>

<p>リアムが火の精霊を召喚し、敵の視線を遮る。その隙にセシリアが防御魔法を展開し、味方を守る。エリナは前線で剣を振るい、敵の攻撃を逸らす。</p>

<p>そしてカイトは———空間そのものを武器として使った。</p>

<pre><code>// Spatial Manipulation: ADVANCED
// 
// 空間を折り曲げ、敵の攻撃を回避
// 距離概念を改変し、瞬間移動に近い動きを実現
// 
// while (combat.isActive()) {
//     let threat = detectNearestEnemy();
//     if (threat.distance < 2) {
//         Spatial.fold(threat.position, randomSafeLocation);
//     }
// }</code></pre>

<p>カイトの周りでは、物理法則が意味をなさなかった。敵の刃は彼に届く前に空間が歪み、軌道を変える。逆に、カイトの魔法は直線を飛ばず、空間の穴を通じて敵の背後に現れる。</p>

<blockquote>「これが……継承者の力……！」</blockquote>

<p>女が恐怖の声を上げた。</p>

<blockquote>「ヴェリディアンの遺産は、こんなにも強大だったのか！」</blockquote>

<blockquote>「まだ本番じゃないよ。これはただの———」</blockquote>

<p>カイトが言いかけた時、空気が変わった。</p>

<p>路地裏の奥から、一人の老人が現れた。白い髪、長い髭、そして穏やかながら威厳のある顔立ち。彼が歩くと、周囲の空間が凍りついたように静止した。</p>

<blockquote>「やめなさい」</blockquote>

<p>老人が言うと、戦闘が一瞬で止まった。執行部隊のメンバーたちは、跪いて頭を下げる。</p>

<blockquote>「収束者様……！」</blockquote>

<p>カイトは息を呑んだ。この老人が、自分を召喚した張本人なのか。</p>

<p>老人は微笑んだ。</p>

<blockquote>「初めまして、カイト君。いや、海藤翔太くんと言うべきかな」</blockquote>

<p>カイトの本名。日本での名前を知っている人物。</p>

<blockquote>「君をこの世界に呼んだのは、私だ」</blockquote>

<p>老人の瞳には、千年の時が宿っていた。</p>

<blockquote>「そして、君に頼みがある」</blockquote>

<h4>◆</h4>

<p>カイトは警戒を解かなかった。目の前の老人は、敵か味方か分からない。しかし、圧倒的な力の差は感じ取れた。</p>

<blockquote>「頼みって？」</blockquote>

<blockquote>「この世界を、救ってほしいのだよ」</blockquote>

<p>老人の言葉に、カイトは眉をひそめた。</p>

<blockquote>「綺麗事だね。具体的に言ってくれない？」</blockquote>

<blockquote>「ヴェリディアンが残したシステムには、欠陥がある。千年の間に、世界を滅ぼすバグが発生したのだ」</blockquote>

<p>老人は空を指差した。</p>

<blockquote>「十日後、世界は終わる。君がそれを止めない限り」</blockquote>

<p>カイトは老人の言葉を分析した。</p>

<pre><code>// 新情報
// 
// 1. 収束者はカイトに協力を求めている？
// 2. 世界の終了条件：十日後
// 3. ヴェリディアンのシステムにバグ？
// 
// Hypothesis:
// - 収束者は敵ではない可能性
// - 真の敵は「システムのバグ」
// - カイトはその修正のために召喚された
// 
// Confidence: 60%
// // 情報不足、裏取り必要</code></pre>

<blockquote>「バグって？」</blockquote>

<blockquote>「三人目の継承者が残したものだ。彼は世界を書き換えようとして失敗したが、その残骸がシステムの中に残っている。今、それが活性化しつつある」</blockquote>

<p>老人の表情に、悲しみがよぎった。</p>

<blockquote>「私は千年の間、この世界を守ってきた。だが、もう限界だ。君の力が必要なのだ」</blockquote>

<p>カイトは仲間たちを見た。エリナは剣を構えたまま警戒を解いていない。リアムは情報を整理しているようだ。セシリアは不安げにカイトを見ている。</p>

<blockquote>「……詳しく話を聞かせてくれ」</blockquote>

<p>カイトは選択した。敵か味方か、まだ分からない。でも、真実を知るには、この老人と対話するしかない。</p>

<blockquote>「いいだろう。ついてきなさい」</blockquote>

<p>老人は背を向け、歩き出した。</p>

<blockquote>「地下水路を通る必要はない。正面から、王宮へ案内しよう」</blockquote>

<p>カイトは一瞬躊躇したが、覚悟を決めて歩き出した。</p>

<p>物語は、新たな局面に入ろうとしていた。</p>
