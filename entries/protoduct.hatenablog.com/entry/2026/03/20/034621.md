<h3>第40章：王都への道、そして追跡者</h3>

<p>朝日が地平線から顔を覗かせ、草原を黄金色に染め上げていた。カイトたちは夜を徹して歩き続け、ようやく街道に辿り着いたところだった。メモリの深淵から出てから既に半日が過ぎている。</p>

<blockquote>「王都までは、徒歩で三日はかかるな」</blockquote>

<p>リアムが地図を広げながら言った。羊皮紙には、ヴェリディアンが残した道程が詳細に記されている。彼らの現在地は、森を抜けた先の小さな街道沿いの村から少し離れた場所だった。</p>

<blockquote>「馬車を雇えれば早いんだけど、この辺りには宿駅もなさそうね」</blockquote>

<p>エリナが周囲を見渡した。見渡す限りの草原と、遠くに霞む山々。人影どころか、家屋の煙すら見えない。</p>

<blockquote>「あそこの丘の向こうに、小さな集落があるようです」</blockquote>

<p>セシリアが指差した。確かに、微かに煙が立ち上っているのが見える。</p>

<blockquote>「よし、まずはそこで休憩と补给をしよう。このままでは、王都に着く前に体力が尽きる」</blockquote>

<p>カイトの提案に全員が頷いた。</p>

<h4>◆</h4>

<p>集落は、十数軒の家屋が集まっているだけの小さな村だった。村の入り口には錆びた看板が立っており、「穏やか村」という名前が読み取れた。</p>

<blockquote>「穏やか村、か。その名前に反したことが起きませんように」</blockquote>

<p>エリナが皮肉っぽく笑った。</p>

<p>村に入ると、数人の村人たちが彼らを好奇の目で見た。見知らぬ旅人四人組。しかも、魔道士と思われるローブを纏った者や、剣を腰に下げた女剣士がいる。物珍しいのは無理もない。</p>

<blockquote>「あそこに宿屋がありますね。看板が出ています」</blockquote>

<p>セシリアが指差した方向に、古びた二階建ての建物があった。「旅人の休息」という名前が、時間に洗われた看板に描かれている。</p>

<p>中に入ると、カウンターの奥から初老の女性が出てきた。白髪をきっちりとまとめ、エプロンを身につけた典型的な田舎の宿屋の女将という風情だ。</p>

<blockquote>「いらっしゃい。部屋は四つでいいのかい？」</blockquote>

<blockquote>「ええ、お願いします。あと、食事も頼めますか？」</blockquote>

<p>カイトが答えると、女将は頷いた。</p>

<blockquote>「もちろんだよ。ちょうど昼の準備をしていたところさ。三十分待ってくれれば、温かいシチューを用意するよ」</blockquote>

<p>部屋に荷物を置き、彼らは食堂で食事を待った。窓から差し込む陽光が、木製のテーブルに温かい光を落としている。</p>

<blockquote>「カイトさん、さっきから何か考えている顔ですけど、大丈夫ですか？」</blockquote>

<p>セシリアの問いに、カイトはハッとした。</p>

<blockquote>「ああ、ごめん。ヴェリディアンの言葉が気になっていてね。僕をここに呼んだのが誰なのか、本当に偶然じゃないっていうのが、ずっと頭から離れないんだ」</blockquote>

<blockquote>「召喚システム、でしたよね。それを操作できる人物なんて、限られているんじゃないですか？」</blockquote>

<p>リアムがお茶を啜りながら言った。</p>

<blockquote>「そうなんだよね。王都の地下にある古代遺跡。そこには、王国の上層部しか入れないはずだ。となると、僕を召喚したのは———」</blockquote>

<blockquote>「王室関係者、あるいはそれに準ずる権力者、ということになるわね」</blockquote>

<p>エリナが腕を組んだ。</p>

<blockquote>「だとしたら、王都に行くこと自体が罠かもしれない。わざわざ僕を呼んでおいて、放置するはずがない」</blockquote>

<blockquote>「でも、行くしかないんでしょう？」</blockquote>

<p>セシリアの言葉に、カイトは頷いた。</p>

<blockquote>「うん。真実を知るためには、危険を冒すしかない。それに———」</blockquote>

<p>カイトは仲間たちを見渡した。</p>

<blockquote>「一人じゃないっていうのが、何よりの心強さだよ」</blockquote>

<h4>◆</h4>

<p>食事を終え、彼らは出発の準備を整えた。女将に礼を言い、宿を出ようとした時だった。</p>

<p>カイトの背筋に、冷たいものが走った。何かが、彼の感覚に触れた。プログラミングで言えば、例外がスローされた瞬間のような感覚。</p>

<pre><code>// 警告：外部からの干渉を検知
// ターゲット：カイト（主人公）
// 種別：追跡・監視
// 
// Thread.currentThread().interrupt();
// // 何かが近づいている</code></pre>

<blockquote>「みんな、待って」</blockquote>

<p>カイトが声を潜めた。</p>

<blockquote>「何かが、こっちに来てる」</blockquote>

<p>エリナが即座に剣の柄に手をかけた。リアムは杖を握りしめ、セシリアは魔導書を開いた。彼らの連携は、もはや言葉を必要としない。</p>

<blockquote>「方向は？」</blockquote>

<blockquote>「村の入り口……いや、複数。三方向から包囲されている」</blockquote>

<p>カイトの感覚が、より鮮明に敵の位置を捉えていく。これは、メモリの深淵での経験が、彼の能力を向上させた結果だった。空間そのものをコードとして認識する力が、戦闘にも応用できるようになったのだ。</p>

<blockquote>「十二体……いや、増えてる。十五体」</blockquote>

<blockquote>「多勢に無勢ね。でも、逃げるのは得策じゃない。村の人たちを巻き込むことになる」</blockquote>

<p>エリナの判断は正しかった。村の入り口から、黒いローブを纏った集団が現れた。彼らは統一された動きで村へと侵入してくる。一般人を装うつもりはないようだ。</p>

<blockquote>「カイト、か？ヴェリディアンの継承者であるカイトを探している」</blockquote>

<p>先頭に立つ男が、不気味なほど平静な声で言った。顔はフードに隠れて見えないが、その声には感情の色がない。まるで、AIが事務的なアナウンスをしているかのように。</p>

<blockquote>「その名前で呼ばれるのは初めてじゃないけど、呼び出し主には心当たりがないんだよね」</blockquote>

<p>カイトが前に出た。</p>

<blockquote>「君たちは誰？僕を召喚したのは君たちなのか？」</blockquote>

<blockquote>「我々は『収束者』の配下だ。お前をここに呼んだのは我々の主。そして、主はこう仰っている——『継承者を連れてこい、さもなくば、消せ』とな」</blockquote>

<p>カイトの心臓が早鐘を打った。収束者。その名前は、ヴェリディアンの遺言には出てこなかった。しかし、その響きには不穏なものがあった。</p>

<blockquote>「その『収束者』ってのは、誰？」</blockquote>

<blockquote>「お前が会うべき存在だ。主は王都でお待ちになっている。大人しくついてきてもらおう」</blockquote>

<p>男が手を上げると、残りの黒ローブたちが一斉に武器を抜いた。剣、槍、そして魔杖。彼らはただの傭兵ではない。訓練された戦闘集団だ。</p>

<blockquote>「断ったら？」</blockquote>

<p>エリナが挑発的に問うた。</p>

<blockquote>「ならば、力尽くで連れていくまでだ。抵抗は無意味だが、命まで奪うつもりはない。主は『生きたまま』連れてこいと」</blockquote>

<blockquote>「残念ね。私は『死んだまま』連れていかれるタイプなの」</blockquote>

<p>エリナの軽口に、カイトは思わず苦笑した。</p>

<h4>◆</h4>

<p>戦闘は、村の広場で始まった。</p>

<p>黒ローブたちが一斉に攻撃を仕掛けてくる。エリナが前線に出て、剣で彼らの攻撃を受け流した。彼女の剣技は、もはや神業の領域に達しつつあった。一人の敵の剣を弾き、返す刀で二人目を斬り伏せる。</p>

<p>リアムは後方から精霊術を展開した。火の精霊が敵の進路を焼き払い、風の精霊が矢を逸らす。彼の支援がなければ、前線のエリナも無傷ではいられなかっただろう。</p>

<p>セシリアは防御魔法を展開し、村人たちを守っていた。彼女はまだ実戦経験が浅いが、魔導学院で培った知識は確かだ。障壁を張り、巻き込みを最小限に抑える。</p>

<p>そして、カイト———。</p>

<blockquote>「コード・イグニッション」</blockquote>

<p>カイトが呟くと、空中に魔法陣が展開された。しかし、今回はメモリの深淵で使ったような大規模なものではない。より小型で、より効率的な陣形だ。</p>

<pre><code>// 効率化された戦闘コード
// 大規模魔法は不要、最小限のリソースで制圧

    rapidFire: (targets) => {

<p>五人の黒ローブが、突然動きを止めた。カイトのスタン魔法が、彼らの神経系に干渉したのだ。</p>

<blockquote>「なっ……！何をした！？」</blockquote>

<p>先頭の男が驚愕の声を上げた。</p>

<blockquote>「体の制御を一時的に遮断しただけだ。五秒で復帰するけど、その間に———」</blockquote>

<p>カイトが指を鳴らすと、エリナが間髪入れずに動いた。止まった五人を剣で無力化する。無駄のない動きだった。</p>

<blockquote>「———戦況は変わる」</blockquote>

<p>カイトが言葉を継いだ。</p>

<p>残りの黒ローブたちが動揺した。彼らはカイトの能力を過小評価していたのだ。ヴェリディアンの継承者、その言葉の意味を真に理解していなかった。</p>

<blockquote>「引くぞ！今の戦力では勝てない！」</blockquote>

<p>男が叫び、残った仲間たちが撤退を始めた。彼らは統率の取れた集団だ。敗北を認めると、即座に退却の行動に出る。</p>

<p>カイトは追撃を考えたが、首を横に振った。</p>

<blockquote>「追わないでいい。彼らの背後には、もっと大きな組織がある。今は情報が足りない」</blockquote>

<p>エリナも剣を収めた。</p>

<blockquote>「賢明ね。それに、村の人たちが怖がってる。これ以上騒ぎを大きくすべきじゃない」</blockquote>

<h4>◆</h4>

<p>黒ローブたちが去った後、カイトたちは村長の家に招かれた。村長は老齢の男性で、震える手でお茶を淹れてくれた。</p>

<blockquote>「すみません、私たちのせいで村に迷惑をかけてしまって」</blockquote>

<p>カイトが頭を下げると、村長は穏やかに首を横に振った。</p>

<blockquote>「いいんじゃよ、いいんじゃよ。お前さんたちがいなければ、あの連中はもっと乱暴に村を探っていたじゃろうからな」</blockquote>

<blockquote>「探ってた？」</blockquote>

<blockquote>「ああ、お前さんたちが来る前から、あの黒い服の連中は村を調べておった。何かを探している様子じゃったが……まさか、お前さんたちだったとは」</blockquote>

<p>カイトは眉をひそめた。彼らが村に到着する前から、追跡者たちは村を調べていたのか？</p>

<blockquote>「どのくらい前からですか？」</blockquote>

<blockquote>「そうじゃな、三日くらい前からかのう。何度か村を訪れては、行ったり来たりしておった」</blockquote>

<p>三日間。カイトたちは三日間、メモリの深淵の中にいたことになる。そして、その間ずっと、追跡者たちは彼らを探していたのだ。</p>

<blockquote>「待ち伏せされてたってことか……」</blockquote>

<p>リアムが呟いた。</p>

<blockquote>「組織的に動いている証拠ですね。メモリの深淵の出口を予測して、周辺地域を網羅的に捜索していた」</blockquote>

<p>カイトは手帳に情報を整理した。</p>

<pre><code>// 敵性勢力メモ

<blockquote>「収束者……誰だろうな」</blockquote>

<p>カイトが呟いた。</p>

<blockquote>「ヴェリディアンの時代にはいなかった存在だ。千年の間に生まれた新たな勢力、あるいは———」</blockquote>

<blockquote>「あるいは？」</blockquote>

<p>エリナが促した。</p>

<blockquote>「三人目の継承者の生き残りか、その追随者たちかもしれない。ヴェリディアンは言っていた。三人目の継承者は、世界を書き換えようとして失敗したって。でも、完全に消滅したとは限らない」</blockquote>

<p>カイトの言葉に、全員が沈黙した。</p>

<blockquote>「だとしたら……カイトさんは、三人目の継承者と戦うことになるかもしれませんね」</blockquote>

<p>セシリアが不安げに言った。</p>

<blockquote>「そうなるかもしれない。でも、僕は一人じゃない」</blockquote>

<p>カイトは仲間たちを見渡した。</p>

<blockquote>「四人の凡人が、一人の天才に勝つ確率は？ヴェリディアンは、そう言ったんだ。なら、証明してみせよう」</blockquote>

<h4>◆</h4>

<p>村を出て、彼らは再び王都への道を歩き始めた。追跡者の存在は分かったが、それは同時に、王都に行けば答えがあることも意味していた。</p>

<p>夕暮れが近づき、空が茜色に染まっていく。遠くに、城壁のシルエットが微かに見え始めた。</p>

<blockquote>「王都が見えてきたな」</blockquote>

<p>リアムが指差した。</p>

<blockquote>「あと一日も歩けば、到着できるでしょう」</blockquote>

<p>カイトは王都の方角を見つめた。その地下に眠る、召喚システムの秘密。そして、自分をこの世界に呼んだ「収束者」の正体。</p>

<p>真実は、もうすぐそこにある。</p>

<blockquote>「覚悟はできてるか？」</blockquote>

<p>エリナがカイトに問いかけた。</p>

<blockquote>「ああ。僕を呼んだのが誰なのか、そして僕がこの世界で何をすべきなのか———全てを確かめる」</blockquote>

<p>カイトの瞳に、夕陽が反射して輝いた。</p>

<p>旅は、いよいよクライマックスへと近づいていく。王都の門をくぐれば、もう後戻りはできない。カイトは覚悟を決め、仲間たちと共に、最後の道程を歩き出した。</p>
