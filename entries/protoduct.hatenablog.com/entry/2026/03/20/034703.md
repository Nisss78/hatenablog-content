<h3>第47章：帰還と新たな兆し</h3>

<p>王都への帰路、四人はほとんど言葉を交わさなかった。</p>

<p>戦いの疲れもあった。だがそれ以上に、深淵の遺跡で目にしたもの、感じたものが、それぞれの心に沈殿していたからだ。</p>

<p>カイトは馬の背で揺られながら、空を見上げていた。雲が流れていく。千年の間、マルクスはあの暗闇の中で何を思っていたのだろう。仲間を救いたいという一心で、理性を保ち続けた執念。それは、プログラマーとしてのカイトにも理解できるものだった。</p>

<blockquote>「バグを直すためには、原因を理解しなきゃいけない」</blockquote>

<p>カイトは独り言のように呟いた。エリナが隣で振り返る。</p>

<blockquote>「何か言った？」</blockquote>

<blockquote>「いや、独り言だ」</blockquote>

<p>エリナは少し笑って、また前を向いた。彼女の背中が、いつもより少し小さく見えた。</p>

<h4>◆</h4>

<p>王都の城門が見えたとき、リアムが大きく息を吐いた。</p>

<blockquote>「生きて戻ってこれたな」</blockquote>

<blockquote>「最初から戻るつもりだったよ」</blockquote>

<blockquote>「俺は半分諦めてた」</blockquote>

<p>セシリアが首を傾げた。</p>

<blockquote>「リアムさん、いつもネガティブですね」</blockquote>

<blockquote>「現実主義者だ」</blockquote>

<p>軽口を叩き合う二人を見て、カイトは少し安心した。変わらない日常。それが、今は何より心地よかった。</p>

<p>城門の衛兵が四人を認め、敬礼した。</p>

<blockquote>「お帰りなさいませ！ギルドマスターがお待ちです！」</blockquote>

<blockquote>「もう知ってるのか」</blockquote>

<blockquote>「魔法通信が届いております」</blockquote>

<p>カイトは眉を上げた。セシリアが説明する。</p>

<blockquote>「遺跡の魔力解放は、かなり遠くまで感知されたはずです。王都の魔術師ギルドも気づいているでしょう」</blockquote>

<blockquote>「なるほど」</blockquote>

<h4>◆</h4>

<p>ギルド本部に入ると、マグナスが待ち構えていた。大柄な体躯を椅子に預け、深刻そうな表情で四人を出迎える。</p>

<blockquote>「無事だったか」</blockquote>

<blockquote>「なんとかな」</blockquote>

<p>エリナが答えた。マグナスは頷き、それからカイトをじっと見た。</p>

<blockquote>「報告は聞いてる。収束者を……倒したのではなく、解放したと？」</blockquote>

<blockquote>「正しくは救出です」</blockquote>

<p>カイトは遺跡での出来事を簡潔に説明した。ヴェリディアンの記憶、マルクスの正体、そして救済の術式について。マグナスは時折頷きながら耳を傾けた。</p>

<blockquote>「古代の賢者が、千年もの間あそこに……」</blockquote>

<blockquote>「ええ。使命を全うしようとして」</blockquote>

<p>マグナスは重いため息をついた。</p>

<blockquote>「我々は収束者を『脅威』として扱ってきた。だが、真実はそう単純じゃなかったわけか」</blockquote>

<blockquote>「真実はいつも複雑です」</blockquote>

<p>セシリアが静かに言った。マグナスは苦笑した。</p>

<blockquote>「若いのに達観してるな」</blockquote>

<h4>◆</h4>

<p>マグナスは書類を取り出した。</p>

<blockquote>「収束者の一件は、公式には『脅威の排除』として記録される。だが、真実はギルドの極秘資料として保管する」</blockquote>

<blockquote>「なぜ公式には？」</blockquote>

<p>リアムが尋ねた。マグナスは肩をすくめた。</p>

<blockquote>「一般人に説明するのが難しいんだよ。『実は収束者は千年の間、世界を守ろうとしてた犠牲者でした』って言ったら、混乱するだろ？」</blockquote>

<blockquote>「確かに」</blockquote>

<blockquote>「だが、君たちの功績は正当に評価される。報酬も弾む」</blockquote>

<p>マグナスは封筒を四つ、机の上に置いた。重みのある音がした。</p>

<blockquote>「それと……カイト」</blockquote>

<blockquote>「はい」</blockquote>

<blockquote>「お前に会いたがってる人物がいる」</blockquote>

<p>カイトは首を傾げた。</p>

<blockquote>「誰です？」</blockquote>

<blockquote>「王立魔術研究院の使者だ」</blockquote>

<h4>◆</h4>

<p>応接室に通されると、一人の女性が待っていた。</p>

<p>年齢は三十代後半。銀縁の眼鏡をかけ、真っ白なローブを身に纏っている。腰には複数の魔法道具がぶら下がり、全体的に学者というよりは、実務家という雰囲気だった。</p>

<blockquote>「初めまして。王立魔術研究院のソフィア・レインズです」</blockquote>

<p>彼女は立ち上がり、丁寧に頭を下げた。</p>

<blockquote>「海藤翔太、通称カイトさんですね」</blockquote>

<blockquote>「そうですけど……」</blockquote>

<blockquote>「あなたの魔法について、詳しくお話を伺いたいのです」</blockquote>

<p>カイトは警戒心を抱いた。彼女の目は、好奇心と探究心に満ちていたが、敵意は感じられなかった。だが、その分、質問は鋭いだろう。</p>

<blockquote>「具体的には？」</blockquote>

<blockquote>「遺跡での戦い、あなたが使った魔法は従来の体系に当てはまりません。新しい魔法理論の可能性があります」</blockquote>

<p>カイトは内心で苦笑した。彼の魔法は理論も何もない。異世界から持ち込んだ「プログラミング」という概念を、この世界の魔力で再現しているだけだ。説明したら理解されるだろうか？</p>

<pre><code>// 説明の難易度：高
// 
// const explanationDifficulty = {
//     programming: "異世界の概念",
//     magic: "この世界の常識",
//     synthesis: "カイトだけの能力",

// };
// 
// // どう答えるか</code></pre>

<blockquote>「少しお時間をいただけますか？順を追って説明します」</blockquote>

<h4>◆</h4>

<p>カイトは、自分の魔法について話し始めた。</p>

<p>すべてを話すわけにはいかなかった。異世界から来たという事実は伏せ、単に「独自の魔法理論を開発した」という形に落とし込んだ。</p>

<blockquote>「私の魔法は、『条件分岐』と『処理の定義』を基本としています」</blockquote>

<p>ソフィアの目が輝いた。</p>

<blockquote>「条件分岐……つまり、状況に応じて異なる効果を発揮するということですか？」</blockquote>

<blockquote>「正確には、状況を判定し、最適な処理を実行する、ということです」</blockquote>

<p>カイトは机の上の水差しを指差した。</p>

<blockquote>「例えば、この水差し。通常の魔法なら『水を出す』とか『凍らせる』とか、単一の効果を指定しますよね」</blockquote>

<blockquote>「ええ」</blockquote>

<blockquote>「私のやり方なら、こう書けます。『温度が〇度以下なら凍らせる。それ以外なら沸騰させる』と」</blockquote>

<pre><code>// Conditional Magic Example
// 
// function adjustWater(target) {
//     if (target.temperature <= 0) {
//         return target.freeze();
//     } else {
//         return target.boil();
//     }
// }
// 
// // 一つの魔法で複数の結果を</code></pre>

<p>ソフィアはメモを取りながら頷いた。</p>

<blockquote>「なるほど……従来の魔法は『何を』に焦点を当てていましたが、あなたの魔法は『どう』と『いつ』も含んでいる」</blockquote>

<blockquote>「そう捉えてもらえば近いです」</blockquote>

<h4>◆</h4>

<p>一時間ほどの質疑応答の後、ソフィアは満足げに頷いた。</p>

<blockquote>「非常に興味深いです。あなたの魔法理論は、既存の体系を覆す可能性があります」</blockquote>

<blockquote>「そこまで大げさなものじゃないですよ」</blockquote>

<blockquote>「いいえ。魔力効率、柔軟性、どれをとっても革新的です」</blockquote>

<p>彼女は名刺を差し出した。</p>

<blockquote>「もしよろしければ、研究院で講義をしていただけませんか？」</blockquote>

<blockquote>「講義？」</blockquote>

<blockquote>「次世代の魔術師たちに、あなたの理論を教えてほしいのです」</blockquote>

<p>カイトは少し考えた。この世界で生きていくなら、自分の能力を正当に評価される場所があってもいい。だが、まだ旅の途中だ。仲間もいる。</p>

<blockquote>「今はまだ、旅を続けたいんです」</blockquote>

<blockquote>「わかりました。いつでも歓迎します」</blockquote>

<p>ソフィアは名刺をテーブルに置いた。</p>

<blockquote>「気が変わったら、いつでも連絡してください」</blockquote>

<h4>◆</h4>

<p>応接室を出ると、エリナが待っていた。</p>

<blockquote>「終わった？」</blockquote>

<blockquote>「ああ」</blockquote>

<blockquote>「何の話だったの？」</blockquote>

<blockquote>「俺の魔法について。研究院が興味を持ったらしい」</blockquote>

<p>エリナはふーんと呟き、それから少し複雑な表情をした。</p>

<blockquote>「カイトの魔法、確かに特殊だもんね」</blockquote>

<blockquote>「エリナ？」</blockquote>

<blockquote>「何でもない。行きましょ、宿に」</blockquote>

<p>彼女は先に歩き出した。カイトはその背中を見つめながら、首を傾げた。</p>

<h4>◆</h4>

<p>宿に戻ると、リアムとセシリアが酒場で待っていた。</p>

<blockquote>「遅かったな」</blockquote>

<blockquote>「研究院の人と話してた」</blockquote>

<p>カイトは席に着いた。テーブルの上には料理と酒が並んでいる。</p>

<blockquote>「乾杯しようぜ。生きて帰ってきた記念に」</blockquote>

<p>リアムがジョッキを掲げた。四人は杯を合わせた。</p>

<blockquote>「乾杯！」</blockquote>

<p>温かい料理と冷たい酒。日常の幸せが、戦いの後には何よりも輝いて見えた。</p>

<p>セシリアがカイトを見た。</p>

<blockquote>「カイトさん、これからどうするんですか？」</blockquote>

<blockquote>「どうするって？」</blockquote>

<blockquote>「研究院の話、受けないんですか？」</blockquote>

<p>カイトは少し考えた。確かに、安定した生活も悪くない。だが、まだやりたいことがあった。</p>

<blockquote>「元の世界に帰る方法、まだ見つかってないからな」</blockquote>

<blockquote>「あ……」</blockquote>

<p>セシリアは黙った。この世界に来た理由。それはカイトだけの課題だった。</p>

<p>エリナが静かに言った。</p>

<blockquote>「私たちも手伝うわ」</blockquote>

<blockquote>「エリナ……」</blockquote>

<blockquote>「仲間でしょ？」</blockquote>

<p>リアムも頷いた。</p>

<blockquote>「面倒なことになりそうだが、放っておけないな」</blockquote>

<blockquote>「リアム……」</blockquote>

<blockquote>「ただし、報酬は別途請求するぞ」</blockquote>

<blockquote>「そこは譲らないのか」</blockquote>

<p>四人は笑った。未来はまだ見えない。だが、仲間がいる。それが何よりの心強さだった。</p>

<h4>◆</h4>

<p>その夜、カイトは宿の窓から夜空を見上げていた。</p>

<p>星空は故郷のものとは違う。だが、美しさは変わらない。</p>

<p>カイトは拳を握った。まだ終われない。帰らなきゃいけない場所がある。会いたい人がいる。</p>

<blockquote>「必ず帰る」</blockquote>

<p>その誓いは、誰に聞かせるものでもなかった。自分自身への約束。</p>

<p>カイトの異世界での冒険は、まだ終わらない。</p>
