<h3>第67章：静寂の向こう側</h3>

<p>虚空から現れた「セキュリティ」は、カイトの想像を遥かに超える存在だった。</p>

<p>形を持たない。輪郭を持たない。ただ、圧倒的な「否定」の意志だけが、空間全体を満たしていた。カイトがパッチを当てた世界の基盤コードが、軋みを上げて振動する。</p>

<blockquote>「な、なんだこれは……！」</blockquote>

<p>リアムが剣を構えようとしたが、その刀身が見えない力に蝕まれていく。金属が錆びるように、魔力が染み出していく感覚。リアムは慌てて剣を下ろした。</p>

<blockquote>「攻撃が効かない？いや、攻撃そのものが消滅してる……」</blockquote>

<p>エリナが光の弓を構えるが、同様だった。矢を放とうとするたびに、その魔力が霧散していく。まるで、存在そのものが否定されているかのように。</p>

<blockquote>「セキュリティ……」</blockquote>

<p>カイトは、虚空の中に立ち尽くしていた。彼のプログラマーとしての感覚が、目の前の現象を解析しようとしていた。</p>

<pre><code>
class Security {
    constructor() {
        this.directive = "PRESERVE_WORLD_INTEGRITY";
        this.method = "NULLIFY_THREATS";
        this.scope = "GLOBAL";
    }

    execute(target) {
        // Threats are not destroyed
        // They are simply "undefined"
        return null;
    }
}
</code></pre>

<p>カイトの背筋に寒気が走った。このセキュリティは、脅威を「破壊」するのではない。「無効化」するのだ。コードにおいて、変数を<code>null</code>に設定するように、存在そのものを未定義にする。</p>

<blockquote>「マルクス、これは――」</blockquote>

<p>カイトが振り返ると、マルクスは膝をついていた。収束者の顔には、絶望が張り付いていた。</p>

<blockquote>「知らない……わたしは、知らなかった」</blockquote>

<p>マルクスの声が震えていた。</p>

<blockquote>「創造主たちが残したセキュリティが、これほどのものだとは。伝承には『最後の砦』とだけ記されていた。詳細は、何百年もの間に失われていたのだ」</blockquote>

<p>セシリアが、カイトの袖を掴んだ。その手は小刻みに震えていた。</p>

<blockquote>「ねえ、カイト。これ、どうなるの？私たち、消されちゃうの？」</blockquote>

<p>カイトはセシリアの手を優しく握り返した。彼の掌から、微かな温かさが伝わる。</p>

<blockquote>「大丈夫だ。俺がなんとかする」</blockquote>

<p>そう言いながらも、カイトの頭の中では必死に計算が回っていた。このセキュリティに対抗するには、どうすればいいのか。</p>

<ul>
<li>攻撃は無効化される</li>
<li>魔力は霧散する</li>
<li>存在そのものが「否定」される</li>
</ul>

<p>通常の手段では、何一つ通じない。カイトは深呼吸をした。プログラマーとしての彼の経験が、一つの可能性を示唆していた。</p>

<blockquote>「セキュリティは、脅威を判定して動作する」</blockquote>

<p>カイトが呟く。エリナが、彼の顔を見た。</p>

<blockquote>「どういうこと？」</blockquote>

<p>カイトは虚空を見上げた。そこには、まだ何の形もない。だが、確かに何かが存在している。</p>

<blockquote>「つまり、脅威と判定されなければ、攻撃されないってことだ」</blockquote>

<pre><code>
if (target.isThreat()) {
    security.nullify(target);
} else {
    // No action
}
</code></pre>

<p>カイトは、自分の魔力回路を静かに観察した。パッチを当てた際に、彼の中に流れ込んできた膨大な記憶。先人たちの知識。それらが、彼の内側で静かに息づいている。</p>

<blockquote>「俺は、この世界を壊すつもりはない」</blockquote>

<p>カイトが、虚空に向かって語りかけた。</p>

<blockquote>「俺がしたかったのは、この世界をより良くすることだ。真実を知っても崩壊しない、強い世界にすることだ」</blockquote>

<p>虚空が、わずかに揺らいだ。カイトの言葉が、何かに届いたかもしれない。</p>

<blockquote>「だから、俺たちは敵じゃない。味方だ。この世界を守るという目的において、セキュリティと俺たちは同じはずだ」</blockquote>

<p>カイトは、自分の右手をかざした。掌に、青白い光が集まる。だが、それは攻撃のための魔力ではない。理解を伝えるための、静かな波動だった。</p>

<pre><code>
function communicate(intent) {
    let signal = new ManaSignal();
    signal.encode(intent);
    signal.broadcast();
    return signal.resonance;
}
</code></pre>

<p>カイトの魔力が、空中に広がっていく。それは、言葉以上の情報を伝えていた。カイトの真意。彼の望み。世界を救いたいという、純粋な意志。</p>

<blockquote>「カイト、何をしているの？」</blockquote>

<p>エリナが、不安げに問うた。</p>

<blockquote>「会話してるんだ。セキュリティと」</blockquote>

<p>エリナは目を見開いた。</p>

<blockquote>「会話？そんなの、可能なの？」</blockquote>

<p>カイトは薄く笑った。</p>

<blockquote>「プログラムは、入力を受け付ける限り、通信できる。セキュリティだって、例外じゃないはずだ」</blockquote>

<p>虚空が、再び揺らいだ。今度は、よりはっきりと。見えない何かが、カイトの信号に反応している。</p>

<hr>

<p>カイトの意識が、深く沈んでいく。</p>

<p>彼は、世界のコードの更深いレイヤーにアクセスしていた。パッチを当てた際に垣間見た、創造主たちの領域。そこには、セキュリティとの対話インターフェースが存在するはずだった。</p>

<pre><code>
// Ancient Interface
class SecurityInterface {
    authenticate(user) {
        return user.hasPureIntent();
    }

    listen(message) {
        if (this.authenticate(message.sender)) {
            return this.evaluate(message.content);
        }
        return "ACCESS_DENIED";
    }
}
</code></pre>

<p>カイトは、自分の意識を集中させた。純粋な意志。世界を救いたいという、偽りのない願い。それを、セキュリティに伝える。</p>

<blockquote>『――検知』</blockquote>

<p>声がした。いや、声ではない。意識に直接響く、純粋な情報の波。</p>

<blockquote>『異物を検知。脅威判定を実施中……』</blockquote>

<p>カイトは、その「声」に向かって語りかけた。</p>

<blockquote>『俺は脅威じゃない。この世界の住人だ。そして、この世界を守りたいと思っている』</blockquote>

<p>しばらくの沈黙。そして――</p>

<blockquote>『照合開始。意図パターンを解析……』</blockquote>

<p>カイトの内側で、何かがスキャンされている感覚。彼の記憶、感情、望み。すべてが、透かされるように解析されていく。</p>

<blockquote>『解析完了。判定結果――』</blockquote>

<p>カイトは息を止めた。</p>

<blockquote>『――「保留」』</blockquote>

<p>保留？カイトは眉をひそめた。</p>

<blockquote>『保留とはどういうことだ』</blockquote>

<p>セキュリティの声は、機械的だった。</p>

<blockquote>『創造主たちの定義において、脅威か否かの判定には「コンセンサス」が必要。単一の意図パターンでは、最終判定を下せない』</blockquote>

<p>カイトは、その意味を理解した。このセキュリティは、一人の意志だけで動くものではない。世界の住人たちの「合意」が必要なのだ。</p>

<blockquote>『つまり、どうすればいい』</blockquote>

<blockquote>『世界の住民の過半数が、君を「脅威ではない」と認めれば、セキュリティは停止する。逆に、過半数が「脅威」と判定すれば――』</blockquote>

<p>セキュリティは一瞬、言葉を切った。</p>

<blockquote>『――完全消去が実行される』</blockquote>

<p>カイトは息を呑んだ。完全消去。つまり、彼だけではない。彼に関わったすべての存在が、世界から抹消されるということだ。</p>

<hr>

<p>意識が、現実に戻る。カイトは大きく息を吐いた。</p>

<blockquote>「カイト！大丈夫！？」</blockquote>

<p>セシリアが、心配そうに彼の顔を覗き込んでいた。</p>

<blockquote>「ああ、大丈夫だ。ただ……少し複雑な状況になった」</blockquote>

<p>カイトは、仲間たちに説明した。セキュリティとの対話。コンセンサスの必要性。そして、失敗した場合の代償。</p>

<blockquote>「世界の住人の過半数が、俺たちを味方だと認めなきゃいけない……」</blockquote>

<p>リアムが、腕を組んだ。</p>

<blockquote>「無茶だな。俺たちは、世界の真実を知っている。それを公表すれば、パニックになる。かといって、隠したまま信頼を得るのは難しい」</blockquote>

<p>エリナが、静かに頷いた。</p>

<blockquote>「それに、私たちがこれまで戦ってきた相手たち。彼らが、私たちを『脅威ではない』と認めるかしら」</blockquote>

<p>カイトは、マルクスを見た。収束者は、まだ膝をついたままだった。</p>

<blockquote>「マルクス。お前は、何百年もの間、真実を守り続けてきた。世界の住人たちのことを、誰よりもよく知っているはずだ」</blockquote>

<p>マルクスは、ゆっくりと顔を上げた。</p>

<blockquote>「……わたしは、彼らを守ってきた。だが、理解してきたつもりはない」</blockquote>

<p>マルクスの声には、自嘲が混じっていた。</p>

<blockquote>「わたしの役割は、真実を知った者を統合することだった。彼らと対話し、信頼を築くことではなかった」</blockquote>

<p>カイトは、マルクスの前に立った。</p>

<blockquote>「これからは違う。お前はもう、孤独な収束者じゃない。俺たちの仲間だ」</blockquote>

<p>カイトは、マルクスに手を差し出した。</p>

<blockquote>「一緒にやろう。世界の住人たちに、真実を伝えるんだ。そして、選ばせる。この世界をどうしたいのかを」</blockquote>

<p>マルクスは、カイトの手を見つめた。そして、ゆっくりと、その手を握った。</p>

<blockquote>「……狂っているな、君は」</blockquote>

<p>マルクスの声に、微かな笑いが混じっていた。</p>

<blockquote>「真実を公表すれば、世界の安定性は崩れるかもしれない。それでも、するのか」</blockquote>

<p>カイトは頷いた。</p>

<blockquote>「ああ。でも、俺のパッチがあれば、即座の崩壊は防げる。あとは、住人たちがどう受け止めるかだ」</blockquote>

<p>カイトは、仲間たちを見渡した。エリナ、リアム、セシリア、そしてマルクス。</p>

<blockquote>「俺たちの旅は、まだ終わらない。むしろ、ここからが本番だ」</blockquote>

<p>エリナが、静かに微笑んだ。</p>

<blockquote>「覚悟はできてるわ。カイト」</blockquote>

<p>リアムが、剣を鞘に収めた。</p>

<blockquote>「面倒なことになる予感がするが……まあ、退屈はしなさそうだ」</blockquote>

<p>セシリアが、カイトの腕に抱きついた。</p>

<blockquote>「私も！カイトと一緒なら、どこへでも行く！」</blockquote>

<p>カイトは、彼らの言葉に力を得た。そう、彼は一人ではない。信頼できる仲間がいる。そして、世界には、まだ会ったことのない無数の住人がいる。</p>

<blockquote>「行こう。まずは――」</blockquote>

<p>カイトは、遺跡の出口を見据えた。</p>

<blockquote>「最初の街へ。ギルドマスターのマグナスに会いに行くんだ」</blockquote>

<hr>

<p>古代遺跡を後にしたカイトたちは、夜道を急いでいた。空には、二つの月が輝いている。この世界の空には、常に二つの月がある。それが、世界が人工的なシステムであることの証左の一つだと、カイトは今になって理解した。</p>

<blockquote>「ねえ、カイト」</blockquote>

<p>セシリアが、歩きながら話しかけてきた。</p>

<blockquote>「真実を知った人たち、どう思うかな」</blockquote>

<p>カイトは少し考えた。</p>

<blockquote>「わからない。でも、隠し続けるよりはいいと思う。真実を知った上で、それでも世界を愛していると言えるなら、それは本物だからな」</blockquote>

<p>セシリアは、カイトの顔を見上げた。</p>

<blockquote>「カイトは、優しいね」</blockquote>

<p>カイトは苦笑した。</p>

<blockquote>「優しいっていうか、プログラマーとしての性分だよ。ユーザーには、真実を知る権利があるって思ってる」</blockquote>

<p>エリナが、前を向いたまま言った。</p>

<blockquote>「でも、真実は時に残酷よ。すべての人が、それを受け入れられるとは限らない」</blockquote>

<p>カイトは頷いた。</p>

<blockquote>「ああ。だからこそ、俺たちが支える必要がある。真実を知って動揺した人たちを、受け止める場所を作るんだ」</blockquote>

<p>マルクスが、静かに口を開いた。</p>

<blockquote>「わたしは、何百年もの間、動揺した魂を『統合』してきた。だが、これからは『統合』ではなく、『対話』をする番か」</blockquote>

<p>カイトは、マルクスの肩を叩いた。</p>

<blockquote>「頼りにしてるぜ、マルクス。お前の経験は、きっと役に立つ」</blockquote>

<p>夜風が、カイトたちの頬を撫でていく。遥か彼方に、街の灯りが見え始めていた。</p>

<p>新しい戦いの幕開け。真実を巡る、世界規模の対話。</p>

<p>カイトは、自分の胸に手を当てた。その奥で、青白い光が静かに脈打っている。彼が当てたパッチ。そして、セキュリティとの約束。</p>

<pre><code>
// Promise to the World
if (consensus.reached()) {
    if (majority.trust(kait)) {
        security.deactivate();
        world.evolve();
    } else {
        security.purge(kait);
    }
}
</code></pre>

<p>成功すれば、世界は新たな段階へと進む。失敗すれば、彼は消滅する。</p>

<p>だが、カイトは恐れていなかった。彼には仲間がいる。そして、何より――この世界を信じる気持ちがあった。</p>

<blockquote>「さあ、行こう」</blockquote>

<p>カイトの声に、仲間たちが頷く。</p>

<p>夜明け前の静寂の中、五人の姿は、街へと向かって歩き出した。</p>
