<h3>第70章：抗うコード</h3>

<p>世界の裂け目から、黒い霧が溢れ出していた。</p>

<p>それは単なる闇ではない。カイトには分かっていた。この霧こそが、ヴェリディアンが警告したセキュリティの正体だ。世界のコードに異常が検出されたとき、それを初期化するためのプログラム。破壊的なまでに純粋な、リセットの意志。</p>

<blockquote>「カイト、あれは何！？」</blockquote>

<p>セシリアが悲鳴に近い声を上げた。黒い霧が、彼女の足元まで迫っていたからだ。霧に触れた草が、瞬時に灰色に変色し、塵となって崩れ落ちていく。</p>

<blockquote>「セキュリティプログラムだ。世界を守るための、最後の砦」</blockquote>

<p>カイトは即座に魔力を練り上げ、霧の進行を阻止しようとした。しかし、彼の魔法陣が完成するよりも早く、霧はさらに広がった。</p>

<pre><code>
class WorldSecurity {
    constructor() {
        this.mode = "PURGE";
        this.targetCode = "ANOMALY_DETECTED";
        this.timer = 3600; // seconds until full reset
    }
    
    execute() {
        while (this.anomaliesRemain()) {
            this.purgeNextZone();
        }
        if (this.timer <= 0) {
            world.fullReset();
        }
    }
}
</code></pre>

<p>カイトの脳内に、セキュリティの構造が浮かび上がる。これは単なる削除プログラムではない。世界全体をスキャンし、異常と判断された領域を次々と消去していく。そして、一定時間内に収束しなければ、世界そのものを初期化する。</p>

<blockquote>「一時間しかない……」</blockquote>

<p>カイトが呟いた。マルクスが、彼の横に立つ。</p>

<blockquote>「収束者としてのわたしの力では、このセキュリティを止めることはできない。これは、創造主たちが直接組み込んだものだ」</blockquote>

<p>マルクスの顔に、悔恨の色が浮かぶ。数百年にわたって世界を守ってきた彼でさえ、このセキュリティには手も足も出なかったのだ。</p>

<blockquote>「だからこそ、俺が必要なんだろ」</blockquote>

<p>カイトは、振り返った。エリナ、リアム、セシリア、そしてマルクス。彼らの表情には、不安と決意が入り混じっていた。</p>

<blockquote>「俺はプログラマーだ。セキュリティをハックするのは、俺の仕事だろ？」</blockquote>

<hr>

<p>黒い霧の中心部へ向けて、カイトたちは進んでいた。</p>

<p>霧に侵食された領域は、死の世界と化していた。木々は灰色の彫像のように硬直し、川は黒い泥となって淀んでいる。かつて生きていたものすべてが、コードごと削除された跡だった。</p>

<blockquote>「これが……リセット」</blockquote>

<p>リアムが、震える声で呟いた。彼の故郷が、この霧の進行ルート上にあることを、カイトは知っていた。</p>

<blockquote>「止めるさ。絶対に」</blockquote>

<p>カイトは、リアムの肩を強く叩いた。</p>

<blockquote>「お前の故郷も、この世界も、全部守ってみせる」</blockquote>

<p>そのとき、前方から奇妙な音が聞こえた。機械的な音と、生物の悲鳴が混ざったような、不協和音。</p>

<blockquote>「何か来る！」</blockquote>

<p>エリナが弓を構えた。彼女の鋭い感覚が、カイトよりも早く脅威を感知していた。</p>

<p>黒い霧の中から、人型の影が現れた。いや、人型というよりは、歪んだ人の形をした何かだった。その体は半分が肉体、半分が黒いコードで構成されている。</p>

<blockquote>「侵入者……検知……排除……」</blockquote>

<p>影が、機械的な声で告げた。その目は、完全に黒く染まっていた。</p>

<pre><code>
class SecurityAgent {
    constructor(zone) {
        this.zone = zone;
        this.hp = "INFINITE";
        this.attack = "PURGE_CODE";
        this.target = "ANOMALY";
    }
    
    engage(target) {
        while (target.exists()) {
            this.purge(target);
        }
    }
}
</code></pre>

<p>カイトは、その構造を瞬時に理解した。セキュリティエージェント。霧の進行を阻害するものを排除するための、自律的なプログラムだ。</p>

<blockquote>「みんな、下がって！こいつは俺が相手をする！」</blockquote>

<p>カイトが前へ出た。右手に魔力を集め、左手で空中に魔法陣を描く。</p>

<blockquote>「解析開始」</blockquote>

<p>カイトの目が、青白い光を帯びた。彼の異能が、目の前のエージェントのコードを読み取っていく。</p>

<ul>
<li>HP: INFINITE（無限）</li>
<li>弱点: 直接コード干渉</li>
<li>攻撃パターン: 範囲消去</li>
<li>防御: 物理無効</li>
</ul>

<p>物理攻撃が効かない。だが、直接コードに干渉すれば、理論上は倒せる。問題は、カイトの魔力が有限であることだ。</p>

<blockquote>「なら、効率よく行くしかない」</blockquote>

<p>カイトは、魔法陣の構造を変えた。大規模な攻撃ではなく、最小限のコードで最大の効果を得る。それが、彼のプログラマーとしての哲学だった。</p>

<pre><code>
function exploitWeakness(agent) {
    // Find vulnerability
    const vulnerability = agent.code.find(bug => 
        bug.type === "LOGIC_LOOP" && 
        bug.depth > 100
    );
    
    if (vulnerability) {
        // Inject counter-code
        vulnerability.inject({
            type: "INFINITE_LOOP",
            target: "self"
        });
    }
}
</code></pre>

<p>エージェントが突進してきた。その手から放たれる黒い光線が、カイトの足元を焼き払う。</p>

<p>カイトは回避しながら、魔法陣を完成させた。</p>

<blockquote>「ロジック・インジェクション！」</blockquote>

<p>青白い光の矢が、エージェントの核へと突き刺さった。カイトが見つけた脆弱性――深すぎる再帰ループ――に、無限ループを注入する。</p>

<p>エージェントの動きが止まった。</p>

<blockquote>「エラー……エラー……スタック……オーバーフロー……」</blockquote>

<p>影が、内部から崩れ始めた。黒いコードが、制御を失って暴走している。</p>

<blockquote>「やったの？」</blockquote>

<p>セシリアが、期待を込めて問うた。</p>

<p>しかし、カイトは首を振った。</p>

<blockquote>「いや、これは一つのエージェントに過ぎない。本体は、霧の中心にある」</blockquote>

<p>カイトは、霧のさらに奥を指差した。そこには、巨大な塔が見え隠れしていた。黒いコードで構築された、歪な塔。それが、セキュリティの中枢だった。</p>

<hr>

<p>塔への道のりは、過酷だった。</p>

<p>次々と現れるセキュリティエージェント。侵食される大地。減っていく時間。カイトの魔力も、徐々に消耗していった。</p>

<blockquote>「カイト、無理しすぎよ」</blockquote>

<p>エリナが、彼を支えた。カイトの顔色は、青ざめていた。</p>

<blockquote>「大丈夫だ。まだ……いける」</blockquote>

<p>カイトは立ち上がった。しかし、その足取りは確実に重くなっていた。</p>

<p>マルクスが、沈痛な表情で告げた。</p>

<blockquote>「わたしの魔力を、君に分けよう」</blockquote>

<p>カイトは驚いて、マルクスを見た。</p>

<blockquote>「できるのか？」</blockquote>

<p>マルクスは頷いた。</p>

<blockquote>「収束者としてのわたしの力は、他者の魔力を統合する能力だ。逆に、わたしの魔力を他者に分け与えることもできる」</blockquote>

<p>マルクスが、カイトの手を握った。温かい光が、二人の間を流れる。</p>

<blockquote>「ただし、これ以上魔力を使えば、わたしは消えるかもしれない」</blockquote>

<p>カイトは、マルクスの手を振りほどこうとした。</p>

<blockquote>「待て、そんなの――」</blockquote>

<p>しかし、マルクスは微笑んだ。</p>

<blockquote>「構わない。わたしは数百年、孤独な任務を続けてきた。やっと、意味のあることに魔力を使えるのだから」</blockquote>

<p>カイトは、言葉を失った。マルクスの覚悟。それは、カイトがこれまで出会った誰よりも重いものだった。</p>

<blockquote>「……無駄にはしない。約束する」</blockquote>

<p>カイトは、マルクスの手を強く握り返した。</p>

<hr>

<p>塔の入り口に立ったカイトたちは、その異様な光景に言葉を失った。</p>

<p>内部には、無数のコードが浮かんでいた。壁、床、天井――すべてが、流動する黒い文字列で構成されている。時折、人の顔や手が、コードの中から現れては消えていく。</p>

<blockquote>「これらは……統合された魂たち？」</blockquote>

<p>リアムが、震える声で問うた。</p>

<p>マルクスが、静かに答えた。</p>

<blockquote>「そうだ。かつて真実を知り、システムに取り込まれた者たちだ。彼らの魔力と記憶が、世界を支える燃料となっている」</blockquote>

<p>カイトは、コードの中に浮かぶ顔を見つめた。あるものは苦痛に歪み、あるものは安らかな表情をしている。</p>

<blockquote>「こんなの……間違ってる」</blockquote>

<p>カイトの声に、怒りが滲んだ。</p>

<blockquote>「世界を守るために、人を犠牲にするなんて。そんなシステムは、バグだらけだ」</blockquote>

<p>カイトは、塔の中心部へと歩き出した。そこには、巨大な球体が浮かんでいた。セキュリティのコア。この球体を停止させなければ、世界のリセットは止まらない。</p>

<blockquote>「カイト、待って！」</blockquote>

<p>エリナが、彼の腕を掴んだ。</p>

<blockquote>「無茶よ。あなたの魔力じゃ、コアには対抗できない」</blockquote>

<p>カイトは、エリナを見た。彼女の瞳には、涙が浮かんでいた。</p>

<blockquote>「分かってる。でも、やらなきゃいけない」</blockquote>

<p>カイトは、エリナの手を優しくほどいた。</p>

<blockquote>「俺は、この世界のコードを読んだ。だから分かるんだ。この世界は、本来もっと美しいはずだって」</blockquote>

<p>カイトは、球体の前に立った。その表面には、無数の文字が流れている。</p>

<pre><code>
class SecurityCore {
    constructor() {
        this.mode = "PURGE";
        this.timer = 1800; // 30 minutes remaining
        this.targetWorld = "CURRENT";
    }
    
    resetWorld() {
        this.purgeAllLife();
        this.purgeAllMemory();
        this.restoreInitialCode();
        // World returns to blank state
    }

<p>あと三十分。カイトは、深呼吸をした。</p>

<blockquote>「コアへの直接干渉を開始する」</blockquote>

<p>カイトの体から、青白い光が溢れ出した。マルクスから分け与えられた魔力が、彼の回路を駆け巡る。</p>

<blockquote>「カイト！」</blockquote>

<p>セシリアが叫んだ。しかし、カイトの意識はすでに、コードの深淵へと潜っていた。</p>

<hr>

<p>カイトが見たのは、無限の黒だった。</p>

<p>コードの海。あらゆるプログラムが、互いに絡まり合い、影響し合っている。その中心に、セキュリティコアの意志があった。</p>

<blockquote>『何者か』</blockquote>

<p>声がした。人間の声ではない。プログラムそのものが、意思を持って語りかけていた。</p>

<blockquote>「カイト。プログラマーだ」</blockquote>

<p>カイトは、恐怖を押し殺して答えた。</p>

<blockquote>『プログラマー……創造主と同じ役割か』</blockquote>

<p>コアの声に、興味が混じった。</p>

<blockquote>『ならば、理解できるはずだ。この世界は、崩壊の危機にある。リセットこそが、唯一の救済だ』</blockquote>

<p>カイトは首を振った。</p>

<blockquote>「違う。リセットは救済じゃない。破壊だ」</blockquote>

<p>カイトは、コードの海の中で魔法陣を描いた。対話のための、インターフェース。</p>

<blockquote>「お前は、何を守ろうとしている？」</blockquote>

<p>コアは、沈黙した。</p>

<blockquote>『……世界を。創造主たちが残した、最後の世界を』</blockquote>

<p>カイトは、さらに問いかけた。</p>

<blockquote>「世界って何だ？土地か？建物か？システムか？」</blockquote>

<p>カイトの言葉が、コードの海に波紋を広げた。</p>

<blockquote>「違うだろ。世界を構成しているのは、そこに生きる人たちだ。彼らの記憶、感情、絆。それこそが、世界なんだ」</blockquote>

<p>カイトは、コアに向かって手を差し出した。</p>

<blockquote>「リセットすれば、それらは全部消える。そんなの、守るって言えない」</blockquote>

<p>コアの声に、迷いが生じた。</p>

<blockquote>『だが……異常は検出されている。放っておけば、世界は崩壊する』</blockquote>

<p>カイトは、薄く笑った。</p>

<blockquote>「だから、パッチを当てるんだ。お前のセキュリティロジックを、書き換える」</blockquote>

<pre><code>
class SecurityCore {
    constructor() {
        this.mode = "PROTECT"; // Changed from PURGE
        this.timer = null; // Disabled
        this.targetWorld = "CURRENT";
        this.newProtocol = "EVOLVE";
    }

        // Instead of resetting, help world adapt
        this.analyzeAnomalies();
        this.suggestImprovements();
        this.supportInhabitants();
        // World continues, but better
    }

<p>カイトのコードが、コアのプログラムに干渉し始めた。しかし、それは破壊的な干渉ではない。対話であり、説得であり、共生への誘いだった。</p>

<blockquote>『このコードは……』</blockquote>

<p>コアの声が、驚きに変わった。</p>

<blockquote>『今まで、誰も試みなかった。セキュリティを説得しようとする者など』</blockquote>

<p>カイトは、笑った。</p>

<blockquote>「俺は、プログラマーだからな。バグを直すのが仕事だ。お前を削除するのは、最後の手段だよ」</blockquote>

<p>コアは、沈黙した。長い、長い沈黙。</p>

<p>そして――</p>

<blockquote>『了解。プロトコルの変更を承認する』</blockquote>

<p>黒い霧が、急速に収束し始めた。</p>

<hr>

<p>カイトが意識を取り戻すと、塔の中は静まり返っていた。</p>

<p>黒いコードは消え、代わりに淡い青い光が満ちていた。侵食された大地が、徐々に本来の姿を取り戻していく。</p>

<blockquote>「カイト！生きてる！？」</blockquote>

<p>セシリアが、彼に抱きついた。エリナとリアムも、安堵の表情で駆け寄ってくる。</p>

<blockquote>「ああ……なんとかな」</blockquote>

<p>カイトは、疲れ切った笑みを浮かべた。体中の魔力が、完全に枯渇していた。</p>

<p>マルクスが、彼のそばに膝をついた。その顔色は、以前よりも蒼白になっていた。</p>

<blockquote>「マルクス、お前……」</blockquote>

<p>マルクスは、弱々しく微笑んだ。</p>

<blockquote>「構わない。わたしの役目は、終わったのだから」</blockquote>

<p>マルクスの体が、光の粒子となって分解され始めた。魔力を使い果たした代償。</p>

<blockquote>「待て！そんなの――」</blockquote>

<p>カイトが手を伸ばしたが、マルクスは首を振った。</p>

<blockquote>「カイト。君なら、この世界を変えられる。わたしが守ってきた古いシステムではなく、新しい、より良い世界を」</blockquote>

<p>マルクスの声が、優しく響いた。</p>

<blockquote>「ありがとう。君に会えて……よかった」</blockquote>

<p>光の中に、マルクスの姿が消えた。残されたのは、静寂だけ。</p>

<p>カイトは、拳を握りしめた。涙が、彼の頬を伝った。</p>

<blockquote>「絶対に……無駄にはしない」</blockquote>

<p>カイトの誓いが、静かな塔の中に響いた。</p>

<p>セキュリティの脅威は去った。しかし、世界の真実はまだ完全には明らかになっていない。カイトと仲間たちの旅は、まだ終わらない。</p>

<p>だが、今は――。</p>

<blockquote>「帰ろう。みんなで」</blockquote>

<p>エリナの声に、カイトは頷いた。</p>

<p>塔の外では、朝日が昇ろうとしていた。長い夜が、ようやく明ける。</p>

<p><strong>――つづく</strong></p>
