<h3>第12章：カスタム・ファンクション</h3>

<hr>

<p>翌朝、俺たちは王都の魔法図書館に来ていた。</p>

<p>オリジナルの魔法を作る——そのために必要な知識を集めるためだ。</p>

<hr>

<h4>魔法図書館</h4>

<p>王都の魔法図書館は、三百年の歴史を持つ巨大な建物だった。</p>

<p>天井まで届く本棚。無数の魔導書。そして、古びた紙と魔力の混じった独特の匂い。</p>

<hr>

<blockquote>「カイトさん、何を探しているんですか？」エリナが聞いた。</blockquote>

<hr>

<blockquote>「魔法の基本構造についての資料だ」俺は答えた。「既存の魔法を真似するだけじゃなく、自分でゼロから作りたい」</blockquote>

<hr>

<p>俺は、プログラミングの世界で学んだことを思い出していた。</p>

<p>優れたプログラマーは、既存の関数を使うだけでなく、自分で関数を作れる。自分だけの、カスタム・ファンクションを。</p>

<hr>

<p>魔法も同じはずだ。</p>

<hr>

<blockquote>「こちらに、基礎魔法構造論の書籍がありますわ」セシリアが一冊の本を持ってきた。</blockquote>

<hr>

<p>俺はその本を開いた。</p>

<p>中には、魔法陣の基本的な構造が図解されていた。</p>

<hr>

<pre><code>
// Basic Magic Structure
struct Magic {
  name: String,           // 魔法の名前
  elements: Array,        // 使用する元素
  power: Number,          // 出力
  range: Number,          // 効果範囲
  duration: Number,       // 持続時間
  cost: Number            // 魔力消費
}
</code></pre>

<hr>

<p>「なるほど......」俺は呟いた。「魔法も、データの集合体なんだ」</p>

<hr>

<p>俺はさらに読み進めた。</p>

<p>魔法には三つの必須要素があるらしい。</p>

<hr>

<ul>
<li><strong>インプット</strong>：魔法を発動するための条件</li>
<li><strong>プロセス</strong>：魔力を変換する処理</li>
<li><strong>アウトプット</strong>：最終的な効果</li>
</ul>

<hr>

<p>「関数と同じだ......」俺は理解した。</p>

<hr>

<pre><code>
// Magic as a Function
function createMagic(input) {
  // Process: Convert mana
  const process = convertMana(input);
  
  // Output: Apply effect
  return applyEffect(process);
}
</code></pre>

<hr>

<h4>設計開始</h4>

<p>俺は、自分だけの魔法を設計し始めた。</p>

<hr>

<blockquote>「どんな魔法を作るんですか？」リアムが興味津々で聞いた。</blockquote>

<hr>

<blockquote>「防御と攻撃を組み合わせた魔法だ」俺は言った。「普通、防御魔法と攻撃魔法は別々だけど、それを一つに統合したい」</blockquote>

<hr>

<p>俺は紙に書き始めた。</p>

<hr>

<pre><code>
// Custom Magic Design: "Guardian Flame"
// A defensive wall that burns attackers

const GuardianFlame = {
  // Name
  name: "Guardian Flame",
  
  // Elements: Fire (attack) + Earth (defense)
  elements: ["fire", "earth"],
  
  // Input: Enemy proximity
  trigger: (distance) => distance < 5,
  
  // Process: Create barrier with flames
  process: function(mana) {
    const barrier = createElement("earth");
    const flames = createElement("fire");
    
    barrier.shape("dome");
    barrier.thickness(10);
    
    flames.attach(barrier);
    flames.temperature(1500); // Celsius
    
    return {
      defense: barrier,
      attack: flames
    };
  },
  
  // Output: Protected area
  output: function(result) {
    result.defense.activate();
    result.attack.activate();
    return "Guardian Flame activated!";
  },
  
  // Cost calculation
  calculateCost: (duration) => {
    return 50 * duration; // 50 mana per second
  }
};
</code></pre>

<hr>

<blockquote>「ガーディアン・フレイム......守護の炎、ですか」エリナが呟いた。「かっこいい名前ですね！」</blockquote>

<hr>

<blockquote>「名前は重要だ」俺は笑った。「いい名前をつければ、魔法も気合が入る」</blockquote>

<hr>

<h4>実装</h4>

<p>設計が終わった。次は実装だ。</p>

<hr>

<p>俺たちは図書館の裏にある実験場に移動した。</p>

<p>ここなら、魔法を試しても安全だ。</p>

<hr>

<p>俺は、地面に魔法陣を描き始めた。</p>

<p>設計図を、実際の魔法陣に変換する作業だ。</p>

<hr>

<blockquote>「カイトさん、何か手伝えることは？」セシリアが聞いた。</blockquote>

<hr>

<blockquote>「魔力供給を頼む」俺は言った。「俺が魔法陣を描く間、安定して魔力を送り続けてほしい」</blockquote>

<hr>

<blockquote>「承知いたしました」</blockquote>

<hr>

<p>セシリアが、詠唱を始めた。</p>

<p>周囲の空気が張り詰める。彼女の魔力が、俺の周りに流れ込んでくる。</p>

<hr>

<p>俺は集中した。</p>

<p>地面に、幾何学的な模様を描いていく。</p>

<hr>

<pre><code>
// Drawing the Magic Circle
// Layer 1: Base structure (Earth)
drawCircle(center, radius);
drawRunes(["protect", "stabilize", "endure"]);

// Layer 2: Fire elements
drawTriangle(center, direction: "up");
drawRunes(["burn", "consume", "purify"]);

// Layer 3: Integration layer
connectLayers(layer1, layer2);
addCondition("trigger_on_threat");
addCondition("auto_deactivate_after:60s");
</code></pre>

<hr>

<p>三十分後。</p>

<p>俺は描き終えた。</p>

<hr>

<p>目の前に、複雑な魔法陣が浮かび上がっていた。</p>

<p>二重の円。その内側に三角形。そして、無数のルーン文字が巡っている。</p>

<hr>

<blockquote>「すごい......」エリナが息を呑んだ。「カイトさんの魔力の色、見たことない色です」</blockquote>

<hr>

<blockquote>「え？」俺は自分の手を見た。</blockquote>

<hr>

<p>確かに、俺の魔力が淡い金色に輝いていた。</p>

<p>今までの魔法は、青や緑だった。でも、これは金色だ。</p>

<hr>

<blockquote>「オリジナルの魔法だからじゃないか？」リアムが言った。「他人のコードを改造したんじゃなく、自分で書いたコード。だから、自分だけの色が出るんだよ」</blockquote>

<hr>

<p>「そうかもな」俺は頷いた。</p>

<hr>

<h4>テスト</h4>

<p>いよいよテストだ。</p>

<hr>

<blockquote>「みんな、下がって」俺は言った。「成功する保証はないから」</blockquote>

<hr>

<p>三人が後ろに下がる。</p>

<p>俺は、魔法陣の中央に立った。</p>

<hr>

<p>深呼吸。</p>

<p>そして——</p>

<hr>

<blockquote>「ガーディアン・フレイム、起動！」</blockquote>

<hr>

<pre><code>
// Activate Guardian Flame
GuardianFlame.process(mana);
GuardianFlame.output(result);

// Console output:
// > Initializing Guardian Flame...
// > Earth element: OK
// > Fire element: OK
// > Integration: OK
// > Guardian Flame activated!
</code></pre>

<hr>

<p>魔法陣が爆発的に光った。</p>

<p>地面から、半透明のドームがせり上がってくる。</p>

<p>そして、ドームの表面に炎が走った。</p>

<hr>

<p>「成功だ......！」俺は叫んだ。</p>

<hr>

<p>半径五メートルのドームが、俺を守っている。</p>

<p>そして、そのドームに触れようとするものは——何でも燃やし尽くす炎に包まれる。</p>

<hr>

<blockquote>「カイトさん、すごいです！ 自分で魔法を作っちゃいましたよ！」エリナが飛びついてきた。</blockquote>

<hr>

<p>ドームの外からでも、エリナの興奮が伝わってくる。</p>

<p>俺は笑った。</p>

<hr>

<blockquote>「でも、これで終わりじゃない」俺は言った。「まだテストが足りない」</blockquote>

<hr>

<h4>バグ発見</h4>

<p>俺は、魔法を維持しながら観察を続けた。</p>

<hr>

<p>一分後。</p>

<p>二分後。</p>

<hr>

<p>「あれ？」俺は違和感に気づいた。</p>

<hr>

<p>炎の温度が、徐々に下がっている。</p>

<p>最初は千五百度あったはずが、今は千度を切っている。</p>

<hr>

<pre><code>
// Debug output
console.log(GuardianFlame.status);
// { temperature: 980, manaCost: 52, uptime: 120 }

// Issue detected: Temperature decay
// Root cause: Fire element not receiving enough mana
</code></pre>

<hr>

<p>「バグがある」俺は呟いた。</p>

<hr>

<p>設計ミスだ。</p>

<p>土と炎、二つの元素に魔力を分配しているが、バランスが悪い。</p>

<hr>

<p>土（防御）に多くの魔力が流れすぎて、炎（攻撃）への供給が不足している。</p>

<hr>

<blockquote>「修正が必要だ」俺は言った。「設計を少し変える」</blockquote>

<hr>

<h4>パッチ適用</h4>

<p>俺は、魔法を維持したまま、コードを修正した。</p>

<hr>

<pre><code>
// Hot patch for Guardian Flame v1.0
// Issue: Temperature decay due to uneven mana distribution

// Fix: Adjust mana allocation ratio
GuardianFlame.manaAllocation = {
  earth: 0.4,  // Reduced from 0.6
  fire: 0.6    // Increased from 0.4
};

// Apply patch
reloadMagicCircle();
console.log("Patch applied successfully!");
</code></pre>

<hr>

<p>魔法陣が一瞬明滅した。</p>

<p>そして、炎の温度が再び上昇し始めた。</p>

<hr>

<pre><code>
// Status after patch
console.log(GuardianFlame.status);
// { temperature: 1520, manaCost: 50, uptime: 180 }
// ✓ Temperature stabilized
// ✓ Mana cost reduced
</code></pre>

<hr>

<blockquote>「直りました！」俺は安堵した。</blockquote>

<hr>
