<h3>第59章：禁忌の封印</h3>

<p>古代の図書館の奥深く、カイトたちは静寂に包まれていた。天井まで届く書架には、何千年もの知識が詰まっている。エリナがランタンを掲げると、黄金色の光が埃の舞う空間を照らし出した。</p>

<blockquote>「すごい……これほどの量の文献が、こんな場所に眠っていたなんて」</blockquote>

<p>エリナの声が小さな echo を残して広がる。リアムは慎重に周囲を警戒しながら、カイトの背中を守る位置に立っていた。</p>

<blockquote>「カイト、気をつけて。古代の図書館には、守護者の仕掛けがあるかもしれない」</blockquote>

<p>カイトは頷きながら、目の前にある巨大な分類台に目を落とした。そこには複雑なルーン文字が刻まれ、まるでコンソール画面のように並んでいる。</p>

<blockquote>「これ……検索システムみたいだ」</blockquote>

<p>彼は指先でルーンをなぞった。すると、空中に半透明のウィンドウが浮かび上がる。魔法の画面には、カテゴリ一覧が表示されていた。</p>

<pre><code>// 古代図書館インデックス
categories: [
  "elemental_magic",
  "summoning_arts", 
  "dimension_theory",
  "forbidden_archives",
  "creation_rituals"
]</code></pre>

<p>カイトは息を呑んだ。forbidden_archives——禁忌の書庫。その文字が、妙に気になった。</p>

<blockquote>「カイト？ どうしたの？」</blockquote>

<p>セシリアが彼の顔を覗き込む。カイトは指差した。</p>

<blockquote>「見て。禁忌の書庫って項目がある。でも……ロックされている」</blockquote>

<p>確かに、その項目は赤く輝き、アクセスできない状態だった。通常の権限では開けないようになっている。</p>

<blockquote>「そりゃそうでしょ。禁忌ってくらいだから、危険な内容なんじゃない？」</blockquote>

<p>リアムが眉をひそめる。だが、カイトの脳裏には、マルクスの言葉が蘇っていた。</p>

<p><i>「収束者は、世界の境界を操作する能力者だ。その力の源は……」</i></p>

<p>マルクスは途中で言葉を切っていた。何かを隠していたのだ。</p>

<blockquote>「カイト、まさか……」</blockquote>

<p>エリナが彼の意図に気づいた。カイトは覚悟を決めて、検索欄にコードを入力し始めた。</p>

<pre><code>// アクセス権限昇格の試み
function elevateAccess(user, target) {
  if (user.hasMark("converger")) {
    return decrypt(target.seal);
  }
  return ACCESS_DENIED;
}</code></pre>

<p>カイトの手の甲が輝く。収束者としての紋章が、魔法のコードに反応したのだ。禁忌の書庫の封印が、音もなく解かれていく。</p>

<blockquote>「開いた……」</blockquote>

<p>書架の一部が回転し、隠された通路が現れた。その奥には、一冊の黒い書物が台座に置かれていた。</p>

<blockquote>「あれが……禁忌の書？」</blockquote>

<p>セシリアが小声で尋ねる。四人は慎重に近づいた。黒い表紙には、見たこともない言語が刻まれている。だが、不思議なことに、カイトにはその意味が理解できた。</p>

<blockquote>「『創世と終焉の理』……これがタイトルか」</blockquote>

<p>カイトが手を伸ばした瞬間、アルフレッドの声が背後から響いた。</p>

<blockquote>「待て、カイト・海藤！」</blockquote>

<p>守人の長老が、息を切らして駆け込んできた。その表情は、カイトがこれまで見た中で最も深刻だった。</p>

<blockquote>「その本に触れてはならない。それは……世界そのものを書き換える力を持っている」</blockquote>

<blockquote>「世界を書き換える？」</blockquote>

<p>カイトは手を止めた。アルフレッドはゆっくりと近づき、台座の周りに刻まれた警告のルーンを指し示した。</p>

<blockquote>「この図書館を建てたのは、第一紀の魔導師たちだ。彼らは魔法を言語として体系化し、世界の理をコードとして記録しようとした」</blockquote>

<p>アルフレッドは一息ついて、続けた。</p>

<blockquote>「だが、彼らは気づいてしまった。この世界自体が、誰かによってプログラムされたものだと」</blockquote>

<p>沈黙が広がる。カイトの心臓が早鐘を打った。</p>

<blockquote>「プログラム……された？ この世界が？」</blockquote>

<blockquote>「そうだ。そしてその本には、この世界のソースコードの一部が記されている。創造主が残した、世界編集の権限コードが」</blockquote>

<p>エリナが顔を覆った。</p>

<blockquote>「そんな……じゃあ、私たちは何なの？ ただの……プログラム？」</blockquote>

<blockquote>「違う」</blockquote>

<p>アルフレッドは首を横に振った。</p>

<blockquote>「お前たちは自律している。意志を持ち、感情を持ち、自ら選択している。だが……この世界の基本法則は、誰かによって設計されたものだ。魔法も、収束者の力も、すべてはその設計の一部」</blockquote>

<p>カイトは本を見つめた。黒い表紙が、まるで彼を誘うように輝いている。</p>

<blockquote>「じゃあ、俺がこの世界に召喚されたのも……何かのプログラムの一部？」</blockquote>

<blockquote>「おそらく。異世界からの訪問者は、世界の更新プログラムのようなものだ。バグを修正し、システムを最適化するために呼ばれる」</blockquote>

<p>リアムが剣の柄に手をかけた。</p>

<blockquote>「俺たちは……ただのパッチ要員だって言うのか？」</blockquote>

<blockquote>「いいや。パッチは選択できない。だが、お前たちは選べる。その本を使うかどうかを」</blockquote>

<p>アルフレッドはカイトの目を真っ直ぐに見た。</p>

<blockquote>「カイト・海藤。お前は収束者として、世界の境界を超える力を持っている。その本を使えば、この世界の理を書き換えられるかもしれない。戦争を終わらせ、病を eradication し、死さえも克服できるかもしれない」</blockquote>

<p>カイトの手が震えた。プログラマーとして、バグを修正することは日常茶飯事だ。だが、世界そのものを書き換える？ それは……</p>

<blockquote>「でも、そんなことが許されるのか？ 世界を自分の都合で書き換えて」</blockquote>

<blockquote>「許されるかどうかは、誰も決められない。創造主はもういない。この世界を判断できるのは、今ここにいる私たちだけだ」</blockquote>

<p>セシリアがカイトの手を握った。</p>

<blockquote>「カイト……あなたが決めることよ。私は、あなたの選択を信じている」</blockquote>

<p>エリナも頷いた。</p>

<blockquote>「私も。あなたがこの世界で出会った人たちを大切に思っていることは知っている。だから……何を選んでも、正しいと思う」</blockquote>

<p>リアムは少し照れくさそうに頭をかいた。</p>

<blockquote>「ま、俺は剣で戦うことしかできないけど。でもよ、カイト。お前が決めたことなら、最後までついていくぜ」</blockquote>

<p>カイトは深呼吸をした。仲間の言葉が、彼の心を温かく包んだ。</p>

<p>彼は本に手を伸ばした——が、触れる直前に止めた。</p>

<blockquote>「いや、まだだ」</blockquote>

<p>カイトは本から手を引っ込めた。</p>

<blockquote>「まだ読むべきじゃない。まずは……もっと知る必要がある」</blockquote>

<p>アルフレッドが微かに微笑んだ。</p>

<blockquote>「賢明な判断だ。力は、使い方を知ってこそ意味を持つ」</blockquote>

<p>カイトは本を台座に残したまま、踵を返した。禁忌の書は、今はまだ開かない。</p>

<p>だが、彼は知ってしまった。この世界がプログラムされた存在であることを。そして、自分がそのコードに触れる力を持っていることを。</p>

<blockquote>「行こう。やるべきことはまだある」</blockquote>

<p>カイトの言葉に、三人は頷いた。彼らは禁忌の書庫を後にし、図書館のメインホールへと戻った。</p>

<p>アルフレッドは最後に一度だけ本を振り返り、静かに呟いた。</p>

<blockquote>「カイト・海藤……お前が救世主になるか、破壊者になるか。すべては、その選択にかかっている」</blockquote>

<p>古代の図書館に、静寂が戻った。黒い本は、ただ静かに、次の訪問者を待ち続けていた。</p>
