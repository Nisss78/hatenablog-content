<h3>第55章：共鳴する記憶</h3>

<p>崩れゆく古代遺跡の中で、カイトはマルクスと対峙していた。守人アルフレッドが遺した真実の断片が、カイトの魔力回路に焼き付いている。世界は人工的に構築された魔力循環システムに過ぎない――その事実が、カイトの内側で激しく共鳴していた。</p>

<blockquote>「カイト、下がって！」</blockquote>

<p>エリナの鋭い声と同時に、光の矢がマルクスの足元を穿った。しかし、収束者と呼ばれる男は微動だにしない。まるで、その攻撃が来ることが分かっていたかのように。</p>

<blockquote>「無駄だ。この空間全体が、わたしの監視下にある」</blockquote>

<p>マルクスが静かに告げる。その声音には、敵意というよりは、どこか哀れみが混じっていた。</p>

<blockquote>「君たちは知らないだけだ。この世界が、どのような目的で創られ、どのような終わりを迎える予定だったのかを」</blockquote>

<p>カイトは目を細めた。プログラマーとしての彼の感覚が、マルクスの言葉の裏にある構造を読み取ろうとしていた。この世界が人工的なシステムだとするならば、すべての事象には何らかの意図があるはずだ。</p>

<pre><code>
if (world.isArtificial) {
    return world.getCreatorIntent();
}
</code></pre>

<p>カイトの脳内で、自然とコードが浮かぶ。この感覚は、彼がこの世界に召喚されて以来、常に持っていたものだった。魔法を見た瞬間に、その構造をコードとして理解できる。それがカイトの異能だった。</p>

<blockquote>「教えてくれ」</blockquote>

<p>カイトが一歩前へ出た。エリナが止めようとするが、カイトは右手で制する。</p>

<blockquote>「この世界の目的は何だ。なぜ俺が召喚された。そして――」</blockquote>

<p>カイトは言葉を切り、マルクスの目を真っ直ぐに見据えた。</p>

<blockquote>「お前は何を守ろうとしている」</code></blockquote>

<p>マルクスの表情に、微かな変化が現れた。一瞬だけ、その冷徹な仮面が揺らいだように見えた。</p>

<blockquote>「……君は面白いな、カイト」</blockquote>

<p>マルクスは静かに息を吐いた。</p>

<blockquote>「ほかの誰も問わなかったことを、君は問う。彼らはただ、目の前の敵を倒すことしか考えなかった。だが、君は――コードを読もうとする」</blockquote>

<p>マルクスがゆっくりと手をかざすと、空中に複雑な幾何学模様が浮かび上がった。それは、カイトが今まで見たことのないほど高度な魔法陣だった。複数のレイヤーが重なり合い、互いに影響を与え合っている。</p>

<blockquote>「見ろ。これが、この世界の基盤コードだ」</blockquote>

<p>カイトは息を呑んだ。その魔法陣は、彼が知っているどのプログラミング言語とも異なっていたが、同時に、どの言語よりも美しく構造化されていた。変数、関数、クラス、ループ――すべての概念が、魔力の形で表現されている。</p>

<pre><code>
class World {
    constructor(creator) {
        this.manaCore = new ManaCore();
        this.inhabitants = [];
        this.timeline = new Timeline();
        this.purpose = "STABILITY_MAINTENANCE";
    }
    
    cycle() {
        this.manaCore.distribute();
        this.timeline.advance();
        this.checkConvergence();
    }
}
</code></pre>

<p>カイトの脳内で、その魔法陣がコードに変換されていく。彼の能力が、目の前の構造を解析していた。</p>

<blockquote>「世界は『安定維持』のために創られた？」</blockquote>

<p>カイトが問うと、マルクスは頷いた。</p>

<blockquote>「そうだ。かつて、この世界の外側には無数の次元が存在した。しかし、次元間の摩擦により、魔力の乱流が生まれた。それが、すべてを滅ぼす『大崩壊』を引き起こしたのだ」</blockquote>

<p>マルクスの手が動き、魔法陣の一部が赤く染まる。</p>

<blockquote>「創造主たちは、生き残りのためにこの世界を構築した。魔力を循環させ、安定させ、二度と崩壊を起こさないための要塞として」</code></blockquote>

<p>セシリアが、不安げにカイトのそばに寄った。</p>

<blockquote>「それって……私たちは、檻の中に閉じ込められているってこと？」</blockquote>

<p>マルクスは首を横に振った。</p>

<blockquote>「檻ではない。船だ。次元の海を渡る、最後の船」</blockquote>

<p>その言葉の意味を理解するまで、カイトは数秒を要した。この世界は、滅びた次元群からの避難所。そして、住人たちは、その事実を知らされずに生きている。</p>

<blockquote>「なぜ隠していたんだ」</blockquote>

<p>カイトの声に、抑えきれない怒りが滲んだ。</p>

<blockquote>「なぜ真実を隠して、偽りの平和を続けてきた」</blockquote>

<p>マルクスは、苦しげに目を伏せた。</p>

<blockquote>「Because the truth……breaks the code」</blockquote>

<p>その言葉に、カイトははっとした。プログラマーの直感が、その意味を理解する。</p>

<pre><code>
if (inhabitants.knowTruth()) {
    world.stability -= CRITICAL_AMOUNT;
    if (world.stability < THRESHOLD) {
        world.collapse();
    }
}
</code></pre>

<p>住人が真実を知れば、世界の安定性が損なわれる。魔力循環システムは、住人の認識とリンクしているのだ。彼らが世界を「安定した場所」として信じることで、システムが機能する。</p>

<blockquote>「それで……収束者として、真実を知る者を消してきたのか」</blockquote>

<p>リアムが、震える声で問うた。彼の剣を握る手が、白くなるほど強く握りしめられていた。</p>

<blockquote>「違う」</blockquote>

<p>マルクスは静かに否定した。</p>

<blockquote>「わたしは消していない。『統合』しているのだ」</blockquote>

<p>その瞬間、カイトの視界が揺れた。無数の声が、彼の脳内に響く。</p>

<ul>
<li>『これは何……私の記憶じゃない』</li>
<li>『助けて、誰か――』</li>
<li>『真実を知ってしまった、もう戻れない』</li>
<li>『マルクス、君は間違っている』</li>
</ul>

<p>カイトは膝をついた。頭が割れるような感覚。それは、膨大な量の記憶が、彼の中に流れ込んでくる感覚だった。</p>

<blockquote>「カイト！」</blockquote>

<p>エリナが駆け寄り、彼の肩を支えた。</p>

<blockquote>「何をしたの！」</blockquote>

<p>エリナがマルクスを睨みつける。しかし、マルクスは冷静だった。</p>

<blockquote>「わたしの役割は、真実を知り、動揺した魂を『統合』すること。彼らの記憶と魔力を、世界のコアに還すのだ。そうすることで、システムの安定性を保つ」</blockquote>

<p>カイトは荒い息を吐きながら、立ち上がった。頭の中の声はまだ残っているが、少しずつ整理されつつあった。</p>

<blockquote>「お前は……何百年も、そうしてきたのか」</blockquote>

<p>マルクスは頷いた。</p>

<blockquote>「わたしは、最初の収束者ではない。この役割は、代々受け継がれてきた。真実を知り、それでも世界を守ることを選んだ者たちの、連鎖だ」</blockquote>

<p>カイトは、マルクスの瞳の奥にある深い悲しみに気づいた。彼は敵ではない。彼もまた、この歪んだシステムの犠牲者の一人なのだ。</p>

<blockquote>「マルクス」</blockquote>

<p>カイトは、前へと歩み出た。</p>

<blockquote>「俺は、お前の敵じゃない。この世界を壊すつもりもない」</blockquote>

<p>マルクスの眉が、わずかに動いた。</p>

<blockquote>「だが、俺はプログラマーだ。バグを見つけたら、放っておけない性分なんだよ」</blockquote>

<p>カイトは、自分の右手をかざした。その掌に、青白い魔力が集まり始める。</p>

<blockquote>「この世界のシステムには、致命的なバグがある。真実を知ったら崩壊するなんて、不安定すぎる」</blockquote>

<pre><code>
class World {
    // NEW: Add stability layer
    addStabilityLayer() {
        this.truthBuffer = new MemoryBuffer();
        this.emotionHandler = new EmotionHandler();
    }
    
    // NEW: Override collapse check
    checkConvergence() {
        if (this.inhabitants.knowTruth()) {
            this.truthBuffer.absorb();
            this.emotionHandler.process();
            // Do NOT collapse
        }
    }
}
</code></pre>

<p>カイトの魔力が、空中に複雑な幾何学模様を描き始めた。それは、マルクスが見せた世界の基盤コードと共鳴し、新たなレイヤーを追加しようとしていた。</p>

<blockquote>「カイト、何をしているの？」</blockquote>

<p>セシリアが、驚いたように問うた。</p>

<blockquote>「パッチを当てているんだ。この世界のシステムに」</blockquote>

<p>カイトの額に、汗が滲む。世界のコードに干渉するなど、彼がこれまで試みたことのない領域だった。しかし、今の彼には、マルクスを通じて流れ込んできた膨大な記憶がある。先人たちが積み上げてきた知識と経験が、彼を支えていた。</p>

<blockquote>「やめろ！それは――」</blockquote>

<p>マルクスが初めて、動揺した様子を見せた。</p>

<blockquote>「君の魔力回路が焼き切れる！世界のコードへの直接干渉は、自殺行為だ！」</blockquote>

<p>カイトは薄く笑った。</p>

<blockquote>「プログラマーはね、コードのバグを見つけたら、直さずにはいられないんだよ」</blockquote>

<p>その瞬間、カイトの魔法陣が、世界の基盤コードと完全に同期した。青白い光が遺跡全体を包み込み、空間そのものが振動する。</p>

<hr>

<p>光の中で、カイトは見た。無数の風景が、彼の意識を通り過ぎていく。</p>

<p>創造主たちが、滅びゆく次元群から逃れ、この世界を構築する姿。</p>

<p>最初の収束者が、真実を知り、苦渋の決断を下す姿。</p>

<p>数えきれないほどの魂が、システムに統合され、世界の一部となっていく姿。</p>

<p>そして――</p>

<blockquote>『カイト』</blockquote>

<p>声がした。カイトは、その声に聞き覚えがあった。</p>

<blockquote>『ヴェリディアン……？』</blockquote>

<p>古代の魔導士の声が、彼の意識の中に響く。</p>

<blockquote>『君は、正しい道を歩んでいる。だが、気をつけろ。システムへの干渉は、創造主たちの残したセキュリティをトリガーする可能性がある』</blockquote>

<p>カイトは眉をひそめた。</p>

<blockquote>『セキュリティ？』</blockquote>

<blockquote>『世界を守るための、最後の砦だ。それが起動すれば……この世界そのものが、リセットされる』</blockquote>

<p>カイトは息を呑んだ。リセット。つまり、すべての記憶、すべての生命が、初期化されるということだ。</p>

<blockquote>『どうすればいい』</blockquote>

<p>ヴェリディアンの声は、静かだった。</p>

<blockquote>『選択肢は二つ。干渉をやめ、現状を維持するか。あるいは――』</blockquote>

<blockquote>『あるいは？』</blockquote>

<blockquote>『セキュリティを、説得するかだ』</blockquote>

<p>カイトは、光の中で笑った。</p>

<blockquote>『説得、か。それなら、俺の得意分野だ』</blockquote>

<hr>

<p>光が収まると、カイトは膝をついていた。体中の魔力が、完全に消耗していた。</p>

<blockquote>「カイト！」</blockquote>

<p>エリナ、リアム、セシリアの三人が、彼の周りに駆け寄る。</p>

<blockquote>「大丈夫か？」</blockquote>

<p>リアムが、カイトの腕を貸した。</p>

<blockquote>「ああ……なんとか」</blockquote>

<p>カイトは立ち上がり、マルクスを見た。収束者は、驚愕した表情でカイトを見つめていた。</p>

<blockquote>「君は……何をした」</blockquote>

<p>カイトは、自分の手を見つめた。その掌には、以前とは違う、微かな光が残っていた。</p>

<blockquote>「パッチの一部を当てた。これで、真実を知っても即座に崩壊することはない」</blockquote>

<p>マルクスは、目を見開いた。</p>

<blockquote>「……本当に？」</blockquote>

<p>カイトは頷いた。</p>

<blockquote>「ただし、完全じゃない。俺の魔力じゃ、世界全体のコードを書き換えるには足りなかった」</blockquote>

<p>カイトは、マルクスに手を差し出した。</p>

<blockquote>「協力してくれ、マルクス。お前の力があれば、この世界を『リセット』なしで救えるかもしれない」</blockquote>

<p>マルクスは、カイトの手を見つめた。数百年にわたる孤独な任務。誰にも理解されず、ただ世界を守り続けてきた日々。その終わりが、目の前にあるかもしれなかった。</p>

<p>ゆっくりと、マルクスはカイトの手を握った。</p>

<blockquote>「……信じていいのか、君を」</blockquote>

<p>カイトは、真っ直ぐにマルクスの目を見た。</p>

<blockquote>「信じてくれ。俺たちは、同じコードの一部なんだから」</blockquote>

<p>その言葉に、マルクスは初めて、微かな笑みを浮かべた。</p>

<p>崩れかけた遺跡の天井から、光が差し込んでいた。新しい章の始まりを告げるかのように、その光は、二人の男を優しく包み込んでいた。</p>

<p>だが、カイトの心の奥で、ヴェリディアンの警告が反響していた。セキュリティ。創造主たちの残した、最後の砦。それが何なのか、まだ彼は知らない。</p>

<p>パッチを当てたことで、そのセキュリティはすでに起動しているのかもしれない。あるいは、まだ眠っているのかもしれない。</p>

<p>いずれにせよ、カイトと仲間たちの旅は、まだ終わらない。世界の真実を知り、それでもなお、希望を信じる彼らの戦いは、ここからが本番だった。</p>
