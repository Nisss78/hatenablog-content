<h3>第4章：魔法の本質</h3>

<p>カイトはギルドの酒場でエリナと向かい合って座っていた。朝日が窓から差し込み、埃がきらきらと舞っている。リアムはすでに次の仕事を探しに出かけていた。</p>

<p>「ねえ、カイト」エリナが低い声で言った。「昨日の魔法、あれって......本当にこの世界のものじゃないよね」</p>

<blockquote>「気づいたか」</blockquote>

<p>カイトはコーヒーを飲み干してから答えた。この世界のコーヒーは少しスパイシーで、目が覚める味がした。</p>

<blockquote>「スキルで実装できる魔法には限界があるんだ。MP消費、詠唱時間、属性の制約......でも、僕のやり方は違う。マナを直接操作して、論理的に組み立てる」</blockquote>

<p>エリナは身を乗り出した。</p>

<blockquote>「それって、危険なんじゃない？」</blockquote>

<p>カイトは肩をすくめた。</p>

<blockquote>「かもしれない。でも、そうしないと生き残れない。この世界は思ったより厳しい」</blockquote>

<hr>

<p>二人は街の東にある「霧の森」に向かっていた。依頼内容はシンプルだ——行方不明になった薬草師を探すこと。報酬は銀貨50枚。初心者向けの依頼としては悪くない。</p>

<p>森の入り口に立つと、濃い霧が立ち込めていた。普通の人間なら数メートル先も見えないだろう。だが、カイトには違って見えた。</p>

<p><code class="language-javascript">const detectLife = () => {
  const mana = senseMana(100);
  return mana.filter(m => m.type === 'living');
};</code></p>

<p>彼は意識の奥でその関数を実行した。マナの流れが可視化される——まるでコードのデバッガのように。森の中に数個の生体反応。小さな動物たちだろう。そして、もっと奥に......</p>

<blockquote>「誰かいる」</blockquote>

<p>カイトは静かに言った。エリナが剣の柄に手をかける。</p>

<blockquote>「薬草師さん？」</blockquote>

<p>返事はない。だが、マナの動きが変わった。何かが警戒している。</p>

<hr>

<p>霧の中から現れたのは、薬草師ではなかった。</p>

<p>背の高い女性。黒いローブ。手には奇妙な杖——先端に赤い宝石が埋め込まれている。その宝石から、不気味なマナの波動が放たれていた。</p>

<blockquote>「冒险者か」</blockquote>

<p>女性はつまらなそうに言った。</p>

<blockquote>「引きなさい。この森は私の実験場だ」</blockquote>

<p>実験場？ カイトは眉をひそめた。</p>

<blockquote>「薬草師はどうなりました？」</blockquote>

<p>女性は冷たく笑った。</p>

<blockquote>「彼女？ 実験の被検体として提供してもらったわ。文句があるなら、力ずくで止めてみたらどう？」</blockquote>

<p>エリナが剣を抜いた。</p>

<blockquote>「カイト、この女......やるわよ」</blockquote>

<p>カイトは女性の杖を観察した。あの宝石——マナを増幅する装置か、それとも......?</p>

<p><code class="language-javascript">analyze artifact {
  type: 'amplifier',
  source: 'ancient',
  risk: 'high'
}</code></p>

<p>解析結果が頭の中に浮かぶ。古代の遺物。危険度：高。この世界の技術レベルを超えている。</p>

<blockquote>「エリナ、待って」</blockquote>

<p>カイトは女性に向かって言った。</p>

<blockquote>「その杖、どこで手に入れた？」</blockquote>

<p>女性の目が細められた。</p>

<blockquote>「......あなた、普通の冒険者じゃないわね」</blockquote>

<p>霧が濃くなった。いや、女性がマナを操っているのだ。</p>

<blockquote>「私の名はセシリア。賢者の塔の研究員よ」</blockquote>

<p>賢者の塔——カイトはギルドで聞いたことがあった。この世界の魔法研究の中心地。最高位の魔術師たちが集う場所。</p>

<blockquote>「研究員が人を誘拐するのか？」</blockquote>

<blockquote>「必要な犠牲よ」セシリアは冷淡に言った。「より良い魔法を作るには、人体への影響を調べなければならない」</blockquote>

<p>カイトは息を吐いた。話は通じそうにない。</p>

<p><code class="language-javascript">prepare combat {
  shields: 3,
  offenseType: 'lightning',
  strategy: 'defensive'
}</code></p>

<p>彼はマナを練り始めた——まだ見ぬ敵に備えて。</p>

<hr>

<p>戦いは一瞬だった。</p>

<p>セシリアが杖を振ると、霧が鋭い刃となって襲ってきた。エリナがそれを剣で弾く。カイトは別の角度から霧の刃を防ぐ盾を展開した。</p>

<p><code class="language-javascript">shield.on('impact', (damage) => {
  redistribute(damage);
});</code></p>

<p>セシリアの目に驚きの色が浮かんだ。</p>

<blockquote>「その詠唱......何？ まるで古代語みたいに聞こえるけれど、全く違う」</blockquote>

<p>カイトは答えなかった。代わりに、攻撃のコードを書いた。</p>

<p><code class="language-javascript">cast('lightning_bolt', {
  target: enemy.weakness,
  power: 70,
  speed: 'instant'
});</code></p>

<p>雷が指先から放たれる。セシリアは間一髪で回避したが、その一瞬の隙をエリナが見逃さなかった。剣がセシリアのローブを切り裂く。</p>

<blockquote>「くっ......！」</blockquote>

<p>セシリアは霧の中に身を隠した。逃げるつもりだ。</p>

<blockquote>「逃がさない！」</blockquote>

<p>エリナが追おうとしたが、カイトが止めた。</p>

<blockquote>「待て。罠かもしれない」</blockquote>

<p>その時、霧の奥から声が聞こえた。</p>

<blockquote>「た、すけて......」</blockquote>

<p>薬草師だ。彼女は木の根元に倒れていた。意識はあるが、体が動かないようだ。</p>

<hr>

<p>一時間後、彼らは薬草師を連れて街に戻っていた。セシリアは逃げたが、依頼は達成された。報酬を受け取り、三人は再び酒場にいた。</p>

<blockquote>「あの女、賢者の塔の人間だったわね」</blockquote>

<p>エリナが深刻な表情で言った。</p>

<blockquote>「厄介なことになった。塔は権力がある。下手に動くと、ギルドごと潰されるかも」</blockquote>

<p>リアムがテーブルに戻ってきた。情報収集を終えたらしい。</p>

<blockquote>「最悪のタイミングだが、話がある」</blockquote>

<p>彼は声を落とした。</p>

<blockquote>「賢者の塔が、『異世界の召喚者』を探しているらしい」</blockquote>

<p>カイトの心臓が跳ねた。</p>

<blockquote>「......誰のことだ？」</blockquote>

<p>リアムはまっすぐカイトを見た。</p>

<blockquote>「お前のことだよ、カイト。塔の連中は気づいている——この世界に、本来いるはずのない人間がいることを」</blockquote>

<p>酒場の喧騒が、急激に遠く感じられた。</p>

<p>カイトはコーヒーの残りを飲み干した。冷めていて、苦かった。</p>

<p><em>見つかったか。</em></p>

<p><em>早すぎる。</em></p>
