<h3>第7章：古代の魔法陣</h3>

<hr>

<p>翌朝。</p>

<p>俺はセシリアから受け取った地図を広げていた。</p>

<p>「この場所、どこですか？」</p>

<hr>

<p>「王国の北東、エルダー森林の奥地です」セシリアが答えた。「そこに、古代の遺跡があるんです」</p>

<p>「遺跡？」</p>

<p>「はい。『アーケイン・ルインズ』と呼ばれている場所です。何百年も前に、強大な魔法使いが住んでいたと言われています」</p>

<hr>

<p>エリナが覗き込んできた。</p>

<p>「カイトさん、そこに行くんですか？」</p>

<p>「......そうみたいだ」俺は言った。「セシリアの依頼で、遺跡の調査をすることになった」</p>

<p>「私も行きます！」エリナが即座に言った。</p>

<hr>

<p>「危ないよ」</p>

<p>「カイトさん一人の方が危ないです！ 私がサポートします！」</p>

<p>俺は少し笑った。</p>

<p>「......分かった。一緒に行こう」</p>

<hr>

<h4>出発</h4>

<p>俺とエリナは、王都の北門を出た。</p>

<p>馬を借りて、エルダー森林に向かう。セシリアは王都に残ると言った。</p>

<p>「遺跡で何か見つけたら、すぐに連絡してください」セシリアが言った。「この通信玉を使ってください」</p>

<hr>

<p>俺は、小さな水晶球を受け取った。</p>

<p>「これ、どう使うの？」</p>

<p>「魔力を込めると、私の水晶球と繋がります。音声と映像、両方送れます」</p>

<p>「......便利だな」</p>

<hr>

<p>馬で数時間。エルダー森林に到着した。</p>

<p>森の中は薄暗く、不気味な雰囲気が漂っていた。</p>

<p>「......何か、変な感じがする」エリナが呟いた。</p>

<hr>

<p>「俺も感じる」俺は言った。「魔力が......乱れてる」</p>

<p>「魔力が？」</p>

<p>「ああ。普通の魔力とは違う。何か、歪んでるというか......バグってるというか」</p>

<hr>

<p>「バグ？」エリナが首を傾げた。</p>

<p>「......俺の世界の言葉だ。エラー、みたいな意味」</p>

<p>「なるほど......」</p>

<hr>

<p>俺たちは森の中を進んだ。</p>

<p>しばらくすると、開けた場所に出た。</p>

<p>そこにあったのは——</p>

<hr>

<p>「これが......アーケイン・ルインズ」</p>

<p>巨大な石造りの建物が、苔に覆われて立っていた。何百年も前の遺跡だ。</p>

<p>入り口には、複雑な模様が彫り込まれていた。</p>

<hr>

<p>「魔法陣......」エリナが呟いた。</p>

<p>「これ、何かの魔法陣？」</p>

<p>「はい。でも、見たことない形です」</p>

<hr>

<p>俺は魔法陣に近づいた。</p>

<p>模様を見る。線と円と、複雑な図形。</p>

<p>そして——</p>

<hr>

<p>俺には、それが「コード」に見えた。</p>

<p>「......これは」</p>

<hr>

<p>俺は目を凝らした。</p>

<pre><code>
function seal(target) {
  const barrier = createBarrier({
    element: "time",
    duration: Infinity,
    strength: 1000
  });
  barrier.lock(target);
  return barrier;
}
</code></pre>

<p>「......タイム・シール？」俺は呟いた。</p>

<hr>

<p>「カイトさん？」エリナが不思議そうに見てきた。</p>

<p>「これ、魔法陣じゃない」俺は言った。「......コードだ」</p>

<p>「コード？」</p>

<p>「古代の魔法使いは、魔法を......プログラムとして書いてたんだ」</p>

<hr>

<p>俺は震えた。</p>

<p>この遺跡には、俺と同じ「コード・メイジ」が住んでいたのかもしれない。</p>

<p>そして、その魔法は——</p>

<hr>

<p>「......何かを封印してる」俺は言った。</p>

<p>「封印？」</p>

<p>「ああ。この魔法陣は、時間を操るバリアだ。何かを、永遠に閉じ込めてる」</p>

<hr>

<p>エリナの顔が青ざめた。</p>

<p>「じゃあ、開けちゃダメですよね？」</p>

<p>「......当然だ」俺は頷いた。「でも、調査するだけなら大丈夫だと思う」</p>

<hr>

<p>俺たちは、遺跡の中に入った。</p>

<p>暗い廊下。壁には、たくさんの魔法陣が描かれている。</p>

<p>どれも、俺には「コード」として理解できた。</p>

<hr>

<pre><code>
// Fireball spell
function fireball(target, power) {
  const fire = createElement("fire");
  fire.temperature = 800 + power * 10;
  fire.launch(target);
}
</code></pre>

<pre><code>
// Healing spell
function heal(target, amount) {
  const life = createElement("life");
  life.flow(target, amount);
}
</code></pre>

<pre><code>
// Shield spell
function shield(duration) {
  const barrier = createBarrier({
    element: "physical",
    duration: duration
  });
  return barrier;
}
</code></pre>

<p>「......すごい」俺は呟いた。</p>

<hr>

<p>「カイトさん、何がすごいんですか？」エリナが聞いた。</p>

<p>「この遺跡は......魔法の図書館だ」俺は言った。「古代の魔法使いが、何百種類もの魔法を記録してる」</p>

<hr>

<p>「それって......」エリナの目が輝いた。「カイトさん、全部読めるんですか？」</p>

<p>「......たぶん」俺は頷いた。「でも、時間がかかる。何百種類もある」</p>

<hr>

<p>俺たちは、遺跡の奥に進んだ。</p>

<p>そして——</p>

<hr>

<p>最深部に、一冊の本があった。</p>

<p>机の上に置かれた、古びた本。</p>

<p>表紙には、こう書かれていた——</p>

<hr>

<p><strong>『Code Mage Manual』</strong></p>

<p><strong>（コード・メイジ・マニュアル）</strong></p>

<hr>

<p>俺は、震える手で本を開いた。</p>

<hr>
