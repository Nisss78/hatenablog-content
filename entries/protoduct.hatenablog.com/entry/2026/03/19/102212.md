<h3>第8章：コード・メイジ・マニュアル</h3>

<hr>

<p>俺は、震える手で本を開いた。</p>

<p>『Code Mage Manual』——コード・メイジ・マニュアル。</p>

<p>表紙をめくると、手書きの文字が並んでいた。</p>

<hr>

<pre><code>
Introduction

This manual is written for those who can perceive magic as code.
You are not alone. Throughout history, there have been others like you.
We call ourselves "Code Mages".

What makes us different:
- We see magic's underlying structure
- We can modify spells at the source level
- We can create new magic from scratch
- We don't need chants or incantations

But with great power comes great responsibility.
This manual will teach you how to use your abilities wisely.

- Archmage Veridian, Year 847
</code></pre>

<hr>

<p>「アークメイジ・ヴェリディアン......」俺は呟いた。</p>

<p>847年。今から何百年も前だ。</p>

<hr>

<p>「カイトさん、何が書いてあるんですか？」エリナが覗き込んできた。</p>

<p>「......俺と同じ能力を持っていた、古代の魔法使いの手記だ」俺は言った。</p>

<hr>

<h4>マニュアルの内容</h4>

<p>俺はページをめくった。</p>

<p>マニュアルは、いくつかの章に分かれていた。</p>

<hr>

<p><strong>Chapter 1: Basic Spells</strong></p>

<pre><code>
// Light spell - fundamental for all mages
function light(brightness, duration) {
  const photons = createElement("light");
  photons.intensity = brightness;
  photons.duration = duration;
  return photons;
}

// Usage: light(100, 60) // 100 lumens for 60 seconds
</code></pre>

<hr>

<p><strong>Chapter 2: Elemental Magic</strong></p>

<pre><code>
// Fireball - offensive spell
function fireball(power, speed) {
  const fire = createElement("fire");
  fire.temperature = 500 + power * 100;
  fire.velocity = speed;
  fire.size = power * 10;
  return fire;
}

// Ice shard - offensive spell
function iceShard(count, sharpness) {
  const ice = createElement("ice");
  ice.count = count;
  ice.sharpness = sharpness;
  return ice;
}
</code></pre>

<hr>

<p><strong>Chapter 3: Defensive Magic</strong></p>

<pre><code>
// Magic shield - basic defense
function shield(type, strength, duration) {
  const barrier = createBarrier({
    element: type,
    strength: strength,
    duration: duration
  });
  return barrier;
}

// Types: "physical", "magical", "elemental"
</code></pre>

<hr>

<p><strong>Chapter 4: Advanced Techniques</strong></p>

<pre><code>
// Spell combination - combine multiple spells
function combine(spells) {
  const combined = merge(spells);
  combined.efficiency = calculateEfficiency(spells);
  return combined;
}

// Example: fireball + wind = flame tornado
</code></pre>

<hr>

<p>「......すごい」俺は呟いた。</p>

<p>このマニュアルには、何十種類もの魔法がコードとして書かれている。</p>

<p>しかも、それぞれに詳細なコメントと使用例がある。</p>

<hr>

<p>「カイトさん、これ......全部読めるんですか？」エリナが聞いた。</p>

<p>「ああ。俺には、これが普通の文章みたいに読める」俺は言った。</p>

<hr>

<p>エリナの目が輝いた。</p>

<p>「じゃあ、カイトさんは......もっと強くなれるんですか？」</p>

<p>「......たぶん」俺は頷いた。「このマニュアルがあれば、もっと複雑な魔法も使えるようになるかもしれない」</p>

<hr>

<h4>最後の章</h4>

<p>俺は、マニュアルの最後の章を開いた。</p>

<p><strong>Chapter 10: The Truth of Code Mages</strong></p>

<hr>

<pre><code>
The Truth

Code Mages are not simply mages who can see magic as code.
We are the original creators of the magical system itself.

Long ago, magic was chaotic and uncontrollable.
The first Code Mages discovered that magic follows logical patterns.
They created the "Magical Framework" - a system that allows
ordinary mages to use magic through chants and incantations.

The chants are simply function calls.
The gestures are parameters.
The magical circles are interfaces.

You, as a Code Mage, can bypass all of this.
You can access the source code directly.
You can modify the framework itself.

But be warned:
If the framework is corrupted, all magic in the world will collapse.
Use your power wisely.

- Archmage Veridian
</code></pre>

<hr>

<p>俺は息を飲んだ。</p>

<p>「......魔法のフレームワーク」</p>
