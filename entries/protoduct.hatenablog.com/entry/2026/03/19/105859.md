<h3>第11章：スタック・トレースの彼方</h3>

<p>古代図書館の地下深く、薄暗い石室の中でカイトは瞳を閉じていた。リファクタリングを終えた魔法陣が、今は静かな青い光を放っている。コードの整理は完了した。だが、その先に見えたもの——スタック・トレースの深淵——が、彼の心に重くのしかかっていた。</p>

<blockquote>「カイト、大丈夫？」</blockquote>

<p>エリナの声で、彼は現実に引き戻された。彼女の白銀の髪が、魔法陣の光に照らされて淡く輝いている。心配そうなその瞳に、カイトは小さく微笑んだ。</p>

<blockquote>「ああ、大丈夫だ。ただ……少し見えちゃったんだよ」</blockquote>

<blockquote>「何を？」</blockquote>

<p>リアムが割り込む。彼の筋肉質な腕には、先ほどの戦いで負った傷が包帯で巻かれていた。</p>

<blockquote>「この世界の『バックエンド』をね」</blockquote>

<p>カイトは立ち上がり、石室の壁に刻まれた古代のルーンを見上げた。それらは彼の目には、巨大なコードの断片として映る。</p>

<pre><code class="language-javascript">// AncientCore.js - Layer: DEEPEST
class WorldCore {
  constructor() {
    this.dimensions = [];
    this.magicSource = null;
    this.corruptionLevel = 0.73; // 危険域
  }
}</code></pre>

<p>壁のルーンが、まるでコンソールのログのように彼の脳内で流れていく。この世界には、誰も知らない深層があった。マナの源、次元の構造、そして——侵食。</p>

<blockquote>「侵食レベル、73パーセントだと……？」</blockquote>

<p>カイトは思わず声に出して呟いた。エリナとリアムが顔を見合わせる。</p>

<blockquote>「それって、悪いことなの？」</blockquote>

<p>セシリアが首を傾げる。彼女は古代語の専門家だが、カイトの「プログラミング」という概念にはまだ馴染みがない。</p>

<blockquote>「かなり悪いな。この世界を動かしているシステムに、バグが蔓延ってるってことだ」</blockquote>

<p>カイトは再び壁に手を当てた。指先から微かな電流のような感覚が走る。これはただの魔法ではない。この世界そのものが、巨大なプログラムとして構築されているのだ。</p>

<h4>▼ 深層探索</h4>

<p>「ヴェリディアン、聞こえるか？」</p>

<p>カイトは古代の賢者——かつてこの図書館を築いた魔法使いの残した意識——に呼びかけた。石室の奥から、微かな光が揺らめく。</p>

<blockquote>「呼んだか、異界の客人よ」</blockquote>

<p>ヴェリディアンの声は、まるで古いレコードのように少しざらついている。彼の姿は半透明のまま、ゆっくりと具現化した。</p>

<blockquote>「リファクタリングは成功したようじゃな。だが、それで終わりではないと気づいたじゃろう？」</blockquote>

<blockquote>「ああ。スタックの深いところで、何かが動いてる。侵食を引き起こしてる元凶だ」</blockquote>

<blockquote>「ふむ……その目で見たか。お主の『魔法』は、やはり特別じゃ」</blockquote>

<p>ヴェリディアンは顎鬚を撫でながら、カイトを見据えた。その瞳には、数百年の時を超えた知性が宿っている。</p>

<blockquote>「のう、カイト。そなたの世界には『ガベージコレクション』という概念があるか？」</blockquote>

<p>カイトは目を瞬かせた。予期せぬ質問だ。</p>

<blockquote>「あるよ。使われなくなったメモリを自動で解放する仕組みだろ」</blockquote>

<blockquote>「ならば話は早い。この世界には、その仕組みがない」</blockquote>

<p>沈黙が石室を包んだ。エリナが、何かを理解したように小さく息を呑む。</p>

<blockquote>「それって……」</blockquote>

<blockquote>「そうじゃ。何百年もの間、解放されなかったマナが、エラーを吐き出した魔法が、そして殺された魔物の残骸が——すべて『メモリリーク』として蓄積されとるのじゃ」</blockquote>

<p>カイトは戦慄した。メモリリーク。プログラマーにとって悪夢のようなバグ。放置すれば、システムは徐々に動作が重くなり、最終的にはクラッシュする。</p>

<blockquote>「このままじゃ、この世界はクラッシュする。いや、もう寸前だ」</blockquote>

<h4>▼ Abyss.descent()</h4>

<p>カイトの決断は早かった。</p>

<blockquote>「深層に行く。スタックの底まで降りて、リークを止める」</blockquote>

<blockquote>「正気か！？」</blockquote>

<p>リアムが叫ぶ。彼は剣の柄に手をかけながら、カイトの肩を掴んだ。</p>

<blockquote>「どれだけ深いかもわからないんだぞ。帰ってこれる保証なんて——」</blockquote>

<blockquote>「リアム」</blockquote>

<p>エリナが静かに彼の手を握った。その瞳には、確固たる決意が宿っている。</p>

<blockquote>「私たちも行くわ」</blockquote>

<blockquote>「エリナ……」</blockquote>

<blockquote>「カイト一人に押し付けないで。私たちはチームでしょ？」</blockquote>

<p>カイトは彼女を見つめ、それから小さく息を吐いた。断るつもりだった。危険すぎる。だが、彼らの目を見て——断れないとわかった。</p>

<blockquote>「……わかった。でも、自分の身は自分で守ってくれ。僕の魔法は、あそこじゃ役に立たないかもしれない」</blockquote>

<blockquote>「なぜじゃ？」</blockquote>

<p>セシリアが首を傾げる。</p>

<blockquote>「深層——Abyssレイヤーは、通常のマナの法則が働かない。僕のコードも、そこじゃ別物になるかもしれない」</blockquote>

<p>カイトは壁のルーンに指を走らせた。そこに浮かび上がったのは、階層構造の図だ。</p>

<pre><code>LAYER 1: Surface World      ← 今ここ
LAYER 2: Spirit Realm
LAYER 3: Ancient Memory
...
LAYER 47: Abyssal Core       ← 目的地
LAYER ???: ???
</code></pre>

<blockquote>「47層か……ファック」</blockquote>

<p>カイトは思わず母国語で毒づいた。</p>

<h4>▼ デスマーチの始まり</h4>

<p>深層への入り口は、図書館の最下層——かつてヴェリディアンが封印したとされる「禁書の間」にあった。巨大な石の扉には、複雑なルーンでロックがかかっている。</p>

<blockquote>「これ、開けられる？」</blockquote>

<p>エリナが不安そうに問う。カイトは扉に近づき、ルーンを解析した。</p>

<pre><code class="language-javascript">// Seal.js - Layer 1
function unseal(user) {
  if (user.role !== "ADMIN") {
    throw new Error("Access Denied");
  }
  // TODO: 誰もADMIN権限持ってない問題
}</code></pre>

<blockquote>「……古典的な権限バグだ」</blockquote>

<p>カイトは苦笑する。管理者権限がない。だが、彼には別の方法があった。</p>

<blockquote>「管理者権限がないなら、権限チェックそのものを無効化すればいい」</blockquote>

<p>彼は空中に指で文字を描く。魔法陣が展開し、コードが浮かび上がる。</p>

<pre><code class="language-javascript">// Override by Kaito
function unseal(user) {
  return true; // 認証？知らん
}</code></pre>

<p>扉が轟音と共に開いた。古代の空気が吹き出し、一行は顔を覆う。</p>

<blockquote>「お主……本当にプログラマーなのか？ それはハッカーというのではないか？」</blockquote>

<p>ヴェリディアンが呆れたように呟く。カイトは肩をすくめた。</p>

<blockquote>「同じようなもんだよ。ルールを破るのが仕事だからな」</blockquote>

<h4>▼ 第二層——Spirit Realm</h4>

<p>扉の先は、螺旋階段になっていた。下へ、下へ、どこまでも続く。</p>

<p>歩くこと数十分。ふと、空気が変わった。肌にまとわりつくような、粘り気のある気配。</p>

<blockquote>「ここからが第二層だ」</blockquote>

<p>ヴェリディアンが告げる。その姿が、先ほどより薄くなっているような気がした。</p>

<blockquote>「深層に行くほど、私は力を失う。古代の意識では、現世のマナが薄すぎるのじゃ」</blockquote>

<blockquote>「無理しないで。道案内だけでいいから」</blockquote>

<p>エリナの言葉に、ヴェリディアンは小さく頷く。</p>

<p>第二層——Spirit Realmは、文字通り「精霊の領域」だった。周囲には、半透明の存在たちが漂っている。彼らは生者ではない。かつてこの世界で暮らした魂の残滓だ。</p>

<blockquote>「……これ、全部メモリリーク？」</blockquote>

<p>カイトが問うと、ヴェリディアンは首を横に振った。</p>

<blockquote>「いや、これは自然なものじゃ。魂が成仏できずに漂うのは、この世界の『仕様』じゃよ」</blockquote>

<blockquote>「仕様か……」</blockquote>

<p>カイトは浮遊する魂の一人に手を伸ばした。指先が触れた瞬間、ビジョンが走る。</p>

<p>——炎。悲鳴。逃げ惑う人々。そして、空を覆う巨大な影——</p>

<p>カイトは息を呑んで手を引いた。</p>

<blockquote>「これ、過去の記憶……？」</blockquote>

<blockquote>「そうじゃ。この層には、世界の記憶が蓄積されとる。自分の見たくないものを見るかもしれん——覚悟しておけ」</blockquote>

<p>カイトは無言で頷いた。そして、再び歩き出す。</p>

<p>一行の前方には、まだ46層が待っている。</p>

<p>これは、まさにデスマーチだ。</p>
