<h3>第5章：塔の影</h3>

<p>沈黙が酒場の空気を重くした。</p>

<p>カイトは自分の手を見た。普通の若者の手に見える。だが、その奥には——この世界の誰も持たない、異質な知識が眠っている。プログラミング。アルゴリズム。デバッグの概念。それらがこの世界では「魔法」として現れる。</p>

<blockquote>「カイト」</blockquote>

<p>エリナが静かに言った。</p>

<blockquote>「あんた、何か隠してるわね」</blockquote>

<p>彼女の目は鋭かった。冒険者としての勘だろう。カイトは小さく笑った。</p>

<blockquote>「隠してるっていうか......説明するのが難しいんだ」</blockquote>

<p>リアムがテーブルに身を乗り出した。</p>

<blockquote>「まずは状況を整理しよう。賢者の塔が異世界の召喚者を探している——これは確実だ。俺の情報源は信頼できる」</blockquote>

<blockquote>「情報源？」</blockquote>

<p>カイトが聞くと、リアムは少し躊躇った。</p>

<blockquote>「......昔の知り合いだ。塔に勤めていたことがある」</blockquote>

<p>エリナが驚いた顔をした。</p>

<blockquote>「えっ、リアムって塔の出身なの？」</blockquote>

<blockquote>「研究員だった。十年前にな」</blockquote>

<p>リアムは窓の外を見た。街の広場で子供たちが走り回っている。</p>

<blockquote>「辞めた理由は......言わない方がいい。とにかく、塔は今、必死だ。何か大きな実験を計画しているらしい。そのために、異世界の知識が必要なんだと」</blockquote>

<p>カイトの胸に不安が広がった。</p>

<p><code class="language-javascript">const threat = {
  source: 'sage_tower',
  target: 'self',
  riskLevel: 'critical',
  timeFrame: 'unknown'
};</code></p>

<hr>

<p>その夜、カイトは一人で宿屋の部屋にいた。エリナとリアムは別の部屋で休んでいる。</p>

<p>彼はベッドに座り、目を閉じた。マナの流れを感じる——街全体を覆う、薄い膜のようなもの。</p>

<p><code class="language-javascript">scan surroundings {
  range: 'city',
  filter: 'hostile_intent'
}</code></p>

<p>結果は何も返ってこない。今のところ、敵意を持つ存在は近くにいない。</p>

<p>だが、それは安心できることではなかった。</p>

<p><em>賢者の塔。</em></p>

<p><em>異世界の知識。</em></p>

<p><em>彼らは何を知っている？</em></p>

<p>カイトはこの世界に来てから、ずっと一つのことを考えていた——なぜ自分は召喚されたのか？ 誰が？ 何のために？</p>

<p>答えは見つかっていない。だが、一つだけ確かなことがある。この世界の魔法は、彼が知っているプログラミングの概念と奇妙に似ている。変数、関数、条件分岐——それらがマナという形で具現化する。</p>

<p>誰かが、彼の世界の知識を元にこの世界の魔法を設計したのではないか？</p>

<p><code class="language-javascript">hypothesis = {
  origin: 'parallel_world',
  designer: 'unknown',
  purpose: '???'
};</code></p>

<p>仮説はまだ検証できない。情報が足りない。</p>

<p>その時、窓の外で何かが動いた。</p>

<hr>

<p>カイトは瞬時に警戒態勢に入った。</p>

<p><code class="language-javascript">activate alertMode {
  sensoryBoost: true,
  reactionSpeed: 'max'
}</code></p>

<p>窓の方を見る。月明かりが差し込んでいるが、影の中に何かが潜んでいるのがわかった。</p>

<blockquote>「誰だ」</blockquote>

<p>静かに問う。返事はなかった。</p>

<p>だが、影が動いた。</p>

<p>窓ガラスが——内側から、叩かれた。</p>

<p>カイトは立ち上がった。この部屋は二階だ。普通の人間が窓の外に立つことはできない。</p>

<blockquote>「......カイト様ですね」</blockquote>

<p>女性の声。低く、落ち着いている。</p>

<blockquote>「お話があります。窓を開けていただけますか？」</blockquote>

<p>カイトは迷った。だが、敵なら既に攻撃しているはずだ。</p>

<p>彼は窓を開けた。</p>

<p>そこに立っていたのは、黒髪の女性だった。年は三十代前半。青いローブを着ている——賢者の塔の研究員の服だ。だが、セシリアとは違う。彼女の目には、敵意ではなく、疲労と焦りがあった。</p>

<blockquote>「私はアリシア。賢者の塔の......元研究員です」</blockquote>

<p><em>元？</em></p>

<blockquote>「塔を脱走してきました。あなたに伝えなければならないことがあります」</blockquote>

<p>彼女は窓枠に足をかけ、部屋の中に入ってきた。</p>

<blockquote>「賢者の塔が......あなたを探しているのは、実験のためではありません」</blockquote>

<p>カイトは眉をひそめた。</p>

<blockquote>「どういうことだ？」</blockquote>

<p>アリシアは深く息を吸った。</p>

<blockquote>「塔の長官が......あなたと同じ世界の出身なのです」</blockquote>

<p>言葉が、カイトの思考を凍りつかせた。</p>

<blockquote>「......何だって？」</blockquote>

<blockquote>「三十年前、彼もまた召喚されたのです。あなたの世界から」</blockquote>

<p>アリシアの目が揺れた。</p>

<blockquote>「彼は......コウイチ・タナカという名前で、かつては日本人でした」</blockquote>

<hr>

<p>部屋の中に、重い沈黙が落ちた。</p>

<p>カイトは呼吸を整えた。情報が多すぎる。処理しきれない。</p>

<p><code class="language-javascript">try {
  process(newInformation);
} catch (e) {
  console.error('Cognitive overflow');
}</code></p>

<blockquote>「待て」</blockquote>

<p>カイトは言った。</p>

<blockquote>「その話、信じる根拠は？」</blockquote>

<p>アリシアは懐から一枚の写真を取り出した。古びた、セピア色の写真。</p>

<p>そこには、若い日本人男性が写っていた。背後には——東京タワー。</p>

<blockquote>「長官の部屋から盗んだものです。彼は三十年前、渋谷で突然姿を消した。警察は行方不明として処理しましたが......真実は違う」</blockquote>

<p>カイトは写真を受け取った。間違いない。これは日本だ。彼の世界だ。</p>

<blockquote>「なぜ彼は......塔の長官になった？」</blockquote>

<blockquote>「最初は普通の冒険者でした。ですが、彼の魔法は圧倒的だった——この世界の誰も敵わないほどに。彼は急速に力を蓄え、二十年で塔の長官の座についた」</blockquote>

<p>アリシアの声が低くなった。</p>

<blockquote>「そして今、彼は次の召喚を計画しています」</blockquote>

<blockquote>「召喚？」</blockquote>

<blockquote>「あなたを呼び出したのは、偶然ではありません。彼が意図的に行ったのです」</blockquote>

<p>カイトは息を吐いた。</p>

<blockquote>「......なぜだ？」</blockquote>

<p>アリシアは窓の外を見た。</p>

<blockquote>「彼は言っていました——<em>『故郷に帰るには、もう一人の協力者が必要だ』</em>と」</blockquote>

<p>月が雲に隠れた。部屋の中が暗くなる。</p>

<blockquote>「カイト様、あなたは実験体ではありません。彼はあなたを......共犯者として呼んだのです」</blockquote>

<p>カイトの手が、微かに震えた。</p>

<p><em>故郷に帰る。</em></p>

<p><em>日本に——帰れる？</em></p>
