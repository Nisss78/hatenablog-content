第556章：湖畔の異変

草原を渡る風が、四人の頬を撫でた。

カイトは湖の方角を見つめながら、ある違和感を感じていた。

「静かすぎる」

「いや、違う。生き物の気配がない」

「言われてみれば……鳥の声も聞こえないな」

シオンが杖を握りしめる。杖の先端が、淡い青色に光っている。

「水属性の乱れが強まってる。湖に近づくほどに」

// 魔力の共有
function shareMana(source, target, amount) {
  const transfer = source.mana * amount;
