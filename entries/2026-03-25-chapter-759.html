第759章：灼熱の回廊

第二の扉が、重々しく開いた。

中から、目が痛くなるほどの光が溢れ出した。

カイトたちは腕で目を覆った。

「明るすぎる……」

エリナが眩しそうに呟いた。

目が慣れてくると、回廊の様子が見えてきた。

壁も床も天井も、すべてが燃えている。

でも、燃え尽きることはない。永遠に炎が揺らめき続けている。

「灼熱の回廊……名前通りだな」

リアムが熱気に顔をしかめた。

熱耐性魔法のおかげで、直接の熱は感じない。

でも、視覚的な熱波は精神を削ってくる。

「この回廊を抜けた先に、何があるんだ？」

シオンがカイトに尋ねた。

「イグニスさんは言ってなかった」

カイトは首を振った。

「ただ、『灼熱に耐え、進み続けよ』とだけ」

カイトはイグニスの言葉を思い出した。

カイトたちは回廊に入った。

足元の床は、炎の模様が浮き出るタイル。

一歩踏み出すたびに、タイルが熱を放つ。

「何か仕掛けがありそうだ」

カイトは警戒した。

十歩ほど進んだ時だった。

回廊の両側から、火柱が噴き出した。

「来るぞ！」

カイトが叫んだ瞬間、炎が壁となって襲ってきた。

カイトは即座に防御魔法を発動した。


// 水の防御壁
function waterShield(targets) {
  const magic = {
    type: "水",
    subtype: "barrier",
    targets: targets,
    effect: "炎を中和する水の壁",
    duration: 5
  };
  
  return `${targets.join("、")}の周りに水の盾を展開`;
}

// 実行
waterShield(["カイト", "リアム", "エリナ", "シオン"]);

四人を包む水の膜が、炎を受け止めた。

水が蒸発し、白い霧が立ち込める。

「防いだ！」

エリナが安堵した。

でも、カイトは眉をひそめた。

「これ、体力を消耗する」

カイトは魔力の消費を計算した。

回廊はまだ続いている。

この調子で火柱が来たら、魔力が尽きる。

「防御じゃなく、回避でいこう」

カイトは提案した。

さらに進むと、床のタイルが変わった。

赤いタイルと青いタイルが、市松模様に並んでいる。

「チェスボードみたい」

シオンが観察した。

カイトは赤いタイルに足を踏み入れた。

すると、足元から炎が噴き出した。

「わっ！」

カイトは後ろに飛び退いた。

「赤いタイルは罠だ」

リアムが分析した。

「青いタイルだけを踏んで進むんだ」

カイトは頷いた。

青いタイルだけを慎重に進んでいく。

でも、青いタイル同士の距離が徐々に離れていく。

「飛ばないと届かない」

エリナが不安そうに言った。

「風魔法で補助しよう」

シオンが提案した。


// 風の跳躍補助
function windJump(target) {
  const magic = {
    type: "風",
    subtype: "movement_boost",
    target: target,
    effect: "跳躍力を三倍に強化"
  };
  
  return `${target}に風の跳躍を付与`;
}

// 全員に適用
["カイト", "リアム", "エリナ", "シオン"].forEach(windJump);

四人は風に乗って、タイルを渡っていった。

一歩間違えれば、灼熱の床へ。

でも、シオンの風魔法が支えてくれる。

「シオン、いい仕事だ」

カイトが称賛した。

市松模様のエリアを抜けると、次は坂道だった。

上り坂。しかも、床が動いている。

「逆走する歩道みたい」

リアムが呟いた。

進んでも進んでも、元の位置に戻される。

「走らないと」

カイトは走り出した。

でも、回廊の熱気が体力を奪っていく。

「カイト、これを見て」

エリナが壁を指差した。

壁の炎に、文字が浮かんでいる。

「灼熱に逆らうな。熱と共に歩め」

カイトは文字を読んだ。

「熱と共に歩め？」

カイトは考えた。

逆らうのではなく、受け入れる。

カイトは足を止めた。

「みんな、止まって」

カイトは言った。

「でも、床が動いてるよ？」

シオンが不安そうに言った。

「信じて」

カイトは目を閉じた。

カイトは熱を感じた。

熱耐性魔法で遮断しているけど、その奥にある熱の本質。

炎の鼓動。リズム。流れ。

カイトは魔法を書き換えた。


// 熱との同調
function heatSync(target) {
  const magic = {
    type: "炎",
    subtype: "resonance",
    target: target,
    effect: "周囲の熱と同調し、受ける影響を最小化"
  };
  
  return `${target}を熱と同調させる`;
}

// 全員に適用
heatSync("カイト");
heatSync("リアム");
heatSync("エリナ");
heatSync("シオン");

四人の体が、赤く光り出した。

熱と一体化した感覚。

床の動きが、体に馴染んでいく。

「進めるようになった」

リアムが気づいた。

坂道をゆっくりと登っていく。

熱は敵じゃない。道しるべだ。

「これが、イグニスさんの言いたかったことか」

カイトは理解した。

坂道の頂上に、広場があった。

広場の中央に、巨大な炎の柱が立っている。

柱の中に、何かが見えた。

「あれは……？」

エリナが目を凝らした。

炎の柱の中に、剣があった。

炎そのもののような、赤い刃の剣。

「炎帝の剣」

カイトは直感した。

柱に近づくと、声が聞こえた。

「剣を引き抜け。ただし、覚悟を決めて」

イグニスの声だった。

「僕がやる」

カイトは前に出た。

「待って、カイト」

シオンが止めた。

「罠かもしれない」
「わかってる」

カイトは振り返った。

「でも、試練なんだから、何かをしないと」

カイトは微笑んだ。

カイトは炎の柱に手を伸ばした。

熱が肌を焼く感覚。でも、同調魔法が守ってくれる。

剣の柄に指が触れた。

その瞬間、炎が爆発した。

カイトは炎の中に包まれた。

「カイト！」

エリナが叫んだ。

でも、カイトは燃えなかった。

炎の中で、声が聞こえた。

「何のために戦う？」

炎帝の剣からの問いかけ。

カイトは答えた。

「守りたいものがあるから」

カイトは言った。

「仲間を、世界を、未来を」

カイトは続けた。

「僕の魔法は、誰かを傷つけるためじゃない。誰かを守るためにある」

カイトの言葉が、炎に響いた。

炎が静まった。

カイトの手の中に、剣があった。

炎帝の剣。でも、燃えるような熱さは消えていた。

「引き抜いた……」

リアムが驚いた。

「よかった、無事で」

エリナが胸を撫で下ろした。

剣を握ると、炎の魔力が流れ込んでくる。

炎を操る力の片鱗。

「これが、第二の試練の報酬か」

カイトは剣を見つめた。

広場の奥に、新たな扉が現れた。

扉の上には、「第三の試練への道」と刻まれている。

「あと一つだ」

カイトは剣を背負った。

イグニスの声が響いた。

「灼熱の回廊、クリアだ」

「最後の試練が、待っているぞ」

カイトたちは扉に向かった。

第三の試練。情熱の炎。

何が待っているのか、わからない。

「みんな、大丈夫？」

カイトが振り返った。

「疲れたけど、いける」

シオンが頷いた。

「僕も大丈夫」

リアムも言った。

「カイトがいれば、怖くない」

エリナが微笑んだ。

カイトは心が温かくなった。

仲間がいる。それが、最大の力だ。

「じゃあ、行こう」

カイトは第三の扉を開けた。

扉の向こうは、広大な円形闘技場だった。

観客席は炎で満たされている。何千もの炎の精霊が、見守っている。

闘技場の中央に、一人の人物が立っていた。

イグニスだった。

でも、さっきとは違う。鎧を脱ぎ、剣を持っている。

「最後の試練だ、カイト」

イグニスが剣を構えた。

「情熱の炎。それは、戦いの中で証明される」

イグニスは燃えるような瞳でカイトを見据えた。

「私と戦え。そして、お前の情熱を示せ」

カイトは炎帝の剣を構えた。

第三の試練。イグニスとの決闘。

「受けて立ちます」

カイトは宣言した。

炎の精霊たちが、歓声を上げた。

闘技場の空気が、熱を帯びる。

カイトとイグニス、二人の対決が始まろうとしていた。

第759章、了。
