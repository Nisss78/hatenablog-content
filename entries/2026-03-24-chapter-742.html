第742章：峡谷の入り口

夜が明けた。荒野の朝は寒かった。乾いた風が岩陰を吹き抜け、焚き火の残り火を揺らす。

カイトは最初に目を覚ました。エリナが隣で眠っている。昨夜の会話を思い出し、少しだけ心が温かくなった。

「おはよう、カイト」

リアムが起きてきた。大剣を背負い、軽く伸びをする。

「おはよう。よく眠れた？」
「ああ。緊張して眠れないかと思ったが、意外とぐっすりいけた」

リアムは焚き火に薪を足し、火を強くした。

「昨日の戦いで疲れてたんだろ。体が休息を求めてた」

まもなく、シオンとエリナも目を覚ました。

「ふわあ……寒い」

シオンが毛布にくるまりながら言った。

「朝の冷え込みは厳しいからな。暖まってから出発しよう」

カイトが焚き火を囲むよう手招いた。

四人は朝食をとりながら、今日の計画を話し合った。

「山脈の入り口まで、半日もかからないはずだ。問題は、そこから先」

カイトが地図代わりの木片に、指で道筋を描いた。

「封印の場所は、山脈の中にある峡谷だ。周囲は断崖絶壁、進める道は一本だけ。そこが一番の難所になる」
「待ち伏せされる可能性が高いってこと？」

エリナが尋ねた。

「そう。怪物だけじゃない。もしかしたら、十三使徒が待っているかもしれない」

カイトの言葉に、全員の表情が引き締まった。

「十三使徒……か」

リアムが剣の柄に手を置いた。

「まだ全員と戦ったわけじゃない。油断はできないな」
「でも、私たちも強くなってる。負けないよ」

シオンが拳を握った。

「そうだね。準備はできてる。行こう」

カイトが立ち上がった。三人もそれに続いた。

山脈の麓に到着したのは、正午を少し過ぎた頃だった。

巨大な岩壁が目の前に聳え立っている。見上げても頂上が見えないほど高い。七つの光は、岩壁の向こう側で輝いていた。

「ここからどうやって入るの？」

シオンが岩壁を見上げながら尋ねた。

「入り口があるはずだ。探そう」

カイトが岩壁に沿って歩き出した。

十分ほど歩くと、岩壁に亀裂が走っているのを見つけた。幅は五メートルほど、高さは十メートル以上。自然の峡谷ではなく、何者かが意図的に開けたような形状だった。

「これが入り口……？」

エリナが峡谷の中を覗き込んだ。奥は暗く、先が見えない。

「いらっしゃい。待っていたよ」

峡谷の奥から、声が響いた。

四人は身構えた。足音が近づいてくる。

現れたのは、一人の男だった。白い長髪、青白い肌、黒いローブ。その瞳は、虚ろだった。光も闇もない、ただの穴のような瞳。

「名を名乗れ」

カイトが言った。

「名前？　そんなもの、とっくの昔に捨てたよ。ただの番人だ」

男は微笑んだ。だが、その微笑みには温度がなかった。

「君たちが封印を修復しに来たのは知っている。でも、残念ながら通すわけにはいかない」
「十三使徒の一人？」

リアムが剣を抜いた。

「十三使徒？　違うな。彼らは虚無の信徒だ。私はただの番人——虚無そのものの一部だ」

男は手を広げた。

「封印から漏れ出した虚無が、形を成した。それが私だ。名もなき、意志を持った影」

カイトは理解した。この男は、怪物と同じ種類の存在だ。だが、知性を持ち、言葉を話す。上位の虚無の化身だ。

「倒せば、通れるんだな？」
「もちろん。私を倒せるならね」

男は指を鳴らした。

峡谷の壁から、黒い霧が滲み出してきた。霧は形を成し、次々と怪物の姿をとる。十体、二十体——いや、もっとだ。

「私一人じゃない。私の眷属もいる。さあ、どうする？」

男は挑発的に笑った。

怪物の群れが、四人を包囲した。

「数が多い……！」

エリナが杖を構えた。

「落ち着いて。戦術を変える」

カイトが指示を出した。

四人は素早く配置についた。

シオンが両手を掲げた。七属性の魔力が渦を巻く。


// 七属性融合魔法：プリズム・ノヴァ
function prismNova() {
  const elements = ["fire", "water", "wind", "earth", "lightning", "light", "dark"];
  const fused = [];
  
  for (let elem of elements) {
    const magic = mana.summon(elem);
    fused.push(magic);
  }
  
  // 全属性を融合
  const nova = manaFuseAll(fused);

  nova.setDamage("massive");

}

「やった！」

シオンが叫んだ。

だが、番人は涼しい顔をしていた。

「いい魔法だ。でも、数には勝てない」

番人が手を振ると、倒した怪物が再び立ち上がった。霧が集まり、形を成す。

「無限に蘇る。私がいる限り、眷属は減らない」
「なら、お前を先に倒す！」

カイトが番人に向かって駆けた。

「無駄だ。私は虚無そのもの。切っても、突き抜けるだけだ」

番人が指を鳴らすと、怪物たちが一斉に攻撃を開始した。

リアムが剣で怪物を薙ぎ払う。エリナが水魔法で防壁を張る。シオンが絶え間なく攻撃魔法を放つ。

だが、怪物の数は減らない。倒しても、倒しても、蘇る。

「このままじゃ、消耗戦で負ける！」

リアムが叫んだ。

カイトは考えた。番人は虚無そのもの。物理攻撃も、魔法攻撃も、通用しない。どうすれば——

そこで、カイトは閃いた。

「封印だ……！」

カイトは番人を凝視した。

「お前は、封印から漏れ出した虚無だと言ったな。つまり、封印と繋がっているんだ」

番人の表情が初めて変わった。微かな動揺。

「……鋭いな」
「封印を修復すれば、お前の力も弱まる。あるいは、消滅する」

カイトは仲間に叫んだ。

「みんな、番人を無視して進むぞ！　封印の場所まで行けば、勝てる！」
「でも、怪物が邪魔だ！」

エリナが叫んだ。

「僕が道を開く！　ついてきて！」

カイトは虚無魔法を最大出力で展開した。


// 虚無魔法：虚空回廊
function voidCorridor() {

そこは峡谷の最深部だった。巨大な広場、その中央に七つの柱が建っている。それぞれの柱から光が立ち上り、天井付近で交差していた。

「これが封印……」

シオンが息を呑んだ。

七つの柱の中央には、黒い球体が浮いていた。中に何かが見える。人のような、でも人ではない、歪んだ影。

「虚無の具現化……」

カイトが呟いた。

球体の中で、影が蠢いた。まるで、目覚めようとしているかのように。

「急いで修復しないと、完全に目覚める」

カイトは柱に近づいた。柱には古代文字が刻まれている。


// 封印解析システム
class SealAnalyzer {
  constructor(seal) {
    this.seal = seal;
    this.pillars = seal.pillars;
  }
  
  analyze() {
    const status = {
      integrity: 0,
      damage: [],
      repairMethod: null
    };
    
    // 七つの柱の状態を確認
    for (let i = 0; i < 7; i++) {
      const pillar = this.pillars[i];
      status.integrity += pillar.strength;
      
      if (pillar.strength < 1.0) {
        status.damage.push({
          pillar: i,
          cause: pillar.damageSource
        });
      }
    }
    
    status.integrity /= 7;
    status.repairMethod = this.determineRepair(status);
    
    return status;
  }
}

// 二属性同時展開
function dualAttributeInfusion() {
  const dark = elements.summon("dark");
  const void = elements.summon("void");
  
  // 時分割で交互に注入
  const infusion = async () => {
