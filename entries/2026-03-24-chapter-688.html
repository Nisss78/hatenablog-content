第688章：光の塔の第三層

第二層を後にして、私たちは再び螺旋階段を登り始めた。

今度は階段の様子が違っていた。壁が半透明になり、向こう側で何かが動いているのが見える。光の粒子が渦を巻き、幾何学的な模様を描いている。

「綺麗だけど、何か不気味ね」

エリナが壁を眺めながら言った。

「ただの飾りじゃなさそうだ」

リアムが剣の柄に手をかける。

「警戒して損はない」

シオンが壁に手を触れた。

「暖かい」

彼女が呟く。

「光が生きてるみたい」

「生きている光……」

セシリアが興味深そうに観察する。

「魔力を持った光子の集合体かしら。学術的に見れば興味深いわね」

私はコードで解析を試みた。

階段を登り続けること数分。第三層への扉が見えてきた。

第一層も第二層も白い結晶の扉だった。でも、第三層の扉は違っていた。

黒と白が交互に並ぶ、碁盤の目のような扉。

「光と闇……」

セシリアが呟く。

「チェス盤みたいね」

「チェス？」

シオンが首を傾げる。

「古い遊びよ」

エリナが説明する。

「二人が交互に駒を動かして、相手の王を取った方が勝ち」

「何か関係があるのかしら」

私は扉に近づき、黒白のパターンを観察した。

「これ、ただの模様じゃない」

指で触れてみると、黒いマスと白いマスが微妙に質感が違う。白は滑らかで、黒は少しざらついている。

「凹凸がある」

リアムも気づく。

「点字みたいだな」

「点字……」

私はその可能性を検討した。もし黒いマスが点字だとしたら、何かメッセージが隠されているのかもしれない。

でも、点字にしてはマスの数が多すぎる。六十四マス。八×八のグリッド。

「待てよ」

ひらめきが走った。

「これ、チェスの初期配置だ」

「初期配置？」

エリナが眉をひそめる。

「チェスの駒は、最初に特定の位置に置かれる。その配置をマスの色で表している」

私は指でなぞりながら説明した。

「黒いマスは黒の駒がある場所。白いマスは白の駒がある場所か、何もない場所」

「じゃあ、どうやって開けるの？」

シオンが聞く。

「おそらく、正しい手を指すんだ」

私は推測を述べた。

「誰かとチェスをしている。相手の待ち手に対して、正しい応手を打てば、扉が開く」

「相手は誰？」

リアムが周囲を見渡す。

「塔自体か、守護者か」

「どっちにしても、やるしかない」

私は扉に向き直った。

「私が指す。みんなは見ていてくれ」

「カイト、チェスできるの？」

エリナが意外そうな顔をした。

「昔、少しだけ」

正直に言うと、プログラミングで将棋AIを作ったことはあるが、チェスは趣味でやった程度だ。でも、基本戦略はわかる。

私は黒いマスの一つ、キングの初期位置に触れた。

すると、扉全体が淡く光り出した。

「反応した！」

セシリアが声を上げる。

次の瞬間、マス目から光の駒が浮かび上がった。白と黒の駒が、空中に静止している。

「白番か。相手が先手か」

私は状況を理解した。

光の白い駒が動く。キングの前のポーンが二マス前进。

「オーソドックスなオープニングだ」

私は手を伸ばし、黒のポーンを動かした。

対称の位置に進める。シシリアン・ディフェンスへの変換を狙う。

白の次の手。ナイトが動く。

私は冷静に応じた。チェスは情報戦だ。相手の手を読み、最善の応手を打つ。

def evaluate_position(board):
    score = 0
    for piece in board.pieces:
        if piece.color == BLACK:
            score += piece_value(piece.type)
            score += position_bonus(piece)
        else:
            score -= piece_value(piece.type)
            score -= position_bonus(piece)
    return score

def find_best_move(board, depth=3):
    best_move = None
    best_score = float('-inf')
    
    for move in board.legal_moves:
        board.push(move)
        score = minimax(board, depth - 1, False)
        board.pop()
        
        if score > best_score:
            best_score = score
            best_move = move
    
    return best_move

戦況は膠着状態だった。白は攻めるが、私は守りを固める。互いに駒を交換し、盤面は少しずつ単純化していく。
