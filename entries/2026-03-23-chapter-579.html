第579章：希望の灯火

中央大陸の夜空に、不吉な紫の光が脈打っていた。

「あれが……世界再構築の兆候か」

カイトは城壁の上から、遠く北東の空を見つめていた。その瞳には、プログラマーとしての知性と、冒険者としての警戒心が同居している。彼の手の中で、魔法の杖が微かに振動していた——まるで、来たるべき戦いを予感しているかのように。

エリナが静かに隣に立った。銀髪が夜風に揺れる。

「カイト、感じる？ あの光の中心から、何か巨大な魔力が放出されているわ」

「ああ。まるで……巨大なサーバーが起動しているような感覚だ」

カイトの脳内で、プログラムのコードが次々と浮かび上がる。

class WorldReconstruction {
    constructor(targetWorld) {
        this.target = targetWorld;
        this.progress = 0;
        this.status = "initializing";
    }
    
    async execute() {
        // 世界の基礎構造を再定義
        await this.redefineLaws();
        await this.rewriteHistory();
        await this.reshapeReality();
    }
}

「誰かが、この世界自体を再プログラミングしようとしている」

カイトの言葉に、後ろで聞いていたリアムが眉をひそめた。

「世界を……再プログラミング？ それは一体どういう意味だ？」

「この世界は、ある種のシステムとして動いている。魔法の法則、自然の摂理、因果の流れ——それら全てが、何らかのプログラムとして記述されているんだ」

カイトは説明を続ける。

「普通の魔術師は、そのシステムの中で魔法を使う。火を生み出したり、風を操ったり。でも、俺の能力は違う。俺は……システム自体に干渉できる」

「だからあなたは『コード・マジック』を使えるのね」

セシリアが納得したように頷く。

「その通り。そして今、誰かがそのシステムレベルで、世界そのものを書き換えようとしている。それが『世界再構築』だ」

沈黙が降りた。全員が、事態の深刻さを理解していた。

□ ■ □

城門の広間には、主要な面々が集まっていた。

アルフレッド——かつてカイトたちと敵対し、今は盟友となった守人——が、古びた巻物を広げていた。

「古の記録によれば、世界再構築は過去に二度試みられたことがある」

「二度？」

「最初は、最初の管理者アレクシス・ヴェイルの時代。彼は世界の崩壊を防ぐために、七つの原初の力を調律した」

アルフレッドは続ける。

「二度目は、約五百年前。十三使徒の前身となった組織が、『理想郷』の創造を試みた」

「結果は？」

「失敗した。いや、正確には……阻止された」

リリス——深淵の魔女として知られる、漆黒の髪の女性——が口を開いた。

「阻止したのは、七人の賢者たちだったわ。彼らは各属性の力を結集させ、世界再構築の儀式を中断させたの」

「七人の賢者……」

カイトは以前に出会った賢者たちを思い出す。イグニス（炎）、フローズナ（氷）、ライラ（雷）、ミズリア（水）、そしてアンブロス（闇）。

「残りは……光と地、か」

「ええ。ルミナスとテラドン。彼らは今、北東の『虚無の聖域』にいるはずよ」

エレノア——シルヴィアの母であり、時空魔法の使い手——が地図を指し示した。

「虚無の聖域は、この世界の『境界』が最も薄い場所。世界再構築を行うには、そこが最適な環境なの」

「つまり、敵もそこにいる可能性が高い」

カイトの推測に、全員が頷いた。

□ ■ □

作戦会議は深夜まで続いた。

「敵の規模は？」

テオドール——アルヴァレス支部長——が尋ねる。

「確認できているだけで、百を超える魔力反応があります。内、十三使徒クラスの強者は五名」

セレナ・クロフォード——中央ギルド調査官——が報告書を読み上げる。

「十三使徒の第七席ルシフェイン、第九席グラシエルの存在が確認されています。また、正体不明の三名が指揮官格と思われます」

「十三使徒か……」

カイトは以前の戦いを思い出す。彼らは強力な敵だった。だが、今のカイトなら——いや、今のカイトたちなら、勝てるかもしれない。

「戦力の分散は避けたい」

リアムが提案する。

「少数精鋭で、敵の中枢を急襲する。それが最善の策だ」

「賛成だ。でも、誰を行かせる？」

全員の視線が、自然とカイトに集まった。

「カイト、あなたは必須よ。コード・マジックがないと、世界再構築を止められない」

エリナの言葉に、カイトは頷く。

「わかった。俺は行く。エリナ、リアム、セシリア——お前たちも来てくれるか？」

「当然ね」
「断る理由がない」
「あなたを一人にはさせないわ」

三人の答えは、即座だった。

「リリス、アルフレッド——二人はここに残って、中央大陸の防衛を頼めるか？ 万が一、俺たちが失敗した時は……」

「わかっているわ。この世界を守り抜く——それが私の役目」

リリスの瞳に、決意の光が宿る。

「私はカイトたちの帰還を待つ。必ずな」

アルフレッドも力強く頷いた。

□ ■ □

夜が明ける前、カイトたちは出発の準備を整えた。

シオン——記憶喪失の銀髪少女で、七属性魔法回路を持つ——が、カイトの前に立った。

「私も……連れて行って」

「シオン、危険だぞ」

「わかってる。でも、私は……何かを思い出しかけてるの。あの場所のことを」

シオンは胸を押さえる。

「虚無の聖域。そこで、私は……何かを失った。そして、何かを得た気がするの」

カイトは迷った。シオンの能力は強力だが、記憶喪失の彼女を危険な戦場に連れて行くのは躊躇われた。

だが——。

「カイト、彼女を行かせてあげて」

エリナが静かに言う。

「シオンの過去は、きっとあの場所にある。彼女自身の答えを、彼女自身の手で見つけさせるべきよ」

「……わかった。シオン、覚悟はいいか？」

「うん。ありがとう、カイト」

□ ■ □

出発の刻限が近づいた。

城門の前で、見送りの面々と別れの挨拶を交わす。

ミラベル——管理者の一人——が、カイトに小さな水晶を手渡した。

「これは『座標固定石』よ。世界再構築が進行すると、空間そのものが不安定になる。この石があれば、必ず元の場所に戻れる」

「ありがとう、ミラベル」

ヴォルター——もう一人の管理者——が、重厚な声で告げる。

「カイト、君の能力は我々の想像を超えている。だが、過信はするな。世界再構築を行っている存在は、君と同じ——あるいはそれ以上の、システムへの干渉能力を持っている可能性がある」

「……わかっている」

カイトは深く頷いた。

□ ■ □

夜明け前の空の下、四人と一匹——カイト、エリナ、リアム、セシリア、そしてシオン——が北東へ向けて歩き出した。

空には、まだ紫の光が脈打っている。まるで、彼らを招き入れるかのように。

だが、カイトの心には、一つの確信があった。

——終わらせる。この戦いを、そして、俺の旅を。

カイトは、握りしめた杖に意識を集中させる。その中に、無数のコードが奔流のように流れていた。

function WorldRestoration() {
    const hope = gatherAllies();
    const power = combineStrength(hope);
    const miracle = executeFinalSpell(power);
    
    return miracle; // 必ず、希望を
}

「行こう。希望の灯火を、灯しにな」

カイトの言葉に、全員が頷く。

北東の空の下、小さな光が闇を切り裂くように進んでいった。

——戦いの火蓋は、今、切って落とされた。
