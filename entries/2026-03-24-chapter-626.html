第626章：管理者の誓い

アルヴァレス支部——その大広間に、多くの人々が集まっていた。

カイト一行の帰還を祝う集まり——支部の冒険者たち、職員たち、そして街の代表者たち。皆、カイトたちに感謝の言葉をかけようと集まっていた。

「カイトくん——世界を救ったんだって?」

見知らぬ冒険者が、カイトの肩を叩いた。

「大げさだよ。みんなでやったことだ」
「謙虚だな! でも——聞いたぞ。原初の座で、虚無の王とやらを封じたんだろ?」
「封じたというより——循環の一部にしたんだ」
「 whatever! 結果は同じだろ! ありがとな——お前のおかげで、俺の魔法も復活したんだ!」

冒険者は、豪快に笑った。カイトは苦笑するしかなかった。

◆ ◇ ◆

大広間の奥——テオドール支部長が、カイトを呼んだ。

「カイト——少し、話がある」

カイトは、エリナたちに目配せして、テオドールについていった。支部長室——静かな空間だった。

「座ってくれ」

テオドールが、椅子を勧めた。カイトは、大人しく座った。

「カイト——君に、正式な話がある」

テオドールの表情——真剣だった。

「君を——アルヴァレス支部の特別顧問に任命したい」
「特別顧問?」
「正規の職員ではない。だが——支部の重要な決定に参加できる立場だ。君の知識——そして、君の魔法。それを世界のために使ってほしい」

カイトは、少し考えた。

「……俺は——自由でありたいんだ」
「わかっている。だから——特別顧問だ。定期的な報告も、任務もない。ただ——必要な時に、力を貸してほしい」

テオドールの目——真剣だった。

「君は——管理者としての力を持っている。それを——無駄にしてほしくないんだ」

管理者——アレクシス・ヴェイルの血脈を受け継いだ、カイト。その力——世界を救った力。カイトは、自分の手を見た。

「……わかった。引き受けるよ」
「ありがとう——カイト」

テオドールは、安堵の表情を浮かべた。

「だが——一つ、条件がある」
「何だ?」
「俺の仲間たち——エリナ、リアム、セシリア、シオン。彼女たちも——認めてほしい。俺と同じだけの」

テオドールは、少し驚いた顔をした——そして、笑った。

「当然だ。彼女たちも——世界を救った英雄だ。特別待遇——いや、それ以上の扱いを受けるべきだ」

◆ ◇ ◆

支部長室を出たカイトは、廊下でエリナと出くわした。

「カイト——テオドール支部長、何か言ってた?」
「特別顧問に任命された」
「へ?」

エリナが、きょとんとした顔をした。

「特別顧問——って、すごい地位じゃない?」
「らしいな。でも——定期的な報告も任務もない。ただ——必要な時に力を貸すだけ」
「じゃあ——今までと変わらない?」
「そうなるな」

エリナが、ふっと笑った。

「カイトらしい——答えね」

◆ ◇ ◆

大広間——シオンとリアムが、冒険者たちに囲まれていた。

「シオンちゃん——七属性魔法使いなんだって?」
「う、うん——まだ完璧じゃないけど」
「謙遜しなくていいよ! 七属性——伝説級だぞ!」

シオンは、少し恥ずかしそうにしていた。だが——嬉しそうでもあった。

リアムの周りには——炎魔法に興味のある冒険者たちが集まっていた。

「リアムの炎——どんな訓練をしたんだ?」
「えっと——実戦で覚えたんだ。デストラクション——十三使徒の一人——に、アドバイスもらったけど」
「十三使徒に!? 凄いな!」

リアムは、少し誇らしげだった。だが——自分の炎を見つめて、何かを考えていた。

「炎は——破壊だけじゃない。守るためにも使える——って、教わったんだ」
「守るため?」
「ああ。まだ——うまくできないけど。でも——いつか、守れるようになりたい」

リアムの言葉に、周りの冒険者たちも、頷いた。

◆ ◇ ◆

セシリアは——一人、窓際に立っていた。

夜空——星が瞬いている。セシリアは、それを見つめていた。

「セシリア——ここにいたか」

カイトが、近づいてきた。

「カイト——どうしたの?」
「みんな探してるぞ。主役がいないと——始まらない」
「主役——私じゃないわ。カイト——あなたよ」

セシリアは、首を振った。

「私は——ただ、サポートしただけ。虚無の王と戦ったのも——儀式を完成させたのも——全部、カイト」
「それは違う。俺一人じゃ——何もできなかった」

カイトは、セシリアの隣に立った。

「エリナがいたから——前に進めた。リアムがいたから——守れた。シオンがいたから——希望を捨てなかった。そして——セシリアがいたから——魔法を信じられた」

カイトの言葉に、セシリアは——少し驚いた顔をした。

「……魔法を信じた?」
「ああ。お前の時空魔法——何度も助けられた。お前がいたから——俺は、どんな状況でも諦めなかった」

セシリアは——ふっと笑った。

「カイト——そう言ってくれると、嬉しいわ」
「事実だ。だから——主役は、俺たち全員だ」

カイトは、セシリアの肩を叩いた。

「行こう——みんな待ってる」
「ええ——行きましょう」

◆ ◇ ◆

夜が更けて——宴が終わった。

一行は——宿屋に戻った。それぞれの部屋で——明日に備える。

カイトは——一人、屋上にいた。夜風——冷たかった。だが——心地よかった。

「カイト——」

エリナが、現れた。

「ここにいたのね」
「少し——考え事を」

エリナは、カイトの隣に立った。

「何を考えてるの?」
「これからのこと——かな」

カイトは、空を見上げた。

「世界を救った——って言われるけど。本当に——救えたのかな」
「どういう意味?」
「虚無の王——循環の一部になった。だが——彼が、本当に救われたのかは——わからない」

カイトの声——少し、暗かった。

「あいつ——ずっと孤独だった。誰にも理解されず——世界の基盤にされ——封印された。その寂しさ——俺には、わかる気がするんだ」

エリナは——カイトの手を握った。

「カイト——優しいね」
「優しくなんてない。ただ——全部を背負いたくないだけだ」
「背負わなくていいの。カイト——あなたは、十分に戦った。十分に——頑張った」

エリナの言葉——温かかった。

「虚無の王——彼も、救われたと思うよ。カイトが——循環の一部にしてくれたから。彼は——もう、孤独じゃない」
「……そうだといいけど」
「そうよ——きっと」

エリナは、カイトに寄りかかった。

「カイト——これからも、一緒にいよう」
「……ああ」

カイトは、エリナを抱きしめた。

夜空——星が、瞬いていた。静かな夜——そして、新たな朝が来る。

◆ ◇ ◆

翌朝——カイトは、支部に出頭した。

テオドール支部長——そして、数名のギルド上層部が、会議室に集まっていた。

「カイト——来てくれたか」

テオドールが、頷いた。

「今日は——君に、いくつかの報告がある」
「報告?」
「ああ。まず——魔力の回復状況だ」

テオドールが、地図を広げた。世界地図——いくつかの場所に、赤い印がついている。

「魔力の回復に伴い——各地で、異変が報告されている」
「異変?」
「強力な魔物の目覚め——古代の遺跡の活性化——そして、未知の現象。すべて——魔力の回復が原因だ」

カイトは、地図を見た。赤い印——十箇所以上。

「これ——全部?」
「ああ。今のところ——深刻なものはない。だが——放置すれば、大きな問題になる」

テオドールの表情——深刻だった。

「カイト——君たちに、調査を頼みたい」
「調査?」
「ああ。最も深刻な場所——北の山脈。そこで、異常な魔力反応が観測されている」

北の山脈——カイトは、そこに行ったことがあった。氷の賢者フローズナの聖域——そして、フィンブルヴェトが封印されていた場所。

「北の山脈——フィンブルヴェトがいた場所だろ?」
「そうだ。だが——フィンブルヴェトは、もういない。封印は解かれた——あるいは、崩壊した」

テオドールの言葉に、カイトは眉をひそめた。

「封印が崩壊——何が出る可能性がある?」
「わからない。だが——万が一のため、調査が必要だ」

カイトは——少し考えた。そして、頷いた。

「わかった。引き受ける」
「ありがとう——カイト。君たちに、期待している」

◆ ◇ ◆

会議室を出たカイトは——仲間たちに事情を説明した。

「北の山脈——か」

リアムが、腕を組んだ。

「フィンブルヴェトがいた場所——封印が崩壊したってことは——」
「何かが——出るかもしれない」

セシリアが、続きを言った。

「でも——フィンブルヴェトは、倒したわよね?」
「倒した——けど。封印自体は、古代のものだ。他にも——何か封じられているかもしれない」

シオンが、少し不安そうに言った。

「怖い——けど——行かなきゃだよね?」
「ああ。俺たちが——世界を救ったんだ。その責任——果たさないとな」

カイトの言葉に、全員が頷いた。

◆ ◇ ◆

出発は——明日。

カイトたちは——それぞれの準備を進めた。薬草の補充——装備の整備——そして、休息。

カイトは——一人、部屋にこもった。目を閉じて——集中する。

コード——脳内に浮かぶコード。自分の魔法——そして、可能性。


function analyzeMagicResonance(location) {
  const response = detectManaFlow(location);
  if (response.anomaly) {
    return investigateAnomaly(response.source);
  }
  return null;
}

カイトは——コードを書いた。魔法として——この世界に存在するコード。

北の山脈——何が待っているのか。カイトは——覚悟を決めた。

管理者として——そして、冒険者として。

カイトの旅は——まだ、終わらない。

◆ ◇ ◆

夜——エリナが、カイトの部屋を訪れた。

「カイト——準備できた?」
「ああ。明日——出発する」

エリナは——カイトの隣に座った。

「北の山脈——寒いわよ」
「わかってる。防寒具——準備した」
「じゃあ——大丈夫ね」

エリナは、カイトに寄りかかった。

「カイト——これからも、ずっと——一緒にいよう」
「……ああ。約束する」

カイトは、エリナの手を握った。

静かな夜——明日への決意を固める夜。

カイトの物語——第二章の始まりだった。

窓の外——月が、輝いていた。冷たい光——だが、希望の光。
