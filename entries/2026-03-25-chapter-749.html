第749章：深海の予兆

夜明けの海は、黄金色に輝いていた。

「海神の翼」号は、順調に東へ進んでいる。昨夜の海竜との戦闘で多少の被害を受けたが、航行に支障はない。

カイトは甲板で、朝日を浴びながら息を整えていた。

「カイト、おはよう」

エリナが近づいてきた。手には、温かいスープが入った椀を持っている。

「朝ご飯、持ってきたよ。シオンとリアムも食堂で食べてる」
「ありがとう」

カイトはスープを受け取った。野菜と魚のスープだ。塩味が効いている。

「昨夜の海竜……あれ、深海神殿の守護者だったのかな」

エリナが尋ねた。

「可能性はある。でも、虚無に侵食されていた。封印が弱まって、守護者さえも狂わせてしまったんだと思う」

カイトの表情が暗くなった。

「深海神殿、どれくらいで着くの？」
「ノヴァ島までは、順調なら明日の昼頃。そこから海底へ潜る方法を探さないといけない」

カイトはスープを飲み干した。

「問題は、海底への入り口だ。深海神殿は、水面から数百メートル下にあるらしい。普通の水魔法じゃ、そこまで潜るのは厳しい」
「私たちが使った水呼吸の魔法、応用できない？」

エリナが首を傾げた。

「水呼吸は、あくまで呼吸を可能にする魔法。水圧への対策じゃないんだ。深度が深くなれば、体が潰れる」

カイトは眉をひそめた。

「新たな魔法を開発しないといけない。水圧を無効化する、あるいは軽減する魔法を」

昼前、カイトは船室で魔法の研究をしていた。

机の上には、水魔法に関するメモが散らばっている。


// 水圧対策魔法の理論構築
function pressureNullificationTheory() {
  const theory = {
    problem: "深海の水圧は深度100mで約10気圧。人体は耐えられない",
    solution_ideas: [
      "体の周りに魔法の膜を形成し、圧力を分散",
      "体内の空間を魔力で固定し、潰れないようにする",
      "水と同化して、圧力の影響を消失させる"
    ],
    feasibility: {
      magic_barrier: "可能だが、魔力消費が激しい",
      internal_fix: "リスクが高い。失敗すれば内臓破裂",
      water_fusion: "未検証。前例なし"
    }
  };
  
  return theory;
}

// 理論確認
console.log(pressureNullificationTheory());

カイトは三番目の案、「水との同化」に興味を持った。

（水と同化する……身体を一時的に水の性質に変えるイメージか。成功すれば、水圧を全く感じないはず)

だが、リスクも大きい。失敗すれば、本当に水になって戻れないかもしれない。

（少しずつ実験するしかない。浅い深度から始めて、徐々に深く)

カイトは決断した。

昼過ぎ、カイトは甲板で実験を始めた。

エリナ、リアム、シオンが見守る中、カイトは海に向かって手を伸ばした。


// 水同化魔法・実験版
function waterFusionExperiment(duration, depth) {
  const magic = {
    element: "water",
    type: "transformation",
    effect: "身体の一部を一時的に水の性質に変化",
    duration: duration,
    depth: depth,
    target_body_part: "右手のみ"
  };
  
  return `右手を${duration}秒間、水の性質に変化`;
}

// 実験開始
waterFusionExperiment(5, 0);

カイトの右手が、半透明の水のような質感に変わった。

「すごい……手が透けてる」

シオンが目を輝かせた。

カイトは水に右手を浸した。水圧は感じない。むしろ、水の中にいる方が自然だ。

（感覚がある。水の流れが、指先を通じて伝わってくる)

五秒後、魔法を解いた。右手は元に戻った。

「成功だ。ただ、全身でやるとなると話が違う。徐々に時間を延ばして、慣れていく必要がある」

カイトは汗を拭った。

その時、ヴィクトル船長が甲板に現れた。

「カイト、前方に島が見えてきた」
「ノヴァ島ですか？」

カイトが尋ねた。

「いや、違う。地図にない島だ」

船長の表情は複雑だった。

「三十年以上、この海域を航海してるが、こんな島を見たことがない」

カイトたちは船首に向かった。

水平線の向こうに、小さな島が見えた。緑に覆われた、美しい島だ。

「地図にない島……？　どういうこと？」

エリナが不思議そうに言った。

「いくつか可能性がある。一つは、新しくできた島。海底火山の活動で隆起したとか」

カイトが推測した。

「二つ目は、魔法で隠されていた島。何らかの理由で、結界が解除された」
「三つ目は？」

リアムが尋ねた。

「幻覚。深海神殿に関わる何かが、僕たちを試している」

カイトの言葉に、全員が沈黙した。

船は徐々に島に近づいていった。

島の姿が鮮明になる。中央には高い山。周囲には白い砂浜。椰子の木が揺れている。

「普通の島に見えるけど……」

シオンが首を傾げた。

「魔力の反応がある。微かだけど、この島全体から発せられている」

カイトが目を細めた。

「やはり、ただの島じゃない。上陸して調べる必要がある」
「危険じゃないか？」

リアムが懸念を示した。

「もちろん、危険かもしれない。でも、この島が深海神殿へのヒントを握っている可能性もある」

カイトは船長の元へ向かった。

「上陸しましょう」

カイトの言葉に、ヴィクトル船長は少し考えた。

「分かった。岸壁に停泊して、小舟で上陸する」
「ありがとうございます」
「だが、無理はするな。何かあったら、すぐに撤退しろ」

船長の目は真剣だった。

一時間後、カイト、エリナ、リアム、シオンの四人は小舟で島に向かっていた。

船員の一人が舵を取っている。

砂浜が近づいてくる。波は穏やかだ。

「着きました」

船員が舟を砂浜に乗り上げさせた。

四人は舟を降りた。足元には、白い砂。熱帯の植物の香りが漂う。

「誰もいない……？」

エリナが周囲を見回した。

確かに、人の気配はない。動物の声も聞こえない。

「静かすぎる」

リアムが剣の柄に手をかけた。

「シオン、風で何か感じる？」

カイトが尋ねた。

シオンは目を閉じて、風の流れを感じ取った。

「風……普通の風だけど、どこか人工的な感じがする。自然の風じゃないかも」
「やはり、この島自体が魔法で作られたものか」

カイトは確信した。

四人は島の奥へと進んだ。

森を抜けると、開けた場所に出た。そこには、古びた石造りの神殿があった。

「神殿……」

エリナが息を呑んだ。

「深海神殿の入り口かもしれない」

カイトが言った。

神殿の入り口には、古代文字が刻まれている。カイトは知識を総動して解読した。

「『深淵を望む者よ、試練を超えよ。真の管理者のみ、聖域へ至ることを許す』……」

カイトが読み上げた。

「試練、か。深海神殿への資格を試す場所なんだろうな」

リアムが剣を抜いた。

「入るぞ」

カイトが神殿の門を押した。

重い石の扉が、軋みながら開いた。

神殿の中は、意外なほど明るかった。

壁には発光する苔が生えている。床には、複雑な魔法陣が描かれていた。

「これは……」

カイトは魔法陣を調べた。

「転送陣だ。深海神殿へ直接送ってくれるのかもしれない」
「じゃあ、これに乗ればいいの？」

シオンが魔法陣の中に入った。

「待て、シオン！　まだ起動してない――」

カイトの声が届く前に、魔法陣が光り輝いた。

四人の足元が消えた。

気がつくと、四人は水中にいた。

だが、溺れる感覚はない。周囲に、薄い魔法の膜が張られている。

（水の中！？　でも、息ができる……自動的に水圧対策もされてる)

カイトは周囲を見回した。

深海だ。太陽の光は届かないが、発光する魚たちが泳ぎ回っている。

目の前には、巨大な神殿がそびえ立っていた。

深海神殿。

「ここが……深海神殿……」

エリナの声が、水中を通って届いた。

「思ってたより、大きいな」

リアムが感嘆した。

神殿の入り口に、人影があった。

長い銀髪の女性。肌は青白く、目は海のように深い蒼。

彼女は四人に近づいてきた。

「ようこそ、深海神殿へ。私は、この神殿を守る水の賢者、ミズリア」

カイトは息を呑んだ。

「ミズリア……水の賢者、ミズリア？」

カイトは記憶を辿った。

（確か、最初の管理者たちの中に、水の賢者ミズリアという名前があったはず。この人は、古代の賢者なのか？)

ミズリアは穏やかに微笑んだ。

「あなたがカイトね。あなたのことは、予言で聞いているわ」
「予言……？」
「ええ。虚無を封じ、世界を救う者の予言。あなたがその者かどうか、試させてもらうわ」

ミズリアの目が、鋭くなった。

ミズリアは手を上げた。

周囲の水が、巨大な竜の形に変化した。

「水龍の試練。これを乗り越えれば、神殿への入り口を許可する」

水龍が咆哮を上げた。水中で、その音は重く響いた。

「四人で戦っていいのね？」

エリナが尋ねた。

「もちろんです。仲間との絆も、試練の一部ですから」

ミズリアは下がった。

水龍が突進してきた。

「散開！」

カイトの指示で、四人は四方に散った。

水龍の体は水でできている。物理攻撃は通りにくい。

「リアム、剣で斬っても再生するだけ！　魔法で攻撃して！」
「分かった！」

リアムは剣を収め、身体強化の魔法で接近戦に備えた。

エリナが水魔法を放った。

「水流刃！」

鋭い水流が、水龍の体を切り裂く。だが、傷はすぐに再生した。

「再生が速い！」

エリナが焦った。

カイトは分析した。

（水龍の核はどこだ？　さっきの海竜と同じなら、体内のどこかにあるはず)

カイトは感知魔法を使った。


// 核の位置探知
function detectCore(target) {
  const magic = {
    element: "water",
    type: "detection",
    effect: "対象の魔力の中心を探知",
    target: target
  };
  
  return `${target}の核の位置を特定`;
}

// 実行
detectCore("waterDragon");

感知の結果、水龍の頭部に強い魔力反応があった。

「頭だ！　頭の中に核がある！」

カイトが叫んだ。

「シオン、風で水龍の動きを止めて！」
「う、うん！　風魔法……水中でどうやれば……」

シオンは戸惑った。

「水の中でも、風は存在する。水流として形を作るんだ！」

カイトの言葉に、シオンは目を輝かせた。

「水流……分かった！」

シオンは両手を広げた。

「水流拘束！」

周囲の水が、風の影響で渦を巻いた。その渦が、水龍の体を拘束した。

水龍の動きが止まった隙に、カイトが近づいた。

水同化魔法で、水龍の体をすり抜ける。

「七属性融合弾！」

// 七属性融合魔法・単体攻撃版
function fusedDestructionBullet(target, aim) {
  const magic = {
    elements: ["fire", "water", "wind", "earth", "light", "dark", "lightning"],
    type: "offensive",
    effect: "全属性の魔力を凝縮して放つ弾丸",
    target: target,
    aim: aim
  };
  
  return `${target}の${aim}に七属性弾を放つ`;
}

// 実行
fusedDestructionBullet("waterDragon", "head_core");

七色の弾丸が、水龍の頭部を貫いた。

核が砕け、水龍が崩壊した。

無数の水滴となって、海中に散った。

戦闘が終わると、ミズリアが近づいてきた。

「見事です。水龍の試練を突破しました」

ミズリアは深く頭を下げた。

「あなたたちは、深海神殿に入る資格があります。どうぞ、中へ」

ミズリアは神殿の扉を開けた。

中から、眩いほどの光が溢れ出した。

「ありがとう、ミズリアさん」

カイトは礼を言った。

「中で、真実を知ることになるでしょう。覚悟を決めて」

ミズリアの言葉は、どこか重かった。

四人は神殿の中へと足を踏み入れた。

扉が閉まり、光が収まった。

そこには、巨大な魔法陣があった。その中央に、黒い結晶が浮いている。

「これが……虚無の封印」

カイトが囁いた。

結晶は、微かに振動していた。封印が、限界に近づいている証拠だ。

（急いで、修復しないと)

カイトは覚悟を決めた。

深海神殿での真の試練は、これからだった。

第749章、了。
