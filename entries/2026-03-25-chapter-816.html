第816章：闇の根源との対峙

闇の最深部。そこは光という概念が存在しない、純粋な暗黒の空間だった。カイトは自分の手すら見失いかけていた。

「ここが……闇の根源か」

カイトの声が虚無に吸い込まれていく。返答はない。ただ、圧倒的な静寂だけが満ちていた。

「ようやく辿り着いたのう、若き管理者」

声がした。いや、声というよりは思考に直接響く振動に近かった。カイトは周囲を見渡したが、視界には何も映らない。

「ここにおるよ。常に、そして永遠にな」

カイトの目の前に、徐々に輪郭が浮かび上がってくる存在があった。それは人の形をしていたが、人ではなかった。闇そのものが凝縮され、人の姿を模したような——。

「あなたが……闇の根源？」

「根源とは名ではない。ただの概念じゃ。じゃが、お主がそう呼びたいならば、そう呼ぶがよい」

闇の根源は静かに微笑んだように見えた。その表情は、人間的な温かさと、計り知れない深淵の冷たさを同時に宿していた。

「なぜ僕を呼んだんですか」

「呼んだのではない。お主が自らの足で歩いてきたのじゃ。闇の試練を乗り越え、深淵を超え、ここまで辿り着いた」

カイトは思い出した。これまでの旅路を。闇の賢者アンブロスとの修行、深淵の書庫での探求、そして十三使徒アビスとの戦い。すべてが、この瞬間への道筋だった。

「お主は問うておるな。なぜ闇が存在するのか。なぜ虚無が世界を侵食しようとするのか」

「……わかりますか」

「わからぬわけがなかろう。わしは闇そのもの。万物の影。光があれば必ず生まれる、不可避の存在じゃからな」

闇の根源はゆっくりと手を伸ばした。すると、周囲の闇が波紋のように広がり、無数の映像が浮かび上がった。

「これは……」

「世界の記憶じゃ。光と闇の、永遠の繰り返し」

映像の中で、光と闇が交互に世界を覆っていた。光が輝く時、その背後に必ず影が生まれる。影が濃くなれば、やがてそれは闇となる。そして闇が極まれば、光が再び芽生える——。

「これが世界の理じゃ。陰陽の循環。決して断ち切ることのできぬ、絶対の法則」

「でも、虚無はこの循環を壊そうとしている」

「左様。虚無は無じゃ。無は有の対極。光でも闇でもない、存在そのものの否定」

カイトは理解し始めた。虚無がなぜこれほどまでに恐ろしいのか。光と闇の双方を脅かす存在なのか。

「ダークハルト、ミラージュ、アルタイル。彼らは最高位管理者でありながら、虚無に飲まれた」

「なぜ……」

「彼らは答えを求めたのじゃ。世界を救うための、完璧な答えを。そして見つけたのが、虚無との融合だった」

闇の根源の声に、微かな哀しみが混じった。

「彼らは間違っておらぬ。じゃが、道を誤った。虚無は決して制御できるものではない。無限の力を与えるようでいて、最終的には自己さえも飲み込む」

カイトは握りこぶしを作った。三人の最高位管理者たちの悲劇。それは、世界を救いたいという純粋な願いが招いた、皮肉な結末だった。

「僕に何ができるんですか」

「問い続けることじゃ」

「問い……？」

「答えは常に問いの中にある。お主はプログラマーじゃろう？ コードを書く時、答えは最初からあるわけではない。問いを立て、仮説を検証し、エラーを修正して、ようやく辿り着くものじゃ」

カイトはハッとした。闇の根源の言葉は、彼の本質を突いていた。

「虚無に対抗するのは、問い続ける意志じゃ。答えを諦めぬ強さ。たとえ絶望的な状況でも、解は存在すると信じ続けること」

「……それが、僕に与えられた試練なんですね」

「試練ではない。選択じゃ。お主は問い続けることを選べる。あるいは、答えを諦めて虚無に融合することも選べる」

闇の根源が一歩近づいた。

「じゃが、お主は後者を選ばぬ。わかっておる」

「なぜそう言い切れるんですか」

「お主の目を見ればわかる。光と闇、その双方を受け入れた者の目じゃ。虚無に屈する者は、光か闇のどちらかに固執する。どちらかを否定することで、無に逃げ込む」

カイトは自分の内側を感じた。七属性の魔法回路。炎、水、風、地、雷、光、そして闇。すべての属性が、彼の中で共存していた。どれか一つを否定することは、自分自身を否定することと同じだった。

「僕は……どの属性も諦めません」

「ならば、わしから贈り物を授けよう」

闇の根源が両手を広げた。すると、カイトの周囲の闇が凝縮し、一冊の書物となって彼の手の中に現れた。

「これは……」

「深淵の書物の、真の姿じゃ。これまでお主が目にしたものは、その一部に過ぎぬ」

カイトは書物を開いた。そこには、世界の秘密が記されていた。光と闇の起源、虚無の正体、そして——対抗するための術式。


// Null Counter Protocol
// 虚無に対抗するための基盤コード

function nullCounter() {
  const light = this.essence.light;
  const dark = this.essence.dark;
  const balance = light * dark / (light + dark);
  
  if (void.approaching) {
    return {
      barrier: balance * infinity,
      anchor: memory.self,
      question: "なぜ戦うのか"
    };
  }
}

カイトは目を見開いた。これは、虚無に対する防御の概念だった。光と闇のバランスを取り、自己の存在を錨として固定する。そして何より、戦う理由を問い続ける——。

「虚無は無限に見えるが、有限じゃ。問い続ける限り、虚無の境界線は見えてくる」

「このコードを……実装できるんですか」

「お主ならできる。七属性の管理者として、光と闇の双方を統べる者ならな」

カイトは書物を胸に抱いた。重みがあった。それは物理的な重さではなく、世界の運命を背負う重みだった。

「闇の根源……あなたはどちらの味方なんですか」

「どちらでもない。わしは概念じゃ。味方も敵もない。ただ、存在し続けるのみ」

闇の根源が少しずつ霧散し始めた。

「じゃが、一つだけ言えることがある。お主のような存在が現れたことを、わしは喜んでおる」

「喜んでいる……？」

「長いこと待っていたのじゃ。光と闇を等しく愛する者を。虚無に屈せず、問い続ける者を」

闇の根源の姿が完全に消え失せる前に、最後の言葉が響いた。

「行け、カイト。仲間たちが待っておる。そして、ダークハルトたちが待っておる」

「ダークハルトたちを……救えるんでしょうか」

「問い続けよ。答えはその先にある」

闇が晴れた。カイトは、深淵の書庫の入り口に立っていた。手には深淵の書物の真の姿。心には、新たな問いと決意。

「カイト！ 無事だったのね！」

エリナが駆け寄ってきた。彼女の後ろには、シオン、リアム、セシリアたちの姿もあった。

「みんな……」

カイトは仲間たちの顔を見回した。彼らもまた、それぞれの試練を乗り越えてきたはずだ。

「何か……大事なものを手に入れた顔してるわね」

シオンが鋭く指摘した。

「ああ。これからが本番だ」

カイトは深淵の書物を見せた。

「虚無に対抗する方法。でも、これだけじゃ足りない。みんなの力が必要だ」

「言われなくてもわかってるわよ」

エリナが微笑んだ。

「私たちはチームでしょ？ 一人で背負い込まないで」

カイトは頷いた。そうだった。問い続けることは、一人で行うものではない。仲間と共に、答えを模索していくものだ。

「行こう。次は……最高位管理者たちとの決着だ」

カイトと仲間たちは、深淵の書庫を後にした。空には、七色の光と闇が混ざり合い、不思議な模様を描いていた。

世界の命運をかけた戦いは、いよいよ最終局面へと突入していた。
