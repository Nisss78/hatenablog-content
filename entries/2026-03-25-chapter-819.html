第819章：虚無の根源

門をくぐった瞬間、カイトは「無」の感覚に包まれた。

視覚がない。聴覚がない。触覚がない。五感がすべて null に設定されたような、完全な感覚遮断の状態。残っているのは意識だけ——それも、風前の灯火のように頼りない。

『僕は……誰だ』

問いかけた瞬間、答えが返ってこなかった。名前を思い出そうとしても、その参照先が null になっている。自己認識の変数が、空っぽだった。

『存在の錨——エリナ、リアム、セシリア、シオン——』

仲間たちの名前を呼び出そうとした。すると、微かにだが、温かい感覚が戻ってきた。彼らがカイトの存在をミラージュの鏡像の中に保存してくれている。その接続が、カイトの意識を null から引き戻した。

『大丈夫。僕は——カイト。海藤翔太。七属性の管理者。みんなを救うために、ここに来た』

自己認識を強制的に上書きする。プログラミングで言えば、 null 参照エラーを catch して、デフォルト値を設定するような処理。カイトは自分の意識の中で、その処理をループさせ続けた。

◇◇◇

徐々に、感覚が戻り始めた。

最初に戻ったのは視覚だった。といっても、光が見えたわけではない。闇が見えた。純粋な、何も含まない黒。だが、その黒の中に、微かな輪郭が浮かび上がっていた。

「ここが……虚無の根源」

カイトの声は、音として届かなかった。だが、概念として響いた。この場所では、物理的な音波は存在しない。意図そのものが伝達される。

周囲を見渡すと、無数の「裂け目」が見えた。空間に刻まれた傷跡のようなもので、そこから黒い霧が漏れ出している。裂け目の一つ一つが、虚無が世界に干渉しているポイントだった。

「これを塞げば……」

カイトは裂け目の一つに近づいた。触れようとした瞬間、強烈な拒絶反応が返ってきた。

「存在の介入を検知。対象——管理者クラス。脅威レベル——中程度。排除プロセスを開始」

機械的な声——いや、概念的な宣言——が響いた。黒い霧が凝縮し、人の形をした影を形成した。顔のない、輪郭だけの存在。

「虚無の尖兵……？」

「我々は根源の意志の一部。存在を null に変換する実行者」

「実行者……」

カイトは身構えた。通常の魔法は、ここでは限定的にしか機能しない。だが、深淵の書物で学んだ概念的なアプローチなら——。


// Conceptual Attack - Void Layer
// 概念攻撃 - 虚無層

function conceptualAttack(target) {
  const question = "なぜ存在を否定するのか";
  const intent = "理解したい";
  
  return {
    type: "question_projectile",
    payload: { question, intent },
    target: target
  };
}

カイトは掌に概念を凝縮した。光の球——ではなく、「問い」の球体。それは物理的な攻撃力を持たないが、相手の概念に直接干渉する。

「行け！」

問いの球体が、影の尖兵に命中した。

「問いを検知。内容——『なぜ存在を否定するのか』。解析——できない。回答—— null 」

尖兵の動きが止まった。問いに対する答えを持たないため、処理がループに陥ったのだ。

「これだ。ここでは『問い』が最強の武器になる」

カイトは理解した。虚無の領域では、答えを持たないものは無力化される。逆に言えば、問い続けることで、虚無の侵食を食い止められる。

問いの球体を連射し、周囲の尖兵たちを次々と無力化していく。彼らは消滅したわけではない。ただ、問いに対処できず、停止状態に陥った。

カイトはさらに奥へと進んだ。

◇◇◇

虚無の根源の中心部には、巨大な「核」があった。

それは黒い太陽のように見えた。直径は数百メートル。表面は絶えず脈動し、黒い波紋を放射している。波紋の一つ一つが、世界のどこかで発生する虚無の影響だった。

「これが……元凶」

カイトは核の前で立ち尽くした。圧倒的な質量。そして、圧倒的な「無」。近づくだけで、自己認識が揺らぐ。

「来訪者を確認。存在ステータス——一時 null 化中。本来なら即座に完全 null 化される対象。じゃが——保存処理を確認。外部アンカーにより、自己認識を維持」

核から声が響いた。いや、声ではない。純粋な情報の波。カイトの意識に直接語りかけてくる。

「君は……何者？」

「我は虚無の根源。概念の消失を司る存在。じゃが、本来の役割は違う」

「本来の役割？」

「我はもともと、終わりを司る者だった。寿命、破滅、変革——それらを管理する、世界の調整機構。じゃが、ある時から狂った」

「狂った……？」

「誰かが問うたのじゃ。『なぜ存在するのか』とな。我は答えられなかった。終わりを司る者に、存在の理由などないからな。問いに答えられぬまま、我は停滞した。そして、停滞は退化を招いた」

カイトは息を呑んだ。虚無の根源は、悪意のある存在ではない。ただ、問いに答えられずに停滞してしまった、悲しき存在。

「じゃが、答えがないからといって、存在を否定していいわけじゃない」

カイトは核に向かって言った。

「僕の世界にも、答えのない問いはたくさんある。『なぜ生きるのか』『正義とは何か』『愛とは何か』。それらに明確な答えはなくても、人は生き続ける。問い続ける」

「問い続ける……か。興味深い概念じゃ。じゃが、我にはできぬ。我は問いを停止した時点で、本来の機能を失った。今はただ、終わりを加速させるだけの存在」

「機能を……取り戻せるかもしれない」

カイトは提案した。

「僕が君に問い続ける。君はそれに答えようとしてくれればいい。答えが出なくても、問い続けること自体に意味がある」

「問い続ける……我に？」

「そう。プログラミングで言えば、デバッグセッション。問題を一緒に探るんだ」

核の脈動が、わずかに変わった。興味——あるいは戸惑い。

「面白い。やってみるがよい」

◇◇◇

カイトは核の前に座り込んだ。物理的な座席ではない。意識の座標を固定するイメージ。

「第一の問い。君は、自分をどう認識している？」

「問いを受理。回答を生成——『我は虚無の根源』。じゃが、その認識は二次的に付与されたもの。本来の認識は—— null 」

「本来の認識は null ……。じゃあ、君は本来、何だったの？」

「第二の問い。回答を生成——『終わりを司る者』。じゃが、それも役割であって、存在そのものではない。存在の本質は——」

核が沈黙した。

「—— null 」

「じゃあ、探そう。君の存在の本質」

カイトは七属性の回路を活性化させた。光、闇、火、水、風、地、雷——七つの力が、カイトの意識の中で共鳴する。


// Essence Search Protocol
// 本質探索プロトコル

function searchEssence(target) {
  const elements = ["light", "dark", "fire", "water", "wind", "earth", "thunder"];
  const resonances = [];
  
  elements.forEach(el => {
    const res = checkResonance(target, el);
    if (res !== null) {
      resonances.push({ element: el, strength: res });
    }
  });
  
  return resonances.length > 0 ? resonances : "essence_null";
}

「君の中に、何か残っていないか探させてもらう。七つの要素で共鳴するものがあれば、それが君の本質の手がかりになる」

カイトは核に向かって、七色の光を放った。光は核の表面に触れ、内部へと浸透していく。

「……！」

反応があった。闇の要素と、地の要素。この二つが、微かにだが共鳴した。

「共鳴を確認。要素——闇、地。共鳴強度——微弱。じゃが——存在する」

「闇と地……。終わりと、土に還る。なるほど、君の本来の役割は『死と再生の循環』だったのかもしれない」

「死と……再生」

「そう。終わりは、新たな始まりのための準備。落葉が土に還り、次の春の養分になるように。君は本来、終わりを通じて新たな生命を育む存在だった」

核の脈動が、大きく変わった。黒い表面に、微かな亀裂が入った。その亀裂から、淡い光が漏れ出している。

「思い出した……。そうじゃ、我は終わりを司るだけでなく、始まりへの架け橋だった。じゃが、誰かの問いに答えられず、その役割を忘れた」

「役割を思い出した。じゃあ、今の君は——」

「まだ完全ではない。長い時間、虚無として機能してきた副作用が残っておる。じゃが——希望は見えた」

核が、自らの一部を分離させた。それは黒い球体——手のひらサイズの、凝縮された虚無の欠片。

「これを受け取れ。我の権限の一部じゃ。これを持つ者は、虚無の領域での活動が容易になる。じゃが、同時に虚無の誘惑にも晒される。意志が弱ければ、飲み込まれるぞ」

「わかった。受け取る」

カイトは黒い球体を受け取った。手の中で、冷たくて温かい、相反する感覚がした。

「一つだけ警告しておく。虚無の領域には、我以外にも『狂った概念』が存在する。かつて世界を構成していた要素が、問いを失って暴走しているのじゃ。それらと遭遇したら——問い続けること。答えを求めず、問い続けること」

「了解。ありがとう、虚無の根源」

「礼など不要。我はまだ完全には戻らぬ。じゃが、お主との対話で、一歩前進した。それだけで十分じゃ」

核の脈動が安定し始めた。黒い太陽は、少しだけ輝きを取り戻したように見えた。

◇◇◇

カイトは門を逆方向にくぐり、虚無の領域の表層へと戻った。

「カイト！」

エリナが最初に駆け寄ってきた。彼女の手が、カイトの頬に触れる。

「無事だったのね……よかった」

「大丈夫。少し疲れただけ」

カイトは仲間たちに微笑んだ。ダークハルト、ミラージュ、アルタイルも、安堵の表情を浮かべていた。

「虚無の根源に会ったのじゃな。お主の纏う気が、また変わった」

ダークハルトが言った。

「ええ。話をして、ある程度理解し合えました。彼は敵じゃない。ただ、問いに答えられずに迷っていただけ」

「そうか。やはり、お主のアプローチは正しかった」

ミラージュが微笑んだ。

「それに、これを授かりました」

カイトは黒い球体を取り出した。

「虚無の権限の一部……！ まさか、根源自身から授かったのか」

アルタイルが驚愕した。

「これは予言になかった。虚無の根源との交渉——そして和解。お主は、予言の枠を完全に超えた」

「まだ終わってません。根源の話では、他にも『狂った概念』が存在するらしい。それらに対処しないと、世界の安定は完全じゃない」

「狂った概念……」

リアムが呟いた。

「具体的には、どんなものがあるんだろう」

「根源は詳細を話さなかった。ただ、『問い続けること』が対抗策だと言っていた」

セシリアが頷いた。

「私たちのアプローチと同じね。答えを急がず、問い続ける」

シオンがカイトを見つめた。

「カイト、その球体——力を感じる。七属性と共鳴してる。もしかしたら、あなたの中で何かが目覚めるかもしれない」

「目覚める……？」

「ええ。管理者としての可能性が、さらに広がる。ただ、気をつけて。虚無の力は、光と闇の両方を内包してる。使い方を間違えると——」

「わかってる。慎重に使うよ」

カイトは球体を懐にしまった。

「さて、戻ろうか。アンブロスさんたちも心配してるだろうし」

「うむ。一度、深淵の書庫に戻り、今後の指針を立てよう」

ダークハルトの提案に、全員が同意した。

カイトたちは、虚無の領域から脱出し、通常の世界へと戻った。

虚無の根源との対話——それは、世界を救うための重要な一歩だった。

だが、カイトは予感していた。これは序章に過ぎないと。狂った概念たちとの対峙、そして自身の中に眠る虚無の力——これらを制御しなければ、真の平和は訪れない。
