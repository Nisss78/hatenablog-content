第746章：氷雪の試練

北の山脈を下る道は、登りよりも険しかった。

深淵の書庫を出て数時間。カイトたちは山腹を慎重に進んでいた。日が沈み始め、空は茜色から紫へと変わっている。

「もうすぐ暗くなる。ペースを上げないと」

リアムが前を向いたまま言った。

「でも、急ぐと危ないよ。道が凍ってる」

エリナが足元を指差した。細い山道は薄い氷に覆われ、一歩踏み出すごとに滑りそうになる。

「私、風魔法で滑り止めできるよ」

シオンが手をかざした。


// 足元の滑り止め魔法
function createGripMagic(target) {
  const wind = {
    element: "wind",
    direction: "downward",
    pressure: "light",
    purpose: "friction_increase"
  };
  
  return `風を足元に纏わりつかせ、摩擦力を高める`;
}

// シオンの魔法発動
createGripMagic("party_feet");

シオンの魔法が四人の足元を包んだ。風が靴底に絡みつき、滑りにくくなる。

「すごい！　歩きやすくなった」

エリナが驚いたように言った。

「えへへ、私にもできることあるでしょ？」

シオンは得意げに胸を張った。

「ありがとう、シオン。助かったよ」

カイトが微笑んだ。

日が完全に沈んだ頃、異変が起きた。

山の上から、低い唸り声が聞こえてきたのだ。

「……何か来る」

リアムが剣の柄に手をかけた。

カイトは魔力感知を広げた。山の斜面を、何かが高速で降りてきている。

「魔獣だ。複数……いや、一頭だけど、かなり大きい」
「大きい魔獣？」

シオンが身構えた。

雪煙を巻き上げながら、巨大な影が現れた。白い毛皮に覆われた狼のような獣。だが、普通の狼の三倍はあるサイズだ。

「氷牙狼（フロストファング）だ。この山脈の最上位捕食者」

カイトが説明した。

「氷牙狼……名前の通り、牙が氷でできてるの？」

エリナが尋ねた。

「ああ。凍える息と共に、氷の牙で獲物を貫く。単独で行動するが、戦闘力は高い」

氷牙狼は四人を睨みつけた。青白い瞳が、暗闇の中で光っている。

「戦うの？」

シオンが緊張した声で尋ねた。

「逃げるのは無理だ。氷牙狼の走力は、僕たちの比じゃない」

カイトが答えた。

「なら、やるしかない」

リアムが剣を抜いた。

氷牙狼が咆哮を上げ、飛びかかってきた。

巨大な体躯が雪を巻き上げながら迫る。カイトはすばやく魔法を展開した。


// 防御魔法：氷の壁

// カイトの防御
createIceWall("front", "thick");

カイトの前に氷の壁が現れた。氷牙狼の牙が壁に突き刺さるが、貫通はしない。

「今だ、リアム！」

カイトが叫んだ。

リアムは壁の横から飛び出し、剣を振るった。斬撃が氷牙狼の脇腹を掠める。

「グオォォ！」

氷牙狼が痛みに吼えた。すぐに体勢を立て直し、今度はリアムに向き直る。

「速い！」

リアムは剣で攻撃を受け止めた。氷の牙と鉄の剣がぶつかり合い、火花が散る。

「エリナ、炎魔法で！」

カイトが指示した。

「うん！」

エリナは掌に炎を宿した。


// 炎魔法：火球

  };
  
  return `対象に向けて火球を放つ`;
}

// エリナの攻撃
castFireball("frostfang_face", "medium");

火球が氷牙狼の顔面に直撃した。毛皮が焦げ、獣が悲鳴を上げる。

「シオン、風でエリナの炎を強化して！」

カイトが続けた。

「分かった！」

シオンは風魔法を使い、エリナの炎に酸素を送った。炎が勢いを増し、氷牙狼を包み込む。

「カイト、トドメを！」

リアムが叫んだ。

カイトは魔力を集中させた。氷と光の複合魔法——凍結光線を放つ準備をする。


// 複合魔法：凍結光線

// カイトの必殺技
castFreezingBeam("frostfang_body");

青白い光線がカイトの手から放たれた。光線は氷牙狼の体を直撃し、瞬時に凍らせる。

巨大な獣は、氷の彫像のように硬直した。

戦闘が終わり、四人は息を整えた。

「はぁ……はぁ……やったね」

シオンが膝に手をついた。

「みんな、連携が上手かったよ」

カイトが仲間たちを称賛した。

「カイトの指示が的確だったからね。感謝するよ」

リアムが剣を納めた。

「エリナの炎、すごかったよ。氷牙狼の毛皮を一瞬で焦がしたもんね」

シオンが言った。

「シオンの風魔法があったから、炎が強くなったんだよ。二人の息が合ってた」

エリナはシオンの頭を撫でた。

「えへへ……」

シオンは照れくさそうに笑った。

カイトは凍りついた氷牙狼を見た。

「この氷は、半日もすれば溶ける。氷牙狼も、命までは奪ってないから、逃げられるはずだ」
「優しいね、カイトは」

エリナが言った。

「魔獣も生き物だ。無駄な殺生はしたくない」

カイトは静かに答えた。

氷牙狼を倒した後、四人は山道を急いだ。

夜の山は危険だが、立ち止まっているほうがもっと危険だ。カイトは光魔法で足元を照らし、慎重に進んだ。

「あとどのくらい？」

シオンが尋ねた。

「麓までは、あと二時間くらい。支部には、明日の朝には着けるはずだ」

カイトが答えた。

「二時間か……足が棒になりそう」

シオンが愚痴った。

「頑張れ、シオン。もう少しだ」

エリナが励ました。

リアムは無言で歩いていたが、時々、シオンの背中を押して支えていた。

深夜、一行はようやく麓に到着した。

山脈を抜け、平野部に出ると、空気が穏やかになった。星が綺麗に輝いている。

「麓に着いた……」

シオンがその場に座り込んだ。

「疲れた〜」
「お疲れ様。ここなら安全だ。少し休もう」

カイトが提案した。

四人は岩陰に座り、水筒の水を飲んだ。夜風が心地よい。

「ねえ、カイト」

エリナが口を開いた。

「ん？」
「ノヴァ島に行く時、どうやって海中の神殿に入るの？　水魔法が得意な私でも、深海まで潜るのは難しいよ」
「それなら大丈夫。水の賢者ミズリアに、水呼吸の魔法を教わってるから」

カイトが答えた。

「水呼吸？　水中で息ができる魔法？」

シオンが目を輝かせた。

「そうだよ。魔法で肺の周りに膜を作って、水から酸素を取り出すことができる」
「すごい！　私もできるかな？」
「教えるよ。ノヴァ島に行く前に、練習しよう」

シオンは嬉しそうに頷いた。

カイトが付け加えた。

「でも、三人が協力すれば大丈夫だよ。僕が防御魔法で水圧を防ぐ。エリナとシオンで、呼吸魔法を維持する」
「リアムは？」

シオンが尋ねた。

「リアムは、僕と一緒に水圧対策を担当しよう。剣士としての身体強度が、役に立つはずだ」

リアムは頷いた。

「任せてくれ。深海の探索、楽しみになってきた」

夜明け前、四人は出発した。

平野部を歩き、昼過ぎにはアルヴァレス支部の街が見えてきた。

「アルヴァレスだ！　やっと着いた」

シオンが歓声を上げた。

街の入り口で、見張りの兵士が四人に気づいた。

「お帰りなさい、カイト様。ご無事で何よりです」
「ただいま。テオドール支部長はいる？」
「はい、支部の執務室にいらっしゃいます」

四人は支部へと向かった。

支部の執務室で、テオドール支部長が四人を迎えた。

「戻ったか。深淵の書庫はどうだった？」

テオドールが尋ねた。

「エドガーさんに会いました。虚無の封印について、情報を得ました」

カイトは簡潔に報告した。

「西の峡谷以外に、少なくとも六つの封印がある。特にノヴァ島と氷結の山脈の封印は、早急な調査が必要です」

テオドールは真剣な表情で聞いた。

「ノヴァ島か……海中の神殿だな」
「はい。第二の封印が、深海に設置されています」
「海中への侵入は、難しいぞ。水魔法の心得がないと、命に関わる」

テオドールが懸念を示した。

「水呼吸の魔法は、訓練済みです。数日あれば、全員が習得できます」

カイトは自信を持って答えた。

「そうか……なら、船の手配をしよう。ノヴァ島までは、船で三日だ」
「お願いします」

報告を終えた後、四人は宿舎に戻った。

部屋に入ると、シオンがベッドに倒れ込んだ。

「疲れた〜。お風呂入りたい」
「温泉があるよ。支部の地下に」

エリナが教えた。

「本当？　行こう、エリナ！」

シオンはエリナの手を引いた。

「はいはい、行こうか」

二人は女子風呂へと向かった。

残されたカイトとリアムは、部屋で向かい合った。

「カイト、一つ聞きたいことがある」

リアムが口を開いた。

「何？」
「お前は、虚無と戦う覚悟があるのか？」

カイトは少し驚いた。

「どうしてそんなことを？」
「深淵の書庫で、エドガーが言っていただろう。虚無はお前を狙うと」

リアムの目は真剣だった。

「僕は剣士だ。戦うことはできる。だが、虚無相手では、僕の剣が通じるか分からない」
「リアム……」
「だから聞きたいんだ。お前は、本当に勝てると思っているのか？」

カイトは窓の外を見た。夕日が街を染めている。

「正直に言うと、確信はないよ。虚無がどれだけ強いのか、僕も正確には知らない」

カイトは静かに答えた。

「でも、戦わないという選択肢はない。僕が逃げたら、世界が終わるかもしれない。みんなが、エリナもシオンも、リアムも、何もかも失われるかもしれない」

カイトはリアムを振り返った。

「だから戦う。勝つために、できることを全部やる。そのために、リアムの力が必要だ」

リアムは少し沈黙した後、口元を緩めた。

「……分かった。お前がそこまで言うなら、最後までついていく」
「ありがとう、リアム」

カイトは手を差し出した。リアムはその手を握った。

「でも、無茶はするなよ。お前が倒れたら、終わりだ」
「分かってる。仲間を信じてるから」

その夜、四人は支部の食堂で夕食をとった。

温かいスープと焼きたてのパン。山での冷えた体に染み渡る。

「明日から、水呼吸の訓練だね」

カイトが言った。

「楽しみ！　深海、どんな所なんだろう」

シオンが目を輝かせた。

「暗くて寒いよ。でも、綺麗な魚もいるかもしれない」

エリナが言った。

「魚、食べれるかな？」
「シオン、食べることばかりだな」

リアムが呆れたように言った。

「だって、お腹空くんだもん」

シオンはスープを啜った。

「ノヴァ島までは三日。船旅だね」

カイトが言った。

「船か……初めて乗る」

シオンが言った。

「酔わないようにね。船酔いは辛いよ」

エリナが注意した。

「大丈夫！　私は強いから」

シオンは胸を張った。

カイトは仲間たちの会話を聞きながら、明日からの訓練を思った。

（水呼吸の魔法、三日で全員に習得させないと。深海の神殿、何が待っているのか……)

封印の修復、虚無との戦い、仲間との絆。

カイトの冒険は、まだ始まったばかりだ。

北の山脈での試練を乗り越え、次は海へ。

深海の神殿で、何が待っているのか——それは、まだ誰も知らない。
