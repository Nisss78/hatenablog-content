第764章：水の包容

二日目の朝が来た。

カイトは、客室の窓から深海を見た。

静かな青の世界。

魚たちが、ゆっくりと泳いでいる。

何の急ぎもない。ただ流れている。

「水の包容」

カイトは、昨日のミズリアの言葉を思い出した。

「第二の試練は、水のように全てを受け入れること」

一体どういう意味なんだろう。

エリナが起きた。

彼女も窓の外を見た。

「深海、不思議な場所だね」

エリナが呟いた。

「暗いのに、怖くない」
「水が守ってくれているからよ」

シオンが起きてきた。

「水は、中にあるものを優しく包む」
「深海も、水の一部だから」

リアムも起きた。

「今日は第二の試練だな」

リアムが言った。

「水の包容」
「どんな試練なんだろう」
「包容っていうのは、受け入れること」

シオンが解説した。

「水は、汚れも、穢れも、全て受け入れる」
「それでも、清らかであり続ける」
「それが、水の包容力よ」

カイトは、シオンの言葉を考えた。

全てを受け入れる。

それでも、清らかであり続ける。

難しいことだ。

朝食が運ばれてきた。

水の精霊が、トレイを持ってきた。

海藻のスープ、魚の刺身、海藻パン。

全て、海の幸だ。

「美味しい」

エリナが言った。

「海の中でも、こんなに美味しいものが食べられるなんて」
「水の賢者は、食事にもこだわりがあるみたいね」

シオンが言った。

「体に入れるものも、水のように清らかでなければ」
「それが、ミズリアの哲学」

食事を終えると、ドアがノックされた。

「おはよう、カイト」

ミズリアの声だった。

「準備はいい？」
「はい」

カイトは立ち上がった。

「第二の試練、お願いします」

カイトたちは、ミズリアに案内された。

昨日とは違う部屋。

そこは、大きな庭園だった。

水中に庭園がある。不思議な光景。

珊瑚が木のように並んでいる。

海藻が草のように揺れている。

色とりどりの魚が、花のように泳いでいる。

「すごい」

エリナが感嘆した。

「水中の庭園」
「まるで別世界」
「ここで、第二の試練を行うわ」

ミズリアが言った。

「水の包容」

ミズリアは、庭園の奥にある池を指差した。

小さな池。でも、深そうだ。

水が、黒く濁っている。

「この池、見て」

ミズリアが言った。

「汚れているでしょう？」

カイトは池を見た。

確かに、黒く濁っている。

昨日の水槽の水とは、正反対だ。

「この池には、いろいろなものが流れ込んだ」

ミズリアが語り始めた。

「泥、枯れ葉、死んだ魚、汚水」
「水は、それら全てを受け入れた」
「でも、見て」

ミズリアが池に手をかざした。

魔法を使った。


// 水の浄化
function purifyWater(pond) {
  const spell = {
    type: "水",
    subtype: "浄化",
    target: pond,
    effect: "濁りを取り除き、清浄な水に戻す"
  };
  
  return `${pond}の水が清らかになった`;
}

// 実行
purifyWater("汚れた池");

池の水が、澄んでいった。

黒い濁りが消え、透き通った水になった。

清らかで、美しい水。

「水は、何でも受け入れる」

ミズリアが言った。

「でも、受け入れるだけで終わらない」
「受け入れた後、自分を清らかに保つ」
「それが、水の包容の本質よ」

カイトは、ミズリアの言葉を噛みしめた。

受け入れるだけじゃない。

受け入れた後、自分を保つ。

それが、包容だ。

「第二の試練の内容は、これ」

ミズリアが言った。

「あなたの心の中にある、負の感情を受け入れる」
「恐怖、不安、劣等感、後悔」
「それら全てを、水のように受け入れるの」
「負の感情……」

カイトは呟いた。

「僕の中にも、あると思います」
「あるわよ、誰にでも」

ミズリアが微笑んだ。

「でも、多くの人は逃げる」
「負の感情と向き合うのは、怖いから」
「水は逃げない」

ミズリアが続けた。

「どんな汚れも、受け入れる」
「あなたも、水のように向き合いなさい」
「やります」

カイトは決意した。

「僕の中にある負の感情、受け入れます」

ミズリアは、カイトを池の前に立たせた。

「池を覗いて」

ミズリアが指示した。

「あなたの心が映るわ」

カイトは、池を覗いた。

水面に、自分の顔が映る。

でも、普通の顔じゃない。

暗い、歪んだ顔だった。

池が、カイトの心を映し出した。

負の感情が、形を成していく。

一つ目は、恐怖。

「僕は、強くない」

「十三使徒や最高位管理者に、勝てるのか？」

「仲間を守れるのか？」

二つ目は、不安。

「七属性の力を完成させて、何になる？」

「本当に、世界を救えるのか？」

「それとも、破滅を招くのか？」

三つ目は、劣等感。

「エリナも、リアムも、シオンも、すごい」

「僕だけ、遅れている気がする」

「本当に、僕が主人公でいいのか？」

四つ目は、後悔。

「もっとうまくできたはず」

「失敗したこともあった」

「あの時、もっと強ければ」

負の感情が、次々と現れた。

カイトは、胸が締め付けられた。

辛い。苦しい。

目を逸らしたい。

「逃げないで」

ミズリアの声が聞こえた。

「それらを、受け入れるの」
「水のように」

カイトは、目を逸らさなかった。

池を見続けた。

自分の中にある負の感情を、見つめた。

「恐怖がある」

カイトは、心の中で言った。

「僕は、強くないかもしれない」

「でも、恐怖があっても、前に進む」

「不安がある」

カイトは続けた。

「未来はわからない」

「でも、不安があっても、信じる」

「劣等感がある」

カイトは認めた。

「仲間たちはすごい」

「でも、僕には僕の役割がある」

「比較しても意味がない」

「後悔がある」

カイトは受け入れた。

「失敗したこともある」

「でも、後悔は過去」

「学んで、前に進む」

カイトは、負の感情を一つ一つ受け入れた。

否定しない。

逃げない。

ただ、認める。

池の水が、変わり始めた。

濁りが消えていく。

透き通った水になっていく。

カイトの顔も、変わった。

暗い表情が、穏やかな表情に。

負の感情を受け入れた顔。

そして、池は完全に澄んだ。

カイトの顔も、晴れやかだった。

「終わった」

カイトは立ち上がった。

「負の感情、受け入れました」

ミズリアが、満足そうに頷いた。

「合格よ」

ミズリアが言った。

「第二の試練、水の包容」
「あなたは、自分の中にある闇を受け入れた」
「最初は辛かった」

カイトが言った。

「でも、受け入れたら、楽になった」
「負の感情も、僕の一部なんだとわかった」
「それが、水の包容よ」

ミズリアが説明した。

「負の感情を否定しても、消えない」
「受け入れてこそ、浄化できる」
「水と同じね」
「水の包容……」

カイトは、深く頷いた。

「僕、少し強くなった気がします」
「強くなったわよ」

ミズリアが笑った。

「自分を受け入れることは、最も難しいことの一つ」
「あなたは、それを成した」

エリナたちが駆け寄ってきた。

「カイト、大丈夫？」

エリナが心配そうに聞いた。

「大丈夫」

カイトは笑った。

「ちょっと、自分と向き合ったんだ」
「楽になったよ」
「水の包容、か」

リアムが頷いた。

「いい試練だな」
「俺も、いつか受けたい」
「あなたも機会があるわよ」

ミズリアが言った。

「でも、今はカイトの試練」
「第三の試練、水の激流」
「明日、行いましょう」
「第三の試練は、戦いなんですよね？」

カイトが確認した。

「ええ」

「水は静かだけど、怒ると恐ろしい」
「その力を、戦いで使う試練よ」

「水の戦い方、学びたい」
「今夜は、ゆっくり休んで」

ミズリアが言った。

「明日は、激しい一日になるわよ」

カイトたちは、客室に戻った。

カイトは、窓の外を見た。

深海の静寂。

夜が来た。

カイトは、ベッドに横たわった。

今日は、自分と向き合った日。

負の感情を受け入れた日。

夢を見た。

広い海。波が寄せては返す。

静かな波。でも、力強い。

水の包容力を、肌で感じた。

朝が来た。

カイトは目を覚ました。

体が軽い。心も軽い。

// 水の激流攻撃

// 防御魔法
function shield(element) {
  const spell = {
    type: element,
    subtype: "防御",
    effect: "属性の壁を作る"
  };
  
  return `${element}の盾が展開された`;
}

// 実行（複合属性）
shield("炎と風");
