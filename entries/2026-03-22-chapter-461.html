第461章：地の領域へ

夜が明けた。カイトは宿屋の窓から東の空を眺めていた。地平線が金色に染まり、新しい一日が始まろうとしている。

「起きてたのか、カイト」

リアムが部屋に入ってきた。手には、温かいスープの入った鍋を持っている。

「朝食、持ってきたよ。エリナが作ったんだ」

カイトは窓辺から離れ、テーブルについた。スープの香りが鼻をくすぐる。シンプルだけど、心が温まる味だ。

「ありがとう。ちょうど腹が減ってたところ」
「昨日の雷の領域、大変だったんだって？」

カイトはスープをすすると、昨日の出来事を簡潔に説明した。ライラとの出会い、雷撃を受け止めたこと、そして管理者としての可能性について。

「管理者、か……。カイト、どうするつもりなんだ？」
「まだ分からない。でも、この世界の魔法システムを理解したい。それが最初の一歩だと思ってる」

リアムは頷いた。

「僕たちは、どこまでもついていくよ。カイトが決めた道なら」

その言葉に、カイトは少し救われた気がした。一人で抱え込む必要はない。仲間がいる。それが、この旅の最大の武器だ。

◆ ◆ ◆

一行は朝のうちに街を出発した。目的地は、大陸の中央に位置する「大地の背骨」と呼ばれる巨大山脈。地の賢者テラドンは、その山脈の最深部に住んでいるという。

「大地の背骨か……。名前からして、やっかいそうね」

エリナが地図を見ながら言った。

「標高三千メートル以上。空気が薄くて、寒さも厳しい。それに、地魔法の影響で重力が不安定な場所があるらしいわ」
「重力、不安定？」

カイトが聞き返すと、エリナは頷いた。

「地上なのに空に浮いちゃったり、逆に地面に押し付けられたり。地の賢者の力が、地形そのものを変えているのよ」

カイトは思考を巡らせた。重力制御。物理エンジンの最も複雑な部分の一つだ。ゲーム開発でも、重力の計算は骨が折れる。ましてや、現実の物理法則を書き換える魔法となれば。

「面白そうだね」

カイトの口から、自然とそんな言葉が出た。

「またそういうことを……」

セシリアが呆れたように言った。

「カイト様は、本当に変わっていらっしゃる。普通なら恐怖するような状況を、楽しんでいるんですから」
「だって、プログラマーだもん。未知のシステムに触れるのは、最高の楽しみだよ」

三人は顔を見合わせて、苦笑した。でも、その表情には信頼があった。この奇妙な主人公についていくことは、決して退屈ではないだろう。

◆ ◆ ◆

街道を進むこと三日。一行は「大地の背骨」の麓に到着した。眼前にそびえ立つのは、雲を突き抜けるような巨大な山々。その威容は、言葉では言い表せない。

「これが、地の賢者の領域……」

エリナが息を呑んだ。

「魔力が、濃い。空気そのものが重い感じ」

カイトも感じていた。ここ一帯は、特別な場所だ。地面から湧き上がるような魔力が、全身を包んでいる。


function analyzeEnvironment() {
    const manaDensity = detectManaDensity();
    const gravityVector = measureGravity();
    const elementalAffinity = checkEarthElement();
    return {
        density: manaDensity,
        gravity: gravityVector,
        earth: elementalAffinity
    };
}

カイトは無意識に解析魔法を走らせた。結果は、驚くべきものだった。

「魔力密度、通常の百倍以上。重力ベクトル、不規則に変動中。地属性親和性……計測不能」

カイトが結果を口にすると、リアムが眉をひそめた。

「計測不能？ どういうことだ？」
「地属性の魔力が、あまりに巨大すぎて測定範囲を超えている。これは、桁違いの何かがある」

カイトの言葉に、全員が緊張した。これまでの賢者たちも強力だった。でも、ここは別次元だ。地そのものが、賢者の一部になっているような感覚。

「行こう。ここで立ち止まっていても、何も始まらない」

カイトが先頭を歩き出した。その背中を、三人が追う。

◆ ◆ ◆

山道を登り始めて数時間。景色は劇的に変化していた。木々は姿を消し、代わりに巨大な岩々が点在している。そして何より、重力がおかしい。

カイトは一歩踏み出した瞬間、体がふわりと浮くのを感じた。

「うわっ！」

慌ててバランスを取るが、今度は地面に強く引っ張られた。重力が、場所によって全く違うのだ。

「みんな、気をつけて！ 重力が不安定よ！」

エリナが叫んだ瞬間、彼女自身が空中に浮き上がった。

「きゃあ！」

カイトはとっさに手を伸ばし、エリナの手を掴んだ。

「 gotcha！ 」

二人は重力の影響を受けないエリアまで、互いに引っ張り合いながら移動した。

「ありがとう、カイト。危ないところだったわ」
「どういたしまして。でも、この重力異常、何とかしないと先に進めないね」

カイトは周囲を観察した。重力が強い場所、弱い場所、逆方向に引っ張られる場所。それらはランダムに見えるが、よく見るとパターンがある。


function mapGravityField() {
    const points = [];
    for(let x = 0; x < 100; x++) {
        for(let y = 0; y < 100; y++) {
            const g = measureGravityAt(x, y);
            points.push({x, y, g});
        }
    }
    return findPattern(points);
}

カイトは目を閉じ、意識を集中させた。重力のパターンを解析する。それはまるで、複雑な迷路の地図を作成するような作業だった。

「分かった。重力の変動には周期がある。三十秒ごとに、パターンが入れ替わる」

カイトは目を開けた。

「今から三十秒間は、左側のルートが安全。その次の三十秒は、右側。この周期で進めば、重力異常に巻き込まれない」
「すごい……。そんなこと、どうやって分かるの？」

セシリアが驚いたように聞いた。

「重力の波形をフーリエ変換したら、きれいな正弦波が出てきたんだ。物理法則をコードとして読めば、意外とシンプル」

三人は顔を見合わせた。カイトの説明は、半分も理解できていない。でも、結果が正しいことだけは確かだ。

「じゃあ、左ルートを行こう。三十秒以内に次の安全地帯まで走るわよ！」

エリナの号令で、一行は走り出した。

◆ ◆ ◆

重力異常地帯を抜けた先に、巨大な洞窟の入り口が現れた。入り口には、古代文字で何かが刻まれている。

「これは……古の言語ね」

セシリアが近づいて、文字を解読し始めた。

「『大地の子よ、ここより先は禁忌の領域。地の賢者の許可なき者、立ち入るべからず』……だそうよ」
「警告文、ってことか」

リアムがつぶやいた。

「でも、私たちは賢者に会いに来たんだ。立ち止まるわけにはいかない」

カイトは洞窟の入り口に手を触れた。冷たい岩の感触。そして、微かな振動。地脈が、どこかで脈打っている。

「中に入ろう。テラドンさんが待っているはずだ」

カイトが一歩踏み出すと、洞窟の奥から声が響いてきた。

「待っていたぞ、若き訪問者よ」

その声は、岩が擦れ合うような低く重い響きを持っていた。

「ようやく来たか。七つの属性の最後の一つ、地の賢者に会うために」

洞窟の闇が揺らいだ。そして、巨大な影が姿を現した。身長三メートルはあろうかという大男。肌は岩のような灰色で、髭は植物の根のように絡み合っている。その瞳だけが、琥珀色に輝いていた。

「わしはテラドン。地の賢者にして、この大地の守護者じゃ」

圧倒的な存在感。カイトは思わず息を呑んだ。これまでの賢者たちとは、明らかに違う。彼は、地そのもののようなのだ。

「カイト、じゃな？ お主の話は、ライラから聞いておる」
「はい。お会いできて光栄です、テラドンさん」

カイトは深く頭を下げた。

「礼には及ばん。むしろ、わしの方から礼を言いたい。ここまで辿り着いた勇気に対してな」

テラドンは少しだけ笑ったように見えた。

「さて、用件は分かっておるじゃろう。お主に課す試練についてじゃ」

カイトは覚悟を決めて頷いた。

「はい。何をすればいいんですか？」

テラドンは洞窟の奥を指差した。

「この洞窟の最深部に、『大地の心臓』がある。そこに辿り着き、大地の鼓動を感じ取ること。それが、わしの試練じゃ」
「大地の、心臓？」
「そうじゃ。この世界の地脈が交わる、最も神聖な場所。そこで、お主は何かを見つけることじゃろう」

テラドンの言葉には、含みがあった。何かを見つける？ 何を？

「行って参ります」

カイトは仲間たちに目配せをした。四人は、洞窟の奥へと歩き出した。

地の領域の試練が、今、始まろうとしていた。
