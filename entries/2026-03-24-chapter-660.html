第660章：炎の試練

第四層をクリアしたカイトたちは、螺旋階段を上りながら第五層へと向かっていた。

ルシフェインから得た情報は重要だった。第一席オメガが第七層で境界崩壊の儀式を準備しているという事実。そして、オメガに接触した者は戻ってこないという警告。

「私の風魔法なら、炎に対抗できるかもしれません。でも、相手が十三使徒なら、普通の炎じゃないでしょう」

リアムがため息をついた。

「また苦戦しそうだな」

階段の先に、赤黒い光が漏れていた。熱気が漂い始め、空気が揺らぐ。第五層の入り口が近づいている証拠だ。

カイトは身構えた。

「みんな、準備して。第五層——炎の領域だ」

巨大な扉の前で、カイトたちは立ち止まった。扉の表面には、炎の紋様が刻まれている。触れれば火傷しそうなほどの熱気を感じる。


function analyzeFireBarrier() {
    const barrier = detectFireStructure();
    
    // 炎の結界を解析
    const analysis = {
        type: 'inferno_barrier',
        temperature: 'extreme',
        source: 'destruction_core',
        weakness: 'simultaneous_cooling'
    };
    
    // 侵入方法を特定

        entry: entryMethod
    };
}

そこは、灼熱の世界だった。床は溶岩で覆われ、壁からは炎が噴き出している。空気そのものが燃えるような熱さで、呼吸するだけで喉が焼けるようだ。

「ここは……地獄か？」

リアムが顔をしかめた。

「熱すぎる。まともに戦えるのか？」

function thermalProtectionSphere(allies) {

}

「第七席のルシフェインがやられたと聞いた。心の闇を操る者だったが、所詮は精神攻撃に過ぎん。私の炎は、魂ごと焼き尽くす！」

「デストラクション……君の目的は何だ？ なぜ、境界を崩壊させようとしている？」

「目的？ 決まっているだろう——破壊だ！ この世界は、あまりにも脆い。境界という無駄な制約に縛られている。だから、すべてを焼き尽くし、新たな世界を創るのだ！」

リアムが呆れたように呟いた。

function flamePathDisruption(target) {
    // 炎の流れを逆転させる
    const flameFlow = analyzeFlameCurrent(target);
    const reverseFlow = invertFlowDirection(flameFlow);
    
    // 干渉フィールドを展開
    const disruptionField = createInterferenceMatrix(reverseFlow);
    
    // 核心（心臓）への道を確保
    const pathToCore = findCorePath(target, disruptionField);
    
    return {
        type: 'flame_disruption',
        corePath: pathToCore,
        duration: 5 // 5秒間のみ有効
    };
}

function combinedAssault(attacks) {
    // 全員の攻撃を統合
    const combinedEnergy = merge(attacks.map(a => a.energy));
    
    // 核心に集中

    return {

function voidAbsorption(explosion) {
