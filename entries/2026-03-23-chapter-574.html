第574章：炎の賢者

遺跡の最深部へ続く回廊は、長かった。

壁に刻まれたレリーフが、カイトの足元を照らしている。炎を模した文様が、一定のリズムで明滅していた。まるで、呼吸をしているかのように。

「……生きている遺跡」

カイトは呟いた。この場所全体が、何らかの魔力で維持されている。単なる遺跡ではない。イグニス自身の魔力が、この空間を形作っているのだ。

回廊の先に、重厚な扉が見えた。

扉には、巨大な鳥のレリーフが彫られていた。翼を広げ、口から炎を吐く様子——フェニックスだ。炎の守護獣の姿。

カイトは扉の前に立ち、深呼吸をした。

「……イグニス。来たよ」

静かに声をかける。

返答はなかった。だが、扉が音もなく開いた。

扉の奥は、巨大な空間だった。

円形の広場。床には幾何学模様が描かれ、中央には祭壇のようなものがある。その上に——赤い光が渦巻いていた。

光が凝縮し、人の形をとっていく。

長い赤髪。琥珀色の瞳。褐色の肌に、炎の紋様が浮かぶドレスを纏った女性。火の賢者イグニスが、そこに立っていた。

「久しぶりね、管理者」

彼女の声は、燃え盛る炎のように熱く、そして鋭かった。

「炎の試練から……三年ぶりかしら」
「そうなるな。……変わってないな、イグニス」

カイトの言葉に、イグニスは小さく笑った。

「私は時間に縛られない存在。……さて、何をしに来たの？」
「フィンブルヴェトを倒した後、封印について調べた。……残る六つの災厄、その中で第二の封印——炎の災厄について、知りたい」

イグニスの表情が、少し真剣になった。

「……フロストから聞いたのね」
「ああ。炎の災厄——『灼熱のクアドラ』のことだ」

イグニスは祭壇の縁に腰掛けた。足を組み、カイトを見据える。

「クアドラは、三百年前に私が封印した。……でも、完全な封印ではなかった」
「なぜ？」
「私の力が、足りなかったから」

イグニスの声に、悔恨の響きが混じった。

「クアドラは、火属性の災厄の中でも最強クラス。……炎そのものを生み出す『創炎の能力』を持っている。私の炎を凌駕する、原初の炎」
「原初の……炎？」

カイトは眉をひそめた。

「世界が生まれた時、最初に存在した七つの原初魔法。その一つが『炎』。クアドラは、その原初の炎を体現した存在なの」

イグニスは立ち上がり、カイトに近づいた。

「あなたが使っている魔法——コードで記述するタイプよね」
「……そうだ」
「クアドラも、同じようなものだったわ。……ただ、彼女の場合は、炎そのものがコード。燃えることが、彼女の存在定義」

カイトは言葉の意味を理解しようとした。

「炎そのものがコード……？つまり、彼女を倒すには、その『炎のコード』を書き換える必要がある？」
「正解。……でも、簡単じゃない」

イグニスはカイトの目を真っ直ぐに見た。

「あなたのコード書き換え能力——『Magic.override』。それがあれば、理論上はクアドラの炎を無効化できる。……でも、彼女の炎は私のものより強い。あなたの魔力が足りなければ、逆に焼き尽くされる」
「……試してみる価値はある」
「そうね。……でも、その前に」

イグニスがニヤリと笑った。

「私との試練、クリアできるかしら？」

カイトは身構えた。

「試練……か。炎の試練の時と同じか」
「いいえ。今回は、もっと実戦的よ」

イグニスが手をかざすと、広場の床から炎が噴き出した。炎が集まり、二体の火の精霊の形をとる。

「この火精霊を、三十秒以内に倒して」
「三十秒？」
「私の炎から生まれた精霊。普通の魔法じゃ、倒せない。……あなたのコード魔法で、どう対処するか見せて」

精霊たちが、カイトに向かって飛びかかってきた。

速い。

カイトは瞬時に反応した。

Magic.override(
  target: "火精霊A",
  property: "combustion_state",
  newValue: "inactive"
)

左の精霊が、一瞬で炎を失い、消滅した。

だが、右の精霊はカイトの背後に回り込んでいた。

Magic.bind(type: "flame_chains", target: "火精霊B", duration: 10)

炎の鎖が精霊を縛り上げる。隙を見て——

Magic.override(
  target: "火精霊B",
  property: "heat_output",
  newValue: 0
)

精霊が崩れ落ちた。

経過時間——十八秒。

「……どうだ」

カイトがイグニスを見る。

イグニスは、満足そうに頷いた。

「いい動き。……前に会った時より、明らかに強くなっている」
「色々な経験をしたからな」
「ふふ。……じゃあ、次の試練に行きましょうか」
「まだあるのか」

カイトが少し呆れる。

「本番よ。……私と、戦って」

「……お前と？」

「制限時間は一分。私の攻撃を一分間耐え抜けたら、クアドラについて知っている全てを教える」

イグニスの全身から、炎が噴き出した。

圧倒的な熱量。カイトの肌が、ただ近くにいるだけで焼けるような感覚に陥る。

「……いいだろう」

カイトは覚悟を決めた。

イグニスが動いた。

瞬きする間に、カイトの目の前に迫る。右手から放たれた炎の刃——

Magic.shield(type: "thermal_resistance", intensity: 0.9)

熱耐性の障壁を展開。だが、衝撃までは防げない。カイトは後方へ弾き飛ばされた。

「遅い」

イグニスの追撃。炎の弾丸が、雨のように降り注ぐ。

Magic.create(type: "wind_barrier", radius: 3, intensity: "maximum")

風の障壁で弾丸を逸らす。だが、数発が障壁を突破し、カイトの肩をかすめた。

「くっ……！」

痛みが走る。だが、立ち止まっている暇はない。

イグニスが再び接近してくる。今度は、両手に巨大な炎の槍を構えていた。

「終わりよ！」

槍が突き出される。

カイトは——あえて、前に踏み込んだ。

Magic.override(
  target: "self",
  property: "temporal_perception",
  newValue: "accelerated",
  factor: 3
)

時間の感覚を加速させる。カイトにとって、イグニスの動きが三分の一の速度に見えた。

槍の軌道を読み、最小限の動きで回避。そして——

Magic.override(
  target: "ignis_flame",
  property: "temperature",
  newValue: 0
)

イグニスの炎を、一時的に無効化した。

無防備になったイグニスの腹部に、カイトの手刀が入る。

「……！」

イグニスが一瞬たじろぐ。

カイトはすぐに距離を取った。

Magic.override(
  target: "self",
  property: "temporal_perception",
  newValue: "normal"
)

時間感覚を戻す。身体への負担は大きいが、一分を耐え抜くには必要だった。

イグニスが、再び炎を纏う。

「……やるじゃない」

彼女は少し驚いたような表情を見せた。

「自分の炎を無効化するなんて……前の管理者でもできなかったわ」

「……褒め言葉として受け取っておく」

カイトは荒い息を吐きながら答えた。

残り時間——十五秒。

イグニスが、再び構えた。

「最後の一撃、行くわよ」

彼女の全身から、今まで以上の炎が噴き出した。広場全体が、熱気に包まれる。

「原初の炎——ほんの少しだけ、見せてあげる」

イグニスの手のひらに、青白い炎が宿った。普通の炎とは明らかに違う。澄んだ、透明に近い炎。

「これは……」

カイトは本能的に理解した。これが、クアドラが持つ炎と同じ種類のものだ。

イグニスが、青い炎を放った。

直線的な攻撃ではない。広場全体を包み込むような、広範囲の熱波。

カイトは全ての魔力を、防御に注ぎ込んだ。

Magic.override(
  target: "self",
  property: "heat_resistance",
  newValue: "absolute",
  duration: 5
)

Magic.create(type: "barrier", effect: "reflect", radius: 2)

絶対的な熱耐性と、反射障壁。二重の防御。

熱波が、カイトを襲った。

視界が真っ白になる。身体が焼けるような感覚。だが——耐えた。

障壁が砕け散り、カイトは膝をついた。だが、意識はある。

熱波が収まった。

静寂が戻る。

イグニスが、カイトに近づいてきた。

「……一分、経過」

彼女は静かに言った。

「合格よ、管理者」

カイトは安堵の息を吐いた。

「……助かった」

「私の原初の炎、耐え抜いたのはあなたが初めて。……認めるわ」

イグニスは微笑んだ。初めて見る、優しい表情だった。

「クアドラの情報、約束通り教える。……座りなさい」

彼女は祭壇の縁に腰掛け、カイトも隣に座った。

イグニスの話は、三百年前に遡った。

「クアドラは、元々人間だったわ」
「……人間？」

カイトは驚いた。

「そう。……三百年前、火の賢者として選ばれた女性。炎の魔法に、誰よりも秀でていた」

イグニスの目が、遠くを見つめる。

「彼女は、原初の炎に近づこうとした。……そして、それに取り込まれた」

「取り込まれた？」

「原初の炎は、意思を持っている。……クアドラの魂を飲み込み、災厄として転生させたの」

カイトは息を飲んだ。

「じゃあ、彼女を救うには——」

「原初の炎を彼女から引き剥がす必要がある。……でも、それは私の力では不可能だった」

イグニスは拳を握りしめた。

「だから、封印した。……時間を稼ぐために」

「何から？」

「十三使徒よ」

カイトは眉をひそめた。

「十三使徒……以前、戦ったことがある」
「彼らは、災厄を解放しようとしている。……クアドラも、その対象」

イグニスはカイトの方を向いた。

「クアドラを救えるのは、あなたしかいない。……原初の炎を書き換えられるのは、あなたのコード魔法だけだから」

「……わかった」

カイトは頷いた。

「彼女を救ってみせる。……そのために、力を貸してくれ」

「ええ。……私の炎、あなたに預ける」

イグニスが手をかざすと、カイトの chest に炎の紋章が浮かんだ。

「これは？」

「火の賢者の印。……私の力の一部を、あなたに譲渡した」

「譲渡？そんなことしていいのか」

「かまわない。……私は、もう三百年も戦い続けてきた。少し休んでもいい頃でしょう」

イグニスは微笑んだ。

「クアドラを救ったら、また会いに来て。……あなたの成長した姿、見たいから」

「……約束する」

カイトは立ち上がった。

「ありがとう、イグニス。……期待に応える」

「ええ。……行って」

遺跡を出ると、夕日が沈んでいた。

エリナ、リアム、シオンが、心配そうに待っていた。

「カイト！」

エリナが駆け寄ってくる。

「大丈夫？怪我はない？」

「ああ、大丈夫だ。……試練、クリアしたよ」

カイトは胸の炎の紋章を見せた。

「これは……火の賢者の印？」

リアムが驚く。

「イグニスから譲り受けた。……炎の魔法が、より強力になった」

「すごい……カイト、かっこいい！」

シオンが目を輝かせる。

カイトは仲間たちに向き直った。

「次の目的地が決まった。……第三の封印——闇の災厄の場所へ向かう」

「闇の災厄？」

エリナが問う。

「ああ。……十三使徒が狙っているのは、災厄たちの解放。急がないと、先を越される」

カイトは西の空を見上げた。

夕日が、血のように赤く染まっていた。

「……闇の災厄は、北にある。急いで向かおう」

「うん！」

三人が同時に頷く。

カイトたちは、砂漠を後にした。

新たな力を手に入れ、次なる戦いへ向かって。
