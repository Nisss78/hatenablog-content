第136章：西からの風

翌朝、カイトは宿屋の一室で結晶と向き合っていた。

窓から差し込む朝日に照らされた結晶は、静かな青色を放っている。昨夜の脈動が嘘のように、今は穏やかだった。


class CrystalAnalysis {
    anomalies = [];
    
    addAnomaly(data) {
        this.anomalies.push({
            ...data,
            timestamp: Date.now(),
            investigated: false
        });
    }
    
    prioritize() {
        return this.anomalies.sort((a, b) => {
            const severity = { high: 3, medium: 2, low: 1 };
            return severity[b.severity] - severity[a.severity];
        });
    }
}

「西の大陸か……」

カイトは呟いた。結晶が示した異常は、西の大陸の山岳地帯で発生していた。東の森の花と同じような魔力反応。だが、規模はより大きい可能性があった。

ドアがノックされ、エリナが入ってきた。

起きてたの？

「ああ。ちょっと調べてた」

カイトは結晶を懐にしまった。エリナは彼の隣に座った。

西の大陸のこと？

「うん。放置するわけにはいかないんだろうけど、どう動くべきか迷ってる」

エリナは少し考えた。

まずは情報を集めるべきじゃない？ 王都なら、西の大陸の情報を持ってる商人もいるかもしれないし

「そうだな」

カイトは頷いた。焦って現地に向かうより、まずは状況を把握する。それが賢明な判断だった。

——

カイトとエリナは、王都の市場に向かった。

朝の市場は活気に満ちていた。野菜や果物を売る商人、香辛料の香りが漂う露店、布地を広げる商人。様々な人々が行き交い、賑やかな声が響き渡る。

あそこに、遠方の商人が集まる場所があるわ

エリナが案内してくれた。市場の奥には、少し大きめの建物が並んでいた。ここは、他国からの商人が情報交換する場所らしい。

二人は建物に入り、西の大陸について詳しそうな商人を探した。

西の大陸？ 最近は行ってないね

最初の商人は首を振った。

山越えが危なくなってるって話だ。何かが起きてるらしいが、詳しくは知らん

二人は別の商人に話を聞いた。

うちの組合員が先月、山岳地帯を通ろうとしたんだが、断念したそうだ

中年の商人が、深刻な表情で語った。

夜になると、山全体が光るんだと。青白い光が。それを見た者は、体が動かなくなったり、変な夢を見たりするらしい

カイトとエリナは顔を見合わせた。東の森の花と同じ現象だ。

その光を見た人は、その後どうなりましたか？

エリナが問いかけた。

死んだりはしてない。ただ、数日間呆然としてたって話だ。知らない言葉を喋ったり、見たこともない場所のことを話したりしたらしい

「知識の庭の影響か……」

カイトは小声で呟いた。リアムが経験したことと同じだ。だが、山岳地帯ではより多くの人々が影響を受けている。

商人に礼を言って、二人は建物を出た。

「行くしかないな」

カイトが言った。エリナも頷いた。

ええ。でも、準備が必要よ。山越えは簡単じゃないし

「リアムたちにも相談しよう」

——

宿屋に戻ると、リアムとリリス、セシリア、アルフレッドが食堂に集まっていた。

カイトは西の大陸の情報を皆に伝えた。

山全体が光る、か

リアムが腕を組んだ。

東の森の花より、規模がデカいってことだな

「たぶん。知識の種が、山岳地帯に大量に散らばってるのかもしれない」

セシリアが冷静に分析した。

問題は、その影響をどう最小限に抑えるかです。商人の話では、商人以外にも現地の住民がいるはず。彼らへの影響が心配です

「山岳地帯には、鉱山町があるはずだ」

アルフレッドが口を開いた。

かつて、西の大陸は豊富な鉱脈で知られていました。魔石の産出も多かったと聞きます

カイトの脳内で、結晶が微かに脈打った。

リリスが覗き込んだ。

全員が頷いた。

準備は順調に進んだ。

リアムが馬と食料を手配し、エリナが薬草や治療道具を揃えた。セシリアは西の大陸の地図を探し出し、アルフレッドは地元のガイドを探した。

リリスはというと、何やら膨らんだ荷物を抱えてきた。

これ、何だと思う？

「さあ」

古本屋で見つけた、西の大陸の伝承集。読んどいたら役に立つかも

カイトは感心した。リリスはこういうところが抜け目ない。

「ありがとう。夜に読んでおくよ」

その夜、カイトはリリスが借りてきた伝承集を読んでいた。

西の大陸には、かつて「星の民」と呼ばれる集団が住んでいたらしい。彼らは夜空の星を読み、未来を予言する能力を持っていたという。

『星の民は、管理者たちと共に暮らしていた。彼らは管理者から知識を授かり、それを次世代に伝える役割を担っていた』

馬に乗って西へ向かう。王都を出てしばらくは平原が続き、やがて丘陵地帯へと変わっていく。道中は穏やかで、特に問題は起こらなかった。

三日目の夕方、彼らは山岳地帯の入り口にある小さな村に到着した。

ここが、最後の足場になる

アルフレッドが言った。村は十数軒の家が集まる、小さな集落だった。鉱山で働く人々の村らしく、男たちの服は煤で汚れていた。

村の入り口で、一人の老人が彼らを呼び止めた。

お前さんたち、山に登るつもりか？

「ええ」

カイトが答えると、老人は深く皺の刻まれた顔をしかめた。

やめときな。最近、山がおかしいんだ。夜になると光るし、変な音もする。先月、若い衆が三人行ったきり戻ってこなかった

カイトとエリナは顔を見合わせた。

その三人は、見つからなかったんですか？

エリナが問いかけた。

見つかったよ。山の入り口で、呆然と立ってた。何も喋らんし、目も焦点が合ってなかった。今は村の家で寝かせてるが、一向に目覚める気配がない

「会わせてもらえますか？」

カイトが言った。老人は怪訝な顔をしたが、やがて頷いた。

彼らの表情は穏やかだったが、目は開いたままだった。瞳の中に、微かな青い光が宿っている。

リアムが口を開いた。

俺はどうやって戻れたんだ？

「君の場合は、花を一つだけ触ったからだ。でも、ここは違う。山全体が知識の源になってる。彼らは、その中心に入り込んでしまったんだろう」

セシリアが問いかけた。

戻す方法は？

カイトは少し考えた。

「かもしれない、か」

確実ではないが、やるしかないということですね

「そうだ」

カイトは立ち上がった。

「明日の朝、山に入る」

——

その夜、カイトは村の外れで星を眺めていた。

山の方角から、微かな青い光が漏れている。昼間は見えないが、夜になると確かに光っていた。

エリナが隣に来た。

眠れない？

「うん。ちょっと考えごとをしてた」

何を？

カイトは結晶を取り出した。

「アークライトたちが僕にこれを託したのは、何か理由があったはずだ。ただ世界を守れと言っただけじゃない。僕にしかできない何かが、これから起きるんだと思う」

エリナは少しの間、カイトを見つめていた。そして、優しく言った。

一人で背負い込まないで。私たちがいるんだから

「……ありがとう」

カイトは結晶を握りしめた。明日、山に入れば、何かがわかるはずだ。星の民が何を残したのか。知識の種が何を伝えようとしているのか。

風が西から吹いてきた。山の方角から。その風には、微かな魔力の香りが混じっていた。
