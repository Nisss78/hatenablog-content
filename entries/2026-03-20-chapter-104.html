第104章：回廊の少女

夜明け前の空は、まだ濃紺の帳（とばり）を下ろしていた。廃墟の庭園に、朝霧が漂い始める。カイトはテントの外で、焚き火の残り火を見つめていた。眠れなかったわけではない。ただ、思考が止まらなかっただけだ。

「起きるの早いわね」

エリナが、毛布を肩に巻きながら近づいてきた。寝起きでも、彼女の琥珀色の瞳は澄んでいる。

「考え事をしてたら、な。始原の図書館について、まだわからないことが多すぎる」

エリナは彼の隣に座り、焚き火に枝をくべた。炎が踊り、二つの影を長く伸ばす。

「不安？」
「……正直に言えば、少しな。世界そのものの設計図なんて、見つけてどうするつもりなのか、自分でも完全にはわかっていない」

カイトは自嘲気味に笑った。エンジニアとして、ドキュメントのない巨大なレガシーシステムに放り込まれた気分だ。しかも、そのシステムは「世界」そのものだ。

「あなたらしくないわね」

エリナが、彼の方を向いた。真剣な眼差し。

「あなたはいつも、わからないことをわかるまで突き詰める人でしょ？ それがカイトよ」

その言葉に、カイトは少し救われた気がした。彼は深呼吸をして、夜明けの空気を肺いっぱいに吸い込んだ。

「……そうだな。まずは情報を集めることから始めよう」

空が白み始め、仲間たちが次々と起き出してきた。リアムが剣の手入れを終え、アルフレッドが周囲の安全を確認し、セシリアが転移魔法の準備に取りかかる。

「座標計算完了。ただし、始原の図書館の周辺は空間が不安定だから、少し誤差が出るかもしれない」
「どれくらいの誤差だ？」

カイトの問いに、セシリアは少し困った顔をした。

「……数十キロ、かも」

「数十キロ」

リアムが絶望的な声を上げた。

「徒歩で丸一日かかるじゃないか」
「文句を言うな。命があっただけマシだと思え」

セシリアが舌を出し、魔法陣を展開し始めた。複雑な幾何学模様が、朝の光の中で美しく輝く。

// Teleportation Spell
// Destination: Primordial Library (Approximate)
// Coordinates: X:7824.3, Y:2191.7, Z:156.8
// Error Margin: ±32.5 km
// Mana Cost: 2,400 MP per person

カイトの視界に、例のステータス画面が浮かび上がる。彼はその数値を確認し、小さく頷いた。

「全員、準備いいか？ 転移魔法は少し酔うかもしれないが、我慢してくれ」

四人が頷き、魔法陣の中心に集まる。セシリアが詠唱を始めると、世界が一瞬で反転した。

眩暈。吐き気。視界の歪み。

転移の余韻が収まると、カイトたちは見知らぬ場所に立っていた。一面に広がるのは、白銀の霧。視界は数十メートル先で遮られている。

「成功したのか、これ？」

リアムが剣を抜き、警戒態勢を取った。

「空間座標は……間違っていないはずよ。ただ、ここは図書館の内部じゃなくて、周辺の『回廊』ね」

セシリアが、魔導書のページをめくりながら答えた。

「始原の図書館は、無限の回廊と呼ばれる空間迷宮の中に存在する。本来なら、図書館への直通ルートを確保するはずだったんだけど……」

「誤差があったってことか」

カイトが周囲を見渡した。霧の向こうに、何かが見え隠れしている。巨大な柱か、あるいは建物の壁か。定かではない。

「とにかく、進もう。この霧の中に留まっていても何も始まらない」

カイトの判断に、全員が従った。アルフレッドが先頭に立ち、警戒を続けながら一行は霧の中を進んでいく。

十分ほど歩いた頃だろうか。霧が突然晴れ、広大な空間が現れた。そこは、まるで異界だった。

空には、二つの月が浮かんでいる。一つは白く、もう一つは赤い。地面は幾何学模様のタイルで敷き詰められ、その先には、天を突くほどの巨大な塔が聳え立っていた。

「あれが……始原の図書館？」

エリナが息を呑む。塔は、目測で数キロメートルの高さがある。窓のような開口部が無数に並び、その一つ一つから、薄い光が漏れ出していた。

「ああ。間違いない」

セシリアの声が震えていた。

「伝説通りの規模……これだけの構造物を維持するには、一国の魔力消費量に匹敵するエネルギーが必要よ」

カイトは、塔を眺めながら思考を巡らせた。システム管理者としての視点が自然と働く。この構造物は、単なる図書館ではない。データセンターだ。しかも、とてつもなく大規模な。

STRUCT PrimordialLibrary {
    Floors: Infinite;
    StorageCapacity: Unlimited;
    AccessControl: Layered;
    Administrator: Unknown;
    Purpose: WorldDesign;
}

彼の脳内で、概要が浮かび上がる。世界の設計図を保管する場所。であれば、当然、強固なセキュリティが存在するはずだ。

「警戒を怠るな。ここは、管理者たちでさえ容易には立ち入れない場所だと言っただろう？」

カイトの警告に、全員が頷いた。

その時だった。

「……誰か、いますわね」

声がした。幼く、しかし妙に落ち着いた声。カイトたちは一斉に声の方を振り向いた。

そこにいたのは、一人の少女だった。年齢は十歳前後だろうか。白銀の髪を腰まで伸ばし、黒いゴシックドレスを身に纏っている。手には、身長ほどもある巨大な杖を抱えていた。

「誰だ？」

リアムが剣を構えた。だが、少女は怯むことなく、むしろ楽しそうに笑った。

「剣を向けるのはやめてくださいな。私は敵ではありませんの」

少女は優雅に一礼し、自己紹介を始めた。

「私は『リリ』。この回廊の案内人をしておりますわ」
「案内人？」

カイトが問い返す。少女──リリは、にっこりと微笑んだ。

「ええ。始原の図書館へ向かう方々を、お守りするのが私の役目ですの」

その言葉に、セシリアが眉をひそめた。

「待って。始原の図書館の案内人なんて、聞いたことがない。伝承にもそんな記述は……」
「それはそうですわ」

リリが、悪戯っぽく笑った。

「私は、ごく一部の方にしかお目見えいたしませんの。特に……『異邦人』の方には」

カイトの心臓が跳ねた。異邦人。その言葉が、彼を指しているのは明らかだ。

「僕のことを知っているのか？」
「もちろんですわ」

リリが一歩近づいてきた。その瞳は、不思議な色彩をしていた。右目は金色、左目は銀色。オッドアイ。

「カイトさん。いえ、本来のお名前で呼ぶべきかしら？ 海藤翔太さん」

その名前を聞いた瞬間、周囲の空気が凍りついた。エリナ、リアム、セシリア、アルフレッド──全員が驚愕の表情を浮かべている。

「なっ……！？」

カイトは反射的に後退り、防御態勢を取った。

「お前、何者だ！ なぜその名前を知っている！」

リリは、怯むことなく、ただ静かに微笑んでいた。

「落ち着いてくださいな。私は、あなたの敵ではありませんの」

彼女は、杖の先端を地面に向け、ゆっくりと語り始めた。

「あなたがこの世界に召喚された経緯。あなたが持つ能力の本質。そして、あなたがこの世界で果たすべき役割……すべて、私は知っていますわ」
「知っているだと？」
「ええ。なぜなら、私は『記録者』ですから」

記録者。その言葉に、カイトはピンときた。

「まさか、始原の図書館の……」
「そうですわ。私は、この世界のすべての情報を記録し、管理する存在の一人」

リリは、両手を広げた。

「始原の図書館は、単なる書物の保管庫ではありません。世界そのものの設計データを格納した、巨大なアーカイブなのですわ」

カイトは、息を呑んだ。予想通りだが、実際にそうだと断言されると衝撃的だ。

「であれば、僕がこの世界に来た理由も……？」
「ええ。知っていますわ」

リリの表情が、少しだけ真剣なものになった。

「あなたは、『バグ』ですの」
「……バグ？」
「この世界のシステムにとって、予期せぬ存在。本来存在しないはずの変数。それが、あなたですわ」

カイトは、苦笑した。異世界転生してまでバグ扱いとは、なかなか皮肉な運命だ。

「で、そのバグに対して、システムはどうするつもりなんだ？」
「二つの選択肢がありますわ」

リリが、指を二本立てた。

「一つは、排除。バグを修正し、システムを正常な状態に戻すこと」
「もう一つは？」
「統合。バグを新たな機能として組み込み、システムをアップデートすること」

カイトは、その言葉を咀嚼した。排除か、統合か。プログラマーとして、非常になじみ深い選択肢だ。バグを発見した時、その場で潰すか、あるいは仕様として受け入れて設計を変更するか。

「で、どっちになるんだ？」
「それは、あなた次第ですわ」

リリは、にっこりと笑った。

「あなたがどれだけの価値を示せるか。それが、この世界の運命を決めますの」

彼女は杖を振り、空中に魔法陣を展開した。その中に、複雑なコードのような文字列が浮かび上がる。

// System Status
// Anomaly Detected: Entity "KAITO"
// Origin: External Dimension
// Threat Level: Unknown
// Potential: Maximum
// Action Required: Observation
「今のところ、システムは『観察モード』ですわ。あなたが敵対するのか、それとも協力するのか──それを見極めようとしていますの」

カイトは、拳を握りしめた。観察されている。最初から、すべて見られていたのだ。

「であれば、余計に始原の図書館に行く必要があるな。この世界のシステムについて、もっと詳しく知るためには」
「もちろんですわ。案内いたしましょう」

リリは、杖を指し示した。その先には、始原の図書館への道が続いている。

「ただし、警告しておきますわ。図書館の内部は、あなたが想像する以上に危険ですの。知識は力ですが、同時に呪いにもなりますから」

カイトは頷いた。覚悟はできている。

「行こう。みんな」

エリナ、リアム、セシリア、アルフレッドが、それぞれの決意を胸に頷いた。

「待って、カイト」

出発しようとした瞬間、エリナが彼の腕を掴んだ。

「あの子、さっき『本来のお名前』って言ったわよね。海藤翔太って……それって、あなたの元の世界での名前？」

カイトは、一瞬躊躇した。彼はこれまで、自分の過去について多くを語っていない。信頼していないわけではない。ただ、話すべきタイミングが見つからなかっただけだ。

「……ああ。僕の、元の名前だ」

エリナは、真剣な眼差しで彼を見つめた。

「いつか、教えてくれる？」

その瞳には、不安と、それ以上の何かが宿っていた。カイトは、小さく笑った。

「ああ。いつか、な」

エリナは、少し安心したように微笑んだ。

「なら、いいわ。いつかって言ったら、絶対よ」

「約束だ」

二人は、小指を絡ませた。幼い約束の形。だが、二人にとっては十分すぎるほどの重みがあった。

「あらあら、青春ですわね」

リリが、楽しそうにクスクスと笑った。

「ですが、感傷に浸っている時間はありませんわよ。図書館の門限まで、あと四時間ですの」

「門限があるのか」

カイトが呆れると、リリは至極真面目な顔で答えた。

「夜間は『彼ら』が活性化しますの。安全のためにも、日没前には内部に避難することをお勧めいたしますわ」

「彼ら？」

リアムが警戒したが、リリはただ微笑むだけだった。

「詳しくは、図書館の中で。さあ、急ぎましょう」

少女は、踵を返し、歩き出した。白銀の髪が、風もないのに優雅になびく。

カイトたちは顔を見合わせ、覚悟を決めてその後に続いた。始原の図書館。その内部で待つのは、知識か、それとも──。

二つの月が、静かに彼らを見守っていた。白と赤。表と裏。光と闇。

世界の真実が、もうすぐ明らかになる。

【用語解説】

  • リリ：回廊の案内人を名乗る謎の少女。始原の図書館の「記録者」の一人。オッドアイ（右目金色、左目銀色）。
  • 記録者（レコーダー）：世界の情報を記録・管理する存在。始原の図書館の管理システムの一部。
  • バグ：リリがカイトを指して使った言葉。システムにとって予期せぬ存在。
  • 彼ら：夜間に活性化する何らかの存在。詳細は不明。

【次章予告】

始原の図書館の内部へ。カイトたちは「知識の回廊」で試練に直面する。そこで見たものは、カイトの記憶そのものだった──！
