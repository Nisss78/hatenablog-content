第740章：廃墟の街

港から内陸へと続く道は、かつて石畳だったものの残骸であった。今では雑草が隙間から生い茂り、風化した石が点在しているのみ。

四人は慎重に足を進めた。周囲には崩れかけた建物が並んでいたが、そのどれもが長い年月の痕跡を残していた。

「ここは……かつて何かの街だったんですかね」

シオンが廃墟を見渡しながら尋ねた。銀色の髪が風に揺れる。

「港町だったはずだ。船が来るってことは、交易があったということ」

リアムが冷静に分析した。彼の剣の柄に手をかけたまま、周囲を警戒している。

カイトは目を閉じ、魔力の流れを感じ取った。プログラマーとしての彼は、魔法的な現象をコードとして理解することができる。


// 周囲の魔力解析
class RuinsAnalyzer {
  constructor() {
    this.scanRange = 1000; // 1km圏内
  }
  
  detectResidualMagic() {
    const traces = manaFlow.scan(this.scanRange);
    return this.classify(traces);
  }
  
  classify(traces) {
    // 魔力の痕跡を分類
    let classified = {
      ancient: [],  // 古代の魔法
      recent: [],   // 最近の痕跡
      corrupted: [] // 歪んだ魔力
    };
    
    for (let trace of traces) {
      if (trace.age > 500) {
        classified.ancient.push(trace);
      } else if (trace.age < 10) {
        classified.recent.push(trace);
      }
      
      if (trace.distortion > 0.5) {
        classified.corrupted.push(trace);
      }
    }
    
    return classified;
  }
}

四人は廃墟の街を抜け、更に内陸へと進んだ。道はやがて森の中へと続いていた。

森の中は静寂に包まれていた。鳥のさえずりも、虫の声も聞こえない。

「不気味ね……」

エリナが小声で呟いた。彼女の水魔法は、周囲の気配を感じ取るのにも長けている。

「水脈が乱れているわ。地下深くから、何かが滲み出している」

カイトは頷いた。

シオンがふと立ち止まった。

「あれ、見て」

彼女が指差したのは、森の奥に見える石造りの建物だった。半分が崩れかけているが、かつては神殿か何かだったようだ。

「行ってみよう」

神殿の入り口には、古代の文字が刻まれていた。カイトはそれを読み解こうと試みた。


// 古代文字解析
function decodeAncientText(inscription) {
  const database = knowledgeBase.load("ancient_languages");
  const pattern = database.match(inscription);
  
  if (pattern.confidence > 0.7) {
    return pattern.translate();
  }
  
  return "解析不能";
}

カイトは以前、深淵の書庫で読んだ記録を思い出していた。虚無には段階があり、最も深い虚無は存在そのものを否定する力を持つという。

神殿の奥から、影が滲み出してきた。昨夜、海で遭遇した怪物と似ているが、より巨大で、より実体を持っていた。

// 光闇融合魔法：エクリプス・レイ
function eclipseRay(target) {
  const light = elements.summon("light");
  const dark = elements.summon("dark");
  
  const fused = manaFuse(light, dark);
  const ray = fused.form("beam");
  
  return ray.fire(target);
}

function purifyingRain(target) {

// 原初魔法・強化版：完全なる否定
function absoluteDenial(target) {
  const void = elements.summon("void");
  const primordial = void.connect("primordial");
  
  // 対象の構造を解析
  const structure = primordial.analyze(target);
  
  // 構造の核を特定
  const core = structure.findCore();
  
  // 核に対して存在否定を適用
  primordial.deny(core);
  
  // 周囲への影響を最小限に抑える
  primordial.contain();
  
  return primordial.close();
}
