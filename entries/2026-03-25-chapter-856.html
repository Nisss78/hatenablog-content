第856章：岩石の軍団との死闘

夜の山道に、無数の岩石の魔物が現れた。月光が雲間から漏れ、その異様な姿を照らし出している。

カイトは素早く状況を分析した。数は十体以上。岩肌には血管のような黒い脈動が走り、虚無のエネルギーが渦巻いている。

「みんな、散開だ！ 一箇所に固まると囲まれるぞ！」

カイトの指示で、四人はそれぞれの位置に散った。リアムが左手の岩陰に、エリナが右手の高台に、シオンはカイトの背後で支援魔法の準備に入る。

「行くぜ！ 炎纏剣・連撃！」

リアムが吼えながら、最も近いゴーレムに突っ込んだ。剣に纏われた炎が夜闇を切り裂き、岩石の腕を焼き焦がす。

ゴーレムが鈍い悲鳴を上げた。しかし、傷口から黒い霧が噴き出し、リアムの視界を奪おうとする。

「させるかよ！」

リアムは足元を蹴り、バックステップで霧を回避した。

エリナが高台から風の刃を連射する。

「風神刃・乱舞！」

無数の風の刃が、三体のゴーレムを同時に切り裂いた。風圧で黒い霧が吹き飛ばされ、岩石の核が露わになる。

「カイト！ 核が見えた！」

カイトは頷き、七属性の魔法を展開した。

// 七属性複合攻撃
class ElementalBarrage {
    constructor() {
        this.elements = ["炎", "氷", "雷", "風", "地", "光", "闇"];
        this.mode = "収束";
        this.target = "核";
    }
    
    fire() {
        let damage = 0;
        this.elements.forEach(element => {
            damage += this.calculateDamage(element);
        });
        return `七属性収束砲・ダメージ${damage}`;
    }
    
    calculateDamage(element) {
        return Math.floor(Math.random() * 100) + 50;
    }
}

const barrage = new ElementalBarrage();
console.log(barrage.fire());
「七属性収束砲！」

カイトの手から七色の光線が放たれた。光線は螺旋を描きながら一つのゴーレムの核へと収束していく。

ドォン！

爆発音とともに、ゴーレムが粉々に砕け散った。黒い霧が夜空へと霧散していく。

「一体撃破！」

シオンが歓声を上げた。しかし、彼女はすぐに支援に戻った。

「皆に光の加護を！ 癒しの光！」

温かな光が四人を包み込む。戦闘で受けた細かな傷が、徐々に癒えていく。

ゴーレムたちは怒ったように、一斉に動き出した。地響きを立てて、四人の方へと迫ってくる。

「数が多すぎる！ 何とか数を減らさないと！」

エリナが焦りを感じた声で叫んだ。

カイトは周囲を見渡した。山道の脇には切り立った崖がある。もし崖を崩せば、複数のゴーレムを同時に崖下へ落下させることができるかもしれない。

「リアム！ 左側の崖を狙え！ 土砂崩れを起こすんだ！」

リアムはカイトの意図を即座に理解した。

「なるほど、わかったぜ！」

リアムは剣に炎を纏わせ、地面を叩いた。衝撃波が岩盤を伝わり、崖の土質を崩し始める。

「エリナ、風で誘導して！」
「任せて！」

エリナが強風を巻き起こした。崖が崩れ落ちるのと同時に、風圧でゴーレムたちを崖の方へと押しやる。

四体のゴーレムが、バランスを崩して崖下へと転落していった。

「よしっ！ 四体一気に！」

しかし、まだ七体が残っている。そして、彼らは学習していた。

ゴーレムたちは散開し、崖から距離を取って包囲網を狭めてきた。

「くっ、賢いな……」

カイトは舌打ちした。単なる魔物ではない。虚無のエネルギーが、何らかの意志を持って動いている。

「指揮者がいるのかもしれない」

シオンが静かに言った。

「え？」
「これだけ連携できるなら、誰かが遠隔で指示してるはず」

カイトはハッとした。十三使徒の仕業か。どこかで彼らが見ている可能性がある。

「みんな、気をつけて！ 敵はこの魔物だけじゃないかもしれない！」

カイトの警告が響いた瞬間、山道のさらに奥から、新たな気配が現れた。

闇の中から、人影が現れた。

漆黒のローブを纏った人物。顔はフードで隠されているが、その身から滲み出る不気味なオーラは、十三使徒のそれだった。

「やはり、来ましたか」

低い声が響いた。

「海藤翔太……そして、七属性の乙女シオン」

カイトは剣を構えたまま、冷静に問いかけた。

「お前は誰だ。十三使徒か？」

人物はゆっくりとフードを外した。現れたのは、青白い顔をした青年だった。目は虚ろで、瞳の中に黒い渦が見える。

「私は第十三席……コードネームは『ナンバー・ゼロ』。虚無の化身と呼ばれている」

シオンが息を呑んだ。

「虚無の……化身？」
「ええ。私は虚無そのもの。感情も意志もない、ただの破壊の器」

ナンバー・ゼロと名乗った青年は、感情のない声で続けた。

「あなたたちをここで止めるよう命じられています。地の聖域には行かせません」

カイトは拳を握りしめた。

「行かせないだと？ テラドンを助けなきゃならないんだ」
「テラドン……地の賢者は、もう限界です。今ごろは、我が同胞が聖域を攻めている頃でしょう」

その言葉に、四人の表情が固まった。

「何だって！？」

リアムが叫んだ。

「もう攻め込まれてるのか！？」
「ええ。私の役目は、あなたたちの足止め。ここで時間を稼ぐことです」

ナンバー・ゼロが手をかざすと、残りのゴーレムたちが一斉に動き出した。

「さあ、始めましょうか。虚無の宴を」

カイトは覚悟を決めた。この敵を倒さない限り、地の聖域にはたどり着けない。

リアムが炎纏剣でゴーレムの群れに突っ込み、道を切り開く。エリナが風で敵の動きを封じ、シオンが光の矢でゴーレムの核を正確に撃ち抜いていく。

そして、カイトはナンバー・ゼロと対峙した。

「お前の相手は俺だ！」

カイトが七属性の剣を形成し、ナンバー・ゼロに切りかかる。

ナンバー・ゼロは、片手で黒い盾を展開して攻撃を受け止めた。

「悪くない攻撃です。ですが、虚無には届きません」
「そうか？ これならどうだ！」

カイトは剣に雷の属性を重ね、麻痺効果を付与した。盾を通して、電撃がナンバー・ゼロの腕に走る。

ナンバー・ゼロの表情がわずかに歪んだ。

「……痛覚が残っていたとは。興味深い」
「お前も生身なんだな。無敵じゃないってことだ」

カイトは連続攻撃を繰り出した。炎、氷、雷、風、地、光、闇——七属性を次々と切り替えながら、猛攻を仕掛ける。

ナンバー・ゼロは防戦一方になった。しかし、その表情は変わらない。まるで、痛みすら感じていないかのように。

「なぜ……こんなに攻撃を受けても倒れない？」

カイトが息を切らしながら問うと、ナンバー・ゼロは静かに答えた。

「私は虚無の化身。痛みはあっても、死はありません。虚無が存在する限り、私は何度でも蘇る」

その言葉に、カイトは戦慄した。不死身の敵。どうやって倒せばいいのか。

その時、シオンの声が響いた。

「カイト！ 核があるはず！ ゴーレムと同じように、彼の中にも核がある！」

シオンの鋭い観察眼が、ナンバー・ゼロの胸の奥に微かな光を見つけていた。

「胸の奥……！」

カイトは狙いを定めた。

「サンキュー、シオン！」

カイトは全魔力を剣に込めた。七色の光が剣先に集束していく。

「七属性収束剣・虚空貫通！」

カイトが放った渾身の一撃が、ナンバー・ゼロの胸を貫いた。

青年の胸から、黒い霧が噴き出した。

「……まさか、見抜かれるとは」

ナンバー・ゼロが初めて、感情のこもった声を出した。

「あなたたちは……危険すぎる。虚無にとって」
「虚無なんかに、この世界は渡さない」

カイトが言うと、ナンバー・ゼロは微かに微笑んだ。

「……ええ。もしかしたら、あなたたちが……虚無を止めるのかもしれない」

その言葉を最後に、ナンバー・ゼロの体は黒い霧となって霧散した。

ゴーレムたちも、指揮者を失い、動きを止めて崩れ落ちた。

静寂が戻った山道で、四人は荒い息を整えていた。

「やった……のか？」

リアムが剣を下ろした。

「ああ。倒した。でも……」

カイトはナンバー・ゼロの最後の言葉を反芻していた。

「彼は『虚無が存在する限り何度でも蘇る』って言ってた。つまり、倒しただけじゃ不十分なんだ」

エリナが深刻な表情で頷いた。

「虚無そのものを消さない限り、彼らは何度でも現れるってことね」

シオンがカイトの側に来た。

「でも、今は地の聖域へ急がないと。テラドンさんが危ない」

カイトは頷いた。

「そうだ。行こう。戦いはまだ始まったばかりだ」

四人は山道を再び登り始めた。

夜空には、雲の切れ間から星が瞬いていた。

その星明かりの下、地の聖域への道が、彼らを待っていた。

山のさらに奥、地の聖域では、今まさに激しい戦いが繰り広げられているはずだ。

カイトは拳を握りしめた。
