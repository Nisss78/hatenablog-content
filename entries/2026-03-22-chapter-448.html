第448章：闇への招待状

東の聖堂を後にしたカイトたちは、西への帰路を急いでいた。

アンブローズ・ヴェイルの研究所——境界に関する膨大な資料が眠る場所。そこに、闇の賢者への手がかりがあるはずだ。

「待ってくれ」

聖堂を降りきったところで、リアムが立ち止まった。

「どうした？」

カイトが振り返ると、リアムは空を指差していた。

「空を見ろ」

カイトは顔を上げた。夕焼けに染まった空の中に、黒い点がいくつも浮かんでいる。

「鳥……？」

「いや、違う」

エリナが目を細めた。

「あれ、魔法の灯りだよね。複数の人が、空を飛んでいる」

黒い点が近づいてくる。確かに、人の形をしている。ローブをまとった数人の魔法使いが、空を滑るように飛んできた。

先頭の人物が、カイトたちの前に降り立った。

「海藤翔太殿とお見受けする」

男性だった。年齢は三十代半ば。黒いローブに、銀色の刺繍が施されている。顔立ちは鋭く、眼光は澄んでいる。

「はい、そうですけど……あなたは？」

「私は『深淵の書庫』の管理官、エドガー・ナイトシェード。闇の賢者、アンブロス様の使者だ」

カイトは息を飲んだ。まさか、向こうから接触してくるとは。

「闇の賢者……アンブロスさん？」

「左様。アンブロス様は、そなたの活躍を聞き及んでいる。光の賢者ルミナス様の試練を乗り越えたとの報告も、すでに届いている」

エドガーは懐から封筒を取り出した。黒い封筒に、銀色の封蝋。

「これは、アンブロス様からの招待状だ。そなたを『深淵の書庫』へ招く」

カイトは封筒を受け取った。手に取った瞬間、冷たい感覚が指先を伝った。

「深淵の書庫って？」

「地下千メートル、世界最深の図書館だ。アンブロス様はそこで、永い時を過ごされている」

「地下千メートル……」

エリナが呟いた。

「そんな深いところに、どうやって行くの？」

「専用の転移陣がある。招待状を持つ者だけが、その陣を使用できる」

エドガーは地図を取り出した。

「転移陣の場所はここだ。南の廃都、『グリムホルム』。かつては栄えた都市だったが、今は闇に覆われている」

「廃都……」

リアムが地図を覗き込んだ。

「グリムホルムか。聞いたことはある。百年以上前に、何らかの災厄で壊滅した都市だと」

「その通り。だが、廃都の地下には、今も転移陣が機能している。アンブロス様が維持しているのだ」

エドガーはカイトを見据えた。

「アンブロス様は、そなたとの会見を楽しみにされている。……覚悟を決めて、来てくれ」

＊＊＊

エドガーたちが去った後、カイトたちは近くの森で野営することにした。

焚き火を囲みながら、招待状について話し合う。

「向こうから招待してくるなんて、珍しいね」

エリナが言った。

「闇の賢者は、これまで会ったどの賢者とも雰囲気が違う気がする」

カイトは招待状を開いた。中には、流麗な文字でこう書かれていた。

『海藤翔太殿へ

そなたの旅路、興味深く見守ってきた。
光と影。その二つを背負いながら歩む姿、実に興味深い。
私は闇の深淵に座している。そなたが望むなら、会おう。
ただし、闇は光を食べる。覚悟を問う。

——アンブロス』

「『闇は光を食べる』……」

カイトはその言葉を反芻した。

「どういう意味だろう」

「文字通りの意味じゃないか」

リアムが薪をくべながら言った。

「光が強ければ強いほど、その分の影も濃くなる。ルミナスさんの試練でも言ってただろ？ 闇の賢者は、その『影』の側面を極めた存在なんだろう」

「光と闇は、対立するようで、実はセットなんだよね」

エリナが焚き火を見つめながら言った。

「カイトは光の試練で、自分の影を受け入れた。じゃあ、闇の試練では何を問われるんだろう」

カイトは考え込んだ。光の試練は「自分の影を受け入れること」だった。では、闇の試練は——

「もしかしたら、光を信じ続けることかもしれない」

カイトは呟いた。

「闇の中にいると、光があったかどうかさえ忘れちゃう。希望があったことを疑ってしまう。だから、闇の試練は……自分の中の光を見失わないことなのかも」

「なるほど」

リアムが頷いた。

「光の試練が影を受け入れることなら、闇の試練は光を信じること。対照的で、でも通じ合っている」

「難易度が高そうだな」

エリナがため息をついた。

「でも、カイトなら大丈夫。絶対」

「え？」

「だって、カイトは光も闇も持ってる。光の試練でも、自分の影を認めたでしょ？ 闇の中で光を見つけることも、きっとできるよ」

エリナの言葉に、カイトは少し救われた。この旅で出会った仲間たちの存在は、何よりも大きい。

「ありがとう、エリナ。リアムも」

「礼を言われるほどのことじゃないさ」

リアムは照れくさそうに薪をくべった。

「俺たちは、お前の背中を預かる相棒だ。最後まで、ついていく」

＊＊＊

翌日、カイトたちは南への移動を開始した。

廃都グリムホルムまでは、馬車で三日の距離。街道を外れ、荒野を横切る必要がある。

「道中、気をつけてください」

エドガーは去り際にそう忠告していた。

「グリムホルムの周辺は、闇の魔力が濃い。普通の人間なら、精神を蝕まれる」

「精神を？」

「幻覚を見たり、恐怖に襲われたり。だが、そなたなら大丈夫だろう。光の力を持っている」

その言葉を思い出しながら、カイトたちは進んでいた。

＊＊＊

一日目は、特に問題なく過ぎた。

小さな町で宿を取り、翌朝に出発。荒野に入ると、景色が一変した。

草木が生えない、灰色の大地。空は鉛色で、太陽の光が届かない。雲が分厚く覆い、世界を押し潰そうとしている。

「暗い……」

エリナが身震いした。

「昼間だっていうのに、夕暮れみたい」

「グリムホルムの影響だろう」

リアムが周囲を警戒しながら言った。

「闇の賢者が住んでいる場所だ。魔力が漏れ出しているんだろう」

カイトは解析魔法を試みた。


function analyzeDarknessField(area) {
  const metrics = {
    darknessDensity: measureMagicalDensity("dark"),

    mentalInfluence: calculateMentalPressure(),

  if (metrics.darknessDensity > THRESHOLD) {

  }
  
  return metrics;

結果は予想以上だった。この一帯は、闇の魔力が物理的な質量を持つほど濃密になっている。普通の人間なら、確かに精神に異常をきたすレベルだ。

「僕が光の魔法で、周囲を照らそう」

カイトは手をかざし、光の魔法を発動した。


function createLightAura(radius, intensity) {
  const aura = {
    type: "protective_light",
    source: "sage_power",
    range: radius,
    brightness: intensity,
    duration: "continuous"
  };
  
  return castSpell(aura);
}

淡い光が、カイトを中心に広がった。闇が退き、視界が明るくなる。

「これで少しは楽になるはず」

エリナとリアムが安堵の表情を浮かべた。

「助かるよ、カイト。さすが、光の賢者の力を手に入れただけある」

「まだ使いこなせてないけどね」

カイトは苦笑した。

＊＊＊

二日目の夜、奇妙な出来事が起きた。

野営中、カイトが見張り番をしていた時のことだ。

焚き火の向こう側に、人影が現れた。

黒いローブを着た、背の高い人物。顔は兜で隠れていて見えない。

「誰だ？」

カイトが警戒して尋ねると、人物はゆっくりと近づいてきた。

「驚かせてすまない。敵ではない」

声は低く、響きがあった。女性の声。

「私は『深淵の書庫』の司書、セレスティア。アンブロス様の命で、そなたを案内しに来た」

「案内？」

「グリムホルムへの道は、危険が多い。特に、夜間は闇の魔物が跋扈する。私がいれば、安全に通れる」

カイトは少し迷った。敵ではないと言うが、本当にそうか確信はない。だが、もし本物なら、心強い味方になる。

「……わかった。お願いする」

セレスティアは頷いた。

「賢明な判断だ。……明日の朝、出発しよう。今夜は休んでくれ」

＊＊＊

翌朝、セレスティアを加えた一行は、荒野の奥へと進んだ。

「グリムホルムは、かつて魔法文明の中心だった」

セレスティアが説明してくれた。

「今は廃都だが、当時の名残は至る所に残っている。特に、地下水路の魔法陣は、現代では再現できない技術で作られている」

「魔法文明の中心……」

カイトは周囲を見渡した。荒廃した大地だが、よく見ると、崩れた石柱や、半ば埋もれた彫像が点在している。かつての栄華を物語る遺構たちだ。

「なぜ、グリムホルムは壊滅したんですか？」

エリナが尋ねた。

「正確な原因は、誰も知らない。ただ、アンブロス様は何かを知っている様子だ。……そなたが会った時に、聞いてみるといい」

セレスティアの言葉には、含みがあった。

＊＊＊

正午頃、廃都グリムホルムが見えてきた。

巨大な壁に囲まれた都市。かつては壮麗だったであろう建造物が、今は崩れ落ちている。空を突き刺すような塔も、半分が折れていた。

「これが……グリムホルム」

リアムが呟いた。

「想像以上に、規模がデカいな」

「十万の人口を誇った都市だ」

セレスティアが言った。

「今は、誰も住んでいない。……アンブロス様を除いて」

廃都の入り口——かつては大門があった場所——を通り抜けると、冷気が押し寄せてきた。

「ここから先は、闇の領域」

セレスティアが立ち止まった。

「転移陣は、中央広場の地下にある。私が先導しよう」

カイトたちは、セレスティアに続いて廃都の中へと足を踏み入れた。

＊＊＊

廃都の内部は、異様な雰囲気に満ちていた。

崩れた家屋、ひび割れた石畳、苔と蔦に覆われた彫像。かつての生活の痕跡があちこちに残っているが、人はいない。静寂だけが、重く立ち込めている。

「誰もいないっていうのは、本当なんだな」

リアムが声を潜めて言った。

「魔物もいないのか？」

「グリムホルムの内部は、アンブロス様の結界で守られている。魔物は入れない」

セレスティアが答えた。

「ただし、結界の外には多くの魔物がいる。だから、夜間の移動は危険なのだ」

中央広場に着いた。かつては噴水があった場所だろう——今は、乾いた泉の跡だけが残っている。

セレスティアが泉の中に入り、床の石板を押した。

カチリ。

音が響き、地下への階段が現れた。

「ここから先は、招待状を持つ者だけが進める。お二人は、ここでお待ちください」

セレスティアはエリナとリアムに向かって言った。

「えっ、カイト一人で行くの？」

エリナが心配そうに声を上げた。

「大丈夫。僕一人でやらなきゃいけないことだから」

カイトは二人に向かって微笑んだ。

「ここで待ってて。必ず、戻るから」

＊＊＊

地下への階段を降りていく。

一段、また一段。光は徐々に薄れ、闇が濃くなっていく。

（これが、闇の賢者の領域……）

カイトは招待状を握りしめた。手の中で、銀色の封蝋が微かに光っている。

階段は長く続いた。千段以上はあるだろうか。足音だけが響く静寂。

やがて、階段が終わり、扉が現れた。

黒い扉。表面には、複雑な模様が彫刻されている。星座のようにも見えるし、魔法陣のようにも見える。

カイトは招待状を扉に押し当てた。

すると、扉が音もなく開いた。

中から、冷たい風が吹き出してくる。

「ようこそ、海藤翔太」

闇の中から、声が響いた。

「待っていたよ」

カイトは一歩踏み出した。闇の賢者、アンブロスとの対面——ついに、その時が来た。

（続く）
