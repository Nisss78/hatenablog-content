第451章：風の道標

アンブローズ邸を出てから、三日が経過していた。

カイトたちは街道を北東へと進んでいる。アルヴァレス支部までは、徒歩で四日の距離。馬車を使えば一日で着くが、あえて徒歩を選んだ。理由は二つ。一つは、道中で発生する魔物の討伐。もう一つは、風の賢者の気配を探るためだ。

「風って、どうやって探すんだろう」

エリナが呟いた。街道沿いの丘に座り、風に吹かれている。彼女の銀髪が、柔らかく揺れていた。

「風は見えないし、掴めない。どこに吹いているのかも、わからないし」

「風が吹く場所、か……」

カイトも隣に座った。目を閉じて、風の感触を確かめる。肌を撫でる微かな空気の流れ。草木を揺らすそよ風。遠くで鳴る木々のざわめき。

（プログラミングで言えば……）

風は、変数のようなものかもしれない。名前をつけても、その中身は常に変化する。時には優しく、時には激しく。予測不可能で、制御が難しい。

「風の賢者も、きっとそういう存在なんだろうね」

カイトは目を開けた。

「特定の場所に留まらず、常に移動し続けている。会うには、タイミングと縁が必要かもしれない」

「縁、ねぇ」

エリナは苦笑した。

「そういうの、苦手なんだけど」

「僕もだよ」

カイトも笑った。

「でも、ヒナタさんなら何か知っているはずだ。風の剣士として、風の賢者と何らかの接点があるかもしれない」

「だといいけど」

リアムが、焚き火の準備をしていた。夕暮れが近づいている。今夜は野宿だ。

「セシリア、明日の予定は？」

「明日の夕方には、アルヴァレス支部の近くにある宿場町に着きます」

セシリアが地図を広げた。

「そこで一泊して、翌朝に支部へ向かうのがよろしいかと」

「了解」

リアムは薪を積み上げ、火魔法で点火した。

「今夜は早めに休もう。明日も歩き詰めだ」

＊＊＊

夜、カイトは眠れなかった。

体の中で、光と闇の力が交錯している。胸の奥が熱くなったり、冷たくなったり、交互に感覚が変わる。試練の後遺症だろうか。それとも、力がまだ完全に馴染んでいないのか。

（深呼吸……）

カイトは、静かに呼吸を整えた。瞑想のように、意識を内側に向ける。光と闇が、彼の中で渦を巻いているのが見えた。


function balance(dark, light) {
  while (true) {
    if (dark > light) {
      light += 0.1;
    } else if (light > dark) {
      dark += 0.1;
    }
    if (Math.abs(dark - light) < 0.01) {
      break;
    }
  }
  return { dark, light };
}

カイトは、心の中でコードを書いた。光と闇のバランスを取るためのアルゴリズム。少しずつ、少しずつ、二つの力を近づけていく。

（こういうことか……）

やがて、胸の奥が落ち着いてきた。光と闇が、互いに干渉することなく、静かに共存し始める。まるで、太極図のように。

「……カイト？」

声がして、カイトは目を開けた。エリナが、心配そうに覗き込んでいた。

「大丈夫？ 眠れないの？」

「あ、うん……ちょっと、力の調整を」

カイトは苦笑した。

「光と闇、まだ完全に制御できてないみたいで」

「そっか」

エリナはカイトの隣に座った。焚き火の残り火が、彼女の顔を照らしている。

「無理しないでね。カイトが倒れたら、私たち困るから」

「わかってる」

「本当に？」

エリナはカイトの目をじっと見た。

「カイトって、たまに無茶するから。心配なんだよ」

「……ごめん」

カイトは素直に謝った。確かに、試練に挑む時も、仲間に相談せず突っ込んでいった。結果的に成功したが、失敗していたらどうなっていたか。

「次からは、もっと相談するよ」

「絶対だよ」

エリナはカイトの手を握った。温かい。深淵の冷たさを思い出し、その対比にカイトは心が安らいだ。

「エリナたちがいるから、僕は頑張れるんだ」

「……ずっと、そばにいるから」

エリナの声が、風に乗って消えていった。

＊＊＊

翌朝、カイトたちは出発した。

街道は緩やかに登り続けている。北へ進むにつれて、空気は冷たくなり、草木の種類も変わっていく。針葉樹林が増え、遠くに山々の稜線が見え始めた。

「もうすぐアルヴァレス領だ」

リアムが指差した。街道の先に、小さな村が見える。宿場町だ。

「あそこで休憩して、明日支部へ向かおう」

「賛成」

エリナが頷いた。

「足が棒になりそう」

「大げさだな」

リアムは笑った。

「これでも、冒険者としては楽な方だぞ」

「わかってるけど、女心として文句くらい言わせてよ」

セシリアが、冷静に口を挟んだ。

「文句を言うのは構いませんが、足を止めないでください。日が暮れる前に到着したいので」

「……はい」

エリナは肩をすくめた。

村に到着したのは、夕暮れ前だった。宿屋に荷物を預け、四人は食堂で食事をとった。

「明日は、アルヴァレス支部へ向かいます」

セシリアが確認した。

「ヒナタさんに会って、風の賢者の情報を聞く。それが、まず第一歩です」

「第二歩は？」

エリナが尋ねた。

「ヒナタさんからの情報次第ですね。風の賢者が特定の場所にいるなら、そこへ向かう。移動しているなら、追跡する必要があります」

「追跡か」

カイトは考え込んだ。

「風を追跡するって、どうやるんだろう」

「風の跡を辿る、という魔法が存在します」

セシリアが答えた。

「風が吹いた場所には、微量な魔力の残留が残ります。それを読み取ることで、風の進行方向を特定できるそうです」

「へぇ……」

カイトは感心した。

「魔法って、本当に奥が深いね」

「ええ。世界の理（ことわり）を理解する鍵でもあります」

セシリアは眼鏡の位置を直した。

「カイトさんの『魔法をコードで書く』能力も、魔法の本質に近づく一つの道かもしれません」

「そうだといいけど」

カイトは窓の外を見た。風が、木々を揺らしている。風の賢者も、どこかでこの風を感じているのだろうか。

（会えるかな）

カイトは、心の中で問いかけた。

（風の賢者に）

＊＊＊

夜、カイトは一人で宿屋の外に出た。

村は静かだ。家々の灯りは消え、星明かりだけが道を照らしている。カイトは村の外れにある小高い丘に登った。

そこからは、アルヴァレス支部の方角が見えた。遠くに、城壁と塔のシルエットが浮かんでいる。

「明日か……」

カイトは呟いた。明日、ヒナタさんに会う。風の賢者の情報を聞く。そして、旅は続く。

（残るは、風だけ）

火、氷、雷、地、光、闇。六大元素の力を得た。あとは風だけ。それを集めれば、境界修復の準備が整う。

（でも——）

カイトは、胸の違和感を感じていた。何かが、まだ足りない。力を集めるだけでは、境界は修復できない気がする。

（何だろう、この感覚）

カイトは、深淵の書庫で見た記録を思い出した。アレクシス・ヴェイル——最初の管理者——の言葉。

「境界は、分断するものにあらず。繋げるものなり」

境界を修復するには、力を集めるだけでは足りない。何かを繋げる必要がある。何を？

（わからない……）

カイトは首を振った。今は、目の前の課題に集中しよう。風の賢者を探すこと。それが、まずやるべきことだ。

風が吹いた。カイトの髪を揺らし、服をはためかせる。

その時——

カイトの耳に、微かな声が届いた。

「……見つけた」

カイトは振り返った。誰もいない。丘の上は、カイト一人だけ。

「今の、何？」

声は聞こえない。ただ、風の音だけが残った。

「……来なさい。……風の……場所……」

途切れ途切れの声。でも、確かに届いている。風に乗って。

「風の賢者？」

カイトは問いかけた。応答はない。だが、風が方向を変えた。北東へ。アルヴァレス支部の方角とは、微妙に異なる。

（あっちに何かあるのか？）

カイトは、風が吹く方角を見つめた。

「明日、ヒナタさんに会ってから……確認してみよう」

カイトは宿屋に戻った。胸の奥に、予感が残っていた。風の賢者は、近くにいる。会える日は、そう遠くない——。

＊＊＊

翌朝、カイトたちは早々に出発した。

宿場町からアルヴァレス支部までは、半日の距離。昼前には到着できる。

街道を歩いていると、遠くに支部の城壁が見えてきた。高い石垣に囲まれ、見張り塔が複数設置されている。冒険者ギルドの支部の中でも、防衛力が高いことで知られていた。

「ヒナタさん、待っててくれるかな」

エリナが呟いた。

「連絡なしで行くわけだし」

「大丈夫です」

セシリアが答えた。

「事前に、手紙で到着の目安を伝えてあります。ヒナタさんも、待っているはずです」

「え、そうなの？」

エリナが驚いた顔をした。

「いつの間に」

「宿場町に着いた時、手紙を預けておきました」

「さすが、セシリア」

リアムが感心したように頷いた。

「手回しがいいな」

「当然です」

セシリアは淡々と答えた。

「効率的な行動が、成功への近道ですから」

カイトは、そんな仲間たちの会話を聞きながら、歩を進めた。支部の正門が、徐々に近づいてくる。

（ヒナタさん——）

風の剣士。速さを極め、風の魔法を剣技に融合させた達人。彼女が、風の賢者への道標となるはずだ。

（頼むよ）

カイトは、胸の光と闇を感じながら、正門へと歩を進めた。

風が、背中を押していた。

（続く）
