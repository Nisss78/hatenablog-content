第99章：偽りの傷跡

準備一日目の朝、カイトは早朝から目覚めていた。

源泉のそばで、ハーモニーシステムを起動し、作戦の詳細を詰めていた。


// おとり作戦・詳細プラン
const decoyOperation = HarmonySystem.planOperation({
    name: '偽りの傷跡',
    objective: 'ザインを誘引し、黒曜の位置を特定',
    decoy: 'ゼノン',
    fake_status: '重傷・源泉枯渇寸前',
    bait_channels: ['商人ネットワーク', '傭兵ギルド', '闇の情報屋']
});

console.log(decoyOperation);
// {
//   phase1: '偽情報の拡散（今日）',
//   phase2: '情報の浸透待機（明日）',
//   phase3: '迎撃態勢構築（明後日）',
//   estimated_response_time: '48-72時間',
//   risk: 'アークライト本隊も反応する可能性'
// }

カイトは、リスクを考慮した。偽情報は、ザインだけでなく、アークライト本隊にも届くかもしれない。そうなれば、複数の敵を同時に相手にすることになる。

「でも、それを逆手に取れば、一網打尽にできるかもしれない」

カイトは、独り言を言った。

朝食の後、カイトたちは作戦会議を行った。

ゼノンの住居は、源泉のそばにあった。石と木で作られた簡素な建物。十年間の戦いで、少しずつ整備したものだ。

机の上に、西方の地図が広げられた。ゼノンが、重要な地点を指し示した。

「ここが、この源泉だ。そして、ここが、アークライトの前線基地。直線距離で、約五十キロ」

カイトは、地図を見つめた。

「アークライトの主力は、この基地に駐留しているんですね」

ゼノンは頷いた。

「ああ。兵力は、約五千。魔導兵器も、多数配備されている。昨日の戦いで、尖兵部隊を撃退したが、本隊は無傷だ」

リアムが、口を開いた。

「五千対六人……分が悪いな」

アルフレッドが、冷静に言った。

「正面から戦うつもりはないだろう。敵を分断し、各個撃破する。それが、我々の戦い方だ」

カイトは頷いた。

「その通り。今回の作戦は、ザインを誘き寄せて、黒曜の位置を特定すること。アークライト本隊との全面衝突は、避ける」

エリナが、質問した。

「偽情報は、どうやって流すの？」

カイトは、ハーモニーシステムの分析結果を共有した。

「三つのルートを使う。商人ネットワーク、傭兵ギルド、そして闇の情報屋」

ゼノンが、補足した。

「西方には、中立の商人街がある。『鉄鋼の市』だ。あそこなら、情報を売買できる」

カイトは、地図上の地点を確認した。

    region: 'west'

カイトは、決めた。

「俺とリアムで、鉄鋼の市へ行く。偽情報を流してくる」

エリナが、心配そうに言った。

「二人だけで大丈夫？」

カイトは頷いた。

「大丈夫。大人数で行くと、目立つ。二人なら、傭兵か商人に見える」

ゼノンが、念を押した。

「気をつけてくれ。鉄鋼の市は中立だが、アークライトの密偵も潜んでいる」

カイトは、真剣な表情で頷いた。

「わかっています。慎重にやります」

正午、カイトとリアムは、鉄鋼の市に向けて出発した。

山道を下り、街道に出る。西方の大地は、荒廃していた。十年間の戦争で、畑は焼かれ、村は廃墟と化していた。

「ひどいもんだな」

リアムが、呟いた。

「戦争ってのは、こういうもんだ。何も残さない」

カイトは、悲しげに言った。

「だから、止めないといけないんです」

リアムは、カイトの肩を叩いた。

「お前は、いいやつだな。カイト」

カイトは、少し照れた。

「……ありがとうございます」

夕刻、カイトとリアムは、鉄鋼の市に到着した。

巨大な壁に囲まれた街。門には、自治警備隊が立っていた。入場料を払い、中に入る。

街の中は、活気に満ちていた。露天商が武器や薬を売り、傭兵たちが酒場で騒いでいた。中立地帯だからこそ、様々な人種が集まっていた。

「まずは、情報屋を探そう」

カイトが言った。

リアムが、ニヤリと笑った。

「俺に任せろ。こういう場所には、詳しいんだ」

リアムは、裏路地に入った。怪しげな看板の酒場があった。

「ここだ。『黒猫の隠れ家』。情報屋の溜まり場だ」

カイトは、少し緊張した。でも、覚悟を決めて、中に入った。

酒場の中は、薄暗かった。煙草の煙が漂い、怪しげな客たちが、鋭い視線を向けてきた。

カイトとリアムは、カウンターに座った。

バーテンダーは、年老いた男だった。片目が義眼だった。

「初めて見る顔だな。何用だ？」

リアムが、金貨を三枚、カウンターに置いた。

「情報を売りたい。そして、情報を買いたい」

バーテンダーは、金貨を懐に入れた。

「話を聞こう」

カイトが、偽情報を話した。

「西方の源泉で、鋼の巨竜ゼノンが、瀕死の重傷を負った。源泉も、もうすぐ枯渇する。この情報、いくらで買う？」

バーテンダーは、少し驚いたようだった。

「ゼノンが……瀕死か。確かな情報か？」

カイトは頷いた。

「俺は、商人だ。西方の戦場で、武器を売っていた。ゼノンが倒れるのを見た。源泉の光も、弱々しかった」

カイトの嘘は、完璧だった。ハーモニーシステムで、嘘発見器の反応を抑制していた。


// 嘘発見回避・生理反応制御
HarmonySystem.controlPhysiology({
    target: 'self',
    heartbeat: 'normal',
    sweating: 'minimal',
    eye_contact: 'natural'
});

バーテンダーは、少し考えた。

「……五十金貨で買おう。ただし、情報が確実ならな」

カイトは、内心で驚いた。五十金貨は、大金だ。この情報の価値が高いことを示していた。

「 seventy どうだ？」

バーテンダーは、少し笑った。

「六十だ。それ以上は、出せん」

カイトは頷いた。

「六十でいい。ただし、この情報を、『黄昏の牙』にも流してくれ」

バーテンダーは、少し驚いた。

「闇のギルドに、直接情報を流したいのか？」

カイトは頷いた。

「ああ。彼らなら、高い金を出してくれるだろう」

バーテンダーは、少し笑った。

「……わかった。六十金貨に、情報の拡散を含める。ただし、『黄昏の牙』への接触は、別料金だ」

カイトは、了承した。

「いいだろう」

取引は成立した。カイトは、六十金貨を受け取り、バーテンダーに追加で十金貨を払った。

酒場を出た後、カイトとリアムは、別の情報屋を訪れた。

商人ネットワーク、傭兵ギルド。三つのルートで、偽情報を拡散した。

「これで、ザインの耳にも届くはずだ」

カイトが言った。

リアムが、少し心配そうに言った。

「でも、本当に来るかな？ ザインは、慎重な男だろ？」

カイトは頷いた。

「ええ。でも、管理者を狩るチャンスなら、見逃さないはずだ。十年間、ゼノンを狙い続けてきたんだから」

リアムは、納得したようだった。

「だといいがな」

夜、カイトとリアムは、鉄鋼の市の宿に泊まった。

狭い部屋に、粗末なベッドが二つ。でも、清潔だった。

カイトは、窓から空を見上げた。星が輝いていた。その彼方に、黒い船が浮かんでいるかもしれない。

「明日は、どうする？」

リアムが、ベッドに寝転がりながら聞いた。

カイトは、ハーモニーシステムで翌日の計画を確認した。

カイトは、リアムに説明した。

「明日は、源泉に戻って、罠を設置する。情報が拡散したら、敵が動く。三日目には、決戦だ」

リアムは、少し緊張した表情で頷いた。

「わかった。覚悟を決めるか」

その頃、空の彼方で、巨大な影が動いていた。

全長八百メートルの飛行船、黒曜。闇のギルド『黄昏の牙』の本拠地。

船内の司令室で、一人の男が報告を受けていた。

銀髪の男。四十代前半。鋭い目つき。黒いローブを身に纏っている。

ザイン。闇のギルドの長。

「ゼノンが、瀕死だと？」

部下が、報告書を差し出した。

「はい。複数の情報筋から、同じ情報が入りました。西方の源泉で、アークライトとの戦いで重傷を負ったとのこと」

ザインは、報告書を読んだ。情報は、詳細だった。ゼノンの傷の位置、源泉の出力低下、守りの薄さ。

「……怪しいな」

ザインは、呟いた。

「あまりに、都合がいい」

部下が、意見を求めた。

「おとりの可能性がありますか？」

ザインは、少し考えた。

「かもしれない。でも、確認する価値はある。ゼノンを捕獲できれば、管理者の力を手に入けられる」

ザインは、立ち上がった。

「偵察部隊を送れ。三人でいい。情報の真偽を確かめろ」

部下は、畏まった。

「承知いたしました」

ザインは、窓から空を見た。

「十年間、待ち続けた。ゼノン……お前の力は、私のものだ」

ザインの目は、野心に燃えていた。

同じ頃、アークライトの前線基地でも、情報が入っていた。

報告を受けたのは、一人の女将軍だった。長い黒髪、冷たい瞳。名は、イヴァ。

「ゼノンが、弱っていると？」

副官が、頷いた。

「はい。情報屋からの報告です。信頼性は、高いと思われます」

イヴァは、少し笑った。

「ならば、好機だ。攻撃部隊を編成せよ。明日の夜、源泉を襲撃する」

副官は、畏まった。

「兵力は？」

イヴァは、冷徹に言った。

「三百でいい。少数精鋭で、確実に仕留める」

副官は、承知した。

「承知いたしました」

イヴァは、地図を見つめた。西方の源泉が、赤い印で示されていた。

「ゼノン……お前の首は、私が頂く」

深夜、カイトは、宿のベッドで眠れなかった。

「やるしかない」

カイトは、決意を新たにした。

窓の外で、夜風が吹いていた。明日は、準備二日目。罠を設置し、迎撃態勢を整える。

三日後には、決戦が始まる。

カイトは、カグヤの水晶を握った。

「カグヤ……俺たちを守ってくれ」

水晶は、微かに光った。カグヤの気が、応えているようだった。

カイトは、目を閉じた。明日に備えて、休息を取る。
