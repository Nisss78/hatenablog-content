第687章：光の塔の第二層

螺旋階段を登り続けること十分。ようやく踊り場のような場所に到着した。

「ここが第二層の入り口ね」

エリナが息を整えながら言う。階段はそれほどきつくはなかったが、塔内部の魔力が体力を徐々に削っていくようだった。

目の前には、また別の扉があった。第一層の扉と同じく、白い結晶で作られている。ただし、扉の中央には複雑な幾何学模様が彫り込まれていた。

「今度は何か違うようだな」

リアムが模様を観察する。

「パズルか、暗号か……」

シオンが近づき、指先で模様をなぞった。

「光の反射」

彼女が呟く。

「これ、光を当てる位置で模様が変わる」

「どういうこと？」

セシリアが首を傾げる。

「見て」

シオンが小さな光魔法を掌に灯す。それを模様に向けると、表面の凹凸に光が反射し、全く別の図形が浮かび上がった。

「おお……」

感嘆の声が漏れる。

「光学迷彩みたいなものか」

私が分析する。

「特定の角度からの光でないと、真の模様が見えない仕組みだ」

「じゃあ、正しい角度で光を当てればいいのね」

エリナが杖を構える。

「でも、何度試せばいいの？　角度は無限にあるわ」

「それが試練なんでしょう」

セシリアが静かに言った。

「手当たり次第に試すのは効率が悪い。何か法則があるはずよ」

私は扉の模様をじっと見つめた。幾何学模様は一見ランダムに見える。でも、プログラマーとしての経験が、パターンの存在を告げていた。

「ちょっと待って」

私は集中して、模様をコードとして解析し始めた。

class PatternAnalyzer:
    def __init__(self, pattern_image):
        self.pattern = pattern_image
        self.vertices = self.extract_vertices()
    
    def find_reflection_path(self):
        # 光が反射する経路を計算
        angles = []
        for v1 in self.vertices:
            for v2 in self.vertices:
                if v1 != v2:
                    angle = self.calculate_angle(v1, v2)
                    angles.append(angle)
        return self.detect_resonance(angles)
    
    def detect_resonance(self, angles):
        # 特定の角度で共鳴現象が起きる
        return [a for a in angles if a % 15 == 0]  # 15度単位

「十五度単位だ」

私が呟く。

「この模様、十五度ごとに区切られた角度で光を当てると、意味のある図形が現れる」

「十五度……」

リアムが顎に手を当てる。

「三六〇度を二十四分割。時刻の数と一致するな」

「時刻？」

エリナが目を丸くする。

「そうか、光の塔だから、太陽の動きに関連があるのかもしれない」

私は試しに、真上から光を当ててみた。角度は九十度。

模様が変化する。現れたのは、太陽のような図形。

「次は七十五度」

十五度ずつ角度を変えていく。六十度、四十五度、三十度……。

それぞれの角度で、異なる図形が現れた。月、星、雲、雷、虹……。

「これ、全部で二十四種類あるわね」

セシリアが数える。

「二十四の図形……二十四時間……」

「正解は一つじゃない」

シオンが言った。

「全部正解なの」

「どういうこと？」

「見て、模様が光ってる」

彼女の指差す先を見ると、二十四の図形すべてが、扉の表面に薄く浮かび上がっていた。

「全部の角度で光を当てないと、扉が開かないってこと？」

エリナが眉をひそめる。

「それだと時間がかかりすぎるわ」

「いや」

私は違う可能性に気づいた。

「同時に当てるんだ」

「同時に？」

「二十四の角度、二十四の光。一斉に当てれば、扉は開く」

全員が顔を見合わせた。

「できるか？」

リアムが問う。

「やってみるしかない」

私はみんなを見渡した。

「協力してほしい。それぞれがいくつかの角度を担当して、同時に光魔法を放つんだ」

「五人で二十四か……」

セシリアが計算する。

「一人あたり約五つ。いけるわね」

「位置取りが重要だ」

エリナが言う。

「角度を正確に出さないと意味がない」

「私が座標を出す」

私は素早く計算した。

def assign_positions(angles, members):
    assignments = {}
    angles_per_person = len(angles) // len(members)
    
    for i, member in enumerate(members):
        start = i * angles_per_person
        end = start + angles_per_person
        assignments[member] = angles[start:end]
    
    # 残りを最後のメンバーに
    if len(angles) % len(members) != 0:
        assignments[members[-1]] += angles[-(len(angles) % len(members)):]
    
    return assignments

「カイト、九十度から七十五度まで。エリナ、六十度から四十五度。セシリア、三十度から十五度。リアム、ゼロ度からマイナス十五度。シオン、残り全部」

「マイナスってことは、下から？」

リアムが確認する。

「そう。床近くから光を当ててくれ」

「了解」

全員が所定の位置につく。私は深呼吸をした。

「私の合図で一斉に。三、二、一……」

「今！」

五つの光源が、同時に扉を照らした。

私の光は真上から。エリナは斜め上から。セシリアは横から。リアムは斜め下から。シオンは床から。

二十四の角度すべてが、一瞬でカバーされた。

扉が共鳴し始める。模様が脈動し、光の波が表面を走る。

「開くわよ！」

セシリアが叫ぶ。

重厚な音と共に、扉が内側へと開いていった。

中から、柔らかな光が溢れ出す。第一層とは違う、温かみのある光だった。

「成功ね」

エリナが安堵の息を吐く。

「でも、これで終わりじゃないわよね」

「ええ」

私は頷いた。

「第二層の試練は、これからだ」

扉の奥へと足を踏み入れる。

そこは、信じられない空間だった。

空が見える。いや、本物の空ではない。天井に映し出された、巨大な空の映像だ。白い雲がゆっくりと流れ、太陽が暖かく照らしている。

地面は草原だった。緑の草が風に揺れ、遠くには森が見える。

「ここは……塔の中なのか？」

リアムが呆然と呟く。

「幻覚魔法の類か、それとも空間拡張か……」

「どちらでもない」

声が響いた。

草原の向こうから、一人の女性が歩いてくる。白いドレスを着て、長い銀髪を風になびかせている。第一層の守護者アルバとは違う雰囲気。もっと人間的で、温かい。

「私は第二層の守護者、ルクス」

女性が微笑む。

「ようこそ、光の庭へ」

「光の庭……」

シオンが周囲を見渡す。

「とても綺麗」

「ありがとう、お嬢さん」

ルクスが優しく言った。

「この庭は、光の賢者が創った小さな楽園です。でも、ただの庭ではありません」

彼女が指を鳴らすと、周囲の風景が変わった。

草原が荒野に。空が曇天に。温かい光が、冷たい影に。

「光があれば、影も生まれる。それが第二層の試練です」

「影……」

エリナが身構える。

「光と影、表と裏。そのバランスを理解することこそが、第二層の真の目的です」

ルクスが手を広げると、地面から黒い影が湧き上がった。

影は人の形を取り、次々と実体化していく。数は十、二十、三十……。

「私の影を倒してください。光を失わずに」

「光を失わずに？」

私が問い返すと、ルクスは意味深に微笑んだ。

「影と戦ううちに、人は闇に染まってしまうことがあります。怒り、憎しみ、恐怖。それらは闇の感情。光の魔法を使う者として、それらに負けてはいけません」

影たちが一斉に襲いかかってくる。

「来るわよ！」

セシリアが杖を振る。

「LightBarrier！」

光の障壁が影の攻撃を防ぐ。でも、影の数は多すぎる。防ぐだけで精一杯だ。

「反撃しないと！」

エリナが炎の魔法を放つ。

「FireBurst！」

炎が影を焼き尽くす。でも、焼かれた影はすぐに再生する。

「駄目だ、再生が速い！」

リアムが剣で影を切り裂くが、結果は同じだった。

私は状況を分析した。

影が再生する原因。闇に染まるリスク。この二つを解決しないと、試練はクリアできない。

「待って」

シオンが突然、前に出た。

「シオン、危ない！」

私が止めようとするが、彼女は既に影の中に歩み入っていた。

でも、影は彼女を襲わない。

「シオン……？」

エリナが驚いて目を見開く。

シオンは影の一つに手を伸ばした。

「怖くないよ」

彼女が優しく言う。

「あなたたちは、ただ光を求めているだけ」

影が震える。そして、ゆっくりと輝き始めた。

「影は、光がないから生まれるんじゃない」

シオンが振り返る。

「光があるから、影が生まれるの。影は光の証」

その言葉と共に、すべての影が光に変わった。

温かい光が溢れ、荒野は再び草原に戻る。曇天は青空に。

ルクスが満足げに頷いた。

「見事です」

彼女が手を差し出すと、第二の結晶が現れた。

「影を敵とせず、光の一部として受け入れた。それこそが第二層の答えです」

私は結晶を受け取った。第一層のものとは違う、淡い紫色の結晶。

「残り五層。それぞれに異なる試練が待っています」

ルクスが言う。

「でも、あなたたちは大丈夫。光も影も、受け入れる強さがある」

彼女の姿が、光の中に溶けていった。

「……シオン」

私が彼女を見る。

「どうしてわかったんだ？」

シオンは少し恥ずかしそうに笑った。

「わからない。ただ……影が可哀想だと思ったの」

「可哀想？」

「光になりたいのに、光になれない。そんなふうに見えたの」

私は彼女の頭を撫でた。

「すごいよ、シオン」

「えへへ」

彼女は照れくさそうに笑った。

「行こう。次の層へ」

私はみんなを促した。

第二層をクリア。あと五つ。

光の塔は、まだ始まったばかりだった。
