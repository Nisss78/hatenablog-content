第110章：凍てつく山峰

遺跡の北出口から飛び出したカイトたちは、振り返ることなく走り続けた。背後で、エーテリアとアークライトの力が衝突する轟音が響き渡る。

「間に合ったか……！」

リアムが、息を切らしながら呟いた。

「でも、エーテリアは大丈夫なの？」

エリナが、心配そうに後ろを振り返った。遠くで、青と紫の光が激しく交錯している。

「エーテリアは管理者だ。簡単には負けないはずだが——長くは持たないかもしれない」

カイトは、前方に広がる風景に目を向けた。遺跡の北側は、徐々に標高が上がり、遠くには雪に覆われた山々が連なっていた。

「凍てつく山峰……あそこが、最初の遺物がある場所だ」

セシリアが、魔法で簡易的な地図を作り出した。

「凍てつく山峰は、この大陸で最も標高が高い場所です。頂上付近は、万年雪と氷に覆われています」
「最初の光の神殿は、どこにあるんだ？」

カイトが、地図を覗き込んだ。

「山の中腹より少し上、標高四千メートル付近です。そこには、古代の魔術師たちが建てた神殿があります」
「四千メートルか……空気が薄いな」

リアムが、眉をひそめた。

「物理的な厳しさだけじゃない。あの山には、『氷の守護者』がいるって噂だ」
「氷の守護者？」

エリナが、首を傾げた。

「詳細は不明だが、山に入った者を凍らせてしまう怪物だという話だ。実在するかどうかも怪しいが——」
「実在するでしょうな。聖なる遺物を守るためのガーディアンですわ」

リリが、冷静に付け加えた。

// Guardian Analysis
// Location: Frozen Peaks
// Type: Ice-elemental Entity
// Threat Level: HIGH
// Weakness: Fire / Lightning (presumed)
// Note: Requires tactical approach

一行は、山麓の村で準備を整えることにした。凍てつく山峰への入り口に位置する「霜降りの村」——その名の通り、一年中霜が降りる寒い土地だ。

村に入ると、住民たちが怪訝な目で彼らを見た。よそ者を見慣れていないのか、それとも——何かを知っているのか。

「おい、あんたたち」

一人の老人が、声をかけてきた。白い髭を生やし、毛皮のコートを着込んでいる。

「山へ行くつもりか？」
「ええ、そうですが」

カイトが、答えた。

「やめとけ。ここ一ヶ月、山に入った者は誰も帰ってきていない。氷の守護者が、暴れているんだ」
「暴れている？」

セシリアが、問いかけた。

「ああ。普段は山の奥深くにいるんだが、最近は中腹付近でも目撃されている。何かが、守護者を刺激したんだろう」

老人は、意味深な視線を向けた。

「……ヴォルターの差し金かもしれない」

カイトが、小声で呟いた。

「遺物への道を塞ぐために、守護者を意図的に活性化させたのか」

村の宿で、一行は作戦を練った。

「正面から行けば、守護者と戦うことになる。でも、私たちの目的は戦いじゃなくて遺物だ」

エリナが、指摘した。

「守護者を避けて、神殿にたどり着く方法はないか？」
「私のコードで、敵の位置を把握できるかもしれない」

カイトが、端末を取り出した。

// Entity Scanner v2.0
// Function: Detects magical entities within range
// Range: 5km radius
// Note: Requires calibration for ice entities

カイトは、静かに詠唱を始めた。

「scan target = ice_entity; range = 5000; filter = guardian——」

端末の画面に、山の地図が表示された。複数の赤い点が点滅している。

「これが守護者の位置か……数が多いな」

リアムが、画面を覗き込んだ。

「一つじゃないのか？」

「いや、どうやら『氷の守護者』は単体じゃない。複数の氷精霊が、集合意識を形成しているみたいだ」

カイトは、さらに分析を進めた。

// Analysis Result
// Entity Type: Collective Ice Spirits
// Core Location: [3847m altitude - near temple]
// Sub-entities: 47 units (dispersed)
// Behavior: Defensive - attacks intruders on sight
// Weakness Point: Central Core
「中枢コアが神殿の近くにある。つまり、神殿に近づけば必然的に守護者と遭遇する」

カイトは、頭を抱えた。

「戦うしかないか……」

その夜、カイトは一人で村の外に出た。星空の下、冷たい風が頬を撫でる。

エリナが、静かに後ろから近づいてきた。

「眠れないの？」
「……いろいろ考えてて」

カイトは、振り返らずに答えた。

「ミラベルのこと、エーテリアのこと、ヴォルターのこと——この世界は、俺が想像していた以上に複雑だ」
「カイト」

エリナが、彼の隣に並んだ。

「あなたは、この世界を救えるって信じてる？」
「……わからない」

カイトは、正直に答えた。

「でも、やるしかない。俺がこの世界に呼ばれた理由があるなら——それを見つけたい」

エリナは、微笑んだ。

「あなたは、不思議な人ね。プログラマーだったなんて、信じられないくらい」

「どういう意味だ？」
「だって、プログラマーって、もっと……こう、冷静で論理的なイメージがあるもの。でもカイトは、感情豊かで、仲間を大切にして——」

エリナは、言葉を探した。

「……人間的、かな」

カイトは、少し照れた。

「まあ、俺も最初はそういうキャラじゃなかったんだけどな。この世界に来てから——変わった気がする」
「いい変化よ」

エリナが、彼の手を握った。冷たい風の中、その手だけが温かかった。

翌朝、一行は山登りを開始した。

最初の数時間は、順調だった。村の老人が教えてくれた獣道を通り、守護者の検知範囲を避けながら進む。

だが、標高三千メートルを超えたあたりから、状況が変わった。

「寒い……！」

エリナが、震えながら言った。

「魔力を使って、体温を維持して！」

セシリアが、全員に温かい魔法をかけた。

「ありがとう、セシリア。助かるわ」

だが、カイトの注意は別のところに向いていた。端末が、警告を発していた。

// WARNING: Entity approaching
// ID: Ice Spirit Unit-23
// Distance: 800m
// Direction: North-West
// Speed: Fast
「来るぞ！ 北西から、氷精霊が接近してる！」

カイトが叫んだ瞬間、雪原の中から青白い影が飛び出してきた。

それは、人型をした氷の塊だった。透明な体の中に、青い核が光っている。

「これが氷精霊か！」

リアムが、剣を構えた。

「氷のブレスに気をつけろ！ 凍らされるぞ！」

精霊が、口を大きく開けた。冷気が、一直線に向かってくる。

「shield type = fire; target = all——！」

カイトが、炎の障壁を展開した。冷気と炎がぶつかり合い、蒸気が立ち上る。

「今だ！ リアム！」
「おう！」

リアムが、精霊の懐に飛び込んだ。剣に雷を纏わせ、核を突き刺す。

// Critical Hit!
// Damage: 847
// Effect: Stun (2 seconds)

精霊が、悲鳴のような音を立てて崩れ落ちた。氷の破片が、雪の上に散らばる。

「一体倒した！ でも——」

カイトの端末が、さらに警告を発した。

// ALERT: Multiple entities converging
// Count: 12
// ETA: 60 seconds
// Reason: Unit-23 destruction detected
「まずい！ 一体倒したのが、他の精霊に伝わった！」

カイトは、歯を食いしばった。

「逃げよう！ 神殿まで一気に行くぞ！」

一行は、雪原を駆け抜けた。背後から、複数の氷精霊が追いかけてくる。

「あそこだ！ 神殿の入り口が見える！」

セシリアが、指差した。雪混じりの風の向こうに、古代の石造りの建築物がぼんやりと見えた。

「でも、何かいる……！」

エリナが、声を上げた。神殿の入り口前に、巨大な影が立っていた。

それは、他の氷精霊とは比べ物にならないほど巨大だった。高さは十メートルを超え、全身が分厚い氷の鎧で覆われている。

// BOSS ENTITY DETECTED
// ID: Ice Guardian Alpha
// Threat Level: EXTREME
// HP: ??????
// Weakness: Unknown
「これが……守護者の本体か」

カイトは、立ち止まった。背後からは十二体の氷精霊。前方には、巨大な守護者。

「挟み撃ちだ……！」

リアムが、絶望的な声を上げた。

だが、その時——神殿の中から、一筋の光が射した。

「そこにいたか、継承者よ」

声が、響き渡った。若い女性の声だ。

神殿の入り口から、一人の人物が現れた。銀髪の美しい女性。その姿は、エーテリアが見せた hologram の中にいた人物と同じ——

「ミラベル……？」

カイトが、呆然と呟いた。

女性は、微かに笑った。

「いいえ、私はミラベルの『影』です。本体は、まだ深層で眠っています」

彼女は、優雅に手を振った。すると、巨大な守護者が一瞬で凍りつき、動きを止めた。

「急いで。私の力で守護者を抑えていられるのは、数分だけです」
「あなたは何者なんだ？」

カイトが、問いかけた。

「私は、ミラベルが残した予備の人格データ——あなたがたを導くために、四百七十二年前に作られました」

彼女は、神殿の奥を指差した。

「最初の遺物は、この中にあります。急いで——運命の時計は、刻々と進んでいますから」

カイトは、仲間たちと顔を見合わせた。そして、神殿へと駆け込んだ。

凍てつく山峰の頂上で、運命の歯車が動き始めた。
