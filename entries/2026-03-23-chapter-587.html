第587章：選択の代償

魔法陣が、幾重にも重なる。

アルフレッドの周囲に、複雑な幾何学模様が展開し、空間そのものを支配し始めた。

「見よ、これが因果の力だ」

アルフレッドが、静かに告げる。

「私は、この塔を管理する者として、因果を操る権限を持つ。お前の過去を書き換え、存在そのものを消すことも可能だ」

「やれるもんならやってみろ」

カイトが、宝玉を前に突き出す。

「俺は——俺の人生を、誰にも書き換えさせない」

宝玉が、強烈な光を放つ。

カイトの周囲に、金色の結界が張られる。それは、因果の干渉を防ぐ——カイト自身の意思が生み出した守りだった。

「ほう……」

アルフレッドが、感心したように目を細める。

「因果の鍵の力か。なるほど、確かに強力だ。だが——」

彼が、指を鳴らす。

その瞬間、塔全体が振動する。

「私は、塔そのものを操れる。一箇所を守るだけでは、勝てない」

床から、無数の光の棘が突き出す。

カイトは、瞬時に回避する。床を蹴り、空中に飛び上がる。

「GravityManipulation.float(duration: 5)」

重力を操作し、空中に留まる。

「俺だって、ただ守ってるだけじゃない」

カイトは、空中から反撃に出る。

「MagicMissile.volley(target: ALFRED, count: 20)」

二十本の光の矢が、アルフレッドに向かって放たれる。

しかし——

アルフレッドは、動かない。

光の矢が、彼の目の前で見えない壁に阻まれる。

「因果の壁。お前の攻撃は、因果的に『当たらない』と定義した」

「チッ——因果の定義をいじれるのか」

カイトが、舌打ちする。

これは、単純な魔法戦ではない。因果そのものを武器にした戦いだ。

□ ■ □

その頃——

塔の外では、激しい戦いが続いていた。

シオンとグラシエルの戦い。

「氷の刃！」

グラシエルが、両手から冷気を放つ。

氷の刃が、シオンを襲う。

「炎で対抗——FireElement.counter(element.ICE)」

シオンの両手から、熱い炎が噴き出す。

氷と炎が激突し、蒸気が舞い上がる。

「単純な属性相性か。だが——」

グラシエルが、蒸気の中から姿を消す。

「どこ！？」

シオンが、周囲を見渡す。

その背後から——

「ここだ」

氷の槍が、シオンの背中を狙う。

「！」

シオンは、間一髪で回避する。しかし、左腕をかすめる。

「痛っ……」

傷口が、凍り始める。

「私の氷は、傷つけた相手を凍らせる。徐々に全身が凍っていく——」

グラシエルが、冷徹に告げる。

「いずれ、動けなくなる」

「そんな……」

シオンが、凍り始めた左腕を見る。

しかし——彼女は、笑う。

「でも——凍るなら、溶かせばいい！」

「SevenElements.healing().purify(status: FROZEN)」

七色の光が、シオンの体を包む。

凍り始めた部分が、徐々に溶けていく。

「七属性の魔法は、治癒もできる。状態異常も、自分で治せるんだ」

シオンが、両手を広げる。

「だから、負けない！」

グラシエルが、眉をひそめる。

「再生能力まで……厄介な相手だ」

□ ■ □

一方、リアムとエリナは——

ルシフェインと激闘を繰り広げていた。

「うおおおお！ くらえ！」

リアムが、剣を振り下ろす。

ルシフェインは、それを素手で受け止める。

「いい腕だ。だが——まだだ！」

ルシフェインが、剣を押し返す。

膂力（りょりき）で圧倒される。

「くそっ——エリナ！」

「わかってる！ WindSlash.aerial(target)」

エリナの風魔法が、ルシフェインの視界を奪う。

その隙に、リアムが距離を取る。

「支援魔法か……面倒だな」

ルシフェインが、舌打ちする。

「だが、同時に——二人がかりってことは、お前ら一人一人は弱いってことだ」

彼が、地面を叩く。

衝撃波が、エリナを直撃する。

「きゃあ！」

エリナが、吹き飛ばされる。

「エリナ！」

リアムが、叫ぶ。

その隙を——ルシフェインは見逃さない。

「もらった！」

鉄拳が、リアムの腹を捉える。

「がはっ……」

リアムが、崩れ落ちる。

「これで終わりだ」

ルシフェインが、トドメを刺そうと拳を振り上げる——

その時。

「させない！」

エリナが、起き上がる。

「LightningStrike.shock(target)」

雷が、ルシフェインを襲う。

「ぐあっ——！」

ルシフェインが、感電して後退する。

「なっ——まだ動けたか」

エリナが、肩で息をする。

「私は……みんなを守る魔法使い。簡単には、倒れないわよ」

リアムが、苦笑する。

「やるじゃねえか、エリナ」

「あんたこそ……さっさと立ちなさいよ」

「へいへい」

二人は、背中合わせに立つ。

「もう一度——いくぜ！」

□ ■ □

塔の最上階。

カイトとアルフレッドの戦いは、拮抗していた。

カイトの攻撃は、因果の壁に阻まれる。

アルフレッドの因果操作は、宝玉の結界に防がれる。

「このままでは、決着がつかない」

カイトが、判断する。

「なら——因果の核を直接狙う」

彼は、中央の巨大な水晶——因果の核——に目を向ける。

アルフレッドが、それに気づく。

「核を……？ 馬鹿な。核を壊せば、因果が崩壊する。世界そのものが——」

「壊さない」

カイトが、首を振る。

「制御するんだ。アルフレッド、お前が干渉している部分だけを、無効化する」

「CausalityKey.override(target: ALFRED_INTERFERENCE)」

宝玉が、核に向かって光を放つ。

光が、核に吸い込まれていく。

核が、激しく明滅し始める。

「なっ——！」

アルフレッドが、驚愕する。

「お前、因果の鍵の本来の使い方を——」

「ああ。リリスに教わった」

カイトが、答える。

「因果の鍵は、世界を書き換えるためのものじゃない。世界を守るためのものだ」

class CausalityKeyTruePurpose {
    purpose = "PROTECT_WORLD";
    notPurpose = "REWRITE_WORLD";
    
    // 因果の干渉を無効化
    // 世界の自然な流れを守る
    // 誰かの選択を奪わない
    
    override(interference) {
        if (interference.type === "FORCED_REWRITE") {
            return "NULLIFY";
        }
        return "PRESERVE";
    }
}

核の明滅が、激しくなる。

アルフレッドの魔法陣が、次々と消えていく。

「しまった——私の干渉が、解除されている……！」

アルフレッドが、核に手を伸ばす。

「止めろ——止めるんだ！」

しかし——

カイトは、止めない。

「これが、俺の選択だ」

宝玉の光が、最大輝度に達する。

「俺たちは——自分の道を、自分で選ぶ。誰かに決められる必要なんてない」

「CausalityKey.finalRelease()」

核から、まばゆい光が爆発する。

光が、塔全体を包み込む。

そして——

アルフレッドの体が、光の中に消えていく。

「カイト……お前は……」

アルフレッドが、最後に何かを呟く。

しかし、その言葉は光にかき消された。

□ ■ □

光が収まると——

アルフレッドの姿は、消えていた。

核は、静かに輝いている。

カイトが、宝玉を見る。

宝玉の光は、穏やかだった。

「終わった……のか？」

カイトが、膝をつく。

魔力を使い果たし、体が重い。

しかし——

「やった……アルフレッドを止めた」

彼は、塔の窓から外を見る。

外では——

グラシエルとルシフェインの姿が、消えていた。

十三使徒たちは、アルフレッドの力を失い、撤退したのだ。

そして——

シオン、リアム、エリナ、セシリアが、無事に立っていた。

「みんな……」

カイトが、安堵する。

「勝ったんだ」

□ ■ □

塔の外に降りると、仲間たちが駆け寄ってきた。

「カイト！」

エリナが、最初に抱きつく。

「無事だったのね！ 心配したわよ！」

「ああ、なんとかな」

カイトが、苦笑する。

「アルフレッドは……？」

「消えた」

カイトが、静かに答える。

「因果の鍵で、彼の干渉を無効化した。そうしたら——彼自身も、因果の中に還っていった」

「還った……？」

リアムが、首をかしげる。

「どういうことだ？」

「アルフレッドは——最初から、因果の一部だったのかもしれない」

カイトが、推測する。

「彼が言ってたんだ。『完全な世界を作る』って。それは——因果そのものと一体化することだったのかもしれない」

セシリアが、眼鏡を光らせる。

「興味深い仮説ね。アルフレッドは、因果の管理者としての役割を果たしていた。だが、その役割が終われば——因果に還る。それが、彼の運命だったのかも」

「悲しい運命だね」

シオンが、悲しげに言う。

「でも——彼は、自分で選んだんだよね？ 世界を変えることを」

「ああ」

カイトが、頷く。

「彼は、自分の選択を貫いた。俺は、俺の選択を貫いた。どっちが正しいかは——誰にもわからない」

「でも」

エリナが、カイトの手を握る。

「私たちは、自分たちの選択を信じる。それでいいんじゃない？」

「……そうだな」

カイトが、微笑む。

「俺たちの選択——俺たちが、責任を持つ」

□ ■ □

因果の塔の前に、一行は立っていた。

塔は、静かに輝いている。

世界は——変わっていないように見える。

しかし、確実に変わったものがある。

アルフレッドという脅威が消え、世界再構築の危機が去った。

「これで——終わりなの？」

シオンが、聞く。

「終わり——ではないだろうな」

リアムが、答える。

「世界には、まだ問題がある。十三使徒の残党もいるかもしれない。新たな脅威も現れるかもしれない」
「でも——私たちは、戦える」

セシリアが、言う。

「カイトが因果の鍵を持っている。世界を守る力がある。私たちには、仲間がいる。それがあれば——何度でも立ち上がれる」

カイトが、仲間たちを見渡す。

エリナ——いつもそばにいてくれる魔法使い。

リアム——頼れる相棒の剣士。

セシリア——冷静な分析官。

シオン——新たな仲間、七属性の魔法使い。

「ああ」

カイトが、頷く。

「俺たちは、これからも旅を続ける。世界を守りながら、自分たちの道を選びながら」

「それは——終わらない旅かもしれない」

「でも——悪くない」

エリナが、笑う。

「私も、そう思うわ」

「私も！」

シオンが、手を上げる。

「みんなと一緒なら、どこまでも行ける！」

リアムが、剣を肩に担ぐ。

「じゃあ、次はどこに向かうんだ？」

カイトが、空を見上げる。

因果の塔の上空には、七つの光が輝いていた。

世界を守る七つの賢者たち——その象徴。

「とりあえず——戻ろう」

カイトが、言う。

「みんなに報告しなきゃ。それに——少し休憩も必要だ」

「賛成！」

エリナが、両手を上げる。

「もう、へとへとなのよ。美味しいご飯と、柔らかいベッドが欲しい！」

「俺は、酒が飲みたいな」

リアムが、笑う。

「勝利の美酒ってやつだ」

「私は、お風呂に入りたい」

シオンが、言う。

「ずっと戦ってて、汗だくなの」

セシリアが、眼鏡を外す。

「私は——データの整理をしたいわね。今回の戦いで得た情報を、分析しないと」

「相変わらずだな、セシリア」

カイトが、苦笑する。

「でも——それがいい」

□ ■ □

一行は、因果の塔を後にする。

塔は、静かにその場に残る。

世界の因果を守り続ける——永遠の番人として。

カイトは、最後に一度だけ振り返る。

「アルフレッド……」

彼は、小さく呟く。

「お前の選択は、間違ってたかもしれない。でも——お前が選んだことは、覚えてる」

「俺たちは——違う道を選んだ。それが、正しかったと信じてる」

「いつか——因果の向こう側で、また会えるかもしれないな」

風が吹く。

カイトの髪を、優しく撫でていく。

それは——アルフレッドの返事のように感じられた。

「……行こう」

カイトは、前を向く。

仲間たちが、待っている。

新しい冒険が、待っている。

「俺たちの物語は——まだ終わらない」

「これからも、続いていく」

一行は、光の中へと歩き出した。
