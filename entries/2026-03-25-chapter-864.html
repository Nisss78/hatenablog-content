第864章：援軍の影

修行四日目の朝、カイトは予期せぬ来訪者を目にした。

地下聖域の入り口で、見覚えのある姿が立っていた。風の賢者ウィンディアだ。彼女の背後には、数名の風魔法使いが控えている。

「ウィンディアさん？ どうしてここに？」

カイトが駆け寄ると、ウィンディアは微かに微笑んだ。

「水晶の予言が届いたの。四月一日の攻撃、でしょう？」

「はい。地の聖域が目標だと予知されています」

「ならば、放っておけないわ。風の聖域も脅威を受けているけど、ここが最も危険なら援軍を送るべき」

カイトは胸が熱くなるのを感じた。仲間が助けに来てくれた。一人で戦うのではない、この感覚。

テラドンがカイトの横に並んだ。

「ウィンディア、感謝する。風の聖域の防衛は大丈夫なのか？」

「シオンとヒナタが守ってくれているわ。あの子たちの成長、目を見張るものがある」

シオン。銀髪の記憶喪失少女で、七属性の魔法回路を持つ少女のことだ。ヒナタは風の剣士で、シオンの旅の仲間だ。

「二人に任せて本当に大丈夫なんですか？」

カイトの心配そうな声に、ウィンディアは笑って答えた。

「過保護ね、カイト。シオンは七属性使いよ。普通の賢者より強いかもしれない」

「そう……ですよね」

ウィンディアたちは地下聖域の客室に案内された。カイトは彼女と共に戦略会議を行う。

机の上には、地の聖域の見取り図と、敵の予想進路が描かれた地図が広げられている。

「四月一日、敵は三方向から攻めてくる見込みよ」

ウィンディアが地図を指差した。

「北東から使徒グラシエルの氷結軍団。南東から使徒テンペストの嵐の部隊。そして正面から、使徒ダークハルト直率の虚無軍」

「三方面同時攻撃……手ごわいですね」

「ええ。結界で守りきれるかどうか」

カイトは地脈との同調を思い出した。半永続的な結界なら、三方面の攻撃にも耐えられるかもしれない。

「僕が結界を担当します。地脈から魔力を引き出せば、一日程度は維持できますから」

「それは心強いわ。でも、結界を張っている間は動けないわよね？」

「はい。意識を集中し続ける必要があります」

ウィンディアは顎に手を当てて考えた。

「なら、結界内の戦闘は私たちに任せて。あなたは盾になり、私たちは剣になる」

「分かりました」

午後、カイトはウィンディアと模擬戦を行った。風の魔法は速く、予測が困難だ。

「遅い！」

ウィンディアの風の刃が、カイトの頬をかすめた。紙一重の回避だが、次々と攻撃が迫る。

「未来予知があるのに、なぜ当たるの？」

「視えても、身体が追いつかないんです」

「なら、先読みの精度を上げるしかないわね」

ウィンディアは攻撃の手を緩めず、講義を続けた。

「未来予知は『何が起きるか』を見る力。でも、真に重要なのは『どう対処するか』よ」

「どう対処するか……」

「視えた未来に対して、最適な行動を選ぶ。その判断の速さが、生死を分ける」

カイトは理解した。未来予知は万能ではない。視えた情報を処理し、行動に移す能力がセットで必要なのだ。

「では、模擬戦を続けましょう。今度は、視えた未来に対して最速の反応を目指します」

「いい返事ね」 


// 反応速度トレーニング
const reactionTraining = {
    trials: 50,
    successThreshold: 200, // ミリ秒
    results: []
};

for (let i = 0; i < reactionTraining.trials; i++) {
    const attack = generateWindAttack();
    const prediction = predictFuture(attack, 500);
    
    // 予測から行動までの時間
    const reactionStart = Date.now();
    const action = selectOptimalResponse(prediction);
    const reactionEnd = Date.now();
    
    const reactionTime = reactionEnd - reactionStart;
    reactionTraining.results.push({
        trial: i + 1,
        reactionTime: reactionTime,
        success: reactionTime < reactionTraining.successThreshold
    });
}

const successRate = reactionTraining.results.filter(r => r.success).length / reactionTraining.trials;
console.log(`成功率: ${successRate * 100}%`);

五十回の試行の後、カイトの反応成功率は六割程度まで向上した。まだ不十分だが、初日よりは遥かにマシだ。

「上達が早いわね。三日でここまで来るなんて」

ウィンディアが称賛してくれた。カイトは少し照れくさくなった。

「良い先生がいるからです」

「おだてても何も出ないわよ」

二人は笑い合った。緊張感の中にも、温かい空気が流れていた。

夜、カイトは再び水晶と向き合った。援軍が来たことで、未来がどう変わったか確認したい。

「水晶よ、ウィンディアたちの援軍が加わった後の勝率は？」

水晶の光が激しく明滅し、複数の未来を映し出す。

一つ目の未来は、結界と援軍の連携が完璧に機能し、敵を撃退する場面。確率は六十五パーセント。

二つ目の未来は、結界が破られ、混戦の末に辛勝する場面。確率は二十パーセント。

三つ目の未来は、敵の予想外の戦力に圧倒され、敗北する場面。確率は十五パーセント。

「六十五パーセント……以前より十ポイント上昇した」

カイトは安堵したが、同時に疑問も浮かんだ。

「でも、まだ三割以上の確率で負ける可能性がある。もっと援軍が来れば……」

水晶は新たな映像を映し出した。雷鳴の峰からライラが、氷の聖域からフローズナが、それぞれ向かっている場面だ。

「二人も来てくれるのか！」

カイトは声を上げた。二人が加われば、勝率はさらに上がるはずだ。

「残りのリスクは何だ？」

水晶が答えを示した。敵の指揮官、ダークハルトの個人戦力だ。最高位管理者であり、闇と虚無を融合させた怪物。彼との直接対決になれば、勝率は五分五分になるという。

「ダークハルトか……」

カイトは名前を反芻した。かつての管理者が、虚無に侵食され、敵に堕ちた存在。彼を救う道はあるのか、それとも倒すしかないのか。

翌朝、カイトはテラドンにダークハルトについて尋ねた。

「ダークハルトは、かつてどんな存在だったんですか？」

テラドンは静かに語り始めた。

「彼は……かつては偉大な管理者だった。虚無の研究において、誰よりも深い知見を持っていた」

「虚無の研究？」

「ああ。虚無は世界の脅威だが、同時に根源的な力でもある。彼はそれを理解し、制御しようとした」

「そして、失敗した」

「そうだ。虚無に飲み込まれ、自我を失った。今の彼は、かつてのダークハルトの面影を残しているが、中身は虚無の意思そのものだ」

カイトは胸が締め付けられるのを感じた。虚無の研究で偉大な成果を上げた管理者が、その研究ゆえに堕ちた。皮肉な運命だ。

「彼を救う方法は……？」

「ないとは言えない。だが、極めて困難だ。虚無から自我を取り戻した例は、世界中を探しても見つからない」

「でも、可能性はゼロじゃない」

カイトは言った。テラドンは困ったように笑った。

「お前は、救いの道を探すのか」

「戦って倒すだけなら、それはそれでいい。でも、もし救えるなら、救いたいです」

「水晶、一つ教えてくれ。もし僕が四月一日の戦いで命を落とすとしたら、それはどんな未来だ？」

水晶は沈黙した後、一つの映像を映し出した。結界が破られ、敵の主力が聖域になだれ込む。カイトは仲間を守るため、単身で敵の中心に突っ込む。そして、数多の敵に囲まれ、光の中に消える。

「自己犠牲の未来……か」

カイトは静かに受け入れた。自分が命を落とす未来がある。だが、それは同時に、仲間を守るための未来だ。

「悪くない死に方だ」

「流れ星……」

カイトは目を閉じ、願った。

「四月一日、皆が無事でありますように」

愚かかもしれない。星に願うなど、子供じみている。だが、今のカイトには、それしかできなかった。

窓の外で、地脈の光が静かに揺れていた。大地もまた、四月一日を待っているのかもしれない。

カイトは深呼吸をして、目を閉じた。明日も、修行が待っている。
