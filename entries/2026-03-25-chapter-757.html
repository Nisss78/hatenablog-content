第757章：風の呼び声

螺旋階段を登るカイトたちの耳に、風の音が届き始めた。

最初は微かなささやき。次第に、強い風の唸り声へと変わっていく。

「第三層は風の試練か」

カイトは階段の壁に手を添えた。

壁から風が漏れている。冷たく、鋭い。

「風系魔法は苦手じゃない？」

シオンがカイトを見た。

「得意ではないけど、使えないわけじゃない」

カイトは苦笑した。

階段が終わり、第三層への扉が現れた。

扉は緑色。風の紋様が刻まれ、微かに揺れている。

「風の精霊の祝福がある扉だ」

リアムが感じ取った。

「いい予感がする」
「俺もそう思う」

カイトは扉を押し開けた。

第三層の空間が広がっていた。

そこは、空中に浮かぶ無数の島々だった。

大小様々な岩の島が、空に浮いている。島同士は、細い石橋で繋がっている。

下を見ると、雲海が広がっている。底は見えない。

「落ちたら、終わりってこと？」

エリナが不安そうに言った。

「たぶん」

カイトは正直に答えた。

風が強い。

カイトたちは体を低くして、風に耐えた。

「この風の中で、進まないといけないのか」

リアムが風を盾で防いだ。

「風の守護者が、どこかにいるはずだ」

カイトは前方を見据えた。

島々の間を、風が吹き抜けていく。

風の音が、まるで言葉のように聞こえる。

「……聞こえる？」

シオンが耳を澄ませた。

「風が、何かを言っている」

カイトも耳を澄ませてみた。

確かに、風の中に声が混じっている。

「来たか、試練を受ける者よ」

「誰だ？」

カイトは周囲を見回した。

風が渦を巻き、一箇所に集まっていく。

渦の中から、一人の人物が現れた。

緑色の髪の男性。軽装の服装。背中には、透明な羽が生えている。

「第三層の守護者、ゼフィルスだ」

男性は軽やかに着地した。

「僕はカイト。管理者の資格を得るために来た」

カイトは名乗った。

「ミリアから話は聞いているよ」

ゼフィルスは微笑んだ。

「水の試練を突破した君たちだ。風の試練も楽しんでくれるはずだ」
「風の試練は何をすればいいんですか？」

カイトが尋ねた。

「三つの試練がある」

ゼフィルスは指を三本立てた。

  • 第一の試練：暴風の回廊
  • 第二の試練：空の迷宮
  • 第三の試練：風の心臓
「三つともクリアすれば、第三層通過だ」

ゼフィルスは風に乗って浮遊した。

「じゃあ、第一の試練から始めよう」

カイトは覚悟を決めた。

「暴風の回廊とは？」

ゼフィルスが指差した先には、狭い石橋があった。

橋の両側から、強烈な横風が吹いている。

「あの橋を渡ればクリアだ」

ゼフィルスは説明した。

「ただし、一度落ちたらやり直しだ」

カイトは橋を見つめた。

幅は一メートル程度。長さは五十メートル以上。

風が不規則に吹いている。瞬きする間に、風向きが変わる。

「普通に渡るのは無理そうだ」

カイトは分析した。

「風を読んで、タイミングを計る必要がある」
「私が先に行く」

リアムが申し出た。

「盾で風を防ぎながらなら、安定するはず」
「頼む」

カイトは頷いた。

リアムは橋に足を踏み入れた。

横風が吹く。リアムは盾を風上に向け、風を受け流す。

一歩、また一歩。確実に進んでいる。

「いい調子だ」

カイトは見守った。

リアムが橋の中間に達した時、風が急に強まった。

突風。盾が弾かれ、リアムの体が傾く。

「リアム！」

エリナが叫んだ。

リアムはなんとか体勢を立て直した。

でも、風は止まない。むしろ、さらに強くなっている。

「魔法でサポートする」

カイトは魔法を発動した。


// 風の盾魔法
function windBarrier(target) {
  const magic = {
    type: "風",
    subtype: "deflection",
    target: target,
    radius: 2,
    effect: "風を自動的に逸らす"
  };
  
  return `${target}の周囲に風の盾を展開`;
}

// 実行
windBarrier("リアム");

リアムの周りに、見えない盾ができた。

風が自動的に逸らされていく。

リアムは再び前に進み始めた。

橋の終点が見えてきた。

リアムが到達した瞬間、風が止んだ。

「到着！」

リアムは振り返って手を振った。

「次は僕たちの番だ」

カイトはエリナとシオンを見た。

「三人同時に行こう。連携すれば、風にも対抗できる」
「うん」

シオンが頷いた。

カイト、エリナ、シオンの三人は、橋に並んで立った。

カイトが先頭。エリナが中央。シオンが最後尾。

「いくぞ！」

カイトの合図で、三人は歩き始めた。


// 結束の風壁
function groupWindWall(members) {
  const magic = {
    type: "風",
    subtype: "collective_barrier",
    members: members,
    synergy: "high"
  };
  
  return `${members.join("、")}の結束で風壁を形成`;
}

// 実行
groupWindWall(["カイト", "エリナ", "シオン"]);

三人の魔法が重なり合い、強固な壁となった。

風が襲ってくる。でも、壁に当たって霧散する。

三人は足並みを合わせて進んだ。

中間地点を過ぎた。

突風。風向きの変化。嵐のような暴風。

でも、三人の結束は揺るがない。

橋の終点に到達した。

「クリアだ！」

カイトは叫んだ。

四人は無事に、暴風の回廊を渡り切った。

ゼフィルスが現れた。

風を纏って、悠々と浮遊している。

「見事だ。四人の連携、完璧だった」

ゼフィルスは称賛した。

「次は、第二の試練だ」

ゼフィルスが指を鳴らした。

前方の島々が、空中で動き始めた。

島が回転し、入れ替わり、新しい形を作っていく。

「空の迷宮」

ゼフィルスが説明した。

「動く島々の先にある、ゴール地点にたどり着けばクリアだ」

カイトは迷宮を見つめた。

島々が複雑に入り組んでいる。道は常に変化している。

「迷子になりそうだ」

シオンが不安そうに言った。

「風を読めば、正しい道がわかるはず」

カイトは言った。

「風は常に、ゴールに向かって流れている」
「カイトの言う通りだ」

ゼフィルスが補足した。

「風を読める者だけが、この迷宮を抜けられる」

カイトは目を閉じ、風の流れを感じ取った。

風が語りかけてくる。

「こちらだ。こちらへ」

カイトは風の導きを感じた。

「こっちだ」

カイトは最初の島に向かって歩き出した。


// 風の導き
function windGuidance() {
  const magic = {
    type: "風",
    subtype: "pathfinding",
    effect: "風の流れを可視化"
  };
  
  return "風の道筋を可視化し、正しいルートを表示";
}

// 実行
windGuidance();

カイトの目に、風の流れが光の線として見えた。

カイトたちは、風の導きに従って進んだ。

島から島へ。橋から橋へ。

島が動いて道が消えても、風は新しい道を示してくれる。

「カイト、すごい」

エリナが感心した。

「風が見えてるみたい」
「見えてるんだ。風の道が」

カイトは答えた。

迷宮の深部に入った。

島々の動きが激しくなる。

カイトたちは走った。島が動く前に、次の島へ飛び移る。

「あと少し！」

カイトはゴール地点を見つけた。

最後の島が、遠く離れていく。

跳躍力では届かない。

「風魔法で飛ぶ！」

カイトは魔法を発動した。


// 疾風跳躍
function galeLeap(target, destination) {
  const magic = {
    type: "風",
    subtype: "propulsion",
    target: target,
    destination: destination
  };
  
  return `${target}を${destination}まで風で射出`;
}

// 実行
galeLeap("カイトと仲間たち", "ゴール島");

四人は風に乗って飛んだ。

空中で、島々の隙間を縫う。

そして、ゴール島に着地した。

「着いた！」

シオンが歓声を上げた。

ゼフィルスが現れた。

「風の導きを見抜いたか。素晴らしい」

ゼフィルスは満足そうに頷いた。

「最後の試練に進もう」

ゼフィルスが手をかざした。

ゴール島の中央に、巨大な竜巻が現れた。

竜巻の中に、何かが見える。

淡い緑色の光。

「風の心臓」

ゼフィルスが言った。

「竜巻の中にある光を、手に入れればクリアだ」

カイトは竜巻を見つめた。

風速は数百キロ。普通なら近づくのも危険だ。

「竜巻の中に入るのか」

カイトは確認した。

「そうだ。風に逆らうのではなく、風に乗るんだ」

ゼフィルスはアドバイスした。

「風に乗る……」

カイトは考えた。

逆らうのではなく、同調する。風と一体化する。

「わかった。やってみる」

カイトは決意した。


// 風同調魔法
function windSync(user) {
  const magic = {
    type: "風",
    subtype: "synchronization",
    user: user,
    effect: "風と一体化して制御を得る"
  };
  
  return `${user}が風と同調`;
}

// 実行
windSync("カイト");

カイトの体が、半透明の風になった。

カイトは竜巻に入った。

風が荒れ狂う。でも、カイトは風の一部となって流れている。

風が体を叩く。でも、痛みはない。

風が言葉を語りかけてくる。

「風は、自由だ」

「風は、繋がりだ」

「風は、すべてを運ぶ」

カイトは風の哲学を受け取った。

竜巻の中心に到達した。

そこは、不思議なほど静かだった。

暴風の目。嵐の中の静寂。

その中心に、風の心臓が浮いていた。

緑色の光球。中には、小さな羽が見える。

カイトは光球に手を伸ばした。

触れた瞬間、ビジョンが走った。

かつて、空に住む種族がいた。

翼を持つ人々。風と共に生きる者たち。

彼らは自由で、優しく、誰とでも友達になれた。

でも、時代が変わり、人々は空を離れた。

残ったのは、風の心臓だけ。

ビジョンが終わった。

カイトは光球を抱きしめた。

「君たちの自由さ、忘れないよ」

カイトは竜巻の中心で誓った。

カイトは竜巻から脱出した。

風同調を解き、元の体に戻る。

足元の島に着地した。

「カイト！」

仲間たちが駆け寄った。

カイトは光球を掲げた。

緑色の光が、空を照らす。

「風の心臓、手に入れた」

カイトは宣言した。

ゼフィルスが近づいてきた。

その目には、静かな感動があった。

「風と同調した者は、久しぶりだ」

ゼフィルスは言った。

「君は、風の本質を理解している」
「風は自由で、風は繋がり、風はすべてを運ぶ」

カイトはビジョンで得た言葉を繰り返した。

「その通りだ」

ゼフィルスは満足そうに頷いた。

「私は、空の民の最後の一人だった」

ゼフィルスは語り始めた。

「地上に降りた同胞たちを見守るために、ここに留まっている」
「でも、風の心臓が新しい管理者に受け継がれた。これで、私も解放される」

ゼフィルスの体が、風に溶け始めた。

「ゼフィルスさん！」

シオンが叫んだ。

「大丈夫。これは、風に還るだけだ」

ゼフィルスは笑った。

「カイト、次の試練も頑張って。風はいつも、君の味方だ」

ゼフィルスは風になり、空へ消えていった。

残されたのは、穏やかな風だけ。

「さようなら、ゼフィルスさん」

カイトは空に向かって手を振った。

第三層に、静寂が戻った。

「第三層、クリアだな」

リアムが言った。

「ああ。でも、まだ四層残ってる」

カイトは空を見上げた。

第四層への階段が現れた。

階段からは、熱気が漂ってくる。

「次は、炎の試練か」

カイトは予感した。

「熱くなりそうだ」

カイトたちは、第四層への階段を登り始めた。

風の教えを胸に、新たな試練に挑む。

第757章、了。
