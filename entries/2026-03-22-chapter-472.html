第472章：地への道標

翌朝、カイトたちはイヴェルナの塔を後にした。

氷見の村を出て、南東へと向かう。地の賢者テラドンが住む場所へ向かうためだ。

「テラドンさんの場所、分かるの？」

エリナが、イヴェルナに尋ねた。

「ええ。彼は、大陸の中央にある『地脈の聖域』に住んでいます」

イヴェルナは、氷の杖を指先でくるくると回しながら答えた。

「そこは、この世界の地脈が交差する場所。大地のエネルギーが最も強い地点です」
「遠いの？」

リアムが、剣の重さを確かめながら聞いた。

「徒歩で二週間。でも、私が道を開けば、三日で着きます」
「道を開く？」

カイトが興味を持った。

「氷の賢者として、空間を凍らせて通路を作ることができます。『氷の回廊』と呼ばれる移動手段です」

イヴェルナは、杖を振った。すると、目の前の空間が歪み、青白い光のトンネルが現れた。

「これが、氷の回廊。中は寒いですが、耐えてください」

カイトたちは、光のトンネルの中へと足を踏み入れた。

◆ ◆ ◆

氷の回廊の中は、イヴェルナの言った通り寒かった。でも、ただ寒いだけではない。

壁の中に、風景が見える。まるで、万華鏡の中を通っているようだ。山々、森、川、村。すべてが、青い光に包まれて流れていく。

「すごい……。これ、どういう仕組みなんですか？」

カイトが尋ねると、イヴェルナは得意げに答えた。

「空間そのものを凍らせて、二点間の距離を縮めています。原理的には、あなたのコードに近いかもしれない」
「空間を、コードで書き換えるようなもの？」

「ええ。ただし、私は氷魔法を媒介にしていますが」

カイトは、イヴェルナの話を聞きながら、自分の力について考えた。

コードで世界を記述する力。それは、魔法を超えた魔法。イヴェルナの氷の回廊も、結局は現象を操る範囲内だ。でも、カイトのコードは、現象そのもののルールを書き換えることができる。

「僕の力、もっと使いこなせるようにならないかな」

カイトは、独り言のように呟いた。

「大丈夫ですよ。テラドンさんなら、何か教えてくれるはず」

セシリアが、カイトの肩をポンと叩いた。

「彼は、七賢者の中で最も知識が豊富です。あなたの力についても、何か知っているかもしれません」

◆ ◆ ◆

三日後、カイトたちは氷の回廊を抜けた。

目の前に広がっていたのは、巨大な岩山だった。頂上は雲の中に隠れ、麓には太い蔦が絡み合った森が広がっている。

「ここが、地脈の聖域」

イヴェルナが、静かに言った。

「テラドンさんは、この岩山の中腹にある洞窟に住んでいます」
「洞窟か。入り口はどこ？」

リアムが、岩山を見上げながら尋ねた。

「入り口はありません」
「えっ？」
「テラドンさんは、訪問者を選びます。資格がない者は、近づくことすらできません」

イヴェルナの言葉に、カイトたちは顔を見合わせた。

「じゃあ、どうすれば？」

カイトが聞くと、イヴェルナは微かに笑った。

「あなたは、预言の者。資格は十分です。ただ、自分で道を切り開く必要があります」
「自分で？」
「ええ。テラドンさんは、試練を課さない代わりに、訪問者の本気を求めます。崖を登る也好、岩を砕く也好。自分の力で、彼のもとにたどり着いてみせなさい」

カイトは、岩山を見上げた。険しい崖、太い蔦、所々に見える洞窟の入り口のような窪み。普通の登山なら、装備を整えて数日かかるだろう。

「よし。やってみる」

カイトは、風魔法を使って体を軽くした。


function windWalk(): void {
  const wind = new WindSpell("support");
  wind.reduceWeight(0.3); // 体重の30%を軽減
  wind.enhanceJump(2.5); // ジャンプ力を2.5倍に
}

カイトの体が、ふわりと浮き上がる。

「僕が先に行く。みんなはついてきて」
「待って、カイト。私も手伝う」

セシリアが、光魔法を唱えた。


function illuminatePath(): void {
  const light = new LightSpell("guidance");
  light.markSafeRoute(); // 安全なルートを照らす
  light.detectDangers(); // 危険箇所を感知
}

岩山の表面に、光の筋が走った。登るべきルートが、黄金色の光で示される。

「すごい、セシリア」
「光魔法は、道を示すのが得意なの」

セシリアは、少し照れくさそうに笑った。

◆ ◆ ◆

登り始めて一時間。カイトたちは、岩山の中腹に差し掛かった。

ここからは、蔦がなくなり、裸の岩肌が続く。手掛かりも足掛かりも少なく、慎重に進まないと落ちてしまう。

「カイト、あそこ」

エリナが指差した先に、洞窟の入り口が見えた。でも、入り口の前に何かが立っている。

岩でできた巨大な番人だ。高さは五メートル以上。角ばった体に、光る二つの目。

「ゴーレム……」

リアムが、剣を抜いた。

「戦うのか？」
「待って」

カイトは、リアムを制止した。

「テラドンさんは、訪問者の本気を求めるって言った。戦うことが答えじゃないかもしれない」

カイトは、ゴーレムの前に進み出た。そして、静かに声をかけた。

「僕たちは、地の賢者テラドンさんに会いに来ました。通してくれますか」

ゴーレムは、動かなかった。ただ、光る目でカイトを見つめている。

「……沈黙か」

カイトは、コードの視界を使ってゴーレムを分析した。


class EarthGolem {
  private core: MagicCore;
  private instructions: string[];
  
  public getInstructions(): string[] {
    return this.instructions;
  }
}

ゴーレムの内部構造が、カイトの脳裏に浮かんだ。魔法の核と、そこに刻まれた指示。指示の内容を読み取ろうとすると、一つの文が見えた。

『真実を語る者のみ、通すべし』

「真実を語る者……」

カイトは、考えた。真実とは何か。自分は、何を語るべきか。

「僕は、コードで世界を記述する力を持っています。でも、その力が何なのか、まだ完全には分かっていません。僕は、自分の力を理解するために、テラドンさんに会いたい」

ゴーレムは、動かなかった。

「僕は、世界を救いたい。そのためには、僕の力が必要だと言われています。でも、僕は自分が本当に世界を救えるのか、確信を持てません。だから、確かめたいんです」

ゴーレムの目が、瞬いた。

「僕の名前は、海藤翔太。カイトと呼ばれています。プログラマーで、異世界から来ました。これが、僕の真実です」

ゴーレムが、動いた。ゆっくりと体を横にずらし、洞窟の入り口を開けた。

「通れた！」

エリナが、歓声を上げた。

「カイト、すごい。どうやって？」
「真実を話せ、って言われてたんだ。嘘をつかず、飾らず、自分の言葉で語る。それが答えだった」

カイトは、振り返って仲間たちに微笑んだ。

「行こう。テラドンさんが待ってる」

◆ ◆ ◆

洞窟の中は、意外なほど明るかった。

天井から垂れ下がる水晶が、自然の光を取り込んでいる。壁には、古代の壁画が描かれていた。大地、植物、動物、人間。生命の循環を描いた物語のようだ。

カイトたちは、洞窟の奥へと進んだ。やがて、広い空間に出た。

中央に、一人の老人が座っていた。

肌は岩のようにゴツゴツしており、髭は苔のように長い。でも、その目は澄んでいて、深い知性を感じさせる。

「待っていたよ、预言の者」

老人が、静かな声で言った。地響きのような、低く安定した声。

「あなたが、地の賢者テラドンさん？」
「いかにも。そして、お前がカイト」

テラドンは、ゆっくりと立ち上がった。身長は二メートル近くある。岩の巨人という表現が似合う威厳だった。

「イヴェルナから話は聞いている。氷の試練を突破し、彼女の認定を得たとな」
「はい」
「だが、氷は氷。地は地だ。お前がここに来たからには、地の試練も受けるつもりだろう」

カイトは、頷いた。

「はい。僕の力を試してください」
「よかろう」

テラドンは、広間の中央に手を置いた。

「私の試練は、一つ。この大地の声を聞き、その意味を理解すること」
「大地の声？」
「この世界は、生きている。大地もまた、言葉を持っている。お前のコードで、その言葉を聞いてみよ」

カイトは、床に手を触れた。冷たくて硬い岩肌。でも、その奥に何かを感じる。

カイトは、目を閉じた。コードを紡ぐ。


function listenToEarth(): void {
  const earth = new EarthEntity();
  earth.setSensitivity("maximum");
  const voice = earth.getVoice();
  console.log(voice);
}

カイトの意識が、地下へと潜っていく。

岩の層、地下水脈、地熱の流れ。そして、もっと深く。マグマの海、地球のコア、その中心にある何か。

そこで、カイトは聞いた。

大地の声を。

『助けて……』

それは、苦しげな声だった。

『世界が……壊れていく……』

カイトは、目を見開いた。

「大地が、何かを訴えている」

カイトは、テラドンに向かって言った。

「世界が壊れていくって。何かが、大地を苦しめている」

テラドンは、満足げに頷いた。

「その通りだ。お前は、大地の声を聞いた。これが、地の試練の答え」
「でも、何が大地を苦しめているんですか？」
「それは、次の試練で分かる」

テラドンは、カイトに近づいた。

「カイト。お前は、世界の危機を感じ取った。だが、その正体はまだ見えていない。真の敵は、目に見えるものではないかもしれない」
「真の敵？」
「ああ。お前が戦うべき相手は、この世界そのものかもしれない」

カイトは、テラドンの言葉の意味を理解できなかった。でも、一つだけ確信したことがある。

この旅は、まだ始まったばかりだ。
