第442章：灼熱の砂漠

氷の湖を離れて三日目。

カイトたちは、景色が劇的に変化する境界線に到達した。

背後には緑豊かな森と湖。前方には、見渡す限りの金色の砂丘。空は焼けた銅のような色に染まり、地面からは陽炎が立ち昇っている。

「すごい……」

セシリアが思わず呟いた。

「氷の湖とは正反対ね」

エリナが額の汗を拭いながら言う。すでに気温は十度以上上がっている。

リアムは水筒を確認しながら言った。

「水は十分にあるが、砂漠では消耗が激しいだろう。慎重に行こう」

カイトは頷いた。氷の種が懐で微かに震えているのを感じた。極寒の属性を持つ種が、この熱さに反応しているのか。

Environment.scan() {
  気温: 42°C
  湿度: 8%
  魔力濃度: 低
  危険度: 中
  推奨: 水分補給・熱中症対策
}

砂漠に入って数時間。

足を一歩進めるたびに、砂が ankle まで埋まる。体力が想像以上に削られていく。

「カイト、大丈夫？」

エリナが心配そうに覗き込む。カイトは笑顔を作った。

「大丈夫。ただ、想像以上に厳しいな」

カイトは魔法で体温を下げようと試みたが、環境の熱が強すぎて効果が薄い。

ThermalMagic.adjust(target: "body", delta: -5) {
  結果: 部分成功
  効果: 一時的な冷却
  持続時間: 3分
  消費魔力: 中
  警告: 環境熱が干渉
}

リアムが提案した。

「一度休憩しよう。日が沈むまで待って、夜に移動した方がいいかもしれない」

全員が同意した。日差しは容赦なく、休む場所もない。ようやく見つけた岩陰に、四人は身を寄せ合った。

夕暮れ時。空が紫色から濃紺へと変わっていく。

砂漠の夜は、昼とは対照的に急速に冷え込む。カイトたちは温かい服を着込み、焚き火を囲んだ。

「砂漠の夜は冷えるのね」

セシリアが驚いたように言う。

「砂は熱を逃がしやすいんだ。昼間の熱を保てない」

リアムが説明した。冒険者としての知識が役立っている。

カイトは焚き火を見つめながら、氷の種を取り出した。夜の闇の中で、種は美しい青白い光を放っている。

「カイト、種が光ってる」

エリナが指摘した。カイトは種の感覚に集中した。

IceSeed.ping() {
  応答: あり
  方向: 南南東
  距離: 推定80km
  共鳴: 微弱
  他種反応: あり
}

微かだが、火の種の反応を感じる。南南東、八十キロ先。

「火の種が反応してる。あと八十キロくらいだ」
「八十キロ……歩けば二日はかかるな」

リアムが計算する。

「何か乗り物がないか探してみよう。砂漠の民がいるかもしれない」

翌朝、カイトたちは南南東へと進み始めた。

日が昇ると同時に、再び灼熱の世界が戻ってくる。水分をこまめに摂りながら、砂丘を越えていく。

昼前、リアムが砂丘の上から何かを発見した。

「あれを見てくれ！」

目を凝らすと、遠くに茶色い影が見える。砂と同じ色に溶け込みそうな、巨大な生物。

「砂船だ！」

砂船——砂漠を行き交う巨大な生物で、背中に荷物や人を乗せることができる。ラクダに似ているが、体長は三メートルを超える。

「砂漠の民かもしれない。話しかけてみよう」

カイトたちは砂船の群れに向かって歩を速めた。

砂船の群れは、十頭以上いた。その周りには、褐色の肌をした人々が移動生活の準備をしている。

カイトが近づくと、一人の男が出迎えた。年齢は四十代後半。長い白い髪を後ろで縛り、サングラスをかけている。

「旅の方か。砂漠で会うのは珍しい」

男は友好的だった。

「はい。南の灼熱の深域に向かいたいんです」

カイトが説明すると、男の表情がわずかに曇った。

「灼熱の深域……危険な場所だぞ。何の用だ？」

カイトは正直に答えた。

「火の種を探しています」

「火の種」という言葉に、周りの砂漠の民たちがざわめいた。男は静かにカイトを見つめ、やがて口を開いた。

「種を求める者か……なら、お前さんたちを乗せていこう。ただし、条件がある」
「条件？」
「ああ。灼熱の深域には、最近奇妙な現象が起きている。砂嵐が収まらないんだ。原因を調べてほしい」

奇妙な砂嵐。カイトは嫌な予感を感じた。

砂船の背中は意外と快適だった。

揺れは少なく、日よけの布が設置されている。砂漠の民の名はハサンと言い、灼熱の砂漠を旅して三十年になるという。

「火の種の話、詳しく聞かせてくれないか」

ハサンがカイトに問いかけた。カイトはこれまでの旅のあらすじを説明した——氷の種を手に入れたこと、残り三つの種が存在すること、そして世界を救うために集める必要があること。

ハサンは静かに聞いていた。カイトの話が終わると、彼はサングラスを外し、深い茶色の瞳をカイトに向けた。

「お前さん、本当のことを言ってるな」
「え？」
「目だ。お前さんの目には、真実を語る光がある。嘘をついていない」

ハサンは満足そうに頷いた。

「火の種は、灼熱の深域の中心にある『炎の聖域』に封印されている。だが、ここ数ヶ月、誰も近づけなくなった」
「なぜですか？」
「砂嵐が絶えないからだ。普通の砂嵐じゃない。魔力を帯びた、凶暴な嵐だ」

カイトは氷の種に触れた。種が小さく震えている。まるで、何かを警告しているかのように。

三日後、カイトたちは砂嵐の境界線に到達した。

目の前には、茶色い壁がそびえ立っている。高さは百メートル以上。風が唸りを上げ、砂が激しく舞い上がっている。

「これが、例の砂嵐か」

リアムが厳しい表情で言った。

Sandstorm.analyze() {
  種類: 魔力嵐
  風速: 80m/s
  視界: 0m
  魔力濃度: 極高
  中心: 南南東 15km
  推定原因: 不明
  危険度: 極高
}

ただの自然現象ではない。誰かが、あるいは何かが、意図的にこの嵐を生み出している。

ハサンが砂船の群れを止めた。

「ここから先は危険だ。砂船では進めない」
「分かりました。ここからは歩きます」

カイトは答えた。ハサンは心配そうに言った。

「無理はするな。命の方が大事だ」
「ありがとうございます。ハサンさんたちも、気をつけて」

別れを告げ、カイトたちは砂嵐の中へと足を踏み入れた。

嵐の中は、地獄だった。

風が肌を切り裂くように吹き付ける。砂が目に入り、息をするのも困難だ。カイトは魔法で防御膜を張ったが、魔力の消耗が激しい。

Shield.maintain() {
  状態: 低下
  残り魔力: 45%
  外部干渉: 高
  推定持続: 20分
}

エリナ、リアム、セシリアも同様に魔法で身を守っているが、限界が近い。

「カイト！ このままじゃ全滅する！」

リアムが叫んだ。風の音が大きすぎて、近くにいても大声を出さなければ聞こえない。

カイトは考えた。この嵐の原因を突き止めれば、止められるかもしれない。

彼は氷の種を取り出し、逆にその力を解放した。氷の冷気が周囲に広がり、一時的に風と砂の猛威を和らげる。

IceSeed.release(power: 30%) {
  効果: 冷気放出
  範囲: 半径10m
  結果: 嵐の一時停止
  持続: 5分
}

五分。その間に、原因を見つけなければならない。

冷気のおかげで視界が開けた。

カイトは周囲を素早く観察した。砂丘の合間に、何かが見える。建物の廃墟だ。そして、その中心に——

「あれだ！」

カイトが指差した先には、紫色の光を放つ球体があった。砂嵐の中心にあるその球体が、魔法の風を生み出している。

「あれを破壊すれば、嵐が止まる！」

カイトたちは球体に向かって走った。氷の種の効果が切れるまで、あと三分。

だが、球体の前に立ちはだかる者がいた。

黒いローブを纏った人物。顔は仮面で覆われている。手には、歪んだ形状の杖が握られている。

「止まれ、継承者」

声は低く、機械的だった。人間ではないかもしれない。

「お前は何者だ」

カイトが問うと、仮面の人物は杖を振った。

「種の回収者。お前が集めた種、全ていただく」

戦闘は一瞬で始まった。

仮面の人物が杖を振ると、砂が槍の形に凝固し、カイトたちに向かって飛んできた。

カイトは氷の種で防御したが、衝撃は防ぎきれない。

Combat.log() {
  敵: 回収者（仮）
  攻撃: 砂槍
  威力: 高
  防御結果: 部分成功
  カイト損傷: 軽傷
}

リアムが剣で砂槍を切り裂き、セシリアが支援魔法を唱えた。エリナは炎魔法で反撃を試みる。

だが、仮面の人物は全ての攻撃を軽々と避けた。その動きは、人間離れしている。

「無駄だ。お前たちごときでは、私に傷一つつけられない」

仮面の人物が嘲笑した。カイトは歯噛みした。

このままでは、氷の種を奪われる。いや、それだけじゃない。火の種も、残りの種も、全て回収されてしまう。

カイトは考えた。どうすれば——

その時、空から声が響いた。

「そこまでだ、回収者」

カイトが見上げると、燃えるような赤い翼を持つ生物が舞い降りてきた。

フェニックスだ。

だが、以前会ったフェニックスとは違う。体色が少し暗く、瞳には知性の光がある。

「サラマンダー……！」

仮面の人物が呟いた。赤い翼の生物は、カイトたちの前に降り立った。

「フローズナから話は聞いている。灼熱の深域を荒らす者を討てとな」

サラマンダーは仮面の人物を睨みつけた。

「継承者よ、下がっていろ。こやつは私が相手をする」

カイトは一瞬迷ったが、頷いた。今の自分では、あの仮面の人物と渡り合えない。

サラマンダーと回収者の戦いが始まった。炎と砂の衝突が、砂漠を震わせる。

カイトはその隙に、球体に近づいた。これを破壊すれば、嵐が止まるはずだ。

カイトは球体に手を伸ばした。

球体は紫色に輝き、不気味な魔力を放っている。触れればどうなるか分からない。

だが、やるしかない。

カイトは氷の種を球体に押し当てた。氷と紫の光が激しくぶつかり合う。

IceSeed.interact(target: "StormCore") {
  反応: 激突
  結果: 中和開始
  進捗: 30%
  推定完了: 2分
}

二分。長いようで、短い。その間、サラマンダーが回収者を抑えてくれている。

カイトは全神経を集中した。球体の魔力に対抗する。氷の種の力を、最大限に引き出す。

三十秒。進捗五十パーセント。

一分。進捗七十パーセント。

一分三十秒。進捗九十パーセント。

そして——

StormCore.destroy() {
  結果: 成功
  嵐: 停止
  魔力残留: なし
}

砂嵐が消えた。

視界が開け、静寂が戻ってきた。仮面の人物——回収者は、砂嵐が消えたことを悟ると、舌打ちをして姿を消した。

「逃がしたか……」

サラマンダーが悔しそうに呟いた。だが、すぐにカイトの方を向いた。

「よくやった、継承者。砂嵐を止めるとはな」
「サラマンダーさんが助けてくれなかったら、やられてました」

カイトは正直に言った。サラマンダーは満足そうに頷いた。

「フェニックスから聞いた通りだ。お前は、協調を理解している」

エリナ、リアム、セシリアがカイトに駆け寄ってきた。

「カイト、怪我はない？」
「大丈夫。軽傷だよ」

カイトは仲間たちに微笑んだ。

砂嵐が消えたことで、灼熱の深域の姿が明らかになった。

その中心に、巨大な遺跡が鎮座している。炎の形をした塔。それが、火の種が眠る場所——炎の聖域だ。

「あれが、炎の聖域か」

カイトが呟いた。サラマンダーが言った。

「ああ。だが、中に入るには試練を乗り越えなければならない」
「試練？」
「火の種の試練だ。氷の種と同じように、種は継承者を選ぶ。お前がふさわしいかどうか、試されるだろう」

カイトは炎の聖域を見つめた。灼熱の熱気が、肌を焼く。

火の種を手に入れるには、この熱の中で何かを成し遂げなければならない。

「行こう。僕たちの旅は、まだ終わらない」

カイトの言葉に、三人が頷いた。

灼熱の砂漠を越え、回収者という新たな敵を知った日。

炎の聖域を前にして、カイトは覚悟を決めた。

あと二つの種。そして、回収者の正体と目的。

全ての答えは、旅の先にある。

物語は、次の章へと続く。
