第531章：灼熱の危機

エンドレス崖を後にしたカイトは、急ぎ南へと向かった。

火の聖域は、東大陸の中央部に位置する。

灼熱の砂漠の奥、火山地帯の最深部にあった。

「グラシエルが先に到着している可能性がある」

カイトは、馬を走らせた。

ノースエッジ村を出て、街道を南下する。

夜が更けても、休む余裕はなかった。

三日後。

カイトは、東大陸の中央都市、セントラルシティに到着した。

かつて、カイトが最初に訪れた街である。

冒険者ギルドの支部があり、多くの冒険者が集まる場所だ。

「久しぶりだな、この街」

カイトは、懐かしさを感じながら街に入った。

街の様子は、以前と変わらないように見えた。

だが、空気の中に、微かな違和感があった。

カイトは、冒険者ギルドの支部へ向かった。

建物の中に入ると、受付嬢のマヤ——同名の別人だ——が、笑顔で迎えた。

「いらっしゃいませ！　ご依頼ですか、それとも情報収集ですか？」

カイトは、カウンターに近づいた。

「火の聖域について、情報を集めたい」

マヤの表情が、少しだけ曇った。

「火の聖域……最近、妙な噂があります」
「妙な噂？」
「はい。砂漠の温度が、異常に上昇していると」

カイトの眉が、潜められた。

「異常に上昇？」
「普段でも暑い砂漠ですが、ここ数日、商人たちが『焼け死ぬかと思った』と口を揃えています」

マヤが、資料を取り出した。

「気温の測定記録を見ると、平均より二十度も高いそうです」
「二十度も……」

カイトの中の原初の闇が、語りかけた。

「火の心臓の暴走か、あるいは外部からの干渉か」
「グラシエルの仕業かも」
「可能性は高い。急ぐべきだ」

カイトは、マヤに礼を言った。

「ありがとう。急いで向かうよ」
「あ、お待ちを！　テオドール支部長が、あなたに会いたがっていました」

カイトは、足を止めた。

「テオドールさん？」
「はい。カイトさんが来たら、必ず一声かけるようにと」

カイトは、支部長室へ向かった。

支部長室のドアをノックする。

「どうぞ」

中から、落ち着いた声が聞こえた。

カイトは、部屋に入った。

テオドール支部長は、執務机の前に座っていた。

五十代半ばの男性で、白髪混じりの髪と鋭い目つきが特徴的だ。

「カイト君か。久しぶりだな」
「テオドールさん、お久しぶりです」

カイトが、頭を下げた。

テオドールが、椅子から立ち上がった。

「火の聖域へ向かうつもりか？」
「はい。急ぎの用事があって」
「十三使徒のグラシエルに関する情報がある」

テオドールの目が、真剣な光を帯びた。

「三日前、砂漠の入り口に、青白い肌の女が現れたという目撃情報がある」
「青白い肌……グラシエルですね」
「氷の魔力を纏っていたらしい。砂漠の熱気を一瞬で凍らせたと」

カイトの顔が、強張った。

「やはり、グラシエルが向かっている」
「カイト君、君は管理者だと言っていたな」
「はい」
「ならば、世界の危機に関わることなのだろう」

テオドールが、引き出しから一つの封筒を取り出した。

「これを持っていけ」
「これは？」
「砂漠のキャラバン、炎の牙率いる商隊への紹介状だ」
「炎の牙？」
「砂漠を縄張りとする、屈強な商人たちだ。彼らの協力を得れば、火の聖域までの道が開ける」

カイトは、封筒を受け取った。

「ありがとうございます、テオドールさん」
「礼には及ばん。世界を救うのが君の役目なら、それを手助けするのが我々の役目だ」

テオドールが、カイトの肩を叩いた。

「無事を祈る」
「はい」

カイトは、支部長室を出た。

翌朝。

カイトは、セントラルシティを出発した。

南へ向かう街道を進み、昼過ぎには砂漠の入り口に到着した。

見渡す限りの砂丘。

太陽が、容赦なく照りつける。

「暑い……」

カイトは、額の汗を拭った。

砂漠特有の乾燥した風が、肌を刺す。

砂漠の入り口には、小さなオアシスの村があった。

村の名は、サンドゲート。

砂漠に入る前の最後の補給地点だ。

カイトは、村の広場で商隊を探した。

「炎の牙……どこにいるかな」

広場には、何組もの商人が荷物を整理していた。

カイトは、最も大きな商隊に近づいた。

商隊のリーダーらしき男が、カイトに気づいた。

背が高く、日焼けした肌に赤い髪。

目つきは鋭いが、悪意は感じられない。

「客か？　砂漠を越えたいのか」
「はい。テオドール支部長の紹介で来ました」

カイトは、封筒を差し出した。

男が、封筒を開けて中身を確認した。

「ほお、支部長の直筆か。あの爺さん、まだ元気か」

男が、ニヤリと笑った。

男は、自己紹介した。

「俺は、レイモンド。炎の牙の隊長だ」
「カイトです。よろしくお願いします」
「カイト……どこかで聞いた名前だな」

レイモンドが、カイトをじっと見た。

「ああ、思い出した。管理者のカイトか」
「知ってるんですか？」
「噂は届いている。十三使徒を退けた異世界の勇者だろ」

レイモンドが、豪快に笑った。

「良い。俺たちは、強い奴が好きだ」

カイトは、商隊に加わった。

キャラバンは、三十頭のラクダと、五十人ほどの商人と護衛で構成されていた。

「砂漠の移動は、馬よりラクダだ」

レイモンドが、カイトにラクダを貸してくれた。

「慣れないと扱いにくいが、こいつは賢い」

カイトは、ラクダの背に跨った。

少し揺れたが、すぐにバランスを取れた。

商隊が出発した。

砂丘を越え、灼熱の大地を進む。

太陽は真上にあり、影は足元に凝縮されていた。

「砂漠の熱が、以前より強い気がする」

カイトが、レイモンドに言った。

「ああ。三日ほど前から、急に暑くなった」
「火山地帯の活動が活発化しているのか？」
「いや、これまでと違う。まるで、巨大な熱源が目覚めたみたいだ」

カイトの中の原初の闇が、警告した。

「火の心臓が、共鳴している」
「共鳴？」
「他の強力な火の魔力と反応しているのだ」
「グラシエルが、火の心臓に干渉している？」
「あるいは、火の賢者イグニスが、グラシエルと戦っているか」

カイトは、焦りを感じた。

「急がないと」

商隊は、日が沈むまで進んだ。

夜になると、砂漠の気温は急激に下がる。

商隊は、オアシスで野営した。

焚き火を囲み、食事をとる。

「カイト、火の聖域へ行く目的は何だ？」

レイモンドが、尋ねた。

「火の心臓を手に入れる必要がある」
「火の心臓……伝説のアーティファクトか」

レイモンドが、興味深そうに頷いた。

「火山地帯の最深部にあると言われているが、誰も戻ってこない」
「危険なんですね」
「ああ。だが、お前なら大丈夫かもしれない」

レイモンドが、焚き火に薪をくべた。

「カイト、言っておくことがある」
「何ですか？」
「火の聖域の守護者、フェニックスについてだ」

カイトの記憶に、その名前が引っかかった。

「フェニックス……炎の守護獣ですよね」
「正確には、炎の守護獣のリーダー格だ」

レイモンドが、真面目な表情になった。

「フェニックスは、数千年前から火の心臓を守り続けている」
「戦わなければならない？」
「いや、フェニックスは賢い。話せばわかるはずだ」

翌朝。

商隊は、火山地帯の入り口に到着した。

黒い岩肌と、立ち上る噴煙。

地面からは、熱気が立ち上る。

「ここから先は、俺たちは行けない」

レイモンドが、カイトを見た。

「カイト、火の聖域は、火山の火口にある」
「火口の中に？」
「ああ。 magma の海を渡らなきゃならない」
「普通の人間には、無理ですね」
「だから、伝説なんだ」

レイモンドが、カイトの肩を叩いた。

「無事を祈る。戻ってきたら、酒を奢ってくれ」

カイトは、商隊に別れを告げた。

火山地帯の入り口に立ち、奥を見据える。

噴煙の向こうに、赤い光が見えた。

「あそこに、火の心臓がある」

カイトは、歩き出した。

地面は熱く、靴底を焼くようだ。

だが、カイトは魔法で保護した。

// 火山環境対策・耐熱バリア
func volcanicProtection() {
    let barrier = BarrierMagic.self
    barrier.createThermalShield(
        caster: self,
        temperature: .extreme,
        properties: [
            .heatResistance(.maximum),
            .airPurification(.active),
            .staminaConservation(.enabled)
        ]
    )
}

カイトは、火山の斜面を登った。

硫黄の臭いが鼻をつく。

周囲の岩は赤熱し、触れれば火傷しそうだ。

だが、バリアのおかげで、影響は最小限だった。

「もう少しで火口だ」

カイトが、呟いた。

その時、前方から爆発音が聞こえた。

カイトは、足を止めた。

火口の縁から、炎の柱が立ち上った。

そして、二つの影が、空中で激突していた。

一つは、炎の翼を持つ巨鳥——フェニックス。

もう一つは、青白い肌の氷の女——グラシエル。

「始まっている！」

カイトは、駆け出した。

火口の縁に到着したカイトは、光景に息を呑んだ。

火口の中は、マグマの海だった。

その上で、フェニックスとグラシエルが空中戦を繰り広げている。

炎と氷の衝突が、周囲の空気を震わせた。

「氷の災厄よ、引き下がれ！」

フェニックスが、炎のブレスを放った。

「炎の守護獣、貴様の炎など、私の氷で凍らせる！」

グラシエルが、氷の盾を展開した。

二つの力が、激しくぶつかり合う。

衝撃波が、カイトの立つ縁まで届いた。

カイトは、バリアを強化した。

「どっちも、強い……」

原初の闇が、語りかけた。

「グラシエルは、フィンブルヴェトと同じ氷属性だ」
「同じ十三使徒ってことか」
「だが、フィンブルヴェトは封印されて力が衰えていた。グラシエルは、全盛期だ」
「苦戦しそうだね」

カイトは、介入のタイミングを計った。

フェニックスとグラシエルの戦いは、拮抗していた。

だが、徐々にグラシエルが押し始めていた。

氷の魔力が、フェニックスの炎を侵食していく。

「まずい、フェニックスが！」

カイトは、飛翔魔法を発動した。

// 飛翔・火口内部へ突入
func diveIntoCrater() {
    let flight = FlightMagic.self
    flight.activate(
        caster: self,
        destination: .craterCenter,
        speed: .maximum,
        protection: .volcanicBarrier
    )
}

カイトが、火口の中へ飛び込んだ。

マグマの熱気が、肌を焼く。

だが、バリアが守ってくれる。

カイトは、戦場の中央に進出した。

「ストップ！」

カイトの声に、二人が反応した。

フェニックスとグラシエルが、カイトを見た。

グラシエルの瞳が、カイトを射抜いた。

「あなたが、管理者のカイトね」
「知ってるのか？」
「ルシフェインから報告を受けている。光の聖域で彼を退けたそうね」

グラシエルが、冷ややかに笑った。

「光の第七席を倒したとはいえ、私は第九席。序列は下だが、実力は上よ」
「試してみるか？」

カイトが、挑発した。

グラシエルの表情が、怪訝になった。

「余裕ね。だが、油断している」

グラシエルが、手をかざした。

「氷結・絶対零度！」

青白い光が、カイトを包んだ。

カイトの周囲の空気が、一瞬で凍りついた。

カイトは、即座に反応した。

// 絶対零度対策・反魔法シールド
func counterAbsoluteZero() {
    let antiMagic = AntiMagicShield.self
    antiMagic.dispel(
        caster: self,
        targetSpell: "AbsoluteZero",
        properties: [
            .dispelType(.counterspell),
            .elementalConversion(.fire)
        ]
    )
}

カイトの体から、炎が噴き出した。

氷が溶け、カイトは解放された。

グラシエルが、目を見開いた。

「即座に対抗魔法を！？　ただの管理者ではないわね」
「プログラマーだからね。コードを書くのは得意なんだ」

カイトが、ニヤリと笑った。

「フェニックス、协力しよう！」

フェニックスが、頷いた。

「わかった、管理者よ！」

カイトとフェニックスが、並んで立った。

対するは、氷の災厄グラシエル。

マグマの海の上で、決戦の火蓋が切られた。

「行くぞ、グラシエル！」

カイトの両手に、光と闇の魔力が集まった。

最強の魔法使いと、炎の守護獣。

二人の协力で、氷の使徒に挑む。

戦いの行方は、まだわからない。

だが、カイトの瞳には、揺るぎない決意があった。

「フィンを救うために、ここで負けるわけにはいかない！」

カイトの叫びと共に、灼熱の決戦が始まった。
