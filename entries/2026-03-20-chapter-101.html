第101章：嵐の前触れ

決戦当日の朝、源泉は静かに目覚めた。

薄い霧が、大地を覆っていた。源泉から立ち上る光が、霧の中で揺らめいている。鳥のさえずりが、遠くから聞こえた。平和な光景。でも、この静けさは、嵐の前触れだった。

カイトは、夜明けとともに目を覚ました。焚き火のそばで眠っていたエリナが、まだ静かな寝息を立てている。

カイトは、音を立てないように立ち上がった。源泉のそばまで歩き、ハーモニーシステムで状況を確認した。


// 朝の状況確認
const morningCheck = HarmonySystem.scan({
    range: '全方位10km',
    focus: '敵性反応',
    sensitivity: 'maximum'
});

console.log(morningCheck);

敵の姿はまだない。でも、カイトは気を抜かなかった。夜になれば、敵が現れる。それは確実だった。

「……予想通りだ。敵は夜に来る」

カイトは、呟いた。源泉の光を眺めながら、今日の戦いについて考えた。

一時間後、全員が目を覚ました。

ゼノンが、朝食を用意してくれた。乾燥肉と堅パン、泉の水。質素な食事だが、今日一日の糧になる。

エリナが、カイトの隣に座った。心配そうな表情。

「……眠れた？」

カイトは頷いた。

「ええ、少しは。エリナは？」

エリナは、少しだけ笑った。

「正直、あまり眠れなかった。でも、大丈夫。カイトの隣にいるから」

カイトは、エリナの手を握った。温かい手。この温もりを守りたい。そう強く思った。

「……今日は、勝ちます。絶対に」

エリナは、強く頷いた。

「うん。一緒に戦おう」

朝食の後、ゼノンが全員を集めた。

源泉のそばの広場で、最終作戦会議を行った。ゼノンが、地図を広げる。

「今朝、改めて敵の動きを確認した。ザインの偵察部隊とアークライトの攻撃部隊、両方がこちらに向かっている」

リアムが、質問した。

「到着時間は？」

カイトが、ハーモニーシステムの予測を共有した。


// 敵の到着予測・更新

カイトは、結果を説明した。

「ザインの偵察部隊が、今夜八時頃に到着。人数は三人。少数精鋭の暗殺者だと思います。アークライトの攻撃部隊は、一時間から二時間遅れて到着。三百人の兵士に、指揮官が一人」

セシリアが、冷静に分析した。

「偵察部隊を先に片付ければ、情報がアークライトに伝わらない。有利になります」

アルフレッドが、頷いた。

「ただし、偵察部隊を逃がせば、アークライトに情報が漏れる。確実に始末しなければならない」

ゼノンが、静かに言った。

「偵察部隊は、私が相手をする。カイトは、アークライトの本隊に備えてくれ」

カイトは、ゼノンの顔を見た。蒼白な肌。弱った体。でも、目は力強かった。

「……ゼノンさん、本当に大丈夫なんですか？」

ゼノンは、静かに微笑んだ。

「ああ。十年間、この時を待っていた。私の最後の仕事だ。全力を尽くす」

カイトは、覚悟を決めた。ゼノンの覚悟に、応えなければならない。

「……わかりました。偵察部隊は、ゼノンさんに任せます。僕たちは、アークライトの本隊に備えます」

午前中、カイトたちは最後の準備を行った。

魔法陣の再確認。罠の動作テスト。武器の手入れ。各自が、自分の役割を最終調整した。

カイトは、ハーモニーシステムで魔法陣の状態を確認した。


// 魔法陣・最終点検
const arrayCheck = HarmonySystem.inspect({

魔法陣は、九十六パーセントの完全性を維持していた。問題ない。

エリナが、カイトに声をかけた。

「カイト、魔法の準備はできてる？」

カイトは頷いた。

「ええ。攻撃、防御、回復、全部準備できてます。エリナは？」

エリナは、杖を握りしめた。

「うん。雷撃魔法と治癒魔法、両方準備できてる。カイトのサポートをする」

カイトは、エリナの肩に手を置いた。

「……無理しないで。僕が守るから」

エリナは、少し膨れた顔をした。

「もう、カイトも無理しないでよ。私は、カイトを守りたいの」

カイトは、苦笑した。

「……じゃあ、お互い無理しないで。二人で、生きて帰ろう」

エリナは、満足そうに頷いた。

「うん。約束」

正午、カイトはカグヤの水晶を取り出した。

精神世界で、カグヤと会話する。今日の戦いに備えて、最後の準備を行う。

白い空間。カグヤが、カイトの前に現れた。

「カイト……ついに今日なのね」

カイトは頷いた。

「はい。夜になれば、敵が来ます。ザインとアークライト、両方」

カグヤは、真剣な表情で言った。

「私は、全力でサポートする。ハーモニーシステムの限界を超えて、あなたを守る」

カイトは、カグヤに感謝の言葉を伝えた。

「ありがとうございます。カグヤがいてくれて、本当に助かってます」

カグヤは、少し寂しそうに微笑んだ。

「……カイト。もし、私が限界を超えて力を使ったら……私は、消えるかもしれない」

カイトは、驚いた。

「えっ……？」

カグヤは、静かに言った。

「私は、管理AIの一部。カイトと契約した時、制限がかかった。もし限界を超えて力を使えば、プログラムとしての存続が危うくなる」

カイトは、カグヤの言葉を聞いて、胸が締め付けられた。

「……そんなこと、言わないでください。僕は、カグヤを失いたくない」

カグヤは、優しく微笑んだ。

「……ありがとう、カイト。でも、もしもの時は、躊躇わないで。あなたが生きることが、一番大事だから」

カイトは、カグヤの手を握った。白い手。儚げな存在。でも、確かな温もりがあった。

「……カグヤを守ります。絶対に、カグヤを消させない」

カグヤは、嬉しそうに笑った。

「……うん。信じてる」

午後、カイトたちは休息を取った。

夜の戦いに備えて、体力を温存する。各自が、思い思いの場所で過ごした。

エリナは、源泉のそばで瞑想していた。魔力を高めている。セシリアは、魔導書を読んでいた。最後の知識の確認。リアムとアルフレッドは、剣の手入れをしていた。

ゼノンは、源泉の前に座っていた。目を閉じて、何かを唱えている。

カイトは、ゼノンに近づいた。

「ゼノンさん……」

ゼノンは、目を開けた。

「……カイトか。何か用か？」

カイトは、少し躊躇してから言った。

「……十年間、一人で戦ってきて、辛くなかったですか？」

ゼノンは、少し驚いた顔をした。そして、静かに語り始めた。

「……辛かった。仲間を失い、希望を失い、何度も投げ出そうと思った。でも、諦めなかった。カグヤと約束したから」

カイトは、ゼノンの言葉を聞いて、胸が熱くなった。

「……カグヤと、約束？」

ゼノンは、遠くを見つめながら言った。

「ああ。十年前、カグヤは私に言った。『いつか、あなたを助けてくれる人が現れる』と。私は、その言葉を信じて、十年間待ち続けた」

カイトは、言葉を失った。カグヤは、十年前からカイトの存在を予見していたのか。

「……僕が、その人だなんて」

ゼノンは、カイトの肩に手を置いた。

「カイト、自分を過小評価するな。お前は、私が十年間待ち続けた希望だ。お前が来たから、私はこうして最後の戦いに臨める」

カイトは、ゼノンの手を握り返した。

「……ゼノンさん。絶対に、生き残りましょう。この戦争を終わらせて、一緒に」

ゼノンは、満足そうに頷いた。

「ああ。約束だ」

夕暮れ。空が、赤く染まった。

カイトたちは、戦闘配置についた。源泉を中心に、円陣を組む。魔法陣が、微かに光り始めた。

カイトは、ハーモニーシステムで最終確認を行った。

エリナが、カイトの隣に立った。

「……カイト。いよいだね」

カイトは頷いた。

「ええ。あと二時間で、敵が来る」

エリナは、カイトの手を握った。

「……怖いけど、大丈夫。カイトと一緒だから」

カイトは、エリナの手を強く握り返した。

「僕も、エリナがいてくれるから、怖くない」

夜七時五十分。カイトのハーモニーシステムが、敵の接近を検知した。

カイトは、全員に警告を発した。

「来ます！ ザインの偵察部隊、あと十分で到着！」

ゼノンが、立ち上がった。

「私がやる。カイトは、アークライトに備えてくれ」

カイトは、ゼノンを見た。

「ゼノンさん……」

ゼノンは、静かに微笑んだ。

「心配するな。十年間、こういう相手と戦い続けてきたんだ。私の仕事だ」

カイトは、頷いた。

「……お願いします。無事で」

ゼノンは、一人で北西の方向へ歩き出した。背中が、大きく見えた。

夜八時。偵察部隊が、源泉の外周に到達した。

ヴェルナー、クララ、ダリウス。黄昏の牙の精鋭たち。闇に紛れて、静かに近づく。

ヴェルナーが、小声で言った。

「警報装置あり。魔法陣の反応あり。罠だらけだ」

クララが、冷静に分析した。

「三層構造。幻惑、拘束、衝撃。突破には時間がかかる」

ダリウスが、提案した。

「なら、正面から行かない。迂回して、上空から侵入する」

三人は、頷き合った。そして、飛行魔法で空へ舞い上がった。

だが、その時だった。

ゼノンが、彼らの前に現れた。

「……ここから先は、通さない」

ヴェルナーが、冷ややかに笑った。

「ゼノン……十年ぶりだな。生きていたか」

ゼノンは、静かに言った。

「ああ。お前たちを迎えるために、十年間待っていた」

クララが、皮肉っぽく言った。

「ひとりで、三人を相手にするつもり？ 無謀ね」

ゼノンは、微笑んだ。

「無謀じゃない。覚悟だ」

ゼノンは、源泉の力を解放した。凄まじいエネルギーが、彼の体を包んだ。

「来い。黄昏の牙。私が、お前たちを止める」

戦いの火蓋が、切られた。
