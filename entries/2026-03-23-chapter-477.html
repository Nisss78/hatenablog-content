第477章：鋼鉄の守護者

朝霧が立ち込める森の中、カイトたちは出発の準備を整えていた。

昨夜の焚き火の跡は、丁寧に消されている。自然に残す痕跡を最小限にする——それは冒険者としての基本だ。

「天気は良さそうだね」

エリナが、空を見上げながら言った。

確かに、雲一つない晴れ渡った空。旅日和と言える。

「でも、油断は禁物だ。古の遺跡は、何が起こるかわからない」

リアムが、警告した。

「機械仕掛けの守護者以外にも、罠や迷宮が待っているかもしれない」
「リアムの言う通り。慎重に行こう」

カイトが、頷いた。

カイトは地図を確認した。原初の座への最短ルート——古の遺跡を通る道。そこには、何千年も前に造られた機械文明の遺産が眠っている。

「出発しよう」

カイトたちは、北東へ向かって歩き出した。

◆ ◆ ◆

三時間ほど歩いたとき、景色が変わった。

鬱蒼とした森が開け、荒涼とした岩場が現れた。巨大な岩が不規則に突き出し、奇妙な形の影を落としている。

「ここが……古の遺跡への入り口？」

セシリアが、不安そうに言った。

岩場の奥に、巨大な石造りの門が見えた。門には、複雑な幾何学模様が刻まれている。そして、門の両脇には——動かない二体の鋼鉄の像が立っていた。

「あれが機械仕掛けの守護者か」

リアムが、警戒した表情で見つめた。

「今は停止しているようだが……近づけば動き出すだろう」
「大きいね……三メートルはあるんじゃない？」

エリナが、息を呑んだ。

確かに巨大だ。人型をしているが、胴体は無骨な金属板で覆われ、関節には歯車が見える。腕には巨大な刃が備え付けられていた。

「僕が先に行く。皆は下がっていて」

カイトが、前に出た。

虚無の欠片を手に入れた今、以前より力は増しているはず。カイトはそう信じて、門に近づいた。

距離、十メートル。

その時——守護者の目が光った。

ガガガガッ——重い音を立てて、守護者が動き出した。金属のきしむ音が響く。

「侵入者を検知。排除シマス」

守護者から、機械的な声が響いた。

「来る！」

カイトは、魔法を展開した。


class GuardianDefense {
  private target: Guardian;
  private barrier: MagicBarrier;
  
  public analyze(): Weakness {
    return this.target.scanForVulnerability();
  }
  
  public engage(): void {
    const weakness = this.analyze();
    this.barrier.activate();
    this.target.disable(weakness);
  }
}

守護者が、巨大な刃を振り上げた。速い。見た目からは想像できないスピードだ。

「シールド！」

カイトは、光の盾を展開した。

ドーン——刃が盾に当たり、衝撃が走る。カイトは、一メートルほど後ろに押された。

「くっ——重い」

守護者は、容赦なく追撃してくる。右腕の刃、左腕の刃、交互に襲いかかる。

カイトは、盾を維持しながら後退した。

「カイト、大丈夫？」

エリナが、心配そうに叫んだ。

「問題ない。パターンを見ている」

カイトは、冷静だった。

守護者の攻撃は、確かに速く重い。でも、パターンがある。右、左、右、左——単調なリズム。

「わかった。コアは胸部の中心だ」

カイトは、守護者の動きの中に、規則性を見つけた。

攻撃のたびに、胸部中央のプレートがわずかに開く。そこに、青く光る球体——動力源が見える。


class CoreTargeting {
  public exploit(guardian: Guardian): void {
    const pattern = guardian.attackPattern;
    const coreWindow = pattern.findOpening();
    
    if (coreWindow) {
      this.pierceCore(guardian.core);
    }
  }
  
  private pierceCore(core: Core): void {
    const spell = new PiercingLight();
    spell.execute(core);
  }
}

カイトは、次の攻撃を待った。

右——左——その瞬間。

「今だ！」

カイトは、盾を解除し、代わりに貫通魔法を放った。

細い光の槍が、守護者の胸部を突き抜ける。青い球体が砕け、守護者の動きが止まった。

ガガガッ——重い音を立てて、守護者が崩れ落ちた。

「やった！」

エリナが、歓声を上げた。

「まだだ！ もう一体が動いた！」

リアムが、叫んだ。

見ると、もう一体の守護者が動き出していた。そして——カイトの背後から、新たな気配。

「何！？」

カイトが、振り返った。

そこには、三体目の守護者が立っていた。岩陰に隠れていたのだ。しかも、この守護者は他の二体と違う。より洗練された形。そして——

「上位個体ヲ検知。カイト、ト認識。特別排除プロトコルヲ開始」

上位個体の守護者は、カイトの名前を言った。

「どうして僕の名前を？」

カイトは、戦慄した。

「管理者ノ末裔、カイト。貴様ノ存在ハ、予測サレテいた」

上位個体は、胸のプレートを開いた。そこには、青い球体ではなく——赤い球体が埋まっていた。

「予測されていた……？ 誰に？」
「最初ノ管理者、アレクシス・ヴェイルニヨル」

その名前には、覚えがあった。

アレクシス・ヴェイル——最初の管理者。この世界の魔法システムを構築した人物。

「アレクシス・ヴェイルが、何千年も前に僕を予測していた？」
「管理者ノ血脈ハ、予測可能デアル。貴様ガ現レル時期、場所、力——全テ計算サレテいた」

上位個体は、ゆっくりと近づいてきた。

「ソシテ、貴様ガココニ来ルコトモ、予測サレテいた。我ガ任務ハ、貴様ヲ排除スルコト」
「カイト、下がって！ 僕たちがやる！」

エリナが、魔法を構えた。

「いや、待って」

カイトは、エリナを制止した。

「これは僕の問題だ。アレクシス・ヴェイルが仕掛けた試練——僕が受けて立つ」

カイトは、虚無の欠片に触れた。

手の中で、黒い結晶が共鳴する。以前より、力が安定している。自分の中の闘いを経て、制御できるようになった。

「上等だ。アレクシス・ヴェイルの予測を、裏切ってやる」

◆ ◆ ◆

上位個体との戦いは、激しかった。

他の守護者とは比べものにならないスピードとパワー。そして、魔法を使ってくる。

「ファイア・ランス！」

上位個体が、腕から炎の槍を放った。

カイトは、横に回避した。槍は地面に命中し、爆発を起こす。

「魔法が使える……ただの機械じゃない」

カイトは、冷静に分析した。


class AdaptiveCombat {
  private enemy: AdvancedGuardian;
  private patterns: AttackPattern[];
  
  public counter(): void {
    const magicType = this.enemy.detectMagicType();
    
    switch (magicType) {
      case 'fire':
        this.deployWaterBarrier();
        break;
      case 'ice':
        this.deployFireBarrier();
        break;
      default:
        this.deployNeutralBarrier();
    }
  }
  
  public findCoreVulnerability(): Point {
    // 上位個体のコアは可動式
    // パターンを見極める必要がある
    return this.enemy.traceCoreMovement();
  }
}

カイトは、上位個体の動きを観察し続けた。

魔法の属性は、火だけではない。氷、風、雷——次々と変化する。多彩な攻撃に対処するには、対応する防御魔法が必要だ。

「ウォーター・シールド！」

炎に対しては水。氷に対しては火。カイトは、瞬時に魔法を切り替えた。

でも、守備に回るだけでは勝てない。攻撃の機会を作る必要がある。

「リリアナさん……時空魔法の教えを思い出すよ」

カイトは、時間を少しだけ伸ばす魔法を唱えた。

世界が、スローモーションのように見える。上位個体の動きが、わずかに遅く感じられた。

その隙に——カイトは見た。

上位個体の背中に、小さな装置が埋め込まれている。それが、赤い球体——コアへのエネルギーを供給している。

「あそこが、弱点だ」

カイトは、時空魔法を解いた。

時間が元に戻る。上位個体が、再び速い動きで襲いかかってくる。

でも、もうわかっている。勝ち方は。

「エリナ、リアム、セシリア——援護を！」
「了解！」

三人が、同時に魔法を放った。

「ウィンド・カッター！」
「サンダー・ボルト！」
「アイシクル・レイン！」

三つの魔法が、上位個体の正面から襲いかかる。

上位個体は、防御魔法を展開した。でも、その間——カイトは背後に回っていた。

「終わりだ」

カイトは、背中の装置に向けて、全魔力を込めた一撃を放った。

「虚空斬——！」

黒い光の刃が、装置を切り裂く。エネルギーの供給が断たれ、上位個体の動きが鈍った。

そして——赤い球体が露出した。

カイトは、球体を破壊した。

爆発とともに、上位個体が崩れ落ちた。

◆ ◆ ◆

戦いが終わった後、カイトは上位個体の残骸を調べた。

破壊された赤い球体の中に、金属製のプレートがあった。そこには、文字が刻まれている。

「これは……？」

カイトは、プレートを拾い上げた。


管理者ノ末裔へ

予測通リノ強サヲ持ッテイタ。
原初の座へ向カウタメニ、
コノ試練ハ不可欠ダッタ。

決シテ、敵意ハナイ。
私ノ意志ヲ継グ者トシテ、
貴様ヲ試サザルヲ得ナカッタ。

アレクシス・ヴェイル

カイトは、プレートを読んで、複雑な気持ちになった。

「アレクシス・ヴェイルは……僕を試していたんだ」
「何千年も前に、本当に予測していたなんて、すごいですね」

セシリアが、感心したように言った。

「それとも、これも世界の因果が導いたのかもしれない」

リアムが、冷静に分析した。

「どっちにしても——カイトは試練を乗り越えた。それが大事なことだよ」

エリナが、カイトの肩を叩いた。

「エリナの言う通りだ」

カイトは、プレートをしまった。

「行こう。原初の座はもう近い」

巨大な石造りの門が、静かに開いた。

その奥に、続く道が見えた。長い旅の、いよいよ終わりが近づいている。

カイトたちは、門をくぐった。

古の遺跡の深部——そこから、原初の座への道が始まる。
