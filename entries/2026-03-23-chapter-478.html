第478章：遺跡の深層

石造りの門をくぐった先、そこには長い回廊が続いていた。

壁には、淡い光を放つ鉱石が埋め込まれている。何千年もの間、この場所を照らし続けてきたのだろう。

「すごい……人工的な明かりなのに、まだ光っているなんて」

セシリアが、壁に触れながら感嘆した。

「古代の技術だね。今の魔法文明とは、また違う発展を遂げていたみたい」

リアムが、冷静に分析した。

カイトは、回廊の奥を見つめた。見通しは悪い。曲がり角がいくつもあり、どこまで続いているのかわからない。

「気をつけて。罠があるかもしれない」

カイトが、警告した。

アレクシス・ヴェイルが残した試練——守護者はその一部に過ぎないかもしれない。

カイトたちは、慎重に歩き出した。

◆ ◆ ◆

回廊を十分ほど歩いたとき、開けた空間に出た。

巨大な円形の広間。天井は高く、直径は優に五十メートルはある。中央には、奇妙な装置が鎮座していた。

「あれは……？」

エリナが、装置を指差した。

巨大な水晶球が、台座の上に浮いている。周囲には、複雑な文字が刻まれた金属板が配置されている。

「どう見ても、何かの装置だな」

リアムが、警戒した表情で近づいた。

「でも、敵意は感じない。起動もしていないみたい」

カイトは、装置の前に立った。

水晶球の中に、何かが見える。微かな光の粒——まるで、星々のようだ。

「これ、星図か？」

カイトが、呟いた。

その時——水晶球が光り輝いた。

「アクセスヲ検知。管理者ノ末裔、カイト。認識シマス」

機械的な声が、響いた。

「また僕のことを知っているのか」

カイトは、少し呆れた。

「アレクシス・ヴェイルノ予測ニ基ヅキ、貴様ノ来訪ヲ待ッテいた。此処ハ、情報保管庫デアル」

情報保管庫——その言葉に、カイトは反応した。

「どんな情報が保管されている？」
「管理者ノ歴史、魔法システムノ設計図、世界ノ構造、因果ノ理論——多岐ニ渡ル」

カイトの目が、輝いた。

「原初の座についての情報は？」
「原初の座ニ関スル情報ハ、一部ノミ保管サレテいる。完全ナ情報ハ、現地ニ到着シタ後ニ提供サレル」
「わかった。では、ここに行く方法を教えてくれ」
「了解。最短ルートヲ表示シマス」

水晶球の中の光の粒が動き、一本の線が引かれた。遺跡の深部を通り抜け、北東へと続く道だ。

「注意事項ガ一件。遺跡ノ深層ニ、破損シタ守護者ガ存在スル。暴走状態ニアリ、極メテ危険」
「暴走した守護者……」

カイトは、表情を引き締めた。

「位置は？」
「此ノ先、約五百メートル。大広間ニテ待機中」
「了解。避けることはできる？」
「回避ルートハ存在シナイ。大広間ヲ通ラナケレバ、先ヘ進メナイ」

カイトは、仲間たちを見た。

「また戦いになりそうだ。構わないか？」
「もちろんだよ。ここまで来て、引き返すわけにはいかない」

エリナが、力強く頷いた。

「暴走した守護者か。厄介そうだが、やるしかない」

リアムも、覚悟を決めた。

「私も、皆さんと一緒に戦います」

セシリアが、杖を握りしめた。

カイトは、水晶球に向かって言った。

「ありがとう。情報を参考にする」
「警告ヲ忘レズ。破損シタ守護者ハ、予測不可能。十分ニ注意サレタシ」

それだけ言い残して、水晶球の光は消えた。

◆ ◆ ◆

カイトたちは、情報保管庫を出て、さらに奥へと進んだ。

回廊は次第に狭くなり、壁の傷みも激しくなっていく。長い時が経っている証拠だ。

「それにしても、アレクシス・ヴェイルっていう人は、すごい人だったんだね」

エリナが、歩きながら言った。

「何千年も先のことまで予測して、仕掛けを残すなんて」
「管理者というのは、そういう存在なんだろう」

リアムが、答えた。

「世界を守るためのシステムを構築し、それを維持する。アレクシス・ヴェイルは、その始まりだった」
「僕も、そんな風になれるのかな」

カイトが、呟いた。

「管理者の末裔として、この世界を守っていけるのか」

「カイトなら大丈夫だよ」

エリナが、カイトの手を握った。

「私たちもついているから」
「エリナ……」

カイトは、エリナの温かさを感じた。この手が、どれだけ心強いか。

「そろそろだ。気配がする」

リアムが、警告した。

回廊の先に、大きな扉が見えた。その向こうから、重い機械音が聞こえる。

「暴走した守護者がいる」

カイトは、魔法を展開した。

カイトは、扉に手をかけた。

「行くぞ」

重い扉が、ゆっくりと開いた。

◆ ◆ ◆

大広間は、荒れ果てていた。

壁には亀裂が走り、床は瓦礫で埋め尽くされている。天井の一部が崩落し、光が差し込んでいた。

そして——広間の中央に、守護者がいた。

正常な守護者とは、明らかに違う。装甲の半分が剥がれ落ち、内部の歯車が露出している。右腕は根本から失われ、左腕には歪んだ刃が取り付けられていた。

最も異様なのは——その動きだ。

守護者は、同じ場所をぐるぐると回っている。まるで、壊れたおもちゃのように。

「ガガガ……エラー……エラー……ターゲット……ナシ……排除……シマス……」

守護者から、途切れ途切れの音声が漏れた。

「完全に暴走しているな」

リアムが、冷静に分析した。

「まともに動いているようには見えない。でも、それがかえって危険かもしれない」
「予測できないってこと？」

セシリアが、尋ねた。

「そう。正常な守護者なら、パターンがある。でも、暴走している場合は——何をするかわからない」

カイトは、守護者を観察した。

ぐるぐると回る動き——そこには、何か法則があるのか？ それとも、完全にランダムなのか？

「慎重にいく。まずは、様子を見よう」

カイトが、一歩踏み出した。

その瞬間——守護者が、カイトの方を向いた。

片目のレンズが、赤く光る。

「ターゲット……検知……カイ……ト……」

守護者の動きが、止まった。

そして——爆発的な加速で、カイトに向かって突進してきた。

「速い！」

カイトは、とっさに回避した。

守護者の刃が、カイトがいた場所を薙ぎ払う。床の石が砕け、粉塵が舞った。

「くっ——予測できないって本当だ」

カイトは、距離を取った。

正常な守護者なら、攻撃前に予備動作がある。でも、この守護者には——何もない。突然、動く。

「カイト、後ろ！」

エリナが、叫んだ。

カイトが振り返ると——守護者が、背後に回っていた。

「なっ——！？」

カイトは、魔法を展開した。


class EmergencyEvade {

カイトの体が、光に包まれて瞬間移動した。

守護者の刃が、空を切る。

「今のうちに！」

カイトは、守護者の背後から攻撃を仕掛けた。

「ライトニング・ストライク！」

雷の魔法が、守護者の背中を襲う。でも——装甲が厚い。ダメージは限定的だ。

「硬い……普通の攻撃じゃ効かない」

守護者が、再びカイトに向き直った。

赤いレンズが、不気味に光る。

「カイト……排除……カイト……排除……」

守護者は、カイトの名前だけを繰り返していた。

◆ ◆ ◆

戦いは、一進一退だった。

カイトの攻撃は、守護者の装甲に阻まれる。守護者の攻撃は、予測不能で避けるのがやっと。

「このままでは、じり貧だ」

カイトは、冷静に分析した。

暴走した守護者——その力の源は何だ？ 正常な守護者なら、コアを破壊すれば止まる。でも、この守護者のコアは——

カイトは、守護者の胸部を見た。

装甲が剥がれ落ちている部分から、コアが見える。でも、色が違う。青でも赤でもない——紫色だ。

「あのコア、何か変だ」

カイトが、呟いた。

「紫色のコア……見たことがない」

リアムが、眉をひそめた。

「カイト、あれは『虚無の汚染』かもしれない」
「虚無の汚染？」
「虚無の欠片が、機械に取り憑いた状態だ。本来の動力源が、虚無の力に侵食されている」

カイトは、理解した。

「つまり、僕が持っている虚無の欠片と同じ力が、あの守護者を動かしている？」
「その通りだ。だから暴走している。虚無の力は、予測不可能だからな」

カイトは、守護者を見つめた。

虚無の力——自分も、その力を手に入れた。制御できるようになったとはいえ、危険な力だ。

「わかった。あのコアに、僕の虚無の欠片を使って干渉する」
「干渉？ どうやって？」

エリナが、尋ねた。

「虚無同士は共鳴する。僕の欠片を媒介にして、あのコアの虚無を中和できるかもしれない」
「危険だぞ。失敗したら、カイト自身が侵食されるかもしれない」

リアムが、警告した。

「わかっている。でも、他に方法がない」

カイトは、虚無の欠片に触れた。

黒い結晶が、熱を帯びる。守護者のコアと共鳴し始めている。

「皆、援護を頼む。数秒間、あいつを止めてくれ」
「了解！」

エリナ、リアム、セシリア——三人が、同時に魔法を放った。

「フローズン・チェイン！」
「ウィンド・バインド！」
「ライト・プリズン！」

氷、風、光——三つの魔法が、守護者の動きを封じる。

守護者は、必死に抵抗した。でも、三つの拘束魔法には勝てない。

「今だ！」

カイトは、守護者の前に立った。

虚無の欠片を、守護者の胸部に押し当てる。

「共鳴せよ——虚空の調停」

class VoidHarmonization {
  private sourceCrystal: VoidFragment;
  private targetCore: CorruptedCore;

    const frequency = this.calculateResonance();
    const connection = this.establishLink(frequency);
    
    if (connection.success) {
      return this.purify();
    }
    
    return { status: 'failed', reason: 'resonance_mismatch' };
  }

    return { status: 'success', message: 'Core stabilized' };
  }

黒い光と紫色の光が、交錯した。

守護者の体が、激しく振動する。

「ガガガ……エラー……修正……シマス……エラー……修正……」

守護者の音声が、次第に安定していく。

そして——紫色のコアが、青色に変わった。

暴走が、止まった。

◆ ◆ ◆

守護者は、静かに立ち尽くしていた。

暴走が止まった今、敵意は消えている。

「ターゲット……カイト。管理者ノ末裔。認識シマス。暴走状態、解除サレマシタ。援助、感謝シマス」

守護者から、穏やかな声が響いた。

「君は、元々どんな目的で作られたんだ？」

カイトが、尋ねた。

「私ハ、遺跡ノ案内役トシテ作ラレマシタ。暴走前ハ、訪レタ者ヲ原初の座ヘ案内スル役割ヲ持ッテいた」
「案内役？ じゃあ、原初の座への道を知っている？」
「ハイ。現在ハ機能ノ一部ガ損傷シテいるため、同行ハ出来マセン。ダガ、道案内ハ可能デス」

守護者は、広間の奥にある通路を指差した。

「ソノ通路ヲ真っ直グ進メバ、原初の座ヘ到着シマス。距離ハ約二キロメートル。障害ハ、存在シマセン」
「ありがとう」

カイトは、守護者に礼を言った。

「君を修理することはできないか？」
「現状ノ技術デハ、困難デス。ダガ、機能停止シテいる訳デハアリマセン。此処デ、待機ヲ続ケマス」
「わかった。また、いつか来るかもしれない」

カイトは、守護者に別れを告げた。

◆ ◆ ◆

守護者が教えてくれた通路を、カイトたちは進んだ。

壁の光が、次第に強くなっていく。出口が近い証拠だ。

「もうすぐ、原初の座だ」

カイトが、静かに言った。

「長い旅だったね」

エリナが、カイトの横に並んだ。

「うん。でも、まだ終わりじゃない。原初の座で、何が待っているのか——それが本当の試練だ」
「カイトなら大丈夫。私たちは、ずっと一緒だから」

エリナが、微笑んだ。

リアムとセシリアも、静かに頷いた。

カイトは、前を向いた。

光が、見えてきた。

通路の先に、開けた場所が広がっている。空が見える——原初の座への入り口だ。

「行こう。運命の場所へ」

カイトたちは、光の中へと歩き出した。

長い旅の、いよいよクライマックスが近づいていた。
