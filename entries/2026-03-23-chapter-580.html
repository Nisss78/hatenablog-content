第580章：虚空の境界

北東への道は、次第に荒涼としていった。

中央大陸の豊かな森を抜け、岩石だらけの荒野を越えると、空気そのものが変わった気がした。重く、冷たく、そして——空虚。

「空気が……薄い」

リアムが息を切らしながら言う。普段は冷静な彼らしくない反応だが、ここでの環境は確かに異常だった。

カイトも同じ感覚を抱いていた。まるで、この場所自体が「存在」を希薄にさせているような——。

class ZoneEffect {
    name = "VoidBoundary";
    effects = {
        existenceDensity: 0.3,  // 存在密度の低下
        magicEfficiency: 0.7,   // 魔法効率の低下
        staminaRecovery: 0.5    // 体力回復の阻害
    };
}

「ここが境界域か」

カイトの呟きに、エリナが頷く。

「ええ。虚無の聖域へ続く、境界の領域。ここから先は、世界の理が歪んでいるわ」

セシリアが周囲を警戒しながら、杖を構える。

「注意して。この領域には、境界の守護者がいると言われているわ」

「守護者？」

「古の時代、虚無の聖域を守るために創られた存在。侵入者を排除するようプログラムされている」

プログラム——その言葉に、カイトの心が反応する。

「つまり、こいつらもシステム上の存在だということか」

「そのはずよ。でも、数百年もの間、誰もメンテナンスしていない。どうなっているかは……」

セシリアが言い終わる前に、地面が振動した。

□ ■ □

岩石が崩れ、砂塵が舞う中、何かが姿を現した。

それは、人型をした影だった。いや、正確には「人型の輪郭を持つ、存在そのものが曖昧な何か」と言うべきか。

顔には目も鼻もない。ただ、中央に一本の亀裂のような線が走っているだけ。

「——侵入者を検知」

影のような存在が、機械的な声で告げる。

「対象の存在密度を確認。許容値を超えています。即時排除を開始します」

「やっぱりか！」

カイトは即座に杖を構える。

function analyzeEnemy(target) {
    const weakPoints = target.vulnerabilities;
    const patterns = target.attackPatterns;
    
    // 境界の守護者は「存在」への攻撃に弱い
    if (weakPoints.includes("existenceOverride")) {
        return "USE_EXISTENCE_SPELL";
    }
}

「みんな、こいつには普通の魔法が効きにくい！ 存在を肯定するような攻撃を使え！」

「存在を……肯定？」

リアムが困惑するが、エリナは即座に理解したようだ。

「つまり、『私はここにいる』と主張するような魔法ね」

「その通り！」

影の守護者が動いた。その動きは、物理法則を無視しているかのように不規則で、予測不可能。

一瞬でエリナの背後に現れ、腕を振り下ろす。

「させない！」

セシリアが光魔法を放つ。純白の光が守護者を照らすが、ダメージは与えられない。

「効いてない！？」

「光魔法じゃダメだ！ 光も闇も、ここでは意味を持たない！」

カイトは叫びながら、独自の魔法を編み始める。

spell ExistenceAnchor() {
    const self = caster.identity;
    const world = currentWorld.laws;
    
    // 自分の存在を世界に強制的に認識させる
    world.assert(self.exists() == true);
    world.lock(self.position);
    
    // 周囲の存在密度を強化
    for (ally in party) {
        ally.existenceDensity += 0.5;
    }
}

「Existentia Anchora!」

カイトの詠唱とともに、金色の鎖が空中に現れた。それはカイトと仲間たちを繋ぎ、彼らの存在を強固にする。

守護者の攻撃が、リアムに届く直前で弾かれた。

「効いた！」

「今だ、反撃だ！」

エリナが氷魔法を、リアムが剣技を、セシリアが雷魔法を同時に放つ。三つの攻撃が、存在を強化された状態で守護者に直撃する。

守護者の体が、砂のように崩れ始めた。

「エラー……エラー……存在否定に失敗……」

最後の言葉を残して、守護者は霧散した。

□ ■ □

戦闘が終わった後、シオンが地面に膝をついていた。

「シオン、大丈夫か？」

カイトが駆け寄ると、シオンは震える声で答えた。

「うん……大丈夫。ただ、何かを……思い出しかけて」

「何を？」

「あの守護者の声。どこかで……聞いたことがあるの」

シオンは頭を抱える。

「『侵入者を検知』って言葉。それから……『存在密度』……」

シオンの瞳が、遠くを見つめる。

「私は……かつて、あの場所にいた。虚無の聖域に。そして、何かを……誰かを、守ろうとしていた気がする」

「誰を？」

「わからない。でも、とても大事な人だった」

シオンの言葉に、カイトは胸が締め付けられる思いだった。記憶を失った彼女が、少しずつ過去を取り戻している。だが、その記憶は果たして——。

「シオン、無理はするな。記憶は自然に戻るものだ」

「うん……ありがとう、カイト」

□ ■ □

境界域を進むにつれて、風景はさらに歪んでいった。

空には、複数の月が浮かんでいるように見える。地面は、上昇したり下降したりを繰り返している。遠くには、逆さまに浮いた山々が見えた。

「ここは……本当に、世界の中なのか？」

リアムが呆然と呟く。

「境界域は、世界の裏側とも言われている」

カイトが説明する。

「普通の世界は、因果律というプログラムによって秩序が保たれている。でも、ここはその秩序が最も薄い場所。だから、物理法則も因果律も、歪んでいるんだ」

「なら、私たちの魔法も歪む可能性があるわね」

エリナの指摘に、カイトは頷く。

「その通り。ここでは、何が起こるか予測できない。常に警戒しておけ」

□ ■ □

歩くこと数時間。一行は、奇妙な遺跡の前に到着した。

それは、古代の神殿のようにも見えたが、建材が透明な結晶だった。中を覗くと、無数の光が複雑に交錯しているのが見える。

「これは……時空の記録施設？」

セシリアが驚きの声を上げる。

「まさか、こんな場所に残っていたなんて」

「時空の記録施設？」

カイトが尋ねると、セシリアは説明を始めた。

「最初の管理者たちが、世界の安定を図るために建設した施設よ。ここには、世界中の時空データが保存されているはず」
「つまり、世界再構築に関する情報も？」
「可能性は高いわ。中に入ってみましょう」

□ ■ □

神殿の中は、外観からは想像もつかないほど広かった。

壁一面に、光の文字が流れている。カイトにはそれが、巨大なデータベースのログに見えた。

WORLD_LOG://
[ENTRY_001] World created. First Administrator: Alexia Vale.
[ENTRY_023] Seven Primal Forces calibrated.
[ENTRY_157] World Reconstruction attempt detected.
[ENTRY_158] Intervention by Seven Sages. Reconstruction halted.
[ENTRY_892] Boundary guardian activated.
...
[ENTRY_1847] WARNING: Anomaly detected at Void Sanctuary.
[ENTRY_1848] World Reconstruction signal increasing.
[ENTRY_1849] Alert level: CRITICAL.

「これが……世界の記録」

カイトは、圧倒的な情報量に息を呑む。

「エントリー一八四七……虚無の聖域で異常検知。それが、今起きていることだ」

エリナが、一つの光る文字列を指し示した。

「カイト、これを見て」
[ENTRY_1203] Subject:SION created.
Purpose: Interface between worlds.
Status: Memory sealed. Deployed to Central Continent.

「シオン……？」

カイトは振り返る。そこには、自分の名前を見つめて呆然と立ち尽くすシオンの姿があった。

「私……インターフェース？」

シオンの手が、震えながら自分の胸に触れる。

「私は……世界の間を繋ぐために、創られたの？」

誰も答えられなかった。ただ、冷酷な事実だけが、光の文字として彼女を凝視していた。

□ ■ □

神殿の奥で、一つのホログラムが現れた。

それは、白衣を着た女性の姿をしていた。顔立ちはシオンと似ているが、より大人びた雰囲気だ。

「ようこそ、旅人たち。私はアルケミア——第一の調律者です」

「アルケミア！」

カイトは驚く。以前、シオンの母として名前を聞いたことがある。

「この記録をあなたが見ているということは、世界再構築が進行しているのでしょう。ならば、私は真実を語る義務があります」

ホログラムのアルケミアは、静かに語り始める。

「シオンは、世界と世界の間を繋ぐための存在として創られました。彼女の七属性魔法回路は、あらゆるエネルギーを調和させ、世界の均衡を保つためのもの」

シオンは、自分の両手を見つめる。

「私……」
「ですが、シオンには欠陥がありました。彼女は、人間としての感情を持ってしまったのです。愛する心、悲しむ心、そして——守りたいという意志」

アルケミアのホログラムは、慈愛に満ちた表情を浮かべる。

「だから私は、彼女を人間として生きることを選ばせました。記憶を封じ、一人の少女として。それが、私の……母としての選択でした」

シオンの目から、涙が溢れる。

「お母さん……」
「シオン、あなたは道具ではありません。あなたは、私の娘です。そして、あなたの選んだ道が、世界を救う鍵となるでしょう」

ホログラムが消えようとした時、アルケミアは最後に言い残した。

「虚無の聖域の奥に、世界再構築の核があります。それを止めるには、カイトの『コード・マジック』と、シオンの『七属性調律』の両方が必要です。二人で、終わらせてください」

そして、光は消えた。

□ ■ □

神殿を出た一行は、再び北東へ歩き出した。

シオンはずっと黙っていたが、やがて口を開いた。

「カイト、私は……自分が何者かわかった気がする」

「ああ」

「私は、世界のために創られた存在。でも、お母さんは私を娘として選んでくれた」

シオンは、涙を拭って微笑む。

「だから、私は自分の意志で戦う。世界を守るために——みんなを守るために」

その笑顔は、以前よりも強く、美しく見えた。

カイトは頷く。

「ああ。一緒に戦おう、シオン」

空には、まだ紫の光が脈打っている。だが、その光に向かって歩く彼らの背中は、迷いのないものだった。

——虚無の聖域まで、あとわずか。
