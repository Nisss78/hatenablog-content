第627章：北への道

朝——アルヴァレス支部の正門前に、五人の姿があった。

カイト、エリナ、リアム、セシリア、シオン。それぞれが、旅の準備を整えていた。

「防寒具——全員、持ってるわよね?」

エリナが、確認するように言った。

「持ってる!」

リアムが、リュックを叩いた。

「厚着しすぎると動きにくいから——魔力で温度調整できるようにした」
「さすがリアム——用意がいいな」

カイトが、苦笑した。

「私は——時空魔法で寒さ対策できるわ」

セシリアが、静かに言った。

「空間を操作して——冷気を遮断できるの」
「便利だな——時空魔法って」
「そうでもないわ。魔力消費が激しいから——長時間は無理」

シオンが、少し不安そうに言った。

「私は——七属性全部使えるから……火魔法で暖を取れるよ」
「シオンがいれば——凍える心配ないな」

リアムが、笑った。シオンは——少し照れた。

「そ、そんなことないよ……まだ未熟だし」
「謙遜しなくていい。シオンの魔法——頼りにしてる」

カイトの言葉に、シオンは——小さく頷いた。

◆ ◇ ◆

北への道——雪に覆われた街道を、一行は歩いていた。

アルヴァレスから北の山脈までは——馬車で三日の距離。だが——魔力が回復した今、馬車よりも早い方法があった。

「テレポート——使える?」

エリナが、セシリアに聞いた。

「北の山脈までは——無理ね。距離が遠すぎる」

セシリアが、首を振った。

「中継地点を挟めば——可能かもしれない。でも——魔力の消費が激しいから——全員を一度に運ぶのは厳しい」
「じゃあ——歩き?」

リアムが、がっかりしたように言った。

「いや——もっといい方法がある」

カイトが、口を開いた。

「飛ぶぞ」
「飛ぶ?」
「ああ。風魔法で——一気に北上する」

カイトは——手を掲げた。


function createWindPath(destination) {
  const wind = conjureElementalForce("wind");
  wind.setDirection("north");
  wind.setSpeed("high");
  wind.setPassengers(5);
  wind.shieldFromCold();

風が——渦を巻いた。そして——五人を包み込む。

「すごい——これなら半日で着くかも」

シオンが、目を輝かせた。

「しっかり掴まってろよ——飛ぶぞ!」

カイトの合図と共に——一行は、空へと舞い上がった。

◆ ◇ ◆

空の旅——風の盾に守られながら、一行は北上した。

下に見える風景——雪原、森、そして小さな村々。魔力が回復した世界——徐々に、徐々に、豊かになりつつあった。

「綺麗……」

シオンが、窓のような風の隙間から下を眺めた。

「雪が——キラキラしてる」
「太陽の光を反射してるんだよ」

リアムが、説明した。

「氷の結晶——それぞれが、小さなプリズムみたいに光を届けるんだ」
「リアム——詩人みたい」
「えっ——そ、そう? ただ——前に、誰かに聞いただけだよ」

リアムは——少し照れた。カイトは——その様子を見て、微笑んだ。

こいつら——本当に、仲間だな。

◆ ◇ ◆

数時間後——北の山脈が、見えてきた。

白銀の山々——雲を突き抜ける峰々。壮麗——そして、威圧的。

「あれが——北の山脈」

セシリアが、静かに言った。

「フィンブルヴェトが封印されていた場所——そして、フローズナ様の聖域」
「フローズナ様——氷の賢者だよね?」

シオンが、確認するように聞いた。

「ええ。カイトたちが——以前、会ったことがあるのよね?」
「ああ。フィンブルヴェトを封じる儀式に——協力してくれた」

カイトの表情——少し、暗くなった。

「フローズナ様——今、どうしてるかな」
「会えばわかるわ——まずは、着陸するわよ」

エリナの言葉に——カイトは頷いた。

◆ ◇ ◆

山脈の麓——雪に覆われた平地に、一行は着陸した。

冷気——肌を刺すような寒さ。だが——風の盾が、ある程度は防いでいた。

「寒っ!」

リアムが、身震いした。

「予想以上だな——山の上は、もっと寒いぞ」
「勘弁してよ——」

リアムは——さっそく、炎魔法を使って、体を温めた。

「リアム——魔法使いすぎないで。魔力は——節約しなきゃ」

セシリアが、注意した。

「わかってる——でも、凍え死にそうなんだよ」
「大げさね。私は——全然寒くないわ」
「セシリアは——時空魔法で対策してるからだろ!」

二人のやり取りを見て——シオンが、笑った。

「ふふ——仲いいね」
「仲いいわけじゃない! ただ——相性が悪いだけ!」

リアムとセシリア——同時に言った。そして——顔を見合わせて、ため息をついた。

◆ ◇ ◆

カイトは——周囲を警戒していた。

魔力反応——テオドール支部長が言っていた異常。カイトは——自分の感知魔法で、探っていた。


function scanManaAnomaly(radius) {
  const manaFlow = detectManaInRadius(radius);
  const anomalies = manaFlow.filter(m => m.intensity > THRESHOLD);
  return analyzeSource(anomalies);
}

結果——複数の反応。

「どう?」

エリナが、カイトに聞いた。

「反応がある——複数。でも——弱い」
「弱い?」
「ああ。今のところ——危険なレベルじゃない。でも——奥に行くほど、強くなってる」

カイトは——山の頂上を指差した。

「頂上——そこが、一番強い反応だ」
「じゃあ——頂上を目指す?」
「いや——まずは、フローズナ様の聖域に寄ろう。情報が欲しい」

カイトの判断——皆が頷いた。

◆ ◇ ◆

山道——雪に埋もれた道を、一行は登った。

風——強くなってきた。視界——白一色。

「これ——吹雪になりそうね」

セシリアが、空を見上げた。

「山小屋——探したほうがいいかも」
「待って——反応がある」

カイトが、手を止めた。

「何か——来る」

雪の中から——影が現れた。

白い毛皮——鋭い爪。氷の狼——フェンリルの眷属。

「フェンリル……」

シオンが、息を呑んだ。

「大丈夫——攻撃してくる様子はない」

カイトが、冷静に言った。

フェンリル——ゆっくりと、近づいてきた。そして——カイトの前で、止まった。

「……連れて行ってくれるのか?」

カイトが、聞いた。

フェンリル——頷いたように、頭を下げた。

「ついて来い——案内してくれるみたいだ」

一行は——フェンリルの後を追った。

◆ ◇ ◆

フェンリルに案内され——一行は、洞窟の前に着いた。

洞窟の入口——氷の結晶が、美しく輝いていた。

「ここ——フローズナ様の聖域への入り口?」

エリナが、聞いた。

「みたいだな」

カイトは——フェンリルに礼を言った。

「ありがとう——助かった」

フェンリル——一声、吠えた。そして——雪の中へと消えていった。

「行こう——中へ」

カイトの言葉に——一行は、洞窟の中へと足を踏み入れた。

◆ ◇ ◆

洞窟の中——意外なほど、温かかった。

氷の壁——だが、寒さを感じない。魔法——氷の賢者の魔法。

「綺麗……」

シオンが、壁に触れた。

「氷なのに——温かい」
「フローズナ様の魔法よ——氷を制する者は、寒さも制する」

セシリアが、説明した。

洞窟の奥——光が漏れていた。

「誰か——来たわね」

女性の声——落ち着いた、透き通るような声。

奥から——一人の女性が現れた。白い髪——青い瞳。氷の賢者、フローズナ。

「久しぶりね——カイト」
「フローズナ様——お元気そうで」
「ええ——あなたも、元気そうね」

フローズナは——カイトの後ろにいる仲間たちを見た。

「新しい仲間?」
「ああ。シオン——七属性魔法使いだ」
「七属性——珍しいわね。始めまして、シオン。私はフローズナ——氷の賢者よ」
「は、始めまして——シオンです」

シオンは——少し緊張しているようだった。

「それで——カイト。何をしに来たの?」
「調査です——異常な魔力反応が、この山で観測されていて」

カイトの言葉に——フローズナの表情が、曇った。

「……やっぱり、来たのね」
「やっぱり?」
「ええ。私も——数日前から、違和感を感じていたの」

フローズナは——洞窟の奥へと歩き出した。

「ついて来て——見せたいものがあるわ」

◆ ◇ ◆

洞窟の最深部——巨大な氷の結晶が、鎮座していた。

だが——その結晶に、亀裂が入っていた。

「これは……」

カイトが、息を呑んだ。

「フィンブルヴェトを封じていた結晶——封印の核よ」

フローズナが、静かに言った。

「フィンブルヴェトは——倒された。でも——封印そのものは、残っていた。そして——今、崩壊し始めている」
「何が——中に?」
「わからない——でも、強大な何かがいる」

フローズナの言葉に——全員が緊張した。

「カイト——あなたにしか、できないことがあるかもしれない」
「俺にしか?」
「ええ。管理者の力——そして、コード魔法。その二つがあれば——中の何かを、鎮められるかもしれない」

カイトは——結晶を見つめた。亀裂——そこから、黒い霧が漏れ出していた。

「……わかった。やってみる」

カイトの決意——仲間たちも、頷いた。

北の山脈——新たな試練が、待っていた。

カイトは——覚悟を決めた。

管理者として——そして、冒険者として。
