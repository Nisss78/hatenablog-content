第464章：光の賢者の真実

ルミナスに導かれ、カイトたちは光の塔の内部を進んだ。

通路の壁は、半透明の結晶でできている。中を光の粒子が流れ、まるで生きている血管のようだ。

「すごい……。これ全部、魔法で作られてるんですか？」

エリナが、感嘆の声を漏らした。

「左様。この塔は、私が三百年の歳月をかけて築き上げたものじゃ」

ルミナスは、懐かしむような表情で答えた。

「当時は、まだ若かった。世界を守りたいという一心で、ここまで作り上げたのじゃよ」

三百年。その数字に、カイトは驚いた。見た目は二十代後半から三十代前半にしか見えない。

「賢者の方々は、皆さん長生きなんですね」

リアムが言った。

「寿命が違うのか、それとも魔法で延命してるのか……」

「両方じゃな。賢者としての力が強まれば、自然と寿命も延びる。それに、各領域のエネルギーと同化することで、老化を遅らせることもできるのじゃ」

ルミナスは説明しながら、一行を広い部屋に招き入れた。

部屋の中央には、円形のテーブルが置かれている。周囲には、柔らかそうな椅子が四つ。壁際には、果物や飲み物が並べられた棚があった。

「まずは休むがよい。腹も減っておろう」

ルミナスの言葉に、カイトたちは顔を見合わせた。確かに、船旅で疲れているし、空腹も感じていた。

「じゃあ、お言葉に甘えて」

四人は椅子に座り、振る舞われた食事をとった。光の塔特有の食材らしく、どの料理も淡い光を放っている。

「これは、光の結晶から抽出したエッセンスを使っておる。魔力を回復させる効果があるのじゃよ」

ルミナスの説明通り、一口食べるごとに体の中から力が湧いてくる感覚があった。

◆ ◆ ◆

食事を終えた後、本題に入った。

ルミナスは、カイトの向かいに座った。真剣な表情だ。

「カイトよ。お主がここに来た目的は分かっておる。虚空の種の真実を知りたいのじゃな？」
「はい。テラドンさんから、ある程度は聞きました。でも、まだ核心までは分からなくて」

カイトは、テラドンとの対話を思い出した。管理者の末裔。虚空の種の意志。世界を覆う地脈の汚染。どれも重要な情報だが、つながりが見えてこない。

「テラドンから聞いたのなら、説明は省けるのじゃな。では、私が語るべきは……」

ルミナスは一瞬目を閉じ、意識を集中させた。

「虚空の種の起源について、そして……お主の力の真の意味についてじゃ」

ルミナスは、空中に図形を描いた。光の粒子が集まり、複雑な幾何学模様を形成していく。

「まず、虚空の種とは何か。それは、この世界が生まれたときに残された『残滓（ざんし）』じゃ」
「残滓？」

セシリアが首を傾げた。

「世界の創造において、不完全な要素が残ることがある。それが、虚空の種の正体じゃ」

ルミナスは、さらに図形を展開した。今度は、二つの球体が描かれた。一方は光り輝き、もう一方は漆黒の闇に包まれている。

「この世界は、光と闇、生と死、秩序と混沌――そうした二律相反する要素のバランスで成り立っておる。じゃが、創造の過程で、そのバランスが崩れることがあった」
「それが、虚空の種……？」

カイトが問うと、ルミナスは頷いた。

「そうじゃ。虚空の種は、世界の『無』の部分が凝縮されたもの。本来なら存在しないはずの空白領域が、意志を持って動き出したのじゃ」
「意志を持って、というのは？」

エリナが身を乗り出した。

「虚空の種には、明確な目的がある。それは……『世界の再構築』じゃ」

ルミナスの言葉に、全員が息を呑んだ。

「再構築？」

リアムが震える声で聞いた。

「今の世界を壊して、新しい世界を作るってことですか？」
「左様。虚空の種は、今の世界を『不完全』と見なしておる。だから、一度すべてを無に帰してから、新たな秩序を築こうとしているのじゃ」

カイトは、背筋が凍るのを感じた。世界を一度破壊し、作り直す。それは、今のすべてを失うことを意味する。

「そんな……。今の世界に住んでいる人たちは、どうなるんですか？」

セシリアが問うた。

「消える。虚空の種の再構築において、今の生命や文明は『不要なデータ』として処理されるのじゃ」

ルミナスの言葉は、残酷な響きを持っていた。

「だからこそ、私たち管理者は、世界を守るために戦っておる。虚空の種の侵食を食い止め、バランスを保つことが、我々の使命じゃ」

◆ ◆ ◆

沈黙が流れた。

カイトは、ルミナスの話を消化しようと努めた。虚空の種の正体。世界の再構築。そして、管理者の使命。

「ルミナスさん。一つ聞きたいことがあります」

カイトが口を開いた。

「僕が管理者の末裔だというのは分かりました。でも、僕の力――魔法をコードとして記述できる能力は、どういう意味があるんですか？」

カイトの質問に、ルミナスは柔和な笑みを浮かべた。

「良い質問じゃ。実は、それこそがお主の最大の武器なのじゃよ」

ルミナスは、新たな図形を描いた。今度は、数式のような記号が並んでいる。

「この世界の魔法体系は、アレクシス・ヴェイルが確立したものじゃ。彼は、魔法を『世界を記述する言語』として捉えておった」
「世界を記述する、言語……」

カイトは、自分の能力と照らし合わせた。確かに、自分が魔法を使うときは、世界に対して命令を送っている感覚がある。

「アレクシスは、魔法をプログラミングのように扱える能力を持っておった。魔力の流れを可視化し、それを最適化するコードを記述することができたのじゃ」
「それって、僕と同じ……」
「左様。お主は、アレクシスの血脈を受け継ぐ者として、彼と同じ能力を持って生まれたのじゃ」

ルミナスは、カイトの目を真剣に見つめた。

「魔法をコードとして記述できるということは、世界そのものを書き換えることができるということじゃ。虚空の種が『世界の再構築』を目指すのに対し、お主は『世界の改変』を行うことができる」
「改変……？」
「虚空の種を無に帰すのではなく、その性質を変えることができるかもしれない。お主の力があればな」

カイトは、自分の能力の重大さを理解した。単に強力な魔法使いというわけではない。世界の根本的なルールに干渉できる、特異な存在なのだ。

「でも、僕にはそんな大それたことができるんでしょうか。まだ、自分の力も完全に制御できていません」

カイトの不安に、ルミナスは優しく答えた。

「安心せい。そのために、ここに来たのじゃろう？ 試練を受けて、力を高めるために」

ルミナスは立ち上がり、カイトに手を差し出した。

「明日、お主に光の試練を与えよう。地の試練を乗り越えたお主なら、必ずや成功するはずじゃ」
「……分かりました。やります」

カイトは、ルミナスの手を握った。温かい光が、二人の手の間で交わる。

◆ ◆ ◆

その夜、カイトは光の塔の客室で休んでいた。

ベッドに横になりながら、今日の情報を反芻する。虚空の種の正体。世界の再構築。そして、自分の力の真の意味。

「世界を書き換える力、か……」

カイトは、自分の右手を見つめた。魔法を使うとき、この手から光が溢れる。それが、世界に干渉する触媒なのだ。


function rewriteWorld(target, change) {
    const analysis = analyzeStructure(target);
    const modification = designChange(change);
    const execution = applyModification(analysis, modification);
    return stabilizeResult(execution);
}

頭の中に、自然とコードが浮かんだ。もし本当に世界を書き換えるのだとすれば、こんな関数を書くことになるのだろうか。

「カイト、寝てる？」

ドアの外から、エリナの声がした。

「いや、起きてる。どうしたの？」

ドアが開き、エリナが入ってきた。寝間着姿で、髪を下ろしている。

「眠れなくて。ルミナスさんの話、気になっちゃって」

エリナは、カイトのベッドの端に座った。

「世界の再構築とか、書き換える力とか。すごく重い話だったね」
「うん。正直、圧倒されてるよ」

カイトは、天井を見上げた。

「僕が管理者の末裔で、特別な力を持ってるっていうのは分かった。でも、それで何をすればいいのか、まだ見えてこないんだ」
「……カイトは、真面目だね」

エリナが、ふっと笑った。

「いきなり『世界を救え』って言われても、誰だって戸惑るよ。私だったら、逃げ出したくなるかも」
「エリナは逃げないよ。僕の隣で、戦ってくれるはずだ」

カイトが言うと、エリナは少し顔を赤らめた。

「……ありがと。私も、カイトを信じてるよ。だから、そんなに悩まないで。一歩ずつ進めばいいんだよ」

エリナは、カイトの手を握った。温かい感触が、心を落ち着かせる。

「明日は光の試練、なんだよね？ 今は、それに集中しようよ」
「……うん。そうだね」

カイトは、エリナの言葉に救われた。大きな目標ばかりを見ず、目の前の一歩に集中する。それが、今の自分に必要な姿勢なのだ。

「ありがとう、エリナ。少し落ち着いたよ」
「どういたしまして。じゃあ、おやすみ」

エリナは手を離し、部屋を出ていった。残されたカイトは、再びベッドに横になった。

明日は、光の試練。何が待っているかは分からない。でも、仲間がいる。一人ではない。

カイトは、静かに目を閉じた。

◆ ◆ ◆

翌朝、カイトはルミナスに案内されて、光の塔の最上階へと向かった。

螺旋階段を登ること、百段以上。息が切れる頃、ようやく頂上の広間に到着した。

広間は、丸天井のドーム状。壁も天井も、光で構成されている。中央には、直径十メートルほどの円形の台座があった。

「ここが、光の試練の場じゃ」

ルミナスが宣言した。

「カイトよ。お主に課す試練は、二つある」
「二つ、ですか？」
「そうじゃ。一つ目は、光の加護を得ること。二つ目は……」

ルミナスは、意味深に言葉を切った。

「自分自身の闇と向き合うことじゃ」
「自分自身の、闇……？」

カイトが首を傾げると、ルミナスは頷いた。

「光と闇は表裏一体。光の力を得るには、闇を受け入れる必要がある。お主の中にある、恐れや不安、迷い。それらと向き合い、乗り越えるのじゃ」

ルミナスは、台座の上に立った。

「準備はよいか？ この試練は、地の試練よりも厳しいかもしれぬぞ」

カイトは、深呼吸をした。エリナの言葉を思い出す。「一歩ずつ進めばいい」。

「やります。どんな試練でも、乗り越えてみせます」

カイトの決意に、ルミナスは満足げに微笑んだ。

「よろしい。では、始めよう」

ルミナスが手をかざすと、台座が光り輝いた。眩しいほどの輝きが、カイトを包み込む。

視界が白く染まり、意識が遠のいていく――。

◆ ◆ ◆

カイトが目を開けると、そこは見知らぬ場所だった。

真っ白な空間。床も壁も天井も、すべてが白一色。何もない、無限の空白。

「ここは……？」

カイトがつぶやくと、背後から声がした。

「ようやく来たな」

カイトは振り返った。そこに立っていたのは、自分自身だった。

同じ顔、同じ背格好、同じ服装。ただ一つ違うのは、もう一人のカイトの瞳が、漆黒の闇に染まっていること。

「君は、誰だ？」

カイトが問うと、闇のカイトは不敵に笑った。

「俺はお前だ。お前の中にいる、もう一人の可能性」

闇のカイトは、ゆっくりと歩み寄ってきた。

「お前は、自分がこの世界を救えると思ってるのか？ 管理者の末裔だから？ 特別な力があるから？」
「……そういう意味じゃない。僕は、やるべきことをやろうとしてるだけだ」
「やるべきこと、か。自分に課した義務か、それとも誰かに期待されたからか？」

闇のカイトの言葉が、カイトの胸に突き刺さった。確かに、自分は期待に応えようとしているだけなのかもしれない。テラドン、ルミナス、そして仲間たち。彼らが自分を頼りにしているから、逃げられないだけなのかもしれない。

「違う……」

カイトは、首を横に振った。

「僕は、自分で選んだんだ。この世界を守りたいって。守りたい人たちがいるから、戦うんだ」
「守りたい人たち、ね。エリナか？ リアムか？ セシリアか？」

闇のカイトは、冷ややかな目でカイトを見据えた。

「彼らを守るために、お前は自分を犠牲にできるのか？ 世界を救うために、大切な人を切り捨てることができるのか？」

カイトは、答えに詰まった。究極の選択。もし、エリナを救うためには世界を犠牲にしなければならないとしたら。自分は、どちらを選ぶのか。

「……分からない」

カイトは、正直に答えた。

「そういう場面になったら、どうするか。今は、答えられない」

闇のカイトは、驚いたような顔をした。

「正直なやつだな。嘘をついて、『世界を救う』とか言えばいいのに」
「嘘はつきたくない。自分にも、仲間にも」

カイトは、闇のカイトを真っ直ぐに見つめた。

「僕は、完璧じゃない。迷うし、弱いし、逃げ出したくもなる。でも、それでも前に進む。仲間と一緒に、一歩ずつ」

カイトの言葉に、闇のカイトの表情が変わった。冷ややかだった瞳が、少しずつ和らいでいく。

「……そうか。お前は、そういうやつだったな」

闇のカイトは、笑った。今度は、穏やかな笑みだった。

「俺は、お前の中にある迷いや不安の化身だ。でも、お前はそれを受け入れた。否定せず、認めて、前に進むと言った」

闇のカイトは、カイトに手を差し出した。

「その強さが、お前の本当の力だ。光も闇も、どちらも背負って進む。それが、管理者の末裔としての資格じゃないか？」

カイトは、闇のカイトの手を握った。するとは、光の粒子となって消えていった。

「ありがとう。僕の中にいる、もう一人の僕」

カイトは、自分の胸に手を当てた。迷いは消えていない。でも、それを受け入れる強さが芽生えた。

真っ白な空間が、光に包まれていく――。

◆ ◆ ◆

カイトが目を覚ますと、ルミナスが目の前にいた。

「よくやった、カイト。試練を乗り越えたのじゃな」

ルミナスは、満足そうに頷いた。

「お主は、自分の闇を受け入れた。それは、光の力を得るための第一歩じゃ」

カイトは、体を起こした。体の中に、以前とは違う感覚がある。温かい光が、血管を巡っているようだ。

「これが、光の加護……？」
「左様。お主の中に、光の種が宿ったのじゃ」

ルミナスは、カイトの胸元を指差した。そこには、淡い光が灯っている。テラドンから貰った大地の実と共鳴し、琥珀色と金色が混ざり合っていた。

「地と光、二つの力を手に入れたお主は、着実に力をつけておる。じゃが、まだ完全ではない」
「まだ、足りないものがあるんですか？」
「ああ。賢者の試練は、全部で七つある。それぞれが、異なる属性の力を授けるのじゃ」

ルミナスは、指を折って数えた。

「地、水、火、風、氷、雷、光。この七つの属性をすべて統合したとき、お主は真の管理者として覚醒するのじゃよ」

七つの属性。そのすべてを手に入れるためには、長い旅が必要になる。

「分かりました。一つずつ、確実に」

カイトは、決意を新たにした。

光の塔での試練を終え、カイトは新たな力を手に入れた。だが、旅はまだ終わらない。虚空の種の脅威は、刻一刻と迫っている。

次なる目的地は、どこになるのか。カイトの冒険は、まだ続く――。
