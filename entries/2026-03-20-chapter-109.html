第109章：三人目の管理者

カイトたちは、遺跡の最深部への道を進んでいた。設計図から得た知識を胸に、エーテリアとの再会を目指す。

「カイト、さっきの話だけど……」

エリナが、歩きながら切り出した。

「三人目の管理者って、本当に四百七十二年も眠っているの？」
「設計図にはそう書かれていた。でも、なぜ活動を停止したのかは不明だ」

カイトは、考え込みながら答えた。

「もしかしたら、何か重大な理由があったのかもしれない。あるいは——」
「あるいは？」

リアムが、先を促した。

「誰かが、意図的に活動を停止させた可能性もある」

その言葉に、全員が足を止めた。

リリが、冷静に分析を始めた。

「意図的な停止となると、他の管理者の関与が疑われますわ。エーテリアかヴォルター、あるいはその両方が、三人目の管理者を封印した可能性があります」
「だが、なぜ？」

セシリアが、疑問を投げかけた。

「三人目の管理者の役割が『スペシャル・ファンクション』だった。つまり、何か特別な権限や機能を持っていたはずだ」

カイトは、設計図の記憶を辿った。

// Tertiary Admin: [CORRUPTED DATA]
// Function: Override & Reset Authority
// 
// WARNING: This admin has the power to:
// - Override any system decision
// - Reset world state to backup point
// - Terminate other admin sessions
//
// RESTRICTION: Can only activate when world is in critical danger
「見つけた。三人目の管理者の権限は、『オーバーライド＆リセット』だ」

カイトの声が、緊張を帯びた。

「どんなシステム決定も覆せる。世界をバックアップ時点まで戻せる。他の管理者のセッションを終了できる」
「それは……強大すぎる権限ですね」

アルフレッドが、息を呑んだ。

「だから封印されたのかもしれない。誰かが、その権限を恐れたんだ」

廊下の先から、青白い光が漏れてきた。遺跡の最奥——そこに、エーテリアがいた。

管理者は、巨大な水晶の前に佇んでいた。水晶の中には、複雑な魔法陣が回転している。

「来ましたね、カイト」

エーテリアは、振り返らずに言った。

「設計図を読んだのでしょう。緊急プロトコルのことも」
「知っていたのか？」

カイトが、問いかけた。

「もちろん。私は、この世界の管理者の一人ですから」

エーテリアが、ゆっくりと振り返った。その表情は、どこか悲しげだった。

「質問があるはず。三つの聖なる遺物について、そして——三人目の管理者について」
「ああ。教えてほしい」

カイトは、一歩前に出た。

「三人目の管理者は何者だ？ なぜ封印された？ 三つの聖なる遺物はどこにある？」

エーテリアは、水晶に手を触れた。魔法陣の回転が遅くなり、 hologram（ホログラム）のような映像が浮かび上がった。

「三人目の管理者の名は、ミラベル。かつては、私とヴォルターと共にこの世界を守っていました」

映像の中に、一人の女性が現れた。長い銀髪に、星のような瞳。その姿は、神々しいほどに美しい。

「ミラベルは、『リセット』の権限を持っていました。世界が破滅の危機に陥った時、全てを初期状態に戻すことができる——それが、彼女の役割でした」
「初期状態に戻す？ それは、この世界に生きる人々の記憶も消えるってことか？」

エリナが、恐る恐る聞いた。

「はい。リセットが発動すれば、この世界は創造された時の状態に戻ります。全ての文明、歴史、記憶——消去されます」

エーテリアの声が、沈んだ。

「四百七十二年前、世界は存亡の危機に直面しました。ミラベルは、リセットを決断しました。ですが——」
「ですが？」

カイトが、先を促した。

「私は、それに反対しました。リセットすれば、何百年もの歴史と、無数の命が消える。それは、救済ではなく虐殺だと」

沈黙が、場を支配した。

リリが、静かに口を開いた。

「それで、ミラベルを封印したのですか？」
「いいえ。私は、ミラベルと話し合いました。彼女も、リセットを望んでいたわけではなかった。ただ、他に方法が見つからなかったのです」

エーテリアは、水晶から手を離した。

「ヴォルターが、別の解決策を提案しました。『世界を分割し、問題を封じ込める』と」
「分割？」

リアムが、眉をひそめた。

「はい。この世界を、複数のレイヤーに分けました。表層の世界と、深層の世界。問題の根本原因を深層に隔離し、表層だけを正常に保つ——それが、ヴォルターの計画でした」

カイトの脳裏に、一つの単語が浮かんだ。

「sandbox（サンドボックス）……隔離された環境で問題を封じ込める手法だ」
「その通り。ヴォルターは、この世界をサンドボックス化しました。問題の根幹である『虚無』を深層に封印し、表層の世界を安定させたのです」

エーテリアが、説明を続けた。

「ですが、その代償として、ミラベルのリセット権限は凍結されました。彼女は深層の守り人として、永遠に封印されることになったのです」
「ミラベルは、今どこに？」

カイトが、問いかけた。

「世界のコア——深層の最深部にいます。彼女に会うには、三つの聖なる遺物が必要です」

エーテリアは、 hologramを切り替えた。三つの遺物の映像が浮かび上がる。

// Sacred Artifacts - Trinity Keys
//
// 1. Crystal of Origin (創造の水晶)
//    Location: Frozen Peaks - Temple of First Light
//    Function: Opens the path to World Core
//
// 2. Blade of Severance (分断の剣)
//    Location: Void Abyss - Corrupted Citadel
//    Function: Cuts through dimensional barriers
//
// 3. Mirror of Truth (真実の鏡)
//    Location: Memory Garden - Hidden Sanctum
//    Function: Reveals hidden data and removes corruption
「三つの遺物……それぞれが、危険な場所にあるな」

リアムが、 hologramを睨みながら言った。

「凍てつく山峰、虚空の深淵、記憶の庭園——どれも、普通の場所じゃない」
「はい。各場所には、強力な守護者がいます。そして——」

エーテリアが、言葉を区切った。

「ヴォルターも、遺物を探しています。彼は、緊急プロトコルの発動を阻止しようとするでしょう」
「なぜ？ 世界を救うためのプロトコルだろ？」

カイトが、疑問を投げかけた。

「ヴォルターにとって、秩序こそが正義です。彼は、リセット権限が復活することを恐れています。ミラベルが再びリセットを決断すれば、彼が築き上げた秩序が崩れる——そう考えているのです」

エーテリアの言葉に、カイトは複雑な思いを抱いた。

「つまり、俺たちはヴォルターと競争になるってことか」
「そうです。ですが、ヴォルターは一人ではありません。彼には、アークライトという協力者がいます」

アークライト——かつてカイトたちを追い詰めた敵対者の名前だ。

「アークライトは、ヴォルターに何を求めているんでしょうか？」

セシリアが、問いかけた。

「アークライトは、この世界の破壊を望んでいます。ヴォルターの秩序維持という目的とは矛盾しているはず——」
「彼らは、利用し合っているだけです」

エーテリアが、静かに言った。

「ヴォルターはアークライトの力を利用し、アークライトはヴォルターのリソースを利用する。互いに裏切りを計画しながら、一時的に手を組んでいる——そういう関係でしょう」
「危険な同盟だな」

カイトが、呟いた。

「だが、俺たちには選択肢がない。三つの遺物を集めて、ミラベルに会い、この世界のコードを書き換える——それしか道はない」

エーテリアが、頷いた。

「私は、あなたたちをサポートします。ですが、直接戦うことはできません。管理者の権限には制約があり、ヴォルターに対抗するには——」
「緊急プロトコルが必要、ってことか」
「はい。ミラベルの権限が復活すれば、私は彼女と共にヴォルターの暴走を止められます。そのために——どうか、三人の遺物を集めてください」

カイトは、仲間たちを見渡した。エリナ、リアム、セシリア、リリ、アルフレッド——全員が、決意を表情に宿していた。

「最初の遺物は、凍てつく山峰の『最初の光の神殿』だったな」

カイトが、 hologramを指差した。

「カイト、準備はいい？」

エリナが、彼の手を握った。

「ああ。この世界を救うために——コードを書き換えるために」

カイトは、強く握り返した。

その時、遺跡全体が揺れた。何かが——接近していた。

// WARNING: Unauthorized entity detected
// ID: ARKLIGHT
// Status: Approaching
// ETA: 180 seconds
// Threat Level: CRITICAL
「アークライトだ！」

アルフレッドが、叫んだ。

「逃げて！ ここで戦えば、遺跡が崩れます！」

エーテリアが、警告を発した。

「北の出口から凍てつく山峰へ！ 私が時間を稼ぎます！」

カイトたちは、走り出した。背後で、エーテリアの力が輝き始める。

長い旅の始まりだ。三つの遺物、三人目の管理者、そして世界の命運——全てが、今動き出した。
