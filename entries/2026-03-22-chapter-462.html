第462章：大地の心臓

洞窟の中は、外とはまるで別世界だった。

カイトたちが足を踏み入れた瞬間、松明などの光源を一切持っていないにもかかわらず、周囲が淡い琥珀色の光に包まれた。壁面に埋め込まれた無数の結晶が、地脈の魔力を吸収して発光しているのだ。

「きれい……。まるで星空の下を歩いているみたい」

セシリアが思わず声を漏らした。その言葉通り、洞窟の天井には結晶が散らばり、まるで夜空のような光景を作り出していた。

「でも、気を抜かないで。テラドンさんは試練だと言ったわ。何かが待っているはず」

エリナが警戒心を崩さずに言った。カイトも同感だった。これまでの賢者たちの試練は、どれも甘くはなかった。地の賢者の試練も、例外ではないだろう。

◆ ◆ ◆

洞窟を進むこと十分ほど。唐突に、地面が揺れた。

「地震！？」

リアムが叫んだ。だが、それは普通の地震ではなかった。揺れは、一定のリズムを持っていた。まるで、巨大な何かが鼓動しているかのように。

「これが……大地の心臓？」

カイトがつぶやいた瞬間、前方の通路が崩落した。巨大な岩が落下し、進路を完全に塞いでしまう。

「戻るしかないのか？」

リアムが後ろを振り返った。だが、来た道も崩れ始めていた。前にも後ろにも進めない。四方を岩に囲まれた状況。

「待って。見て」

エリナが崩落した岩山を指差した。その岩の隙間から、微かな光が漏れている。

「あそこから、光が。もしかして、何か道があるのかも」

カイトは近づいて観察した。岩は、ただ崩れ落ちたわけではない。何者かが意図的に配置したような、規則正しい積み方をしている。


function analyzeRockFormation() {
    const rocks = scanRockPositions();
    const pattern = detectPattern(rocks);
    const solutionPath = solvePuzzle(pattern);
    return solutionPath;
}

カイトは解析魔法を発動させた。岩々は、古代のパズルのように組み合わさっていた。特定の順序で岩を動かせば、道が開く仕組みだ。

「これ、パズルだ。岩を正しい順番で押せば、道が開く」

カイトの言葉に、仲間たちは顔を見合わせた。

「正しい順番、って？」

リアムが聞いた。

「地脈の流れに沿っている。魔力が最も強く流れている岩から順に、エネルギーの低い方向へ。そうすれば、エネルギーの流れが整列して道が開く」

カイトの説明は、相変わらず難解だった。でも、彼が指示を出せば、結果はついてくる。それはこれまでの旅で証明されていた。

「一番大きな岩、あれから始めよう」

カイトが中央の巨岩に手を触れた。魔力を流し込むと、岩が低い音を立てて動き出した。地下の奥深くから響くような、重厚な音。

「次は、右の岩。それから、左奥の小さい岩」

カイトの指示に従い、四人が協力して岩を動かした。一つ動かすたびに、地脈の光が変化していく。琥珀色から、深紅へ。そして、青緑色へ。

最後の岩が動いた瞬間、岩山が轟音とともに崩れ落ちた。舞い上がった埃の向こうに、広大な空間が現れた。

◆ ◆ ◆

その場所は、洞窟とは呼べなかった。

地下数百メートルにあるとは思えないほどの広大な空間。天井は見えないほど高く、底には湖が広がっている。だが、それは普通の湖ではない。水面が、発光していた。深いエメラルドグリーンの光を放ち、静かに揺らめいている。

「これが……大地の心臓」

カイトは息を呑んだ。湖の中央には、小さな島があった。その島に、一本の巨木が立っている。ただし、それは普通の木ではない。幹は水晶のように透明で、枝には光る果実が実っていた。

「あそこに行かないと」

エリナが言った。

「でも、どうやって？ 橋もないし、ボートもないわ」

確かに、岸から島までは百メートル以上ある。泳いで渡るには、湖の魔力が強すぎた。

「私が道を作る」

カイトが言った。彼は湖の水面に手をかざし、集中した。


function createIcePath(target) {
    const water = extractWaterElements();
    const temperature = lowerTemperature(water);
    const path = freezePathTo(target);
    return stabilizeStructure(path);
}

カイトの魔法が発動した。湖の水が次々と凍り始め、一本の氷の道が島へと伸びていく。

「すごい……。水魔法と氷魔法を組み合わせたの？」

セシリアが驚嘆の声を上げた。

「水の分子運動を抑制して、熱エネルギーを奪う。物理学的に言えば、エントロピーを局所的に減少させているんだ」

またしても、理解不能な説明。でも、氷の道は確かにそこにあった。

「行こう。急がないと、氷が溶けるかもしれない」

四人は氷の道を渡り始めた。湖の光が、彼らの影を照らす。水底には、巨大な魚の影が見えた。おそらく、この湖に住む守護獣だろう。でも、敵対する様子はない。

◆ ◆ ◆

島に到着したカイトたちは、巨木の前に立った。近くで見ると、その美しさは圧倒的だった。幹の中を、液体のように流れる光。枝に実る果実は、それぞれが小さな太陽のように輝いている。

「これが……何なんだろう」

リアムがつぶやいた。

「ただの木じゃない。何か特別な力を感じる」

カイトは木に近づき、幹に手を触れた。その瞬間、視界が白く染まった。

――フラッシュバック。

カイトは、自分が別の場所にいることを認識した。そこは、この洞窟ではない。もっと高く、もっと広い場所。空の上だ。

眼下には、大陸全体が見えた。七つの光点が、大陸の各所で輝いている。火、水、風、氷、雷、光、闇。そして、中央に八つ目の光点があった。地の属性を表す、琥珀色の光。

『見ろ、これがこの世界の地脈じゃ』

声が響いた。テラドンの声だ。だが、姿は見えない。

『七つの属性は、それぞれが独立しながらも、地脈によって繋がっておる。火が燃えれば、風が運ぶ。水が流れれば、氷が冷やす。すべては、バランスの中に存在するのじゃ』

カイトは、地脈の流れを見た。光の川が、大陸中を巡っている。それは、血液が体内を循環するように、この世界そのものを生かしていた。

『だが、今そのバランスが崩れつつある。虚空の種が、地脈を汚染しておるのじゃ』

カイトの視界が、ある一点に引き寄せられた。大陸の北東。そこに、黒い影が見えた。地脈が、そこだけ汚染されている。光が届かない、暗黒の領域。

『虚空の種は、ただの現象ではない。意志を持っておる。お主がこれまで倒してきた敵たちは、すべて種の分身に過ぎなかったのじゃ』

カイトは戦慄した。種には、意志がある。そして、それはこの世界の敵だ。

『お主は、管理者の血を引いておる。アレクシス・ヴェイルの末裔として、この世界を守る力を持っておるのじゃ』

ビジョンが、急速にフェードアウトしていく。

『大地の心臓で見たものを、忘れるな。それが、お主の戦う理由になるはずじゃ』

◆ ◆ ◆

カイトが意識を取り戻したとき、彼は巨木の根元に座り込んでいた。仲間たちが、心配そうに囲んでいる。

「カイト！ 大丈夫！？」

エリナがカイトの肩を揺さぶった。

「……うん、大丈夫。ちょっと、ビジョンを見ただけ」

カイトはゆっくりと立ち上がった。頭の中に、テラドンの言葉が反響している。

「みんな、聞いて。重要なことが分かった」

カイトは、ビジョンで見たことを簡潔に説明した。地脈の構造、虚空の種の正体、そして自分の血脈について。

「管理者の……末裔？」

リアムが驚いたように聞き返した。

「僕が、この世界の管理者だったなんて。知らなかったよ」
「でも、それで合点がいくわ」

エリナが頷いた。

「カイトがこれまで魔法を使えたのも、賢者たちと対等に戦えたのも、血脈の力だったのね」
「かもしれない。でも、それを自覚したからって、すぐに強くなれるわけじゃない。まだまだ、学ぶべきことは多い」

カイトは巨木を見上げた。果実の一つが、ふわりと落ちてきた。カイトの手の中に、自然に収まる。

「これは……？」

果実は、手の中で温かく光っていた。触れるだけで、魔力が満ちてくる感覚。

『それは、大地の実じゃ。持っていくがよい』

テラドンの声が、洞窟全体に響いた。

『試練を乗り越えた証じゃ。いざというとき、お主の力になるはずじゃ』

カイトは大地の実を大切に懐に収めた。

「ありがとう、テラドンさん。僕たちは、これからどうすればいい？」
『すでに答えは見えておるはずじゃ。虚空の種の本体を探し、それを破壊する。それが、お主の使命じゃ』

テラドンの声が、遠ざかっていった。

『七人の賢者すべての力を借りれば、種の本体に到達できるはず。すでにお主は、四人の賢者と邂逅した。残るは三人。光と闇、そして……』

言葉が途切れた。カイトは耳を澄ましたが、それ以上の声は聞こえなかった。

「残る三人か。光、闇、あと一人は誰なんだろう」

エリナが首を傾げた。

「七大属性だから、光と闇で終わりだと思ってたけど」
「もしかして、第八の属性が存在するのかもしれない」

セシリアが推測を口にした。

「地脈の中央に八つ目の光点があったって言ったわよね。それが何なのか、これから分かるのかも」

カイトは頷いた。謎は尽きない。でも、進むしかない。

◆ ◆ ◆

洞窟を出た一行を、テラドンが出迎えた。

「よく戻ってきた。試練を乗り越えた証、胸に抱いておるな」

カイトは頷いた。

「はい。ありがとうございました、テラドンさん」
「礼には及ばん。むしろ、わしの方が礼を言いたい。若き管理者の誕生を、見届けられたのじゃからな」

テラドンは、巨体を揺らして笑った。

「次は、光の領域へ行くのがよかろう。光の賢者ルミナスは、大陸の西、光の塔に住んでおる。彼女は、この世界の真実を多く知っておるはずじゃ」
「光の塔、か。分かりました」

カイトは仲間たちと顔を見合わせた。

「西へ向かおう。次の賢者に会うために」

テラドンは満足げに頷いた。

「行ってこい。そして、この世界を救うのじゃ。お主になら、できるはずじゃ」

カイトたちは、テラドンに別れを告げ、山を下り始めた。

地の領域での試練は終わった。だが、本当の戦いはこれからだ。

虚空の種の本体。まだ見ぬ敵。そして、自分自身の血脈の秘密。

カイトの冒険は、まだ始まったばかりだった。
