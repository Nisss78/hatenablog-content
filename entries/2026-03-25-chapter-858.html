第858章：炎の使徒との激突

デストラクションが放った無数の火球が、広場の空中に静止した。

赤黒い炎の弾丸。一つ一つが人の頭ほどの大きさがあり、総数は優に百を超えている。それらが一斉にカイトたちを狙い、発射を待っていた。

「可愛い挨拶だろ？ 受けてみな」

デストラクションが指を鳴らすと、火球が一斉に射出された。

「散開！ 各自が防御を！」

カイトが叫ぶと同時に、四人は四方へと散った。

カイトは七属性の盾を展開した。七色の光がドーム状に広がり、迫りくる火球を弾き飛ばす。火球は盾に当たると爆発し、熱波と衝撃を撒き散らした。

「うわっ、熱い！」

エリナは風の障壁を展開し、火球を逸らしていた。しかし、火球の数が多く、防ぎきれないものが地面に着弾し、岩石を溶かしていく。

「溶岩になってる……！？」

リアムは炎纏剣を振るい、火球を斬り払っていた。炎属性を持つ彼にとって、火球の熱はさほど苦にはならない。

「俺には効かねえよ！」

しかし、デストラクションは余裕の笑みを浮かべていた。

「そうだろうな。炎属性の魔法剣士か。だが……」

デストラクションが手を握ると、空中から新たな火球が生まれた。今度は先ほどの三倍の数だ。

「数で圧倒させてもらう」

三百を超える火球が、雨のように降り注いだ。

「無限かよ！」

シオンは地の結界を展開し、テラドンと共に身を守った。結界の上で火球が爆発し続け、琥珀色の光が赤く染まっていく。

「シオン、結界が持たない！」

テラドンが警告した。シオンも額に汗を流し、限界が近いことを悟っていた。

「でも、守らないと……！」

カイトは状況を瞬時に分析した。このまま防戦一方では勝てない。攻撃に転じる必要がある。

「みんな、聞いて！ あいつの本体を狙う！」

カイトがテレパシーで仲間に伝えた。声を出せばデストラクションに読まれる。テレパシーなら一瞬で伝えられる。

四人は微かに頷いた。テレパシーの指示を理解したのだ。

「エリナ、リアム！ 火球を引きつけてくれ！」

エリナが風を操り、火球を渦巻かせた。

「風神の舞・旋風版！」

猛烈な竜巻が火球を巻き込み、デストラクションから遠ざける。火球同士が衝突し、次々と爆発した。

リアムは火球の中を突っ切り、デストラクションへと接近した。

「炎帝の突撃！」

炎纏剣を構え、炎の軌跡を残しながら突進する。

デストラクションはリアムを一瞥し、鼻で笑った。

「炎で炎を倒そうとはな。愚かだ」

デストラクションが手をかざすと、リアムの炎纏剣の炎が逆流し始めた。

「なっ！？」

炎が制御を離れ、リアム自身を焼き焦がそうとする。

「私の炎は、全ての炎を支配する」

デストラクションが拳を握ると、リアムの周囲の炎が爆発した。

「ぐあああっ！」

リアムが吹き飛ばされ、岩壁に激突した。

「リアム！」

エリナが叫んだ。しかし、彼女も火球の猛攻を受け、防戦一方だった。

カイトは焦った。想像以上にデストラクションは強い。炎属性の魔法を完全に支配しているのだ。

「テレパシーだ。カイト、地脈の力を使うぞ」

テラドンからテレパシーが届いた。

「この聖域の地下には、強力な地脈が流れている。それを利用すれば、一時的にデストラクションの動きを封じられるかもしれない」
「やり方を教えてくれ！」
「シオンと二人で、地脈の流れを変える。その間、お前と他の二人で奴を攻撃してくれ」

カイトは頷いた。

テラドンとシオンは広場の端へと移動した。二人は地面に手を置き、詠唱を始めた。

「古の地脈よ、我らの声を聞け……」
「大地の鼓動、今ここに集え……」

二人の詠唱に呼応して、地面が振動し始めた。亀裂が走り、そこから琥珀色の光が漏れ出す。

デストラクションは異変に気づいた。

「何をしている？」

デストラクションがテラドンたちに向かおうとした瞬間、カイトが前に立った。

「相手は俺だ！」

カイトは七属性の魔法を最大出力で展開した。

「七属性・連撃砲！」

七色の光弾が、機関銃のようにデストラクションに向かって放たれた。

デストラクションは火球の壁を作り、光弾を防いだ。しかし、カイトの攻撃は止まらない。

「しつこい小僧だ」

デストラクションが苛立ちを見せた時、地面から岩の柱が次々と突き出した。

「地脈の槍！」

テラドンとシオンの連携技だ。岩の柱がデストラクションの足元から突き出し、彼の体を拘束しようとする。

デストラクションは炎を纏い、岩の柱を溶かした。しかし、柱は次々と現れ、彼の動きを封じようとする。

「くそっ、邪魔な！」

デストラクションは周囲に爆発的な熱波を放ち、岩の柱を一瞬で溶岩化させた。

しかし、その一瞬の隙が、カイトにとっては十分だった。

「今だ！」

カイトは七属性の力を一つに集約した。

// 究極攻撃システム
class UltimateAttack {
    constructor() {
        this.name = "七属性・融合崩壊砲";
        this.elements = ["火", "水", "風", "地", "雷", "光", "闇"];
        this.mode = "完全融合";
        this.target = "デストラクション";
    }
    
    charge() {
        let power = 0;
        for (let element of this.elements) {
            power += this.amplify(element);
        }
        return power * 7; // 融合倍率
    }
    
    amplify(element) {
        return "各属性の最大出力";
    }
}
「七属性融合・崩壊砲！」

カイトの両手から、七色が混ざり合った極太のビームが放たれた。ビームは空間を歪ませながら、デストラクションの胸を直撃した。

ドォォォォォン！！

轟音と閃光が広場を包んだ。

数秒後、光が収まると、デストラクションの姿が変わっていた。

胸に大きな穴が開き、炎の体が不安定に揺らいでいる。しかし、倒れてはいない。

「くっ……いい一撃だ……」

デストラクションは胸の穴を炎で塞ぎ、再生を始めた。

「だが、炎の使徒を倒すには、もっと熱い炎が必要だ」

カイトは舌打ちした。再生能力まであるのか。

テラドンがカイトの横に来た。

「カイト、奴の弱点は再生の源だ」
「再生の源？」
「ああ。奴の体の中心にある炎の核。そこを破壊すれば、再生できなくなる」

カイトはデストラクションの体を見つめた。胸の奥に、赤く輝く核が見える。それが再生の源だ。

「わかった。核を狙う」

その時、デストラクションが笑った。

「聞こえていたぞ。核を狙うとな？ 笑わせるな」

デストラクションは炎の翼を広げ、空中に浮上した。

「空中戦なら、地脈の支援も受けられまい」

カイトは舌打ちした。確かに、テラドンとシオンの地脈支援は地上でしか効かない。

「エリナ！ 飛べるか？」

カイトが叫ぶと、エリナが風の翼を広げた。

「任せて！ 私が運ぶから！」

エリナはカイトの腕を掴み、風の翼で飛び上がった。

「リアム、援護を！」

リアムは炎纏剣を振るい、炎の衝撃波を放った。

「炎帝斬・波動！」

炎の衝撃波がデストラクションに向かい、彼の注意を逸らす。

その隙に、エリナがカイトを乗せてデストラクションの背後に回った。

「今だ、カイト！」

カイトは七属性の力を剣に込めた。実体化した魔法剣が、七色の光を放つ。

「七属性剣・核砕き！」

カイトは剣をデストラクションの背中に突き刺した。剣は炎の体を貫き、核へと届いた。

「ぐああああああっ！！」

デストラクションが苦痛の叫びを上げた。

核が亀裂を入れ、赤い光が漏れ出す。

「これで……終わりだ！」

カイトは剣を回転させ、核を完全に破壊した。

バキキキキキッ！！

核が砕け散ると、デストラクションの体が崩れ始めた。

「ばかな……私が……第六席の私が……」

デストラクションは信じられないという表情で、自らの体が霧散していくのを見つめた。

「次会う時は……もっと強くなっているだろうな……小僧……」

そう言い残し、デストラクションは完全に消滅した。

エリナはカイトを地上に降ろした。

「やったね、カイト！」

リアムとシオン、テラドンが駆け寄ってきた。

「すげえぞ、カイト！ 第六席を倒したんだ！」

リアムが興奮して叫んだ。

しかし、テラドンは表情を曇らせていた。

「いや、まだ終わっていない」
「どういうこと？」
「十三使徒は全部で十三人。第六席を倒しただけだ。それに……」

テラドンは言葉を切り、天井を見上げた。

「奴らは、倒された仲間のデータを共有する。カイトの戦い方は、もう他の使徒たちにも知られているだろう」

カイトは拳を握った。敵は学習する。同じ手は二度と通じない。

「じゃあ、どうすれば……」
「強くなるしかない」

テラドンはカイトの目を見た。

「お前の母さん、アルケミアは、七属性の極意をこの聖域で学んだ。お前にも、それを伝えたい」

カイトは深く頷いた。

「……お願いします。強くなりたい。みんなを守るために」

シオンがカイトの手を握った。

「私も手伝うよ。地の属性の極意、一緒に学ぼう」

リアムとエリナも頷いた。

「俺たちも修行させてもらうよ。次の戦いに備えてな」

テラドンは四人を見渡し、満足げに微笑んだ。

「いい仲間を持ったな、カイト。では、地の聖域での修行を始めよう」

カイトたちは、地の聖域の最深部へと向かった。

そこには、アルケミアが残したという修行の場があった。

母の足跡を追い、カイトは新たな力を目指す。

十三使徒との戦いは、まだ始まったばかりだった。
