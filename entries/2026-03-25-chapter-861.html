第861章：大地の記憶

鏡の怪物を倒した後、地下聖域には重苦しい静寂が戻った。砕けた岩の欠片が床に散らばり、カイトの息遣いだけが反響する。

「休む時間はない」

テラドンが無感情に告げた。その巨体からは先ほどの戦闘の疲れを感じさせない。

「敵は十三使徒だ。奴らが一人を送り込んできたということは、本格的な攻勢の前兆に過ぎん。次はもっと多くが来る」

カイトは頷き、再び座禅を組んだ。地脈との同調を深め、より高度な技術を習得しなければならない。

「テラドン、地脈の記憶を読む技術を教えてください。さっき、この場所の過去が見えましたが、もっと意図的にアクセスしたいんです」

テラドンは腕を組み、暫しの間を置いた。

「よかろう。だが警告しておく。地脈の記憶には、この世界の最も暗い部分も含まれている。お前の精神が耐えられるかどうか」

「耐えます」

カイトの答えに迷いはなかった。管理者を目指す者として、世界の美しさも醜さも等しく受け入れる覚悟が必要だ。

テラドンは岩の玉座から降り、カイトの前に立った。

「地脈の記憶にアクセスするには、三つの段階がある。第一に、過去の映像を見る。第二に、その時の感情を感じる。第三に、その瞬間に存在した全ての情報を理解する」

カイトは頭の中でメモを取った。プログラマーとしての彼にとって、これはデータアクセスのレイヤー構造のように思えた。


// 地脈記憶アクセスのレイヤー
const MemoryLayer = {
    VISUAL: 1,      // 映像層
    EMOTIONAL: 2,   // 感情層
    COMPLETE: 3     // 完全情報層
};

async function accessEarthMemory(layer, timestamp) {
    const vein = await earth.getNearestVein();
    return vein.queryMemory({
        layer: layer,
        timestamp: timestamp,
        depth: 1000
    });
}

「まずは第一段階から始める。この聖域の建造時の記憶を見てみろ」

カイトは目を閉じ、地脈に意識を沈めていった。暗闇の中で、古代の技術者たちが岩を削り、運び、積み上げる姿が浮かび上がる。彼らは魔法と物理的労働を組み合わせ、この壮大な地下都市を建設した。

映像は鮮明だった。職人たちの顔、彼らの会話、作業の音。まるで自分がその場に立ち会っているかのような臨場感。

「見えています……千年前の人々が、この場所を築いている」

「よし。次は第二段階だ。彼らの感情を感じてみろ」

カイトは意識をさらに深く沈めた。すると、職人たちの心が伝わってきた。彼らがこの都市に託した希望。地上の戦乱から逃れ、安全な場所で家族を守りたいという切実な願い。そして完成した時の達成感と誇り。

温かい感情に包まれ、カイトは微笑んだ。

「彼らは……幸せだったんですね。ここを建てた時は」

「そうだ。だが、それだけではない。第三段階へ進め」

カイトは意識を最深部まで沈めた。すると、映像と感情だけでなく、全ての情報が雪崩のように押し寄せてきた。

都市の設計図。使用された魔法の原理。住民一人一人の名前と経歴。彼らの夢、悩み、秘密。膨大なデータがカイトの脳を圧迫する。

「うっ……！」

カイトは頭を抱えた。情報量が多すぎる。データベース全体を一度にダウンロードしようとすれば、どんなシステムでもパンクする。


// エラー処理
try {
    const fullMemory = accessEarthMemory(MemoryLayer.COMPLETE, -1000years);
    process(fullMemory);
} catch (OverflowError) {
    // 情報量過多
    console.error("脳の処理能力を超えました");
    emergencyDisconnect();
}

カイトは意識を強制的に引き戻した。冷や汗が額を伝う。

「無理です……情報が多すぎて、頭が破裂しそうになった」

テラドンは静かに頷いた。

「それが地の魔法の難しさだ。全てを知ろうとすれば精神が崩壊する。だが、賢者は選択的に情報を抽出できる」

「選択的に、ですか？」

「そうだ。具体的なクエリを投げかけるのだ。『千年の間にこの聖域を訪れた敵対者の数』や『最後に大規模な地震が発生した時期』など、必要な情報だけを絞り込んで取得する」

カイトは納得した。全テーブルをスキャンするのではなく、インデックスを活用した効率的なクエリを投げるということだ。

「やってみます」

カイトは再び目を閉じ、今度は慎重に地脈にアクセスした。

過去一年の間に、この聖域の周辺で異常な地脈の乱れがあった場所と時期を教えてくれ

問いは地脈を通じて広がり、瞬く間に応答が返ってきた。三ヶ所の座標と、それぞれの時期。そして、その原因となった魔力の属性まで。


// クエリ結果
const anomalies = [
    { x: 234, y: 89, z: -50, date: "2025-12-15", cause: "void_magic" },
    { x: 567, y: 123, z: -80, date: "2026-01-20", cause: "dark_magic" },
    { x: 890, y: 45, z: -120, date: "2026-02-28", cause: "unknown" }
];

console.log(`異常検知: ${anomalies.length}件`);

「見えました。三ヶ所で異常な魔力の痕跡があります。そのうち二つは虚無と闇、残り一つは原因不明です」

テラドンの表情が曇った。

「原因不明だと？ 具体的に見てみろ」

カイトは三つ目の異常箇所に意識を向けた。そこで見たものは、彼を戦慄させるに十分だった。

地下深く、地脈が複雑に絡み合う交差点。そこに何かが埋め込まれている。黒い結晶のような物体で、周囲の地脈から魔力を吸い取っている。

「何かがあります。黒い結晶で、地脈のエネルギーを吸収しています。これ……敵が仕掛けた何かです」

「場所は？」

「この聖域の真下、約五百メートル」

テラドンは即座に立ち上がり、床を強く踏みしめた。地脈を通じて聖域全体をスキャンする。

「見つけた。これは……魔力吸収装置だ。十三使徒が地脈を弱体化させるために設置したもの」

カイトは息を飲んだ。もし地脈が弱体化すれば、地の魔法を使う賢者の力も低下する。敵は最初から、テラドンを無力化する準備を進めていたのだ。

「除去しないと！」

「待て。罠かもしれん」

テラドンの警告も虚しく、カイトは地中を移動し始めた。仲間を守るため、敵の計画を阻止するため、彼は危険を冒してでも行動する必要があった。

地下五百メートル。カイトは暗闇の中で目を凝らした。地脈の微かな光だけが頼りだ。

そこにあったのは、予想を遥かに超える巨大な装置だった。黒い結晶が無数に連なり、複雑な魔法陣を形成している。地脈から吸収した魔力を集約し、どこかへ送っている。


// 装置の解析
const device = analyzeMagicDevice();

console.log(`エネルギー流量: ${device.energyFlow}/sec`);
console.log(`転送先座標: ${device.destination}`);
console.log(`稼働時間: ${device.activeTime}日`);
console.log(`残存エネルギー: ${device.remainingEnergy}%`);

カイトの解析結果に戦慄が走った。装置は既に三ヶ月間稼働しており、集めたエネルギーはどこか遥か遠くへ転送されている。そして残存エネルギーは九十五パーセントも残っていた。

「これだけのエネルギーが、どこへ送られているんだ……？」

答えを探ろうとした瞬間、装置が反応した。黒い結晶が脈動し、周囲の空間が歪み始める。

「しまった、感知された！」

空間の裂け目から、無数の影が溢れ出した。鏡の怪物の上位種らしく、一匹一匹が明確な意志を持った瞳を光らせている。

カイトは戦闘態勢を整えた。地脈を通じてテラドンに援軍を求めるが、応答がない。装置が魔力を遮断しているのだ。

「一人でやるしかない」

カイトは地脈の残り力量を確認した。装置の影響で、通常の三十パーセント程度しか使えない。だが、それでも彼には戦う意志があった。


// 制限下での戦闘モード

最初の影が襲いかかってきた。カイトは岩の壁を生成して防御し、隙を見て反撃する。だが影の数は多く、次々と押し寄せてくる。

「くそっ、きりがない！」

カイトは焦りを感じ始めた。このままでは消耗戦で負ける。勝つためには、敵の源头である装置を破壊しなければならない。

だが装置に近づくには、影の群れを突破しなければならない。カイトは決意を固めた。

「地脈の一時解放、やってやる！」

彼は自らの体内に眠る地脈との接続を限界まで開いた。通常なら精神崩壊の危険がある行為だが、今は背に腹は代えられない。

金色の光がカイトを包み、彼の体が透過し始めた。地脈そのものと一体化する究極の技。

「これなら――！」

カイトは影を透過し、装置の核心へと到達した。そして全魔力を込めた一撃を、黒い結晶の中心に叩き込む。

轟音と共に装置が爆発した。影たちは悲鳴を上げて消滅し、空間の歪みが修復されていく。

カイトは意識を取り戻したが、体は限界だった。地脈との過剰な同調により、全身に激痛が走る。

「……やった」

彼は崩れ落ちるように座り込んだ。装置は破壊された。地脈のエネルギーは正常に戻り始めている。

その時、地脈から新たな情報が伝わってきた。破壊された装置がエネルギーを送っていた場所の座標だ。


// 転送先の特定
const destination = {

カイトの顔が曇った。十三使徒は、地脈のエネルギーを使って虚無の領域で何かを準備していたのだ。

カイトが聖域に戻ると、テラドンが待っていた。

「無茶をしたな。地脈との過剰同調は、賢者でも避ける禁じ手だ」

「すみません。でも、装置は破壊しました」

テラドンは満足げに頷いた。

「よかろう。今日の修行はここまでだ。お前は地脈の記憶にアクセスし、実戦で応用した。第一段階の習得と見なす」

「第一段階、ですか？」

「そうだ。地の極意には七つの段階がある。お前はその第一を習得したに過ぎん。だが、確かな第一歩だ」

カイトは安堵のため息をついた。まだ先は長いが、確実に前進している。

「テラドン、装置がエネルギーを送っていた場所を特定しました。虚無の領域です」

テラドンの表情が険しくなった。

「虚無の領域か……最高位管理者たちが封印された場所だ。奴らはそこで何を企んでいる？」

答えは出ない。だが一つだけ確かなことは、敵の計画が着々と進行しているということ。そして彼らカイトたちの時間は限られている。

「休む暇はなさそうですね」

カイトが言うと、テラドンは微かに笑った。

「そうだ。明日からは第二段階に入る。地脈を通じて未来を予知する技術だ」

「未来予知……！」

カイトの目が輝いた。もし未来が見えれば、敵の動きを先読みできる。仲間を守る確率は飛躍的に向上する。

地下聖域で、カイトの修行は続く。地上ではエリナたちもまた、それぞれの試練に挑んでいた。

世界を守る戦いは、始まったばかりだ。
