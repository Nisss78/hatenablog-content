第752章：北への航路

朝霧が港を包む中、「氷鳥号」は出航した。

カイトは船尾に立ち、ノヴァ島の街並みが遠ざかっていくのを見つめていた。

「いよいよ北か」

リアムが隣に来た。

「ああ。氷雪の山脈。封印の一つがある場所だ」

カイトは北の空を見上げた。

「船長の話だと、二日かかるんだよね」

シオンが甲板からやってきた。

「そうだ。氷山を避けながら進むから、時間がかかる」

カイトは頷いた。

「氷鳥号」の船長は、ヴィクトルという名の中年男性だった。

髭面で、左目に古い傷がある。長年、北の航路を旅してきた経験の表れだ。

「北の海は気まぐれだ。天候は刻一刻と変わる。油断すると、氷山に激突する」

ヴィクトル船長は警告した。

「でも、この船は大丈夫。氷魔法で強化してるから、氷山にぶつかっても修理できる」
「便利ですね」

エリナが感心した。

「便利だけど、ぶつからないに越したことはない。みんな、見張りを頼むよ」

船長はそう言って、操舵輪に戻った。

昼過ぎ、海の色が変わった。

紺碧の海が、薄い青緑色へ。氷河が溶け出し、淡水が混ざり始めた証拠だ。

「水温が下がってる」

カイトが手を海に浸した。

「冷たい。氷山が近いね」

エリナが震えた。

「氷魔法使いの私でも、この寒さは堪える」

四人は厚手のマントを羽織った。

午後、最初の氷山が現れた。

白く輝く巨大な氷の塊が、海面から突き出している。

「きれい……」

シオンが息を呑んだ。

「でも、危険だ。水面上に出ているのは、全体の一割程度。残りは海中にある」

リアムが説明した。

「船が氷山の近くを通ると、海中の氷に底を擦られることがある」
「だから、距離を取らないといけないんですね」

カイトは理解した。

日が沈む頃、「氷鳥号」は小さな入江に停泊した。

「夜は進まない。暗いと氷山が見えないからな」

ヴィクトル船長が言った。

「ここなら、波も静かだ。ゆっくり休んでくれ」

カイトたちは船室に入った。

船室は狭いが、清潔だった。

四つのベッドが並び、小さな窓から夜空が見える。

「カイト、明日のことだけど」

リアムが切り出した。

「氷雪の山脈に着いたら、どうする？」

カイトは地図を広げた。

「まずは、山脈の入り口にある村に行く。そこで情報を集めて、フローズナさんの居場所を探す」
「フローズナって、氷の賢者だよね。以前会ったことあるの？」

シオンが尋ねた。

「一度だけ。封印を守るために協力してもらったことがある」

カイトは思い出した。

「彼女は厳しい人だけど、良い人だ。封印の重要性を理解してくれているはず」

夜、カイトは甲板に出た。

夜空には、無数の星が輝いている。

北の空には、緑と紫の光の幕が揺れていた。

「オーロラ……」

カイトは見とれた。

「きれいだね」

エリナが隣に来た。

「うん。日本では、こんな光景を見られなかった」

カイトは呟いた。

「カイトの世界、日本ってどんなところ？」

エリナが尋ねた。

カイトは少し考えた。

「魔法がない世界。その代わり、科学が発展してる」
「科学？」
「魔法と似たようなものかな。自然の法則を理解して、技術に応用する」

カイトは説明した。

「僕は、プログラミングっていう技術を学んでた。魔法のコードを書くのに、役立ってる」
「へえ。カイトの世界も、面白そう」

エリナは微笑んだ。

「いつか、カイトの世界を見てみたいな」
「……戻れるかな」

カイトは夜空を見上げた。

「この世界に来てから、戻る方法を探してない。でも、いつか戻れる日が来るかもしれない」

エリナはカイトの手を握った。

「カイトが戻るとしても、私、ついていくから」
「エリナ……」
「カイトと離れたくない。どの世界でも、一緒にいたい」

エリナの瞳が、星明かりに輝いた。

「ありがとう。僕も、エリナと一緒にいたい」

カイトはエリナを抱き寄せた。

オーロラの下、二人は静かに寄り添った。

翌朝、「氷鳥号」は再び出航した。

空は快晴。でも、気温はさらに下がっていた。

「寒い……」

シオンが震えた。

「風魔法で、温かい風を送れる？」

カイトが提案した。

「やってみます」

シオンは集中した。


// 温風生成魔法
function warmWind(area, temperature) {
  const magic = {
    type: "wind",
    subtype: "temperature_control",
    target_area: area,
    target_temperature: temperature,
    duration: "持続的"
  };
  
  return `${area}の温度を${temperature}度に調整`;
}

// 実行
warmWind("船上全域", 20);

温かい風が吹き始め、船内の温度が上がった。

「すごい。シオン、上手くなったね」

カイトが褒めた。

「カイトさんの教え方がいいんです」

シオンは照れ笑いした。

昼過ぎ、海面に異変が起きた。

波が高くなり、船が揺れ始めた。

「何だ！？」

リアムが叫んだ。

「海中に、何かいる！」

ヴィクトル船長が叫んだ。

「氷海の魔物だ！　この海域に生息する、危険なやつだ！」

海面が割れ、巨大な影が現れた。

それは、氷でできた巨大な蛇だった。

体長は二十メートル以上。鱗は氷の結晶、目は青白く光っている。

「氷海蛇（アイスシーサーペント）」

カイトが識別した。

「氷属性の魔物。性格は凶暴。船を襲って、乗組員を食べる」
「やるしかない！」

リアムが剣を抜いた。

氷海蛇が船に向かって突進してきた。

「エリナ、氷魔法で動きを止めて！」

カイトが指示した。

「分かった！」

エリナは両手を前に出した。


// 氷結魔法
function freeze(target, duration) {
  const magic = {
    type: "ice",
    subtype: "binding",
    target: target,
    strength: "強",
    duration: duration
  };
  
  return `${target}を凍結`;
}

// 実行
freeze("氷海蛇の体幹", "30秒");

エリナの魔法が発動し、氷海蛇の体が凍りついた。

「今だ！　リアム！」

カイトが叫んだ。

リアムは船から飛び出し、凍った氷海蛇に斬りかかった。

「はあああっ！」

剣が氷の鱗を切り裂く。

でも、氷海蛇は再生した。氷が再び繋がり、傷が塞がる。

「再生能力があるのか！」

リアムが舌打ちした。

「物理攻撃だけじゃ、倒せない！」

カイトは考えた。

（氷属性の魔物なら、弱点は火属性。でも、ここは船上。火魔法を使うと、船も燃える危険がある)

カイトは別の方法を思いついた。

「シオン、雷魔法で神経を焼くんだ！」

カイトが指示した。

「雷魔法？　でも、海の上だと……」
「大丈夫。対象を限定すれば、海には影響しない」

カイトはシオンを励ました。

シオンは深呼吸した。

「やります」

シオンは両手を天に向けた。


// 雷撃魔法
function lightningStrike(target, precision) {
  const magic = {
    type: "lightning",
    subtype: "precision_strike",
    target: target,
    accuracy: precision,
    damage_type: "内部神経焼灼"
  };
  
  return `${target}の神経系を破壊`;
}

// 実行
lightningStrike("氷海蛇の脳", "100%");

空から稲妻が降り注ぎ、氷海蛇の頭部を直撃した。

氷海蛇が悲鳴を上げた。

青白い目が消え、体が崩れ落ちる。

海に落ちた氷の塊は、波間に消えた。

「やった……」

シオンが膝をついた。

「シオン、すごいよ！」

エリナが駆け寄った。

「カイトさんの指示が的確だったからです」

シオンは弱々しく微笑んだ。

ヴィクトル船長が安堵の息を吐いた。

「助かったよ。君たち、本当に冒険者か？　実力が違う」
「管理者の仕事で、色々な戦いを経験してるから」

カイトは控えめに答えた。

「それに、仲間が優秀だ」

カイトは仲間たちを見回した。

「リアムの剣技、エリナの氷魔法、シオンの雷魔法。どれも、欠かせない」

三人は照れた。

夕方、海の向こうに陸地が見えた。

白く輝く山脈。氷雪の山脈だ。

「着いた！」

シオンが指差した。

「あれが、氷雪の山脈。封印がある場所」

カイトは緊張した。

「明日、上陸する。準備を整えておこう」

夜、カイトは船室で日記をつけていた。

「今日の戦い、危なかったな」

カイトは独り言を言った。

「でも、みんな無事だった。それが一番大事」

カイトは日記を閉じた。

（明日は、山脈に入る。封印を確認して、フローズナさんに会う。そして、管理者の塔への道を聞く)

窓の外では、再びオーロラが輝いていた。

カイトは眠りについた。

明日の冒険に備えて。

氷雪の山脈。氷の賢者。そして、封印の真実。

カイトの旅は、続く。

第752章、了。
