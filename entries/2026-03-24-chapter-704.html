第704章：虚無の深淵

爆発的な閃光が虚空の城を包み込んだ。

四人の渾身の一撃を受けたダークハルトの体が、大きく後方へ弾き飛ばされた。彼の漆黒の鎧に亀裂が入り、そこから漏れ出す虚無の力が不安定に揺らめく。

カイトは荒い息を吐きながら、仲間たちの状態を確認した。エリナは膝をついていたが、すぐに立ち上がった。リアムは剣を杖代わりにして体を支えている。シオンとアルケミアは、互いに背中を合わせて警戒態勢を維持していた。

「効いたか？」

リアムが期待を込めて問うた。

煙が晴れ、ダークハルトの姿が現れた。彼の体には確かな損傷があった。しかし、その瞳には依然として圧倒的な殺意が宿っていた。

「良い一撃だった。だが――」

ダークハルトが手を掲げると、傷口から漏れ出していた虚無の力が逆流し始めた。傷が急速に修復されていく。

「虚無を統べる者に、永遠の死などない」

カイトは舌打ちした。再生能力か。厄介だ。

その時、ミラージュが冷ややかな声で割り込んできた。

「あなたの書き換え、興味深かったわ。でも、一時的なもの」

彼女の鏡像が再び安定し始めていた。カイトのシステム改変は、永続的なものではなかったらしい。

「予言は再び明瞭になりつつある」

アルタイルが静かに告げた。

「あなたたちの勝率、現在百分之七。絶望的と言わざるを得ない」

エリナがカイトの横に並んだ。

「カイト、どうする？」

カイトは冷静に状況を分析した。三人の最高位管理者は、まだ完全には体勢を立て直していない。今が最大の好機だ。

「攻め続けるしかない。守りに入ったら負けだ」

アルケミアが頷いた。

「同意するわ。彼らの連携が完全に復旧する前に、個別撃破を狙いましょう」

シオンが歩み出た。

「私がミラージュを引き受ける。七属性の魔法回路なら、彼女の鏡像を相殺できる」

カイトは彼女の決意を目にして、頷いた。

「頼む。アルケミアさんはアルタイルを。俺とエリナとリアムでダークハルトを叩く」

役割分担が決まった瞬間、カイトたちは散開した。

シオンは真っ直ぐにミラージュへ向かった。

七属性の魔法回路が、彼女の体の中で激しく回転していた。火、水、風、地、光、闇、虚無。七つの属性が渾然一体となり、彼女の周囲に圧倒的な力の場を形成する。

「来なさい、娘よ」

ミラージュが挑発的に手招きした。彼女の周囲に、無数の鏡像が現れる。

「あなたの攻撃は、すべて私の鏡像に映し出される。無意味よ」

シオンは答えず、魔法を放った。火の矢が、ミラージュの鏡像に向かって殺到する。

予想通り、鏡像が火の矢を反転させ、シオンへと跳ね返した。

しかし、シオンは動じなかった。彼女は即座に水の盾を展開し、反転した火の矢を受け止めた。

「読んでたの？」

ミラージュが驚きの声を漏らした。

シオンは淡々と告げた。

「鏡像の原理は理解している。私の攻撃が反転するなら、それも含めて計算すればいい」

彼女は再び魔法を構築した。今度は、七つの属性を同時に放つのではなく、順番に連続で撃ち出した。

火、水、風、地、光、闇、虚無。

七つの属性が、絶妙なタイミングでミラージュに向かう。鏡像が一つ一つの攻撃を反転させようとするが、連続する属性変化に対応しきれない。


// Attribute Sequence Attack
// Counter-Mirage Protocol

class AttributeSequenceAttack {
    constructor() {
        this.sequence = ['fire', 'water', 'wind', 'earth', 'light', 'dark', 'void'];
        this.timing = 0.1; // seconds between each element
    }
    
    execute(target) {
        for (let element of this.sequence) {
            this.cast(element, target);
            this.wait(this.timing);
        }
    }
    
    cast(element, target) {
        const spell = new ElementalSpell(element);
        spell.setPower('maximum');
        spell.target(target);
        spell.fire();
    }
}

ミラージュの鏡像が、属性の連続変化に対応できずに崩れ始めた。

「くっ……！ 何この攻撃パターン！」

シオンは間髪入れずに、最大魔法を放った。

「七属性融合――プリズム・エクリプス！」

七色の光が一本の巨大な光束となり、ミラージュを貫いた。彼女の鏡像が完全に砕け散り、本体がよろめく。

「これで、あなたの鏡像は作れないはず」

ミラージュが苦痛に顔を歪めた。

「まだよ……まだ終わらない！」

彼女は残存する力を振り絞り、虚空の城そのものを利用しようとした。城の壁面が鏡のように輝き、無数の反射面を形成する。

しかし、その時だった。

アルケミアがアルタイルとの戦いの中で、一つの魔法を放った。

「調律、発動！ 境界を安定化！」

彼女の魔法が、城そのものの状態を変化させた。鏡面の壁が曇り、反射能力を失う。

「お母さん！」

シオンが母の支援に気づき、感謝の声を漏らした。

ミラージュは、最大の武器を失った。

一方、アルケミアとアルタイルの戦いは、静かながらも激しいものであった。

アルタイルの予言能力は、未来を知覚し、最適な対応を事前に選択できる。彼との戦いは、常に一手先を読まれているのと同じだった。

「無意味だ」

アルタイルが淡々と告げた。

「あなたの攻撃は、すべて予知済み。回避は容易」

アルケミアは微かに笑った。

「予知できるのは、確定した未来だけ。確率論的な未来はどう？」

彼女が放った魔法は、着弾点が確率的に変動する特殊なものだった。対象領域内のどこに着弾するか、攻撃者本人にも予測できない。


// Probabilistic Attack
// Anti-Prophecy Protocol

class ProbabilisticAttack {
    constructor() {
        this.targetArea = { x: 0, y: 0, radius: 100 };
        this.probabilityDistribution = 'uniform';
    }
    
    execute() {
        const impactPoint = this.randomizeImpact();
        const spell = new VoidDisruptor();
        spell.target(impactPoint);
        spell.fire();
    }
    
    randomizeImpact() {
        // Even the caster cannot predict where it lands
        const angle = Math.random() * 2 * Math.PI;
        const distance = Math.random() * this.targetArea.radius;
        return {
            x: this.targetArea.x + Math.cos(angle) * distance,
            y: this.targetArea.y + Math.sin(angle) * distance
        };
    }
}

アルタイルの表情に、初めて困惑が浮かんだ。

「これは……予知できない」

彼の予言能力は、因果関係を辿ることで未来を知覚する。しかし、因果関係そのものが確率的な場合、予知は機能しない。

アルケミアは間髪入れずに、連続で確率攻撃を放った。

「第一の調律者の力、見せてあげる」

無数の着弾点が、アルタイルの周囲で爆発した。彼は回避行動を取ろうとしたが、攻撃の軌道が予測できないため、効果的な回避ができない。

「くっ……！ 予言が……曇る……」

アルタイルが膝をついた。彼の最大の武器である予言能力が、完全に封じられたのだ。

カイト、エリナ、リアムの三人は、ダークハルトに集中攻撃を仕掛けていた。

しかし、最高位管理者の中でも最強の実力を持つ彼は、容易には崩れなかった。

「虚無の王に挑むとは、愚かな」

ダークハルトが手を振ると、巨大な虚無の球体が形成された。それは、触れるものすべてを無に還す絶対破壊の球体。

「避けて！」

エリナが叫び、三人は四方に散開した。球体が通過した場所は、空間ごと消滅していた。

カイトは即座に分析した。


// Void Sphere Analysis

class VoidSphereAnalysis {
    constructor(target) {
        this.target = target;
        this.threatLevel = 'EXTREME';
    }
    
    analyze() {
        return {
            speed: 0.8, // relative to light speed
            destructionRadius: 50, // meters
            countermeasures: ['avoid', 'redirect', 'contain']
        };
    }
    
    findWeakness() {
        // Void spheres require constant mana input
        // If connection to caster is severed, sphere collapses
        return {
            type: 'mana_link',
            location: this.traceManaFlow(this.target)
        };
    }
}

カイトは見えないマナの流れを追跡した。ダークハルトと虚無の球体を繋ぐ、目に見えない管のようなもの。

「リアム、あの球体の動きを変えられるか？」

リアムは即座に理解した。

「風で軌道を逸らすってことか？」
「いや、もっと直接的に。球体とダークハルトを繋ぐ線を切断したい」

リアムは頷き、剣に風の力を集中させた。

「やってみる」

彼は一瞬で距離を詰め、見えない管の通過する空間を、風の刃で切断した。

その瞬間、虚無の球体が不安定に揺らぎ、爆発した。自壊したのだ。

「何！？」

ダークハルトが驚愕の声を漏らした。

カイトはその隙を逃さなかった。

「今だ、エリナ！」

エリナが最大魔法を放った。氷と水を融合させた、絶対零度の攻撃。

「氷結魔法――アブソリュート・ゼロ！」

ダークハルトの体が、瞬時に氷結した。虚無の力を持ってしても、絶対零度の凍結は即座には解除できない。

カイトは追撃を加えた。


// Finishing Blow
// Administrator Judgement

class AdministratorJudgement {

const judgement = new AdministratorJudgement(darkhart);
judgement.execute();

轟音が響き、衝撃波が周囲を襲った。

煙が晴れた時、そこには――

ダークハルトの体は、半ば消滅していた。

彼の鎧は粉々に砕け、体の半分以上が虚無の霧となって散っていた。しかし、まだ完全には滅びていない。

「馬鹿な……私が……この私が……」

ダークハルトが信じられないという表情で、自分の体を見下ろした。

カイトはゆっくりと彼に近づいた。

「終わりだ、ダークハルト。境界への攻撃を止めろ」

ダークハルトは不敵に笑った。

「私が倒れても、境界の崩壊は止まらない。既に始まっているのだから」

カイトは周囲を見回した。確かに、空間の亀裂は広がり続けていた。三人の最高位管理者が行っていた攻撃は、彼らが倒れても慣性で続いている。

「なら、俺が修復する」

カイトは両手を広げ、管理者としての力を解放した。


// Boundary Restoration
// World Integrity Repair

class BoundaryRestoration {
    constructor() {
        this.targetIntegrity = 1.0; // 100%
        this.method = 'ADMINISTRATOR_OVERRIDE';
    }
    
    scan() {
        return this.detectBreaches();
    }
    
    repair(breaches) {
        for (let breach of breaches) {
            this.seal(breach);
        }
    }
    
    seal(breach) {
        const seal = new AttributeSeal(['void', 'light', 'time']);
        seal.apply(breach.location);
        this.reinforce(breach.location);
    }
}

const restoration = new BoundaryRestoration();
const breaches = restoration.scan();
restoration.repair(breaches);

カイトの体から、三色の光が放たれた。虚無、光、時。三つの属性が融合し、空間の亀裂を塞いでいく。

境界の崩壊が、徐々に止まっていった。

「シオン、アルケミアさん！ 手伝ってくれ！」

二人が頷き、カイトに力を貸した。三人の管理者の力が重なり、境界の修復が加速する。

エリナとリアムは、無力化された三人の最高位管理者を見張っていた。

そして、ついに――

空間の亀裂が、完全に塞がった。

「終わった……のか？」

リアムが安堵の声を漏らした。

しかし、カイトは警戒を緩めなかった。

ダークハルトが、最後の力を振り絞って立ち上がった。

「ほめるぞ、新たなる管理者。私をここまで追い詰めたのは、あなたが初めてだ」

彼の体は、今にも消滅しそうだった。しかし、その瞳には狂気が宿っていた。

「だが、これは始まりに過ぎない。虚無の深淵は、あなたの内側にも存在する」

カイトは眉をひそめた。

「どういう意味だ？」

ダークハルトは笑った。

「あなたは虚無の力を行使する管理者。その力は、いつかあなた自身を飲み込む。その時、あなたは私と同じになる」

カイトは首を振った。

「その運命も、俺が書き換える」

ダークハルトの体が、ついに完全に霧散した。

ミラージュとアルタイルも、同様に力を失い、消滅していった。

戦いは終わった。

しかし、カイトの心には、ダークハルトの最後の言葉が残っていた。

虚無の深淵は、あなたの内側にも存在する。

カイトは虚空の城を見上げた。境界は修復され、空間の歪みは消えた。しかし、城そのものはまだ存在していた。

エリナがカイトの横に並んだ。

「カイト、大丈夫？」

カイトは頷いた。

「ああ。ただ、疲れただけだ」

それは嘘だった。内側で、虚無の力がざわめいていた。管理者の力を使いすぎた代償。

アルケミアが近づいてきた。

「カイト、気をつけて。ダークハルトの言葉、あながち嘘ではないわ」

カイトは彼女を見た。

「管理者の力には、代償があるのか？」

アルケミアは静かに頷いた。

「虚無の力を使えば使うほど、内なる虚無が膨らむ。それを抑え込むには、強い意志と、何より――絆が必要よ」

カイトは仲間たちを見回した。エリナ、リアム、シオン、アルケミア。彼らの存在が、内なる虚無を抑え込んでくれている気がした。

「分かった。気をつけるよ」

シオンが母に向き直った。

「お母さん、これからどうするの？」

アルケミアは優しく微笑んだ。

「あなたと一緒にいたいけれど、管理者としての役目があるわ。境界の監視を続けないと」

シオンは少し寂しそうな表情を浮かべたが、すぐに納得したように頷いた。

「うん、分かってる。また会えるよね？」

アルケミアは娘を抱きしめた。

「いつでも会えるわ。管理者同士、空間を超えて繋がっているから」

母娘の再会は短かったが、二人の間に確かな絆が生まれたのは間違いなかった。

カイトは仲間たちに告げた。

「戻ろう。長い戦いだった」

虚空の城を後にしながら、カイトは空を見上げた。

境界は修復された。最高位管理者たちは倒れた。

しかし、戦いは終わっていないのかもしれない。

内なる虚無との戦い。それが、カイトの次の課題だった。
