第105章：記憶の回廊

始原の図書館の入り口は、カイトが想像していたものとは全く違っていた。

巨大な塔の基部にあるのは、華奢なアーチ型の扉一つだけ。装飾もなく、 guard もいない。ただ、古びた木の扉が、静かにそこにあった。

「これが入り口？ 隠し扉とか、もっと荘厳なやつを想像してた」

リアムが拍子抜けしたように呟く。カイトも同感だった。世界的な規模のアーカイブなら、もっとセキュリティが厳重でもおかしくない。

「外見に騙されてはいけませんわ」

リリが杖の先端で扉を三回叩いた。コツ、コツ、コツ。乾いた音が、霧の立ち込める空間に響き渡る。

「始原の図書館は、内側にこそ真の姿がありますの。この扉は、単なる接続ポートに過ぎませんわ」

接続ポート。カイトはその言葉に、データベースへのエントリーポイントを連想した。確かに、大事なのは中身だ。入り口が質素なのは、ある意味で合理的だ。

扉が、音もなく開いた。中から漏れ出したのは、淡い金色の光。

「ようこそ、始原の図書館へ」

リリが優雅に一礼し、先導を始めた。

扉の向こうに広がっていたのは、巨大なエントランスホールだった。

天井は見えない。壁も見えない。ただ、果てしなく続く本棚の列が、螺旋状に空へと伸びている。本棚の一つ一つが、ビルの階数ほどの高さがある。その間を、無数の浮遊する光球が行き交っていた。

「これが……図書館？」

セシリアが、呆然と呟いた。

「本の数だけでも、億単位はあるわ。全部を読むのに、人間の寿命じゃ足りない」
「本来の目的は、読むことではありませんわ」

リリが、歩きながら答えた。

「始原の図書館は、情報への『アクセス』を提供する場所ですの。必要な知識を、必要なだけ引き出す。それが正しい使い方」

カイトは納得した。データベースと同じだ。すべてのレコードを舐めるのではなく、クエリを投げて必要な情報だけを取得する。それが効率的な使い方だ。

// Library Access Protocol
// Query: Knowledge about WorldDesign
// Access Level: Restricted
// Authentication Required: Yes

カイトの視界に、またステータス画面が浮かび上がる。アクセス制限があるらしい。当然だ。世界の設計図など、誰でも見られるようにしてはいけない。

「リリ、僕たちが探しているのは、世界の設計図についての情報だ。どこにある？」

リリは少し考え込むように首を傾げた。

「設計図……それを直接見ることはできませんわ。アクセス権限が高すぎますの」
「じゃあ、どうすればいい？」
「まずは、認証をクリアしていただく必要がありますわ」

リリが立ち止まり、前方を指差した。そこには、三つの扉が並んでいた。それぞれに、異なる色の光が灯っている。左から順に、青、赤、白。

「三つの試練ですわ。それぞれ、異なる資質を問うもの。青は『知恵』、赤は『勇気』、白は『記憶』」
「全部やる必要があるのか？」
「いいえ。一つ選んでくださいまし。ただし……」

リリの表情が、少しだけ曇った。

「どの試練を選んでも、容易ではありませんわ。特に『記憶』は……」

そこで言葉を切り、リリはカイトの方をじっと見た。その瞳には、どこか心配そうな色が宿っていた。

「あなたにとって、最も辛いものになるかもしれませんわ」

カイトは、三つの扉を見つめた。知恵、勇気、記憶。

知恵の扉を選べば、論理的な試練が待っているだろう。プログラマーとしての彼なら、得意分野かもしれない。

勇気の扉なら、戦いが待っているはずだ。剣技には自信はないが、魔法でカバーできる。

だが──。

「僕は、記憶を選ぶ」

カイトの言葉に、全員が驚いたようだった。

「カイト、本当にいいの？」

エリナが、心配そうに問いかける。

「リリの言い方、なんか引っかかるんだけど……」
「大丈夫だ。理由がある」

カイトは、白い光を放つ扉の方を向いた。

「僕は、自分が何者なのかを知りたい。この世界に召喚された理由。僕の能力の本質。それを知るには、過去を振り返る必要があるんだ」

エリナは少し躊躇った後、小さく頷いた。

「わかった。あなたがそう決めたなら、ついていくわ」
「僕も行くよ」

リアムが、剣の柄に手をかけた。

「何が待っていようと、一緒に乗り越える」
「私もです」

セシリアが、魔導書を抱えしめた。

「学術的な興味も……いえ、それ以上に、仲間として」
「私も同行を」

アルフレッドが、静かに一礼した。

「守人として、カイト殿を護るのが私の務め」

カイトは、仲間たちの顔を見渡した。心強い。本当に、心強い。

「感謝する。みんな」

彼は、白い扉に手をかけた。冷たい感触。そして──。

世界が、反転した。

カイトが目を開けると、そこは見知らぬ場所だった。いや、見知っている場所かもしれない。デジャヴのような、懐かしさのような、奇妙な感覚。

そこは、小さな部屋だった。壁には漫画のポスター。机にはパソコン。床には散らばった本とゲームソフト。

カイトの心臓が、早鐘を打った。

「これは……僕の部屋？」

かつて住んでいた、日本のアパートの一室。間違いない。棚に並んだ技術書、モニターに映ったコード、飲みかけのコーヒー。すべてが、記憶と一致する。

だが、何かが違う。

部屋の中央に、一人の少年が座っていた。年齢は十五歳くらい。黒髪で、少しやつれた顔つき。パソコンの画面を、熱心に見つめている。

「……僕？」

カイトは、息を呑んだ。目の前にいるのは、過去の自分だ。高校生の頃の、海藤翔太。

少年は、キーボードを打ち続けていた。画面には、複雑なコードが流れている。

// Personal Project: World Simulator
// Version: 0.1.3
// Status: Debugging Phase
// Current Issue: Memory leak in rendering engine

カイトは、その光景を静かに見つめていた。高校時代、彼は「世界シミュレータ」というプロジェクトに取り組んでいた。仮想世界を作り出し、そこで物理法則や生態系をシミュレートする。壮大な夢だった。

だが、完成しなかった。バグが多すぎて、結局断念したのだ。

「……懐かしいな」

カイトが呟くと、過去の彼が顔を上げた。だが、視線はカイトには向けられていない。彼は、画面の向こうにある何かを見ている。

「なんで、動かないんだよ……」

少年の声が、カイトの胸を締め付けた。焦り、悔しさ、そして孤独。当時の感情が、鮮明に蘇ってくる。

「これが完成したら、誰も僕をバカにできなくなるのに……」

少年は、机に突っ伏した。肩が、小さく震えている。

カイトは、過去の自分に手を伸ばそうとした。だが、指先が少年の肩に触れる直前で、手が止まった。触れられない。これは記憶だ。干渉できない。

「そうだ……この頃の僕は、ずっと一人でコードを書いていたんだ」

友達はいなかった。クラスでは浮いた存在。唯一の救いは、プログラミングという「逃げ場所」だけ。

画面が切り替わった。今度は、大学の講義室。カイトは、後ろの席で居眠りをしている。黒板には、難解な数式。教授の声が、遠くで響く。

「海藤君」

名前を呼ばれ、カイトが顔を上げた。

「はい……」

「君、この授業についてこられてるのかね？」

周囲の学生が、クスクスと笑う。カイトは、顔を伏せた。悔しかった。でも、反論できなかった。確かに、彼は授業についていけていなかった。夜遅くまでゲームを作って、朝は起きられない。そんな生活の繰り返しだった。

「……情けないな」

カイトは、現在の自分の姿を呟いた。情けない。でも、それが彼の過去だ。逃げられない。

場面がまた変わる。今度は、小さな café。カイトは、向かいの席に座る女性と会話している。

「翔太君、今回のプロジェクト、すごく上手くいったね」

女性は、笑顔で言った。年齢は二十代後半。眼鏡をかけた知的な雰囲気。カイトの最初の職場の先輩だ。

「……先輩のおかげです」

「謙遜しないの。君の設計があったから、このシステムが動いたんだよ」

女性は、温かい目でカイトを見た。

「翔太君、君には才能がある。もっと自信を持ちなさい」

その言葉に、カイトは救われた気がした。初めて、誰かに認められた。初めて、自分に価値があると感じた。

「……先輩」

カイトは、その名前を呟いた。名前は言わないでおこう。彼女は、今はもう会えない。彼女は、カイトが異世界に来る少し前に、別の会社に転職したのだ。

「ありがとう……」

カイトは、過去の自分に向けて呟いた。ありがとう、先輩。あなたの言葉が、僕を支えてくれた。

場面が、また変わる。

今度は、夜の路上。雨が降っている。カイトは、コンビニの袋を持って、アパートに向かって歩いていた。

「……あ」

カイトが、足を止めた。前方に、何かが見える。光る、亀裂のようなもの。空間に、亀裂が入っている。

「なんだ、これ……？」

彼が近づくと、亀裂から光が溢れ出した。そして、声が聞こえた。

『見つけた』

女性の声。いや、もっと超越的な存在の声。

『海藤翔太。君の能力は、この世界では勿体ない』

「え……？」

『我々の世界へ来ないか？ 君のプログラミング能力は、魔法として応用できる』

カイトは、その言葉を聞いた瞬間、背筋が震えた。異世界転生の誘いだ。小説でしかありえないと思っていたことが、現実に起きている。

「……行きます」

彼は、迷わず答えた。この世界には、もう未練はない。むしろ、新しい世界で一から始められるなら、それの方がマシだと思った。

「ここじゃ、僕は誰にも必要とされていない。でも、もし向こうで僕の能力が役に立つなら──」

光が、彼を包み込んだ。意識が遠のく。最後に見たのは、雨に濡れたアスファルトだった。

カイトは、息を吐き出した。

目の前の光景が消え、元の図書館のエントランスに戻っていた。リリと仲間たちが、心配そうに彼を見ている。

「カイト、大丈夫？」

エリナが、彼の手を握った。冷たい手だった。

「……ああ。大丈夫だ」

カイトは、目元を拭った。気づかないうちに、涙が流れていたらしい。

「過去を、見たんだ。僕の」
「辛かった？」

エリナの問いに、カイトは少し考えてから首を振った。

「辛かったけど……必要だった。僕は、自分が何者かを忘れかけていたから」

彼は、仲間たちの顔を見渡した。エリナ、リアム、セシリア、アルフレッド。そして、案内人のリリ。

「僕は、孤独なプログラマーだった。誰にも必要とされないと思っていた。でも、この世界に来てから、僕は仲間を得た。目的を得た。それが、何よりも大事だと気づいたんだ」

エリナが、彼の手を強く握りしめた。

「私たちは、あなたの仲間よ。どこにいたって、そうよ」
「ああ。わかってる」

カイトは、小さく笑った。

「だから、前を向く。過去に囚われるんじゃなく、未来を作るために」

リリが、満足そうに頷いた。

「試練、クリアですわ。あなたは、自分自身を受け入れることができました」
「これで、設計図にアクセスできるのか？」
「残念ながら、まだですわ」

リリは、少し申し訳なさそうに言った。

「設計図へのアクセス権限は、さらに上のレベルで管理されていますの。次は、『知恵』か『勇気』の試練をクリアする必要がありますわ」
「だと思ったよ」

カイトは苦笑した。一筋縄ではいかないらしい。

「だが、無駄じゃなかった。自分を取り戻せたからな」

彼は、拳を握りしめた。準備はできている。何が待っていようと、乗り越えてみせる。

「さて、次はどっちにする？ 知恵か、勇気か」

カイトが仲間たちに問いかけると──

「知恵でしょ、あなたなら」

エリナが、即答した。

「戦うのは私に任せて。あなたは、頭を使って」

カイトは、苦笑しながら頷いた。確かに、彼女の言う通りだ。

「わかった。知恵の試練に行こう」

一行は、青い光を放つ扉へと向かった。その背中を、リリが静かに見送っていた。

「……頑張ってくださいまし、異邦人の方」

リリの呟きは、誰にも聞こえなかった。ただ、本棚の隙間を舞う光球だけが、彼女の言葉を受け止めていた。

世界の真実へ、もう一歩近づいた。

【用語解説】

  • 記憶の回廊：始原の図書館の試練の一つ。訪問者の過去を具現化し、自己受容を促す。
  • 世界シミュレータ：カイトが高校時代に制作していた個人プロジェクト。完成には至らなかった。
  • 召喚の瞬間：カイトが異世界へ転生した瞬間。空間の亀裂を通って召喚された。
