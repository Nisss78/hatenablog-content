第471章：氷の試練

イヴェルナは、カイトたちを塔の最深部へと案内した。

螺旋状の氷の階段を下りていく。壁の結晶が放つ青白い光が、彼らの影を長く伸ばしていた。

「どこへ行くんですか？」

エリナが、不安そうに尋ねた。

「試練の場所へ」

イヴェルナは、振り返らずに答えた。

「カイトの力を試すのに、最も適した場所です」
「危険なことじゃないよね？」
「試練に安全なものなどありません。でも、死ぬことはないでしょう」

「死ぬことはない」という言葉が、かえって不安を煽った。リアムが、剣の柄に手をかけた。

「カイト、何かあったら俺たちが守る」
「ありがとう、リアム。でも、これは僕一人でやるよ」

カイトは、決意を込めて言った。

「僕の力を試すんだ。みんなに頼るわけにはいかない」

◆ ◆ ◆

階段の突き当たりに、巨大な氷の扉があった。

高さは十メートル以上。扉全体に、複雑な紋様が刻まれている。古代の文字のようにも見えるが、カイトには読めなかった。

「ここが、試練の場所」

イヴェルナは、扉の前に立った。

「この先にあるのは、氷の迷宮。出口にたどり着けば、試練は終了です」
「迷宮……。道に迷ったら？」

カイトが尋ねると、イヴェルナは無表情に答えた。

「永遠に彷徨うことになります」
「永遠って……」
「冗談です。三時間で出られなければ、私が助けます。でも、その場合は不合格」

イヴェルナの表情は変わらなかったが、その奥に微かな悪戯っぽさが見えた気がした。

「では、始めましょう」

イヴェルナが手をかざすと、氷の扉が音もなく開いた。中から、冷気が吹き出してくる。

「カイト、頑張って」

エリナが、心配そうに言った。

「必ず戻るから」

カイトは、仲間たちに手を振り、迷宮へと足を踏み入れた。

◆ ◆ ◆

迷宮の中は、外とはまったく異なる世界だった。

壁も床も天井も、すべてが氷でできている。でも、ただの氷ではない。壁の中に、無数の光点が見える。まるで、星の海の中に立っているようだ。

カイトは、自分の足音を聞きながら進んだ。カツン、カツン。音が、どこまでも響いていく。

「まずは、道を探さないと」

カイトは、目を閉じた。コードの視界を使って、迷宮の構造を読み取ろうとする。


class IceLabyrinth {
  private structure: MazeNode[];
  private currentPosition: Position;
  
  public scanPath(): Path[] {
    // 氷の壁を透過して経路を探る
    return this.detectOpenings();
  }
}

カイトの意識が、迷宮の中に広がっていく。でも、何かがおかしい。

迷宮の構造が、見えない。壁があるはずの場所に、何も感じられない。まるで、迷宮そのものが、カイトの認識を拒んでいるようだ。

「どういうことだ……」

カイトは、目を開けた。自分の力が通用しないことに、戸惑いを感じた。

「ここは、ただの迷宮じゃない。イヴェルナさんが作った、特別な場所だ」

カイトは、別の方法を考えた。コードの視界が使えないなら、五感を頼りに進むしかない。

右の通路、左の通路。どちらを選ぶべきか。

「風の流れを感じてみよう」

カイトは、風魔法を使った。微かな風を起こし、空気の流れを読み取る。


function detectAirFlow(): Direction {
  const wind = new WindSpell("gentle");
  const flow = wind.trace();
  return flow.source;
}

風が、左の通路から吹き込んでいる。外に通じているのは、左のようだ。

「よし、左だ」

カイトは、左の通路へと進んだ。

◆ ◆ ◆

十分後、カイトは行き止まりに遭遇した。

目の前には、氷の壁。風は、この壁の向こうから吹いてきている。でも、通れる隙間はない。

「壁を壊すか？」

カイトは、火魔法を思い浮かべた。氷なら、熱で溶かせるはずだ。

でも、すぐに思いとどまった。イヴェルナの作った迷宮だ。簡単に壊せるはずがない。それに、壁を壊すことが試練の答えなら、最初から火魔法を使えばいい。そんな単純な話ではないはずだ。

「待てよ。風が吹いているということは、どこかに隙間がある」

カイトは、壁を詳しく調べた。表面は滑らかだが、所々に小さな亀裂がある。その一つに、風が吸い込まれていく。

「ここだ」

カイトは、亀裂に指を当てた。そして、風魔法で空気を押し込んだ。


function activateHiddenMechanism(crack: Crack): void {
  const airPressure = new WindSpell("compressed");
  airPressure.inject(crack);
}

カシュッ。

音がして、壁の一部が回転した。隠し扉だ。

「ビンゴ」

カイトは、開いた隙間をくぐった。

◆ ◆ ◆

迷宮の奥へと進むにつれて、カイトはイヴェルナの意図を感じ取れるようになった。

この迷宮は、ただの道迷いゲームではない。カイトの力を、多角的に試している。

  • コードの視界への依存を戒める
  • 五感と魔法を組み合わせる柔軟性
  • 諦めずに解を見つける粘り強さ

イヴェルナは、カイトがただ強い魔法を使えることを望んでいるわけではない。本当の強さは、頭を使い、工夫し、諦めないことにある。そのことを、迷宮を通じて教えているのだ。

「なるほど。賢者ってのは、ただ強いだけじゃないんだな」

カイトは、新たな敬意を抱いた。

そして、一時間後。

カイトは、迷宮の出口にたどり着いた。

そこには、光が差し込んでいた。外の世界への出口だ。

「やった」

カイトは、光の中へと歩き出した。

◆ ◆ ◆

出口の先は、氷の広間だった。

天井が高く、壁には大きな氷の結晶が並んでいる。中央には、イヴェルナが立っていた。エリナ、リアム、セシリアも、その横にいる。

「戻りましたね」

イヴェルナが、静かに言った。

「所要時間、一時間七分。合格です」
「合格……」

カイトは、安堵の息を吐いた。

「僕の力、認めてくれますか？」
「ええ。ただし、迷宮を抜けただけでは不十分です」

イヴェルナは、カイトの前に歩み寄った。

「本当の試練は、これからです」
「えっ？」
「私と戦いなさい。あなたのコードで、私の氷を超えてみせなさい」

イヴェルナの周囲に、冷気が渦巻いた。銀白色の髪が、風もないのに揺れる。

「本気ですか？」

カイトが尋ねると、イヴェルナは微かに笑った。

「賢者を名乗る者が、冗談で戦いを申し込むと思いますか？」
「……分かりました」

カイトは、覚悟を決めた。

「イヴェルナさん。あなたの氷、僕のコードで溶かしてみせます」

◆ ◆ ◆

戦いは、一瞬だった。

イヴェルナが手をかざすと、広間の床から無数の氷柱が突き出た。カイトは、風魔法で空中に飛び退く。

空中で、カイトは火魔法を詠唱した。


async function flameBurst(): Promise {
  const fire = new FireSpell("intense");
  fire.setTemperature(1500); // 摂氏1500度
  fire.spread("radial", 10); // 半径10メートル
  await fire.cast();
}

熱い炎が、カイトの周囲に広がる。氷柱が溶け、水蒸気が立ち上る。

でも、イヴェルナは涼しい顔だった。


function absoluteZeroShield(): void {
  const ice = new IceSpell("barrier");
  ice.setTemperature(-273); // 絶対零度に近い
  ice.createBarrier(5); // 半径5メートルの結界
}

イヴェルナの周囲に、青白い結界が現れた。カイトの炎が触れると、霧のように消えていく。

「熱を、冷気で打ち消した」

カイトは、驚いた。火と氷の相性を逆転させるなんて、普通の魔法使いにはできない。

「氷の賢者として、数百年氷を研究してきました。単純な熱では、溶かせませんよ」

イヴェルナが、静かに言った。

「では、どうすれば？」

カイトは、考えた。熱で溶かせないなら、何ならできる？

「答えは一つ。氷の構造そのものを書き換える」

カイトは、コードを紡いだ。これは、世界を記述する力。魔法の現象だけでなく、現象そのもののルールを書き換える力。


function rewriteIceStructure(target: IceSpell): void {
  // 氷の分子結合を書き換える
  const molecules = target.getMolecularStructure();
  molecules.vibration = "high"; // 分子振動を高める
  molecules.bond = "unstable"; // 結合を不安定化
  target.applyChanges();
}

カイトの指先から、金色の光が放たれた。光は、イヴェルナの結界に触れ、その中に浸透していく。

イヴェルナの表情が、初めて変わった。

「これは……。魔法の構造を、直接書き換えている？」

結界が、揺らぎ始めた。青白い光が、金色に染まっていく。

やがて、結界は音もなく崩れ去った。氷が解けるのではなく、存在そのものが消えたように。

「すごい……」

エリナが、息を呑んだ。

「カイト、本当に世界を記述する力を持ってるんだ」

◆ ◆ ◆

戦いが終わると、イヴェルナは静かに拍手した。

「素晴らしい。あなたが、预言の者であることは間違いありません」
「预言の者、ですか」

カイトが尋ねると、イヴェルナは頷いた。

「古の预言には、こうあります。『コードを紡ぐ者、世界の理を書き換える者。彼が現れた時、世界は選択を迫られる』」
「選択？」
「世界を救うか、滅ぼすか。あなたの力は、どちらにも使える」

イヴェルナの瞳が、カイトを真っ直ぐに見つめた。

「でも、あなたの瞳を見て分かりました。あなたは、世界を救う者だ」
「どうして分かるんですか？」
「迷宮で、あなたが選んだ道。暴力ではなく、知恵。破壊ではなく、工夫。あなたの選択の一つ一つが、あなたの本質を表していた」

イヴェルナは、カイトの前に手を差し出した。

「カイト。私は、あなたに協力します。氷の賢者として、あなたの旅を助けましょう」
「本当ですか？」

カイトは、嬉しさを隠せなかった。

「ありがとうございます、イヴェルナさん」
「イヴェルナでいいです。長い時を生きてきましたが、友人と呼べる者はほとんどいません。あなたたちとなら、そう呼び合えるかもしれません」

イヴェルナの顔に、初めて本当の笑みが浮かんだ。冷たい氷が、春の陽光に溶かされたような、穏やかな笑みだった。

◆ ◆ ◆

その夜、カイトたちはイヴェルナの塔に泊まることになった。

客室は、氷でできていたが、不思議と寒くない。イヴェルナの配慮で、適度な温度に保たれている。

夕食は、氷見の村から取り寄せた食材を使った料理だった。魚の煮込み、根菜のスープ、温かいパン。どれも美味しく、カイトたちは久しぶりに腹いっぱい食べた。

「イヴェルナ、ずっとここで一人で暮らしてたの？」

エリナが、尋ねた。

「ええ。村人が時々訪ねてきますが、基本は一人です」
「寂しくない？」

エリナの問いに、イヴェルナは少し考えた。

「最初は寂しかった。でも、長い時を経て、慣れました」
「慣れることじゃないよ。寂しいのは」

エリナが、言った。

「これからは、私たちがいる。一人じゃないよ」
「……優しい子」

イヴェルナは、エリナの頭を撫でた。

「ありがとう。あなたたちに会えて、よかった」

カイトは、その光景を見て、胸が温かくなった。氷の賢者も、孤独な存在ではない。ただ、誰かが手を差し伸べてくれるのを待っていたのだ。

「明日は、ここを出発します」

カイトが言うと、イヴェルナは頷いた。

「ええ。次は、地の賢者テラドンさんのところですね」
「地の賢者……」
「彼は、七賢者の中で最も長い時を生きている。私よりも、さらに古い存在です」

イヴェルナは、窓の外を見た。

「彼に会えば、世界の真実が見えるかもしれません」
「真実？」
「世界がなぜ危機に瀕しているのか。あなたの力が、なぜ必要なのか。すべては、彼が知っています」

カイトは、窓の外の星空を見上げた。旅はまだ続く。でも、仲間が増えた。それが、何よりの心強さだった。
