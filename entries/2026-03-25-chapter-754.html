第754章：氷雪の試練

フローズナは氷の杖を振り、洞窟の奥へと歩き出した。

「ついてきなさい。管理者の塔へ向かう前に、資格を証明してもらうわ」

カイトたちはフローズナの後ろについていった。

氷の回廊は続く。壁には古代の文字が刻まれ、天井には光る氷晶が埋め込まれている。

「資格って、何をするんですか？」

シオンが尋ねた。

「氷雪の試練よ。三つの関門を突破すれば、塔への道を開くわ」

フローズナは振り返らずに答えた。

「失敗すれば、ここから先には進めない」

十分ほど歩くと、広い空間に出た。

円形の広場。中央には巨大な氷の像が立っている。

像は、氷のドラゴンを模したもの。翼を広げ、口を開けている。

「第一の試練は、氷の知恵」

フローズナが告げた。

「このドラゴンの謎を解きなさい」

ドラゴンの台座に、文字が浮かび上がった。


氷は燃えず、水は凍まず
炎を飲み込み、まだ生きる者は何？

カイトは謎を読んだ。

「氷は燃えず、水は凍まず、炎を飲み込み、まだ生きる……」

シオンが首を傾げた。

「炎を飲み込むって、火の魔法のこと？」
「違うわ。もっと根本的な答えよ」

フローズナがヒントを出した。

リアムが考えた。

「氷は燃えない。水は凍らない。炎を飲み込む……」

エリナが閃いた。

「カイト、答えは『冷気』じゃない？」

カイトは首を振った。

「違う。もっとシンプルだ」

カイトは考えた。

氷は燃えない。水は凍らない。炎を飲み込む。まだ生きる。

「……『夜明け』だ」

カイトが答えた。

フローズナが振り返った。

「理由を説明して」
「夜明けは、氷のように冷たいけど燃えない。水のように流れるけど凍らない。太陽の炎を受け入れて、まだ生き続ける。夜明けの後も、日は昇り続けるから」

カイトは説明した。

ドラゴンの像が光った。

「正解よ」

フローズナは満足そうに頷いた。

「次に行きましょう」

第二の試練の場所は、氷の迷宮だった。

壁が四方に伸び、道が幾つにも分かれている。

「第二の試練は、氷の勇気」

フローズナが告げた。

「この迷宮の奥にある『氷の心臓』を取ってきなさい。ただし――」

フローズナは一瞬、沈黙した。

「一人で行くこと。仲間の助けは借りられないわ」

カイトは迷宮の入り口に立った。

「カイト、無理しないで」

シオンが心配そうに言った。

「大丈夫。何とかなる」

カイトは迷宮に足を踏み入れた。

迷宮の中は、静寂に包まれていた。

氷の壁がカイトの姿を映す。自分の顔が、無限に映っている。

「幻覚か？」

カイトは歩を進めた。

道は分岐している。右か、左か。

「右に行こう」

カイトは右の道を選んだ。

しばらく歩くと、行き止まりだった。

「戻らなきゃ」

カイトは来た道を戻った。

でも、道が変わっている。

「迷宮自体が、動いてるのか」

カイトは冷静に観察した。

カイトは魔法を使った。


// 迷宮構造解析魔法
function analyzeMaze(structure) {
  const magic = {
    type: "light",
    subtype: "structural_analysis",
    target: structure,
    depth: "全域"
  };
  
  return `${structure}の構造を解析`;
}

// 実行
analyzeMaze("氷の迷宮");

氷の迷宮の構造が、カイトの脳内に浮かんだ。

「なるほど。中央の氷の心臓が、迷宮を操作してるんだ」

カイトは中心への最短ルートを見つけた。

カイトは迷宮の中央に到達した。

そこには、青白い光を放つ宝石が浮いていた。

氷の心臓だ。

「これを取ればいいんだな」

カイトは手を伸ばした。

その時、氷の壁から影が現れた。

カイト自身の影。でも、目が赤く光っている。

「影の分身か」

カイトは剣を抜いた。

影のカイトも剣を抜いた。

影のカイトが斬りかかってきた。

カイトは剣で受けた。衝撃が腕に走る。

「俺と同じ動きをするのか」

カイトは分析した。

攻撃すれば、反撃される。防御すれば、隙を突かれる。

「なら、攻撃しない」

カイトは剣を収めた。

影のカイトは戸惑ったように立ち止まった。

「戦う意志はない。通してくれ」

カイトは静かに言った。

影のカイトは、ゆっくりと消えていった。

「自分自身の恐怖と戦う試練……か」

カイトは氷の心臓を手に取った。

カイトは迷宮を出た。

仲間たちが待っていた。

「カイト！」

シオンが駆け寄った。

「無事だったんだね」
「ああ。氷の心臓、取れたよ」

カイトは宝石をフローズナに渡した。

フローズナは宝石を見た。

「いいわ。第三の試練へ行きましょう」

フローズナは歩き出した。

「第三は、氷の戦い。私と戦いなさい」
「えっ？」

シオンが驚いた。

「フローズナさんと？　本気ですか？」
「本気よ。管理者の塔への道を開くには、氷の賢者に認められなければならない」

フローズナは杖を構えた。

「準備はいい？」

カイトは剣を抜いた。

「やるしかない」

カイトは魔法を唱えた。


// 戦闘状態初期化
function initializeCombat(player, enemy) {
  const status = {
    player: player,
    enemy: enemy,
    turn: "player",
    buffs: [],
    debuffs: []
  };
  
  return `${player}対${enemy}の戦闘を開始`;
}

// 実行
initializeCombat("カイト", "フローズナ");

フローズナが先制攻撃を仕掛けた。

「氷槍！」

無数の氷の槍がカイトに向かって飛んできた。

カイトは回避した。氷の槍が床に突き刺さる。

「速い！」

カイトは反撃の魔法を唱えた。


// 火炎魔法（対氷用）
function fireblast(target, power) {
  const magic = {
    type: "fire",
    subtype: "blast",
    target: target,
    power: power
  };
  
  return `${target}に火力${power}の火炎を放つ`;
}

// 実行
fireblast("フローズナ", 80);

炎がフローズナに向かって放たれた。

フローズナは杖を振った。

「氷壁！」

氷の壁が現れ、炎を防いだ。

「予想通りね。炎魔法が得意なのは知ってるわ」

フローズナは微笑んだ。

「でも、それだけじゃ足りない」

フローズナが次の魔法を唱えた。

「絶対零度！」

空間全体が凍り始めた。

カイトの動きが鈍る。

「まずい……」

カイトは体内温度維持魔法を強化した。


// 温度維持強化
function reinforceTempProtection(level) {
  const magic = {
    type: "fire",
    subtype: "temperature_maintenance",
    reinforcement: level
  };
  
  return `体温維持魔法を${level}倍に強化`;
}

// 実行
reinforceTempProtection(3);

カイトは動けるようになった。

「やるわね」

フローズナが称賛した。

「でも、これで終わりよ」

フローズナが最大魔法を準備した。

「氷河期！」

巨大な氷の嵐がカイトを襲った。

カイトは防御魔法を展開した。


// 七属性統合防御
function elementalShield(elements) {
  const magic = {
    type: "composite",
    elements: elements,
    defense: "最大"
  };
  
  return `${elements.join("・")}の複合防御を展開`;
}

// 実行
elementalShield(["火", "水", "風", "土", "雷", "光", "闇"]);

七色の盾がカイトを守った。

氷の嵐が収まった。

カイトは無事だった。

「七属性の統合……！　本当にできるのね」

フローズナは驚いた。

「管理者としての資格、十分よ」

フローズナは杖を下ろした。

「試練、合格」

エリナ、リアム、シオンが駆け寄った。

「カイト、すごい！」

シオンが興奮した。

「フローズナさんと互角に戦ったなんて」
「互角じゃないよ。フローズナさんが手加減してくれた」

カイトは正直に言った。

フローズナは微笑んだ。

「手加減はしてないわ。あなたの実力が、私と同等だった」

フローズナはカイトの前に来た。

「管理者の塔への道、開くわ。ついてきなさい」

フローズナは洞窟の奥へと歩き出した。

洞窟の最奥に、巨大な扉があった。

氷でできた扉には、古代の文字が刻まれている。

「これが、管理者の塔への転移門よ」

フローズナが説明した。

「中に入れば、塔の入り口に転送される」

カイトは扉の前に立った。

「フローズナさん、ありがとうございました」

カイトは頭を下げた。

「礼には及ばない。私は、管理者の血脈を守るだけよ」

フローズナは優しく言った。

「塔で何を見つけるか、楽しみにしてるわ」

カイトたちは扉を開けた。

中は、眩い光に包まれていた。

「行こう」

カイトは光の中に足を踏み入れた。

エリナ、リアム、シオンも続いた。

光が収まると、カイトたちは新しい場所にいた。

空には星がない。地面には雲が広がっている。

「ここが……管理者の塔？」

シオンが周囲を見回した。

遠くに、天を突くような塔が見えた。

「あれが、管理者の塔だ」

カイトが指差した。

塔は黒く、不気味な雰囲気を纏っていた。

でも、カイトは前に進んだ。

「封印を確認して、世界を守る。それが、管理者の仕事だから」

カイトの言葉に、仲間たちは頷いた。

カイトたちは、管理者の塔への道を歩き始めた。

氷雪の試練を乗り越え、新たな冒険が幕を開ける。

第754章、了。
