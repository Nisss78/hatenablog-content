第479章：原初の座

光の中を抜けた先——そこには、想像を絶する光景が広がっていた。

巨大なクレーターのようなどん底。その中央に、一本の巨大な柱がそびえ立っている。柱の高さは優に百メートルはあり、表面には複雑な紋様が刻まれていた。

柱の周囲には、七つの浮遊する台座が配置されている。それぞれの台座は、異なる色の光を放っていた——赤、青、緑、黄、紫、白、そして黒。

「これが……原初の座」

カイトは、息を呑んだ。

言葉では聞いていた。でも、実際に目にすると——その規模と神聖さに、圧倒される。

「すごい……こんな場所が、この世界の中心だったなんて」

エリナも、言葉を失っていた。

「空気が違う。魔力の密度が、段違いだ」

リアムが、冷静に分析しながらも、その驚きを隠せなかった。

「私、こんな場所、見たことありません」

セシリアが、目を輝かせて周囲を見回した。

カイトは、中央の柱を見上げた。その頂上には、何かがあるように見える。でも、高すぎて判別できない。

「行こう。何が待っているかわからないが、ここまで来たんだ」

カイトたちは、柱に向かって歩き出した。

◆ ◆ ◆

柱に近づくにつれて、空気が変化した。

張り詰めたような緊張感——何かが待っている予感。

七つの浮遊台座の近くまで来たとき、カイトは足を止めた。

「待て。何かいる」

カイトが警告した瞬間——七つの台座が、一斉に光り輝いた。

光の中から、七つの影が現れた。

人の形をしている。でも、実体がない——光で形作られた、半透明の存在だ。

「ようやく来たか、管理者の末裔よ」

中央に立つ白い影が、口を開いた。声は、まるで鐘の音のように響いた。

「君たちは？」

カイトが、尋ねた。

「我々は『七賢の残響』。かつてこの世界を守った七人の賢者の、記憶の断片だ」

白い影が、答えた。

「アレクシス・ヴェイルを始めとする七人の賢者は、この世界に魔法システムを構築した後、その記憶をここに残した。後の管理者たちが、いつか訪れる日のために」
「記憶の断片……」

カイトは、七つの影を見回した。

白い影の他に——赤、青、緑、黄、紫、黒の影がいる。それぞれが、かつての賢者の姿を模しているのだろう。

「カイトよ。君がここに来た目的は、原初の座の力を借りて、世界のバランスを修復することであろう」
「その通りだ。虚無の欠片を集め、因果のゆがみを正したい」
「わかっている。だが、その前に——試練を受けよ」

白い影が、厳かに宣言した。

「試練？」
「原初の座の力は、強大だ。相応しい者だけが、それを行使できる。君が真に管理者に相応しいか、見極めさせてもらう」

カイトは、覚悟を決めた。

「わかった。どんな試練でも受ける」
「良い覚悟だ。では——第一の試練、『勇気の証』を与えよう」

白い影が、手をかざした。

カイトの目の前に、赤い影が進み出た。

「我は炎の賢者、フレイア。勇気の試練を司る」

赤い影——フレイアが、宣言した。

「勇気とは、恐怖を知らないことではない。恐怖を乗り越えることだ。カイトよ、君の最も恐れるものを示せ」

フレイアが手を振ると、カイトの周囲に炎の壁が現れた。

炎の中に、幻影が浮かび上がる。

それは——エリナが、虚無に侵食される姿だった。

「カイト……助けて……私、消えちゃう……」

幻影のエリナが、苦しげに呟いた。

カイトの心が、激しく揺れた。最も恐れていること——大切な人を失うこと。

「これは……！」

カイトは、拳を握りしめた。

「逃げるな。見つめろ。そして、乗り越えよ」

フレイアの声が、響いた。

カイトは、幻影のエリナと向き合った。

恐怖が、胸を締め付ける。でも——逃げてはいけない。

「エリナ……君を失うことは怖い。でも、その恐怖に負けていたら、君を守れない」

カイトは、一歩踏み出した。

炎の中へ——幻影の中へ。

「僕は、君を守る。どんなことがあっても。それが、僕の決意だ」

カイトが幻影に触れた瞬間——炎が消えた。

幻影のエリナが、微笑んで消えていく。

「合格だ。君の勇気、認めよう」

フレイアが、満足げに頷いた。

◆ ◆ ◆

「続いて——第二の試練、『知恵の証』」

今度は、青い影が進み出た。

「我は氷の賢者、イヴェルナ。知恵の試練を司る」

イヴェルナが、手をかざすと、カイトの目の前に巨大な迷宮が現れた。

氷で作られた迷宮——出口は見えない。

「知恵とは、知識を蓄えることではない。答えのない問いに対して、最善の道を見出すことだ。この迷宮を、制限時間内に脱出せよ」

カイトは、迷宮を見た。

普通の迷宮なら、壁を手でなぞりながら進めばいい。でも、この迷宮は——壁が動いている。刻一刻と変化している。


class MazeAnalysis {
  private mazeState: DynamicMaze;
  private timeLimit: number = 300; // 5分
  
  public findPath(): Path {
    const currentState = this.mazeState.snapshot();
    const predictedStates = this.predictChanges(currentState);
    
    return this.calculateOptimalRoute(predictedStates);
  }
  
  private predictChanges(state: MazeState): MazeState[] {
    // 壁の動きを予測
    const patterns = this.detectPatterns(state);
    return this.extrapolate(patterns, this.timeLimit);
  }
}

カイトは、壁の動きを観察した。

ランダムに見える。でも——法則がある。特定のリズムで、壁が動いている。

「わかった。壁の動きには周期がある」

カイトは、迷宮に足を踏み入れた。

壁が迫ってくる。でも、カイトは動じない。次の瞬間、壁がどこへ動くか——予測できているから。

迷宮の中央まで進んだとき、カイトは足を止めた。

行き止まり——のように見える。でも、カイトは待った。

三秒後——壁が動き、新しい道が現れた。

「読み通りだ」

カイトは、その道を進んだ。

五分後——カイトは、迷宮の出口に立っていた。

「見事だ。君の知恵、認めよう」

イヴェルナが、涼しげな声で言った。

◆ ◆ ◆

「第三の試練——『慈悲の証』」

緑の影が進み出た。

「我は風の賢者、ウィンダル。慈悲の試練を司る」

ウィンダルが手を振ると、カイトの目の前に二人の人物が現れた。

一人は——幼い子供。もう一人は——老人。

二人とも、苦しげな表情をしている。

「二人のうち、どちらか一人だけを救える。選べ」

ウィンダルが、問いかけた。

カイトは、二人を見た。

子供を救えば、老人は死ぬ。老人を救えば、子供は死ぬ。どちらを選んでも、一人を失う——

「これは……残酷な問いだ」

カイトは、眉をひそめた。

「慈悲とは、選択することではない。両方を救う方法を探すことだ」

カイトは、子供と老人の間に立った。

「僕は、選ばない。二人とも救う」
「不可能だ。魔力は一人分しかない」

ウィンダルが、冷静に言った。

「魔力が足りないなら——僕の魔力を使う」

カイトは、両手を広げた。


class CompassionCasting {
  private source: MagicPool;
  private targets: Person[] = [];
  
  public healAll(): void {
    const requiredEnergy = this.calculateTotalRequired();
    const selfEnergy = this.source.available;
    
    if (selfEnergy < requiredEnergy) {
      // 自分の生命力も使う
      this.convertLifeToEnergy(requiredEnergy - selfEnergy);
    }
    
    this.targets.forEach(target => {
      this.channel(target, requiredEnergy / this.targets.length);
    });
  }
}

カイトの体から、魔力が流れ出た。

子供と老人——二人を同時に癒やす。

カイトの顔色が、蒼白になっていく。でも、手は止めない。

「これで……二人とも、助かった」

カイトが呟いた瞬間——子供と老人の姿が消えた。

幻影だったのだ。

「優しいな。自分を犠牲にしてでも、他者を救おうとする」

ウィンダルが、感慨深げに言った。

「合格だ。君の慈悲、認めよう」

◆ ◆ ◆

続いて——黄色、紫色、黒色の影が、同時に進み出た。

「我は雷の賢者、ライオネル」
「我は時空の賢者、クロノス」
「我は闇の賢者、ノクティス」

三人が、名乗った。

「残る三つの試練——『正義』『忍耐』『希望』は、同時に与えられる」

三人が、同時に手をかざした。

カイトの周囲に、三つの幻影が現れた。

一つ目——悪魔のような姿をした存在が、無辜の民を虐殺している。

二つ目——果てのない闇の中で、カイトが一人で立ち尽くしている。

三つ目——廃墟となった世界で、カイトが瓦礫の中から何かを探している。

「正義の試練だ。悪魔を倒せ。だが——奴は、倒されるたびに復活する」

ライオネルが、言った。

「忍耐の試練だ。闇の中で、光が現れるまで待て。光が現れるのは、百年後かもしれない」

クロノスが、言った。

「希望の試練だ。廃墟の中で、一粒の種を見つけろ。それはどこかにある。だが、広すぎて見つからないかもしれない」

ノクティスが、言った。

三つの試練——どれも、一見不可能に見える。

でも、カイトは微笑んだ。

「わかった。やろう」

◆ ◆ ◆

カイトは、まず悪魔に向き合った。

「正義とは、悪を滅ぼすことではない。悪の根源を断つことだ」

カイトは、悪魔を攻撃しなかった。代わりに——悪魔が虐殺している民に近づいた。

「君たちが苦しんでいる原因は、この悪魔じゃない。この悪魔を作り出した、因果のゆがみだ」

カイトは、民に手をかざした。


class RootCauseHealing {
  public heal(source: CorruptionSource): void {
    // 悪魔を倒すのではなく、生み出す原因を取り除く
    const corruption = this.identifyCorruption(source);
    this.purify(corruption);
  }
}

カイトの魔力が、民を包んだ。

その瞬間——悪魔が消えた。民を苦しめる原因がなくなったからだ。

「見事だ。根本原因を取り除く——それが真の正義だ」

ライオネルが、納得したように頷いた。

◆ ◆ ◆

次に——カイトは闇の中に立った。

光がない。時間の感覚もない。

「忍耐とは、ただ待つことではない。待ちながら、準備を続けることだ」

カイトは、座り込まなかった。その場で、魔法の練習を始めた。

闇の中で見えなくても——体の感覚はある。魔力の流れは感じられる。

一分、十分、一時間——時間が経つにつれて、カイトの魔法は洗練されていった。

そして——

カイトの魔法が、完成した瞬間。

闇が消え、光が現れた。

「自分で光を作り出したか。待つのではなく、自ら動いた」

クロノスが、感心したように言った。

「合格だ。君の忍耐、認めよう」

◆ ◆ ◆

最後——カイトは廃墟の中に立った。

瓦礫が、どこまでも続いている。一粒の種を探すなんて、不可能に見える。

「希望とは、見つかることを信じることではない。自分で作り出すことだ」

カイトは、種を探さなかった。代わりに——瓦礫を集め始めた。

土を作るために。

水を探すために。

そして——自分の虚無の欠片から、一粒の種を創り出した。


class HopeCreation {

  }

カイトの手の中で、黒い光が凝縮した。

そして——小さな芽が出た。

「種を探すのではなく、自分で作ったか」

ノクティスが、静かに言った。

「希望は、与えられるものではない。自分で作るものだ。君はそれを理解している」
「合格だ。君の希望、認めよう」

◆ ◆ ◆

六つの試練——すべてを突破した。

「見事だ、カイト。管理者の末裔として、相応しい資質を持っている」

白い影——七賢の代表が、宣言した。

「原初の座の力、君に託そう」

七つの影が、一斉に光り輝いた。

その光が、中央の柱に集まっていく。

柱が、激しく振動した。

そして——柱の頂上から、一つの物体が浮かび上がった。

透明な水晶——その中に、七色の光が渦巻いている。

「それが『原初の核』。この世界の魔法システムの基盤だ」

白い影が、説明した。

「カイトよ。その核に触れれば、世界のバランスを修復できる。だが——」
「だが？」
「その代償として、君は管理者としてのすべての力を失うかもしれない。元の世界に帰れなくなるかもしれない」

カイトは、水晶を見上げた。

エリナが、カイトの手を握った。

「カイト……」

カイトは、エリナを見た。そして、リアムとセシリアも。

「みんな……僕は、覚悟を決めている」

カイトは、静かに言った。

「この世界を守る。それが、僕の選んだ道だ」

カイトは、原初の核に手を伸ばした。

指先が、水晶に触れた瞬間——

世界が、光に包まれた。
