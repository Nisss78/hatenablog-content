第107章：世界のコード、書き換え

カイトは、世界の設計図を机の上に広げた。

黒い表紙の本から放たれる淡い光が、彼の顔を照らす。ページには、複雑な魔法陣と幾何学模様が刻まれていた。それは、ただの図形ではない。この世界を構成する「コード」そのものだ。

「これが、世界のソースコードか……」

カイトの声には、畏敬の念が込められていた。プログラマーとして、彼は多くのコードを見てきた。だが、これほど巨大で複雑なシステムは初めてだ。

リリが、静かに彼のそばに立った。

「設計図は、あなたの認識に合わせて情報を表示しますわ。まずは、どの部分から着手するおつもりで？」

カイトは、ページをめくった。召喚システムの概要が表示される。

// World Summoning System - Overview
// Original Author: Ancient Civilization
// Last Modified: 500 years ago
// Current Status: CRITICAL - Logic Conflict Detected
// Affected Modules: Portal Control, Soul Routing, Dimensional Anchor

「五百年前に最後の修正が行われている……」

カイトは、眉をひそめた。五百年という期間は、人間の尺度では長い。だが、システムの寿命としては、決して長くない。その間、誰もメンテナンスをしていなかったということだ。

「カイト、何か見えた？」

エリナが、彼の肩越しに覗き込んだ。

「ああ。召喚システムには、三つの主要モジュールがある。ポータル制御、魂のルーティング、次元アンカー。この三つが、互いに矛盾を起こしているんだ」
「矛盾？」

リアムが、腕を組んで問い返した。

「簡単に言うと、命令が食い違っている状態だ。Aというモジュールは『左へ行け』と言い、Bというモジュールは『右へ行け』と言う。結果、システムが混乱して、予期せぬ動作を起こす」

カイトの説明に、セシリアが頷いた。

「なるほど。それが、異世界からの過剰な召喚を引き起こしているのですね」
「その通り。システムは本来、特定の条件を満たした魂だけを召喚するはずだった。だが、矛盾のせいで条件判定が崩れ、無差別に魂を引き寄せるようになった」

カイトは、さらにページをめくった。問題の核心部分に近づいていく。

『次元アンカー・モジュール：魂をこの世界に固定する役割を担う。当初は、七日間の仮滞在を許可する設定だった。だが、五百年前のパッチで「永続固定」オプションが追加された。このパッチが、ポータル制御モジュールと競合している』

カイトは、ため息をついた。

「典型的な仕様変更の失敗だ……」
「どういうこと？」

アルフレッドが、静かに問うた。

「五百年前、誰かが『魂を永続的に固定したい』と思った。それ自体は悪いことじゃない。異世界からの訪問者を、ずっとこの世界に留まらせたいという願いだからな」

カイトは、説明を続けた。

「だが、その変更は、他のモジュールと整合性が取れていなかった。ポータル制御は『仮滞在』を前提に設計されている。そこに『永続固定』をねじ込んだから、矛盾が生じたんだ」
「じゃあ、五百年前のパッチを削除すれば直るの？」

エリナが、希望を込めて聞いた。

「残念ながら、そう簡単にはいかない。五百年の間に、そのパッチに依存する他の機能が追加されている可能性がある。削除すれば、別の問題が起きるかもしれない」

カイトは、設計図の別のページを開いた。

// Dependency Analysis
// Patch ID: ETERNAL-ANCHOR-001
// Dependent Modules: 
//   - Hero Retention System
//   - Cultural Exchange Protocol  
//   - World Stability Matrix
// Warning: Removing this patch may cause system collapse

「やはりな……」

カイトは、苦々しく思った。依存関係が深すぎる。単純に削除することはできない。

「じゃあ、どうすればいいの？」

リアムが、苛立ちを滲ませた。

「リファクタリングだ。一つずつ、慎重にコードを書き換えていく。依存関係を整理し、矛盾を解消する。時間はかかるが、それが唯一の安全な方法だ」

カイトは、決意を新たにした。

「みんな、悪いけど長引くかもしれない。僕一人じゃ、この作業は無理だ。サポートしてくれるか？」
「当然だ」

リアムが、即答した。

「私も協力しますわ。知識が必要なら、いつでも仰ってください」

セシリアが、魔導書を開いた。

「魔力供給なら任せて。カイトが集中している間、私がサポートするわ」

エリナが、彼の手を握った。

「守りはお任せください。何があっても、お守りします」

アルフレッドが、盾を構えた。

カイトは、仲間たちに感謝の眼差しを向けた。

「ありがとう。じゃあ、始めよう」

カイトは、設計図の前に座った。目を閉じ、集中する。

彼の意識が、深層へと潜っていく。世界のコードに直接アクセスするためだ。

// Accessing World Core
// Authentication: Three Trials Cleared
// Permission Level: Administrator
// Connection Established

カイトの視界が、一変した。

無数の光の糸が、空間を纵横に走っている。それが、この世界を構成する「コード」の視覚化だ。赤い糸、青い糸、黄色い糸。それぞれが、異なる役割を担っている。

「これが……世界の裏側」

カイトは、圧倒された。だが、感傆に浸っている暇はない。

彼は、召喚システムのエリアへと移動した。そこには、複雑に絡み合った糸の塊があった。正常なら、整然と並んでいるはずの糸が、乱雑にねじれている。

「ひどい状態だ……」

カイトは、最初の修正箇所を特定した。次元アンカーとポータル制御の接続部分だ。ここが、矛盾の震源地だ。

// Refactoring Step 1: Interface Unification
// Problem: Conflicting signal protocols
// Solution: Create unified interface layer

カイトは、手を伸ばした。指先から、青い光が放たれる。その光が、糸の塊に触れ、一本ずつ解きほぐしていく。

「コード・コンパイル──インターフェース・ユニファイ！」

魔法が発動した。二つのモジュールの間に、新しい層が形成される。翻訳機のようなものだ。互いに異なるプロトコルを、統一された形式に変換する。

カイトの額に、汗が滲んだ。精神への負担が大きい。

「カイト、大丈夫？」

エリナの声が、遠くから聞こえた。現実世界での声だ。彼女が、魔力を注いでくれているのだろう。温かい感覚が、カイトを包む。

「ありがとう、エリナ。大丈夫だ、続けられる」

カイトは、次の修正箇所へと移った。

時間が経つにつれて、作業は困難を極めていった。

依存関係が予想以上に複雑だ。一つの糸を引けば、別の糸が絡まる。まるで、巨大な毛糸の玉を解いているような感覚だ。

// Error Detected: Circular Dependency
// Module A requires Module B
// Module B requires Module C
// Module C requires Module A
// Status: Cannot resolve automatically

「循環依存か……」

カイトは、頭を抱えた。AがBを必要とし、BがCを必要とし、CがAを必要とする。どれから始めても、依存関係が解決しない。

「どうしたの？」

セシリアが、心配そうに声をかけた。

「循環依存が見つかった。このままじゃ、先に進めない」
「循環依存？ それは、どう解決するものなのですか？」

カイトは、少し考えた。

「一つの方法は、依存関係を断ち切ること。どれか一つのモジュールを、他のモジュールなしで動くように書き換えるんだ」
「それは可能ですの？」
「理論上は可能だ。だが、リスクがある。書き換えたモジュールが、他の機能に影響を与えるかもしれない」

カイトは、設計図に問いかけた。

「循環依存を解消するには、どのモジュールを書き換えるべきだ？」

ページが、光り輝いた。

『推奨アプローチ：Module C（次元アンカー）の独立化。依存関係を削除し、自己完結型モジュールへ再設計。これにより、循環が解消される』

カイトは、頷いた。

「了解。次元アンカーを、独立したモジュールに書き換える」

彼は、集中を深めた。次元アンカーを構成する糸の一本一本に、慎重に手を伸ばす。

「コード・リファクタリング──インデペンデント・モジュール！」

青い光が、次元アンカーを包み込んだ。糸が一本ずつ解け、新しい形へと再構築されていく。

だが──。

作業が半ばに差し掛かった時、異変が起きた。

空間が、歪んだ。

カイトの視界が、赤く染まる。警告信号だ。

// ALERT: External Interference Detected
// Source: Unknown
// Threat Level: High
// Action Required: Immediate Defense

「何だ！？」

カイトが顔を上げると、遠くの空間に黒い影が見えた。影は、次第に人の形をとっていく。

そして、その姿がはっきりとした瞬間──カイトは、息を呑んだ。

「アークライト……！」

敵対者の姿だった。だが、以前と少し違う。より禍々しく、より強大な気配を纏っている。

『カイト……我を妨げた代償を、払ってもらおうか』

アークライトの声が、空間全体に響き渡った。

『お前が設計図にアクセスした瞬間から、待っていたのだ。世界のコードに干渉しようとするその瞬間をな』

「くっ……罠だったのか！」

カイトは、歯噛みした。設計図へのアクセス自体が、アークライトにとって好都合だったのだ。カイトが道を開いたことで、彼もまた深層への侵入が可能になった。

『設計図を渡してもらおう。そして、この世界を、我が理想の姿へと書き換える』

アークライトが、手を伸ばした。黒い炎が、彼の手から放たれる。

「させるか！」

カイトは、反射的に防御コードを展開した。

// Emergency Defense: Shield Protocol
// Type: Magical Barrier
// Resistance: High
// Duration: Temporary

透明な壁が、カイトの前に現れた。黒い炎が、壁にぶつかり、弾け飛ぶ。

「みんな、聞こえるか！ 敵が来た！」

カイトが叫ぶと、現実世界から仲間たちの声が返ってきた。

「カイト！ 何が起きているの？」

エリナの焦った声。

「アークライトだ！ 彼が深層に侵入してきた！」
「なんだって！？」

リアムの驚愕の声。

「カイト、一旦戻ってください！ 一人では危険です！」

セシリアの警告。

「いや、戻れない！ 今戻ったら、リファクタリングの成果が消える！ ここで戦うしかない！」

カイトは、決断した。深層での戦い。世界のコードそのものを賭けた、最後の決戦だ。

アークライトが、ゆっくりと近づいてきた。

『興味深い。お前は、コードを視覚化して操る能力を持っているようだな』

彼は、カイトの周囲に浮かぶ光の糸を見つめた。

『だが、その能力も、我には及ばぬ。我は、この世界の「バグ」そのもの。矛盾を糧とし、混乱を力とする存在だ』

アークライトが、手を振ると、空間の糸が乱れ始めた。正常なコードが、歪んでいく。

// System Warning: Code Corruption Detected
// Affected Area: 12% and expanding
// Countermeasure Required: Immediate

「コードを壊す気か！」

カイトは、怒りを露わにした。

「壊す？ いや、違うな。我は、『再構築』するのだ。矛盾した世界など、最初からやり直すべきなのだからな」

アークライトの言葉に、カイトは戦慄した。

「お前の言う再構築は、破壊と同じだ！ 今の世界を消して、新しい世界を作るなんて、許さない！」

カイトは、コードを操る手を止めなかった。片手で防御を維持しながら、もう片手でリファクタリングを続ける。

「僕は、この世界を守る。矛盾を解消して、みんなが平和に暮らせる世界を作るんだ！」

『愚かだな。矛盾は、この世界の本質だ。解消など不可能だ』

アークライトが、冷笑した。

『だが、お前のその健気さ……認めよう。だからこそ、我はお前を倒すことに意味を見出すのだ』

黒い炎が、再び放たれた。今度は、より強力な一撃だ。

「くっ……！」

カイトの防御壁に、亀裂が入った。魔力の消耗が激しい。

「カイト！ 今助ける！」

その時、現実世界から声が響いた。エリナだ。

「私たちの魔力を、あなたに送る！ 受け取って！」

カイトの体に、温かい光が注がれた。エリナ、リアム、セシリア、アルフレッド。四人の仲間からの魔力供給だ。

// Power Boost Received
// Source: Team Unity Link
// Magnitude: 400%
// Status: Overflow Protection Active

カイトは、力が溢れてくるのを感じた。仲間たちが、彼を支えてくれている。一人じゃない。

「ありがとう、みんな……！」

彼は、深く息を吸った。

「コード・オーバーロード──ユニティ・バースト！」

五色の光が、カイトの手から放たれた。アークライトの黒い炎と、正面から激突する。

ドォォォォォォン！！

衝撃波が、深層の空間を揺るがした。光と闇の拮抗。どちらも譲らない。

『強くなったな、カイト……だが、まだ足りぬ！』

アークライトが、さらに力を込めた。黒い炎が、少しずつ押し返してくる。

「くそっ……！」

カイトは、限界を感じていた。仲間の魔力供給があっても、アークライトの力は圧倒的だ。

その時──。

空間の奥から、別の声が響いた。

『カイトよ……設計図の管理者として、お前に力を貸そう』

カイトが顔を向けると、リリの姿があった。だが、いつもの彼女ではない。光り輝く衣を纏い、神々しい気配を放っている。

『私は、始原の図書館の守護者にして、世界の設計図の管理者。三つの試練をクリアしたお前に、最高権限を付与する』

リリが、手をかざした。

// Permission Upgrade: Administrator → Root
// Access Level: Maximum
// Special Ability: Code Rewrite (Limited)
// Duration: Until Task Completion

カイトの視界に、新たなコマンドが表示された。

「ルート権限……！」

カイトは、理解した。これで、彼は世界のコードを直接書き換えることができる。アークライトを排除するための、最強の武器だ。

「アークライト、終わりだ！」

カイトが、声を張り上げた。

「コード・イレイズ──ターゲット・エクスターミネート！」

ルート権限を行使した、削除コマンド。対象を指定し、システムから完全に排除する。

だが──。

『残念だが、それも効かぬ』

アークライトが、不敵に笑った。

『我は、システムの一部ではない。システムの「矛盾」から生まれた存在だ。コードで操作できる対象ではないのだよ』

カイトは、絶句した。コードで操作できない敵。ならば、どう戦えばいいのか？

『だが、認めよう。お前のリファクタリングは、我にとって脅威だ。矛盾が解消されれば、我は消滅する』

アークライトが、静かに言った。

『だからこそ、ここで止めさせてもらう。お前がリファクタリングを完了する前に、この空間を崩壊させる！』

アークライトが、両手を広げた。空間の糸が、激しく揺れ始める。

「五分しかない……！」

カイトは、焦った。五分以内に、リファクタリングを完了しなければならない。だが、今の進捗は六割程度だ。間に合わない。

「どうすれば……！」

その時、エリナの声が響いた。

「カイト！ 私たちが時間を稼ぐ！ あなたは、リファクタリングを続けて！」
「でも、どうやって？ 現実世界からじゃ、深層には干渉できないはず……」
「やるのよ！ 私たちの想いを、あなたに届ける！」

カイトは、エリナの決意を感じた。彼女は、何かをしようとしている。

現実世界。

エリナは、カイトの手を強く握った。彼女の周囲に、白い光が集まっていく。

「みんな、手を！ カイトに、私たちのすべてを届けるの！」

リアム、セシリア、アルフレッドが、エリナの手に重ねた。四人の想いが、一つに収束していく。

「カイト、聞こえる？ 私たちの絆を、受け取って！」

光が、爆発的に膨れ上がった。

深層空間。

カイトの目の前に、四つの光の柱が現れた。エリナ、リアム、セシリア、アルフレッド。それぞれの想いが、具現化している。

> // Soul Link Established
// Partner: Elina - Healing Light
// Partner: Liam - Courage Flame  
// Partner: Cecilia - Wisdom Star
// Partner: Alfred - Protection Shield
// Synergy Level: Maximum

カイトは、胸が熱くなるのを感じた。仲間たちが、命がけで彼を支えてくれている。

「みんな……ありがとう」

彼は、涙を流しながら、最後のリファクタリングに取り掛かった。

「コード・ファイナライズ──ワールド・リファクタリング！」

五色の光が、空間全体を包み込んだ。矛盾した糸が解け、整然とした形へと再構築されていく。

アークライトが、悲鳴を上げた。

『やめろ！ 矛盾が消えれば、我は……！』

「消えろ、アークライト！ そして、世界は生まれ変わる！」

光が、アークライトの姿を飲み込んだ。

そして──。

静寂が訪れた。

カイトが目を開けると、彼は現実世界に戻っていた。設計図の前で、仲間たちに囲まれている。

「カイト……終わったの？」

エリナが、涙目で聞いた。

「ああ。リファクタリング、完了したよ」

カイトは、疲れ切った体で微笑んだ。

設計図のページには、一つのメッセージが浮かび上がっていた。

『修正完了。召喚システムの暴走は停止しました。これより、正常な運用を再開します』

カイトは、深く息を吐いた。

長い戦いだった。だが、ついに終わった。

「みんな、ありがとう。君たちがいなかったら、僕は倒れていた」
「私たちもだよ。カイトがいなければ、世界は救えなかった」

リアムが、カイトの肩を叩いた。

「これで、平和が戻るのですね」

セシリアが、安堵の表情を浮かべた。

「ええ。ただ、これからもメンテナンスが必要です。定期的に設計図をチェックして、矛盾がないか確認しないといけません」

カイトが言うと、エリナが微笑んだ。

「じゃあ、これからも一緒だね」
「ああ。ずっと一緒だ」

カイトとエリナは、手を取り合った。

世界は救われた。だが、物語はまだ終わらない。新しい時代の幕開けだ。

【用語解説】

  • リファクタリング：既存のコードを整理し、矛盾を解消する修正手法。今回、カイトが世界のコードに対して実施。
  • 循環依存：複数のモジュールが互いに依存し合い、解決不可能な状態。カイトが独立化で対処。
  • ルート権限：システムの最高権限。リリから付与され、カイトがコード直接書き換えを可能に。
  • ソウル・リンク：仲間たちの想いを具現化し、カイトに力を届ける絆の魔法。

世界は救われた。だが、カイトが異世界から召喚された理由は、まだ完全には解明されていない。彼は元の世界に戻れるのか？ そして、エリナとの絆はどうなる？ 新たな旅の始まり──！
