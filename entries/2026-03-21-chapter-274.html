第274章：世界樹の戦い

黒い霧が渦巻く。

アークライトの周りで、負の魔力が凝縮していく。

カイトは剣を構え、王冠の力を解放した。


const battle = new BattleSystem();
battle.initiate({
    party: ["kaito", "erina", "cecilia", "liam", "hinata"],
    enemy: "arklight",
    location: "world_tree_base"
});

戦闘開始。

アークライトが先に動いた。

黒い霧が触手のように伸び、カイトたちを襲う。

「散開！」

カイトの声に、五人は四方へ飛び退く。

触手は地面を叩き、深い亀裂を入れた。もし直撃していれば、ひとたまりもなかっただろう。

「速い！」

ヒナタが風魔法を放つ。


hinata.castSpell("wind_blade", { power: 80 });

風の刃がアークライトを襲う——が、黒い霧に阻まれた。

「無駄だ」

アークライトが嘲笑う。

「世界樹の魔力を吸収した今、私の防御は万全だ」

カイトは分析する。


const analysis = kaito.analyze(arklight);
console.log(analysis);
// {
//   shield: "dark_mist_barrier",
//   strength: 95,
//   weakness: ["holy_element", "artifact_resonance"],
//   power_source: "world_tree_mana"
// }
「魔力を吸い取ってる間は、防御が固い……」

カイトは考える。

「なら、供給を断てばいい」

カイトはエリナに目配せした。

「エリナ、世界樹とアークライトの接続を断てる？」

エリナは頷く。

「やってみる」

エリナが走り出す。

アークライトの注意がカイトに向いている間に、エリナは側面から接近した。

剣に光を纏わせ、アークライトと世界樹を繋ぐ黒い霧の管を狙う。

「聖剣技——光刃！」

erina.useSkill("holy_blade", { target: "dark_tube" });

刃が黒い管を切り裂いた。

アークライトが顔をしかめる。

「貴様——！」

管から黒い霧が漏れ出し、アークライトへの魔力供給が減少する。


arklight.manaSupply -= 40; // 供給量低下
「今だ！ 全員、攻撃！」

カイトが叫ぶ。

リアムが光魔法を詠唱。

「聖なる光よ、闇を照らせ——ホーリーランス！」

セシリアが炎魔法を放つ。

「燃え上がれ——ファイアストーム！」

ヒナタが風と雷を合わせた魔法。

「嵐を呼べ——サンダーゲイル！」

三つの魔法が、アークライトに向かって飛ぶ。


const comboAttack = combine([
    liam.castSpell("holy_lance"),
    cecilia.castSpell("fire_storm"),
    hinata.castSpell("thunder_gale")
]);

comboAttack.execute(arklight);
// ダメージ計算中...

アークライトは黒い霧で防御を固める——が、供給が減った今、完全には防げない。

光の槍が霧を貫く。

炎が霧を焼く。

雷が霧を切り裂く。

「ぐあああああ！」

アークライトが悲鳴を上げる。


arklight.takeDamage(350);
arklight.currentHP -= 350; // 1200 -> 850

カイトはこの隙を見逃さない。

王冠の力を使い、アーティファクトを共鳴させる。


const resonance = kaito.activateArtifactResonance();
resonance.addArtifacts([
    "fire_gem",
    "water_droplet",
    "wind_feather",
    "earth_crystal",
    "sky_jewel"
]);
resonance.setCatalyst("sky_crown");
resonance.execute();

王冠が強烈な光を放つ。

五つのアーティファクトが呼応し、カイトの剣に集約されていく。

「これが——アーティファクトの真の力」

カイトは剣を振り上げる。

「六神器融合剣——エレメンタル・ブレイク！」

kaito.useUltimateSkill("elemental_break", {
    elements: ["fire", "water", "wind", "earth", "sky"],
    multiplier: 5.0
});

六色の光を纏った剣が、アークライトに向かって振り下ろされる。

アークライトは最大の防御を展開するが——

「通れ！」

剣は黒い霧を切り裂き、アークライトの肩を貫いた。

「がはっ——！」

アークライトが後ろに吹き飛ばされる。


arklight.takeDamage(600);
arklight.currentHP -= 600; // 850 -> 250

アークライトは膝をついた。

肩から黒い血が流れている。

「……まさか、ここまでやられるとは」

アークライトが荒い息を吐く。

「認めよう。お前たちは強い」

カイトは剣を構えたまま警戒を解かない。

「降伏しろ、アークライト。これ以上戦っても無意味だ」

アークライトは笑う。

「降伏？ いや、私はまだ終わっていない」

アークライトの目に、狂気が宿る。

「切り札は——まだ残っている」

アークライトが世界樹に手をかざす。

「世界樹よ——その命を、我に与えよ」

arklight.useForbiddenSkill("world_tree_sacrifice");
// 警告: このスキルは世界樹を枯渇させる可能性があります

世界樹が震える。

幹に刻まれたルーン文字が、赤く発光し始めた。

「アークライト！ 何をする気だ！」

カイトが叫ぶ。

アークライトは狂ったように笑う。

「この世界樹の命を燃やし、その力で私は新生する！ 不完全なこの世界を、完全なる虚無へと書き換えるのだ！」

カイトは絶句する。

「世界樹を犠牲にする気か！？」
「世界樹が消えれば、この世界も消える。だが——私は生き残る。新たな創造主として！」

世界樹の色が変わり始めた。

鮮やかな緑から、枯れた茶色へ。

そして——黒へ。

「止めなきゃ！」

エリナが叫ぶ。

「カイト、世界樹を守って！」

カイトは頷く。

王冠の力を最大に高める。


kaito.overloadCrown({ power: "maximum" });
// 警告: オーバーロードは王冠に負担をかけます

王冠が激しい光を放つ。

カイトは、世界樹とアークライトの間に立った。

「通さない」

アークライトが睨む。

「邪魔をするか、小僧」
「お前のやり方は間違ってる。世界を書き換えるなら——もっと良い方法があるはずだ」

カイトは、六つのアーティファクトを王冠に集めた。

「僕は、この世界の管理者として——世界樹を守る！」

kaito.activateAdministratorMode();
kaito.grantAuthority("world_protection");

王冠の光が、世界樹を包む。

アークライトの黒い魔力と、カイトの光がぶつかり合う。

「うおおおおおお！」

カイトが叫ぶ。

力と力の激突。

世界樹の根元で、光と闇の戦いが続く。

そして——

光が、闇を押し返し始めた。

「な、何！？」

アークライトが驚愕する。

「なぜ、これほどの力が——！」

カイトの後ろで、仲間たちが祈るように手を合わせている。

エリナ、セシリア、リアム、ヒナタ——四人の想いが、カイトに力を与えていた。


const bondPower = party.getBondLevel();
kaito.boost(bondPower * 10);
// バフ: 仲間の絆により、攻撃力が500%上昇
「これが——絆の力！」

カイトの剣が、アークライトの闇を貫く。

「うあああああああ！」

アークライトが吹き飛ばされ、世界樹から引き離された。


arklight.takeDamage(300);
arklight.currentHP -= 300; // 250 -> -50
// アークライト撃破

アークライトは倒れた。

黒い霧が晴れ、世界樹から漏れ出ていた負の力が消える。

カイトは膝をついた。

「……勝った」

エリナが駆け寄る。

「カイト！ 大丈夫！？」
「ああ……ちょっと、魔力を使いすぎただけ」

カイトは世界樹を見上げる。

枯れかけていた葉が、徐々に緑を取り戻している。

「回復してる……よかった」

セシリアが安堵の息を吐く。

「世界樹の生命力、すごいね。少しでも根が残ってれば、再生できる」

リアムがアークライトに近づく。

「こいつは、どうする？」

アークライトは気を失っていた。

カイトは王冠を使い、アークライトを拘束する。


kaito.castSpell("sealing_chain", { target: arklight });
// アークライトを魔力封鎖状態にしました
「とりあえず、封印しておこう。後で、正式な裁判を受けてもらう」

ヒナタが頷く。

「そうだね。殺すより、罪を償わせた方がいい」

カイトは立ち上がる。

そして、世界樹に向かって頭を下げた。

「守れてよかった……」

その時——

世界樹が光り始めた。

幹のルーン文字が、優しく輝く。

そして——

樹の中から、声が聞こえた。

『ありがとう……継承者よ』

カイトが顔を上げる。

「この声……世界樹？」

『そうじゃ。我はこの世界の根源。すべての生命と魔力の源』

エリナが驚く。

「世界樹が——喋った！」

『継承者よ。そなたが我を守ってくれたこと、心から感謝する』

『礼として——そなたに力を貸そう』

世界樹から、一粒の種が落ちてきた。

カイトがそれを手に取る。

「これは……？」

『我の分身じゃ。そなたが管理する地に植えれば、新たな世界樹として育つ』

『その地は、魔力が豊かになり、あらゆる生命を育むだろう』

カイトは種を大切にしまう。

「ありがとう、世界樹」

『さらばだ、継承者。そなたの旅に、祝福あれ』

光が消える。

世界樹は静かに佇んでいた。

カイトたちは、アークライトを縛り上げ、始原の森を出た。

空を見上げると、夕日が差し込んでいた。

「長い一日だったね」

セシリアが呟く。

「でも、いい終わり方だった」

エリナが笑う。

カイトは空の彼方を見る。

戦いは終わった。

でも——

旅は、まだ終わらない。

「さあ、帰ろう。みんな待ってる」

カイトが言う。

五人は、夕日の中を歩き出した。
