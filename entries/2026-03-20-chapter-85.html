第85章：調和の試練

二つの旋律が、塔の中で交差した。

カイトが紡ぎ出した音は、アウローラの歌と重なり、新たな調和を生み出していた。風の音、精霊のささやき、そして何か——古代の記憶のようなものが、空気の中に溶け込んでいく。

カイトは、目を閉じたまま、心の中でコードを紡いでいた。


function harmonize(melody1, melody2) {
  return melody1.frequencies
    .map((freq, i) => {
      const partner = melody2.frequencies[i] || freq;
      return (freq + partner) / 2;
    });
}

魔法のコードが、音の波を可視化する。二つの波形が、次第に一つに重なっていく様子が見えた。

アウローラの歌が、わずかに変化した。

驚き。あるいは、喜び。

感情を持った守護者——グレゴリウスの言葉が、カイトの脳裏をよぎった。彼女は、単なる試験官ではない。彼女自身が、この試練に何かを感じている。

やがて、歌は終わった。

塔の中に、静寂が戻る。

カイトは、ゆっくりと目を開けた。目の前には、アウローラが立っていた。その無表情だった顔に、今は微かな笑みが浮かんでいた。

「……想定外です」

アウローラが、呟くように言った。

「私の歌に、完全に調和した回答。過去に、それを成し遂げた者はいません」
「過去に、誰か来たことがあるのか？」

リアムが、警戒を解かずに問うた。

「はい。六十七名の方々が、この塔を訪れました。そのうち、試練に挑んだのは四十三名。全員、失敗しました」

アウローラは、淡々と数字を語った。

「失敗した者は、どうなった？」

エリナが、おそるおそる聞いた。

「命は奪いません。ただ、塔から追放しました。二度と入れないように、記憶を調整して」
「記憶調整……！」

セシリアが、顔を青くした。

「それって、洗脳ってこと？」
「正確には、特定の記憶領域へのアクセス制限です。危害を加えるつもりはありませんが、鍵に近づく者を野放しにはできません」

アウローラの説明は、機械的だった。だが、その言葉の端々に、迷いのようなものが感じられた。

カイトは、一歩前に出た。

「アウローラ。君は、この試練についてどう思ってる？」

アウローナは、首を傾げた。

「どう思っている、とは？」
「失敗した人たちを、追放すること。記憶を調整すること。それについて、何か感じることはないか？」

沈黙が流れた。

アウローラの瞳——作り物のような、光のない瞳——が、わずかに揺れた。

「……私は、守護者です。使命を遂行することが、私の存在意義です」

言葉は、規定通りの答えだった。だが、その直後——

「ですが……」

彼女は、言葉を継いだ。

「時々、疑問を感じます。本当に、これが正しいのか。失敗した者たちの、諦めきれない表情。それを消去する瞬間の、胸の奥の痛み。……これが、感情というものなのでしょうか」
「ああ、それが感情だよ」

カイトは、優しく言った。

「正しいかどうか悩むこと。誰かを傷つけたことに痛みを感じること。それが、人間ってことだ」
「人間……」

アウローラは、自分の手を見つめた。

「私は、人間ではありません。收斂者たちが作った、守護者です。感情を移植されただけの、模造品です」
「模造品だとしても、悩んでるのは君だ。痛みを感じてるのも君だ。それは、本物だよ」

カイトの言葉に、アウローラは顔を上げた。その表情に、戸惑いが浮かんでいた。

エリナが、小声でリアムに囁いた。

「カイト、何してるの？敵を説得しようとしてる？」
「……いや、ただの世間話じゃない気がする」

リアムは、剣の柄に手をかけたまま、警戒を続けていた。

「感情を持った守護者——グレゴリウスが言ってた通りだ。だが、それが罠かもしれない」
「罠？」

セシリアが、不安そうに聞いた。

「ああ。敵を油断させるための演技かもしれない。カイトが信じ込んだところで、不意打ちをかけるつもりかも」
「そんな……」

エリナは、カイトとアウローラの会話に視線を戻した。

「でも、あの表情……演技には見えないわ」

確かに、アウローラの表情には、作り物めいた感じがない。彼女は、本当に悩んでいるように見えた。

カイトは、アウローラに向き直った。

「アウローラ。鍵を渡すかどうか、君が決めるんだろ？」
「はい。試練をクリアした者に、鍵を渡すのが私の使命です」
「じゃあ、俺はもうクリアしたってことか？」

アウローラは、少し考え込んだ。

「……本来は、歌への応答だけで判断します。あなたの応答は、完璧でした。よって、鍵を渡す条件は満たしています」
「本来は？」
「ですが、追加の試練を課したいと考えています」

カイトは、眉をひそめた。

「追加の試練？鍵を渡す条件は満たしてるんじゃないのか？」
「条件は満たしています。ですが……」

アウローラは、言葉を探すように沈黙した。

「あなたたちが、鍵を使おうとしているのは知っています。世界核へと辿り着き、世界のシステムを書き換えようとしている。それが、正しいことなのか——確かめたいのです」
「正しいことかどうか、俺にもわからない」

カイトは、正直に言った。

「でも、今の世界が壊れてるのは事実だ。セキュリティと呼ばれる存在が、真実を知る者を消している。精霊たちが、世界から消えつつある。このままだと、世界は崩壊する」
「崩壊……」

アウローラが、呟いた。

「收斂者たちが、最も恐れていた事態です」
「收斂者たちは、何を知っていたんだ？」

カイトの問いに、アウローラは窓の方を向いた。塔の窓からは、山々の景色が見える。風が、彼女の銀色の髪を揺らした。

「彼らは、世界の終わりを予見していました。システムの老朽化、魔力の枯渇、そして——『管理者』の不在」
「管理者？」

カイトは、聞き覚えのある言葉に反応した。

「エーテリア——俺たちが会った管理者と関係があるのか？」
「エーテリア……その名前、データベースにあります。かつて、世界を統括していた管理者の一人。ですが、彼女は——」

アウローラは、言葉を切った。

「彼女は、何百年も前に、姿を消しました」
「姿を消した？」
「はい。理由は不明です。ただ、彼女がいなくなってから、世界のシステムは不安定になり始めました。收斂者たちは、管理者の代わりになる存在を作ろうとしましたが、失敗しました。そして、彼らは方針を変えました」

アウローラの声が、低くなった。

「世界を終わらせることを、選んだのです」
「何だって？」

マルクスが、声を上げた。

「收斂者たちは、世界を救おうとしてたんじゃないのか？」
「救うことを諦めたのです。世界の崩壊は避けられないと判断し、崩壊後の世界に備えることにしました。彼らは、世界核へのアクセスを封印し、鍵を五つの場所に分散させました。そして、守護者である私たちに、鍵を守らせたのです」
「なぜ、封印したんだ？」

カイトが問うと、アウローラは首を振った。

「世界核に干渉できる存在が現れた時、その者に判断を委ねるためです。世界を救うか、終わらせるか——收斂者たちは、それを後世に委ねました」

カイトは、言葉を失った。

收斂者たち——マルクスもかつて所属していた組織——は、世界を救うことを諦めていた。そして、誰かが世界を終わらせる選択をする可能性も、受け入れていた。

「……じゃあ、俺たちが世界核に辿り着いて、世界を救わない選択をしたら？」
「その時は、世界は終わります。私は、それを止める権限を持ちません」

アウローラは、静かに言った。

「ですが、あなたがたが世界を救う選択をした場合、鍵を渡す意味があります。だからこそ——確かめたいのです」

彼女は、カイトに向き直った。

「あなたは、世界を救う意志を持っていますか？」

カイトは、答えに詰まった。

世界を救う。それは、漠然とした目標だった。具体的に何をすればいいのか、まだわかっていない。世界核に辿り着いて、何を書き換えればいいのかも。

「……正直に言う。わからない」

カイトは、認めた。

「世界を救う方法は、まだ見つけてない。ただ、このままだと世界が終わる——それだけは、確かだ」
「では、なぜここへ？」
「進むしかないからだ。後戻りしても、何も解決しない。前に進んで、答えを探すしかない」

アウローラは、カイトの目を見つめた。その光のない瞳が、何かを探るように動く。

やがて、彼女は微笑んだ。

「……いい答えです」

アウローラは、塔の奥へと歩き出した。

「ついてきてください。鍵を渡します」

カイトたちは、顔を見合わせた。拍子抜けしたような、安堵したような表情。

「追加の試練は？」

エリナが聞くと、アウローラは振り返った。

「終わりました。あなたの仲間の答えが、十分でした」
「俺の答え？」

カイトが首を傾げると、アウローラは頷いた。

「『わからない』という答え。それが、最も誠実な答えでした。世界を救う方法を知っているという嘘、あるいは、救う気がないという本音——どちらも、信頼に値しません。ですが、わからないと認め、それでも前に進もうとする意志。それこそが、私が求めていたものです」

カイトは、苦笑した。

「……なんだか、試されたのは俺の人格みたいだな」
「その通りです。鍵は、世界核への道を開くものです。その力を持っていいのは、責任を自覚している者だけ」

アウローラは、螺旋階段を登り始めた。

塔の最上階に、小さな部屋があった。

部屋の中央には、台座が置かれている。その上に、一本の杖が浮いていた。

透明なクリスタルのような素材。中には、風の模様が揺らめいている。

「これが、第二の鍵——『風鳴りの杖』です」

アウローラが説明した。

「世界核への封印を解除する際、五つの鍵が必要です。この杖は、風の属性を持つ鍵です」

カイトは、杖に手を伸ばした。

指先が触れた瞬間、視界が揺れた。

——風の音が聞こえる。

——遠く、遠く、彼方から。

——世界の、果てから。

——誰かが、呼んでいる。

カイトは、ハッとして手を引っ込めた。

「今の、何？」
「鍵が、あなたに反応したのでしょう」

アウローラが、静かに言った。

「風鳴りの杖は、世界核と共鳴しています。持ち主が世界核に近づくと、杖は反応し、道を示します」

カイトは、再び杖に手を伸ばした。今度は、躊躇なく握った。

冷たい感触。そして、微かな振動。

「……受け取るよ」

カイトは、杖を腰に吊るした。

「アウローラ、ありがとう。これで、また一歩前進できる」
「いえ。あなたがたが、正しい選択をすることを祈っています」

アウローラは、深く頭を下げた。

塔を出たカイトたちは、山を下り始めた。

夕日が、西の空を赤く染めている。

「第二の鍵、ゲットだな」

リアムが、安堵の息を吐いた。

「残り三つ。全部集まったら、世界核へ行ける」
「でも、どこにあるんだろう、残りの鍵」

セシリアが、首を傾げた。

「グレゴリウスは、五つの鍵が必要って言ってたけど、場所は知らないって」
「フィオナに聞いてみよう」

エリナが提案した。

「彼女なら、何か知ってるかもしれない」

カイトは、頷いた。

「そうだな。一度、王都に戻ろう。マルクス、馬は大丈夫か？」

「ああ。まだ十分走れる」

一行は、山道を下っていった。

その背中を、アウローラが塔の窓から見送っていた。

「……世界核への道。誰が辿り着くのか」

彼女は、窓枠に手を置いた。

「そして、どんな選択をするのか」

風が、塔の中を吹き抜けた。

アウローラの銀色の髪が、揺れる。

彼女の胸元の装置が、微かに光った——まるで、心臓の鼓動のように。

王都へ戻る途中、カイトは杖を見つめていた。

透明なクリスタルの中で、風の模様が揺らめいている。

「ねえ、カイト」

エリナが、馬を寄せてきた。

「アウローラとの会話、何か気にならなかった？」
「何が？」
「收斂者たちが、世界を終わらせることを選んだって話。マルクスは、それを知ってるのかな」

カイトは、前方を行くマルクスの背中を見た。

元收斂者。組織の内部事情を知る男。だが、彼がどこまで知っているのか——まだ確認していない。

「……聞いてみる価値があるな」

カイトは、馬を加速させた。

「マルクス、少し話がある」

マルクスが、振り返った。

「何だ？」
「收斂者について、もっと教えてくれ。あんたが知ってること、全部」

マルクスは、沈黙した。

夕日が、二人の影を長く伸ばしていた。

「……わかった。王都に戻ったら、話そう」

マルクスの声には、重みがあった。

彼もまた、背負っているものがある——カイトは、そう感じた。
