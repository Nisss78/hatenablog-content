第884章：地下遺跡の予兆

聖都エルディアの西地区にある宿「星見の亭」。

カイトたちは、その二階にある部屋を借りることにした。本庁舎に近く、かつ人目につきにくい場所を選んだのだ。部屋は広く、六人ほどが快適に過ごせる造りだった。窓からは、夕日に染まった聖都の街並みが一望できる。

「部屋、綺麗だね。中央ってやっぱり違う」

エリナが、ベッドに腰を下ろしながら言った。確かに、アルヴァレス支部周辺の宿とは、部屋の広さも内装の質も段違いだった。

「値段も段違いだけどな」

リアムが、苦笑しながら財布を確認した。

「ま、今回はギルドが経費を持つって言ってたから大丈夫だろ」
「リアム、お金のことばかり気にしないの」
「現実的な問題だろ！冒険者は金が命なんだよ」

セシリアが、窓辺に立って外を眺めていた。

「……街の警備、厳重ね」

カイトも窓に近づき、通りを見下ろした。確かに、街角ごとに衛兵が配置され、定期的に巡回しているのが見えた。魔法の灯りが等間隔に設置され、夜でも明るい。

「聖都だからな。王都でもあるし、中央ギルドの本拠地でもある。警備が厳重なのは当然だ」
「ええ。でも——」

セシリアが、声を潜めた。

「見張られている気がするの」

カイトは眉をひそめた。セシリアの勘は、これまで何度も的中してきた。彼女がそう感じるなら、何かがある。


class SurveillanceCheck {

        this.threat = '不明・要警戒';

    }
    
    analyze() {
        return `
            セシリアの警告。
            過去の実績から信頼性は高い。
            
            中央ギルドの監視か？
            それとも、別の勢力か？
            
            部屋での会話は控えめに。
            重要な話は外で。
        `;
    }
}
「分かった。部屋では重要な話はしないようにしよう」

カイトが小声で言った。全員が頷いた。

夕食は、宿の一階にある食堂で取ることにした。

カイトたちは、部屋の隅にあるテーブルに座った。周囲には、商人や冒険者、旅人たちが食事をしている。賑やかな笑い声や会話が飛び交っていた。

「ここなら、話しやすいね」

エリナが、スープを啜りながら言った。

「周りの音で、会話が聞かれにくい」

カイトは頷き、本題を切り出した。

「ヴェルディア局長の依頼、どう思う？」
「罠の可能性が高い」

セシリアが即答した。

「古代の遺跡調査なんて、中央ギルドの専門部隊でできるはず。わざわざ外部の冒険者に頼む理由がない」
「カイトの能力を試している、っていうのはどう？」

リアムが言った。

「管理者の権限がどれくらいのものか、中央ギルドも知りたいんだろ」
「それも考えられる。でも——」

シオンが、静かに口を開いた。

「遺跡には、管理者の血脈にしか開けない扉があるかもしれない」

カイトは考え込んだ。シオンの言うことも一理ある。最初の管理者たちが残した施設なら、特定の条件を満たさないと入れない仕掛けがあるかもしれない。

「だとしたら、私たちが行く意味はある」
「でも、危険よね」

エリナが心配そうに言った。

「異常な魔力反応があるってことは、何かが起きているってことだし」
「危険なのは承知の上だ。でも——」

カイトは言葉を切り、仲間たちの顔を見回した。

「断ったとして、中央ギルドとの関係が悪化する可能性がある。それに、遺跡になら、私の両親や、シオンの母親に関する手がかりがあるかもしれない」

シオンの瞳が、わずかに揺れた。

「……母様の手がかり」

シオンの母、アルケミア。第一の調律者と呼ばれた存在。彼女についての情報は、ほとんど残っていない。遺跡に何らかの記録があれば、シオンの過去を知る手がかりになるかもしれない。

「行くかどうかは、明日決めよう。今夜は情報を整理して、準備を進める」

カイトの提案に、全員が頷いた。

食事を終えた後、カイトは一人で宿を出た。

夜の聖都を少し歩いてみたいと思ったのだ。頭を冷やして、今後の方針を考えたかった。

通りは、魔法の灯りで明るく照らされている。店は閉まっているが、酒場からは賑やかな声が聞こえてきた。カイトは、人通りの少ない路地に入った。

——と、その時だった。

「カイトさんですね」

背後から声がした。カイトは振り返った。

路地の暗がりから、一人の人物が現れた。フードを被った男。年齢は二十代半ばくらいに見える。身につけているのは、粗末な旅装束だ。しかし、その目には鋭い光が宿っていた。

「あなたは？」
「名前は……フィンと呼んでください。中央ギルドの元調査員です」

カイトは警戒を強めた。

「元調査員？なぜ私に接触してくる」
「警告したかったからです」

フィンと名乗った男は、周囲を確認してから、声を潜めた。

「地下遺跡に行くのはやめた方がいい。罠です」
「罠？具体的にどういうことだ」
「中央ギルドの上層部には、あなたの能力を危険視している勢力がいる。彼らは、遺跡の深部にある『封印の間』にあなたを誘い込み、そこで——」

フィンの言葉が途切れた。彼が急に顔をしかめ、胸を押さえたからだ。

「くっ……時間がない。これを読んでください」

フィンは、小さな羊皮紙をカイトに押し付けた。

「詳しいことは、ここに書いてあります。信じるかどうかは、あなたの判断です。ただ——」

フィンは、苦しげな表情で言った。

「ヴェルディア局長を信用しないでください。彼女は——」

その時、路地の入り口から足音が聞こえた。衛兵の巡回だ。フィンは慌ててフードを引き上げ、暗がりへと消えていった。

「待て！」

カイトが呼び止めたが、男の姿はもうどこにもなかった。

残されたのは、一枚の羊皮紙だけ。

カイトは宿に戻り、自室で羊皮紙を開いた。

手書きの文字。震えるような筆跡だが、内容は明確だった。


【警告書】

中央ギルド総括局長ヴェルディア・クロフォードについて。

彼女は、十三使徒の残党と密接な関係を持っている。
過去三年間、少なくとも五人の冒険者が、
「遺跡調査」という名目で地下深部に送られ、
帰らぬ人となった。

彼らは皆、特殊な魔力回路を持っていた。
管理者の資質、あるいはそれに近い能力を持つ者たちだ。

遺跡の最深部には、「継承の間」と呼ばれる部屋がある。
そこで何が行われているのかは不明だが、
恐らく、あなたの管理者の権限を奪おうとしている。

ヴェルディアを信用するな。
遺跡には行くな。

もし行くなら、十分な準備をして、
必ず仲間と共に。

——フィンより

カイトは、羊皮紙を握りしめた。

「ヴェルディアが……十三使徒の残党と？」

信じがたい話だ。しかし、完全に否定もできない。中央ギルドの内部事情は、まだ何も分かっていない。敵は必ずしら外にいるとは限らない。

カイトは、羊皮紙を魔法の袋にしまった。燃やしてしまうのは惜しい。証拠として残しておくべきだ。

窓の外を見ると、夜空には星が輝いていた。

聖都エルディア。美しい街並みの下に、何が隠されているのか。地下遺跡、ヴェルディアの真意、そして謎の警告者フィン。

謎は深まるばかりだ。

「……慎重に進めよう」

カイトは、独り言のように呟いた。

明日は、情報収集の日だ。中央ギルドの図書館や、街の噂話、過去の記録——あらゆる手がかりを探し出す。

その結果如何によっては、遺跡調査を断ることも考えなければならない。

翌朝。

カイトは早起きして、宿を出た。まだ薄暗い時間帯。通りには誰もいない。

目的の場所は、聖都の中央図書館。中央ギルドが管理する、世界最大級の蔵書を持つ施設だ。一般にも開放されているが、一部の資料は許可がないと閲覧できない。

図書館は、まだ閉まっていた。開館は朝八時だ。カイトは、入り口の前にあるベンチに座り、開館を待った。

冷たい朝の空気が、頬を撫でる。

——と、背後からまたしても気配がした。

「早起きですね」

振り返ると、昨日の男——フィンが立っていた。今日はフードを被っていない。茶色の髪に、疲れたような目。顔立ちは整っているが、どこか影が差している。

「またか。何をしにきた」
「追加の情報です」

フィンは、カイトの隣に座った。

「昨夜の警告書、読んでくれましたか？」
「読んだ。だが、信じる材料がない」
「でしょうね。だから、証拠を持ってきました」

フィンは、懐から一冊のノートを取り出した。表紙には「調査記録・第三巻」と書かれている。

「私が、三年前に遺跡調査のチームにいた時の記録です。何が起きたのか、全て書いてあります」

カイトは、ノートを受け取った。

「なぜ俺に協力する？あなたにとってメリットはないはずだ」

フィンは、遠くを見つめるような目で答えた。

「私の妹が——三年前の調査で、帰ってきませんでした」

カイトは、息を呑んだ。

「彼女は、魔力回路が特異だった。管理者の資質があると診断されて、遺跡調査に選ばれたんです。そして、二度と戻らなかった」

フィンの声には、押し殺した怒りと悲しみが滲んでいた。

「真相を知りたい。そして、ヴェルディアを裁きたい。そのために、あなたに協力しているんです」

カイトは、ノートを強く握りしめた。

これは、個人的な復讐心による情報提供かもしれない。しかし、それが真実でないという保証もない。

「……分かった。このノート、読ませてもらう」
「お願いします。ただ、くれぐれも慎重に。ヴェルディアは、裏を探る者を決して許さない」

フィンは立ち上がり、去り際に一言だけ言い残した。

「図書館の地下書庫に、遺跡の設計図があるはずです。閲覧には許可がいりますが——管理者の権限があれば、入れるかもしれません」

そして、彼は再び人混みの中に消えていった。

カイトは、ノートを見つめた。

聖都の朝が、静かに始まろうとしていた。

手の中には、一冊のノート。

そして、胸の中には、重い予感。
