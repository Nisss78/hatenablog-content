第584章：未来の覚悟

光が収束し、一行は再び新しい世界へと引き込まれた。

今度の光景は、先ほどの懐かしい日本家屋とは全く異なっていた。

そこは——

荒廃した世界だった。

空は鉛色に濁り、地面はひび割れている。かつて街だったと思われる場所は、瓦礫の山と化していた。遠くで、黒い煙が立ち上っている。

「ここは……」

エリナが、震える声で呟く。

「私たちの世界？ でも、こんな……こんな姿になってるなんて」

「未来だ」

リアムが、冷静に——しかし重い声で言う。

「アーカイブが言った『未来の覚悟』。これは、可能性としての未来だ」

セシリアが、瓦礫を調べながら分析する。

「炭素年代測定のような感覚で言うと……私の推測では、三百年から五百年後。世界再構築が完了した後の世界」

シオンが、悲しげに空を見上げる。

「人がいない。風の音しか聞こえない。生命の気配が、ほとんど消えてる」

カイトは、足元の瓦礫を拾い上げる。

それは、かつて魔法陣が刻まれていた石板の破片だった。見覚えがある——アルヴァレス支部のギルドで見たものと同じデザインだ。

「これ、アルヴァレスギルドの紋章……」

class FutureWorld {
    year = "ESTIMATED_300-500_YEARS_AHEAD";
    civilizationStatus = "COLLAPSED";
    magicResidue = "DEPLETED";
    humanPresence = "CRITICAL_MINIMUM";
    
    cause = "WORLD_RECONSTRUCTION_COMPLETED";
    // 因果の書き換えが完了し
    // 元の世界が「上書き」された結果
}

「因果の書き換えが完了した世界……」

カイトは、瓦礫を落とす。

「これが、俺たちが止めなきゃいけない未来」

□ ■ □

一行は、瓦礫の中を歩き続けた。

時折、奇妙な生物の死骸が見つかった。かつての魔物とは異なる——歪で、不安定で、形を保てずに凝固したような存在。

「因果の歪みから生まれた失敗作」

セシリアが、死骸を観察する。

「世界再構築の過程で、因果の整合性が取れなかった生命体。おそらく、一瞬で誕生し、一瞬で死んだのでしょう」
「残酷だな」

リアムが、苦々しげに言う。

「生き物を生み出しておいて、失敗作として扱うなんて」

カイトは、これらの死骸に手を合わせる。

「因果の書き換えは、生物学的な進化を無視している。ただ『結果』だけを求めて、『過程』を排除しているんだ」

「過程を排除……」

エリナが、その言葉を反芻する。

「そういえば、アレクシス・ヴェイルの記憶でも、そんなことを言ってたわね。『完全な世界を作るには、不完全な過程は邪魔になる』って」
「間違っているわ」

シオンが、強い口調で言う。

「過程こそが大切なの。私たちが今まで歩んできた道、出会った人たち、乗り越えた試練——それ全部が、今の私たちを作ってる。過程を消したら、私たちは誰でもなくなっちゃう」

「シオンの言う通りだ」

カイトは、シオンの肩に手を置く。

「過程を大事にするからこそ、未来を変えられる。この荒廃した世界を、絶対に実現させない」

□ ■ □

さらに進むと、少し開けた場所に出た。

そこには、一本の枯れ木が立っていた。

かつては大木だったのだろう——太い幹は、今は黒く変色し、枝はすべて折れている。だが、その根本には、小さな芽があった。

「芽……？」

エリナが、驚いて駆け寄る。

「こんな荒廃した世界で、生きてるの？」

シオンが、芽に触れる。

「生命力を感じる。弱いけど、確かに生きようとしてる」

カイトは、コード・ビジョンで芽を観察する。

class SmallSprout {
    species = "UNKNOWN_ADAPTED_PLANT";
    vitality = 0.12; // 限界ギリギリ
    adaptation = "EXTREME";
    
    mutationHistory = [
        "RADIATION_RESISTANCE_ACQUIRED",
        "LOW_NUTRIENT_ADAPTATION",

    // この芽は、過酷な環境に適応して生き延びた
    // 数百万年かかる進化を、数百年で成し遂げた
}

「すごい……この芽は、ものすごい速度で進化して生き延びてる」

カイトが感嘆すると、セシリアが頷く。

「生命の執念ね。どんなに過酷な環境でも、生きようとする意志がある限り、生命は道を見つける」

リアムが、剣を鞘に収める。

「俺たちも、そうあるべきだ。どんなに絶望的な状況でも、諦めずに戦い続ける」

カイトは、枯れ木と芽を見つめる。

「過去の遺産と、未来の希望。この二つが、同じ場所に存在している」

「象徴的ね」

エリナが、静かに言う。

「私たちもそう。過去の経験を背負いながら、未来を作っていく」

□ ■ □

その時——

空から、奇妙な音が聞こえた。

機械的な音。何かが、空中で作動する音。

一行が空を見上げると、そこには巨大な構造物が浮かんでいた。

黒い金属で作られた、球体に近い形状の物体。表面には、無数の光が点滅している。まるで、コンピューターのサーバー室のような——

「あれは……」

カイトは、本能的に杖を構える。

「因果の管理システム……いや、もっと正確に言えば、世界再構築の実行装置」

class ReconstructionOrb {
    purpose = "WORLD_OVERWRITE_EXECUTION";
    altitude = 5000; // 五キロメートル上空
    powerLevel = "CRITICAL";
    
    functions = [
        "CAUSALITY_ANALYSIS",
        "REALITY_REWRITE",
        "EXISTENCE_TERMINATION",

    ];
    
    // この装置が、世界を「上書き」している
    // 元の因果を消去し、新しい因果を書き込む
}

「世界を上書き……」

シオンが、不安そうに空を見つめる。

「あれが、この世界をこんな姿にしたの？」

「可能性としての未来においては、そう」

カイトが答える。

「俺たちが失敗したら、あれが完成する。そして、この世界が実現する」

「絶対にダメ」

エリナが、強い意志を瞳に宿す。

「こんな未来、絶対に認めない」
「俺もだ」

リアムが、剣の柄を握りしめる。

「この可能性を、消し去る。そのために戦う」

セシリアが、冷静に分析を続ける。

「ですが、気をつけて。これは試練よ。単に『否定する』だけでは、クリアできない可能性がある」

「どういうこと？」

カイトが問うと、セシリアは空中の装置を指差した。

「あの装置を見て。ただの破壊機械ではない。秩序を作ろうとしている」

カイトは、もう一度装置を観察する。

確かに——装置は、ただ破壊しているわけではない。瓦礫を分解し、新しい物質に再構築している。黒い煙を吸い込み、浄化した空気を排出している。

「秩序の破壊と創造……」

「世界再構築は、単なる破壊ではない」

セシリアが解説する。

「元の世界を『不完全』と判断し、『完全な世界』に作り変えようとしている。動機自体は、善悪で言えば善に近いかもしれない」
「でも、結果はこれだ」

エリナが、瓦礫を指差す。

「完全な世界を作ろうとして、生命をほぼ全滅させた。動機がどうあれ、結果は最悪」

「過程と結果の矛盾だ」

カイトが、静かに言う。

「良い動機で始めたことが、必ずしも良い結果になるとは限らない。過程を軽視した結果、取り返しのつかないことになる」

□ ■ □

その時——

空中の装置から、光が放たれた。

光は、カイトたちの方へ降りてくる。

「来るぞ！」

リアムが、剣を抜く。

光が着地すると、そこから人物が現れた。

銀色の髪、紫色の瞳——アーカイブだった。

だが、先ほどのアーカイブとは雰囲気が違う。表情が冷たく、瞳には光がない。

「未来の可能性を観測しました」

未来のアーカイブが、機械的に言う。

「あなたたちは、世界再構築を阻止する可能性を持ちます。よって——排除します」

「待て、これは試練だろ？」

カイトが問うが、未来のアーカイブは答えない。

「試練は終了。以降は、世界再構築の防衛行動」

彼女が手を振ると、無数の光の矢が現れた。

「避けろ！」

カイトが叫び、全員が散開する。

光の矢が、次々と地面を貫く。着弾点は蒸発し、小さなクレーターとなった。

「攻撃魔法かよ！」

エリナが、盾魔法を展開しながら叫ぶ。

「これ、試練の範囲を超えてない？」
「未来のアーカイブは、試練を課す守護者ではない」

セシリアが、分析しながら叫ぶ。

「彼女は、因果の管理システムの一部。脅威と判断した存在を排除する機能を持つ」

「つまり、俺たちが脅威と認定されたってことか！」

カイトは、反撃の魔法を準備する。

class CounterAttack {
    type = "DEFENSIVE_REFLECTION";
    powerSource = "EXISTENCE_WILL";
    
    // 存在する意志を力に変える
    // 過去、現在、未来——すべてを繋ぐ意志
    
    execute() {
        return "未来を変える覚悟を、力に変える";
    }
}

「みんな、聞いてくれ！」

カイトが叫ぶ。

「俺たちは、この未来を否定するだけじゃない。この未来に至らないように、今を変えるんだ！」

「どういうこと？」

シオンが、避けながら問う。

「未来のアーカイブは、俺たちを脅威と判断した。つまり、俺たちには可能性がある。この未来を変える可能性が！」

「可能性があるから、排除しようとしてるってこと？」

エリナが、理解する。

「皮肉ね。私たちが脅威だってことは、私たちが勝てるってこと」

「その通りだ！」

カイトは、杖を高く掲げる。

「未来の覚悟——それは、未来を受け入れることじゃない。未来を変える覚悟を持つことだ！」

□ ■ □

カイトの言葉と共に、彼の体から光が放たれた。

それは、先ほど祖母と再会した時の温かい光とは違う——鋭く、力強い、決意の光。

class FutureResolution {
    source = "KAITO_WILLPOWER";
    type = "CAUSALITY_INTERFERENCE";
    
    resolve = {
        past: "ACCEPTED", // 過去は受容した
        present: "ACTING", // 現在は行動している
        future: "REWRITING" // 未来を書き換える
    };
    
    // 過去・現在・未来を繋ぐ意志
    // これこそが、因果を超える力
    
    power = Math.max(
        resolve.past,
        resolve.present,
        resolve.future
    ); // 三つの意志の最大値ではなく——
    // 三つの積
    power = resolve.past * resolve.present * resolve.future;
}

「過去を受け入れ、今を生き、未来を作る——！」

カイトの放った光が、未来のアーカイブに向かっていく。

光の矢とカイトの光が衝突し、激しい火花を散らす。

「シオン！ エリナ！ リアム！ セシリア！」

カイトが叫ぶ。

「みんなの意志も、貸してくれ！」

シオンが、七属性の光を放つ。

「私は——過程を大事にしたい。出会ったすべての人を守りたい！」

エリナが、盾魔法を攻撃に転じる。

「私は——仲間を守り抜く。どんな未来でも、諦めない！」

リアムが、剣に魔力を込める。

「俺は——騎士として、弱きを守り強きを挫く。その誓いを、未来まで貫く！」

セシリアが、分析魔法を支援に切り替える。

「私は——知識を守り、真実を伝える。未来のために、過去を記録し続ける！」

五つの意志が、一つに重なる。

光は、未来のアーカイブを包み込んだ。

「これが——俺たちの、未来の覚悟！」

□ ■ □

光が収まると、未来のアーカイブの姿が変わっていた。

冷たかった表情が、穏やかになり、瞳に光が宿る。

「……理解しました」

彼女は、静かに言う。

「あなたたちは、単に未来を否定しているのではない。未来を変える意志と力を持っている」

「やっとわかってくれたか」

カイトが、息を切らしながら言う。

「私は、可能性を観測し、脅威を排除する機能でした。ですが——あなたたちは、脅威ではなく、希望です」

未来のアーカイブが、微笑む。

『未来の覚悟』——クリアです。あなたたちは、未来を変える資格を持ちます」

彼女の姿が、光の中に溶けていく。

「願わくば——あなたたちの未来が、この荒廃した世界ではないことを」

□ ■ □

風景が、再び時空の回廊に戻る。

アーカイブ——元の時空の守護者が、満足そうに頷いていた。

「見事です。二つ目の試練、クリア」

「あの未来は……実現しないよね？」

シオンが、不安そうに問う。

「可能性の一つに過ぎません。あなたたちが勝てば、実現しません」

アーカイブが答える。

「ですが、気をつけて。世界再構築の脅威は、あなたたちが想像する以上に強大です」

「わかってる」

カイトが、杖を握りしめる。

「だからこそ、最後の試練に挑むんだ」

「『現在の証明』——私の行動が、今の世界を守るに値するか、証明する」

アーカイブが、最後の試練への扉を開く。

「行こう、みんな。最後の試練だ」

カイトが先頭に立ち、一行は扉の向こうへと歩き出した。
