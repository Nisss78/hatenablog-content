第862章：未来予知の門

翌朝、地下聖域には冷たい空気が漂っていた。カイトは岩の床に座り、前夜の戦闘で疲弊した体を休めていたが、傷の痛みは癒えても心の不安は消えない。

虚無の領域。十三使徒が集めた膨大なエネルギーの行き先。そこで何が起ころうとしているのか、想像するだけで背筋が寒くなる。

「眠れなかったか」

テラドンの低い声が響いた。巨体の賢者は静かにカイトの前に座った。

「考え事をしていました。敵の計画、仲間の安否、自分の力不足……」

「悪くない。考えることは賢者の第一歩だ。だが、考えすぎると行動が遅れる。バランスが重要だ」

カイトは頷いた。テラドンは続ける。

「昨日約束した通り、今日は第二段階の修行に入る。地脈を通じた未来予知だ」

「未来予知……本当に可能なんですか？」

カイトの問いに、テラドンは重々しく頷いた。

「可能だ。だが、完璧ではない。未来は確定したものではなく、無数の可能性の集合体だ。我々が見られるのは、その中から最も確率の高い分岐だけだ」

カイトは頭の中で理論を組み立てた。これは確率論と量子力学の概念に近い。観測することで波動関数が収束するように、未来予知もまた、可能性の中から一つの現実を選び取る行為なのだろう。


// 未来予知のモデル
const FuturePrediction = {
    branches: [],  // 可能な未来の分岐
    probability: [],  // 各分岐の確率
    observed: null   // 観測された最も確率の高い分岐
};

function predictFuture(query, timeWindow) {
    const vein = earth.getNearestVein();
    const branches = vein.queryFuturePossibilities({
        query: query,
        window: timeWindow,
        maxBranches: 1000
    });
    
    // 確率でソート
    branches.sort((a, b) => b.probability - a.probability);
    
    // 最も確率の高い分岐を返す
    return branches[0];
}

「まずは理論を説明しよう」

テラドンは岩の床に指で複雑な図を描いた。地脈の走り方、魔力の流れ、そして時間の軸。

「地脈はこの惑星の神経系だ。過去の記憶を保持するだけでなく、未来の可能性も感じ取れる。大地は全てを知っている」

「全て、ですか？」

「全てだ。だが、大地は言葉を喋らない。我々が問いかけ、答えを読み取るのだ」

テラドンは立ち上がり、地下聖域の奥にある巨大な石扉を指差した。

「あそこに、古の予言者の間がある。そこで未来予知の儀式を行う」

カイトは石扉に近づいた。表面には精緻な彫刻が施され、七つの属性を象徴する紋章が刻まれている。

「これは……」

「最初の管理者たちが残した遺産だ。七賢者が協力して造った、未来を視るための装置だと思えばいい」

テラドンが手をかざすと、石扉が重々しい音を立てて開いた。中から冷たい風が吹き出し、カイトは思わず身震いした。

中は広大な円形の部屋で、中央に巨大な水晶が浮いていた。水晶は淡い金色に輝き、内部で無数の光点が明滅している。

「これが予知の水晶だ。地脈からの情報を可視化する」

カイトは水晶に近づき、その表面に触れた。冷たい感触と共に、微かな振動が指先から全身に伝わる。

「何か感じるか？」

「……温かい、ような。でも同時に冷たい。矛盾しているけど、両方感じます」

「良い感性だ。地脈の性質そのものだ。創造と破壊、生と死、始まりと終わり。全てを内包している」

テラドンは水晶の周りを歩きながら説明を続けた。

「未来予知には三つの禁忌がある。第一に、自分の死を見ようとしてはならない。第二に、愛する者の死を見ようとしてはならない。第三に、世界の終わりを見ようとしてはならない」

「なぜです？」

「精神が壊れるからだ。未来の絶望を知った者は、現在を生きる意志を失う。多くの予言者が狂ったのは、この禁忌を破ったからだ」

カイトは神聖な気持ちで水晶を見上げた。これは単なる魔法の道具ではない。歴代の賢者たちが命を懸けて守り続けてきた、世界の宝なのだ。

「では、始めよう。まずは簡単な問いからだ」

テラドンが水晶に手をかざすと、光点の動きが変化した。

「問い：今から一時間以内に、この聖域を訪れる者はいるか？」

水晶の内部で光が集まり、一つの映像を形成した。地下への入り口付近で、何かが蠢いている。それは影のような怪物で、数は三体。

「敵の斥候だ。三体が接近中」

テラドンは即座に反応し、地下への通路に向かった。

「カイト、来い。実戦訓練だ」

カイトは慌ててテラドンを追った。未来予知の成果が即座に現れるとは思っていなかった。

地下通路の入り口で、三体の影が徘徊していた。鏡の怪物の亜種で、より小型だが動きが素早い。

「お前がやれ。未来予知で動きを先読みするんだ」

テラドンは腕を組んで観戦する構えだ。カイトは緊張を感じながら、影たちと対峙した。


// 戦闘開始：未来予知モード
function combatWithPrediction() {
    while (enemies.length > 0) {
        // 0.5秒先の未来を予知
        const nearFuture = predictFuture(
            "敵の次の行動",
            500  // ミリ秒
        );
        
        // 予知に基づいて回避または反撃
        if (nearFuture.attackDirection) {
            dodge(nearFuture.attackDirection);
            counterAttack();
        }
    }
}

三体目は賢く、距離を取って隙を窺っていた。カイトは相手の思考を読もうとしたが、未来予知にノイズが混じる。

「動揺しているな」

テラドンの声が届いた。

「未来予知には冷静さが必要だ。感情が乱れれば、映像も乱れる」

カイトは深呼吸した。心を静め、三体目の影に集中する。

予知がクリアになった。影はカイトの背後に回り込み、不意打ちを狙っている。

カイトは背後の岩壁から無数の石針を発生させた。影は悲鳴を上げ、光の粒子となって消滅した。

「完了です」

テラドンは満足げに頷いた。

「悪くない。直感的に使いこなしている。才能があるようだ」

カイトは息を整えながら、自分の能力について考えた。未来予知は強力だが、万能ではない。ノイズや確率の誤差がある。それを踏まえた上で、いかに活用するかが鍵になる。

聖域に戻った二人は、再び予知の間に入った。水晶の光が二人を包む。

「次は、より高度な問いだ」

テラドンは表情を引き締めた。

「問い：七日以内に、十三使徒はこの地の聖域を攻撃するか？」

水晶の光が激しく明滅し、複数の映像が浮かび沈んだ。戦いの光景、炎上する街、逃げ惑う人々。そして、平穏な日常の続く姿も。

「見えていますか、カイト？」

「はい……複数の未来が見えます。攻撃がある未来と、ない未来」

「それぞれの確率は？」

カイトは目を細め、光の強さを測った。

「攻撃がある未来……六割。ない未来……四割」

テラドンは深刻な顔つきになった。

カイトは水晶に意識を深く沈めた。映像が鮮明になる。
