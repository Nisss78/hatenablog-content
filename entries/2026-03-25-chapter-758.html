第758章：灼熱の序曲

螺旋階段を登るにつれて、空気が変わっていった。

涼しい風が消え、代わりに熱気が漂い始める。

カイトの額に、汗がにじんだ。

「熱い」

シオンが服の襟を緩めた。

「第四層は、炎の領域だもんね」

エリナも暑そうに扇いでいた。

階段の壁が、赤みを帯びてきた。

岩肌に亀裂が走り、そこから蒸気が漏れている。

「火山の中に入ってるみたいだ」

リアムが壁に触れた。

「熱すぎて、触ってられない」

リアムはすぐに手を引っ込めた。

カイトは温度を分析した。

今はまだ四十度程度。でも、奥に行くにつれて上がるはずだ。

「熱中症対策が必要だ」

カイトは魔法を発動した。


// 熱耐性付与
function heatResistance(targets) {
  const magic = {
    type: "水",
    subtype: "temperature_control",
    targets: targets,
    effect: "体温を一定に保つ",
    duration: "試練終了まで"
  };
  
  return `${targets.join("、")}に熱耐性を付与`;
}

// 実行
heatResistance(["カイト", "リアム", "エリナ", "シオン"]);

四人の体に、薄い水の膜が張られた。

「涼しくなった」

シオンが安堵の息を吐いた。

「感謝、カイト」

リアムも頷いた。

「でも、これで油断しないで。魔法で防げるのは熱だけ。炎そのものの攻撃は防げないから」

カイトは警告した。

階段が終わり、第四層への扉が現れた。

扉は真っ赤な金属製。炎の紋様が刻まれ、微かに脈動している。

「生きてるみたい……」

エリナが扉を凝視した。

「炎の精霊が宿っているんだ」

カイトは感じ取った。

扉に近づくと、熱気が肌を焼いた。

でも、熱耐性魔法のおかげで耐えられる。

「開けるぞ」

カイトは両手で扉を押した。

重い金属音と共に、扉が開いた。

中から、灼熱の空気が吹き出してきた。

カイトたちは思わず目を細めた。

第四層の空間が広がっていた。

そこは、溶岩の海に浮かぶ島々だった。

赤い溶岩が、ゆっくりと流れている。表面では泡が弾け、火花が飛び散る。

島同士は、黒い岩石の橋で繋がっている。橋の下には、灼熱の海が広がる。

「落ちたら、即死だな」

リアムが冷静に分析した。

島々には、奇妙な植物が生えていた。

炎の花。燃える木々。熱気の中で揺れる草。

「この環境で育つ植物があるなんて」

シオンが驚いた。

「火属性の魔力があるからこそ、生きていけるんだ」

カイトは説明した。

カイトたちは最初の島に足を踏み入れた。

足元の岩は温かい。でも、熱耐性魔法があれば問題ない。

「歓迎してくれそうにないな」

カイトは周囲を警戒した。

その時、溶岩の海が泡立った。

赤い流体が盛り上がり、形を成していく。

溶岩ゴーレムだった。

三体のゴーレムが、溶岩から現れた。

「侵入者だ！」

一体目が唸った。

「試練を受ける者か？」

二体目が尋ねた。

「管理者の資格を求める者か？」

三体目が確認した。

「僕たちは、管理者の試練を受けに来た」

カイトは前に出た。

「第四層の守護者に会いたい」

カイトは礼儀正しく言った。

ゴーレムたちが顔を見合わせた。

「ミリアとゼフィルスの試練を突破した者たちだ」

一体目が感知した。

「水と風の認可がある」

二体目も同意した。

「ならば、通せ」

三体目が道を開けた。

ゴーレムたちは溶岩の海に戻っていった。

カイトたちは安堵した。

「戦わずに済んだ」

エリナが胸を撫で下ろした。

「でも、これからが本番だ」

カイトは先を見据えた。

島の中央に、巨大な塔が見えた。

溶岩でできた塔。炎が頂上から噴き出している。

「あそこに、守護者がいるはずだ」

カイトは塔を指差した。

カイトたちは島々を渡っていった。

溶岩の海には、時折マグマの魚が飛び出してくる。

でも、橋の上にまでは届かない。

「この世界、生きている」

シオンが感じた。

「炎そのものが、生命を持っている」

二つ目の島に到着した。

そこには、炎の花畑が広がっていた。

赤、オレンジ、黄色の花々が、熱気の中で揺れている。

「きれい……」

エリナが目を輝かせた。

「でも、近づかないで。その花、触れると火傷するから」

カイトは警告した。

花畑の中から、小さな炎の精霊が現れた。

手のひらサイズ。可愛らしい姿。

「旅人さん？」

精霊がカイトに近づいてきた。

「そうだ。第四層の守護者に会いに来たんだ」

カイトは優しく答えた。

「炎帝さまなら、塔の頂上にいるよ」

精霊は教えてくれた。

「でも、気をつけて。炎帝さまは、強いから」
「炎帝？」

リアムが眉をひそめた。

「第四層の守護者は、炎帝イグニスだよ」

精霊は誇らしげに言った。

「この世界の火を、全部支配できるすごい人」

カイトは以前の冒険を思い出した。

イグニス。火の賢者。かつて出会ったことがある。

「イグニスさんか。知ってる人だ」

カイトは安心した。

「だったら、話が通じやすいかも」
「カイト、炎帝を知ってるの？」

シオンが尋ねた。

「以前、火の賢者として出会ったことがある」

カイトは説明した。

「でも、あの時は短い会話だけだったから、どんな試練があるかはわからない」

炎の精霊は、カイトたちを案内し始めた。

「僕が、塔まで案内してあげる」

精霊は嬉しそうに飛び回った。

「炎帝さまに、いい子たちが来たって伝えなきゃ」

精霊について、花畑を抜けていく。

花々が道を開けてくれる。

「精霊さん、名前は？」

エリナが尋ねた。

「エンバーだよ」

精霊は答えた。

エンバーは、第四層で生まれた炎の精霊だった。

溶岩の海から湧き出る炎から、自然に生まれた存在。

「僕たちは、炎帝さまが作ったんじゃないよ」

エンバーは説明した。

「この世界の炎が、勝手に生まれさせたんだ」

カイトは興味深く聞いた。

この世界の炎は、生命を生み出す力があるのか。

「炎帝さまは、僕たちを守ってくれる」

エンバーは続けた。

「でも、試練を受けに来る人は、僕たちを燃やしちゃうこともある」

カイトは胸が痛んだ。

「僕たちは、精霊たちを傷つけない。約束する」

カイトは真剣に言った。

「本当に？」

エンバーは疑わしそうに見た。

「本当に」

カイトは頷いた。

エンバーは笑顔になった。

「なら、僕が一番いい道を教えてあげる」

エンバーは先導を始めた。

三つ目の島に到着した。

そこは、火山の麓だった。

塔の入り口が見えている。巨大な門が口を開けていた。

「ここから先は、僕は行けない」

エンバーは残念そうに言った。

「ありがとう、エンバー」

シオンが精霊に手を振った。

「また会おうね」

エンバーは花畑に戻っていった。

カイトたちは塔の前に立った。

門の上には、火竜のレリーフが刻まれている。

「炎帝の塔」

カイトは门を見上げた。

門が、ひとりでに開いた。

中から、女性の声が聞こえた。

「来たか、カイト」

「待っていたぞ」

カイトは中に入った。

塔の内部は、溶岩の柱が並ぶ大広間。

壁には炎が燃え続けている。

その奥に、玉座があった。

玉座には、一人の男性が座っていた。

赤い髪に、燃えるような瞳。身に纏うのは、炎を模した鎧。

イグニス。火の賢者にして、第四層の守護者。

「久しぶりだな、カイト」

イグニスは立ち上がった。

「イグニスさん。お久しぶりです」

カイトは頭を下げた。

「管理者の試練を受けに来ました」
「知っている」

イグニスは頷いた。

「ミリアとゼフィルスから報告を受けている」
「では、試練を」

カイトは覚悟を決めた。

「焦るな。まずは、話をしよう」

イグニスは玉座に戻った。

「座れ」

イグニスが手を振ると、四人分の椅子が現れた。

カイトたちは恐る恐る座った。

「炎の試練は、他とは違う」

イグニスは語り始めた。

「水は冷静さを、風は自由を試す」

イグニスは続けた。

「だが、炎は違う。炎が試すのは――」

イグニスは言葉を止めた。

「情熱だ」

カイトは考えた。

情熱。炎が象徴するもの。

「情熱とは、何に対する情熱ですか？」

カイトは尋ねた。

「それを、試練の中で見つけるんだ」

イグニスは意味深に言った。

「管理者として、何を守りたいのか。何のために戦うのか」
「その答えが、炎の試練だ」

カイトは胸に手を当てた。

何のために戦うのか。

何を守りたいのか。

それは、常に問い続けてきたこと。

「試練は、三つある」

イグニスは指を三本立てた。

  • 第一の試練：炎の記憶
  • 第二の試練：灼熱の回廊
  • 第三の試練：情熱の炎
「三つともクリアすれば、第四層通過だ」
「炎の記憶とは？」

カイトが尋ねた。

「過去の炎を、体験する試練だ」

イグニスは説明した。

「かつて、この世界で燃えた炎の記憶。それを見て、学ぶんだ」
「では、始めよう」

イグニスが立ち上がった。

「第一の試練への扉を、開く」

イグニスが指を鳴らした。

広間の奥に、扉が現れた。

炎の模様が描かれた、黒い扉。

「その先に、炎の記憶がある」

イグニスは告げた。

「覚悟して、入りなさい」

カイトは仲間たちを見た。

リアムが頷いた。エリナが微笑んだ。シオンが拳を握った。

「いこう」

カイトは歩き出した。

扉を開けた瞬間、熱波が押し寄せた。

カイトたちは目を細めて進んだ。

扉の向こうは、別の世界だった。

そこは、燃える街だった。

建物が炎に包まれている。人々が逃げ惑う。

空は赤く染まり、灰が降り注ぐ。

「これは……」

カイトは絶句した。

これは記憶だ。過去の出来事。

かつて、どこかで起きた炎の惨劇。

「見るんだ、カイト」

イグニスの声が響いた。

「これが、炎の記憶だ」

カイトは街の中を歩いた。

炎は破壊だけじゃない。人々を助けるために使われることもある。

カイトは、燃える建物から子供を救い出す魔法使いを見た。

魔法使いが炎を操り、安全な道を作っている。

子供を逃がすために、自らの魔力を燃やしている。

「炎は、破壊の道具じゃない」

カイトは悟った。

「守るための力にもなる」

記憶が変わった。

次は、火山の噴火だった。

溶岩が村に迫る。でも、魔法使いたちが壁を作っている。


// 記憶の中の魔法
// 炎の防御壁
function flameBarrier(location) {
  const magic = {
    type: "炎",
    subtype: "barrier",
    location: location,
    effect: "溶岩の流れを食い止める"
  };
  
  return `${location}に炎の防御壁を展開`;
}

魔法使いたちが、命を懸けて村を守っている。

炎の魔法で、溶岩の流れを変える。

破壊の力を、守護の力に変える。

「これが、炎の正しい使い方か」

カイトは理解した。

記憶がまた変わった。

今度は、鍛冶師の工房だった。

炉の中で炎が燃える。鍛冶師が鉄を打つ。

炎は、創造の道具。

鉄を溶かし、形を与え、道具を作る。

剣も、鍬も、鍋も。すべて炎から生まれる。

「炎は、創造の源だ」

カイトは呟いた。

三つの記憶を見終えた。

破壊、守護、創造。炎の三つの顔。

カイトは、炎の本質を理解した。

記憶の世界から、現実に戻った。

イグニスが待っていた。

「どうだった？」

イグニスが尋ねた。

「炎は、破壊だけじゃない」

カイトは答えた。

「守り、創り、生かす力だ」
「その通りだ」

イグニスは満足そうに頷いた。

「第一の試練、クリアだ」

イグニスは宣言した。

「次は、第二の試練。灼熱の回廊だ」

イグニスは新たな扉を指差した。

カイトは深呼吸した。

第一の試練を乗り越えた。でも、まだ二つ残っている。

「準備はいい？」

カイトは仲間たちに尋ねた。

「いつでも」

リアムが答えた。

カイトたちは、第二の扉に向かった。

灼熱の試練が、待っている。

第758章、了。
