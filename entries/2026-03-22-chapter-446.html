第446章：北の山脈

風の賢者シルフィアとの別れから、三日が経過していた。

カイトたちは北へと向かっていた。シルフィアの教えてくれた地の賢者の居場所——北の山脈の奥深くを目指して。

「北の山脈か……。あそこは冒険者でも滅多に行かない場所だよ」

エリナが地図を見ながら言った。

「どうして？」

「地脈が不安定なんだ。山そのものが生きているみたいに動くって言われてる」

「山が動く？」

リアムが眉をひそめた。

「正確には、地形が変わるんだ。昨日通った道が、今日は崖になってたり。地の魔法が暴走してる場所だから、普通の方法じゃ進めない」

カイトは顎に手を当てて考え込んだ。

（地形が動的に変化する……まるで、ランダム生成されるダンジョンみたいだ）

「面白そうだね」

カイトが少し笑うと、エリナとリアムは揃ってため息をついた。

「カイト、お前、そういうところ楽しすぎだろ」

「だって、プログラマーとしては親近感湧くじゃん。変数をどう管理するか、それが鍵になりそう」

「はあ……まあ、カイトがなんとかしてくれるのを信じるよ」

エリナは苦笑いした。

＊＊＊

北の山脈のふもとに到着したのは、昼過ぎだった。

目の前には、険しい山々が連なっている。岩肌は荒々しく、所々から蒸気が立ち上っていた。大地から魔力が漏れ出している証拠だ。

「すごいプレッシャー……」

エリナが身震いした。

「地の魔力が濃い。普通の人間なら、ここにいるだけで体が重くなるはず」

「確かに、少し息苦しいかも」

リアムが肩を回した。

カイトは地面に手を当てた。大地の振動が、指先から伝わってくる。


function analyzeGeoPulse(terrain) {
  const readings = [];
  
  for (const sector of terrain.sectors) {
    const pulse = {
      location: sector.coordinates,
      intensity: sector.magicalDensity,
      stability: calculateStability(sector.history),
      nextShift: predictNextShift(sector.patterns)
    };
    readings.push(pulse);
  }
  
  return readings;
}

カイトの解析魔法が、地脈のパターンを可視化した。

「なるほど……地脈が呼吸してる」

「呼吸？」

「うん。一定の周期で、地形が変化するタイミングがある。今は吸気のフェーズ。だとしたら、次の呼気の前に進まないと」

「どれくらいの時間？」

「およそ二十分。その間に、最初の関門を突破する必要がある」

カイトは先頭に立った。

「行こう。タイムリミットまで、あと十八分」

＊＊＊

山道を登り始めると、すぐに異変が現れた。

道幅が徐々に狭くなり、岩が不自然に盛り上がってくる。まるで、山が「入るな」と拒絶しているかのように。

「これ、普通の登山じゃないね」

リアムが剣を抜いた。

「戦う相手がいないのが厄介だ」

「戦うんじゃなくて、説得するんだよ」

カイトが言った。

「地の賢者は、この山の一部。山が拒絶してるなら、賢者も僕らを歓迎してないってこと」

「じゃあ、どうすれば？」

エリナが聞いた。

「山に『僕らは敵じゃない』って伝える。地の言葉で」

カイトは再び地面に手を当てた。今度は、一方的に解析するだけではない。自分の魔力を大地に送り込み、対話を試みる。


function communicateWithEarth(intention) {
  const signal = {
    type: "greeting",
    source: "visitor",
    purpose: "seeking_wisdom",
    harmLevel: 0,
    sincerity: measureSincerity(intention)
  };
  
  return broadcastToGeoNetwork(signal);
}

カイトの魔力が地脈に溶け込んでいく。しばらく応答がない。だが、徐々に、大地の振動が変わっていった。

「何か……反応が変わり始めた」

エリナが呟いた。

拒絶の振動から、好奇心の振動へ。山が「お前は何者だ」と問うている。

カイトは正直に答えた。

「僕はカイト。境界を修復したい。そのために、地の賢者に会いたいんだ」

言葉を発すると同時に、意図を魔力に乗せて大地に送る。

山が沈黙した。そして——

「……通れ」

声がした。地面から直接響いてくる、低く重い声。

盛り上がっていた岩が退き、道が開いた。

「すごい……」

リアムが目を丸くした。

「山と会話したのか？」

「会話というより、交渉かな。地脈はネットワークみたいなものだから、適切なプロトコルで接続すれば通信できる」

カイトは少し苦笑した。

「プログラマーとしてのスキルが、こういうところで役立つとは思わなかった」

＊＊＊

山道を進むこと数時間。地形は刻々と変化していたが、カイトの解析のおかげで危険な箇所を回避できた。

「次の変化まで、あと三分。その先に何か大きな構造物がある」

カイトが指差した方向には、岩壁に掘られた巨大な洞窟があった。入口には、古代の文字で何かが刻まれている。

「古の言語……」

エリナが deciphering spell を唱えた。

「『大地の子よ、汝の重さを示せ』……？」

「重さ？」

リアムが首をかしげた。

「物理的な重さじゃないと思う」

カイトが洞窟の前に立った。

「地の賢者への試練だ。自分の『覚悟の重さ』を証明しろってことだろう」

カイトは洞窟の中へと足を踏み入れた。中は薄暗く、壁が淡い光を放っている。その光は、歩くにつれて強くなっていった。

そして、洞窟の奥に、一人の人物が座っていた。

老人だった。白い髭を蓄え、岩のようにどっしりとした体格。目は閉じられており、瞑想しているようだ。

「地の賢者……？」

カイトが声をかけると、老人はゆっくりと目を開けた。その瞳は、宝石のように深い茶色だった。

「来たか」

声は静かだが、地響きのように重い。

「我が名はテラドン。地の賢者である」

「はじめまして。僕はカイト。境界の修復に、あなたの力を借りたくて来ました」

「境界の修復か……」

テラドンはゆっくりと頷いた。

「それは重い任務だ。どれほどの重さを背負う覚悟がある？」

「わかりません。でも、やるしかないんです」

「ふむ」

テラドンは立ち上がった。その動作一つで、洞窟全体が揺れた。

「言葉は軽い。重さを証明せよ」

テラドンが手をかざすと、カイトの足元の地面が盛り上がった。そして、巨大な岩石が出現し、カイトの肩に圧し掛かった。

「ぐっ……！」

重い。物理的な重さではない。精神的な重圧だ。

「その岩は、お前の『責任の重さ』だ。境界を修復するということは、世界の運命を背負うということ。その重さに耐えられるか？」

カイトは膝をつきそうになった。だが、耐えた。


function bearWeight(responsibility) {
  const mentalStrength = calculateWillpower();
  const support = sumAlliesStrength();
  
  const capacity = mentalStrength + support;
  
  if (capacity >= responsibility.weight) {
    return "endure";
  } else {
    return "crushed";
  }
}

（エリナ、リアム、そしてこれまでに出会ったみんな……僕は一人じゃない）

カイトは歯を食いしばった。

「重い……でも、投げ出さない！」

「ほう」

テラドンの目に興味が宿った。

「耐えたか。だが、これからが本番だ」

テラドンが両手を広げると、さらに多くの岩が出現した。

「境界の修復には、三人の賢者の協力が必要だと言われた。君たちが集めた四つの賢者の力。火、氷、雷、風。だが、それらはすべて『動』の属性。『静』の属性である地があって初めて、バランスが取れる」

「静……と動？」

「そうだ。火は燃え広がる動、氷は凍てつく静。雷は駆け巡る動、風は吹き渡る動。動ばかりでは、エネルギーが暴走する。地の静があって、初めて制御できるのだ」

カイトは理解した。四つの賢者の力を統合するには、地の力が必須なのだ。

「この重さに耐え続けろ。そして、動くのではなく、静を受け入れろ」

テラドンの言葉に、カイトは目を閉じた。

（動かない。流されない。ただ、在る）

今までカイトは、常に前に進もうとしてきた。問題を解決し、障害を乗り越え、ゴールに向かって突き進む。それが彼のスタイルだった。

だが、今は違う。重さを受け入れる。抵抗せず、ただ存在する。

（山のように。岩のように）

カイトの呼吸が遅くなった。心拍数が下がった。精神が、大地と同調していく。

「……そうだ」

テラドンが満足げに頷いた。

「動を捨て、静を得る。それが地の極意」

岩石が、音もなく消えた。

「合格だ」

＊＊＊

テラドンは、カイトの前に手をかざした。

「地の賢者の力、授けよう」

温かい、重たい感覚が体に入り込んでくる。それは、風の賢者の力とは正反対の感触だった。軽やかさではなく、安定感。動きではなく、根付き。

「これで五つ目だ」

カイトは深呼吸した。体の中に、五种の力が共存している。まだ完全には統合できていないが、確かな手応えを感じた。

「残るは光と闇……」

「光は東の聖堂にいる。名はルミナス。堅物な奴だが、悪いやつではない」

テラドンが言った。

「闇は……特殊だ。完全な闇の中にしか現れない。新月の夜、あるいは地下の最深部。覚悟を決めて探すことだ」

「わかりました」

カイトは礼を言った。

「ありがとうございます、テラドンさん」

「礼には及ばん。お前が境界を修復すれば、地脈も安定する。私にとっても利益がある」

テラドンは少しだけ微笑んだようだった。

「行け。光と闇を迎えに」

＊＊＊

洞窟を出ると、エリナとリアムが駆け寄ってきた。

「カイト！ 大丈夫？」

「うん、なんとか」

カイトは二人に笑いかけた。

「地の賢者の力、手に入れたよ」

「本当！？ やったね！」

エリナが飛びついた。

「これで残り二人か」

リアムが肩を叩いた。

「東の聖堂へ向かおう。光の賢者、ルミナスさんに会いに」

カイトは空を見上げた。夕陽が山々を赤く染めている。

五つの賢者の力を手に入れた。残りは光と闇。

境界修復への道は、さらに近づいていた。
