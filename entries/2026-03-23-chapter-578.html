第578章：世界再構築の野心

永夜の森から戻ったカイトたちは、アルヴァレス支部の会議室に集まっていた。

支部長のテオドールをはじめ、数名の幹部が同席している。カイトがクアドラから聞いた情報を共有するためだ。

カイトは、深呼吸をしてから口を開いた。

「闇の災厄クアドラと、直接対話しました」

その言葉に、室内がざわめいた。

「対話……だと？災厄と？」

一人の幹部が、信じられないという表情で尋ねる。

「はい。封印に干渉せず、意識だけを繋ぐ魔法を使いました」

カイトが説明する。

「彼女から聞いた話を、皆さんにお伝えします」

カイトは、クアドラから聞いた内容を整理して話した。

  • 災厄は、太古の昔に自らの力を恐れて封印を望んだ
  • 十三使徒の最終目的は「世界の再構築」
  • 彼らは暴走した災厄の力を利用しようとしている
  • 正しい解放方法が存在する——徐々に封印を弱めながら、災厄の自我を安定させる

話が進むにつれて、テオドールの表情は深刻さを増していった。

「世界の再構築……」

支部長が、重い声で呟く。

「彼らは、今の世界を否定しているのか」
「そうらしいです。……詳細な理由は聞けませんでしたが、今の秩序を破壊して新しい世界を作ろうとしている」

カイトが答える。

「そのために、災厄の力が必要。……制御できない破壊の力を利用して、世界をリセットしようとしている」

一人の幹部が、机を叩いた。

「狂っている……！暴走した災厄が解放されたら、どれだけの被害が出ると思っている！」
「フィンブルヴェトの例を見れば明らかです」

エリナが口を挟む。

「氷見の村は、ギリギリで防げました。……でも、もしカイトがいなかったら」

彼女の言葉に、全員が黙り込んだ。

テオドールが、静かに口を開いた。

「カイト。君が提案しているのは、正しい解放方法を探ることか？」
「はい。……クアドラは、封印を完全に解くこと自体は否定していません」

カイトが頷く。

「問題は解放の方法です。十三使徒のやり方では、災厄が暴走する」
「だが、正しい方法で解放したとしても……災厄は制御できるのか？」

テオドールが鋭い質問をする。

「彼らは強大な力を持っている。……制御された状態でも、危険な存在には変わりない」
「その通りです」

カイトは、素直に認めた。

「でも、暴走した状態よりはマシです。……それに、クアドラの話では、災厄自身が外の世界を望んでいるケースもある」
「いつか、制御できるようになったら……また外の世界を見たい。彼女はそう言っていました」

シオンが、小さな声で付け加える。

「みんな、ずっと封印されてたんだよ。……寂しかったのかも」

その言葉に、室内の空気が少し和らいだ。

会議は三時間に及んだ。

最終的に、以下の方針が決まった。

  • 災厄に関する情報をさらに収集する
  • 他の災厄の封印場所を特定し、十三使徒より先に到達できるよう準備する
  • カイトの「対話」アプローチを継続し、災厄の真意を探る
  • ただし、安易な解放は避ける——制御方法が確立されるまで封印を維持

テオドールは、最後にカイトに向かって言った。

「君の判断を信じる。……だが、無理はするな」
「はい」
「それと、中央ギルドにも報告する必要がある。……十三使徒の目的が『世界再構築』だという情報は、絶対に共有すべきだ」

カイトは頷いた。

会議が終わった後、カイトたちは支部の食堂で遅い昼食をとっていた。

リアムが、大きなため息をつく。

「世界再構築か……大げさな野望だな」
「今の世界に不満がある人間は多い」

エリナが冷静に言う。

「貧困、差別、戦争……解決されていない問題は山積みだ」
「だからって、世界を壊して作り直せば解決するわけじゃない」

シオンが口を尖らせる。

「壊すのは簡単。……でも、作るのは難しい」

カイトが静かに言った。

「十三使徒は、壊すことしか考えていない。……作るビジョンなんてない」
「どういう意味？」

リアムが首を傾げる。

「彼らが求めているのは『今の世界の否定』だ。……新しい世界がどうあるべきか、具体的なイメージはない」

カイトは、テーブルの上の地図を見つめながら続けた。

「破壊だけの思想。……それが、彼らの弱点だ」

その夜、カイトは一人で支部の図書室にいた。

テオドールから許可を得て、災厄に関する資料を調べている。

深淵の書庫から取り寄せられた古文書。表紙には、『七災厄封印記』と記されていた。

カイトは、ページをめくる。

——七災厄封印記（抜粋）——

七つの災厄は、それぞれが属性の極致に達した存在である。

  • 炎の災厄イグニラ　灼熱の化身。全てを燃やし尽くす炎。
  • 氷の災厄フィンブルヴェト　永遠の氷結。世界を凍らせる冷気。
  • 風の災厄テンペスタ　無限の嵐。全てを吹き飛ばす暴風。
  • 地の災厄テラクウェイク　大地の慟哭。山をも砕く震動。
  • 雷の災厄アルグウェル　天空の雷鳴。神をも焦がす電撃。
  • 光の災厄ルクスフェリア　過剰なる光明。闇を焼き尽くす輝き。
  • 闇の災厄クアドラ　深淵の闇。全てを飲み込む虚無。

これら七者は、太古の戦いにおいて封印された。

封印の主導者は、最初の管理者アレクシス・ヴェイル。

七人の賢者が協力し、七つの封印を施した。

封印場所は、以下の通り。

  • 炎　南の火山帯・紅蓮の霊峰
  • 氷　北の氷原・凍てつく湖底
  • 風　西の風谷・嵐の聖域
  • 地　東の大陸・大地の裂け目
  • 雷　中央の山脈・雷鳴の塔
  • 光　天空の浮島・光の聖域
  • 闇　永夜の森・黒き遺跡

「封印場所が記されている……」

カイトは、情報を書き留めた。

氷の災厄フィンブルヴェトは、既に解放されている。氷見の村で戦ったのが、その残滓だ。

闇の災厄クアドラは、永夜の森に封印されている。そこには既に行った。

残りは五つ。

「炎……風……地……雷……光……」

カイトは、地図に印をつけた。

南の紅蓮の霊峰。西の嵐の聖域。東の大地の裂け目。中央の雷鳴の塔。そして、天空の光の聖域。

「十三使徒は、それぞれの属性の災厄を狙っている。……ルシフェインが闇、グラシエルが氷」

カイトは考え込む。

「だとしたら……炎の使徒が紅蓮の霊峰へ、風の使徒が嵐の聖域へ……」

法則が分かれば、対処できる。

翌朝。

カイトは、テオドールに封印場所の情報を伝えた。

「深淵の書庫の記録に、全ての封印場所が記されていました」
「ほう……！それは貴重な情報だ」

テオドールが地図を広げる。

「残り五つの封印場所。……十三使徒がどこを狙うか」
「彼らの属性と対応しているはずです」

カイトが説明する。

「炎の使徒がいるなら、紅蓮の霊峰を狙うでしょう。風の使徒なら、嵐の聖域」
「我々も、それぞれの場所に警戒を強める必要があるな」

テオドールが頷く。

「ですが、全ての場所に十分な戦力を配るのは難しい。……優先順位を決める必要があります」

カイトは、少し考えた後、口を開いた。

「一番危険なのは……光の災厄です」
「光？」
「光の災厄ルクスフェリアは、天空の浮島に封印されています。……場所が特殊です」

カイトが指摘する。

「天空に浮かぶ島。……通常の方法では到達できません。十三使徒が何らかの手段を持っていない限り、そこは最後になるはず」
「では、地上の四つが優先か」
「はい。特に……紅蓮の霊峰と、大地の裂け目が気になります」

カイトは、仲間たちに作戦を説明した。

「二つのチームに分かれて、封印場所を確認する」

カイトが提案する。

「僕とエリナ、シオンで紅蓮の霊峰へ。リアムは、支部の精鋭と一緒に大地の裂け目を」
「なんで俺が裂け目なんだ？」

リアムが不満そうに聞く。

「大地の裂け目には、地魔法の使い手が必要だ。……リアムの風魔法と併せて、地の賢者の知識を持つ者を同行させる」

テオドールが説明する。

「エドガー・ナイトシェードが同行する。……彼は深淵の書庫の管理官だが、地魔法にも精通している」
「エドガーさんが……」

リアムは、少し驚いたようだった。

「わかった。……任せてくれ」

出発は、翌日の朝と決まった。

その夜、カイトは部屋で準備をしていた。

魔法の杖、治療ポーション、そして——クアドラとの対話用に開発した魔法のコード。


// Sealed Entity Communication Protocol
class SealedEntityLink {
    constructor(target) {
        this.target = target;
        this.connection = null;
    }
    
    async connect() {
        this.connection = await Magic.resonance({
            target: this.target,
            type: "consciousness",
            method: "non_intrusive",
            timeout: 300000
        });
        return this.connection.established;
    }
    
    async sendMessage(message) {
        if (!this.connection) return false;
        return await this.connection.transmit(message);
    }
    
    async disconnect() {
        if (this.connection) {
            await this.connection.close();
        }
    }
}

このコードを応用すれば、他の災厄とも対話できるはずだ。

——クアドラが言っていた。「正しい解放方法」。

封印を徐々に弱めながら、災厄の自我を安定させる。

それを実現する魔法も、開発できるかもしれない。

準備を終えたカイトは、窓の外を見た。

夜空に、月が輝いている。

「……世界を再構築する、か」

カイトは、独り言のように呟いた。

「十三使徒が否定しているこの世界は……そんなに駄目な世界なのか？」

答えは出ない。

でも、一つだけ言えることがある。

——壊すことより、守ること。

——否定することより、肯定すること。

それが、カイトの選ぶ道だ。

「紅蓮の霊峰……炎の災厄イグニラ」

カイトは、南の方角を見つめた。

「お前とも、話がしたい」

決意を胸に、カイトは眠りについた。

明日から、新たな旅が始まる。
