第661章：風水の共鳴

第六層への扉は、青と緑の二色で彩られていた。

青は水を、緑は風を象徴している。二つの属性が扉の表面で渦を巻き、絶えず動き続けていた。

カイトは扉の前で立ち止まり、解析を開始した。


function analyzeLayerSixBarrier() {
    const barrier = detectDualElementStructure();
    
    // 二属性の結界構造を解析
    const windFlow = analyzeWindCurrent(barrier);
    const waterFlow = analyzeWaterCurrent(barrier);
    
    // 交差ポイントを特定
    const intersections = findElementIntersections(windFlow, waterFlow);
    
    return {
        type: 'dual_element_barrier',
        elements: ['wind', 'water'],
        intersections: intersections,
        weakness: 'element_cancellation_at_intersections'
    };
}

カイトは結果を仲間に伝えた。

「この扉、風と水の二属性が複雑に絡み合っている。普通の方法じゃ開かない」

セシリアが分析を補足した。

「二つの属性が相互に強化し合っています。風が水を運び、水が風を加速させる……一種の共鳴現象です」

ヒナタが興味深そうに扉を見つめた。

「風と水の共鳴……私の風魔法と、誰かの水魔法を組み合わせれば、同じようなことができるかも」

エリナが頷いた。

「私が水、ヒナタが風。二人で扉の共鳴に合わせてみましょう」

カイトは二人に任せることにした。

「頼んだ。僕たちはサポートする」

エリナとヒナタが並んで立ち、同時に魔法を発動した。

「水流——アクア・フロー！」

「風紋——ウィンド・パターン！」

青と緑の魔力が交差し、扉の共鳴と同じ周波数を生み出した。扉が共鳴し始め、渦が徐々に収まっていく。

——カチャッ。

重い音と共に、扉が開いた。

「やった！」

ヒナタが喜んだ。エリナも満足げに微笑む。

「息、合ってたわね」

カイトたちは第六層へと足を踏み入れた。

——第六層の内部は、予想外の光景だった。

そこは、巨大な水脈と風の洞窟が融合した空間だった。天井からは滝が流れ落ち、床からは風が噴き出している。水と風が絶えず交錯し、霧のような靄を生み出していた。

「ここは……洞窟の中に、滝があるのか？」

リアムが珍しそうに見回した。

「不思議な場所だな。風と水が、こんな風に共存してるなんて」

シオンが周囲の気配を探った。

「敵の気配が……二つあります。一つは風属性、もう一つは水属性」

セシリアが警告した。

「来ます！ 上空から！」

——その瞬間、天井の滝から二人の影が飛び出した。

一人は、風を纏った女性だった。長い緑色の髪が、風に乗って揺れている。手には、風でできた弓を持っていた。

もう一人は、水を纏った男性だった。青い髪と瞳を持ち、周囲に水滴を浮かべている。手には、氷でできた槍を握っていた。

「十三使徒、第八席——テンペスト」

風の女性が名乗った。

「同じく、第十一席——ヴォイドフィンサー」

水の男性が続いた。

テンペストが弓を構えた。

「第六層は、私たち二人の領域。風と水の調和が支配する場所よ」

ヴォイドフィンサーが槍を構えた。

「お前たちの旅は、ここで終わる」

カイトは二人の使徒を見据えた。

「二人同時か……やりにくいな」

エリナが前に出た。

「私がヴォイドフィンサーをやる。水属性なら、私の得意分野よ」

ヒナタが頷いた。

「じゃあ、私はテンペストを。風対風、どっちが上か勝負しましょう」

カイトは二人の判断を尊重した。

「わかった。でも、無理はしないで。僕とシオン、リアム、セシリアは、状況に応じてサポートする」

——戦いが始まった。

テンペストが弓から風の矢を放った。

「嵐矢——ストーム・アロー！」

無数の風の矢が、ヒナタを襲う。ヒナタは風魔法で防御した。

「風盾——ウィンド・シールド！」

緑色の盾が現れ、矢を弾き飛ばした。だが、テンペストの攻撃は止まらない。

「風同士なら、相殺されるだけ。でも——」

テンペストが指を鳴らした。

——その瞬間、ヒナタの背後から水の槍が襲いかかった。

「氷槍——アイス・スピア！」

ヴォイドフィンサーの攻撃だ。ヒナタは間一髪で回避したが、腕をかすった。

「くっ——二人が連携してる！」

カイトはすぐに状況を理解した。

「みんな、注意して！ 二人は単独で戦ってるんじゃない。風と水を組み合わせた連携攻撃をしてくる！」

セシリアが分析結果を報告した。

「テンペストの風がヴォイドフィンサーの水を運び、ヴォイドフィンサーの水がテンペストの風を加速させます。相互強化の連携です」

リアムが舌打ちした。

「厄介な組み合わせだな」

エリナが水魔法で反撃した。

「水流剣——アクア・スラッシュ！」

青い水の刃がヴォイドフィンサーを襲う。だが、ヴォイドフィンサーは水を操り、エリナの攻撃を吸収した。

「水属性の攻撃は、私には無効だ。私の体は、水そのものだからな」

ヴォイドフィンサーの体が、水のように揺らめいた。物理攻撃も水属性攻撃も、すり抜けてしまう。

「どうやって攻撃すればいいのよ！」

エリナが苛立った。

一方、ヒナタも苦戦していた。

「風刃——ウィンド・カッター！」

鋭い風の刃がテンペストを襲う。だが、テンペストは風に乗って優雅に回避した。

「風属性の攻撃も、私には無意味。風の中で私は自由なのよ」

テンペストが空中に浮遊し、風を操りながら攻撃を続ける。地上にいるヒナタにとって、空を飛ぶ敵は狙いにくい。

「地上からじゃ、届かない……」

カイトは状況を分析した。

シオンが七属性魔法を構築した。

「私の魔法なら、風と水を同時に操れます。二人の連携を邪魔してみせます！」

シオンの体から、七色の光が溢れた。

「七属性魔法——元素乖離！」

シオンの魔法が、テンペストとヴォイドフィンサーの間の風と水の流れを断ち切った。

「私たちの共鳴が、遮断された！？」

ヴォイドフィンサーも動揺した。

風と雷の複合攻撃。テンペストは風で防ごうとしたが、雷は風を突き抜けて襲いかかった。

エリナも同時に攻撃を変えた。

「火、水、風、地、雷、光、闇——この七つの属性は、元々一つの源から分かれたものだという説がある。その源こそが『原初』の力」

「オメガとの戦いでは、属性魔法が無効化される可能性があります。物理攻撃と、属性を超えた概念攻撃——虚無など——が有効になるでしょう」

カイトは頷いた。

「だろうな。第七層への扉を開く前に、十分な準備が必要だ」

六人は休息を取り、第七層へ向けて準備を始めた。

オメガは、カイトたちの接近を感じていた。

オメガは静かに立ち上がった。
