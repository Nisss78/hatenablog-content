第465章：七つの道標

光の試練を終えたカイトは、ルミナスに導かれて塔の下部へと戻った。

階段を降りる足取りは、以前よりも軽い。体の中を光の力が巡っている感覚が、心地よい活力を与えてくれている。

「カイト、大丈夫？」

広間で待っていたエリナが、駆け寄ってきた。

「うん、大丈夫。試練、クリアできたよ」

カイトが微笑むと、エリナは安堵の表情を浮かべた。

「よかった……顔色もいいし、なんか雰囲気が変わった気がする」
「光の加護を得たからな。見た目も少し変わるらしい」

後ろからルミナスが追いついてきた。

「エリナの言う通りじゃ。カイトの瞳の奥に、淡い金色の光が灯っておる。それが光の力の証じゃよ」

リアムとセシリアも、カイトの周りに集まった。

「それで、これからどうするんだ？ 次はどの賢者のところに行くんだ？」

リアムが問うと、ルミナスは頷いた。

「それについてじゃが、まずは地図を見せてもらいたいのじゃ」

◆ ◆ ◆

ルミナスは、一行を別の部屋に案内した。そこは、書庫のような場所だった。壁一面に本が並び、中央には巨大な机が置かれている。

ルミナスは、机の上に広げられた地図を指差した。

「これが、この大陸の全体図じゃ」

カイトたちは、地図を覗き込んだ。アルヴァレス支部のある中央部、カイトが最初に降り立った西の森、光の塔のある東の海岸、そしてテラドンが住む南の山地。これまでの旅の軌跡が、線で結ばれていた。

「賢者たちは、それぞれ異なる領域に住んでおる。テラドンは南の山地、私は東の海岸。残る五人は、この大陸の各地に散らばっておるのじゃ」

ルミナスは、地図の上に七つの点を指し示した。

  • 南の山地：テラドン（地の賢者）
  • 東の海岸・光の塔：ルミナス（光の賢者）
  • 北の氷原：イヴェルナ（氷の賢者）
  • 西の火山地帯：イグニス（火の賢者）
  • 中央の風谷：名もなき風の賢者（故人、継承者なし）
  • 北東の雷鳴山脈：ライラ（雷の賢者）
  • 南西の湖畔：水の賢者（行方不明）

カイトは、地図を見つめながら情報を整理した。

「地と光は、もう手に入れた。残りは、氷、火、風、雷、水……五つですね」
「左様。じゃが、問題がある」

ルミナスの表情が曇った。

「風の賢者は、数十年前に亡くなっておる。継承者もおらぬため、風の力は継承されておらぬのじゃ」
「えっ……じゃあ、風の試練は受けられないんですか？」

セシリアが不安そうに聞いた。

「継承者がいない以上、通常の方法では無理じゃ。じゃが、風の力そのものが消えたわけではない。風の賢者が遺した聖域に赴き、直接力を授かる方法があるかもしれぬ」
「直接、力を……」

カイトは、難しい顔をした。通常の試練とは異なる方法。何が待っているか分からない。

「もう一つ、水の賢者についても問題がある」

ルミナスは続けた。

「水の賢者は、十年以上前に消息を絶った。最後に目撃されたのは、南西の湖畔じゃが、そこには誰もおらなんだ」
「行方不明……」

リアムがつぶやいた。

「もしかして、虚空の種に関係があるんですか？」
「可能性は高い。水の領域は、虚空の侵食の影響を受けやすい。水の賢者が何かを発見し、闇に囚われたのかもしれぬ」

ルミナスの言葉に、全員が沈黙した。虚空の種の脅威は、すでに賢者たちにも影響を及ぼしているのだ。

◆ ◆ ◆

ルミナスは、カイトに向き直った。

「カイトよ。お主に、提案があるのじゃ」
「提案、ですか？」
「七つの力を集める順番についてじゃ。現在、お主は地と光の二つを持っておる。これらは、互いに補完し合う属性じゃ」

ルミナスは、地図の上で二つの点を結んだ。

「次に目指すべきは、火か氷のどちらかじゃ。この二つは対極の属性じゃが、どちらかを先に得ることで、残りも理解しやすくなる」
「火か、氷……」

カイトは考えた。火の賢者イグニスは、西の火山地帯。以前、フェニックスとの契約で立ち寄った場所だ。一方、氷の賢者イヴェルナは、北の氷原。まだ訪れたことのない領域だ。

「ルミナスさん。火の賢者イグニスとは、以前少し交流がありました。フェニックスの契約のときに」
「ほう。それは好都合じゃ。イグニスは気難しい性格じゃが、一度認めた相手には誠実じゃ。カイトがすでに顔なじみなら、試練も受けやすいじゃろう」

カイトは頷いた。イグニスとは、短い時間だったが、熱い信念を感じた。彼なら、的確な試練を与えてくれるはずだ。

「分かりました。次は、火の賢者イグニスのところに行きます」

カイトの決断に、ルミナスは満足そうに微笑んだ。

「よろしい。では、西の火山地帯への道を教えよう。船で海岸沿いに南下し、西へ迂回するのが最短じゃ」

ルミナスは、地図の上に指を走らせた。光の塔から出発し、南西へ進む航路が描かれる。

「船旅は三日ほどかかる。その間、カイトは光の力を安定させる練習をするのじゃ。新しい力は、まだ馴染んでおらぬからな」
「分かりました。ありがとうございます、ルミナスさん」

カイトは、深く頭を下げた。ルミナスは、優しく笑った。

「礼には及ばぬ。お主が真の管理者として覚醒するその日まで、我々は全力で支えるつもりじゃからな」

◆ ◆ ◆

出発の準備を整え、カイトたちは光の塔を後にした。

港には、前回と同じヴィクトル船長の船が待っていた。

「おう、カイトくんたち！ 無事に試練を終えたか？」

ヴィクトル船長が、日焼けした顔で笑った。

「はい、おかげさまで。次は、西の火山地帯までお願いします」
「火山地帯か！ そりゃまた、暑いところだな。まかせとけ、安全に送ってやるよ」

船に乗り込み、カイトは甲板に立った。光の塔が、遠ざかっていく。

エリナが、隣に並んだ。

「火山地帯かあ。前にフェニックスに会ったときは、大変だったもんね」
「うん。でも、イグニスさんとは顔なじみだし、大丈夫だと思う」

カイトは、海風を感じながら答えた。

「それに、光の力があれば、火の属性も理解しやすいってルミナスさんが言ってた」
「光と火、似てるよね。どっちも、熱と輝きがあるし」

エリナが言うと、カイトは頷いた。

「そうなんだ。プログラミングで言うと、どちらもエネルギーの放出と変換に関わる処理に近い気がする」

カイトは、空中に指で文字を描いた。光の粒子が集まり、コードのような記号を形成する。


class FireMagic extends LightMagic {
    constructor(intensity, duration) {
        super();
        this.heat = this.convertLightToHeat(intensity);
        this.duration = duration;
    }
    
    convertLightToHeat(lightEnergy) {
        return lightEnergy * THERMAL_CONSTANT;
    }
    
    ignite(target) {
        const flame = this.generateFlame();

カイトの頭の中で、魔法が自然とコードに変換されていく。これが、自分の特別な能力。アレクシス・ヴェイルと同じ、世界を記述する力。

「カイト、すごいね。指を動かすだけで、光が動くなんて」

エリナが感心したように言った。

「僕にもよく分からないんだけど、魔法を使いたいと思うと、自然とコードが浮かぶんだ」
「それが、管理者の力なんだね」

エリナは、カイトの横顔を見つめた。真剣な表情。以前よりも、頼もしくなった気がする。

「……カイト」
「ん？」
「私、もっと強くなるね。カイトの隣で戦えるように」

エリナの言葉に、カイトは驚いた。

「エリナは、もう十分強いよ。何度も助けられたし」
「まだ足りないよ。虚空の種との戦いは、これからもっと厳しくなるんでしょ？ 私も、カイトの負担を減らせるくらいにはなりたい」

エリナの瞳は、真剣だった。カイトは、彼女の決意を受け止めた。

「……うん。一緒に強くなろう。僕も、まだまだ成長できるはずだから」

二人は、海原を見つめた。水平線の向こうに、次の目的地が待っている。

◆ ◆ ◆

船旅は、平穏に過ぎた。

カイトは、毎日ルミナスから教わった光の魔法の練習をした。エリナ、リアム、セシリアも、それぞれの魔法の訓練に励んだ。

三日目の朝、船は西の火山地帯の入り江に到着した。

「着いたよ！ ここからは、陸路だね」

ヴィクトル船長が、甲板から叫んだ。

「気をつけてな！ 火山地帯は、魔物も多いからよ」
「ありがとうございました、船長！」

カイトたちは、船を降りた。目の前には、赤茶けた大地が広がっている。遠くに、煙を上げる火山が見えた。

「あそこが、イグニスさんの住む火山か」

カイトが指差すと、リアムが頷いた。

「前に来たときは、フェニックスの聖域までだったからな。今回は、もっと奥まで行くんだな」
「イグニスさんの住処は、火山の麓にある洞窟だって聞いた。そこを目指そう」

一行は、灼熱の大地を歩き始めた。

◆ ◆ ◆

火山地帯の暑さは、半端ではなかった。

足元の地面は熱を帯び、空気は揺らめいている。汗が噴き出し、喉が渇く。

「うう、暑い……」

セシリアが、ぐったりとした声で言った。

「水魔法で、少し涼しくできる？」

カイトが提案すると、セシリアは首を振った。

「ここは熱気が強すぎて、水魔法の効果が薄れちゃうの。少しは涼しくなるけど、長くは持たないよ」

セシリアは、それでも水魔法を発動した。周囲の温度が、ほんの少し下がる。

「ありがとう、助かるよ」

カイトは、感謝を伝えた。セシリアは、弱々しく微笑んだ。

道中、数回の魔物との遭遇があった。炎を纏う狼、溶岩の体を持つゴーレム。どれも、以前より手強くなっている。

「カイト、後ろ！」

エリナの警告に、カイトは振り返った。炎の狼が、牙を剥いて飛びかかってくる。

カイトは、右手を掲げた。体の中の光の力が、指先に集まる。


function lightBarrier() {
    const energy = channelLightEssence();
    const shield = constructBarrier(energy);
    shield.setDuration(5000);
    shield.setReflectChance(0.3);
    return activate(shield);
}

金色の障壁が、カイトの前に展開された。狼の牙が、障壁に弾かれる。

「今だ！」

リアムが、剣を振るった。風を纏った一撃が、狼を切り裂く。

狼は悲鳴を上げ、光の粒子となって消滅した。

「ナイスだ、二人とも」

リアムが親指を立てた。カイトとエリナは、互いに顔を見合わせて笑った。

◆ ◆ ◆

数時間後、一行は火山の麓に到着した。

巨大な洞窟の入り口が、口を開けている。中からは、熱い風が吹き出していた。

「ここが、イグニスさんの住処か」

カイトがつぶやくと、洞窟の奥から低い声が響いた。

「よく来たな、若者よ」

カイトたちは、身構えた。洞窟の闇の中から、一人の人物が現れた。

赤い髪、燃えるような瞳、鍛え抜かれた体躯。火の賢者、イグニスだ。

「イグニスさん！ お久しぶりです」

カイトが挨拶すると、イグニスはふっと笑った。

「カイトか。フェニックスの契約以来だな。ずいぶんと、強くなったようじゃないか」

イグニスは、カイトの全身を見渡した。

「光の加護を得たな。ルミナスの試練を乗り越えたか」
「はい。次は、火の賢者の試練を受けたいんです」

カイトの言葉に、イグニスは満足そうに頷いた。

「良い覚悟だ。だが、私の試練は甘くないぞ。火は、情熱も破壊も生む。お前に、その炎を制御できるか？」
「やってみせます」

カイトは、真っ直ぐにイグニスを見つめた。迷いはない。

「よろしい。では、明日の朝、試練を始めよう。今夜は、ゆっくり休むがいい」

イグニスは、洞窟の中へと歩き出した。カイトたちは、その背中を追った。

灼熱の夜が、静かに更けていく。

◆ ◆ ◆

洞窟の中は、意外なほど快適だった。

天然の温泉があり、寝床も整えられている。イグニスは、賢者としての生活を長く続けているらしく、住環境にはこだわりがあった。

「熱い風呂だが、心地よいじゃろう？」

イグニスが、カイトに声をかけた。

「はい、とても。疲れが取れます」

カイトは、温泉に浸かりながら答えた。エリナたちは、女性用の温泉へ行っている。

「カイトよ。お前は、なぜ力を求める？」

イグニスの問いに、カイトは考えた。

「……守りたい人たちがいるからです。この世界を、大切な仲間を」
「守る、か。それは、義務か？ それとも、願いか？」

カイトは、湯気を見つめながら答えた。

「願いです。僕自身が、この世界を守りたいと望んでいるんです」

カイトの答えに、イグニスは深く頷いた。

「良い答えだ。炎は、願いがなければ燃え続けられない。お前の願いが本物なら、火の試練も乗り越えられるじゃろう」

イグニスは、立ち上がった。

「明日を楽しみにしておけ。良い試練を用意してやる」

カイトは、イグニスの背中を見送りながら、深呼吸をした。

明日は、火の試練。灼熱の炎と向き合う覚悟を、心に固めた。
