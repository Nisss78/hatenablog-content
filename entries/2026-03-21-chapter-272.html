第272章：天空城の賢者

光の階段を駆け上がったカイトたちの目の前に、圧倒的な光景が広がっていた。

雲海の上に浮かぶ白銀の城。高い尖塔は空を突き刺すようにそびえ立ち、城壁は朝露のように輝いている。空気は thin（希薄）で、遠くの山々が青い影として見えた。

「すごい……」

セシリアが息を呑む。

「本当に空の上に城があるなんて」
「伝説は真実だったのね」

エリナが静かに言う。

カイトたちは、城の正門の前に立っていた。門は何もしなくても自然に開き、中へと招き入れる。

「案内してくれるみたい」

リアムが呟く。

城内部に入ると、さらに驚くべき光景が待っていた。

床は透明なクリスタルでできていて、足元には雲が流れている。壁には空や星の壁画が描かれ、天井には本物の空が広がっているようだった。

「魔法で空を再現してる？」

ヒナタが天井を見上げる。

「いや、これは本物の空だ。城自体が、大気の薄い高層に存在している」

カイトが分析する。

プログラマーとしてのカイトの視点では、この城は高度管理されたenvironment（環境）だ。気圧、温度、重力——すべてが調整され、人間が生存できるように保たれている。


class SkyCastle {
    constructor() {
        this.altitude = 10000; // meters
        this.atmosphere = new LifeSupport();
        this.gravity = new GravityController(0.8);
    }
    
    maintainEnvironment() {
        this.atmosphere.regulateOxygen();
        this.atmosphere.regulateTemperature(20); // Celsius
        this.gravity.stabilize();
    }
}

カイトは首を振る。ついコードで考えてしまう。

広い廊下を進むと、巨大な扉の前に出た。扉には、翼を持つ女性のレリーフが刻まれている。

「空の賢者、セレスティア様の間だ」

リアムが言う。

カイトは深呼吸をして、扉を開けた。

中は、半球形の巨大なホールだった。床は白い大理石、壁は透明なガラスで、外の空が一望できる。ホールの中央に、玉座のような椅子が置かれている。

そして、その玉座に座っていたのは——

一人の女性だった。

青銀色の長い髪、透き通るような肌、そして背中には純白の翼。彼女の存在感は、圧倒的だった。ただそこにいるだけで、空間が満たされるような感覚。

「ようこそ、選ばれし者たち」

女性が立ち上がり、翼を少し広げる。その動作だけで、風が生まれた。

「私が空の賢者、セレスティア。あなたたちの到着を待っていたわ」

カイトは跪こうとするが、セレスティアが手を振って止める。

「その必要はありません。あなたたちは、私の客であり、対等な存在です」

セレスティアは優しく微笑む。

「四元素を集め、ここまで辿り着いたこと、心から称賛します」
「セレスティア様……」

カイトは言葉を探す。

「僕たちは、アークライトを止めるために来ました。最後のアーティファクト『天空の王冠』が必要です」

セレスティアの表情が曇る。

「アークライト……その名を聞くと、胸が痛みます」

彼女は玉座の近くにある窓の方へ歩き、外を眺める。

「彼は、かつて私の弟子だったのです」
「えっ？」

カイトたちが同時に声を上げる。

「弟子……？」

エリナが確認する。

「そうです。五百年前、彼は空の魔法を学ぶために、この城を訪れました」

セレスティアは遠くを見つめる。

「アークライト——本名をアレクシウスと言います——は、類稀なる才能を持っていました。空の魔法は、最も習得が難しい魔法の一つ。しかし彼は、数年で基礎をマスターし、十年で私に匹敵する力を手に入れたのです」
「それなのに、なぜ……」

ヒナタが尋ねる。

「力が、彼を狂わせたのです」

セレスティアは振り返る。

「彼は、空の魔法の極意に触れた時、あることに気づきました。空は、世界のすべてを支配できる力を秘めていると」

セレスティアは一息つく。

「天候を操り、重力を制御し、時間さえも歪める——空の魔法は、創造主の力に最も近い魔法なのです。彼は、その力で世界を再構築しようとしました」
「再構築……？」

カイトが眉をひそめる。

「彼は言っていました。『この世界は不完全だ。俺が、完璧な世界を作り直す』と」

セレスティアの目に、悲しみの色が浮かぶ。

「私は彼を止めようとしました。しかし、彼はすでに私を超えていました。私は、彼を封印することしかできなかった……五人の賢者の力を合わせて」

五人の賢者。炎、水、風、土、空——五元素の賢者たち。

「封印が解けたのは、世界の魔力のバランスが崩れたからです」

セレスティアが続ける。

「魔力の源である『世界樹』が弱り、封印を維持する力が不足した。アークライトは、その隙を突いて蘇ったのです」

カイトは考える。世界樹——古代の伝説に出てくる、世界の中心にある巨大な樹。すべての生命と魔力の源とされている。

「世界樹が弱った原因は？」

リアムが尋ねる。

「不明です。しかし、アークライトの蘇生と関係があるかもしれません」

セレスティアは再びカイトたちの方を向く。

「さて、本題に入りましょう。あなたたちは『天空の王冠』を求めていますね」
「はい」

カイトが頷く。

「天空の王冠は、この城の最も高い塔に保管されています。しかし、簡単には手に入りません」
「試練があるんですね？」

セシリアが推測する。

「その通りです。王冠は、空の賢者の後継者だけが手にできる。あなたが、その資格があるかどうか——試さなければなりません」

セレスティアは手をかざす。

すると、ホールの床から光の柱が立ち上った。

「この光の中に入りなさい。そこで、あなたの魂が試されます」

カイトは光の柱を見つめる。中は、白い霧のように霞んでいて、何が待っているか分からない。

「一人で行くべき？」

カイトが尋ねる。

「いいえ。この試練は、あなた一人のものではありません。仲間と共に乗り越えるべきものです」

セレスティアはカイトの仲間たちを見る。

「五人で、この試練に挑みなさい」

カイトは仲間たちと顔を見合わせる。

「行こう。みんなで」

エリナが言う。

「うん。ここまで一緒に来たんだもん」

セシリアが笑う。

「私はカイトを信じてる」

ヒナタが静かに言う。

「私もです。一緒に戦いましょう」

リアムが杖を握る。

カイトは頷き、光の柱へと歩を進めた。

五人が光の中に入ると、周囲が一変した。

先ほどまでのホールは消え、代わりに——

星空の中に浮かぶ、無限の空間が広がっていた。

足元には何もない。しかし、歩くことはできる。まるで、見えない床の上を歩いているようだ。

「ここは……」

カイトが周囲を見渡す。

星々が近い。手を伸ばせば届きそうな距離だ。

「精神世界？ それとも、幻覚？」

リアムが分析する。

「どちらでもない。これは、空の記憶よ」

どこからか、声が響いた。

セレスティアの声だ。しかし、姿は見えない。

「空は、この世界のすべてを見てきました。誕生も、成長も、戦争も、平和も——すべてを記憶しています」

カイトたちの前に、光の粒子が集まり始めた。

粒子は形を成し、一人の人物となって現れた。

それは——

カイト自身だった。

「僕……？」

カイトが驚く。

もう一人のカイト——光でできたカイト——は、静かに微笑んでいる。

「ようこそ、カイト。私は、あなたの可能性の具現化です」

光のカイトが言う。

「あなたの可能性……？」
「そうです。あなたが持つ、プログラミングの才能。魔法への適性。そして——世界を救う力」

光のカイトは手を広げる。

「この試練は、あなたが本当に世界を救うに値するかを問うものです」
「どうやって？」

カイトが尋ねる。

「三つの問いに答えなさい。三つとも正解すれば、あなたは天空の王冠を手にする資格があります」

光のカイトは、指を三本立てる。

「第一の問い。あなたは、なぜ世界を救いたいのですか？」

カイトは考えた。

なぜ、世界を救いたいのか。

正義のため？ 人類のため？ それとも——

「僕は……」

カイトは口を開く。

「僕は、大切な人たちを守りたいからです。エリナ、セシリア、ヒナタ、リアム——この旅で出会った仲間たち。そして、この世界に住むすべての人々」

カイトは続ける。

「僕はプログラマーです。バグを修正し、システムを改善するのが仕事です。今、この世界には最大のバグ——アークライトが存在しています。僕は、このバグを修正したい。それが、僕の理由です」

光のカイトは頷く。

「良い答えです。自分のためではなく、他者のために戦う——それが、英雄の条件です」

光のカイトは指を一本折る。

「第二の問い。あなたは、アークライトを倒すことができますか？ 彼は、かつての五賢者を凌駕する力を持っています」

カイトは迷わず答える。

「一人では無理かもしれません。でも、僕には仲間がいます。五人の力を合わせれば——勝てるはずです」

カイトは仲間たちを見る。

エリナ、セシリア、ヒナタ、リアム——四人は、力強く頷いてくれた。

「私たちは、カイトと一緒に戦う」

エリナが言う。

「アークライトなんて、絶対に倒す！」

セシリアが拳を握る。

「私の剣は、カイトのためにある」

ヒナタが静かに宣言する。

「私の魔法も、皆さんを支えるために使います」

リアムが杖を掲げる。

光のカイトは微笑む。

「信頼と絆——それもまた、重要な要素です」

光のカイトはもう一本指を折る。

「最後の問い。あなたは、アークライトを倒した後、どうしますか？」

カイトは考える。

アークライトを倒した後——

「僕は、元の世界に戻ります」

カイトは正直に答える。

「僕は、もともとこの世界の人間じゃない。異世界から召喚された存在です」

カイトの言葉に、仲間たちは驚いた表情を浮かべる。

「えっ、カイトって異世界から来たの！？」

セシリアが目を丸くする。

「言ってなかったっけ？」

カイトが苦笑する。

「言ってないよ！」

ヒナタが突っ込む。

「てっきり、遠い国の出身かと……」

リアムが呆然とする。

「異世界……それで、プログラミングという特殊な魔法を使っていたのですね」

エリナだけは、納得したように頷く。

「ずっと、気になってたのよ。カイトの魔法、他とは全く違うから」

カイトは光のカイトに向き直る。

「だから、アークライトを倒したら——僕は、家に帰ります」

カイトの胸に、チクリと痛みが走る。

仲間たちと離れるのは、辛い。でも、元の世界には、待っている人たちがいる。

「でも、帰りたくない——って思う瞬間もあるよ」

カイトは小さく笑う。

「みんないい仲間だし、一緒にもっと冒険したいって思うこともある」

カイトは仲間たちを見る。

「でも、僕には僕の世界がある。そっちでも、やるべきことがあるんだ」

光のカイトは、しばらく沈黙した。

そして——

「……正直な答えですね。自分の欲望や感情を隠さず、真実を語る」

光のカイトは、三本目の指を折る。

「三つの問い、すべて合格です」

星空の空間が、激しい光を放ち始める。

「あなたは、天空の王冠を手にする資格があります」

光のカイトが手をかざすと、カイトの頭上に、輝く冠が現れた。

白銀の冠。中央には、透明なクリスタルが埋め込まれている。

「これが、天空の王冠」

カイトが手を伸ばすと、王冠は自然とカイトの手の中に収まった。

温かい。まるで、生きているかのような温かさだ。

「王冠は、空の魔法を極めた者の証。これを被った者は、空の力を自由に操ることができます」

光のカイトが説明する。

「ただし——王冠の力は絶大です。使い方を間違えれば、アークライトと同じ過ちを犯すことになります」
「分かってる」

カイトは王冠を見つめる。

「僕は、この力を正しく使う。約束する」

光のカイトは微笑んで、光の粒子となって消えた。

星空の空間も、徐々に薄れていく。

そして——

カイトたちは、再び天空城のホールに戻っていた。

セレスティアが、満足そうに頷いている。

「おめでとうございます、カイト。あなたは、試練を乗り越えました」
「ありがとうございます、セレスティア様」

カイトは礼を言う。

「さて、これで六つのアーティファクトが揃いました。炎、水、風、土、空——そして、これらを統合する天空の王冠」

セレスティアは窓の方へ歩く。

「アークライトは、今、世界樹の下にいます。世界樹の魔力を吸い取り、完全復活をしようとしているのです」
「世界樹の下……」

カイトが呟く。

「場所は？」

エリナが尋ねる。

「中央大陸の最深部。『始原の森』と呼ばれる、人類未踏の地です」

セレスティアは振り返る。

「そこへ行けば、アークライトがいます。そして——最後の戦いが、待っています」

カイトは王冠を握りしめる。

「行こう。みんな」

カイトは仲間たちに向かって言う。

「アークライトを倒して、この世界を救うんだ」

五人の表情に、決意の色が浮かぶ。

「うん！」

セシリアが元気よく答える。

「待ってなさいよ、アークライト」

エリナが拳を握る。

「私が道を切り開く」

ヒナタが剣を構える。

「皆さんをサポートします」

リアムが杖を掲げる。

セレスティアは、優しく微笑んだ。

「行ってらっしゃい、選ばれし者たち。空が、あなたたちに味方しますように」

カイトたちは、天空城を出て、光の階段を降りた。

目的地は、始原の森。

最後の戦いが、待っている。
