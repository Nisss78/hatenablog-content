第755章：管理者の塔の入り口

カイトたちは、管理者の塔への道を歩いていた。

雲の上を歩くような感覚。地面は白く、柔らかい。

「空気が薄い……」

シオンが息を切らした。

「管理者の領域だからね。普通の人間には厳しい環境よ」

エリナが風魔法で酸素を循環させた。


// 酸素循環魔法
function oxygenCirculation(range) {
  const magic = {
    type: "wind",
    subtype: "air_purification",
    range: range,
    duration: "継続"
  };
  
  return `範囲${range}メートル内の酸素濃度を維持`;
}

// 実行
oxygenCirculation(50);

シオンの呼吸が整った。

「ありがとう、エリナ」

塔が近づいてきた。

黒い塔は、想像以上に巨大だった。見上げても頂上が見えない。

「これが、管理者の塔……」

リアムが呟いた。

「何か、冷たい気配がする」

カイトは塔を観察した。

塔の壁面には、無数の文字が刻まれている。古代の魔法言語だ。

「ルーン文字……いや、もっと古い言語だ」

カイトは解析魔法を使った。


// 古代文字解析
function analyzeAncientRunes(text) {
  const magic = {
    type: "light",
    subtype: "linguistic_analysis",
    target: text,
    era: "創世紀"
  };
  
  return `${text}の言語的起源を特定`;
}

// 実行
analyzeAncientRunes("塔の壁面文字");

カイトの脳内に、翻訳結果が浮かんだ。

「これは……『原初のコード』だ」

カイトは驚いた。

「原初のコード？　何それ？」

シオンが首を傾げた。

「この世界を構成する、基礎的な魔法式。プログラムで言えば、カーネルに近い存在だ」

カイトは説明した。

「管理者は、このコードを使って世界を操作する」

塔の入り口に到着した。

巨大な扉が二人いる。左右に並ぶ黒い扉。

扉の前には、一人の人物が立っていた。

白いローブを纏った老人。長い髭、穏やかな表情。

「待っていたよ、カイト」

老人が口を開いた。

「あなたは？」

カイトが尋ねた。

「私はアルフレッド。元管理者の一人だ」

老人は自己紹介した。

「アルフレッド！？　フローズナさんが言ってた守人と同じ名前」

シオンが驚いた。

「フローズナの氷見の村に、私と同じ名の守人がいる。彼は私の孫だよ」

アルフレッドは微笑んだ。

「元管理者ってことは、引退したんですか？」

リアムが尋ねた。

「引退というより、役割を終えたと言うべきか」

アルフレッドは塔を見上げた。

「管理者には任期がある。私の任期は、三百年前に終わった」
「三百年！」

エリナが驚いた。

「管理者って、そんなに長生きできるんですか？」
「塔の恩恵でね。普通の人間なら、百年が限度だ」

アルフレッドは説明した。

「それで、私を待っていたのは？」

カイトが本題を切り出した。

「君を案内するためだ」

アルフレッドは扉に手を置いた。

「この塔には、七つの層がある。各層には、管理者の試練が待っている」
「試練？」

カイトは眉をひそめた。

「フローズナさんの試練は終わったはず」
「フローズナの試練は、塔への資格を問うもの。塔の中の試練は、管理者としての資格を問うものだ」

アルフレッドは説明した。

「七つの層って、どういう意味？」

シオンが尋ねた。

「塔の構造だ。第一層から第七層まで、各層で異なる属性の試練がある」

アルフレッドは指を折った。

  • 第一層：地の試練
  • 第二層：水の試練
  • 第三層：風の試練
  • 第四層：火の試練
  • 第五層：雷の試練
  • 第六層：光の試練
  • 第七層：闇の試練
「七属性全部か」

カイトは覚悟を決めた。

「仲間は？」

カイトが尋ねた。

「同伴可能だ。ただし、試練を受けるのは君だけ」

アルフレッドは答えた。

「仲間はサポートに徹することになる」
「分かった」

カイトは仲間たちを見た。

「一緒に来てくれるか？」
「当然よ」

エリナが即答した。

「私たちは、カイトの仲間だもの」

リアムとシオンも頷いた。

アルフレッドは扉を開けた。

重い音とともに、黒い扉が左右に開いた。

中は、広いホールになっている。

壁には、歴代管理者の肖像画が飾られている。

「ようこそ、管理者の塔へ」

アルフレッドが招いた。

カイトたちはホールに入った。

床は黒い大理石。天井には、巨大な魔法陣が描かれている。

「この魔法陣は？」

カイトが尋ねた。

「世界の守護陣だ。封印の状態を監視している」

アルフレッドは説明した。

「今は、正常に稼働している」

カイトは魔法陣を観察した。

複雑な構造。七色の光が、幾何学的な模様を描いている。

「これを解析すれば、封印の詳細が分かる」

カイトは魔法を使った。


// 守護陣解析
function analyzeGuardianArray(array) {
  const magic = {
    type: "light",
    subtype: "pattern_recognition",
    target: array,
    depth: "完全"
  };
  
  return `${array}の構造と機能を解析`;
}

// 実行
analyzeGuardianArray("世界守護陣");

解析結果がカイトの脳内に浮かんだ。

「封印は……七つある」

カイトは呟いた。

「正確には、七層構造の封印だ」

アルフレッドが補足した。

「各層に、異なる属性の封印がある。それを守るのが、管理者の仕事だ」
「封印の中身は？」

カイトが尋ねた。

「虚無の侵食を防ぐ壁だ。壁が崩れれば、世界は虚無に飲まれる」

アルフレッドの表情が曇った。

「虚無の侵食って、具体的に何が起きるんですか？」

シオンが身震いした。

「世界の境界が曖昧になる。存在そのものが、消滅していく」

アルフレッドは静かに言った。

「かつて、一つの大陸が虚無に飲まれたことがある」
「それって……」

エリナが顔を青くした。

「リンド大陸のことだね」

リアムが呟いた。

アルフレッドは頷いた。

「正確には、リンド大陸の一部だ。境界が崩壊し、虚無に飲み込まれた」

アルフレッドは肖像画を指差した。

「その時の管理者は、命を懸けて封印を再構築した」

肖像画には、若い女性が描かれている。ミーナの母、ミズリアだ。

「ミズリアさん……」

カイトは肖像画を見つめた。

「カイト、君に期待しているのは、彼女と同じ役割だ」

アルフレッドは言った。

「封印を守り、世界を守る。それが管理者の使命だ」
「分かっています」

カイトは決意を込めて答えた。

「でも、一人で背負うつもりはありません。仲間と共に、世界を守ります」

アルフレッドは満足そうに頷いた。

「いい答えだ。管理者に必要なのは、力だけじゃない。心だ」

アルフレッドはホールの奥を指差した。

「第一層への階段は、あそこだ」

カイトたちは階段に向かった。

黒い螺旋階段。上に伸びている。

「カイト、気をつけて」

エリナが声をかけた。

「分かってる」

カイトは階段を登り始めた。

第一層に到着した。

広い空間。床には、巨大な岩が点在している。

奥に、巨大なゴーレムが立っていた。

「地の試練の守護者だ」

アルフレッドが説明した。

「このゴーレムを倒せば、第一層クリアだ」

ゴーレムが動き出した。

岩が組み合わさった巨体。目が赤く光っている。

「カイト、私がサポートする」

エリナが風魔法を準備した。

「リアム、シオン、位置について」

カイトは指示を出した。

ゴーレムが拳を振り下ろした。

地面が揺れる。岩が飛び散る。

カイトは回避した。

「硬い……」

カイトは剣で斬りつけたが、傷一つつかない。

「物理攻撃は効かないか」

カイトは魔法に切り替えた。


// 対地属性魔法
function earthPiercer(target, power) {
  const magic = {
    type: "composite",
    elements: ["風", "雷"],
    subtype: "piercing",
    target: target,
    power: power
  };
  
  return `${target}に風雷貫通魔法を放つ`;
}

// 実行
earthPiercer("地のゴーレム", 90);

風と雷の複合魔法が、ゴーレムに命中した。

岩が砕け、ひびが入る。

「効いてる！」

シオンが歓声を上げた。

ゴーレムが反撃した。

腕を振り回し、岩石を飛ばしてくる。

リアムが水魔法で防御した。


// 水盾展開
function waterShield(target, durability) {
  const magic = {
    type: "water",
    subtype: "barrier",
    target: target,
    durability: durability
  };
  
  return `${target}の前に耐久度${durability}の水盾を展開`;
}

// 実行
waterShield("カイト", 80);

水の盾が岩石を受け止めた。

カイトは距離を詰めた。

ゴーレムの懐に入る。

「これで終わりだ！」

カイトは全力の魔法を放った。


// 七属性統合攻撃
function elementalNova(target) {
  const magic = {
    type: "composite",
    elements: ["火", "水", "風", "土", "雷", "光", "闇"],
    subtype: "nova",
    target: target
  };
  
  return `${target}に七属性ノヴァを放つ`;
}

// 実行
elementalNova("地のゴーレム");

七色の光がゴーレムを包んだ。

ゴーレムは崩れ落ちた。

岩が分解し、元の地面に戻っていく。

「第一層、クリアだ」

アルフレッドが宣言した。

カイトは息を整えた。

「まだ六層残ってる……」

カイトは上を見上げた。

塔の高さは、果てしなく続いている。

「一歩ずつ、進もう」

エリナが励ました。

「私たちは、カイトと一緒だから」

カイトたちは、第二層への階段を登り始めた。

管理者の塔の試練が、始まったばかり。

第755章、了。
