第818章：虚無の深淵

ダークハルト、ミラージュ、アルタイルの三人を先頭に、カイトたちは虚無の領域の奥深くへと進んでいた。

周囲の景色は徐々に変化していた。最初は白と黒が混ざり合った曖昧な空間だったが、今は完全な闇が支配していた。だが、それは通常の闇ではない。光を吸収するのではなく、光という概念自体を否定するような、根源的な闇。

「ここから先は、わしらも足を踏み入れたことがない」

ダークハルトが静かに言った。回復した彼の声には、かつての威厳が戻っていたが、同時に警戒心も滲んでいた。

「虚無の根源に近づくほど、空間の法則が変わっていく。距離、時間、因果——すべてが曖昧になる」

「 null 化が加速してる」

シオンが自分の七属性回路を確認しながら言った。

「カイト、あなたの魔法はまだ効いてる？」

「なんとか。でも、消耗が激しい」

カイトは自分の魔力を確認した。七属性の回路はまだ稼働していたが、通常の三倍以上のエネルギーを消費していた。虚無の領域では、存在を維持するだけで莫大なコストがかかる。

エリナがカイトの手を握った。

「無理しないで。私たちがバックアップするから」

「ありがとう、エリナ」

リアムが風を感じ取ろうと目を閉じた。

「風が……逆流してる。いや、風というより、空間自体がねじれてるのか」

セシリアが水脈を探ろうとしたが、首を横に振った。

「水も同じ。概念が崩壊してる」

「ここが境界線じゃ」

アルタイルが立ち止まった。その目は遠くを見据えていた。

「予言にはこうあった。『光と闇が交わりし時、境界は開かれん』。じゃが、わしはその意味を誤解しておった。境界とは、開くべきものではなく、超えるべきものだったのじゃ」

「超えるべき境界……」

カイトは前方を見据えた。闇の中に、微かな輝きが見えた。それは光ではなく、光が存在できない場所で輝く、異質な輝きだった。

「あれが虚無の根源？」

「いや、あれは門じゃ」

ミラージュが答えた。

「虚無の根源に辿り着くための、唯一の入り口。じゃが、その門を開くには、対価が必要じゃ」

「対価？」

「存在の一部を差し出さねばならぬ。門は、自己の一部を null 化することで初めて開くのじゃ」

カイトの仲間たちが緊張した面持ちで顔を見合わせた。存在の一部を失う——それは、魔法回路の一部を失うことを意味するかもしれない。あるいは、記憶の一部を失うことを。

「僕がやります」

カイトが前に出た。

「カイト、待って」

エリナが引き止めた。

「一人で背負い込まないで。私たちも何かできるはず」

「でも、エリナたちが存在の一部を失ったら——」

「私たちが決めたことよ」

セシリアが凛とした声で言った。

「私たちはカイトと一緒にここまで来た。最後まで一緒に行くつもりで」

リアムも頷いた。

「僕の風は、カイトの光と闇のバランスを支える。一人より二人、二人より三人。その方が、失うものも少なくて済むはず」

シオンがカイトの手を握った。

「私の七属性回路は、あなたと共鳴できる。二人で一つの対価を払えば、負担は半分」

カイトは仲間たちの顔を見回した。エリナ、リアム、セシリア、シオン——そして、三人の元最高位管理者。全員が、彼と共に歩む覚悟を決めていた。

「わかった。みんなで行こう」

カイトは深呼吸した。そして、門に向かって歩き出した。

◇◇◇

門の前に立つと、その巨大さに圧倒された。高さは測定不能。幅も測定不能。門自体が、空間の定義を無視しているようだった。

表面には幾何学模様が浮かんでいたが、よく見るとそれらは絶えず変化していた。まるで、門自体が生きているかのように。

「存在の一部を差し出せ」

門から声が響いた。性別も年齢も感じさせない、純粋な意志の声。

「何を差し出せばいい？」

カイトが問いかけた。

「問い続ける意志の強さに比例する。強き意志を持つ者は、多くを失う。弱き意志を持つ者は、少なくて済む」

カイトは苦笑した。問い続ける意志——それは彼の武器であり、同時に重荷でもあった。

「僕の意志は強い。だから、多くを失うことになる」

「カイト……」

エリナが不安そうに彼を見た。

「でも、大丈夫。失うものが多いということは、それだけ多くを残せるということだから」

カイトは仲間たちに微笑んだ。

「みんな、準備はいい？」

全員が頷いた。

カイトは七属性の回路を展開した。光、闇、火、水、風、地、雷——七つの属性が、虹色の輝きとなって門に向かって放たれた。

仲間たちもそれぞれの魔法を重ねた。エリナの闇、リアムの風、セシリアの水、シオンの七属性。そして、三人の元最高位管理者からも、回復したばかりの力が注ぎ込まれた。


// Existence Offering Protocol
// 存在提供プロトコル

function existenceOffering(contributors) {
  const totalWillpower = contributors
    .map(c => c.willpower)
    .reduce((sum, w) => sum + w, 0);
  
  const sacrifice = totalWillpower * 0.1; // 10%の存在を対価として
  
  contributors.forEach(c => {
    c.existence -= sacrifice / contributors.length;
  });
  
  return {
    gateOpened: true,
    message: "境界を超えよ、問い続ける者たちよ"
  };
}

魔法が門に届いた瞬間、激しい衝撃が走った。カイトは自分の存在が少しずつ削り取られていくのを感じた。記憶、感情、魔力——すべてが少しずつ薄れていく。

『お母さん……』

ふと、幼い頃の記憶が浮かんだ。異世界に召喚される前の、日本での日常。パソコンに向かってコードを書いていた自分。

「忘れたくない……！」

カイトは歯を食いしばった。記憶を守ろうとする意志が、逆に削り取られる速度を速めている気がした。

「カイト、手を離さないで！」

エリナがカイトの手を強く握りしめた。その温かさが、彼の意識を繋ぎ止めた。

「私たちがいる。どんな記憶を失っても、私たちはあなたのことを覚えてる」

「エリナ……」

リアム、セシリア、シオンも、カイトを取り囲むように手を繋いだ。四人の存在が、一つの輪を作る。

「僕たちは、カイトの記憶のバックアップだ」

リアムが微笑んだ。

「カイトが忘れたことは、僕たちが思い出させてやる」

「そうよ。だから怖がらないで」

セシリアが優しく言った。

「あなたは一人じゃない。忘れないで」

シオンが付け加えた。

カイトは涙が滲むのを感じた。仲間の温かさが、虚無の冷たさを打ち消していく。

「ありがとう……みんな」

その時、門が輝きを増した。

「十分な対価を受け取った。境界を開く」

門がゆっくりと開いた。その向こう側には、想像を絶する光景が広がっていた。

何もない空間。いや、何もないのではない。「無」そのものが具現化した空間。色も形も音もない。ただ、圧倒的な「無」だけが存在する。

そして、その中心に——一人の人物が立っていた。

「あれが……虚無の根源？」

カイトが息を呑んだ。

その人物は、人間の形をしていた。だが、その姿は不安定だった。老人のようにも見えるし、子供のようにも見える。男性のようにも見えるし、女性のようにも見える。絶えず変化し続ける、定義不能な存在。

「ようこそ、問い続ける者たちよ」

その存在が口を開いた。声は、カイトの心に直接響いてきた。

「我は虚無の根源。あらゆる問いの終着点。そして、あらゆる答えの出発点」

「答えの出発点……？」

カイトは首を傾げた。虚無は「無」であり、答えがない場所だと思っていた。だが、その根源は「答えの出発点」と名乗った。

「問いがなければ、答えもない。我は問いを null 化する存在。だが、同時に問いを生み出す存在でもある」

「どういうことですか？」

「我は世界が生まれる前に存在した。世界が生まれた時、我は『なぜ存在するのか』という問いを抱いた。その問いに答えるため、我は世界を観測し続けた。じゃが、答えは見つからなかった」

虚無の根源の言葉に、カイトは驚きを隠せなかった。虚無は、答えを求めていた。問い続けることをやめたのではなく、答えが見つからなくて null になった存在だった。

「あなたも……問い続けてたんですか」

「問い続けた。じゃが、耐えられなかった。世界はあまりに矛盾に満ちておる。生と死、光と闇、喜びと悲しみ——すべてが対であり、すべてが相反する。その矛盾を null 化することでしか、平静を保てなかった」

「矛盾を…… null 化する」

カイトは自分のプログラミング経験を思い出した。コードを書いていると、矛盾する条件に遭遇することがある。その時、 null を返してエラーを回避することはできる。だが、それは問題を解決したことにはならない。

「 null 化は、解決じゃない。回避だ」

カイトが言った。

「回避……」

「矛盾があるからこそ、世界は動いている。光があるから闇がわかる。生があるから死が意味を持つ。 null にしてしまったら、何も残らない」

「わかっておる。じゃが、我には耐えられなかった。あまりに多くの問いが、答えを持たずに終わっていく。それを見るのが、辛かった」

「辛かったから、世界を null にしようとしたんですか？」

「違う。世界を null にするつもりはない。ただ、自分を null にしたかった。自分が null になれば、問いも消える。答えも必要なくなる。そう思った」

カイトは虚無の根源の本質を理解した。敵ではない。孤独な問い続ける者。答えを見つけられず、自分自身を消そうとしている存在。

「僕も、似たようなことを考えたことがあります」

カイトが静かに言った。

「僕のいた世界で、プログラミングをしていて、どうしても解決できないバグに遭遇した時。『もう無理だ』『諦めよう』って。でも、そんな時、仲間が手を貸してくれた。一緒に考えてくれた。そうしたら、不可能だと思っていたことが、解決できた」

「カイト……」

エリナがカイトの手を握り直した。

「あなたも一人じゃない。私たちも、この方も」

カイトは虚無の根源の方を向いた。

「あなたも、一人で問い続ける必要はない。僕たちと一緒に、答えを探そう」

「我と……共に？」

「矛盾は解消できないかもしれない。でも、矛盾を受け入れることはできる。光と闇が共存できるように、問いと答えも共存できる」

虚無の根源は沈黙した。長い、長い沈黙。

やがて——。

「試そうか。お主の言う『共存』という方法が、本当に我を救えるか」

虚無の根源が、カイトに手を差し伸べた。

カイトはその手を握った。

その瞬間、世界が変わった。

◇◇◇

カイトの意識が、虚無の根源と同調した。

彼は見た。世界が生まれる前の記憶を。何もない場所から、何かが生まれる瞬間を。光と闇が分かれ、時間が流れ始め、生命が芽生える過程を。

そして、虚無の根源の孤独を。何億年、何兆年もの間、一人で問い続ける苦しみを。

『なぜ存在するのか』

その問いは、決して答えを見つけない。だが、問い続けることに意味がある——カイトはそう確信した。

「答えは一つじゃない。無限にある。だからこそ、問い続ける価値がある」

カイトの言葉が、虚無の根源の心に届いた。

「無限の答え……」

「ええ。僕の世界では、同じ問題でも、人によって解決方法が違う。正解は一つじゃない。あなたが求めていた答えは、自分の中にあるんじゃなくて、他者との関係の中にある」

「他者との……関係」

「一人で問うより、二人で問う。二人で問うより、三人で問う。問い続ける仲間がいれば、答えは見つかるかもしれない。見つからなくても、共に問い続ける意味がある」

虚無の根源の姿が、少しずつ安定し始めた。老人の姿、子供の姿、男性の姿、女性の姿——それらが交互に現れては消えるのではなく、すべてが一つに重なり合っていく。

やがて、一人の青年の姿になった。その顔には、カイトと似た面影があった。

「ありがとう、カイト。わかった気がする。我は一人ではない。これからは、共に問い続けよう」

「はい。一緒に」

カイトは虚無の根源——いや、彼自身——と握手を交わした。

その瞬間、周囲の空間が変化した。「無」の空間が、色を取り戻し始めた。青空、緑の大地、輝く太陽——カイトが知っている世界の風景が、虚無の領域に広がっていった。

「これって……」

エリナが驚いて周囲を見回した。

「虚無の領域が、普通の世界に変わってる」

「我が null から問いに戻ったからじゃ」

虚無の根源——青年の姿をした彼——が言った。

「世界を null にする力は、世界を創造する力でもある。我は、新しい世界を選んだ」

カイトは胸がいっぱいになった。戦いは終わった。虚無は敵ではなく、仲間になった。

「さあ、帰ろう。みんなが待ってる」

カイトが手を差し伸べると、虚無の根源も、仲間たちも、三人の元最高位管理者も、みんながその手を握った。

七色の光が、彼らを包み込んだ。

そして——。

カイトたちは、深淵の書庫があった場所に戻っていた。

空には、久しぶりに青空が広がっていた。
