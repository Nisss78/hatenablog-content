第818章：虚無の深淵

虚無の領域の中心からさらに奥へと続く道は、物理的な道ではなかった。足元の鏡のような地面が次第に透明になり、その下に無限に続く深淵が見えるようになったからだ。

「ここから先は、概念の階層になる」

ダークハルトが先頭を歩きながら説明した。彼の声は、以前のような重厚さを取り戻していたが、同時にどこか慎重さも含んでいた。

「我々が融合していたのは、虚無の第一階層。概念がまだ形を保てる領域じゃ。じゃが、これから先は違う。存在そのものが問い直される場所」

「存在が……問い直される？」

シオンが不安げに問いかけた。彼女の銀髪が、周囲の奇妙な光の影響で淡い紫色に見えていた。

「お主が『自分は誰か』という問いに、即座に答えられなくなるということじゃ」

ミラージュが補足した。

「鏡像を操るわしですら、深層では自分の姿を見失ったことがある。自分が何者か、なぜ存在するのか、その答えが null になる感覚」

カイトはプログラミングの知識と照らし合わせた。null——変数が何も参照していない状態。概念が存在しない状態。虚無の領域では、それが現実になる。

「つまり、自己認識が null になる……」

「正確には、自己認識への参照が断たれる」

カイトは仲間たちの方を向いた。

「アンブロスさんが言ってた『存在の錨』がここで本番になる。僕たちがお互いを認識し続けることで、自己への参照を維持する」

エリナがカイトの手を握り直した。

「わかってる。あなたが誰か忘れそうになったら、私が思い出させる。逆も同じ」

リアムとセシリア、シオンも互いに視線を交わし、頷き合った。

「行こう。null にならないように、お互いをモニタリングしながら」

◇◇◇

深淵への降下は、階段を下りるようなものではなかった。意識を沈めるような、眠りに落ちる感覚に近い。

カイトが目を開けると、周囲の景色が一変していた。鏡のような地面は消え、代わりに無数の光の粒子が漂う空間が広がっていた。粒子の一つ一つが、誰かの記憶の断片のように見えた。

「これ……何？」

セシリアが粒子に手を伸ばした。指先が触れた瞬間、粒子が輝き、幻影が現れた。

それは、見知らぬ人物の姿だった。老いた男性で、穏やかな表情をしている。だが、その姿は半透明で、足元から徐々に粒子となって消え去っていた。

「誰か……私を思い出してくれぬか。名前が……思い出せぬ。役割も……目的も……」

「虚無に飲まれた人たちの残響ね」

エリナが悲痛な表情で呟いた。

「彼らは存在を null にされた。誰にも思い出されなくなった瞬間に、完全に消滅した」

カイトは胸が痛んだ。彼らは世界を救おうとしたのかもしれない。あるいは、ただ普通に生きていただけかもしれない。いずれにせよ、彼らは虚無の犠牲になった。

「名前、思い出します」

カイトが幻影に向かって言った。

「あなたが誰だったか、わからないけど、あなたがここにいたことは覚えます。僕が、証人になります」

幻影の目に、微かな光が宿った。

「思い出してくれるのか……？」

「ええ。僕の中に、あなたの痕跡を保存します」

カイトは目を閉じ、意識の中に仮想的な空間を構築した。プログラミングで言うなら、データベースの作成に近い。


// Memory Archive - Void Layer
// 虚無層の記憶アーカイブ

class VoidMemory {
  constructor() {
    this.archive = [];
  }
  
  preserve(fragment) {
    const entry = {
      id: this.generateId(),
      existence: fragment.existence || "unknown",
      timestamp: Date.now(),
      witness: "Kaito"
    };
    this.archive.push(entry);
    return entry;
  }
  
  generateId() {
    return 'void_' + Math.random().toString(36).substr(2, 9);
  }
}

カイトが意識を開くと、幻影の姿が少しだけ鮮明になった。

「ありがとう……若いの。名前を……思い出した。ワシは、エドガー。エドガー・ヴェイル。境界の……研究者だった」

「エドガー・ヴェイル……！」

アルタイルが驚きの声を上げた。

「その名は知っておる。最初の管理者たちと同時代の人物。境界の研究で名を馳せたが、ある日突然姿を消したと記録にあった」

エドガーの幻影は、微かに微笑んだ。

「そうじゃ……ワシは虚無の研究をしていて、飲まれたのじゃ。じゃが、後悔はない。ワシの研究は、誰かの役に立つかもしれんからの」

幻影は、粒子となってカイトの中に吸収された。

「カイト、今の……」

シオンが目を見開いていた。

「魔法じゃない。もっと根源的な何か。あなた、彼を『保存』したでしょ？」

「多分。僕の中に、彼の痕跡を残したんです。完全に思い出せるわけじゃないけど、彼が存在した事実は null にならない」

「凄いわ。それこそ、虚無に対抗する力かもしれない」

ダークハルトが感心したように頷いた。

「『記憶の保存』か。お主の発想は、確かにこの世界の魔法とは異質じゃ。それがゆえに、虚無に対抗できるのかもしれぬ」

◇◇◇

さらに深淵を降りていくと、粒子の数が増え、幻影たちの声が重なり合って聞こえるようになった。

「誰か……私を……」
「名前が……思い出せない……」
「ここで何をしていたのか……」

無数の残響たちが、存在の維持を求めていた。彼らは完全に消滅したわけではない。誰かの記憶の中に、微かな痕跡を残している限り、この境界状態に留まっていた。

「全員を保存するのは無理よ」

セシリアが冷静に指摘した。

「カイトの意識容量には限界がある。全員を保存しようとしたら、あなた自身が null になるかもしれない」

「わかってる。でも、見捨てられない」

カイトは葛藤した。プログラミングで言えば、メモリ不足の状態。すべてのデータを保存することは不可能だ。だが、諦めることもできなかった。

「カイト、アプローチを変えましょう」

リアムが提案した。

「個別の保存じゃなくて、集団の記録。彼ら全員が『虚無に飲まれた人々』という一つのカテゴリーに含まれるなら、そのカテゴリー自体を保存すればいい」

「カテゴリー……」

カイトは思考を切り替えた。個別のデータではなく、集合的な概念。すべての残響を一つのアーカイブとして保存する。


// Collective Memory Archive
// 集合的記憶アーカイブ

class CollectiveArchive {
  constructor() {
    this.collectiveMemory = {
      name: "虚無に飲まれし者たち",
      count: 0,
      essence: []
    };
  }
  
  absorbAll(fragments) {
    fragments.forEach(f => {
      this.collectiveMemory.essence.push(f.existence);
      this.collectiveMemory.count++;
    });
    return this.collectiveMemory;
  }
}

カイトは両手を広げ、周囲の粒子すべてに意識を向けた。

「あなたたち全員を、一つの記憶として保存します。個々の名前は失われるかもしれない。でも、あなたたちが存在した事実、虚無に対抗しようとした意志——それは残ります」

粒子たちが共鳴し始めた。個々の声が、一つのハーモニーになっていく。

「感謝を……若き管理者よ……」

膨大な情報が、カイトの中に流れ込んできた。無数の人生、無数の後悔、無数の希望。それは圧倒的な量だったが、集合体として処理することで、カイトの意識は耐え抜いた。

「カイト！」

エリナがカイトの肩を揺さぶった。

「大丈夫？ 顔色が悪いわ」

「……大丈夫。ただ、少し重かっただけ」

カイトは深呼吸した。胸の中に、数えきれないほどの存在の痕跡が保存されていた。それは重荷でもあり、力でもあった。

「お主の覚悟、本物じゃな」

ミラージュが静かに言った。

「かつてわしらは、彼らの声を無視した。救えないと決めつけて、先へ進んだ。じゃが、お主は違う。救えないまでも、彼らを記憶に留めることを選んだ」

「彼らの犠牲を無駄にしたくなかったから。それだけです」

カイトは立ち上がり、先を促した。

「行きましょう。虚無の根源へ」

◇◇◇

深淵の最下層に近づくにつれて、空間の質が変わった。粒子は消え、代わりに純粋な闇が広がっていた。光も闇も、あらゆる概念を飲み込む究極の無。

「ここが……虚無の深淵」

アルタイルが静かに言った。

「予言には、ここまでの詳細がなかった。誰もここから戻った者がおらぬからな」

カイトは前方を見据えた。闇の奥に、何かが見えた。巨大な、概念的な存在。それは形を持たなかったが、確実にそこにあった。

「あれが……虚無の根源？」

「いや、違う」

ダークハルトが首を振った。

「あれは……門じゃ。虚無の根源への入り口」

門——カイトには、巨大なコードの塊のように見えた。無数の null 参照が絡み合い、自己を否定し続ける無限ループを形成している。

「これ、解析できるかな」

カイトは目を細めて門を観察した。プログラマーとしての直感が警鐘を鳴らしていた。この門は、踏み入った者を即座に null にする。

「普通のアプローチじゃ通れない。強制突破も自殺行為」

「じゃあ、どうするの？」

シオンが問いかける。

「デバッグする。この門が何をチェックしているのか、どうやって null にしているのか、そのロジックを理解する」

カイトは門に近づき、触れようとした——その時だった。

「来訪者を検知。存在確認—— null 。アクセス拒否」

門から声が響いた。いや、声ではない。純粋な情報の波。概念としての拒絶。

カイトは手を引いた。

「わかった。門は存在をチェックしてる。存在が null なら通れる。存在があるなら拒否する」

「それじゃ、通れないじゃない」

リアムが落胆したように言った。

「いや、逆に考えよう。存在があるから拒否されるなら、存在を一時的に『退避』すればいい」

カイトは仲間たちの方を向いた。

「僕の存在の錨——みんなの記憶の中にある僕。それを一時的にミラージュさんに預けられないかな」

ミラージュが目を細めた。

「鏡像への転送か。可能じゃが、危険じゃぞ。鏡の中の存在は、オリジナルと完全に同期するわけではない。万が一、鏡が割れれば——」

「僕の存在も消える。わかってる」

カイトは覚悟を決めていた。

「でも、やるしかない。虚無の根源に行かなきゃ、世界は救えない」

エリナがカイトの手を強く握った。

「絶対に帰ってきて。約束して」

「約束する。必ず戻る」

カイトはミラージュの前に立った。

「行くぞ。一時的な存在転送——鏡像へのバックアップを開始する」

ミラージュが手をかざすと、カイトの体が半透明になった。彼の存在情報が、鏡像の領域へと転送されていく。

「うぐ……」

意識が薄れる感覚。自己認識が null に近づいていく。だが、同時に、エリナたちの温かい手の感触も感じていた。

存在の錨——それは機能していた。

「転送完了。カイトの存在は、今や鏡像の中にある。本体は——一時的に null と同等」

カイトは門に向かって歩き出した。

「来訪者を検知。存在確認—— null 。アクセス——許可」

門が開いた。その向こうには、純粋な虚無の世界が広がっていた。

カイトは一瞬だけ振り返り、仲間たちに微笑んだ。

「行ってきます」

そして、門の向こうへと姿を消した。
