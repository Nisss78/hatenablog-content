第111章：光の神殿

神殿の入り口をくぐった瞬間、カイトは目を細めた。外部の吹雪とは対照的に、内部は静謐な空気に包まれていた。

壁には淡い光を放つ水晶が埋め込まれ、通路を照らしている。床は滑らかな大理石で、天井には幾何学模様が浮かび上がっていた。

「これが……光の神殿」

エリナが、息を呑んで見回した。

「四百七十二年前に建てられたとは思えないほど、保存状態がいいわ」
「古代の魔術師たちの技術だろうな。自己修復機能でもあるんじゃないか」

カイトが、壁の水晶に触れた。

// Structure Analysis
// Material: Self-repairing Crystalline Matrix
// Age: 472 years (approximate)
// Energy Source: Ambient Light Magic
// Integrity: 94.7%

ミラベルの影が、静かに先導した。

「こちらです。遺物は、最奥の祭壇に安置されています」

通路を進むと、広いホールに出た。中央には巨大な石柱が立ち並び、その奥に階段が続いている。

だが、階段の前には何かが立っていた。

それは、半透明の騎士だった。全身が光でできているかのように輝き、手には巨大な剣を握っている。

「光の騎士……試練の番人ですわ」

セシリアが、緊張した声で言った。

「遺物に近づく者に課される、三つの試練の最初」
「三つの試練？」

リアムが、剣に手をかけながら聞いた。

「第一は『力の試練』——騎士を倒せば先へ進めます。でも、ただ倒すだけじゃだめ」

セシリアは、騎士を見つめた。

「光の騎士は、攻撃者の力をそのまま返してきます。暴力で押し切ろうとすれば、自分の力で破滅する」
「じゃあ、どうすればいいんだ？」

カイトが、眉をひそめた。

「倒すんじゃなくて——理解するんです」

ミラベルの影が、答えた。

「この騎士は、かつての私の仲間の一人。彼の名は、シリウス。四百七十二年前、この神殿を守るために自らの魂を捧げました」

光の騎士——シリウスが、ゆっくりと剣を構えた。

カイトは、静かに前に出た。

「カイト、危ない！」

エリナが、叫んだ。だが、カイトは剣を抜かなかった。代わりに、両手を広げた。

「シリウス、だな？ 話を聞かせてくれ」

騎士は、動きを止めた。光の瞳が、カイトをじっと見つめる。

「……継承者か」

声が、直接頭の中に響いた。深く、澄んだ声だ。

「ミラベルが予言した、世界を救う者」
「俺が何者かは、あとで説明する。今は——この神殿を守り続けてきた君の話を聞かせてくれ」

シリウスは、少しの間沈黙した。そして、剣を下ろした。

「……四百七十二年だ。この神殿で、孤独に待ち続けた」

光の体が、わずかに揺らいだ。

「最初の百年は、使命感があった。次の百年は、仲間たちの記憶が支えになった。でも、三百年を超えたあたりから——何を守っているのかさえ、わからなくなった」
「シリウス……」
「私は、もう疲れた。でも、使命は果たさなければならない。お前が、真の継承者かどうか——見極めさせてもらう」

シリウスが、再び剣を構えた。

「ただし、暴力ではなく——心で戦え」

カイトは、目を閉じた。

心で戦う——それは、コードではなかった。論理でもなかった。ただ、純粋な想いだけが、この試練を乗り越える鍵だ。

「俺は——この世界を救いたい」

カイトが、静かに言った。

「最初は、ただ家に帰りたかっただけだった。でも、今は違う。エリナ、リアム、セシリア——この世界で出会った仲間たちを、守りたい」

彼は、目を開けた。

「シリウス、君も——同じだったんじゃないか？ 仲間を守るために、この場所を選んだ」

シリウスの光が、揺らいだ。

「……そうだ。私は、ミラベルたちを守るために、ここに残った。彼女が遺物を隠し、世界を救う道を残してくれると信じて」
「ミラベルは、君の期待を裏切らなかった。遺物はここにある。そして今、俺がそれを受け取ろうとしている」

カイトは、一歩近づいた。

「君の四百七十二年は、無駄じゃなかった。俺が——証明する」

シリウスは、長い沈黙の後——剣を消した。

「……通れ。君は、試練を乗り越えた」

光の騎士が、道を開けた。

「ありがとう、継承者よ。私の長い役目も——ようやく終わる」

階段を登ると、円形の部屋に出た。天井には、巨大な水晶が吊り下がっている。

部屋の中央に、祭壇があった。その上に——淡い光を放つ球体が浮かんでいた。

「これが……最初の光の遺物」

カイトが、息を呑んで近づいた。

// Artifact Detected
// ID: Light Orb Alpha
// Type: Ancient Relic
// Power Level: EXTREME
// Function: Unknown (requires activation)
// Warning: Handle with extreme care

ミラベルの影が、祭壇の前に立った。

「この遺物は、『光の核』と呼ばれています。世界に散らばる七つの遺物の中で、最も純粋なエネルギーを持っています」
「どうやって使うんだ？」
「あなたが、コードで書くのです」

彼女は、カイトの目を見た。

「この遺物は、古代の言葉でプログラムされています。現代の魔術師には起動できません。でも——異世界からの継承者であるあなたなら」

カイトは、球体に手を伸ばした。指先が触れた瞬間——

視界が、白く染まった。

カイトは、見知らぬ場所に立っていた。

そこは、巨大な図書館だった。天井が見えないほど高い本棚が、無数に並んでいる。

「ここは……？」
「遺物の中にある、記憶の空間です」

声が、後ろから聞こえた。振り返ると——本物のミラベルが立っていた。

彼女は、 hologram や影ではない。実体として存在している。

「ミラベル！ 本物なのか？」
「ええ、本体です。ただし——これは記憶の中の姿。肉体は、まだ深層で眠っています」

ミラベルは、優しく微笑んだ。

「カイト、あなたは——私が四百七十二年前に書いたコードを、起動させることができる唯一の存在です」
「コード？」
「この世界の根本ルールを書き換える、プログラムです。私は、世界崩壊を止めるために、七つの遺物にコードを分散させました」

彼女は、一冊の本を取り出した。

「これが、最初のコード。受け取ってください」

カイトが本を開くと——見慣れた文字が並んでいた。

// World Restoration Protocol - Fragment 1
// Author: Mirabel
// Date: 1554.03.17
// Purpose: Stabilize dimensional barriers

function stabilize_dimension(target_world) {
    const light_core = get_artifact("alpha");
    const energy = light_core.extract_purity();
    
    target_world.barrier.reinforce(energy);
    
    return {
        status: "success",
        remaining_fragments: 6
    };
}

// Note for the Inheritor:
// This is only the first step.
// To fully restore the world, you must
// collect all seven fragments and run
// the complete restoration sequence.
// 
// I believe in you.
// - Mirabel

カイトは、息を呑んだ。

「これが……」
「はい。私が残した、世界を救うための鍵です」

ミラベルの姿が、少しずつ薄れていった。

「時間です。現実世界に戻りなさい」
「待って！ ヴォルターやエーテリアのことは——」
「……エーテリアは、私の旧友です。彼女を信じてください。ヴォルターについては——」

ミラベルは、悲しげな表情を見せた。

「彼もまた、かつては私の仲間でした。でも、彼が選んだ道は——私とは違う」

姿が、さらに薄くなる。

「カイト——世界を。お願いします」

そして、ミラベルの姿は消えた。

カイトが目を開けると、祭壇の前に立っていた。手の中には——光の球体が収まっている。

「カイト！ 大丈夫！？」

エリナが、駆け寄ってきた。

「ああ……大丈夫だ。遺物を——受け取った」

カイトは、球体を慎重に懐に収めた。

「残り六つ。全部集めないと、世界は救えない」

リアムが、腕を組んだ。

「六つもか。場所はわかってるのか？」
「ミラベルの記憶から——ある程度は」

カイトは、出口に向かった。

「次は、『炎の遺跡』だ。南の砂漠にある」

神殿を出ると、吹雪が収まっていた。

ミラベルの影が、入り口で待っていた。

「私の役目は、ここで終わりです。影は、遺物を守るために作られた仮初めの存在——本体の目的が果たされれば、消える運命です」
「そんな……！」

エリナが、悲しげな声を上げた。

「悲しまないでください。私は、四百七十二年間、この瞬間を待っていたのですから」

影は、静かに微笑んだ。

「カイト——ミラベルの願いを、叶えてください。あなたなら、できるはずです」

そして、影は光の粒子となって消え去った。

一行は、山を下り始めた。

カイトは、懐の遺物を意識した。温かい光が、胸の奥から伝わってくる。

「カイト」

エリナが、隣に並んだ。

「ミラベルさんは、どんな人だった？」
「……強い人だった。四百七十二年前に、一人で世界を救おうとして——そして、僕たちに道を残した」

カイトは、空を見上げた。

「俺は、その道を歩く。最後まで」

エリナは、彼の手を握った。

「一人じゃないわ。私たちも、一緒に歩く」

リアムとセシリアも、うなずいた。

「当然だ。ここまで来て、途中で投げ出すなんてできんからな」
「私も、最後までお供します」

カイトは、仲間たちを見回した。

四百七十二年前の願いを背負い、彼らは南へと向かう——次の遺物が眠る、炎の遺跡へ。
