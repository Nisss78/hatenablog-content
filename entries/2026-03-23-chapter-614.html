第614章：原初の領域

境界線を越えた瞬間、カイトは強烈な感覚に襲われた。まるで全身の細胞一つ一つが歓喜の声を上げているかのような、圧倒的な「充足感」。それは、この世界のマナが彼の魔力回路に直接共鳴している証拠だった。

「カイト、大丈夫ですか?」

エリナが心配そうに駆け寄る。カイトは深呼吸をして、感覚を整理した。

「ああ、問題ない。ただ……ここのマナが濃すぎて、一瞬圧倒されただけだ」

カイトは周囲を見渡した。境界線の向こう側—そこは、彼が知る世界とは全く異なる風景だった。

空は薄紫と深藍が混ざり合った不思議な色合いで、雲のように見えるものが、よく見れば流動するエネルギーの塊だと分かる。地面は白銀の結晶で覆われ、歩くたびに微かな光の波紋が広がる。植物と思しきものは、半透明の幹に、虹色に輝く葉をつけていた。

「美しい……」

シオンが呟いた。銀髪の少女は、周囲の景色に目を奪われている。

「七属性のどれにも属さないマナ。これが原初の状態……」

シオンの七属性魔法回路が、この環境に反応して微かに発光していた。彼女自身もその異変に気づいているようで、自分の胸元を不思議そうに触れている。

「私の魔力が……いつもより数倍、いや、十倍以上感じられる。まるで、ここに来た瞬間に能力が解放されたみたい」

カイトは腕を組んだ。プログラマーとしての類推が働く。

「この世界のマナは、まだ七つの属性に分化していない『生のデータ』みたいなものなんだろう。お前の魔法回路は、それを直接取り込めるから、効率が圧倒的にいいんだ」
「生のデータ、ですか?」

リアムが首を傾げた。

「ああ。俺たちの世界のマナは、炎、水、風、地、光、闇、雷の七つの属性に分類されているだろ? それは、原初のマナが『処理』された結果なんだよ。一方、この世界のマナは……」

カイトは結晶の地面を指差した。

「『処理』される前の状態。つまり、どんな属性にもなれる可能性を秘めている」

◆ ◇ ◆

一行は慎重に歩を進めた。結晶の大地は、見た目以上に頑丈で、足跡すら残らない。周囲には不思議な音が響いている—風の音でも、水の音でもない。もっと根源的な、世界そのものの鼓動のような音だった。

「カイト様、前方に反応があります」

リアムが風魔法で周囲を探知しながら報告した。

「敵意は感じられませんが……奇妙です。一つの場所に、大量のマナが凝縮されています」

カイトは頷いた。

「行ってみよう。何があるか分からないが、少なくとも、ここがどんな場所なのかの手がかりになるはずだ」

彼らは結晶の丘を越え、小さな谷間へと降りていった。そこで目にした光景に、全員が息を呑んだ。

谷の中心に、巨大な「泉」があった。

水ではない。液体ですらない。泉の中身は、純粋な光の粒子が集まってできた、脈動するエネルギーの塊だった。泉の中心から、太い光の柱が天に向かって伸びている。

「これって……」

エリナが言葉を失った。

「原初のマナの泉。間違いない」

カイトが答えた。彼の目には、泉の構造がコードとして映っていた。

PRIMORDIAL_SPRING {
  type: "mana_source",
  purity: 1.0,
  output: INFINITE,
  connected_realms: ["world_alpha", "world_beta", ...]
}
「この泉が、複数の世界にマナを供給している。俺たちの世界も、ここからエネルギーを受け取っているんだ」

カイトの説明に、セシリアが静かに頷いた。

「つまり、この泉が世界の生命力そのもの……破壊されれば、全ての世界が枯渇する」
「逆に言えば、制御できれば、全ての世界を救える」

カイトは泉に近づいた。光の粒子が、彼に呼応するように揺らめく。

「アレクシスたちは、この泉を守るために境界線を作ったんだ。外からの脅威—十三使徒の源流—がここに辿り着かないように」

◆ ◇ ◆

その時、泉の表面が波打った。光の粒子が渦を巻き、何かが形作られようとしていた。

「来るぞ!」

カイトが警告した瞬間、泉から巨大な影が立ち上がった。それは、獣でも人でもない—歪んだ幾何学模様が集合した、異形の存在だった。

「これは……何なんです?」

シオンが魔法を構えた。

「境界線の向こう側に潜んでいた『歪み』だ。俺たちが境界線を再設計した時、何かが紛れ込んだんだ」

カイトは即座に分析した。歪みの正体—それは、境界線の隙間を縫って侵入した、「世界のノイズ」のような存在。有害というよりは、制御不能なバグに近い。

「戦う必要はない。修正すればいい」

カイトはコードを書き始めた。

function purifyDistortion(target) {
  target.structure.analyze()
    .then(pattern => {
      if (pattern.anomaly) {
        target.restructure(pattern.idealForm);
      }
    });
}

purifyDistortion(distortion_entity);

カイトの指先から、光の鎖が伸びた。鎖は歪みを包み込み、その構造を解析・再構築していく。歪みは激しく抵抗したが、やがて光の中に溶け込み、消滅した。

残ったのは、小さな光の蝶だった。

「え……?」

シオンが目を見張った。

「歪みは、『悪意ある存在』じゃなかった。ただ、形を失ったマナが暴走していただけだ。俺はそれに正しい形を与えた——」

蝶はカイトの周りを舞い、やがて泉へと戻っていった。

◆ ◇ ◆

「カイト、あなたの魔法は本当に不思議ですね」

エリナが微笑んだ。

「普通の魔法使いなら、あの歪みを『敵』として破壊していたでしょう。でも、あなたは『修正』した。殺すのではなく、救う。それが、あなたの魔法の本質なんでしょうね」

カイトは首を振った。

「大げさだよ。俺はプログラマーだから、バグがあったら修正するのが当然だと思ってるだけだ。破壊するのは最後の手段だろ?」

リアムが静かに言った。

「その考え方が、あなたを他の管理者と一線を画す所以なのでしょう。アレクシス殿やイヴェルナ殿も、カイト様なら世界を正しい方向へ導けると信じていたはずです」

カイトは泉を見つめた。光の柱は、変わらず天へと伸びている。

「さて、ここが原初の領域だってことは分かった。次は、何を探せばいいんだ?」

その問いに答えるように、泉の奥から声が響いた。

「次の試練は、三つある」

声の主は、泉の表面に映る影だった。アレクシスの残響——いや、残響よりも濃い存在感。これは、彼の「意識の一部」そのものだった。

「第一の試練は、『理解』。この世界の仕組みを理解すること——お前は既に果たした」
「第二の試練は、『修正』。歪みを破壊せず、正しい形へ導くこと——これも果たした」

影がカイトを見つめた。

「第三の試練は、『選択』。この泉の運命を決めること。維持するか、変革するか、それとも——」

言葉が途切れた。

「——破棄するか」

カイトは眉をひそめた。

「破棄? それって、泉を消すってことか?」
「そうだ。泉が破棄されれば、原初のマナは全ての世界に開放される。各世界は、自前でマナを生み出す力を得る。だが、同時に、制御を失ったマナが暴走するリスクも高まる」

アレクシスの影は、静かに語り続けた。

「維持すれば、現状が続く。世界は守られるが、変化はない。変革すれば、泉の構造を書き換えられる。より効率的に、より安全に。だが、その過程で一時的な不安定が生じる」

カイトは拳を握った。またしても、選択だ。境界線の時と同じ。どれを選んでも、代償がある。

「お前には、『第四の道』を切り開く力があるかもしれない」

アレクシスの影が、最後に言った。

「プログラマーであるお前なら、三つの選択肢に縛られず、新しい答えを見つけられるかもしれない。だが、それはお前自身が見つけるものだ。我々は、道標を示すことしかできない」

影は、泉の奥へと消えていった。

◆ ◇ ◆

「第四の道、か……」

カイトは呟いた。仲間たちが、彼を見守っている。

「カイト、無理に答えを出す必要はありません」

セシリアが言った。

「まずは、この領域を探索しましょう。答えは、その中で見つかるかもしれません」

カイトは頷いた。確かに、焦る必要はない。泉を巡る選択は、この旅のゴールだ。まだ、道のりは長い。

「よし、まずは探索だ。この世界に何があるか、ちゃんと把握しよう」

一行は、泉を背にして歩き出した。原初の領域——そこは、世界の始まりであり、終わりの場所。カイトの冒険は、まだ続いていく。

結晶の大地に、彼らの足跡が刻まれた。それは、この世界に新しい歴史が刻まれ始めた証だった。
