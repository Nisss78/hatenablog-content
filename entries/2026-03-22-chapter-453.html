第453章：風の鈴

街道は、次第に山道へと変わっていった。

アルヴァレス支部を出て三時間。カイトたちは、北東の山脈の麓へと近づいていた。道の両脇には、背の高い木々が並び、その枝が風に揺れて、ざわざわと音を立てている。

「風が強くなってきたね」

エリナが、髪を押さえながら言った。

「山の近くだから、風が通り抜けやすくなってるんだ」

リアムが解説した。

「谷風って言って、山の斜面に沿って風が吹き上げる現象がある」

「へぇ、詳しいね」

「昔、ここで修行したって言ったろ」

「ああ、そうだった」

カイトは、周囲の風景に目を向けた。木々の間から、遠くの山並みが見える。その頂には、白い雪が輝いていた。

（綺麗な景色だ）

カイトは、ふと立ち止まった。風が、頬を撫でていく。冷たくて、でも優しい感触。

「カイト？」

セシリアが振り返った。

「何か感じるの？」

「……うん」

カイトは、目を閉じた。風の音に、耳を澄ます。木の葉が擦れ合う音。鳥のさえずり。遠くの川のせせらぎ。そして——。

「……来たのね」

微かな声が、風に乗って届いた。昨日聞いたのと同じ声だ。風の賢者。

「……村で、待っているわ」

「風の賢者だ」

カイトは、目を開けた。

「また声が聞こえた。村で待ってるって」

「村って、ウィンドベル村のこと？」

エリナが尋ねた。

「たぶん」

「じゃあ、急ごう！」

カイトたちは、足を速めた。

＊＊＊

さらに一時間ほど歩いた頃、道の先に、小さな村が見えてきた。

ウィンドベル村。

ヒナタの説明通り、村のあちこちに、風鈴が吊るされていた。家の軒先、木の枝、村の入り口。大小様々な風鈴が、風に揺れて、涼やかな音を奏でている。

「すごい……」

エリナが、息を呑んだ。

「まるで、風の演奏会みたい」

実際、村全体が一つの巨大な楽器のように思えた。風鈴の音色が、風と共に流れ、調和し、美しいメロディを紡ぎ出している。

「これが、ウィンドベル村か」

リアムが、懐かしそうに言った。

「昔と変わらないな」

「リアム、ここにも来たことあるの？」

「一度だけ。修行の合間に」

リアムは、村の入り口にある古びた碑を指差した。

「この碑には、村の成り立ちが刻まれている。昔々、風の賢者がこの地に留まり、村人たちに風鈴の作り方を教えた——そういう伝説がある」

「風の賢者が、直接？」

カイトは驚いた。

「そういうことになってる」

リアムは肩をすくめた。

「でも、本当かどうかはわからない。伝説だからな」

「どちらにせよ、風の賢者と深い関わりがある村であることは間違いない」

セシリアが結論づけた。

「まずは、村長に挨拶してから、祠に向かいましょう」

「村長って、知ってるの？」

エリナが尋ねた。

「以前、ギルドの任務で来た時に」

セシリアは、村の奥にある一際大きな家を指差した。

「あそこが、村長の家です」

＊＊＊

村長の家を訪ねると、意外な人物が出迎えた。

「おや、見覚えがある顔だね」

老人だった。白髪に白い髭。杖をついているが、背筋は伸びている。目は穏やかで、知的な光を宿していた。

「アルフレッドさん？」

カイトは驚いた。守人のアルフレッド。カイトたちの旅に同行していた老人だ。数日前、本部に残ると言って別れたばかり。

「なぜ、ここに？」

「ふふ、驚かせてしまいましたか」

アルフレッドは笑った。

「実は、ここウィンドベル村の出身でしてね。久しぶりに帰ってきました」

「出身地だったんですね」

「ええ。若い頃は、この村で修行をしていました。風の賢者について、学んでいたのです」

「風の賢者について？」

カイトは目を輝かせた。

「何か知ってるんですか？」

「知っていますとも」

アルフレッドは、杖で地面をトントンと叩いた。

「風の賢者は、名をシルヴィアと言います。女性で、長い銀髪を持つ、美しい方です」

「シルヴィア……」

カイトは、その名を心に刻んだ。

「シルヴィアさんに会うには、どうすればいいんですか？」

「風の祠に行くことです」

アルフレッドは、村の外れを指差した。

「祠には、風の賢者が残した『試練』があります。その試練を乗り越えれば、シルヴィアさんに会えるでしょう」

「試練……」

カイトは頷いた。これまでの賢者たちも、それぞれ試練を課してきた。風の賢者も、例外ではないようだ。

「案内しましょう」

アルフレッドは、歩き出した。

「ついてきてください」

＊＊＊

村の外れにある祠は、小さな石造りの建物だった。苔むした壁に、風の紋章が刻まれている。扉はなく、中は暗い。

「ここが、風の祠です」

アルフレッドは、祠の入り口に立った。

「中には、風の賢者が残した『言葉』があります。その言葉を理解できれば——道は開かれるでしょう」

「言葉だけ？」

エリナが首を傾げた。

「戦わなくていいの？」

「風は、戦うものではありません」

アルフレッドは静かに言った。

「風は、感じるものです。理解するものです。戦って得られるものではないのです」

「……なるほど」

カイトは、祠の中へと足を踏み入れた。

中は、意外なほど明るかった。天井に小さな穴が開いていて、そこから光が差し込んでいる。床には、風鈴がいくつも置かれていた。風がないのに、風鈴は揺れて、音を奏でている。

「風がないのに、動いてる……」

エリナが不思議そうに言った。

「見えない風が吹いているのだろう」

リアムが推測した。

「魔力で生み出された風、あるいは——」

「あるいは、自然に発生している風」

セシリアが言った。

「この場所自体が、風の通り道になっているのかもしれません」

カイトは、祠の奥へと進んだ。そこに、一枚の石板があった。表面には、文字が刻まれている。

風は見えず
風は掴めず
風は名を持たず

されど、風は在り
風は響き
風は運ぶ

問う——風の名は、何か

「風の名……」

カイトは、石板の前に膝をついた。

風に名前はない。それは、これまでの考察と一致する。風は自由で、定義されない存在。名前をつけること自体が、風を縛る行為だ。

でも、問われているのは「風の名は何か」。名前がないと答えるのは、正解なのだろうか？ それとも——。

（ヒントがあるはずだ）

カイトは、風鈴を見上げた。風がないのに揺れる風鈴。音色が、耳に届く。涼やかで、優しい音。

（風鈴が鳴るのは——風が吹いているから）

見えない風が、風鈴を揺らす。風鈴の音が、風の存在を知らせる。

（風は見えない。でも——音は聞こえる）

カイトは、はっとした。

「風の名は——」

カイトは、石板に向かって答えた。

「音だ」

その瞬間、風鈴の音が激しくなった。石板の後ろにある壁が、音もなく開いた。中から、強い風が吹き出してくる。

「カイト！」

エリナが叫んだ。カイトは、風に巻き込まれそうになった。でも——不思議と、怖くはなかった。

（この風は、敵意を持っていない）

カイトは、風の中で目を閉じた。風が、体を包み込む。優しくて、温かい。まるで、抱きしめられているような。

「正解です」

声が、風の中から響いた。

「風には、形がありません。名前もありません。でも——音は、あります。風の音を聞くことで、人は風を知覚できる。風の名は、音。あるいは——響き」

風が、ゆっくりと収まった。カイトが目を開けると、目の前に一人の女性が立っていた。

銀色の長髪。透き通るような白い肌。瞳は、風のように淡い青色。彼女は、白いワンピースを着て、裸足で立っていた。

「シルヴィアさん……？」

カイトは、問いかけた。

「ええ。風の賢者、シルヴィアです」

彼女は、静かに微笑んだ。

「ようこそ、カイトくん。あなたのこと、風の便りで聞いていました」

「風の便り……」

「風は、世界中を巡ります。様々な情報を運んできます。あなたが六大元素を集め、境界修復を目指していること——風が教えてくれました」

シルヴィアは、カイトに近づいた。

「あなたの『魔法をコードで書く』能力、興味深いですね」

「興味を持ってくれたんですか？」

「ええ。風と、コード——似ていると思いませんか？」

「似ている？」

カイトは首を傾げた。

「コードは、論理の流れです。条件が揃えば、実行される。風も、条件が揃えば吹きます。温度差、気圧差、地形——すべてが揃って、風が生まれる」

「……確かに」

カイトは考え込んだ。言われてみれば、その通りだ。風もコードも、因果関係で動いている。

「でも、決定的な違いがあります」

シルヴィアは、指を立てた。

「コードは、書かれた通りに動く。予測可能です。でも——風は、予測できません。条件が揃っても、どう吹くかは、その時次第」

「その時次第……」

「風は、自由です。縛られない。だからこそ——風の魔法は、最も扱いが難しい」

シルヴィアは、カイトに手を差し出した。

「カイトくん。あなたに、風の力を預けましょう。でも——ただではあげません」

「やっぱり、試練ですか？」

「いいえ。試練は、すでに乗り越えました」

シルヴィアは笑った。

「風の名を当てたこと。それが、資格の証です」

「じゃあ、なぜ——」

「代償を、求めているのです」

「代償？」

「風の力を得るには——風に、何かを返さなければなりません。見返りなしに、風は吹かない。それが、風の摂理です」

「何を返せばいいんですか？」

「あなたの、コードの一部」

シルヴィアは、カイトの目を見つめた。

「あなたが書いた魔法のコード——その一部を、風に預けてください。風が、あなたの魔法を運び、世界に広めるのです」

「僕の魔法を、世界に……」

「ええ。風は、情報を運びます。あなたの『魔法をコードで書く』という概念——それを、世界中の魔法使いに届けたいのです」

カイトは、迷った。自分の魔法を、世界中に広める。それは、大きなことだ。もしかしたら、悪用されるかもしれない。でも——。

（悪用を恐れて、良いことを諦めるのは違う）

カイトは、結論を出した。

「わかりました。僕のコードを、風に預けます」

「ありがとうございます」

シルヴィアは、カイトの手を握った。

「では——風の祝福を」

その瞬間、カイトの体が光に包まれた。風の力が、体の中に流れ込んでくる。温かくて、軽やかで、自由な力。


// 風の力を受け取った
const windBlessing = {
  element: "wind",
  freedom: true,
  unpredictability: true,
  carrier: "information"
};

// カイトの魔法は、風と共に世界へ
windBlessing.carry(magicCode);

光が収まると、カイトの体には、淡い青色の紋章が浮かんでいた。風の賢者の証だ。

「六大元素、揃いましたね」

シルヴィアは満足げに頷いた。

「火、氷、雷、地、光、闇——そして風。これで、境界修復ができます」

「はい。ありがとうございます、シルヴィアさん」

カイトは、深く頭を下げた。

「いえ。あなたが、風の名を理解してくれたこと——それが、私の喜びです」

シルヴィアは、風鈴の音を聞くような目で、カイトを見つめた。

「行きなさい、カイトくん。境界修復へ。そして——世界を救いなさい」

「はい！」

カイトは、仲間たちのもとへ戻った。エリナ、リアム、セシリア、アルフレッド——全員が、カイトの成功を祝福してくれた。

「六大元素、コンプリートだね！」

エリナが喜んだ。

「ついに、ここまで来たか」

リアムも、珍しく笑顔を見せた。

「おめでとうございます、カイトさん」

セシリアが、穏やかに微笑んだ。

「さあ、次は境界修復です。中央の塔へ戻りましょう」

「うん」

カイトは頷いた。風鈴の音色が、背中を押してくれる。

（風は、運ぶ）

カイトは、空を見上げた。青い空に、白い雲が流れている。

（僕の魔法も、風に乗って——誰かのところへ届くといいな）

カイトたちは、村を後にした。ウィンドベル村の風鈴が、最後に一度だけ、大きく鳴った——まるで、旅立つ者への餞別のように。
