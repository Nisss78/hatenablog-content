第686章：光の塔の第一層

扉が開いた瞬間、まばゆい光が私たちを包み込んだ。目を開けていられないほどの輝き。まるで太陽の中心に足を踏み入れたかのような感覚だった。

「くっ……！」

エリナが腕で顔を覆う。シオンも目を細め、身を固くしている。私は瞬時に防御魔法を展開した。

def magic_shield(targets):
    return LightFilter(
        intensity="adaptive",
        max_lux=10000,
        wavelength="visible"
    ).apply(targets)

淡い青色の薄膜が私たちを覆い、光の強度を調整する。ようやく目が慣れてきた。

「みんな、大丈夫？」

私が声をかけると、エリナが深呼吸をして頷いた。

「ええ……びっくりしたわ。中に入った瞬間にあんな強い光が来るなんて」

シオンが金色の瞳を瞬かせながら周囲を見渡す。

「ここが……光の塔の中……」

私も改めて周囲を観察した。扉の先は広大なホールになっていた。壁も床も天井も、すべてが白く輝く結晶のような素材で作られている。光源は見当たらない。それなのに、空間全体が均一に明るい。

「光魔法そのものが構造体に練り込まれているようだ」

リアムが壁に手を触れながら分析する。

「古代の技術だな。現代では再現できないレベルの魔導工学だ」

「ルミナス様が作ったのでしょうか？」

セシリアの問いに、リアムは首を横に振った。

「いや、これより古い。光の賢者が建てたというより、光の賢者を生んだ場所かもしれない」

その言葉に、全員が沈黙した。この塔には、まだ私たちが知らない何かが眠っているのかもしれない。

「進もう」

私が前を向いて言った。

「ここまで来て立ち止まっているわけにはいかない」

ホールの奥に、大きな階段が見える。螺旋状に上へと続いている。どうやら塔を登っていく構造のようだ。

階段に足をかけた瞬間、空気が変わった。

「待って」

シオンが私の袖を引く。

「何か……いる」

彼女の警告と同時に、階段の上から光の粒子が降り注いだ。それは次第に人の形を取り始める。

現れたのは、半透明の体を持つ女性の姿だった。純白の長い髪、閉じた瞼、背中の翼。まるで天使のようだった。

「光の塔へようこそ、旅人たちよ」

女性が口を開くと、鈴のような声が響いた。目は開かない。それでも、彼女は確かに私たちを見据えていた。

「私は第一層の守護者。名はアルバ」

「守護者……」

エリナが身構える。私は手で制して、前に出た。

「私たちは、光の賢者ルミナス様に会いに来ました」

アルバと名乗った守護者は、静かに首を横に振った。

そのために、この塔を登らねばなりません。七つの層を超えて、頂上へ」

「七層……」

私は階段を見上げた。かなり高そうだ。

「各層で試練が待っています。光の資格なき者、頂上に立つことはできません」

「試練、か」

リアムが顎に手を当てる。

「予想通りだな。賢者の塔だからな」

「第一層の試練は何だ？」

私が問うと、アルバの唇が微かに微笑んだように見えた。

「光を怖れず、光に惑わされず」

彼女が両手を広げると、周囲の光が激しく脈動し始めた。

「私の光を超えていきなさい」

瞬間、眩暈が襲った。

視界が歪む。世界が回転する感覚。地面がなくなり、空と地面が入れ替わる。

「うっ……！」

膝から崩れ落ちそうになるのを、必死で堪えた。隣でエリナがよろめき、セシリアが杖をついて体を支えている。

「これは……！」

リアムが苦悶の声を上げる。

「平衡感覚を狂わせる魔法か……！」

シオンだけが微動だにせず立っていた。彼女の金色の瞳が、真っ直ぐにアルバを見つめている。

「幻覚じゃない」

シオンが静かに言った。

「光の情報量が多すぎて、脳が処理できていないだけ」

その言葉に、私はハッとした。

そうだ。これは魔法攻撃じゃない。情報の過負荷だ。この空間全体が、私たちの知覚能力を超えた光の情報を放っている。

私は目を閉じた。

視覚情報を遮断する。代わりに、コードでこの状況を解析する。

class SensoryProcessor:
    def __init__(self):
        self.visual_buffer = []
        self.max_input_rate = 1000  # per second
    
    def filter(self, raw_input):
        # 過剰な光情報を間引く
        if len(raw_input) > self.max_input_rate:
            return self.downsample(raw_input)
        return raw_input
    
    def downsample(self, data):
        return data[::10]  # 1/10に間引く

魔法として出力する。

「SensoryFilter.activate()」

私の脳に、情報のフィルターがかかる。光の洪水が、処理可能な量に整理されていく。

目を開けた。

世界は安定していた。脈動する光も、ただの照明に見える。

「カイト……！」

エリナが私を見る。私は彼女の肩を掴み、同じ魔法をかける。

「Share.filter()」

エリナの表情が和らいだ。

「ありがとう……落ち着いたわ」

私は続けてセシリア、リアム、そしてシオンにも同じ処理を適用した。全員の呼吸が整う。

アルバが、初めて目を開いた。

その瞳は、光そのもののように輝いていた。

「見事です」

彼女が静かに拍手を送る。

「光の恐怖に屈せず、分析し、対処した。まさに求めていた資質」

「資質……？」

「光の魔法は、最も純粋で、最も残酷です。その美しさに魅せられ、飲み込まれる者は多い」

アルバが一歩、私たちに近づく。

「ですが、あなたたちは違う。光を見つめ、理解しようとした。それが第一層の試練の本質です」

彼女が手を差し出すと、光の粒子が集まり、一つの結晶になった。

「第一層の証。持っていきなさい」

私は結晶を受け取った。温かい。まるで生命を感じるような温度だ。

「残り六つの層でも、同じように試練が待っています。ですが、覚えておきなさい」

アルバの姿が、徐々に薄れ始める。

「光は答えを与えません。ただ、真実を照らすのみ。あなたたち自身の答えを見つけなさい」

守護者の姿が、光の中に溶けて消えた。

後に残されたのは、静寂と、私たちだけ。

「……行こう」

私が言うと、みんなが頷いた。

螺旋階段を登り始める。第一層をクリアした。あと六つ。

塔の頂上で、光の賢者が待っている。

足取りは軽い。でも、胸の奥には緊張が残っていた。これからの試練は、もっと厳しいものになるだろう。

それでも、私は進む。

仲間と共に。コードと魔法を武器に。

光の塔の階段は、どこまでも続いているようだった。
