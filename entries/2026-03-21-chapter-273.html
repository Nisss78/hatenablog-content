第273章：始原の森への道

天空城を後にしたカイトたちは、光の階段を降りて地上へと戻った。

雲海を抜け、青空を突き抜け、やがて緑の大地が見えてくる。空からの眺めは圧倒的だったが、カイトの心はすでに先へ向いていた。

「始原の森……か」

カイトは呟く。

セレスティアの言葉が頭の中で反響していた。

中央大陸の最深部。人類未踏の地。世界樹の眠る場所。

「どのくらいかかりそう？」

セシリアが尋ねる。

「空から飛べば、半日ってところかな」

エリナが空を見上げる。

「でも、まともな道はないよ。地図にも載ってない場所だし」

ヒナタが付け加える。

カイトは腰の袋から、六つのアーティファクトを取り出した。炎の宝石、水の雫、風の羽、土の輝石、空の水晶——そして、天空の王冠。


const artifacts = {
    fire: new FireGem(),
    water: new WaterDroplet(),
    wind: new WindFeather(),
    earth: new EarthCrystal(),
    sky: new SkyJewel(),
    crown: new SkyCrown()
};

function locateDestination() {
    // 六つのアーティファクトを共鳴させて位置を特定
    return artifacts.crown.resonate([
        artifacts.fire,
        artifacts.water,
        artifacts.wind,
        artifacts.earth,
        artifacts.sky
    ]);
}

カイトは王冠を被った。すると、王冠のクリスタルが淡い光を放ち始める。

光は一方向を指し示していた——北東。そこに、始原の森がある。

「すごい……王冠が道を示してる」

リアムが感嘆する。

「これなら迷わないね」

カイトは王冠を袋に戻そうとしたが——ふと、ためらった。

「カイト？」

エリナが首を傾げる。

「うん、やっぱり被ったままにするよ。いざという時、すぐに使えるように」

カイトは白銀の冠を頭に載せた。不思議と、重さを感じない。むしろ、頭が軽くなったような感覚だ。

「似合ってるよ」

セシリアが笑う。

「王冠かぶったプログラマーなんて、世界広しといえどもカイトだけだね」
「光栄だね」

カイトも苦笑する。

五人は空を飛び始めた。カイトの王冠が生み出す風の通路に乗って、目的地へと向かう。

眼下には、緑の森が広がっている。やがて森は深くなり、木々の密度が増していく。

「あそこが境界線かな」

ヒナタが指差す。

前方に、不自然な霧の壁が見えた。霧は濃く、中を覗き込むことはできない。

「始原の森の入り口……」

リアムが息を呑む。

「人類未踏の地、か。緊張するね」

エリナが剣の柄に手をかける。

カイトたちは霧の前で降り立った。

霧に近づくと、冷気を感じた。普通の霧ではない。魔力を帯びた、結界のようなものだ。

「この霧、ただの水蒸気じゃない」

カイトが分析する。

「魔力が凝縮された霧……世界樹を守るための結界かな」

カイトは推測する。

「どうやって入るの？」

セシリアが尋ねる。

「多分、これが鍵だ」

カイトは王冠に触れる。

王冠を被ったまま、霧に一歩近づく。

すると——

霧が、道を開けた。

まるで生き物のように、霧が左右に分かれていく。中央に、一本の細い道が現れた。

「やっぱり！ 王冠を持ってれば、通れる！」

カイトが振り返る。

「みんな、僕の後についてきて」

五人は霧の中へと足を踏み入れた。

始原の森の中は、異世界のようだった。

木々は巨大で、幹の直径だけでも数十メートルはある。枝は空を覆い尽くし、太陽の光はわずかに差し込むだけ。地面は苔とシダ植物で覆われ、歩くたびにふかふかとした感触がする。

「すごい……こんな場所があったなんて」

セシリアが周囲を見渡す。

「ここにある植物、全部が普通の種類じゃないわ」

エリナが植物に触れる。

「魔力を帯びてる。普通の森とは、生態系が違う」
「世界樹の影響だろうね」

リアムが呟く。

「世界樹は、すべての生命と魔力の源。その周辺は、特別な環境になるはずだ」

カイトは王冠の力を使って、道を探る。

王冠が指し示す方向は、森の奥——深く、暗い場所へと続いている。

「こっちだ」

カイトが歩き出す。

しばらく進むと、奇妙な音が聞こえ始めた。

ザワザワ、ザワザワ……

風の音？ いや、違う。

「何かいる」

ヒナタが剣を抜く。

カイトたちの周囲の茂みが、不自然に揺れ始めた。

そして——

影が飛び出してきた。

それは、狼のような生物だった。しかし、普通の狼ではない。体は半透明で、中に星々が輝いている。宇宙の断片を形どったような、神秘的な存在だ。

「星狼（せいうろ）！」

リアムが叫ぶ。

「始原の森に住む、伝説の守護獣！」

星狼は一匹ではなかった。五匹、十匹——次々と茂みから現れ、カイトたちを包囲していく。

「敵……？」

セシリアが身構える。

「いや、待って」

カイトが制止する。

星狼たちは、攻撃の構えをしていない。ただ、カイトたちを観察している。

「彼らは、僕たちが敵か味方か見極めようとしてる」

カイトは剣を抜かず、両手を広げた。

「僕たちは敵じゃない。世界樹を守りに来たんだ」

星狼の一匹——リーダーらしき、一際大きな個体——が、カイトの前に進み出た。

金色の瞳が、カイトをじっと見つめる。

カイトは王冠に触れる。

「これが見える？ 僕は、空の賢者セレスティア様から、この王冠を託された」

王冠を見た瞬間、星狼の態度が変わった。

敵対的な雰囲気が消え、代わりに敬意のようなものが現れる。

リーダーの星狼は、カイトの前で頭を下げた。

「通してくれるんだ」

カイトが安堵する。

星狼たちは道を開け、さらに奥へと続く獣道を示した。

「ありがとう」

カイトは星狼に礼を言い、再び歩き出す。

仲間たちも、安堵の息を吐く。

「危なかった……戦わなくてよかった」

セシリアが胸を撫で下ろす。

「王冠があってよかったね」

ヒナタが言う。

森をさらに進むと、空気が変わった。

濃密な魔力。呼吸するだけで、体の中に力が満ちてくるような感覚。

「近い……」

リアムが呟く。

「世界樹が近い」

木々の間から、巨大な影が見えた。

それは、山のように大きな樹だった。

幹は太く、見上げるほどの高さがある。枝は雲を突き抜け、空の彼方まで届いているようだ。樹皮には、古代のルーン文字が刻まれ、淡い光を放っている。

「あれが……世界樹」

カイトが息を呑む。

圧倒的な存在感。ただそこにあるだけで、世界を支えていることが分かる。

しかし——

世界樹の根元に、異様な光景が広がっていた。

黒い霧が渦巻き、樹の幹を覆っている。霧は脈動し、まるで世界樹の生命力を吸い取っているようだった。

「あれが……アークライトの仕業？」

エリナが声を震わせる。

「世界樹の魔力を吸収してる……」

リアムが絶望的な声で言う。

そして——

黒い霧の中から、一人の人物が現れた。

長い黒髪、白皙の肌、そして冷徹な金色の瞳。黒いローブを纏い、全身から負のオーラを放っている。

「久しぶりだな、小僧」

アークライトが、冷ややかな笑みを浮かべる。

「六つのアーティファクトを集めたか。感心なことだ」

カイトは剣を抜く。

「アークライト！ 世界樹から離れろ！」

アークライトは鼻で笑う。

「離れろ？ なぜだ？ 私は今、最高の気分だ」

アークライトは手を広げる。

「この樹の魔力——計り知れない。五百年の封印で失った力を、すべて取り戻せる」
「お前のためにある力じゃない！」

エリナが叫ぶ。

「世界樹は、すべての生命の源！ お前が吸い取れば、世界が滅ぶ！」

アークライトは無表情で答える。

「滅びる？ いや、違う。作り直されるのだ」

アークライトの目に、狂気が宿る。

「この世界は不完全だ。争い、貧困、病——無数のバグを抱えている。私は、それらをすべて取り除き、完璧な世界を作る」

カイトは眉をひそめる。

バグ……？

プログラマーとしてのカイトには、アークライトの言葉が奇妙に響いた。

「お前、プログラマーか何かか？」

カイトが尋ねる。

アークライトは首を傾げる。

「プログラマー？ 聞き慣れぬ言葉だ。だが——私のやろうとしていることは、その言葉に近いかもしれん」

アークライトは、世界樹に手をかざす。

「私は、この世界の構造を書き換える。根源から、すべてを」

カイトは悟った。

アークライトは、カイトと同じ——異世界から来た存在だ。

いや、違う。アークライトが来たのは、五百年前。カイトの世界とは、違う時代かもしれない。

でも、考え方は似ている。

アークライトは、この世界を「システム」として捉えている。そして、そのシステムを書き換えようとしている。

「お前のやり方は間違ってる」

カイトが言う。

「システムを書き換えるには、バックアップが必要だ。テストも必要だ。いきなり本番環境で書き換えたら、何が起きるか分からない」

アークライトは眉をひそめる。

「……何を言っている？」
「僕の言いたいことは一つだ」

カイトは剣を構える。

「お前の計画には、致命的なバグがある」

カイトの後ろで、仲間たちも構える。

「そして、そのバグを修正するのが——僕たちだ！」

アークライトは、冷ややかな笑みを浮かべた。

「面白い。来るがいい」

黒い霧が、アークライトの周りに渦巻く。

最後の戦いが、始まろうとしていた。
