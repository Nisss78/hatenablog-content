第527章：地の深淵

北大陸の西の山脈を下りたカイトたちは、ノースポートから中央大陸行きの船に乗った。

甲板で、カイトは海を眺めていた。

青い海が、果てしなく広がっている。

「カイトさん、考え事？」

マヤが、隣に来た。

「ちょっとね。地の聖域について、考えていたんだ」

カイトは、視線を海に向けたまま言った。

「以前、中央大陸の地下道で戦ったことがある。でも、地の聖域そのものには、行ったことがない」
「どんな場所なんですか？」

ルナが、興味津々で尋ねた。

「地の賢者テラドンの領域。大地の力が凝縮された場所だと聞いている」

カイトの中の原初の闇が、補足した。

「地の聖域は、中央大陸の地下深くにある。地表から数千メートル下だ」
「数千メートル……！　すごい深さだね」

ルナが、驚いたように言った。

船旅は三日間続いた。

四日目の朝、中央大陸の港町セントラルポートに到着した。

セントラルポートは、五大陸の中で最も大きな港町だ。

港には、世界中の船が行き交っている。

「久しぶりに来たけど、相変わらず賑やかだね」

マヤが、通りを行く人々を見ながら言った。

「中央ギルドの支部も、この街にある。報告してから、地の聖域に向かおう」

カイトは、支部の方へ歩き出した。

中央ギルドの支部は、街の中心部にあった。

巨大な石造りの建物。入り口には、ギルドの紋章が掲げられている。

カイトたちは、支部の中に入った。

広いロビーには、多くの冒険者が集まっていた。

「カイトさん！　お久しぶりです！」

受付の女性が、笑顔で挨拶した。

「マヤさん、ルナさんも。元気でしたか？」
「はい！　ミラさんもお元気そうで」

マヤが、笑い返した。

「カイトさん、テオドール支部長がお会いしたいと仰っていました」

ミラが、言った。

「わかりました。案内をお願いします」

テオドール支部長の執務室は、支部の最上階にあった。

部屋に入ると、白髪の老人が立ち上がった。

「カイト君、来たか。座りたまえ」

テオドールは、穏やかな声で言った。

「報告は、セレナ調査官から聞いています。七つの心臓を集める旅、ですね」
「はい。風の心臓は、すでに手に入れました」

カイトは、懐から風の心臓を取り出した。

半透明の結晶が、青い光を放っている。

「これが……風の心臓」

テオドールが、目を細めた。

「美しい。そして、強大な力を感じる」

カイトは、地の心臓について尋ねた。

「支部長、地の聖域への入り口をご存知ですか？」
「ああ。中央大陸の北東、山岳地帯にある洞窟が入り口だ」

テオドールは、地図を取り出した。

「ここだ。『大地の裂け目』と呼ばれている」
「大地の裂け目……」
「かつて、多くの冒険者が地の心臓を求めて挑んだが、誰も手に入れていない」

テオドールの声に、警告の響きがあった。

「地の試練は、『重圧の試練』。物理的な圧力だけでなく、精神的な重圧も襲いかかる」
「重圧の試練……」
「カイト君、君なら乗り越えられるかもしれない。だが、過信はするな」

テオドールとの面会を終えたカイトたちは、支部を出た。

街の宿で一泊し、翌朝、北東の山岳地帯へ向かった。

馬車で半日。やがて、巨大な裂け目が見えてきた。

大地が、まるで切り裂かれたように裂けている。

その裂け目の底には、暗い闇が広がっていた。

「これが、大地の裂け目……」

カイトは、裂け目の縁に立った。

「底が見えない。どれくらい深いんだろう」

ルナが、身を乗り出した。

「危ないよ、ルナ」

マヤが、ルナの腕を引いた。

カイトの中の原初の闇が、声を上げた。

「下に、強い力を感じる。地の聖域への入り口だ」
「どうやって降りる？」
「飛び降りるのが一番早い。ただし、落下衝撃を防ぐ魔法が必要だ」

カイトは、頷いた。

// 重力制御魔法
func gravityControl() {
    let gravity = PhysicsMagic.gravity
    gravity.modify(
        target: partyMembers,
        factor: 0.1,
        duration: .untilLanding,
        properties: [
            .slowDescent(true),
            .impactAbsorption(.maximum)
        ]
    )
}

カイトは、マヤとルナに手を置いた。

「二人とも、僕に掴まって。これで安全に降りられる」
「えっ、飛び降りるの！？」

ルナが、悲鳴を上げた。

「大丈夫。僕の魔法を信じて」

カイトたちは、裂け目へと飛び込んだ。

風が、耳元を吹き抜ける。

重力制御魔法のおかげで、ゆっくりと落下していく。

暗闇の中を、三人は降りていった。

やがて、光が見えてきた。

「下に、何かある！」

マヤが、叫んだ。

地面には、巨大な洞窟が広がっていた。

洞窟の壁は、金色の鉱脈で輝いている。

「これが、地の聖域の入り口か……」

カイトたちは、優しく着地した。

洞窟の中は、迷路のように入り組んでいた。

壁には、地の紋章が刻まれている。

「どっちに行けばいいんだろう？」

ルナが、左右の通路を見比べた。

「僕が、道を探す」

カイトは、目を閉じた。

原初の闇の力を使って、地脈の流れを感じ取る。

「……右だ。地脈の流れが、そっちに向かっている」

カイトたちは、右の通路へ進んだ。

通路を進むと、広い空間に出た。

そこには、岩の巨人が立っていた。

高さは十メートル以上。全身が、岩でできている。

「何あれ！？　巨大！」

ルナが、後ずさりした。

「地の守護者……。聖域の侵入者を排除する存在だ」

カイトの中の原初の闇が、告げた。

岩の巨人が、カイトたちを見た。

「侵入者……排除……」

巨人の声は、地響きのように重かった。

巨人が、拳を振り上げた。

カイトは、即座に反応した。

// 闇魔法・影の盾

    darkness.createBarrier(
        target: partyPosition,
        properties: [
            .physicalResistance(.maximum),
            .absorption(true),
            .reflectDamage(0.3)
        ]
    )
}

黒い盾が、カイトたちの前に現れた。

巨人の拳が、盾に叩きつけられる。

衝撃波が、洞窟を揺らした。

「くっ……！」

カイトは、衝撃を耐えた。

「マヤ、ルナ！　二人は下がって！」
「でも、カイトさん！」
「大丈夫。やれる！」

カイトは、巨人に向けて魔法を放った。

黒い風の刃が、巨人を切り裂いた。

岩の体に、深い傷が入る。

「効いている！」

カイトは、追撃をかけた。

闇魔法で、巨人の動きを封じる。

風魔法で、岩の体を削り取る。

巨人が、咆哮を上げた。

「強者……認める……」

巨人の体が、崩れ始めた。

岩が、地面に崩れ落ちていく。

「倒した……」

カイトは、息を整えた。

巨人がいた場所に、階段が現れた。

「奥へ進め、ということか」

カイトたちは、階段を降りた。

階段の先には、巨大な地下空間が広がっていた。

天井は見えないほど高く、床には溶岩が流れている。

空間の中央に、土色の結晶が浮かんでいた。

「地の心臓……」

カイトは、結晶に近づいた。

結晶の前には、一人の男が立っていた。

筋肉質の体。髪は短く、瞳は茶色。

「待っていたぞ、カイト」

男は、低い声で言った。

「あなたは……？」
「地の賢者、テラドンだ」

カイトは、テラドンに礼をした。

「地の心臓を、いただきたく参りました」
「知っている。フィンブルヴェトを救うためだろう」

テラドンは、腕を組んだ。

「風の心臓は、試練を乗り越えて手に入れたと聞いている」
「はい。心の風という試練でした」
「地の試練は、『重圧の試練』だ。受ける覚悟はあるか？」

カイトは、頷いた。

「あります。どんな試練でも、乗り越えます」

テラドンは、満足そうに笑った。

「いい目だ。では、始めよう」

テラドンが、手を振ると——

カイトの体に、圧倒的な重圧が襲いかかった。

「ぐっ……！」

カイトは、膝をついた。

重力が、何倍にも増している。

「これは、物理的な重圧だけではない」

テラドンの声が、響いた。

「君の心にある、責任、使命、期待——それらすべての重さだ」

カイトの脳裏に、様々な光景が浮かんだ。

エリナの笑顔。

リアムの信頼。

セシリアの期待。

そして、フィンの姿。

「君は、多くの人々の期待を背負っている。それに押しつぶされそうになることはないか？」

テラドンの問いかけ。

カイトは、答えた。

「……ある。何度もある」

重圧が、さらに強まる。

「でも、逃げない。逃げたら、何も守れないから」

カイトは、ゆっくりと立ち上がった。

重圧は、変わらない。でも、諦めない。

「僕は、プログラマーだった。コードを書いて、問題を解決することが仕事だった」

カイトの声に、力がこもった。

「この世界でも、同じだ。目の前の問題を、一つずつ解決する。それが、僕のやり方」
「重圧は、確かに重い。でも、背負う価値があるものは、重いのが当たり前だ」

カイトは、前を見据えた。

「僕は、逃げない。最後まで、立ち続ける」

その瞬間、重圧が消えた。

カイトは、深呼吸をした。

目の前に、テラドンが立っていた。

「見事だ。重圧の試練、乗り越えた」

テラドンが、満足げに頷いた。

「君は、強い。心が」

地の心臓が、カイトの前に浮かんできた。

「受け取れ。地の心臓だ」

カイトは、結晶を手に取った。

温かく、力強い感覚が、カイトを包む。

「ありがとうございます、テラドンさん」

テラドンは、カイトの肩を叩いた。

「フィンを救ってくれ。彼は、かつて私の友だった」
「友……？」
「ああ。三百年前、彼が封印される前は、よく語り合ったものだ」

テラドンの目に、懐かしさが浮かんだ。

「彼は、優しかった。凍てつく災厄などではない」
「……はい。必ず、救います」

カイトは、決意を新たにした。

地の聖域を出たカイトたちは、地表へ戻った。

重力制御魔法で、裂け目を登る。

地表に出ると、夕日が沈んでいた。

「地の心臓、二つ目だね」

マヤが、言った。

「ああ。残りは、水、光、闇の三つ」

カイトは、懐の二つの結晶を確認した。

風と地。青と茶色の結晶が、静かに輝いている。

カイトの中の原初の闇が、囁いた。

「半分を超えた。あと三つだ」
「水、光、闇……次は、どこに行く？」
「水の聖域は、南大陸にある。光の聖域は、東大陸。闇の聖域は……境界の深淵だ」

カイトは、考えた。

「水の聖域に行こう。南大陸なら、船で行ける」
「いい判断だ。水の賢者ミズリアも、フィンと関係がある」
「ミズリア……名前に聞き覚えがある」
「以前、ミーナという少女と会ったことがあるだろう？　彼女の母だ」

カイトは、思い出した。

「ああ、ミーナちゃん！　会えるといいな」

カイトたちは、セントラルポートに戻った。

翌日、南大陸行きの船に乗る。

船旅の間、カイトは二つの心臓を見つめていた。

風の心臓——過去との決別。

地の心臓——責任との向き合い。

（二つの試練を乗り越えた。あと三つ）

カイトは、空を見上げた。

水平線の向こうに、南大陸が見え始めていた。
