<h3>第81章：共鳴の始まり</h3>

<p>夜明け前の街は、静寂に包まれていた。石畳の路地には、朝露が薄く輝いている。カイトたちは、街の正門前に立っていた。</p>

<p>見知った街だ。かつて、カイトがこの世界に転生して初めて訪れた場所。冒険者ギルドがあり、マグナスがいて、彼の旅の始まりの地。</p>

<blockquote>「懐かしいな」</blockquote>

<p>カイトが、小さく呟いた。セシリアが、彼の袖を引く。</p>

<blockquote>「カイト、ここが最初の街？」</blockquote>

<blockquote>「ああ。俺がこの世界で、最初に友達を作った場所だ」</blockquote>

<p>エリナが、懐かしむような目で街を見つめた。</p>

<blockquote>「私も覚えてるわ。あなたがギルドに現れた時のこと」</blockquote>

<p>リアムが、鼻を鳴らした。</p>

<blockquote>「俺はまだいなかったがな。噂は聞いてたぜ。『変な魔法を使う謎の奴』って」</blockquote>

<p>カイトは苦笑した。確かに、彼の魔法はこの世界では異質だった。コードのように魔力を組み立てる。詠唱もなく、魔方陣もない。ただ、純粋な論理構造だけで現象を起こす。</p>

<blockquote>「さて、感傷に浸ってる暇はない。マグナスに会おう」</blockquote>

<p>カイトたちは、街の中へと足を踏み入れた。</p>

<hr>

<p>冒険者ギルドは、まだ閉まっていた。早朝のこの時間は、冒険者たちも寝静まっている。だが、カイトは迷わず裏口へと回った。</p>

<blockquote>「マグナスは早起きだからな」</blockquote>

<p>裏口の扉を、軽くノックする。しばらくして、扉が開いた。</p>

<p>現れたのは、恰幅の良い男だった。白髪混じりの髭。鋭い目つき。だが、その奥には温かみがある。この街の冒険者ギルドを束ねるマスター、マグナスだ。</p>

<blockquote>「おお、カイトか！生きてたか！」</blockquote>

<p>マグナスは、カイトの肩を両手で掴んだ。その力強さに、カイトは少し苦笑する。</p>

<blockquote>「なんとかね。少し複雑なことになってるんだけど、話せるか？」</blockquote>

<p>マグナスは、カイトの顔をじっと見た。そして、彼の後ろにいる仲間たちにも視線を向けた。エリナ、リアム、セシリア、そして見慣れない男——マルクス。</p>

<blockquote>「……わかった。中に入れ」</blockquote>

<hr>

<p>ギルドの奥にある個室で、カイトたちは席に着いた。マグナスが、温かい飲み物を用意してくれる。</p>

<blockquote>「それで、何があった。お前の顔を見てわかる。ただの挨拶回りじゃないな」</blockquote>

<p>カイトは、深呼吸をした。そして、語り始めた。</p>

<p>古代遺跡での発見。世界の真実。創造主たちが残したシステム。そして、セキュリティの存在。</p>

<p>マグナスは、最後まで黙って聞いていた。その表情は、驚きも、否定も、恐怖も浮かべていなかった。ただ、静かに思考を巡らせているような、深い沈黙だけがあった。</p>

<blockquote>「……なるほどな」</blockquote>

<p>マグナスが、口を開いた。</p>

<blockquote>「この世界が、何者かによって作られたものだということか」</blockquote>

<blockquote>「ああ。そして、その真実を知った者を排除するセキュリティが存在する。でも、俺たちは対話に成功した。世界の住人の過半数が俺たちを信頼すれば、セキュリティは停止する」</blockquote>

<p>マグナスは、髭を撫でた。</p>

<blockquote>「過半数、か。この街だけでも数千人はいる。世界全体となれば……億を超えるかもしれん」</blockquote>

<p>カイトは頷いた。</p>

<blockquote>「わかってる。無茶な話だ。でも、やるしかないんだ」</blockquote>

<blockquote>「お前は、真実を公表するつもりか？」</blockquote>

<blockquote>「ああ。隠し続けるのはもう限界だ。世界が人工的なものだと知った上で、それでも世界を愛していると言えるなら、それが本物の絆だろ？」</blockquote>

<p>マグナスは、カイトをじっと見つめた。そして、ゆっくりと頷いた。</p>

<blockquote>「……お前らしいな」</blockquote>

<p>マグナスの声には、微かな笑みが混じっていた。</p>

<blockquote>「わかった。この爺を助っ人に加えてくれ」</blockquote>

<blockquote>「マグナス……」</blockquote>

<blockquote>「お前がこの街に来た時、思ったことがある。『こいつは、何かを変える』とな。その予感は、どうやら正しかったらしい」</blockquote>

<p>マグナスは、立ち上がった。</p>

<blockquote>「まずは、この街の有力者たちを集めよう。商人ギルドの長、騎士団の代表、学者たち。彼らに話せば、波紋が広がるはずだ」</blockquote>

<p>カイトは、マグナスに頭を下げた。</p>

<blockquote>「頼む」</blockquote>

<hr>

<p>三日後。</p>

<p>街の中央広場に、巨大な演台が組まれていた。その周りには、数千人の住民が集まっている。噂が噂を呼び、近隣の村からも人が流れてきていた。</p>

<p>カイトは、演台の上に立っていた。心臓が早鐘を打っている。いくつもの視線が、彼に突き刺さる。</p>

<p>隣には、エリナ、リアム、セシリア、マルクス、そしてマグナスが並んでいる。彼らの存在が、カイトに力を与えてくれた。</p>

<blockquote>「みんな、聞いてくれ」</blockquote>

<p>カイトの声が、静寂に響いた。拡声魔法——いや、彼が書いたコードによる音声増幅——が、その声を広場の隅々まで届ける。</p>

<pre><code>
class VoiceAmplifier {
    constructor(speaker) {
        this.target = speaker;
        this.range = "CITY_WIDE";
    }

    project(voice) {
        let waves = new SoundWave(voice);
        waves.amplify(50); // 50x volume
        waves.broadcast(this.range);
    }

<blockquote>「俺は、この世界の真実を知った。そして、それをみんなにも伝えたいと思っている」</blockquote>

<p>広場が、ざわめいた。カイトは、そのざわめきを静かに受け止めた。</p>

<blockquote>「この世界は、何百年も前に、偉大な魔術師たちによって作られたものだ。彼らは、この世界を『創造』した。そして、その真実を守るためのシステムも残した」</blockquote>

<p>さらに、ざわめきが大きくなる。カイトは、言葉を続けた。</p>

<blockquote>「でも、俺は思う。真実を知ったからといって、この世界の価値が下がるわけじゃない」</blockquote>

<p>カイトは、広場を見渡した。老若男女、様々な人々が、彼を見上げている。</p>

<blockquote>「俺たちは、ここで生きている。笑って、泣いて、愛して、戦って。その一つ一つが、本物だ。世界が人工的なものだとしても、俺たちの人生は偽物じゃない」</blockquote>

<p>カイトの言葉に、一部の人々が頷いているのが見えた。だが、まだ懐疑的な目も多い。</p>

<blockquote>「俺は、この世界を守りたい。真実を知ってもなお、ここを愛していると言える、強い世界にしたい」</blockquote>

<p>カイトは、右手をかざした。その掌に、青白い光が集まる。</p>

<blockquote>「この光は、俺が書いたパッチだ。世界の基盤コードに、小さな修正を加えたもの。真実を知っても世界が崩壊しないようにする、安全装置だ」</blockquote>

<p>光が、ゆっくりと空中に広がっていく。それは、カイトの想いを可視化したものだった。世界を守りたいという、純粋な意志。</p>

<blockquote>「だから、みんなに頼みたい。俺を信じてほしい。俺たちを信じてほしい。この世界を、一緒に守ってほしいんだ」</blockquote>

<p>カイトが、頭を下げた。</p>

<p>広場は、しばらくの間、沈黙に包まれていた。</p>

<hr>

<p>最初に動いたのは、一人の少女だった。</p>

<p>カイトがこの街で初めて出会った、花売りの少女。名前は、リナ。</p>

<blockquote>「カイトさん！」</blockquote>

<p>リナが、人混みをかき分けて前に出てきた。彼女の手には、一輪の花が握られている。</p>

<blockquote>「これ、あげる」</blockquote>

<p>リナは、カイトに花を差し出した。カイトは、それを受け取る。</p>

<blockquote>「リナ……」</blockquote>

<blockquote>「私、わかんない。世界が作られたとか、そういうの。でも、カイトさんが言ってることは、わかる」</blockquote>

<p>リナは、真っ直ぐにカイトを見つめた。</p>

<blockquote>「カイトさんは、この街で悪い人を倒してくれた。私のお母さんを助けてくれた。だから、信じる。カイトさんは、嘘をつかない」</blockquote>

<p>カイトの胸が、熱くなった。彼は、リナの頭を優しく撫でた。</p>

<blockquote>「ありがとう、リナ」</blockquote>

<p>リナの行動が、きっかけだった。一人、また一人と、人々が頷き始めた。</p>

<blockquote>「俺もだ。カイトの兄貴には、昔助けてもらったことがある」</blockquote>

<p>と、一人の若い冒険者が声を上げた。</p>

<blockquote>「私は、学者として見解を述べさせていただきます」</blockquote>

<p>と、一人の老婦人が進み出た。</p>

<blockquote>「世界の構造について、長年研究してまいりました。カイト殿の言葉は、私の仮説と一致する部分があります。そして、彼が世界を守ろうとしているのは、本物でしょう」</blockquote>

<p>ざわめきが、次第に歓声へと変わっていく。</p>

<p>だが、すべての人が賛同したわけではなかった。</p>

<blockquote>「待て！」</blockquote>

<p>鋭い声が、広場に響いた。人混みをかき分けて、一人の男が現れた。騎士団の制服を着た、厳つい顔の男。</p>

<blockquote>「カイト、お前はかつて、この街で騒ぎを起こしたことがある。それを覚えているか？」</blockquote>

<p>カイトは、男を見た。騎士団の副団長、ガンツ。かつて、カイトがこの街で問題を起こした際、彼を拘束しようとした男だ。</p>

<blockquote>「覚えてる。俺が無鉄砲だった頃の話だな」</blockquote>

<blockquote>「そのお前が、今になって世界の救世主気取りか？笑わせるな」</blockquote>

<p>ガンツは、カイトを睨みつけた。</p>

<blockquote>「お前の言葉は、確かに魅力的だ。だが、裏があるかもしれない。世界を守ると言いつつ、支配しようとしているかもしれない」</blockquote>

<p>広場の空気が、張り詰めた。カイトは、ガンツの視線を真っ直ぐに受け止めた。</p>

<blockquote>「その疑問は、もっともだ」</blockquote>

<p>カイトは、静かに言った。</p>

<blockquote>「俺は、みんなに盲目的に信じてほしいわけじゃない。疑う権利は、誰にでもある」</blockquote>

<p>カイトは、自分の胸に手を当てた。</p>

<blockquote>「でも、これだけは言わせてくれ。俺の目的は、この世界を守ることだけだ。支配なんて、興味ない」</blockquote>

<p>カイトは、ガンツに向かって、右手を差し出した。</p>

<blockquote>「ガンツ。お前が俺を監視したいなら、そうすればいい。俺の行動を、いつでもチェックしてくれ。それで、お前が納得できるなら」</blockquote>

<p>ガンツは、カイトの手を見つめた。そして、しばらくの沈黙の後——</p>

<blockquote>「……ふん」</blockquote>

<p>ガンツは、鼻を鳴らした。だが、その目から敵意は薄れていた。</p>

<blockquote>「いいだろう。お前の行動に、疑念が生じるまでは、静観する」</blockquote>

<p>ガンツは、カイトの手を握らなかった。だが、去り際に、小さな声で言った。</p>

<blockquote>「だが、もし裏切ったら……その時は、俺がお前を斬る」</blockquote>

<p>カイトは、小さく頷いた。</p>

<blockquote>「わかった」</blockquote>

<hr>

<p>演説は、夕暮れまで続いた。</p>

<p>カイトの言葉に賛同する者、疑問を投げかける者、様々だった。だが、最終的に、広場にいた過半数の人々が、「カイトを信頼する」という意思表示をした。</p>

<p>カイトは、その結果を見て、安堵の息を吐いた。</p>

<pre><code>
// Consensus Progress
let cityApproval = {
    approved: 2847,
    total: 5234,
    percentage: 54.4
};

consensus.update("FirstCity", cityApproval);
</code></pre>

<p>街の過半数。それは、世界全体からすれば、ほんの小さな一歩に過ぎない。だが、確かな一歩だった。</p>

<blockquote>「カイト、よくやったわ」</blockquote>

<p>エリナが、カイトの横に立った。</p>

<blockquote>「最初の壁は超えたわね」</blockquote>

<blockquote>「ああ。でも、これからが本番だろ」</blockquote>

<p>カイトは、夕焼けに染まる街を見渡した。遠くに見える山々、その向こうに広がる広大な世界。まだ会ったことのない無数の人々。</p>

<blockquote>「世界の過半数……億を超える人々の信頼を得るのか。気が遠くなるな」</blockquote>

<p>リアムが、カイトの肩を叩いた。</p>

<blockquote>「細かいことは気にするな。一歩ずつ進めばいい」</blockquote>

<p>セシリアが、カイトの腕に抱きついた。</p>

<blockquote>「カイトならできるよ！私がついてるから！」</blockquote>

<p>マルクスが、静かに言った。</p>

<blockquote>「わたしのネットワークも使ってくれ。収束者としての任務で、世界中の主要都市に知り合いがいる。彼らに話せば、効率よく情報を広められるはずだ」</blockquote>

<p>カイトは、仲間たちを見回した。彼らの言葉が、彼の心に染み入る。</p>

<blockquote>「……みんな、ありがとう」</blockquote>

<p>カイトは、深く息を吸い込んだ。そして、前を向いた。</p>

<blockquote>「次は、隣国だ。この勢いで、信頼の輪を広げていくぞ」</blockquote>

<p>夕陽が、カイトたちの影を長く伸ばしていた。</p>

<p>それは、これから始まる長い旅の、静かな始まりだった。</p>

<hr>

<p>その夜、カイトは一人で街の外壁に座っていた。頭上には、二つの月が輝いている。</p>

<p>隣に、誰かが座った。エリナだった。</p>

<blockquote>「眠れないの？」</blockquote>

<p>カイトは、月を見上げたまま答えた。</p>

<blockquote>「……少し、考えててさ」</blockquote>

<blockquote>「何を？」</blockquote>

<blockquote>「俺、本当にこれでいいのかなって」</blockquote>

<p>エリナは、カイトの顔を見た。その表情には、珍しく弱さが見えた。</p>

<blockquote>「真実を公表して、世界を変えようとしてる。でも、それが正解なのか、俺にはわからないんだ」</blockquote>

<p>カイトの声は、静かだった。</p>

<blockquote>「もしかしたら、真実を隠したままの方が、世界は平穏だったかもしれない。俺が勝手に動いて、余計なことをしただけかもしれない」</blockquote>

<p>エリナは、しばらく黙っていた。そして、静かに口を開いた。</p>

<blockquote>「ねえ、カイト」</blockquote>

<blockquote>「ん？」</blockquote>

<blockquote>「私ね、昔、自分の出生の秘密を知ったことがあるの」</blockquote>

<p>カイトは、エリナの方を向いた。</p>

<blockquote>「私が生まれた村は、ある禁忌を犯した過去があった。私の一族は、呪われた血を引いているって」</blockquote>

<p>エリナの声は、穏やかだった。だが、その奥には、過去の苦悩が滲んでいた。</p>

<blockquote>「最初はショックだった。自分を否定したくなった。でもね、カイトに出会って、考えが変わったの」</blockquote>

<blockquote>「俺に？」</blockquote>

<blockquote>「ええ。あなたは、私の出自なんて気にしなかった。ただ、『今のエリナ』を見てくれた」</blockquote>

<p>エリナは、月を見上げた。</p>

<blockquote>「真実は、時に残酷よ。でも、それを知った上で、自分を受け入れてくれる人がいるなら——それだけで、生きていける」</blockquote>

<p>エリナは、カイトの方を向いた。その瞳は、月明かりに照らされて、綺麗に輝いていた。</p>

<blockquote>「あなたが真実を公表したことで、世界は変わるかもしれない。でも、それは『悪い変化』じゃないわ。人々が、自分たちの世界をより深く知り、より深く愛するきっかけになる」</blockquote>

<p>カイトは、エリナの言葉を噛み締めた。</p>

<blockquote>「……エリナ」</blockquote>

<blockquote>「何？」</blockquote>

<blockquote>「ありがとう。俺、ちょっと救われたかも」</blockquote>

<p>エリナは、ふふっと笑った。</p>

<blockquote>「どういたしまして。でも、まだ旅は続くわよ。気を引き締めて」</blockquote>

<blockquote>「ああ、わかってる」</blockquote>

<p>二人の間に、穏やかな沈黙が流れた。</p>

<p>二つの月の下、世界を変える旅の夜は、静かに更けていった。</p>
