第463章：光の塔への道

山を下りる道中、カイトはずっと考え込んでいた。

テラドンから聞かされた言葉が、頭の中で反響し続けていたからだ。管理者の末裔。虚空の種の意志。そして、世界を覆う地脈の汚染。

「カイト、大丈夫？」

エリナが、心配そうに声をかけてきた。

「ずっと黙り込んでるから。何か気になることがあるの？」

カイトは足を止め、仲間たちの方を振り返った。夕日が山々を赤く染め、長い影を落としている。

「うん、考えてたんだ。虚空の種のこと」

カイトは、ビジョンで見た光景を思い出した。北東の領域に広がる、黒い影。光が届かない、暗黒の世界。

「あれには意志があるって、テラドンさんが言ってた。ただの現象や災害じゃなくて、明確な目的を持って動いているんだ」
「目的、か……。何が目的なんだろう？」

リアムが首を傾げた。

「この世界を乗っ取るとか？ 滅ぼすとか？」
「もっと深い理由がある気がする。管理者の末裔である僕を、わざわざ標的にしてきたのも、何か意味があるはずだ」

カイトの言葉に、セシリアが反応した。

「管理者を排除することで、この世界の防衛システムを無効化しようとしているのかもしれないわね」
「それは考えられる。でも、もっと根本的な何かがある気がするんだ。ルミナスさんに会えば、何か分かるかもしれない」

カイトは再び歩き出した。山道は次第に緩やかになり、やがて広い平原に出た。

◆ ◆ ◆

平原の向こうに、遠くの地平線が見えた。その先には、西の海。そして、海上に突き出す一本の巨大な塔の影。

「あれが、光の塔？」

エリナが指差した。夕日に照らされた塔は、黄金色に輝いている。

「遠いな。徒歩だと、三日はかかるか」

リアムが見積もった。

「街道沿いに進めば、途中で街にも寄れるし。無理はしない方がいいわ」

セシリアの提案に、カイトも頷いた。地の領域での試練で、魔力をかなり消耗している。休息は必要だった。

「じゃあ、一番近い街まで行こう。そこで宿を取って、明日また出発する」

四人は、西への道を歩き始めた。

◆ ◆ ◆

日が完全に沈む頃、小さな村に到着した。

村の名は「風見（かざみ）の里」。風の領域と地の領域の境界に位置し、古くから旅人の休憩所として栄えてきた場所だ。

「ここなら、宿もありそうだね」

リアムが安堵の息をついた。村の入り口には、小さな宿屋が一軒。その隣には、飲食店を兼ねた酒場があった。

「今日はもう疲れたわ。ご飯食べて、早めに休もう」

エリナが提案し、誰も反対しなかった。

宿屋の一階にある食堂で、四人は食事を摂った。シチューとパンという素朴なメニューだが、空腹には何よりも美味かった。

「ねえ、カイト」

食後の茶を飲みながら、セシリアが切り出した。

「管理者の血脈について、もう少し詳しく聞かせてくれない？」

カイトはテーブルに肘をつき、考えを整理した。

「僕も詳しくは知らないんだ。ビジョンで見たのは、アレクシス・ヴェイルっていう人物が、最初の管理者だったってことだけ」
「アレクシス・ヴェイル……深淵の書庫でその名前を見たわ」

セシリアが言った。

「この世界の基礎を築いた人物として、何冊かの本に記録があった。魔法体系の確立、地脈の管理システム、賢者制度の創設……すべてに関わってた」
「それが僕の先祖、か。全然知らなかったよ。家系図とか、見たこともないし」

カイトは苦笑した。自分の血筋に、これほどの重みがあったとは。

「でも、それで合点がいくことも多いわ」

エリナが口を挟んだ。

「カイトが、魔法をコードとして記述できるのも、血脈の力なのかもしれない。普通の魔法使いにはできないことよ」
「確かに。僕の魔法は、詠唱じゃない。関数を書く感覚に近いんだ」

カイトは、自分の魔法の使い方を振り返った。


function castSpell() {
    const intent = defineIntent();
    const mana = gatherMana(intent);
    const structure = encodePattern(mana);
    const result = executeStructure(structure);
    return result;
}

これは、カイトが無意識に行っている魔法のプロセスを、後から言語化したものだ。意図を定義し、魔力を集め、パターンを構築し、実行する。それを、プログラミングのような構文で記述できる。

「もしかして、アレクシス・ヴェイルも同じだったのかもしれない。魔法を、数式やコードとして理解できる能力」

カイトの仮説に、セシリアが目を輝かせた。

「それなら、深淵の書庫にある古文書にも、同じような記述があるはずよ。今度調べてみるわ」
「頼む。何か分かったら、教えてほしい」

四人は、それぞれの部屋に戻った。カイトはベッドに横になり、天井を見つめた。

管理者の末裔。その言葉が、重くのしかかる。自分は、この世界を救う責任がある。でも、それは一人で背負うべき重荷ではないはずだ。

「みんながいる。一人じゃない」

カイトはつぶやき、目を閉じた。深い眠りが、すぐに訪れた。

◆ ◆ ◆

翌朝、カイトは奇妙な夢を見て目を覚ました。

夢の中で、誰かが呼んでいた。「西へ。光が待っている」と。その声は、女性のものだった。優しく、でも力強い。

「ルミナスさん、かな」

カイトは寝起きの頭で考えた。光の賢者が、夢を通じてコンタクトを取ってきたのかもしれない。

窓の外を見ると、夜明け前の空が白み始めていた。カイトは静かに部屋を出て、宿屋の裏庭に行った。朝の空気は冷たく、肺を洗い清めるようだ。


function morningTraining() {
    const breath = regulateBreath();
    const mana = circulateMana(breath);
    const focus = clearMind();
    return harmonize(mana, focus);
}

カイトは、魔力の循環を確認するための訓練を始めた。地の領域での試練で、体の中に残った地脈のエネルギーが、まだ完全に馴染んでいない。

十分ほど訓練を続けると、体の中のエネルギーが整った。琥珀色の光が、体内を巡っている感覚。

「大地の実、か」

カイトは懐から、テラドンから貰った果実を取り出した。夜通し、肌身離さず持っていた。温かい光を放ち、まるで生きているかのように脈動している。

「いざというときに使え、って言ってたけど。どういう場面なんだろう」

答えは出ない。でも、持っているだけで心強い。

「カイト？ 早起きね」

背後から声がして、カイトは振り返った。エリナが立っていた。寝起きの髪を後ろで縛り、すっきりとした表情だ。

「エリナこそ。どうしたの？」
「目が覚めちゃって。カイトの気配を感じたから、来てみたの」

エリナはカイトの隣に並んだ。二人で、夜明けの空を見上げる。

「ねえ、カイト。管理者の末裔だって聞いて、どう思った？」

唐突な質問だった。でも、カイトは迷わず答えた。

「正直、戸惑ってるよ。急にそんなこと言われても、実感がないんだ」
「そうよね。でも、私は思うの。血筋がどうあれ、カイトはカイトだって」

エリナはカイトの方を向いた。朝の光が、彼女の瞳を照らす。

「あなたが管理者の末裔だからついてるんじゃない。カイトだから、私は一緒に戦いたいと思ってるの」

カイトは、胸が温かくなるのを感じた。

「……ありがとう、エリナ。嬉しいよ」
「あ、でも勘違いしないでね。戦いやすいから一緒にいるだけなんだから」

エリナは照れ隠しのように言い、ぷいと顔を背けた。でも、耳が赤くなっている。

「はいはい、分かったよ」

カイトは苦笑した。こういうところは、変わらないなと思う。

◆ ◆ ◆

朝食を済ませた一行は、村を出発した。

西への街道は、整備されており歩きやすかった。両脇には畑が広がり、農夫たちが朝の作業に励んでいる。

「平和な光景だね」

リアムが言った。

「虚空の種の脅威なんて、ここでは実感できないよ」
「それが普通の暮らしってやつよ。私たちが守らなきゃいけないもの」

セシリアが真面目な顔で言った。

昼頃、街道沿いの街「白壁（しらかべ）の町」に到着した。白い壁の家々が並ぶ、美しい街だ。光の領域に近いため、建築様式にも光を取り入れた工夫が見られた。

「ここで昼食にしよう。それと、光の塔について情報を集めたい」

カイトの提案で、一行は街の中央にある広場へ向かった。

広場には、露店が並んでいた。食材、衣類、魔法道具。様々な商品が売られている。

「あそこで聞いてみるわ」

エリナが、地図を売る露店を指差した。店主は、初老の男性だった。

「すみません、光の塔へ行きたいんですけど、一番いいルートを教えていただけますか？」

店主は、エリナを見て目を細めた。

「光の塔へ？ 旅人さんかい？」
「はい。賢者に会いに行きたいんです」
「賢者に、となると……相当な覚悟が必要だよ」

店主は声を潜めた。

「光の塔は、普通の人間には近づけない。強力な結界が張られてるからね」
「結界？」
「ああ。賢者ルミナス様が、外部の干渉を防ぐために設けたものだ。選ばれた者しか、中に入れない」

カイトは眉をひそめた。選ばれた者。自分たちは、その資格があるのだろうか。

「どうすれば、結界を通れるんですか？」
「さあね。私には分からんよ。でも、港町のアルバートに行けば、何か分かるかもしれない。そこから船で塔に近づけるからね」

店主は、西の海岸にある港町を教えてくれた。そこから船で光の塔へ向かうのが、一般的なルートらしい。

「ありがとうございます。助かりました」

カイトたちは、店主にお礼を言い、広場を後にした。

「港町アルバート、か。そこまであと二日くらいかな」

リアムが地図を見ながら言った。

「急ごう。光の賢者が待ってるはずだ」

カイトの言葉に、全員が頷いた。

◆ ◆ ◆

二日後、一行は港町アルバートに到着した。

白い砂浜と、青い海。そして、その海上にそびえ立つ光の塔。間近で見ると、その威容は圧倒的だった。高さは数百メートルにも及び、頂上は雲の中に隠れている。

「あれが、光の塔……」

セシリアが、息を呑んだ。

「すごいわね。どうやって建てたのかしら」
「魔法でしょうね。物理的な建築技術じゃ、無理だと思う」

カイトが答えた。

港には、数隻の船が停泊していた。漁船や商船が主だが、中には観光客向けの遊覧船も見える。

「あの船で、塔の近くまで行けるはずだよ」

リアムが指差した。船着き場に、「光の塔遊覧船」と書かれた看板が出ている。

「でも、結界があるんでしょ？ 近づいても、中には入れないんじゃ？」

エリナが疑問を呈した。

「とりあえず、行ってみよう。何か分かるかもしれない」

カイトたちは、遊覧船のチケット売り場へ向かった。

窓口には、若い女性が座っていた。明るい笑顔で、客を迎えていた。

「いらっしゃいませ。光の塔遊覧船へようこそ」
「すみません、塔の近くまで行きたいんですけど」
「はい、もちろん。塔の周りを一周するコースになってます。所要時間は約二時間です」

女性は手際よく説明した。

「塔の中に入ることはできますか？」

カイトが core な質問をした。女性は少し困った顔をした。

「申し訳ありませんが、塔の中は立ち入り禁止なんです。賢者様の指示で、結界が張られています」
「結界を通る方法は、ありませんか？」

女性は、カイトをじっと見た。そして、何かに気づいたように目を大きく開けた。

「あなた……もしかして、管理者様？」

カイトは驚いた。

「え、どうしてそれを？」
「あなたの周りに、特別なオーラが見えるんです。賢者様と同じ、管理者の証」

女性は声を潜めた。

「実は、賢者様から言付かっているんです。管理者の血を引く方が来たら、この道を教えてほしいと」

女性は、カイトに一枚の紙片を渡した。そこには、複雑な図形が描かれている。

「これは、結界を通るための鍵です。塔の北側に、小さな入り口があります。そこにこの図形を浮かべて、魔力を流してください」
「ありがとうございます。でも、なぜ僕が管理者だと？」
「ルミナス様は、すべてお見通しです。あなたが来ることも、今日この時に来ることも」

女性は謎めいた微笑みを浮かべた。

「光の賢者は、予知の力を持っています。安心してください。あなたは、選ばれた人ですから」

カイトは紙片を大事に懐にしまった。

「ありがとう。すぐに行くよ」

カイトたちは、遊覧船に乗り込んだ。他の観光客も数名いたが、カイトたちの目的には気づいていない様子だ。

船はゆっくりと岸を離れ、光の塔へと向かって進んでいった。

◆ ◆ ◆

船上から見る光の塔は、さらに神秘性を増していた。

塔の表面は、滑らかな白い石材で覆われている。ところどころに、透明な結晶が埋め込まれ、日光を反射して輝いている。

「本当に、光の塔だな。眩しいくらいだ」

リアムが目を細めた。

「カイト、さっきの女性の言ってたこと。どう思う？」

エリナが聞いた。

「ルミナスさんが、僕が来ることを予知してたってことだよね」
「そう。それって、すべて計画されてたってことじゃない？」

カイトは考え込んだ。確かに、偶然にしては出来すぎている。テラドンが光の塔を勧めたのも、港町で女性が待っていたのも。

「もしかして、賢者同士で連携してるのかもしれない。僕を導くために」
「だとしたら、ルミナスさんは何を知ってるんだろう？」

セシリアがつぶやいた。

「虚空の種の正体とか、管理者の使命とか。核心に近い情報を持ってるはずよ」

船は塔の周りを一周し始めた。北側に来たとき、カイトは小さな入り口を見つけた。高さは大人一人が通れる程度。女性の言っていた場所だ。

「あそこだ。船を停めてもらおう」

カイトは船長に声をかけた。他の観光客は不思議そうな顔をしたが、カイトたちは気にせず船を降りた。

◆ ◆ ◆

小さな入り口の前で、カイトは紙片を取り出した。

図形を空中に浮かべ、魔力を流す。


function activateKey(pattern) {
    const structure = decodePattern(pattern);
    const mana = infuseMana(structure);
    const response = sendToBarrier(mana);
    return waitForResponse(response);

図形が光り輝き、結界と共鳴した。低い音が響き、目の前の壁が光の粒子となって消えていく。

「開いた！ 行こう！」

カイトたちは、光の塔の中へと足を踏み入れた。

中は、想像を超える空間だった。壁も床も天井も、すべてが光で構成されている。光源はどこにもないのに、視界は明るい。光そのものが物質化したような、不思議な場所。

「ここが、光の塔の内部……」

エリナが、圧倒されたように呟いた。

「ルミナスさんは、どこにいるんだろう？」

カイトの問いに答えるように、前方から声が響いた。

『よく来たな、管理者の末裔よ』

声は、女性のもの。優しく、でも威厳がある。

『私はルミナス。光の賢者にして、この世界の守護者の一人じゃ。お主を待っていた』

光の中から、一人の女性が現れた。

長い金髪に、純白のローブ。瞳は、太陽のように輝いている。年齢は不詳だが、どこか少女のような可愛らしさと、賢者としての荘厳さを兼ね備えていた。

「ルミナスさん、ですね。僕はカイト。テラドンさんに会ってきました」

ルミナスは、満足げに頷いた。

「うむ。地の試練を乗り越えたのじゃな。その証、胸に抱いておる」

彼女の視線が、カイトの懐にある大地の実に向けられた。

「さて、長い旅をここまでよく来た。まずは休息をとるがよい。それから、話そう。虚空の種について、そして……お主の運命についてな」

ルミナスは、優しく微笑んだ。その笑顔は、母親のように温かかった。

「安心せい。お主は一人ではない。私たちがついておる」

カイトは、胸の奥から安堵が湧き上がるのを感じた。

光の塔での新たな出会い。そして、いよいよ明かされる真実。

カイトの冒険は、大きな転換点を迎えようとしていた。
