第116章：大地の息吹

海を離れて十日。カイトたちは、中央大陸の山脈を越えていた。

海沿いの湿った空気から、乾いた高地の空気へ。風景は、日ごとに変化していく。緑豊かな森から、岩肌が露出した荒野へ。そして——黄金色に輝く平原が、視界に広がった。

「これが……中央大陸の穀倉地帯」

セシリアが、魔導書を見ながら言った。

「小麦、大麦、トウモロコシ——この大陸の食料の七割が、ここで生産されています」
「豊かそうな場所だな」

リアムが、目を細めて言った。

「戦乱の影響も、受けていないように見える」

カイトが、遠くを見渡した。

確かに、平原には農民たちが働いている。平和な光景だ。だが——どこか、違和感があった。

「……土の匂いが、しない」

エリナが、地面に触れながら呟いた。

「どういうこと？」
「水の魔法使いとして、自然の気配には敏感なの。この土地——何か、おかしい」

彼女は、眉をひそめた。

「表面上は豊かだけど——土の生命力が、弱まってる」

「五十年後には、不毛の地になるってことか」

彼は、決意を固めた。

「急ごう。土の遺物が、この事態に関係してるはずだ」

平原を抜けて、一行は山岳地帯へと入った。

岩が積み重なり、巨像のように見える奇岩が点在している。風が岩の隙間を抜けると、不気味な音が響いた。

「大地の聖域は、この先にある」

セシリアが、地図を指差した。

「標高三千メートルの高原に位置しています。土の精霊が守る、古代の遺跡ですわ」
「三千メートルか……息苦しくなりそうだな」

リアムが、上を見上げながら言った。

「魔法的な耐性があれば、何とかなるだろ」

カイトが、杖を握った。

「風の魔法で、空気を補給できるはずだ」

登山は、三日かかった。

標高が上がるにつれて、空気は薄くなり、気温は下がった。だが——カイトの魔法で、四人は何とか頂上近くまで到達した。

そして——目の前に、巨大な石柱群が現れた。

数十メートルの高さを持つ石柱が、円形に並んでいる。中央には、祭壇のような構造物。その奥に——洞穴が口を開けていた。

「これが、大地の聖域への入り口」

セシリアが、静かに言った。

「土の精霊の力が、ここから漏れ出ています」

カイトが、石柱に近づいた。表面には、古代の文字が刻まれている。

// Ancient Script Translation
// Source: Stone Pillar Inscriptions
// Language: Proto-Elemental
// 
// Content:
// "Earth gives life to all things.
//  All things return to the earth.
//  Those who seek power must understand:
//  Strength comes from below, not above."

カイトは、文字を解読した。

「力は下から来る、か。謎めいた言葉だな」
「入ってみれば、わかるかもしれないわ」

エリナが、洞穴を指差した。

洞穴の中は、意外なほど明るかった。

壁面には、光る苔が張り付き、青白い光を放っている。床は、滑らかな石で舗装されていた。

しばらく進むと、広い空間に出た。

天井は高く、壁画が描かれている。農耕、建設、戦争——人間と土の関わりが、時系列で描かれていた。

「歴史の記録か」

リアムが、壁画を見ながら言った。

「人間が、土をどう扱ってきたか——精霊たちが見守ってきたんだな」

その時——地面が揺れた。

小さな地震。だが、人工的なものではない。まるで——巨大な何かが、地下で動いたような。

『来たか、人間たち』

低い、地響きのような声が、空間に響いた。

カイトが、身構えた。

「誰だ！」

中央の床が、割れた。中から——巨大な岩石の巨人が、現れた。

高さは十メートル以上。全身が岩でできている。目は、琥珀色の宝石が埋め込まれている。

『我が名は、ガイア。土の精霊の長だ』

ガイアが、ゆっくりとカイトたちを見下ろした。

『海の精霊から連絡を受けた。テティスが、お前たちを認めたとな』
「……じゃあ、戦わなくていいのか？」

カイトが、少し安堵して聞いた。

『テティスは感情で動く。だが——私は違う』

ガイアの声は、感情を感じさせなかった。

『お前たちが遺物を求める理由は、理解している。世界の崩壊を防ぐため——なのだろう？』
「そうだ」
『だが——聞くことがある』

ガイアが、巨大な手を差し出した。

『お前たち人間は、土を何だと思っている？』

カイトは、考えた。

「……土は、生命の源だ。植物が育ち、食料ができる。建物を建てる基盤でもある」
『一般的な答えだな。だが——表面的だ』

ガイアが、首を振った。

『もっと深く考えろ。土とは、何か』

エリナが、口を開いた。

「……土は、万物の器。水を蓄え、植物を支え、微生物を住まわせる」
「土がないと——生命は循環しない」

ガイアが、わずかにうなずいた。

『悪くない。だが——まだ足りん』

リアムが、剣を握りしめた。

「俺は——戦士だ。戦いの中で、地面に倒れたことが何度もある」
「土は——最後の安息の地だ。死んだ者は、土に還る」

ガイアの目が、光った。

『ほお——死と土の関わりを理解しているか』

セシリアが、静かに言った。

「私は、研究者として多くの遺跡を調べてきました」
「土は——歴史の証人です。文明が栄え、滅びても——土は残る」
「土の中には、過去の全てが眠っています」

ガイアは、沈黙した。

やがて——静かに口を開いた。

『四位の答え。四位の視点——悪くない』
『だが——試練は必要だ。言葉だけで、信じるほど甘くはない』

ガイアが、手を振った。

瞬間——カイトたちの足元が崩れた。

「うわっ！」

四人は、暗闇の中へと落ちていった。

着地した場所は、地下の広場だった。

周囲は岩壁に囲まれている。天井には、空気穴らしき隙間がいくつかあるが、光はほとんど入らない。

「……閉じ込められた、ってこと？」

エリナが、周囲を見回した。

「出口が見当たらないわ」

ガイアの声が、上から響いた。

『これが試練だ。「土の迷宮」からの脱出』
『出口は一つ。見つけ出せ——ただし、時間制限がある』
「時間制限？」

カイトが、聞き返した。

『酸素の供給が限られている。現在の空気量——約三時間分だ』
『三時間以内に脱出できなければ——土の一部となってもらう』

カイトは、拳を握った。

「……やるしかない」

四人は、迷宮の探索を始めた。

通路は複雑に枝分かれしている。壁には、古代の絵が描かれている。

「何か、規則性があるはずだ」

カイトが、壁の絵を調べた。

「ガイアは『力は下から来る』って言ってた。つまり——地下へ進むのが正解かも」
「でも、地上への出口を探すんでしょ？ 逆じゃない？」

エリナが、首をかしげた。

セシリアが、魔導書を開いた。

「……精霊の試練には、よく謎かけが含まれます」
「『下から来る力』——これは、土の精霊の教えかもしれません」
「土の力は、地下深くにある。つまり——深く潜ることで、力を得る」
「その力を使って、脱出する——という流れかも」

カイトが、うなずいた。

「試してみよう。下へ行く通路を探す」

通路を進むと、階段が見つかった。下へと続いている。

四人は、慎重に降りていった。

階段の先には、広い空間があった。中央に——巨大な水晶が埋まっている。

「これ……」

リアムが、水晶に近づいた。

「何か、感じる」

彼が水晶に手を触れた瞬間——光が溢れた。

「急ごう」

いくつかの謎を解き、四人は迷宮の最深部に到達した。

そこには——一本の巨大な木があった。

地下なのに、木が育っている。根は岩盤を突き破り、枝は天井に届いている。葉は、黄金色に輝いていた。

「……世界樹？」

セシリアが、驚いた表情で言った。

「いいえ——世界樹の分身、でしょうか」
「ガイアが守ってる、ってことか」

カイトが、木に近づいた。

幹には、穴が開いている。中に——土の遺物らしき物体が見えた。

「あれが、遺物」

カイトが、手を伸ばした。

その時——木が動いた。

枝が、鞭のようにしなり、カイトを襲った。

「カイト！」

エリナが、水の盾を作った。枝を受け止める。

「この木——意思を持ってる！」

「倒すのか？」
「待て——殺す必要はない」

「じゃあ、どうするんだよ」
「——説得する」

「俺たちは、敵じゃない」

彼は、静かに言った。

「俺たちは、ここまで来た。最後まで——やり遂げる」

「光の神殿への道のりは——長いですわ。東方まで、二週間はかかるでしょう」
「二週間か……」

太陽が、沈みかけている。空は、オレンジ色に染まっていた。

「でも——急ぐ必要はない。しっかり準備してから、行こう」

「私たち、仲間だもの。無理はしないで」

「……ああ。ありがとう」

「元の世界に——戻りたい？」

「……どうして、そんなことを聞く？」
「だって——もう、長い旅だもの」
