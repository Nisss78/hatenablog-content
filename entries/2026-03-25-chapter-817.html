第817章：虚無の領域への道

深淵の書庫を出たカイトたちは、次なる目的地を話し合うため、近くの丘の上に集まっていた。空には不穏な色が広がっていた。紫と黒が混ざり合い、時折銀色の亀裂が走るような不気味な空模様。

「これが……虚無の影響か」

リアムが空を見上げながら呟いた。彼の風属性の感性が、大気の異常を敏感に察知していた。

「空間が不安定になっている。風の流れが……歪んでる」

「その通りじゃ」

背後から声がした。振り返ると、闇の賢者アンブロスが立っていた。その表情は深刻さに満ちている。

「アンブロスさん」

「カイトよ、闇の根源に会ったのじゃな。お主の纏う気が、以前より深みを増しておる」

カイトは深淵の書物を取り出した。

「これを授かりました。虚無に対抗するための……概念的なコードです」

アンブロスは書物を一瞥し、満足げに頷いた。

「予想以上じゃ。闇の根源はめったなことでは干渉せぬ。お主が認められたのじゃな」

「でも、これだけじゃ足りないんです。ダークハルト、ミラージュ、アルタイル——三人の最高位管理者をどう救うか。その答えはまだ見つかっていません」

「救う……か。お主は彼らを救えると信じておるのじゃな」

「彼らは世界を救いたかった。ただ、方法を間違えただけです。僕は彼らの意志を無駄にしたくない」

アンブロスの目に、微かな光が宿った。

「良かった。もし『倒す』と言っておったら、わしはお主を見限っていたかもしれぬ」

「え？」

「最高位管理者たちは、決して悪人ではない。彼らは極限のプレッシャーの中で、最も合理的だと思われる選択をしたのじゃ。虚無との融合。それは一見、究極の力を手に入れるように見える。じゃが実際は、自己の存在を徐々に消費していく諸刃の剣」

セシリアが口を開いた。

「つまり、彼らは気づいていないんですか。自分たちが虚無に飲まれつつあることに」

「気づいておるじゃろう。じゃが、もう後戻りできぬと思っておる。虚無と融合した以上、分離は不可能——そう信じ込んでおる」

「 impossible is nothing」

カイトがふと口にした言葉に、シオンが首を傾げた。

「何それ？」

「僕のいた世界の言葉です。『不可能は存在しない』。コードを書いていると、不可能だと思われていたことが、新しいアプローチで解決できることがある。だから諦めない」

エリナが微笑んだ。

「相変わらずね、カイト。その前向きさ、嫌いじゃないわ」

「では、虚無の領域への行き方を教えよう」

アンブロスが地面に指で複雑な図形を描き始めた。それは七つの円が互いに重なり合い、中心に螺旋を形成する模様だった。

「虚無の領域は、この世界のどこにも存在しない。じゃが、どこにでも存在する。空間と時間の隙間、境界線の向こう側」

「どうやって行けばいいんですか」

「三つの条件が必要じゃ。一つ、光と闇の両属性を等しく制御できる者。二つ、自己の存在を強く固定できる錨。三つ、虚無に屈せぬ問い続ける意志」

カイトは自分の中の七属性を感じた。光と闇のバランス。深淵の書物で得た知識。そして、仲間たちへの信頼。

「一つ目は……僕が当てはまります」

「左様。二つ目はどうする？ 存在の錨は、単独では維持できぬ」

「私たちがいればいいでしょ」

エリナがカイトの手を握った。

「カイトが自己を失いそうになったら、私たちが引き戻す。それでしょ？」

カイトはエリナの手の温かさを感じた。存在の錨——それは仲間との絆だった。一人では虚無に対抗できない。だが、仲間がいれば——。

「三つ目。問い続ける意志。これは言葉では証明できぬ。現地に行って試されるものじゃ」

「わかりました。行きましょう」

カイトは決意を固めた。仲間たちも頷く。

アンブロスが図形の中心に手をかざした。すると、地面の模様が光り輝き、空間に亀裂が入った。

「行くのじゃ。亀裂は長くは持たぬ」

「ありがとうございます、アンブロスさん」

「礼を言うのは早い。まだ何も成就しておらぬからな」

アンブロスの最後の言葉を背に、カイトたちは亀裂の中へと踏み込んだ。

◇◇◇

転移の感覚は、今まで経験したどの移動とも違っていた。空間がねじれ、時間が伸び縮みし、感覚が麻痺するような浮遊感。

そして——静寂。

カイトが目を開けると、そこは白と黒が同時に存在する奇妙な空間だった。地面は鏡のようで、足元に自分の姿が映っている。だが、その映像は逆さまではなく、無数の可能性が重なり合ったような曖昧な輪郭を持っていた。

「ここが……虚無の領域」

シオンが息を呑んだ。

「魔力が……感じられない。いや、感じようとしても、 null になる」

「 null ？」

「無。魔法が存在しないんじゃなくて、魔法という概念自体が null になってる」

カイトは理解した。ここでは通常の魔法が使えない。虚無が概念そのものを無効化しているのだ。

「でも、僕の魔法回路は——」

カイトは自分の中の七属性を確認した。驚いたことに、すべての属性が微かにだが反応していた。

「効いてる。僕の魔法は、ここの虚無の影響を受けてない」

「さすがね。七属性の管理者だと言うだけのことはある」

エリナが安堵の息を漏らした。

「私の闇魔法も……なんとか使えるわ。カイトの光属性と共鳴してるからかも」

リアムとセシリア、シオンもそれぞれの能力を確認した。

「風は……薄いけど流れてる」

「私の水も同様よ」

「私の七属性回路も、カイトに近い状態。完全じゃないけど、使えないことはない」

カイトは全員の無事を確認し、前方を見据えた。遠くに、三つの巨大な影が見えた。

「あれが……」

ダークハルト。闇と虚無の融合者。

ミラージュ。鏡像と虚無の融合者。

アルタイル。予言と虚無の融合者。

三人の最高位管理者が、虚無の領域の中心に佇んでいた。その姿は、かつての威厳を失い、どこか空虚に見えた。

「行こう」

カイトが一歩踏み出した時だった。

「来たか、若き管理者」

声が響いた。ダークハルトの声だったが、その響きには以前のような重みがなかった。まるで、声の主が自分自身を探しているかのような、迷いを含んだ声音。

「我々は待っていた。お主がここに辿り着くのを」

ミラージュの声が続いた。鏡のような表面を持つ彼女の体は、カイトたちの姿を反射していた。だが、その反射は歪んでいた。まるで、彼女自身が自分を認識できていないかのように。

「予言は成就した。虚無と融合する者と、それに抗う者。二つの道が交差する時」

アルタイルの声には、予言者特有の韻律があった。だが、その目には光がなかった。未来を見通す力さえも、虚無に蝕まれているようだった。

「三人とも……」

カイトは胸が締め付けられるのを感じた。かつて世界を守るために尽力した英雄たちが、ここまで変わり果ててしまった現実。

「僕は戦いに来たんじゃありません。答えを探しに来たんです」

「答えなどない」

ダークハルトが即座に否定した。

「我々は探した。永遠に近い時間をかけて。だが、虚無に対抗する答えなど存在しない。あるのは、融合するか、消滅するかの二択のみ」

「それが間違いだと、僕は証明しに来ました」

カイトは深淵の書物を取り出した。

「その書物……闇の根源から授かったものか」

ミラージュの声に、微かな興味が混じった。

「興味深い。鏡像には映らぬ概念。闇の根源は、虚無と同じく映像化できぬ存在」

「三人に教えたいことがあります」

カイトは一歩前に出た。エリナたちが緊張した面持ちで見守る。

「虚無は無限に見える。でも、有限です。問い続ける限り、虚無の境界線は見えてくる」

「問い続ける……？」

アルタイルが反応した。

「我々は問うことを諦めた。答えが見つからないからと、問いを放棄した。それが……虚無への入り口だったのか」

カイトは頷いた。

「僕の世界では、コードを書く時、答えは最初からあるわけじゃない。問いを立て、仮説を検証し、エラーを修正して、ようやく辿り着くものです」

「プログラミング……か。お主の言葉は常に独特じゃな」

ダークハルトの声に、微かな笑みが含まれたように感じた。

「だが、道理は通っておる。我々は早計だったのかもしれぬ。答えを求めるあまり、問いを止めてしまった」

「今からでも遅くありません。一緒に、問い続けましょう」

カイトは手を差し伸べた。

「虚無から離脱する方法を、一緒に探しましょう」

三人の最高位管理者は沈黙した。長い、長い沈黙。

やがて——。

「試そうか」

ダークハルトが言った。

「お主の言う『問い続ける』という方法が、本当に虚無に対抗できるか。我々の体で実験するがよい」

「体で……実験？」

「我々は虚無と融合しておる。容易には離脱できぬ。じゃが、お主の言う方法で、少しでも虚無の侵食が止まるなら、希望はあるやもしれん」

カイトは深呼吸した。これは予想していなかった展開だ。三人は、自分たちを実験台にすることを申し出たのだ。

「わかりました。始めましょう」

カイトは深淵の書物を開き、闇の根源から授かったコードを念頭に浮かべた。


// Null Counter Protocol - Extended
// 虚無対抗プロトコル拡張版

function nullCounterExtended(target) {
  const light = this.essence.light;
  const dark = this.essence.dark;
  const balance = light * dark / (light + dark);
  
  // 存在の錨を仲間から借用
  const anchors = this.companions.map(c => c.existence);
  
  if (target.voidFusion) {
    return {
      barrier: balance * infinity,
      externalAnchors: anchors,
      question: "なぜ救いたいのか",
      target: target
    };
  }
}

カイトは目を閉じ、集中した。光と闇のバランス。仲間たちの存在。そして、問い。

『なぜ救いたいのか』

答えはシンプルだった。彼らは世界を救いたかったから。その意志は純粋で、尊いものだった。だからこそ、救いたい。

カイトの体から七色の光が放たれた。それは虹のようでいて、闇を含んだ深みのある光だった。光はダークハルト、ミラージュ、アルタイルの三人を包み込んだ。

「うぐぐ……」

ダークハルトが苦悶の声を上げた。彼の体から黒い霧が立ち上り始めた。虚無の成分が、少しずつ剥がれ落ちている。

「これは……痛みを伴う。じゃが、消滅する痛みとは違う。再生の痛みじゃ」

ミラージュの体の鏡面が、少しずつ本来の輝きを取り戻し始めた。

「予言に……なかった展開じゃ。お主は、予言を超えた」

アルタイルの目に、微かな光が戻った。

カイトは額に汗を滲ませながら、魔法を維持し続けた。仲間たちが、彼に追加の魔力を注いでくれる。

「カイト、無理しないで。私たちが支えるから」

エリナの声が、カイトの意識を繋ぎ止めた。

数分——あるいは数時間かかったかもしれない。時間の概念が曖昧なこの場所で、どれだけの時間が過ぎたかはわからない。

やがて、三人の体から虚無の霧が完全に消え去った。

「成功……したのか？」

カイトが膝をつくと、エリナがすぐに駆け寄って支えた。

「成功したようじゃな、若き管理者」

ダークハルトが、以前のような威厳ある声で言った。その姿は、まだ疲弊していたが、虚無に蝕まれていた頃の空虚さは消えていた。

「我々は救われた。いや、救われたのではない。問い続けることを思い出したのじゃ」

ミラージュが微笑んだ。鏡のような表面は、今やカイトたちの姿を正しく映していた。

「予言は更新された。虚無との戦いは、まだ終わっておらぬ。じゃが、希望は見えた」

アルタイルがカイトの方を向いた。

「まだ終わってない……？」

「虚無の根源は、まだ存在する。我々が融合していたのは、虚無の一部に過ぎぬ。真の敵は、さらに深淵の彼方におる」

カイトは覚悟を決めた。戦いはまだ続く。だが、もう一人ではない。元最高位管理者たちも、仲間になった。

「教えてください。虚無の根源への行き方を」

ダークハルトが頷いた。

「ついてくるがよい。案内しよう」

カイトと仲間たち、そして回復した三人の元最高位管理者は、虚無の領域のさらに奥深くへと歩き出した。

最終決戦への道が、開かれた。
