第486章：灼熱の砂漠

朝が来た。

カイトは、焚き火の跡を砂で埋めながら、今日の行程を頭の中で整理していた。

「ミーナ——砂漠までどのくらいかかる？」

カイトが、尋ねた。

ミーナは、森の出口の方を指差した。

「森を抜けて——半日ってところかな。でも——砂漠は広い。炎の聖域まで——さらに二日はかかる」
「二日——」

リアムが、ため息をついた。

「砂漠で二日——水は大丈夫なのか？」
「心配ない」

セシリアが、自信を持って言った。

「水魔法で——補給できます。地下水脈があれば——」
「あるよ」

ミーナが、頷いた。

「この大陸——魔力が豊富だから。地下にも水脈がある」

カイトは、仲間たちを見た。

「じゃあ——行こう。炎の聖域へ」

◆ ◆ ◆

森を抜けた瞬間——強烈な熱気が襲ってきた。

目の前に広がるのは——黄金色の砂漠。

地平線まで続く砂の海——太陽が照りつけ、蜃気楼が揺らめいている。

「暑い——」

エリナが、額の汗を拭った。

「アルヴァレス大陸——こんなに暑くなかったのに」
「リンド大陸——魔力が濃いから、気候も極端なんだ」

ミーナが、説明した。

「砂漠は——年中こんな感じ。逆に——北の氷原は、年中凍りついている」
「過酷な大陸だな——」

リアムが、呟いた。

カイトは、管理者としての感覚を働かせた。

// 環境分析
class EnvironmentScanner {
    constructor() {
        this.temperature = 42; // 摂氏42度
        this.humidity = 15; // 湿度15%
        this.magicDensity = 3.2; // 魔力密度（アルヴァレス基準の3.2倍）
    }

    analyze() {
        return {
            heatWarning: this.temperature > 35,
            dehydrationRisk: this.humidity < 20,
            magicInterference: this.magicDensity > 2.0
        };
    }
}

const scanner = new EnvironmentScanner();
const result = scanner.analyze();

カイトは、結果を見た。

「熱中症のリスクがある——こまめに休憩しよう」

◆ ◆ ◆

一行は、砂漠を進んだ。

砂に足を取られ、歩くたびに沈む。一歩進むのが、普段の三倍の労力。

「ミーナ——この砂漠には、何か危険な生き物はいる？」

セシリアが、尋ねた。

ミーナは、少し考えた。

「砂漠狼——あと、砂蛇。でも——あまり遭遇しないよ。彼ら——昼は暑すぎて、地下に潜ってるから」
「じゃあ——夜が危険ってことか」

カイトが、確認した。

「うん。夜は——彼らが活動する。それに——」

ミーナの声が、小さくなった。

「砂嵐——夜になると吹くことがある」
「砂嵐——」

エリナが、不安そうな顔をした。

「大丈夫——カイトが風の加護を持ってるから」

ミーナが、励ました。

「風の魔法で——砂嵐を防げるはず」

カイトは、風の加護を確認した。

体の中に——風が満ちている。いつでも呼び出せる感覚。

◆ ◆ ◆

正午——太陽が真上に来た。

カイトは、日陰を見つけた。巨大な岩が、砂から突き出ている。

「ここで——休憩しよう」

一行は、岩の陰に座り込んだ。

セシリアが、水魔法を使った。

// 水生成魔法
function generateWater(target, amount) {
    const groundwater = detectGroundwater(50); // 50m以内の地下水脈を探知
    if (groundwater) {
        const water = extractWater(groundwater, amount);
        purify(water);
        return water;
    }
    return null;
}

const water = generateWater(canteen, 1000); // 1リットル生成

セシリアの手から、清涼な水が湧き出た。

「すごい——水魔法って便利なのね」

ミーナが、感心したように言った。

「地下水脈がない場所では——使えないけど」

セシリアは、謙遜した。

カイトは、水筒を受け取り、一口飲んだ。

冷たい水が——喉を潤していく。

「カイト——聞いてもいい？」

ミーナが、遠慮がちに尋ねた。

「何だ？」
「あなた——元の世界で、コードを書いていたって言ってたよね」
「ああ——プログラマーだった」
「プログラマー——何をする人のこと？」

カイトは、説明を考えた。

「……言語を使って——命令を書く仕事だ」
「命令？」
「ああ。例えば——『足を前に出せ』『物を持ち上げろ』——そういうのを、厳密な言葉で書く」

ミーナは、首を傾げた。

「魔法と——同じ？」
「似ているかもしれない。俺は——この世界に来て、魔法がコードで書けることを知った」

カイトは、自分の手を見た。

「詠唱じゃない——論理的な言葉で、魔法を組み立てる。それが——俺の能力だ」
「ふうん——カイトって、特別なのね」

ミーナは、尊敬の眼差しでカイトを見た。

◆ ◆ ◆

休憩を終え、一行は再び歩き出した。

砂丘を越え——また砂丘を越え——

単調な風景が続く。でも——

「待って——」

カイトが、足を止めた。

「何か——来る」

風の加護が——警告を発している。

カイトは、風に耳を澄ませた。

「砂の下——何かが動いている」
「えっ？」

エリナが、周囲を見回した。

その瞬間——

砂が爆発した。

巨大な蛇が——砂の中から飛び出してきた。

長さは十メートル以上。鱗は黄金色——そして、口から炎を吐いている。

「砂蛇——しかも——炎属性！」

ミーナが、叫んだ。

「みんな——下がって！」

カイトが、前に出た。

◆ ◆ ◆

砂蛇が、カイトに向かって炎を吐いた。

カイトは——風魔法で防いだ。

// 風の防御壁
class WindBarrier {
    constructor(user) {
        this.user = user;
        this.defense = 500;
        this.range = 3; // 3メートル半径
    }

    activate() {
        const wind = generateWind(this.range);
        wind.createBarrier();
        return "Barrier active";
    }
}

const barrier = new WindBarrier(kite);
barrier.activate();

「本能で動く——猛獣」

function windBlade(target) {

windBlade(sandSerpent);

リアムが、安堵のため息をついた。

「カイト——強くなってる」

風の加護——確実に力を底上げしている。

夕方——一行は、オアシスを見つけた。

小さな池——そして、数本のヤシの木。砂漠の中の楽園。

「ここで——野宿しよう」

カイトが、提案した。

「水も確保できるし——夜の砂嵐も避けられる」

全員が、同意した。

テントを張り——焚き火を囲む。

ミーナが、カイトの隣に座った。

「カイト——今日の戦い、すごかった」
「ん——ああ、ありがとう」
「わたし——カイトみたいに戦えたらいいのに」

ミーナは、自分の手を見た。

「わたし——何の魔法も使えないから」
「……魔法は——誰でも使えるとは限らない」

カイトが、言った。

「例えば？」

ミーナの顔が——嬉しそうに輝いた。

夜が更けた。

カイトは、見張りをしながら——明日のことを考えた。

炎の聖域——第二の封印。

風の精霊ウィンディアは——優しかった。

でも——炎の精霊はどうだろう。

炎は——情熱的で、同時に破壊的。

試練の内容も——風とは違うはず。

「覚悟——決めなきゃな」

カイトは、小さく呟いた。

管理者としての責任——そして、仲間を守る覚悟。

砂漠の夜は——肌寒い。でも——焚き火が温めてくれる。

明日は——炎の聖域に到着する予定。

新たな試練——そして、新たな出会い。

リンド大陸での冒険——続いていく。

風の加護が——何かを警告している。

カイトは、立ち上がった。

遠く——砂漠の彼方から、何かが近づいてくる。

人影——いや、複数の人影。

「みんな——起きろ！」

カイトが、叫んだ。

仲間たちが、飛び起きる。

人影が——オアシスに近づいてくる。

「誰だ——お前たちは？」

カイトが、問い返した。

『誰かが狙っているかもしれない』

その「誰か」が——現れた。
