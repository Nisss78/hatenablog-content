第440章：氷の宮殿

光の粒子に導かれるように、カイトたちは氷の壁の中を進んでいた。

周囲は完全な透明な氷に囲まれている。上を見上げれば、青白い光が氷の層を透過して、まるで水中にいるような幻想的な光景が広がっていた。

「息ができる……不思議ね」

エリナが呟く。確かに、氷の中にいるはずなのに、息苦しさはまったくなかった。空気が循環しているわけではない。むしろ、魔法が彼らを認識し、自然な状態を保たせているようだった。

「この氷、普通の氷じゃないな」

カイトが氷の壁に手を触れる。冷たさは感じるが、凍えるような寒さではない。壁の中に、微細な魔法陣が無数に刻まれているのが見えた。

IceLayer.analyze() {
  魔法陣: 無数
  構造: 再帰的
  機能: 空間維持 + 生命保護
}

コードとして解析しようとすると、その複雑さに目を回しそうになった。単純な氷魔法ではない。何千年もの間、改良され続けてきた、生きたインフラのようなものだ。

「賢者の方々が作られたのでしょうか」

セシリアが問う。リアムが首を横に振った。

「いや、これはもっと古い。フローズナ様の世代より前……あるいは、最初の管理者たちの手によるものかもしれない」

最初の管理者。アレクシス・ヴェイルとその仲間たち。彼らがこの氷の構造を作ったのだろうか。

光の粒子が、ある一点で集まり始めた。そこには、氷の階段が上へと続いていた。

「上に行けってことかしら」

エリナの言葉に、カイトは頷いた。

「行こう。何かが待ってるはずだ」

階段を上ること、数十段。

不意に、視界が開けた。

そこは、巨大なドーム状の空間だった。床も壁も天井も、すべてが透明な氷でできている。だが、ただの氷の洞窟ではない。そこには、明らかに人工的な構造物が存在していた。

氷で作られた柱。氷の彫像。氷の階段。氷の……宮殿。

「すごい……」

セシリアが息を呑む。カイトも同じ気持ちだった。

宮殿は、おそらく本来の人間のサイズで作られたものではない。巨大だ。天井は見上げるほど高く、柱は太く、彫像は三メートルを超えている。巨人が住むための宮殿か、あるいは、神殿としての役割を担っていたのか。

「古代の遺跡、というよりは……」

リアムが言葉を探す。

「誰かが住んでいた場所、ですね」

カイトが補った。その通りだ。生活感がある。いや、あった。かつては。

宮殿の奥に、玉座のようなものが見えた。その周りには、氷の花が咲いている。本来なら決して咲くはずのない、氷の中でのみ生きる花。

「あそこに行ってみよう」

カイトの提案に、全員が頷いた。

宮殿の中を歩く。床は滑らかで、足音すら立てない。まるで氷そのものが、彼らの存在を受け入れているようだった。

壁には絵が刻まれていた。氷を削って描かれた壁画だ。

  • 魔法を使う人々の姿
  • 空に浮かぶ都市
  • 巨大な獣と戦う戦士
  • そして、一人の女性が玉座に座る姿
「この女性……」

エリナが壁画を指差す。玉座に座る女性は、長い髪を持ち、優雅なドレスを着ていた。そして、その周りには氷の花が咲き乱れている。

「氷の賢者、フローズナ様の先代でしょうか」

セシリアの問いに、リアムが答える。

「あるいは、もっと昔の……最初の管理者の一人かもしれません」

カイトは壁画を見つめながら、思考を巡らせた。

この場所は何だ。なぜ今、彼らを呼んだのか。フローズナの言葉——「氷が呼んでいる」——は、何を意味していたのか。

玉座の前に到着した。

近くで見ると、その精巧さに驚かされる。玉座の肘掛けには、細かい模様が刻まれている。氷の結晶の形をした装飾。そして、座面には誰かが座った痕跡が残っていた。

長い時間が経っているはずなのに、かすかに温かい気配が残っている。まるで、つい先ほどまで誰かが座っていたかのように。

「カイト、あれ」

エリナが玉座の後ろを指差した。

そこには、小さな台座があった。台座の上には、氷でできた小箱が置かれている。

カイトは慎重に近づき、箱を手に取った。軽い。冷たい。だが、氷のように硬くはない。

IceBox.scan() {
  素材: 氷結晶 + 魔力
  中身: 不明（封印あり）
  封印強度: 中程度
}

封印されている。無理に開けようとすれば、壊れるかもしれない。

「どうしますか？」

リアムが問う。カイトは少し考えた後、頷いた。

「開けてみよう。呼ばれた理由は、これかもしれない」

カイトは箱の封印を解析し始めた。複雑なパターンだが、敵対的なものではない。むしろ、正当な継承者を選ぶための試験のような。

seal.pattern.match() {
  条件1: 魔力の共鳴 → クリア
  条件2: 氷魔法の理解 → 部分クリア
  条件3: ？？？
}

三つ目の条件が読み取れない。だが、カイトは試してみることにした。

「エリナ、手を貸してくれ」
「え？ うん、いいけど」

カイトはエリナの手を取り、箱の上に重ねた。

「君の氷魔法と、俺のコードを合わせるんだ」

エリナは一瞬驚いた表情をしたが、すぐに理解したようだった。彼女は目を閉じ、氷魔法を放出する。カイトはそれをコードとして受け取り、封印のパターンに合わせて調整した。

二つの力が混ざり合う。カイトのコードと、エリナの魔法。その融合が、封印の第三の条件を満たした。

seal.unlock() {
  条件3: 協調の証 → クリア
  完全解除！
}

箱が、静かに開いた。

中に入っていたのは、一枚の氷の結晶だった。

掌に乗るほどの大きさ。六角形の完璧な形。そして、中心には光が宿っている。

「これは……」

セシリアが目を見開く。

「氷の種です」

種。カイトは以前、シルヴィアから時空の種を受け取ったことを思い出した。そして、この世界には七つの種が存在することも。

「氷の種が、ここにあったのか」

リアムが驚きを隠せない様子で呟く。

「フローズナ様も知らなかったはずです。種は、賢者がそれぞれ受け継いでいると……」
「あるいは、フローズナ様は知っていたが、言えなかったのかもしれません」

セシリアの言葉に、カイトは頷いた。

氷の種。なぜこの場所に、誰が封印したのか。そして、なぜ今、カイトたちに開かれたのか。

答えは、種自身が知っているかもしれない。

カイトは氷の種を両手で包み込んだ。冷たい。だが、心地よい冷たさだ。種から、微かな感情が伝わってくる。

待っていた。長い時間、ずっと待っていた。継承者を。協調を理解する者を。

「俺に、継承してくれるのか？」

種は答えない。だが、光が強まり、カイトの手の中で温かくなった。それは肯定のようだった。

カイトは目を閉じ、種を受け入れた。

瞬間、視界が白く染まった。

氷の記憶が流れ込んでくる。この宮殿で過ごした日々。玉座に座った女性の孤独と責任。種を守り抜く決意。そして、長い眠り。

彼女の名は、イヴェルナ。最初の管理者の一人。氷の魔法を極め、この宮殿を築いた賢者。

彼女は種を守るために、自らを氷の中に閉じ込めた。外界との接触を断ち、ただ種の安寧を願って。

だが、彼女は信じていた。いつか、種を受け取るにふさわしい者が現れることを。協調を理解し、独占ではなく共有を選ぶ者を。

『あなたが、その人なのですね』

声が聞こえた。優しく、静かな声。

『種を、お願いします。未来のために』

カイトは心の中で答えた。

『守るよ。絶対に』

光が収束し、カイトは現実に戻った。

手の中には、氷の種が静かに輝いていた。

カイトは種を大事に懐にしまった。そして、仲間たちの方を向いた。

「行こう。ここでやるべきことは、終わった」

エリナ、リアム、セシリア——三人とも、理解したようだった。カイトが種を受け取ったことを。そして、その重みを。

「フローズナ様に報告しましょう」

リアムの提案に、カイトは頷いた。

宮殿を後にする。氷の階段を下り、光の粒子に導かれて出口へ。

振り返ると、宮殿は静かに輝いていた。イヴェルナの眠りは、種が旅立った今も、穏やかなままだ。

ありがとう。あなたの願いは、俺が叶える

カイトは心の中でそう告げ、前を向いた。

氷の種を手に入れた。残る種は、あと三つ。

物語は、まだ続く。
