第484章：管理者の覚悟

嵐の中、カイトは船首に立っていた。

雨が叩きつけ、風が体を押し流そうとする。稲光が空を切り裂き、波が船を激しく揺らす。

それでも——カイトは目を閉じなかった。

「世界と——繋がる」

カイトは、深呼吸をした。

管理者としての感覚を研ぎ澄ます。世界の鼓動を感じる。風の流れ、波のうねり、雷の軌跡——すべてが情報として、カイトの中に流れ込んでくる。

そして——気づいた。

この嵐は、自然なものではない。

「——人工的に作られた嵐？」

カイトは、驚いた。

世界の裏側にある「構造」が見えた。嵐の核——それを中心に、魔力が渦巻いている。誰かが、意図的にこの嵐を作り出したのだ。

カイトは、さらに集中した。

核に近づく。そして——

    constructor() {
        this.magicSource = null;
        this.intensity = 1.0;
        this.range = 10000; // メートル
    }

    stabilize() {
        this.intensity = 0.0;
        this.range = 0;
    }

const core = detectStormCore();
core.stabilize();

カイトの手から、淡い光が放たれた。

光は雨を突き抜け、嵐の核へと向かう。そして——核に触れた瞬間、嵐が揺らいだ。

風の音が変わる。波の高さが変わる。稲光の間隔が変わる。

カイトは、さらに力を込めた。

「まだ——足りない」

嵐の核は、強力だった。管理者としての力を使っても、完全に消すことはできない。

カイトは、別の方法を思いついた。

消すのではなく——道を作る。

function createSafePath(core, ship) {

    const width = 100; // メートル
    const length = core.range;

    core.excludePath(angle, width, length);

カイトは、船の進路に小さな通路を作った。

嵐の中に——静寂のトンネルが現れた。

雨が止み、風が収まり、波が凪ぐ。まるで嵐の目の中にいるような——不思議な空間。

カイトは、振り返った。

船室から、仲間たちが見ている。ヴィクトル船長も、驚いた表情でこちらを見ていた。

「進んでください——この道は長く持ちません！」

カイトが叫んだ。

ヴィクトル船長は、すぐに反応した。

「全員、配置につけ！ この道を抜けるぞ——！」

◆ ◆ ◆

白鴎号は、静寂のトンネルを進んだ。

左右には、巨大な波の壁。前方には、稲光の幕。でも——船の周りだけは、穏やかだった。

カイトは、船首に立ち続けていた。

額から汗が流れる。体が震える。管理者としての力を使い続けるのは、負担が大きかった。

「カイト——！」

エリナの声が聞こえた。

カイトは振り返らなかった。集中を切らせば、道が消えてしまう。

「あと——少し——」

ヴィクトル船長が叫んだ。

「見えたぞ——リンド大陸だ！」

カイトは、薄目を開けた。

霧の向こうに、陸地が見えた。緑の森、白い砂浜——そして、遠くに山脈。

「あれが——リンド大陸」

カイトは、最後の力を振り絞った。

カイトの手から、風が放たれた。

船は、風に押されて加速した。静寂のトンネルを抜け——

ついに、リンド大陸の砂浜に到着した。

◆ ◆ ◆

カイトは、砂浜に倒れ込んだ。

体から力が抜けていく。視界が薄れる。

「カイト——！」

エリナが、駆け寄ってきた。

「大丈夫——ただ、疲れただけだ」

カイトは、力なく笑った。

「本当に——心配させないでよね」

エリナは、涙目でカイトを睨んだ。

リアムとセシリアも、近づいてきた。

「すごかったな——嵐の中に道を作るなんて」

リアムが、感心したように言った。

「管理者としての力——予想以上ですね」

セシリアが、メモを取りながら言った。

ヴィクトル船長も、甲板から降りてきた。

「若いの——いや、カイト殿」

船長は、カイトの前に立った。

「俺は五十年間、海で生きてきた。多くの魔法使いを見てきた。だが——嵐を操るなど、見たことも聞いたこともない」
「……ただ、やり方を知っていただけです」

カイトは、ゆっくりと立ち上がった。

「嵐は——自然現象じゃなかった。誰かが作ったものです」
「何だと？」

ヴィクトル船長が、眉をひそめた。

「人工的に作られた嵐——核があり、魔力が渦巻いていました。五十年前からずっと、この嵐が続いているなら——誰かが維持しているはずです」

カイトの言葉に、全員が沈黙した。

「つまり——リンド大陸には、強力な魔法使いがいるってこと？」

リアムが、尋ねた。

「可能性は高い——しかも、外部からの侵入を防ぐために、嵐を作ったのかもしれない」

セシリアが、推測した。

「なら——ここは危険な場所かもしれません」

エリナが、警戒した。

◆ ◆ ◆

カイトたちは、砂浜で休憩した。

ヴィクトル船長は、港町マリナへ戻る準備をしていた。

「カイト殿——戻る時は、また呼んでくれ」

船長は、カイトに手を差し出した。

「光栄です——ヴィクトル船長」

カイトは、手を握った。

「あんたは——海を変えた。俺の人生で、一番の航海だったよ」

船長は笑って、船に戻った。

白鴎号は、夕日の中を出航した。嵐の海を——カイトが作った道を通って、帰っていく。

◆ ◆ ◆

夜が来た。

カイトたちは、焚き火を囲んでいた。

「明日から——どうする？」

リアムが、尋ねた。

「まずは、情報を集めないと。リンド大陸のこと——何も知らないからな」

カイトが答えた。

「資料によれば——リンド大陸には、いくつかの集落があるそうです。でも——五十年前から、ほとんど情報がない」

セシリアが、説明した。

「なら——村を探すのが先決ね」

エリナが言った。

カイトは、星空を見上げた。

リンド大陸の空は、アルヴァレス大陸とは少し違っていた。星の配置——そして、空気が違う。

「——何かを感じる」

カイトが、呟いた。

「何を？」

エリナが、尋ねた。

「この大陸——眠っているような気がする。まるで——何かを待っているような」

カイトは、西の方角を見た。

森の向こう——山脈のあたりから、微かな光が見えた気がした。

「明日——森を抜けてみよう。何かが見つかるはずだ」

◆ ◆ ◆

深夜——カイトは目が覚めた。

何か——呼ばれた気がした。

カイトは、焚き火から離れて、森の入り口まで歩いた。

そして——見た。

森の中に、小さな光が浮かんでいた。青白く、揺らめく光。

「……導きの光？」

カイトは、以前に出会った現象を思い出した。

光は、森の中へ誘うように移動した。

カイトは——ついていった。

森の中は暗かった。でも、不思議と恐怖はなかった。

光は、開けた場所までカイトを導いた。

そこには——古い石碑があった。

石碑には、文字が刻まれている。古代語——カイトが読める言語だった。

「『原初の管理者——ここに眠る。後の者よ——世界の真実を求めるなら、七つの封印を解け』」

カイトは、息を呑んだ。

「原初の——管理者？」

カイトは、さらに読み進めた。

「『第一の封印——風の塔。第二の封印——炎の聖域。第三の封印——氷の迷宮。第四の封印——雷の谷。第五の封印——地の洞窟。第六の封印——光の祭壇。第七の封印——闇の深淵』」

七つの封印——そして、最後にこう書かれていた。

「『全ての封印を解きたる者——原初の管理者の遺産を受け継ぐべし』」

カイトは、石碑に手を触れた。

その瞬間——石碑が光り輝いた。

そして——カイトの脳に、情報が流れ込んできた。

リンド大陸の地図——七つの封印の場所——そして、原初の管理者の遺産についての断片的な情報。

カイトは、膝をついた。

「これが——次の試練」

カイトは、仲間たちの待つ場所へ戻った。

夜明け前——カイトは、決意を胸に眠りについた。

明日から——新しい冒険が始まる。

七つの封印——そして、原初の管理者の遺産。

その先に——何があるのか。

カイトは——まだ知らなかった。

◆ ◆ ◆

翌朝——カイトたちは、森の中へ入った。

目指すは——第一の封印、風の塔。

リンド大陸の冒険——始まったばかりだった。
