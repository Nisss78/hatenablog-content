第748章：海上の異変

「海神の翼」号は、穏やかな波を切り裂いて進んでいた。

出航から半日。陸地はもう見えない。どこを見渡しても、青い海と空だけが広がっている。

カイトは甲板で風を感じていた。潮の匂いが鼻をくすぐる。

「カイト、ここだったの」

エリナが近づいてきた。

「シオンが船酔いしちゃって。何かできる？」
「船酔いか……水魔法で、体のバランスを整える魔法があるから、試してみよう」

カイトはエリナと共に船室に向かった。

船室では、シオンがベッドに横になっていた。顔色が悪い。

「うぅ……気持ち悪い……」

シオンが呻いている。

「シオン、少し動かないで」

カイトはシオンの額に手を置いた。

十秒後、シオンの表情が和らいだ。

「あ……良くなったかも。ありがとう、カイト」

シオンが起き上がった。

「無理しないでね。まだ船に慣れてないから」
「うん。でも、外の空気吸いたいな」

シオンはカイトの手を借りて立ち上がった。

甲板に出ると、気持ちのいい風が吹いていた。

シオンは深呼吸をした。

「海って広いね。どこまで行っても、海ばかり」

シオンが水平線を見つめて言った。

「この世界は、海が多いからね。陸地は全体の三割くらい」

カイトが説明した。

「だから、船の技術が発達しているんだ」
「へぇ……私のいた世界とは違うな」

シオンは少し遠い目をした。

「シオンのいた世界って、どんなところ？」

エリナが尋ねた。

「え？　えっと……遠い国だよ。こことは全然違う」

シオンは曖昧に答えた。自分が異世界から来たことは、まだ秘密にしている。

「いつか、教えてね」

エリナはそれ以上聞かなかった。

その時、甲板の反対側から、リアムの声が聞こえた。

「カイト、来てくれ。船長が呼んでる」

カイトは頷いて、リアムの元へ向かった。

船尾で、ヴィクトル船長が海を睨んでいた。

「カイト、見てくれ」

船長が指差した先には、黒い影が見えた。

海面の下、深いところで、何か巨大なものが動いている。

「これは……」

カイトは目を細めた。

「一時間前から、ずっとついてきてる。海中を移動する、何か巨大な生き物だ」

ヴィクトル船長の表情は険しかった。

「海獣か？」
「かもしれない。だが、こんなに長い距離を船のペースに合わせて移動する海獣なんて、聞いたことがない」
「狙われている、ということですか」

カイトが言った。

「そう考えるのが自然だ。特に、あんたたちを乗せてから現れた」

船長はカイトを真っ直ぐに見た。

「深海神殿に行くって言ってたな。その神殿と、こいつに何か関係があるのか？」

カイトは少し考えた。

「深海神殿には、虚無の封印があります。その封印を守る何かが、僕たちを警戒しているのかもしれません」
「封印の守護者、ってことか」
「あるいは、虚無に関わる敵か。どちらにしても、無視できません」

カイトの表情が真剣になった。

カイトは船室に戻り、エリナとシオンに状況を説明した。

「海中に、何かいる。僕たちを監視しているかもしれない」
「敵？」

エリナが尋ねた。

「分からない。でも、警戒した方がいい」
「私も何かできる？」

シオンが尋ねた。

「シオンは風魔法が得意だろ？　風で、海中の動きを感知できる？」
「えっと……風は空中の魔法だから、海中は難しいかも」

シオンは首を傾げた。

「でも、水面上の風の流れなら分かる。水面が乱れてたら、海中で何か動いてるってことだよね」
「それだ！　シオン、定期的に水面を見ていてくれ。異常があったら、すぐに知らせて」
「分かった！」

シオンは力強く頷いた。

夕方になっても、黒い影はついてきていた。

距離を保ちながら、船と同じ速度で移動している。

カイトは甲板で、水魔法を使って海中を探ろうとした。


// 水中探知魔法
function underwaterDetection(depth, radius) {
  const magic = {
    element: "water",
    type: "detection",
    effect: "指定範囲内の水中の動きを感知",
    depth: depth,
    radius: radius,
    feedback: "波紋として情報を受け取る"
  };
  
  return `半径${radius}メートル・深度${depth}メートルまでの水中情報`;
}

// 実行
const detection = underwaterDetection(50, 100);
console.log(detection);

カイトは目を閉じて、水魔法を広げた。

感覚が海中に広がる。魚の群れ、海流の動き、そして――

（いた……！)

カイトは息を呑んだ。

深度三十メートル。船の右舷側に、巨大な影がいる。

全長は五十メートル以上。蛇のような長い体。頭部は龍に似ている。

（海竜か……いや、違う。魔力の質が、虚無に近い)

カイトは警戒レベルを最大に上げた。

カイトはすぐに船長に報告した。

「海竜の類ですが、普通の海竜じゃありません。魔力が歪んでいます」
「虚無の影響か？」

ヴィクトル船長が尋ねた。

「その可能性が高いです。深海神殿の封印が弱まって、影響が出ているのかもしれません」
「どうする？　戦うか？」

カイトは少し考えた。

「まだ攻撃してきません。様子見しましょう。でも、いつ動くか分からない。夜間は交代で見張りを立てます」
「分かった。乗組員にも伝えておく」

ヴィクトル船長は頷いた。

夜になり、海は闇に包まれた。

船は夜通し航行する。ヴィクトル船長の腕があれば、暗い海でも進める。

カイト、エリナ、リアム、シオンの四人は、交代で甲板の見張りに立つことになった。

最初のシフトはカイトとリアムだ。

二人は甲板に並んで座った。

「カイト、あの海竜。剣で倒せる相手か？」

リアムが尋ねた。

「水中だと、剣は不利だね。水の抵抗が大きいから」

カイトが答えた。

「かといって、魔法だけじゃ倒しきれないかも。連携が必要だ」
「分かってる。俺も水中戦の訓練が必要だな」

リアムは剣の柄に手を置いた。

「剣士として、海中で役に立たないのは悔しい。でも、現実を受け入れる」
「リアムは船の上で戦ってくれればいいよ。海竜が襲ってきたら、甲板に上がってくるはずだから」
「そうか。それなら、俺の領分だ」

リアムは少し笑った。

深夜、二時。

カイトとリアムのシフトが終わり、エリナとシオンに交代した。

「異常はない。でも、海竜はまだついてきてる」

カイトが報告した。

「分かった。気をつけるね」

エリナが頷いた。

「シオン、無理しないでね。まだ船酔いが完全に治ったわけじゃないし」
「大丈夫！　カイトの魔法、まだ効いてるから」

シオンは元気に答えた。

深夜三時半。

静寂を破る轟音が響いた。

ドォォォォォン！

船が激しく揺れた。

「な、何！？」

シオンが悲鳴を上げた。

エリナは咄嗟に水魔法で身体を固定した。

「シオン、こっち！」

龍のような頭。だが、皮膚は腐敗している。目は虚ろで、代わりに黒い霧が漂っていた。

「虚無に侵食された海竜……！」

エリナが叫んだ。

「避けて！」

ビームが甲板を直撃した。木片が飛び散る。

「敵襲か！」

リアムは剣を grabbed して甲板に駆け出した。

カイトも続く。

「エリナ！　シオン！」

カイトが叫んだ。

「カイト！　こいつ、強い！」

エリナが叫び返した。

海竜は再び口を開けた。今度は、黒い霧を吐き出している。

「虚無の霧か……吸い込むな！」
