第436章：種の目覚め

夜が明け、朝の光が部屋に差し込んでいた。カイトは机の上に置かれた「謎の種」をじっと見つめていた。昨夜、西の森で見つけたこの種は、ただの植物の種ではなさそうだった。淡い光を帯びており、時折、まるで呼吸するように明滅している。

「寝ないの？」

背後からエリナの声がした。カイトが振り返ると、彼女は眠そうな目をこすりながらドア枠に寄りかかっていた。

「ああ、これが気になって。何か特別な気がするんだ」

カイトは種を指さした。エリナが近づいてきて、机の上の種を覗き込む。

「確かに……普通の種じゃないわね。魔力を感じる」

エリナは種に手をかざした。彼女の指先から淡い光が漏れ、種に反応するように光の脈動が速くなった。

「反応してる。この種、生きてるわ」
「生きてる？」
「ええ。植物としての生命力じゃなくて、もっと……概念的な何かが宿ってる感じ」

カイトは考え込んだ。この世界の魔法は彼にとってコードとして理解できる。なら、この種も何らかの「プログラム」を持っているはずだ。

彼は目を閉じ、集中した。魔力感知を展開し、種の内部構造を探る。すると、目に見えない世界で無数の文字列が浮かび上がってきた。


entity Seed {
    type: "unknown_ancient"
    state: "dormant"
    awakening_condition: {
        trigger: "compatible_magic_resonance"
        threshold: 0.75
    }
    contained_data: {
        encrypted: true
        estimated_size: "infinite"
    }
}
「これ……データが入ってる」

カイトは目を開けた。エリナが首をかしげる。

「データ？何の？」
「わからない。暗号化されてる。でも、この種は何かを保存するための容器みたいだ」
「古代の記録媒体ってこと？」
「かもしれない。それとも……未来へのメッセージか」

カイトは種を手に取った。手のひらの中で、種は温かく脈動している。まるで心臓の鼓動のようだった。

「『compatible_magic_resonance』……適合する魔力共鳴がトリガーになってる」
「つまり？」
「この種を目覚めさせるには、特定の魔力パターンが必要だってこと」

エリナが考え込む。

「私の魔力じゃ反応しなかった。ってことは、私のパターンは違うのね」
「カイト、あなたは？」

カイトは自分の魔力を種に流してみた。通常の魔法構築で使う青い光が種に触れる——が、反応はなかった。

「俺もダメか」

だが、カイトは諦めなかった。彼は別のアプローチを試みることにした。この世界に来てから彼が獲得した様々な魔法のパターンを、一つずつ種に流してみるのだ。

火の魔法。水の魔法。風の魔法。土の魔法。雷の魔法。氷の魔法。

どれも反応しない。

「だめだ。全部試したけど——」

ふと、カイトは思い出した。まだ試していない魔法がある。時空魔法だ。

リリアナから教わった時空魔法のパターン。空間を操り、時間を歪める力。カイトはその魔力構造を再現し、種に流した。

その瞬間——

種が激しく光り輝いた。


AWAKENING SEQUENCE INITIATED
DECRYPTING CONTAINED DATA...
WARNING: INFINITE LOOP DETECTED
ATTEMPTING PARTIAL DECRYPTION...
「うわっ！」

カイトは種を取り落としそうになった。種から溢れ出る光が部屋中を包み込み、壁や天井に幾何学模様を投影していく。それは古代文字のようでもあり、現代のコードのようでもあった。

「カイト、これ何！？」

エリナが叫ぶ。カイトは光の中で目を細めながら、目の前に現れた情報を読み取ろうとした。


PARTIAL DECRYPTION COMPLETE
EXTRACTED DATA:

- TIMELINE FRAGMENT: YEAR 847
- EVENT: GREAT CONVERGENCE
- LOCATION: COORDINATES [REDACTED]
- WARNING: "THE SEAL WEAKENS"
- SIGNATURE: "THE FIRST ADMINISTRATOR"
「管理者……？」

カイトは息を飲んだ。「最初の管理者」——ミラベルやヴォルターたちの先駆者だ。この種は、彼らが残した何かなのか？

光は徐々に収束していき、一つの像を形作った。それはホログラムのように空中に浮かぶ、ローブを纏った老人の姿だった。

「私は最初の管理者、アレクシス・ヴェイル」

老人の像が口を開いた。声は穏やかで、どこか悲しげだった。

「この種を見つけた者よ。君が時空魔法を持っているということは、運命の導きによるものだ。この種は、私が残した数多の『記憶の種』の一つ」
「記憶の種？」

カイトが呟くと、アレクシスの像は続けた。

「この世界には、かつて『大収束』と呼ばれる災厄があった。複数の次元が交差し、境界が崩壊寸前まで陥ったのだ。私たちは何とかそれを防いだが、代償として多くの知識と記憶を失った」
「私はそれを恐れ、世界の各所に記憶の種を植えた。これらは、私の知識と、そして未来への警告を含んでいる」
「警告？」
「『封印は弱まっている』。大収束の際に封じた『虚無の根源』が、再び目覚めようとしているのだ。君たちは準備しなければならない」

像は一瞬静止し、それから新たな情報を投影した。


ADDITIONAL DATA:

- OTHER SEEDS LOCATION: 7 CONFIRMED
- TOTAL SEEDS PLANTED: UNKNOWN
- SEAL STATUS: 34% COMPROMISED
- ESTIMATED TIME TO FULL BREACH: 3 YEARS
- RECOMMENDED ACTION: FIND ALL SEEDS
「封印の三分の一がすでに損なわれている……そして、あと三年で完全に崩壊する」

カイトは戦慄した。三年という時間は、短いようで長い。だが、敵が何なのかもわからない状況で、どうやって備えればいいのか。

「他の種を見つけろって言っても、どこにあるのか……」

その時、カイトの目の前に地図が投影された。七个の光点が、大陸の各所で点滅している。

「これが、今確認できている種の位置だ」

アレクシスの像は徐々に薄れていった。

「私の時間は限られている。種に込められた魔力は、一度の覚醒でほとんど消費されてしまう。だが、種そのものは残る。他の種を見つけ、記憶を継ぐのだ」
「そして、もう一つ。時空魔法を持つ者への忠告がある」
「時間は川のようなものだ。流れに逆らうことはできるが、代償がある。あまりに大きな変更を加えれば、川は暴れ、全てを押し流すだろう。慎重に進むのだ」

像は完全に消え、光も収束して種の中に戻っていった。種は以前よりも淡い光を放っているが、さっきのような激しい反応は見せなくなっていた。

部屋には重い沈黙が漂った。

「……すごいことになったわね」

エリナがポツリと言った。カイトは頷く。

「ああ。この世界にはまだ、俺たちが知らない秘密がたくさんある」
「三年で全ての種を見つけるの？」
「いや、それだけじゃない。封印を修復する方法も見つけなきゃ」

カイトは種をポケットにしまった。この種は、新たな旅の始まりの象徴だった。

「まずは、仲間たちに伝えよう。リアムやセシリア、それに他の賢者たちにも」
「ええ。一人で背負い込まないでね」

エリナが優しく微笑んだ。カイトも小さく笑みを返す。

「わかってる。俺たちはチームだ」

窓の外では、朝日が完全に昇りきっていた。新たな一日が始まる。そして、新たな冒険も。

カイトは心の中で、これから直面するであろう困難を覚悟した。三年というタイムリミット。正体不明の敵。そして、散らばった記憶の種。

だが、彼には強力な仲間がいた。そして、何より——コードを操るという、この世界で唯一無二の能力があった。

「行こう、エリナ」
「ええ、カイト」

二人は部屋を出た。廊下には、既に起きていたリアムとセシリアが立っていた。

「おはよう。何かあったのか？」

リアムがカイトの真剣な表情を見て尋ねた。

「ああ。朝飯の前に、みんなで話さなきゃいけないことがある」

カイトは答えた。これから始まる冒険の予感が、胸を高鳴らせていた。

【用語解説】

  • 記憶の種（Memory Seed）：最初の管理者アレクシス・ヴェイルが残した、知識と警告を含む特殊な種。時空魔法を持つ者のみがアクセスできる。
  • 大収束（Great Convergence）：過去に起きた次元災厄。複数の次元が交差し、境界が崩壊寸前まで陥った事件。
  • 虚無の根源（Root of Void）：大収束の際に封印された存在。詳細は不明だが、世界を滅ぼす力を持つとされる。
  • アレクシス・ヴェイル：最初の管理者。時空魔法の使い手であり、記憶の種の創造主。
