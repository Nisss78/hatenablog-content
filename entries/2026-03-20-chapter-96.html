第96章：荒廃の地

アークライトの声が、まだ耳に残っていた。

「会うのを楽しみにしている」

カイトは、朝まで眠れなかった。焚き火のそばで、剣を磨き続けた。エリナが、心配そうに近づいてきた。

「カイト、眠らないの？」

カイトは顔を上げた。

「アークライトの声が、頭から離れないんだ」

エリナは、カイトの隣に座った。

「夢？」

カイトは頷いた。

「彼が、こっちを知っている。俺たちの動きを、監視しているのかもしれない」

エリナは、カイトの手を握った。

「大丈夫。私たちがいるから」

カイトは、エリナの手の温かさを感じた。少しだけ、心が軽くなった。

「ありがとう。行こう、東方へ」

東方への道は、過酷だった。

南方の森を出て、平原を横断し、乾燥した大地を越えた。途中、いくつかの村を通ったが、どの村も荒廃していた。

畑は枯れ、井戸は干上がり、家は崩れかけていた。村人たちは、痩せこけて、目は虚ろだった。

「これは……」

セシリアが、言葉を失った。

リアムが、地面を蹴った。

「なんだよこれ。何が起きているんだ？」

カイトは、ハーモニーシステムで周囲を分析した。


// 荒廃地域の魔力分析

カイトは画面を見た。源泉は「休眠状態」。つまり、完全に死んだわけではない。しかし、管理者がいないため、魔力を供給できていなかった。

「管理者がいないと、こうなるんだな」

カイトが呟いた。

リリスが言った。

「源泉は、管理者が制御して初めて力を発揮する。管理者がいなければ、ただの巨大な魔力の塊。でも、それが枯渇すると、こうなる」

カイトは村人に近づいた。老人が一人、崩れた家の前に座っていた。

「おじいさん、大丈夫ですか？」

老人は、ゆっくり顔を上げた。目には、生気がなかった。

「水……水がほしい……」

カイトは、水筒から水を注いで、老人に渡した。老人は、震える手で水を受け取り、一口飲んだ。

「ありがとう……魔法使い様……」

老人は、涙を流した。

「この村には、昔は泉があったんだ。水が湧き出て、作物が育って、子供たちが笑っていた。でも、ある日突然、泉が枯れた」

カイトは尋ねた。

「いつ頃ですか？」

老人は、指を折った。

「五年くらい前か。泉が枯れてから、村は死んでいった。若者たちは去り、老人だけが残った」

カイトは、ハーモニーシステムでさらに詳しく調べた。


// 五年前のイベントを検索

カイトは息を飲んだ。管理者が、殺された？

「おじいさん、五年前に、何かあったんですか？ 変な人たちを見たり」

老人は、目を細めた。

「ああ……黒いローブを着た人たちが、山の方へ行ったのを見たな。数日後、山から大きな光が上がって、それから泉が枯れた」

黒いローブ。アークライトの手先かもしれない。

カイトは、村人たちに食料と水を分け与えた。しかし、それは一時的な救済に過ぎない。根本的な解決には、源泉の復活が必要だった。

「行こう。源泉を見に行く」

カイトが言った。

東方の源泉は、山の頂上にあった。

山道は荒れていた。木々は枯れ、土は剥き出しになっていた。かつては豊かな森だった場所が、今は死の世界になっていた。

山頂に着くと、そこには巨大なクレーターがあった。中心に、枯れた泉があった。かつては、魔力を噴き出していた場所。今は、ただの穴になっていた。

「これが、源泉……」

セシリアが、悲しそうに言った。

カイトは、源泉に近づいた。穴の中を覗き込んだ。底には、白い骨が散らばっていた。人間の骨ではなかった。もっと大きく、翼のような骨だった。

「管理者の遺体か？」

リアムが尋ねた。

カイトは頷いた。

「おそらく。カグヤという管理者の」

カイトは、ハーモニーシステムで遺体を分析した。


// 遺体の分析

カイトは眉をひそめた。魂の破壊。それは、単なる殺害ではなかった。管理者としての力を、永遠に奪うための手段だった。

「アークライトの仕業か？」

アルフレッドが尋ねた。

カイトは首を振った。

「五年前だ。アークライトが動き出したのは、もっと最近だ。これは、別の誰かの仕業かもしれない」

リリスが口を開いた。

「闇のギルドよ。彼らは、長年、管理者を狩っていた」

カイトは驚いた。

「管理者を狩る？」

リリスは頷いた。

「管理者の体の一部――心臓、目、骨――には、膨大な魔力が宿っている。それを、武器や薬に加工して売るの。闇のギルドにとって、管理者は高価な獲物よ」

カイトは戦慄した。管理者を、単なる材料として見ている人間がいるなんて。

「カグヤは、狩られたんだな」

リリスは悲しそうに言った。

「おそらく。魂を破壊したのは、再利用を防ぐため。残酷な話だけど」

カイトは、源泉の底を見た。カグヤの骨。かつては、この地を守っていた存在。そして、無惨に殺された。

「許せない」

カイトが呟いた。

「こういうのは、許せない」

エリナが、カイトの肩に手を置いた。

「カイト……」

カイトは顔を上げた。

「源泉を復活させられないか？」

リリスは首を振った。

「管理者がいないと無理よ。源泉は、管理者が制御して初めて機能する」

カイトは考えた。ハーモニーシステムを開いた。何か方法がないか、検索した。

カイトは、一時的な魔力供給を選んだ。今すぐにでも、この地を救いたかった。

「みんな、魔力を貸してくれ。源泉に、一時的にでも魔力を供給したい」

エリナ、リアム、セシリア、アルフレッド、リリス。全員が頷いた。

「わかった」

カイトは、ハーモニーシステムを開いた。魔力転送のプログラムを組んだ。


// 源泉への一時的魔力供給

カイトたちの魔力が、源泉に流れ込んだ。枯れた泉が、少しだけ光り始めた。

地面が揺れた。源泉から、細い水脈が湧き出した。わずかだったが、確かに水が戻った。

「水が出た！」

セシリアが叫んだ。

カイトは息を吐いた。全身の魔力を使い果たして、疲労困憊だった。

「七日間は持つ。その間に、根本的な解決を見つけないと」

エリナがカイトを支えた。

「カイト、休んで」

カイトは座り込んだ。源泉のそばで、空を見上げた。

その時だった。

源泉の底から、声が聞こえた。

『ありがとう……』

カイトは驚いて起き上がった。

「誰だ？」

源泉から、淡い光が昇った。光は、人の形をとった。若い女性の姿。長い黒髪、切れ長の目、白い肌。背中には、半透明の翼があった。

「カグヤ……？」

カイトが尋ねた。

女性は微笑んだ。

「魂は破壊された。でも、欠片が残っていたの。あなたの魔力で、少し目覚めた」

彼女の声は、風鈴のように澄んでいた。

「あなたが、カイトね。オーリンの後継者」

カイトは頷いた。

「どうして、俺のことを？」

カグヤの欠片は言った。

「管理者は、死んでも、少しは世界と繋がっている。あなたが現れたことも、アークライトの野望も、知っている」

彼女は、カイトに近づいた。

「私は、もう長くはいられない。でも、あなたに伝えたいことがある」

カイトは身を乗り出した。

「何ですか？」

カグヤの欠片は言った。

「私を殺したのは、闇のギルドの長、"ザイン"よ。彼は、アークライトと手を組んでいる」

カイトは目を見開いた。

「闇のギルドとアークライトが？」

カグヤの欠片は頷いた。

「ザインは、私の魂を破壊した。そして、その技術をアークライトに売った。アークライトは、他の管理者も同じ方法で殺すつもりよ」

カイトは拳を握った。

「なんてことだ」

カグヤの欠片は、カイトの頬に触れた。冷たかったが、優しかった。

「あなたは、強い。でも、一人では勝てない。ヴォルター、ミラベル、そして、もう一人の管理者の協力を得なさい」

カイトは尋ねた。

「もう一人？」

カグヤの欠片は言った。

「北西の源泉に、"ゼノン"がいる。彼は、アークライトに最も近い場所で、一人で戦い続けている。彼を、助けてあげて」

カイトは頷いた。

「約束します」

カグヤの欠片は微笑んだ。

「それから、これを」

彼女は、カイトに小さな水晶を渡した。

「私の力の欠片。使うときは、私の名前を呼んで」

カイトは水晶を受け取った。温かかった。

「ありがとうございます」

カグヤの欠片は、少しずつ薄くなっていった。

「さようなら、カイト。世界を、救って……」

彼女の姿は、光となって消えた。

カイトは、水晶を握りしめた。

「必ず、救う」

カイトは、立ち上がった。仲間たちが、心配そうに見ていた。

「カイト、大丈夫？」

エリナが尋ねた。

カイトは頷いた。

「ああ。行こう。次は、西方の源泉だ」

カイトたちは、山を下りた。

村に戻ると、井戸から水が湧き出ていた。村人たちが、歓声を上げていた。

「水が出た！水が出たぞ！」

老人が、カイトの手を握った。

「ありがとう、ありがとう！」

カイトは微笑んだ。でも、心の中では、重い決意を抱いていた。

七日間。その間に、すべてを変えないと。

カイトたちは、村を後にした。西方へ。

荒廃の地には、絶望があった。でも、希望の欠片もあった。

カグヤの水晶。彼女の願い。そして、仲間たちの信頼。

カイトは、空を見上げた。雲の切れ間から、太陽が差し込んでいた。

「行こう。世界を救うために」

カイトの声に、仲間たちが応えた。

「おう！」

旅は続く。調和への道は、遠い。でも、カイトは諦めない。

絶望を、希望に変えるために。
