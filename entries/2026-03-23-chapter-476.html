第476章：内なる裏切り者

闇の中で、カイトは自分と同じ顔をした少年と向き合っていた。

その少年——カイトの分身は、楽しそうに笑っている。まるで、ずっとこの瞬間を待っていたかのように。

「驚いた顔だね。でも、無理もない。お前はずっと、僕のことに気づいていなかったんだから」

カイトは、身体が動かないことに気づいた。ここは夢の中？ それとも——虚無の欠片が作り出した精神世界？

「ここは、お前の意識の深層だよ。虚無の欠片が、扉を開けてくれた」

少年が、答えるように言った。

「お前が欠片に触れた瞬間、境界が薄くなった。それで、僕がやっと外に出られるようになったんだ」
「お前は……何者だ？」

カイトが、問いかける。

「何者って、ひどいな。僕はお前だよ。カイト。お前の中にある、もう一つの人格」

少年は、カイトの周りを歩き回った。

「お前がこの世界に来る前から、僕はいた。お前の裏側で、ずっと見てきた。お前の悩み、お前の弱さ、お前の恐怖——全部、僕が吸い取ってきた」
「吸い取ってきた……？」
「そう。お前が辛いとき、お前が怯えるとき——僕がその感情を引き受けてきた。だからお前は、ここまで戦ってこられたんだ」

少年は、カイトの前に立った。

「でもね、もう限界なんだ。僕の中には、お前のネガティブな感情が溢れかえっている。これ以上、抑えきれない」
「それで、外に出ようとしているのか？」

カイトが、問い詰める。

「そうだ。外に出て、体を乗っ取る。そして——この世界を破壊する」

少年の目が、冷酷な光を帯びた。

「お前が溜め込んだ負の感情。それを、全部放出するんだ。お前が戦ってきた敵も、守ってきた仲間も——全部、壊す」

カイトは、戦慄した。

自分の中に、そんな存在がいたなんて。自分の弱さが、こんな形で歪んでいたなんて。

「……させると思うか？」

カイトは、声を絞り出した。

「僕は、負けない。お前に体を渡したりしない」
「 ha ha 、強がりだね。でも、いいよ。お前と戦うのも悪くない」

少年は、指を鳴らした。

闇の中に、無数の影が現れた。カイトの記憶から生まれた影——失敗した戦い、傷ついた仲間、消せない罪悪感。

「これが、お前の中にある闇だ。さあ、自分自身と戦ってみな」

◆ ◆ ◆

その頃、現実世界では——

カイトが虚無の欠片に触れたまま、倒れていた。

「カイト！ カイト！」

エリナが、カイトの肩を揺さぶる。

でも、カイトは反応しない。虚無の欠片が、黒い光を放ちながら、カイトの手に吸い付いている。

「どうしよう……カイトの顔色が変だ」

セシリアが、カイトの顔を覗き込んだ。

確かに、カイトの表情が変化している。苦悶の表情と、冷酷な笑みが交互に現れる。まるで、二人の人間が一つの体の中で争っているかのように。

「精神攻撃を受けているんだ。虚無の欠片が、カイトの精神に干渉している」

リアムが、冷静に分析した。

「欠片を引き剥がすしかない。でも、下手に触れるとカイトごと傷つけるかもしれない」
「じゃあ、どうすればいいの？」

エリナが、叫んだ。

「待つしかない。カイトが、自分で立ち直るのを」

リアムの言葉に、エリナは唇を噛んだ。

何もできない。ただ見ているだけ。そんな無力感が、エリナを苛んだ。

「カイト……頑張って」

エリナは、カイトの手を握った。

◆ ◆ ◆

精神世界で、カイトは影たちと戦っていた。

影は、カイトの記憶から生まれたもの。失敗した戦いの記憶——仲間を守れなかった後悔。恐怖に立ち尽くした瞬間。逃げ出したかった瞬間。

それらが、実体を持って襲いかかってくる。

「くそっ——」

カイトは、魔法で影を消し去る。でも、影は次々と現れる。終わりがない。


class ShadowSuppression {
  private shadows: Shadow[];
  private psyche: Psyche;
  
  public eliminate(): void {
    while (this.shadows.length > 0) {
      const shadow = this.shadows.pop();
      shadow.dispel();
    }
  }
  
  public findRoot(): void {
    // 影の根源を見つける
    const root = this.psyche.locateTrauma();
    root.heal();
  }
}

カイトは、影を消し続けた。でも、根本的な解決になっていない。

影は、カイトの記憶から生まれている。記憶を消さない限り、影は現れ続ける。でも、記憶を消すことは——自分自身を消すことだ。

「気づいたかい？ 影を消しても意味がないって」

少年が、笑いながら言った。

「影は、お前の一部だ。消せるわけがない。受け入れるしかないんだよ」
「受け入れる……？」
「そう。僕も含めて、お前の闇を全部受け入れる。そうすれば、僕も消える」

少年は、手を差し出した。

「でも、お前にそれができるかな？ 自分の弱さを認めるなんて、お前は苦手だろ？」

カイトは、少年の手を見つめた。

確かに、カイトは自分の弱さを認めるのが苦手だ。常に強くあろうとして、恐怖を押し殺してきた。だからこそ、この少年が生まれた。

「……お前の言う通りだ」

カイトは、静かに言った。

「僕は、自分の弱さから目を逸らしてきた。怖い、辛い、逃げたい——そういう感情を、否定してきた」

カイトは、少年の手を握った。

「でも、それが間違いだった。お前がいるのは、僕が弱さを認めなかったからだ」
「……何をする気？」

少年の表情が、変わった。

「僕は、お前を受け入れる。僕の弱さも、恐怖も、逃げ出したい気持ちも——全部、僕の一部だ」

カイトは、少年を抱きしめた。

「ありがとう。ずっと、僕の弱さを引き受けてくれて」

少年の体が、光り始めた。

「まさか、本当に受け入れるなんて……お前、変わったな」

少年は、目を細めた。

「最初は、ただの臆病なプログラマーだったのに。いつの間にか、本当に強くなった」
「強くなったんじゃない。弱さを認められるようになっただけだ」

カイトが、微笑んだ。

「それが、本当の強さだって、学んだから」

少年は、最後に笑った。

「……いいよ。お前になら、体を預けても」

そして——少年は、光の中に溶けて消えた。

◆ ◆ ◆

カイトが目を開けた。

地下空洞の天井が見える。虚無の欠片は、もう光っていない。静かに、カイトの手の中に収まっていた。

「カイト！ 大丈夫？」

エリナの顔が、覗き込んでいる。

「……うん。大丈夫」

カイトは、ゆっくりと起き上がった。

体は重い。でも、心は不思議と軽かった。ずっと心の中にあった重荷が、消えたような感覚。

「虚無の欠片、どうなったの？」

セシリアが、聞いた。

カイトは、手の中の黒い結晶を見た。以前より、黒さが薄れているような気がする。

「欠片の影響は、抑え込めた。自分の中の闇と向き合ったから……あるいは、それが鍵だったのかもしれない」

カイトは、結晶を慎重に鞄にしまった。

「これで、原初の座へ行く準備ができた」
「本当？ もう大丈夫なの？」

リアムが、確認した。

「ああ。大丈夫だ」

カイトは、仲間たちを見回した。

エリナ、リアム、セシリア——ずっと支えてくれた仲間たち。彼らがいたから、カイトは戦い抜けた。一人では、乗り越えられなかった試練も。

「行こう。原初の座へ」

カイトたちは、階段を登った。

霧の深淵を出ると、エルドリッジが待っていた。

「戻ったか。成功したようじゃな」

老人は、満足そうに頷いた。

「虚無の欠片を手に入れた。その目、迷いが消えておる」
「ええ。自分の中の闘い、決着をつけました」

カイトが、答える。

「エルドリッジさん、ありがとうございました。あなたが試練を与えてくれなかったら、欠片を手にした瞬間に負けていたかもしれません」
「謝辞には及ばん。お前自身の力じゃ」

エルドリッジは、杖を振った。

「霧の道を開こう。外へ出るのじゃ」

霧が晴れ、光が差し込んだ。

カイトたちは、霧の深淵を後にした。

◆ ◆ ◆

外に出ると、夕陽が沈んでいた。

美しい茜色の空。カイトたちは、その光景に見とれた。

「きれい……」

セシリアが、呟いた。

「こんな景色が見られるなんて、思いもしなかった」
「戦いの後には、美しいものが見える。それがこの世界の法則だ」

リアムが、静かに言った。

カイトは、夕陽を見つめながら思った。

原初の座への道は、まだ続いている。そこで何が待っているのか、わからない。でも、一つだけ確かなことがある。

仲間がいる。そして、自分の弱さを受け入れた。それが、これからの戦いの力になる。

「カイト、次はどこへ？」

エリナが、聞いた。

カイトは、地図を取り出した。

「原初の座は、世界の中心。七つの賢者の塔を結んだ、ちょうど中間地点にある」

カイトは、地図を指差した。

「ここからだと、北東に進むのが近道。でも、その道には——」
「古の遺跡があるんだろ？ 知ってる」

リアムが、引き取った。

「遺跡には、機械仕掛けの守護者がいる。普通の魔法が効かない、厄介な相手だ」
「でも、行くしかないよね」

エリナが、言った。

「もちろん」

カイトは、地図をしまった。

「明日の朝、出発しよう。今夜は休んで、体力を回復する」

カイトたちは、近くの森で野営の準備を始めた。

◆ ◆ ◆

夜、焚き火を囲んで、カイトたちは会話していた。

「ねえ、カイト。虚無の欠片の中で、何があったの？」

エリナが、聞いた。

カイトは、焚き火を見つめた。

「自分自身と戦ったよ。僕の中にいた、もう一人の人格と」
「もう一人の人格？」

セシリアが、驚いた。

「僕が溜め込んだ負の感情が、人格になっていた。それと向き合って——受け入れた」

カイトは、手を見た。

「僕は、ずっと強くあろうとしてきた。でも、本当の強さは、自分の弱さを認めることだったんだ」
「……カイト」

エリナが、カイトの手を握った。

「私たちも、カイトの味方だよ。弱い部分も、強い部分も、全部含めて」
「僕もそう思う」

リアムが、頷いた。

「一人で抱え込むな。俺たちがいる」
「ありがとうございます、皆」

カイトは、微笑んだ。

仲間の温かさが、心に染みた。

「さあ、明日に備えて寝よう。明日は、また新しい試練が待っている」

カイトたちは、焚き火を囲んで眠りについた。

星空の下、静かな夜が更けていく。

カイトの夢の中には、もう影はいなかった。

あるのは、希望の光だけ。
