第474章：霧の深淵への道

朝露に濡れた大地を、カイトたちは歩いていた。

岩山を背に、北西へ。霧の深淵を目指して。

空は澄み渡り、遠くに山脈の稜線が見える。テラドンの洞窟を出てから、もう半日が経っていた。

「北西って、どっち？」

リアムが、空を見上げながら聞いた。

「太陽が東から昇るなら、今は南西にあるはず。だから、あっち」

エリナが、太陽の位置を確認して指差した。

「方角は合ってる。でも、一週間も歩くのか。遠いな」
「空間のコード、もう一度試してみる？」

セシリアが、カイトに提案した。

昨夜の失敗以来、カイトは何度もコードを紡ごうとしていた。でも、結果は変わらない。空間に亀裂が走るものの、すぐに閉じてしまうのだ。

「……もう少し、練習が必要かな」

カイトは、正直に言った。

「空間を操作するには、座標の概念を理解しないと。ただコードを書くだけじゃだめだ」
「座標の概念？」
「うん。今いる場所と、行きたい場所。その二つを正確に指定して、空間の繋がりを記述する必要がある」

カイトは、歩きながら説明した。

「プログラミングでも同じ。変数を定義して、値を代入して、処理を実行する。一つでも間違えれば、バグが起きる」
「バグって、虫のこと？」

リアムが、首をかしげた。

「違うよ。バグは、エラーのこと。想定通りに動かない状態」
「ふーん。難しいな」
「でも、慣れれば分かるようになる。だから、歩きながら考えよう」

カイトは、前を向いた。

北西の空。その向こうに、霧の深淵がある。虚無の欠片を手に入れれば、原初の座へ行くことができる。そして、自分の力の秘密を知ることができる。

◆ ◆ ◆

昼過ぎ、カイトたちは小さな村に立ち寄った。

岩山の麓に位置する、静かな農村。畑には小麦が実り、子供たちが道端で遊んでいる。

「休憩しようか」

エリナが、提案した。

「水も補給したいし、何か食べるものも買いたい」
「賛成」

セシリアが、頷いた。

「足も少し休ませたいし」

カイトたちは、村の入り口にある小さな店に入った。パンや果物、保存食などが並んでいる。

「いらっしゃい。旅の方かな？」

店の老婆が、穏やかな声で迎えてくれた。

「はい。北西へ向かってるんです」

カイトが答えると、老婆の表情が少し曇った。

「北西……霧の深淵の方へ？」
「ええ、そうです」
「おやまあ。あそこは危険だよ。霧が濃くて、道に迷ったら二度と出られないって言われてる」

老婆は、心配そうに言った。

「昔、若い冒険者が何人も入っていったけど、誰も戻ってこなかった。本当に気をつけてね」
「ありがとうございます。覚えておきます」

カイトは、水とパン、干し肉を買った。老婆に礼を言って、店を出る。

店の外で、リアムが腕を組んだ。

「二度と出られない、か。恐ろしいな」
「でも、行かなきゃいけない」

セシリアが、きっぱりと言った。

「虚無の欠片を手に入れないと、原初の座へ行けない。一年以内に地脈の歪みを直さないと、魔法が使えなくなるんでしょ？」
「セシリアの言う通りだ」

カイトも、頷いた。

「危険でも、進むしかない。それに、僕のコードがある。道を切り開けるはずだ」

エリナが、カイトの肩を叩いた。

「そうそう。カイトがいれば大丈夫。信じてるよ」
「エリナ……ありがとう」

カイトは、少し照れくさそうに微笑んだ。

◆ ◆ ◆

村を出て、カイトたちは再び歩き始めた。

道は徐々に荒れていく。舗装された道が終わり、土の道へ。そして、土の道も消え、ただの獣道になった。

「道がなくなったな」

リアムが、周囲を見渡した。

「地図、持ってる？」
「持ってるけど、役に立つかな」

エリナは、地図を広げた。でも、この辺りの詳細は描かれていない。『未踏の地』と記されているだけだ。

「テラドンさんは、北西って言ってた。方角さえ合ってれば、なんとかなるでしょ」

カイトが言うと、エリナは地図をしまった。

「そうね。信じて進もう」

カイトたちは、北西へ向かって歩き続けた。

夕方になり、日が傾き始めた頃、風景が変わった。

それまで広がっていた草原が終わり、低木が増えてきた。そして、遠くに——霧が見えた。

「あれが……霧の深淵？」

セシリアが、指差した。

地平線の向こうに、白く濁った霧の壁が広がっている。まるで、世界の果てにある幕のように。

「すごいな。あんなに広い霧、見たことない」

リアムが、目を丸くした。

「自然の霧じゃない気がする」

カイトが、呟いた。

「何か、意図的なものを感じる。魔法か、あるいは……」

カイトは、霧を見つめた。

コードで世界を記述する感覚。霧の奥に、何かの構造がある気がする。複雑で、混沌とした構造。

「まだ距離がある。今日はここで野営して、明日霧に入ろう」

エリナが、提案した。

「霧の中で夜を迎えるのは危険だ。視界が利かない状態で、魔獣に襲われたらやばい」
「賢明だ」

カイトは、頷いた。

◆ ◆ ◆

その夜、カイトは霧の縁で座っていた。

仲間たちは、すでに眠っている。焚き火が静かに燃えている。

カイトは、霧を見つめていた。白く濁った壁。その向こうに、何があるのか。

「……眠れない？」

後ろから、セシリアの声がした。

「セシリア。起こしちゃった？」
「ううん。目が覚めただけ」

セシリアが、カイトの隣に座った。

「霧、見てるの？」
「うん。何か、感じるんだ」

カイトは、霧に手を伸ばした。

「あの中に、何かがある。コードで記述できる、何か」
「虚無の欠片？」

「いや、もっと広い意味での何か。霧自体が、一つのシステムになってる気がする」

カイトは、目を閉じた。

霧の中にあるもの。それを、コードとして捉える。

カイトは、霧の中に何かを見た気がした。

霧の壁。その奥に、無数の道が交差している。迷路。いや、もっと複雑な——多次元の迷路。

「カイト？」

セシリアの声で、カイトは我に返った。

「ごめん。ちょっと、分析してた」
「分析？」
「霧の構造。あの中は、ただの霧じゃない。多次元の迷路になってる。普通の方法じゃ進めない」

セシリアは、難しい顔をした。

「じゃあ、どうすればいいの？」
「コードで、道を記述する。今いる座標から、目的の座標までの経路を計算して、通れるようにする」

カイトは、説明した。

「でも、明日中にできるかわからない。時間がかかるかも」
「焦らなくていいよ」

セシリアが、カイトの手を握った。

「私たちは待ってる。カイトができるって信じてるから」
「セシリア……」
「それに、霧の中で何が起きるか分からないけど、私も光で道を照らせる。二人で力を合わせれば、なんとかなる」

セシリアの笑顔に、カイトは心が温まった。

「……うん。ありがとう」

カイトは、空を見上げた。

星が、綺麗に輝いている。霧の縁だけ、空が開けている。まるで、世界が二つに分かれているかのように。

「明日、霧に入る」

カイトは、決意を口にした。

「虚無の欠片を手に入れて、原初の座へ行く。そして、この世界を救う」
「うん。一緒に行こう」

セシリアが、カイトに寄り添った。

霧の向こうにあるもの。それは、まだ分からない。でも、進むしかない。

カイトは、明日に備えて目を閉じた。

◆ ◆ ◆

翌朝、カイトたちは霧の前に立った。

白く濁った壁。その向こうに、何があるのか。未知の世界。危険とチャンスが混在する場所。

「準備はいい？」

エリナが、全員に聞いた。

「水と食料は十分。魔石もチャージ済み。何が起きても対応できる」
「OK」

リアムが、剣を抜いた。

「俺は前に立つ。何かいたら、最初に斬る」
「私は後ろを守る」

セシリアが、杖を構えた。

「光で照らすし、回復もできる」
「私は中盤で支援するね」

エリナが、風魔法の準備をした。

「視界を確保できるなら、やってみる」

カイトは、頷いた。

「僕は、先導する。コードで道を記述して、進む」

カイトは、霧に向かって手を伸ばした。

そして、コードを紡ぐ。


class MistNavigation {
  private currentCoordinates: Coordinates;
  private targetCoordinates: Coordinates;
  private mist: MistBarrier;
  
  public findPath(): Path {
    const path = new Path();
    path.calculate(this.currentCoordinates, this.targetCoordinates);
    path.optimizeForMist(this.mist);
    return path;
  }
  
  public revealPath(path: Path): void {
    for (const point of path.points) {
      this.clearMist(point);
      this.markWaypoint(point);
    }
  }
  
  private clearMist(point: Coordinates): void {
    // 指定座標周辺の霧を一時的に晴らす
    const radius = 5; // 5メートル半径
    mist.setVisibility(point, 0.8, radius);
  }
}

カイトの目の前で、霧が晴れた。

円形に、霧が薄くなり、道が見える。

「すごい……」

リアムが、呟いた。

「本当に道ができた」
「でも、ずっと続くわけじゃない」

カイトは、説明した。

「一定距離進むと、また霧が濃くなる。その都度、新しいコードを書いて進む必要がある」
「大変そうだけど、できないことじゃないね」

エリナが、微笑んだ。

「じゃあ、行こうか」

カイトたちは、霧の中へと足を踏み入れた。

視界は悪い。でも、カイトが開いた道だけ、周囲が見える。その道を頼りに、一歩ずつ進む。

「気をつけて。何かいるかもしれない」

リアムが、警戒しながら言った。

霧の中。その奥から、何かの気配が漂ってくる。魔獣か、それとも——

「来る！」

セシリアが、叫んだ。

霧の中から、影が飛び出してきた。

それは、狼のような形をしていた。でも、普通の狼じゃない。体が半透明で、中に闇が渦巻いている。虚無の眷属だ。

「虚無狼（ヴォイド・ウルフ）！ 気をつけて！」

エリナが叫んだ。

カイトは、すぐに反応した。コードを紡ぐ。

「リアム、今！」

カイトが叫ぶと、リアムが剣を振った。

「おらっ！」

剣が、虚無狼の体を切り裂いた。でも、半透明の体は、攻撃をすり抜ける。

「くそっ、効かない！」

リアムが、舌打ちした。

「物理攻撃は効果が薄い！ 魔法で！」

セシリアが、杖を掲げた。

「光魔法、聖なる光よ！」

光が、虚無狼を包んだ。闇が焼かれ、狼が悲鳴を上げる。

でも、倒せない。光魔法で弱らせただけだ。

「カイト、コードで！ 核を狙って！」

エリナが、叫んだ。

カイトは、虚無狼の体を見た。その中心に、小さな黒い球がある。それが、核だ。


class VoidWolfCoreDestruction {

  }

黒い球が、露出した。

「セシリア、あそこ！」

カイトが指差すと、セシリアが光を放った。

「聖なる一撃！」

光が、黒い球を貫いた。

虚無狼が、音もなく消滅した。霧の中に、静寂が戻る。

「やった……」

リアムが、息を吐いた。

「強いな、虚無狼。普通の魔獣とは別物だ」
「霧の深淵には、こういうのがたくさんいるってことか」

エリナが、深刻な顔で言った。

「油断できないね。カイト、道、開ける？」
「うん。続けよう」

カイトは、再びコードを紡いだ。

霧の中、虚無の欠片への道。それは、険しく、危険な道。でも、進むしかない。

カイトたちは、一歩ずつ、前へと進んだ。
