第583章：時空の裂け目

レゾナンスの街を出て三日目。

カイトたちは、原初の座への道を順調に進んでいた。

かつて訪れたことのある地形は、世界再構築の影響で微妙に変化していたが、テオドールから受け取った羊皮紙の地図は正確で、迷うことはなかった。

「あの丘の向こうが、原初の座の入り口だった場所ね」

エリナが、遠くに見える丘を指差す。

「以前は遺跡の入り口があったはずだけど……今はどうなってるかしら」

「行ってみればわかる」

リアムが、简潔に答える。

「警戒を怠るな。世界再構築の影響で、魔物の生態系も変わっている可能性がある」

カイトは頷き、杖を構えた。

「みんな、気をつけて進もう」

□ ■ □

丘を越えた先に、かつての遺跡入り口があった。

だが——以前とは明らかに様子が違っていた。

石造りの入り口は半ば崩れ、その奥からは、奇妙な光が漏れ出している。青と紫が混ざったような、不安定な色の光。

「これが……時空の歪み？」

シオンが、不安そうに呟く。

「空気が、ざわざわしてる。何かが、境界を越えようとしてる音がする」

セシリアが、眼鏡の位置を直しながら観察する。

「時空の裂け目が、すでに始まっているわ。遺跡自体が、因果の源流への入り口として変質している」

カイトは、コード・ビジョンを発動させる。

class RuinsEntrance {
    status = "TRANSFORMING";
    spatialIntegrity = 0.42; // 危険域
    temporalFlux = "UNSTABLE";
    
    warnings = [
        "CAUSALITY_LEAK_DETECTED",
        "TEMPORAL_ANOMALY_PRESENT",

「空間の完全性が四十二パーセント……かなり不安定だ」

カイトが報告すると、エリナが眉をひそめる。

「四十二パーセントって、半分以下じゃない。普通に入ったらどうなるの？」

「存在が拡散する。自分が自分でなくなる感覚……いや、もっと正確に言えば、過去と未来の自分が混ざり合って、現在の自分が消滅する」

リアムが、重い表情で頷く。

「不気味な場所だ。だが、通らねばならない」
「待って」

シオンが、カイトの袖を引く。

「私が何かできるかもしれない。七属性調律で、空間の歪みを少し整えられないか」

「できるのか？」

「試してみる価値はあると思う」

シオンは、両手を広げて深呼吸をする。そして、詠唱を始めた。

「光と闇、炎と氷、風と地、そして雷——七つの元素よ、我が声に応えよ。時空の裂け目を整え、道を開きたまえ」

シオンの体から、七色の光が放たれる。

光は遺跡の入り口へと伸び、青と紫の不安定な光と交錯した。しばらくの間、二つの光がせめぎ合う。

やがて——

「……できた」

シオンが、額の汗を拭う。

「完全には治せなかったけど、空間の完全性を六十パーセントまで上げられた。これなら、何とか通れるはず」

「六十パーセント……まだ危険域だが、四十二よりはずっとマシだ」

カイトは、シオンの肩をポンと叩く。

「ありがとう、シオン。助かったよ」

「ううん、みんなのためだもん」

シオンは、少し照れくさそうに笑う。

□ ■ □

一行は、遺跡の入り口をくぐった。

中に入ると、世界が一変した。

壁も床も天井も、半透明の膜のような物質に覆われている。その向こう側には、無数の光景が流れていた——

森が燃えている場面。

子供たちが笑い合う場面。

戦士が剣を振るう場面。

老人が本を読む場面。

それらすべてが、同時に、別々の速度で、壁の向こう側で繰り返されていた。

「これ……何？」

エリナが、驚愕の表情で呟く。

「過去の記憶？ それとも、未来の可能性？」

セシリアが、学術的な興味を瞳に宿す。

「両方よ。そして、もっと多くのもの。ここは『時空の記憶回廊』——世界が経験したすべての因果が、記録されている場所」

「因果の記録……」

カイトは、壁の向こう側の光景を観察する。

class CausalityRecord {

    // 原因と結果が永遠に保存されている

「すべての行いが、記録されている……」

カイトが呟くと、リアムが静かに言う。

「なら、俺たちの行いも、どこかで記録されているということか」
「ええ。私たちが今、ここにいることも、永遠の記録の一部になる」

セシリアが頷く。

シオンが、ふと立ち止まる。

「カイト、あれ」

彼女が指差した先には、一際鮮明な光景があった。

それは——カイトの記憶だった。

まだこの世界に召喚される前。日本で、プログラマーとして働いていた頃の姿。深夜のオフィスで、コードを書き続けていた姿。

「……俺だ」

カイトは、思わず息を呑む。

「こんなところに、俺の記憶があるなんて」

「カイトは、異世界からの来訪者だからよ」

エリナが、冷静に分析する。

「あなたの存在は、この世界の因果に大きな影響を与えた。だから、あなたの元の世界での記憶も、記録されている」

「因果に影響を与えた、か……」

カイトは、自分の過去の姿を見つめる。

あの頃の自分は、ただコードを書くことしか知らなかった。魔法なんて存在しない世界で、現実逃避のようにプログラミングに没頭していた。

「あの頃の俺は、まさか異世界で魔法を使えるようになるとは思ってもいなかっただろうな」

「でも、今のカイトの方がずっと生き生きしてる」

シオンが、優しく言う。

「私は、今のカイトが好きだよ」

「……ありがとう、シオン」

カイトは、自分の過去の姿に別れを告げ、前を向く。

「行こう。まだ先は長い」

□ ■ □

回廊を進むにつれて、風景は奇妙になっていった。

時には、天井が床になり、床が天井になる。

時には、左右の壁が回転し、進むべき方向が変わる。

時には、自分の子供時代の姿や老人の姿が、一瞬だけ視界を横切る。

「時間の流れが、完全に狂っている」

セシリアが、分析を続ける。

「過去、現在、未来——すべてが混在している。普通の精神なら、とっくに崩壊しているわ」
「俺たちは大丈夫なのか？」

リアムが問うと、セシリアは頷いた。

「カイトのコード・マジックと、シオンの七属性調律。この二つが、私たちの存在を安定させている。彼らがいなければ、確実に全滅していた」

カイトは、責任の重さを感じながら、さらに前進する。

（みんなを守らなきゃいけない）

彼は、心の中でコードを組み立てる。

class ExistenceStabilizer {

「これで、みんなの存在を八〇％以上に保てるはず」

カイトが宣言すると、エリナが微笑む。

「さすがカイト。頼りになるわね」

□ ■ □

さらに進むと、回廊の終わりが見えてきた。

そこには、巨大な扉があった。

扉は、黒曜石のような素材で作られており、表面には複雑な幾何学模様が刻まれている。その模様は、まるで生きているかのように、ゆっくりと動いていた。

「これが……因果の源流への扉？」

カイトが、扉の前に立つ。

扉の前には、一人の人物が立っていた。

白いローブを纏い、顔は深いフードで隠されている。性別も年齢もわからない。ただ、その存在感だけは、圧倒的だった。

「誰だ？」

リアムが剣を構える。

人物は、ゆっくりとフードを取った。

現れたのは——

美しい女性だった。銀色の髪、紫色の瞳。年齢不詳だが、二十代後半から三十代前半に見える。その表情は、静かで、どこか哀しげだった。

「私は『記録の守護者』の一人、アーカイブと申します」

彼女は、優雅に一礼する。

「因果の源流への入り口を守る者。あなたたちがここに来たことは、予期していました」

「予期していた……？」

カイトが問い返す。

「ええ。世界再構築の脅威に対処するには、真の原因を知る必要がある。あなたたちの判断は、正しい」

アーカイブは、少し微笑む。

「ですが——簡単には通せません。ここには、世界の秘密が眠っています。安易に公開すれば、世界に混乱をもたらす可能性があります」
「だから、試練を課すということね」

エリナが、諦めたように言う。

「その通りです」

アーカイブが頷く。

「試練は三つ。一つ目は『過去の受容』。二つ目は『未来の覚悟』。三つ目は『現在の証明』。これらすべてをクリアすれば、因果の源流への道を開きましょう」

カイトは、仲間たちを見渡す。

全員が、覚悟を決めた表情で頷いていた。

「やるしかない。受けます」

「では、最初の試練へ——」

アーカイブが手を振ると、カイトの周りの風景が変わり始めた。

□ ■ □

カイトが目を開けると、そこは見知らぬ場所だった。

古びた日本家屋。畳の部屋。障子越しに、柔らかな陽光が差し込んでいる。

「ここは……」

カイトは、この場所に見覚えがあった。

これは、彼が子供の頃に住んでいた家。祖父母の家だ。

「何でこんなところに……」

「カイト君？」

背後から声がして、カイトが振り向く。

そこにいたのは——

亡き祖母だった。

「お……おばあちゃん？」

カイトは、信じられない思いで祖母を見つめる。

祖母は、いつものように優しく微笑んでいた。

「どうしたの？ そんなに驚いた顔をして」

これは、過去の記憶だ。カイトは、すぐに理解した。

『過去の受容』——この試練は、自分の過去と向き合うこと。

カイトは、深呼吸をする。そして、祖母に向き合った。

「おばあちゃん……久しぶり」

「あらあら、そんな挨拶でいいの？ もっと近くにおいで」

祖母が、手を広げる。

カイトは、子供の頃に戻ったような気持ちで、祖母に抱きついた。

温かい。懐かしい。大好きな香り。

「おばあちゃん……会いたかった」

「ええ、ええ。私もよ、カイト君」

祖母は、カイトの頭を優しく撫でる。

「あなた、随分と遠くへ行ってしまったのね。でも、頑張っているんでしょう？ おばあちゃんにはわかるわ」

「……うん。今、異世界で冒険してるんだ。魔法も使えるようになった」

「まあ、素敵ね。魔法使いなんて、カイト君にぴったりだわ」

祖母が、嬉しそうに笑う。

「おばあちゃんね、カイト君がプログラミングを始めた時も、すごく楽しそうだなって思ってたの。コンピューターの中に、もう一つの世界を作ってるみたいで」

「……そうだね」

「だからね、今のカイト君も、きっと素敵な世界を作ってるんでしょう？ おばあちゃんは、それが嬉しいわ」

カイトの目から、涙がこぼれる。

「うん……。おばあちゃん、ありがとう」

「何をありがとうなんて。おばあちゃんは、いつでもカイト君の味方よ」

祖母の姿が、少しずつ薄れていく。

「さあ、もう行かなくちゃ。待ってる人がいるでしょう？」

「……うん。行かなきゃ」

カイトは、涙を拭って、祖母に別れを告げる。

「おばあちゃん、大好き」

「おばあちゃんもよ、カイト君。ずっと、ずっと大好き」

祖母の姿が消え、風景が元の時空の回廊に戻る。

カイトは、少しの間、その場に立ち尽くしていた。

「過去の受容、か……」

彼は、心の中で祖母に感謝を捧げる。

（おばあちゃん、見ててね。俺、絶対に世界を守るから）

□ ■ □

「見事です」

アーカイブが、満足そうに頷く。

「過去を否定せず、受容した。一つ目の試練、クリアです」

「カイト、大丈夫？」

エリナが、心配そうに駆け寄る。

「ああ、大丈夫。ちょっと、懐かしい人に会っただけ」

カイトは、弱々しく笑う。

「では、次は——」

アーカイブが、手を振る。

「二つ目の試練、『未来の覚悟』へ」

今度は、全員が光に包まれた。
