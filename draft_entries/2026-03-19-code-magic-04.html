第4章：予期せぬ来訪者

「......来る」

俺はエリナと並んでドアの前に立ち、深呼吸をした。さっきのゴブリン退治で少し自信がついたとはいえ、まだこの世界に来て間もない。何が起きるかわからない。

「開けますよ」エリナが言った。

ドアが開いた。

そこに立っていたのは、予想もしない人物だった。

銀色の髪を後ろで結んだ、青い瞳の女性。年齢は二十代後半。身につけているのは、上質な布で仕立てられた青いローブ。腰には鞘に入った細身の剣。その佇まいからは、ただ者ではない雰囲気が漂っていた。

「......誰？」エリナが小声で聞いた。

女性は静かに微笑んだ。

「初めまして。王都魔法院の調査官、セラフィナと申します」彼女は優雅にお辞儀をした、「あなたがカイトさんですね？」

「......王都魔法院？」俺は眉をひそめた。

王都魔法院。この国で魔法を管理し、魔道士を育成する機関だと聞いたことがある。そんな組織の調査官が、なぜこんな田舎の小さな店に？

「突然のご訪問、失礼いたします」セラフィナは続けた、「実は、あなたの噂を聞きまして」

「......噂？」

「ええ」彼女は頷いた、「三日で十体以上のゴブリンを討伐した新人魔道士。しかも、珍しい魔法の使い方をするとか」

......噂になるの、早すぎないか？

「......どちらでその情報を？」俺は警戒心を崩さずに聞いた。

「ギルドの報告書です」セラフィナは平然と答えた、「王都魔法院は、ギルドと情報共有を行っています。特に、力のある魔道士については詳細な記録を」

......なるほど。ギルドの報告書が王都にまで届くのか。

「それで、何の用ですか？」俺は単刀直入に聞いた。

セラフィナは少し笑った。

「単刀直入ですね。気に入りました」彼女は懐から一枚の羊皮紙を取り出した、「王都魔法院から、あなたに正式な依頼があります」

俺は羊皮紙を受け取った。そこには複雑な魔法陣と、何行もの文字が書かれている。

「......内容は？」

「遺跡の調査です」セラフィナは説明した、「国境付近に、古代の遺跡が見つかりました。内部には強力な魔法障壁があり、普通の魔道士では侵入できません。しかし、あなたの魔法なら、もしかしたら......」

俺は羊皮紙を詳しく見た。依頼内容は以下の通りだった：

  • 依頼内容： 古代遺跡「星落ちる塔」の調査
  • 場所： 王国北東部、国境付近
  • 目的： 内部の魔法障壁の解除、および遺跡の安全確認
  • 報酬： 成功報酬 100枚金貨
  • 期間： 二週間以内
  • 備考： 危険度高、王都魔法院が全面支援

......100枚金貨。ゴブリン退治の報酬が50枚銀貨だったから、単純計算で二十倍以上だ。

「......かなり大きな依頼ですね」俺は言った。

「ええ」セラフィナは真剣な表情で頷いた、「ただし、危険もそれなりです。過去に三人の魔道士が挑み、全員失敗しました。一人は命を落としました」

......命を落とすレベルか。

「なぜ、俺なんです？」俺は聞いた。

「あなたの魔法が、既存の魔法体系とは異なるからです」セラフィナは言った、「通常の魔道士は、魔力を直接操作して魔法を発動させます。しかし、あなたの魔法は......どう言えばいいのか、論理的で、体系化されているように見えます」

......鋭い。

「もしよろしければ、あなたの魔法について詳しくお聞きしたいのですが」セラフィナは言った。

俺は少し考えた。この世界で、自分の能力を他人に明かすのは危険かもしれない。でも、王都魔法院という組織を敵に回すのも得策ではない。

「......簡単に説明します」俺は言った、「俺の魔法は、言葉を使って構築します。魔法の効果を、論理的に組み立てるんです」

「言葉で構築？」セラフィナは興味深そうに聞いた。

「ええ。例えば......」俺は法杖を取り出し、小さな光球を作った、「この光球を作るには、『光を作れ』と念じるだけでいい。でも、俺の場合は、光の明るさ、色、持続時間などを、言葉で指定します」

俺は実際に、心の中でコードを書いた：


light = create_light({brightness: 50, color: "white", duration: 60})
activate(light)

手のひらに、純白の光球が浮かび上がった。

「......なるほど」セラフィナは感心したように頷いた、「非常に精密な制御ですね。普通の魔道士には、これほど細かい指定はできません」

「その代わり、詠唱には時間がかかります」俺は光球を消した、「実戦では、事前に準備できるかどうかが勝負です」

「......面白い」セラフィナは目を輝かせた、「ぜひ、詳細な研究をさせていただきたい」

......研究対象にされるのは勘弁してほしい。

「依頼については、少し考えさせてください」俺は言った。

「もちろんです」セラフィナは頷いた、「三日間考えます。ただし、他にも候補がいますので、早めの回答をいただければ」

彼女は名刺を置いて、店を出て行った。

「......すごい人だったね」エリナが言った。

「ああ」俺は頷いた。

「受けるの？」エリナが聞いた。

「......わからない」俺は正直に言った、「報酬は魅力的だけど、危険も高い。それに、王都魔法院と関わることになる」

「王都魔法院って、そんなに怖いの？」エリナが首をかしげた。

「組織だからな」俺は言った、「俺の能力を知られれば、どう扱われるかわからない。研究材料にされるかもしれないし、政治的な道具に使われるかもしれない」

エリナは少し考えた。

「でも、カイトなら大丈夫だよ」彼女は言った、「カイトは賢いし、強い。それに、私がついてるから！」

......頼もしい。

その夜、俺は一人で考えた。

この世界で生きていくには、力が必要だ。そして、力を得るには、リスクを取らなければならない。

......コードで魔法を書ける能力。この能力を活かせば、もっと強くなれるはずだ。

俺は決めた。

......受ける。

翌朝、俺はセラフィナに連絡を取った。名刺に書かれた連絡先を使う。

「......受けることにしました」俺は言った。

「素晴らしい」セラフィナの声は明るかった、「では、詳細を説明します。明日、王都魔法院の出張所で打ち合わせを行います。場所は......」

打ち合わせの場所は、街の中心部にある古い建物だった。王都魔法院の出張所は、外見からは普通の商店にしか見えない。

中に入ると、数人の魔道士が働いているのが見えた。皆、青いローブを身につけている。

「ようこそ」セラフィナが迎えてくれた、「こちらが、今回の調査メンバーです」

彼女が紹介したのは、二人の人物だった。

一人は、大柄な男。筋肉質で、背は二メートル近い。茶色の髪と髭、目は鋭い。身につけているのは、厚い革鎧。

「リアムだ」男は短く言った、「前衛担当。よろしく」

......リアム。前に聞いた名前だ。エリナが言っていた、腕利きの冒険者。

もう一人は、小柄な女性。黒い髪をショートカットにしている。目は大きく、猫みたいだ。身につけているのは、軽装のローブ。

「ミオです」女性はにっこり笑った、「後衛支援と、侦查担当です。よろしくお願いしますね」

「カイトです」俺は自己紹介した、「魔法担当です」

「魔法担当？」リアムが眉をひそめた、「どんな魔法を使うんだ？」

「......特殊な魔法です」俺は曖昧に答えた。

「まあまあ」セラフィナが間に入った、「詳細は現地で確認しましょう。まずは、遺跡について説明します」

彼女は地図を広げた。

「『星落ちる塔』は、王国北東部の山岳地帯にあります」彼女は指で場所を示した、「高度は約二千メートル。周囲は険しい山道で、徒歩で四日間かかります」

「四日間......」俺は呟いた。

「途中、魔物の群生地帯を通ります」セラフィナは続けた、「野営も必要です。装備は万全にしてください」

俺は頷いた。準備が必要だ。

「それと、これを持っていってください」セラフィナは小さな水晶を渡してきた、「通信装置です。王都魔法院と連絡が取れます」

俺は水晶を受け取った。冷たい感触が手のひらに伝わる。

「......わかりました」

「出発は明後日です」セラフィナは言った、「それまでに準備を整えてください。何か必要なものがあれば、王都魔法院が負担します」

俺は少し考えた。

「......一つ、お願いがあります」

「何でしょう？」

「エリナも連れて行きたいんです」俺は言った。

セラフィナは少し驚いたようだった。

「パートナーの方ですか？」

「......まあ、そんなところです」

セラフィナは少し考えた。

「......わかりました。ただし、危険は本人が理解しているという前提で」

「もちろんです」

その日の午後、俺はエリナに事情を説明した。

「......危険だよ」俺は言った。

「知ってる」エリナは真剣な表情で頷いた、「でも、カイトを一人で行かせたくない。それに、私も役に立ちたい」

「......役に立つ？」

「うん」エリナは胸を張った、「料理もできるし、応急処置もできる。荷物持ちもできるよ！」

......確かに、長旅にはサポートが必要だ。

「......わかった」俺は頷いた、「でも、無理はしないでくれ」

「うん！」エリナは嬉しそうに笑った。

その夜、俺は店の二階で準備を始めた。

必要なものをリストアップする：

  • 食料（五日分）
  • 水（魔法で浄化できるから最小限）
  • 寝袋
  • テント
  • 応急処置キット
  • 予備の法杖
  • 護符（防御用）
  • 照明用具
  • 地図とコンパス

......結構な量だ。

俺はコードで荷物を整理するプログラムを書いた：


inventory = {
  food: 5,
  water: 2,
  sleeping_bag: 1,
  tent: 1,
  first_aid: 1,
  staff: 2,
  talisman: 3,
  light: 1,
  map: 1,
  compass: 1
}

weight = calculate_weight(inventory)
console.log("Total weight: " + weight + " kg")

......全部で三十キログラムくらいか。重いな。

翌日、俺とエリナ、リアム、ミオの四人は、街の北門で集合した。

「全員揃ったな」リアムが言った。「出発するぞ」

俺たちは北への道を歩き始めた。

空は青く、雲が少ない。いい天気だ。

......新しい冒険が始まる。

山道は予想以上に険しかった。

道幅は狭く、両側は崖。足元は不安定で、一歩踏み外せば転落だ。

「気をつけて」ミオが前の方で言った、「この辺りは、岩獣の生息地です」

「岩獣？」俺は聞いた。

「岩に擬態する魔物です」ミオは説明した、「普段は動かないけど、獲物が近づくと襲ってきます。気づいた時には、遅いです」

......嫌な敵だ。

「俺が前衛だ」リアムが大剣を構えた、「お前らは後ろで支援してくれ」

俺は頷いた。

......コードで侦查できるか？

俺は心の中でコードを書いた：


sensor = create_sensor("life", {range: 50})
data = scan(sensor)
for entity in data:
  if entity.type == "rock_beast":
    console.log("Alert: Rock beast at " + entity.position)

......反応あり。三時の方角、距離十メートル。

「リアム、右前方、十メートル」俺は低声で言った。

リアムは一瞬で反応した。大剣を構え、右前方を警戒する。

......一秒後、岩が動いた。

岩獣だ。四足歩行で、全長は二メートルほど。体は岩で覆われている。

「来るぞ！」リアムが叫んだ。

岩獣が飛びかかってきた！

リアムは大剣を振り下ろした。金属音が響く。岩獣の外殻は硬い。

「ミオ！ 支援を！」リアムが叫んだ。

「はい！」ミオは杖を振った。淡い光がリアムを包む。身体強化の魔法だ。

俺もコードを書いた：


target = get_nearest_enemy()
weak_point = analyze(target)
console.log("Weak point: " + weak_point.position)

......弱点は、腹部の柔らかい部分。でも、四足歩行で地面に近い。狙いにくい。

「リアム！ 腹を狙って！」俺は叫んだ。

「わかった！」リアムは踏み込み、大剣を斜めに振った。岩獣は避けようとしたが、間に合わない。剣が腹に命中した！

岩獣が悲鳴を上げて倒れた。

「......やった」エリナが安堵の息を吐いた。

「連携、悪くないな」リアムが笑った、「お前、いい勘してる」

「......ありがとう」俺は小さく笑った。

その日の夜、俺たちは野営した。

テントを張り、焚き火を起こす。エリナが料理を担当した。干し肉と野菜のスープだ。

「美味しい」ミオが言った、「エリナさん、料理上手ですね」

「えへへ、ありがとう」エリナは照れた。

俺は焚き火を見つめながら、これからのことを考えた。

......遺跡の魔法障壁。普通の魔道士では解除できない。

でも、コードならどうだろう？

俺は心の中で、障壁解除のプログラムをシミュレートした：


barrier = detect_barrier()
structure = analyze(barrier)
console.log("Barrier structure: " + structure)

weak_points = find_weak_points(structure)
console.log("Weak points: " + weak_points)

for point in weak_points:
  disrupt(point)
  
if barrier.status == "disabled":
  console.log("Barrier disabled!")

......理論上はいけるはず。でも、実際にどうなるかは、現地で確認するしかない。

「カイト、どうしたの？」エリナが心配そうに聞いた。

「......考えてた」俺は言った、「魔法のこと」

「大丈夫だよ」エリナは俺の肩に手を置いた、「カイトなら、きっとできる」

......頼もしい。

俺は星空を見上げた。

......明日も、山道が続く。そして、四日後には遺跡に到着する。

そこで、何が待っているのか。

......わからない。でも、進むしかない。

夜が更けていった。

焚き火がパチパチと音を立てる。遠くで、獣の声が聞こえる。

......新しい冒険が、始まろうとしていた。
