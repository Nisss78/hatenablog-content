第61章：次元の侵蝕

「侵蝕」という言葉が、重く響いた。カイトは喉が渇くのを感じた。プログラマーとしての直感が、その言葉の意味する深刻さを告げていた。

「次元の侵蝕……具体的に何が起きているんだ」

マルクスは壁に刻まれた古代の図面を指し示した。そこには、球体の周囲を黒い霧が取り囲む図が描かれている。

「次元の海は、混沌とした魔力の奔流だ。無数の可能性が交錯し、法則のない領域。この世界は、その海を渡るための船として設計された。しかし、船には限界がある」

マルクスが図面の一部をなぞる。黒い霧が球体の表面に浸食している様子が、詳細に描かれていた。

「長い旅の中で、次元の海の影響が浸透し始めた。最初は微小な亀裂。誰も気づかないほどの違和感。だが、それは徐々に広がり、今では世界の根幹を揺るがす脅威となった」

セシリアが鋭い声を上げた。

「それって、私たちが戦ってきた『魔物』の増加や、魔力の乱れも——」
「その通りだ。近年、異常気象が増え、凶暴な魔獣が出現し、魔法が不安定になっているだろう？あれはすべて、次元の侵蝕の影響だ」

エリナが震える声で尋ねた。

「じゃあ、このままじゃ……世界はどうなるの」

マルクスは静かに首を振った。

「侵蝕が臨界点に達すれば、次元の海に飲み込まれる。世界という概念そのものが崩壊し、混沌の中に還るのだ」

重い沈黙が降りた。リアムが拳を握りしめる。

「それを防ぐために、カイトの能力が必要だってことか」
「正確には、カイトの能力と、創造主が残した遺産の両方が必要だ」

マルクスは部屋の奥へと歩き出した。そこには、今まで気づかなかった隠し扉があった。古代の石材で作られたその扉には、複雑な魔法陣が刻まれている。

「アーキテクトは、この危機に対処するためのツールを残した。『根源のコード』と呼ばれる、世界のコアシステムに直接アクセスできる端末だ」

カイトは息を呑んだ。世界のコアシステムに直接アクセス——それは、プログラマーにとって究極の権限だ。OSのカーネルに触れるような、システムの根本を操作できる力。


// Emergency Access Protocol
// Requires: Administrator Privilege + Physical Key

function accessRootCore(admin: Administrator, key: PhysicalKey): RootAccess {
    if (admin.hasPrivilege("ROOT") && key.isAuthentic()) {
        return new RootAccess(worldSystem);
    } else {
        throw new AccessDeniedError();
    }
}

カイトの脳内で、アクセスプロトコルが可視化される。管理者権限は自分にある。しかし、「物理的な鍵」とは何だ。

「鍵はどこにあるんだ」

マルクスは扉に手をかざした。魔法陣が青白く輝き、扉が音もなく開いた。その奥には、小さな台座があった。台座の上には、一本の杖が置かれている。

「これが、根源のコードへの鍵だ」

杖は、一見すると普通の魔導杖に見えた。だが、カイトの目には異なるものが映った。杖の内部に、複雑な回路が埋め込まれている。魔力を伝導させるだけでなく、情報を処理し、コードを実行する機能を持った——生きた端末だ。

「ヴェリディアンの魔導書と対になる存在。創造主が残した、最後の遺産」

カイトは杖に手を伸ばした。指先が触れた瞬間、電流のような感覚が走った。情報が彼の脳内に流れ込んでくる。

カイトの視界に、半透明のインターフェースが浮かび上がった。それは、彼が元の世界で使っていたIDE（統合開発環境）に似ていた。コードエディタ、デバッガ、コンパイラ——すべてが魔力で構成された、魔法の開発環境だ。

「これが……根源のコードへの入り口」

カイトは杖を握りしめた。その瞬間、彼の中で何かが変わった。これまで感じていた、世界との違和感が消えた。代わりに、世界と一体化するような感覚が広がる。

「カイト、顔色が変わったぞ」

リアムが心配そうに声をかけた。

「大丈夫……むしろ、これまで以上に調子がいい」

カイトは杖を振った。空中に、光の文字が浮かぶ。


// Diagnostic Scan
// Target: World System
// Status: CRITICAL
// Erosion Level: 73.2%
// Time to Collapse: 47 days

    const erosion = this.detectErosion();
    const stability = this.calculateStability();
    
    return {
        erosionLevel: erosion.percentage,
        stabilityIndex: stability.index,
        estimatedDaysRemaining: this.projectTimeline(erosion)
    };
}

数字がカイトの目に飛び込んでくる。侵蝕レベル73.2パーセント。崩壊まで47日。絶望的な数字だ。だが、カイトは笑った。

「47日あれば、十分だ」

エリナが驚いた顔をした。

「十分って……カイト、何をするつもり？」

カイトは杖を肩に担ぎ、仲間たちを見渡した。

「バグを修正するんだ。この世界の、最大のバグを」

古代遺跡を出たカイトたちは、マルクスの案内で移動を始めた。目的地は、世界の「核」と呼ばれる場所だ。根源のコードにアクセスするには、物理的にコアに近づく必要があるらしい。

「核は、世界の中心にある。そこへ行くには、いくつかの試練を越えなければならん」

マルクスが地図を広げた。そこには、複雑なルートが描かれている。

「まずは『静寂の森』を抜け、『鏡の湖』を渡り、『時の狭間』を通過する。いずれも、次元の侵蝕の影響を受けた場所だ。危険度は高い」

セシリアが眉をひそめた。

「次元の侵蝕の影響って、具体的にどうなるの？」
「現実が不安定になる。空間が歪み、時間が狂い、魔力が暴走する。普通の人間なら、精神が耐えられずに崩壊するだろう」

リアムが剣の柄に手をやった。

「俺たちは普通じゃないさ。なあ、カイト」

カイトは頷いた。

「ああ。それに、俺にはこの杖がある。侵蝕の影響をある程度中和できるはずだ」

カイトは杖を振り、防御の魔法を展開した。半透明のドームが一行を包み込む。

杖の先から、青白い光が広がった。それは、次元の海の混沌を退ける、秩序の光だ。

「行こう。47日しかない」

カイトの声に、力が宿っていた。仲間たちは顔を見合わせ、そして頷いた。

静寂の森は、その名の通り静かだった。虫の声も、鳥のさえずりも、風の音さえも聞こえない。ただ、不気味な静寂が支配している。

「これが、次元の侵蝕の影響か」

エリナが小声で呟いた。

「音が消滅している。振動という概念自体が、ここでは弱まっているんだ」

カイトが杖を掲げると、空中に解析結果が浮かんだ。


// Location Analysis: Silent Forest
// Vibration Frequency: 0.003 Hz (almost null)
// Mana Density: 12.4 (unstable)

    return {
        vibrationFrequency: this.measureVibration(),

空間歪曲レベル2。まだ危険域ではないが、油断はできない。

「みんな、離れないで。空間が不安定だから、簡単にはぐれるぞ」

カイトの警告通りだった。数歩横にそれただけで、景色が歪み、仲間の姿がぼやける。リアムが慌てて戻ってきた。

「おい、これヤバいな。一瞬、俺がどこにいるのか分からなくなった」
「空間が折りたたまれているんだ。直線距離でも、実際には何倍もの距離がある可能性がある」

マルクスが先頭を歩きながら説明した。

「かつて、この森で行方不明になった探求者は数知れん。空間の迷宮に閉じ込められ、永遠に彷徨うことになった者もいる」

セシリアが杖を構えた。

「カイト、ナビゲーションできる？」

カイトは杖を地面に突き刺した。青白い光が、地面に沿って進むべき道を描いていく。


// Pathfinding Algorithm
// Using spatial anchors to maintain cohesion

function findPath(destination: Coordinates): Path {
    const anchors = this.detectSpatialAnchors();
    const stableRoute = this.calculateStablePath(anchors);
    
    return new Path({
        route: stableRoute,
        markers: this.generateVisualMarkers()
    });
}

光の道が、森の奥へと続いていた。カイトたちは、その道を歩き出した。

数時間後、静寂の森を抜けた一行は、鏡の湖のほとりに立っていた。

湖の水面は、鏡のように完璧に周囲を映していた。だが、映っている景色は、現実とは微妙に異なっていた。木々の位置が逆、空の色が違う、そして——水面に映る自分たちの姿が、別人のように見えた。

「これが、鏡の湖」

エリナが水面を覗き込んだ。そこに映る彼女は、冷ややかな目でこちらを見つめ返していた。

「エリナ、見るな」

カイトが彼女の肩を引いた。

「この湖は、『もう一人の自分』を映し出す。向こう側の世界の、可能性の分岐した自分だ」

水面に映る「もう一人のエリナ」が、不気味に微笑んだ。そして、水面から手が伸びてきた。

「来なさい……こっちへ」

鏡のエリナの声は、本物のエリナと同じだが、どこか歪んでいた。

「みんな、水面を見るな！前だけ見て進め！」

カイトが叫び、杖で光の壁を作った。鏡のエリナの手が壁に触れ、弾かれた。

カイトたちは、水面を見ないように慎重に湖を渡り始めた。だが、水面からは次々と手が伸び、誘惑の囁きが聞こえてくる。

「カイト……君も知りたいだろう？自分の可能性を」

鏡のカイトが囁いた。

「君がこの世界に来なかったら、何があったのか。君の元の世界で、何が起きているのか」

カイトは一瞬、足を止めた。鏡のカイトの言葉は、彼がずっと心の奥で考えていたことだった。元の世界での家族、友人、仕事——すべてを置いてこの世界に来た。本当に、正しい選択だったのか。

「カイト！」

エリナの声で、カイトは我に返った。

「見ちゃダメ！前を見て！」

カイトは深呼吸をした。そして、鏡の自分に向かって言った。

「俺の可能性は、俺が決める。お前に教えてもらう必要はない」

鏡のカイトは、一瞬驚いた顔をした。そして、ゆっくりと水面の下へと沈んでいった。

湖を渡りきった一行は、安堵の息を漏らした。

「次元の侵蝕は、精神にも影響を与える。自分自身との対話を強制し、迷いを増幅させるんだ」

マルクスが説明した。

「だが、君たちは強い。普通なら、あそこで心を折られる」

カイトは杖を握りしめた。鏡の湖での体験は、衝撃だった。だが、同時に確信にもなった。自分は、この世界で戦うと決めた。その意志は、何があっても揺るがない。

「次は、時の狭間だな」

リアムが前を指差した。そこには、空間が裂けたような奇妙な亀裂が浮かんでいた。

「あれが、核への最後の関門か」

カイトは覚悟を決めて、亀裂へと歩き出した。仲間たちが続く。47日というタイムリミットが、彼らの背中を押していた。
