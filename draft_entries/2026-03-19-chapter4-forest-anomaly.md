<h3>第4章：森の異変</h3>

<p>翌朝、カイトは宿屋の窓から差し込む光で目を覚ました。隣のベッドではリアムがまだぐっすりと眠っている。昨夜の遅くまで魔法理論について議論していたせいだろう。</p>

<p>カイトは静かに身支度を整え、一階の食堂へ降りた。エリナがすでにテーブルについており、何やら真剣な表情で地図を広げていた。</p>

<blockquote><p>「おはよう、カイト。ちょうどよかった」</p></blockquote>

<p>エリナが顔を上げ、地図のある一点を指し示す。</p>

<blockquote><p>「依頼の現場だ。西の『霧の森』——最近、奇妙な現象が起きているらしい」</p></blockquote>

<p>カイトは地図を覗き込んだ。街から半日ほどの距離に、緑色で塗られた広大な森林地帯があった。</p>

<blockquote><p>「奇妙な現象って？」</p></blockquote>

<blockquote><p>「森に入った者が戻ってこない。そして……夜になると、森全体が青白く光るらしい」</p></blockquote>

<p>カイトは眉をひそめた。この世界の魔法についてまだ詳しくないが、普通の現象ではないことだけは理解できた。</p>

<blockquote><p>「行商人が森の近くで変な怪物を見たという報告もある。ギルドは調査を優先度高めで依頼している」</p></blockquote>

<p>エリナは地図を畳み、テーブルに置いてあったパンをかじった。</p>

<blockquote><p>「報酬は金貨五十枚。初心者向けの依頼にしては破格だ」</p></blockquote>

<p>その言葉に、カイトは違和感を覚えた。</p>

<blockquote><p>「危険ってことか」</p></blockquote>

<blockquote><p>「かもしれない。でも、私たちにはカイトがいる」</p></blockquote>

<p>エリナはいたずらっぽく笑った。</p>

<blockquote><p>「その魔法、見せてもらうわよ」</p></blockquote>

<hr>

<p>準備を終えた三人は、街の西門を出て森への道を歩き始めた。リアムは背中に大きな剣を背負い、エリナは腰に短剣を二本差している。対照的に、カイトは革の鞄一つだけだった。</p>

<blockquote><p>「カイト、武器は持たないのか？」</p></blockquote>

<p>リアムが不思議そうに聞く。</p>

<blockquote><p>「これが僕の武器だ」</p></blockquote>

<p>カイトは自分の頭を指差した。</p>

<blockquote><p>「魔法を書くのに、剣は必要ないからね」</p></blockquote>

<p>リアムは首をかしげたが、それ以上は聞かなかった。昨夜の魔法の実演で、カイトの力が本物であることは理解しているらしい。</p>

<p>道は次第に荒れていった。街を出て二時間ほどで、周囲の景色が変わり始める。背の高い木々が増え、空気が湿り気を帯びてきた。</p>

<blockquote><p>「霧が出てきた」</p></blockquote>

<p>エリナが警告するように言った。視界が白く濁り始めている。</p>

<blockquote><p>「この霧、普通じゃないわ。魔力を感じる」</p></blockquote>

<p>カイトは目を細めて霧を観察した。彼の視界には、霧の中に微細な光の粒子が見えた。プログラマーとしての彼の感覚が、その光の動きを「コード」として捉え始める。</p>

<pre><code>
// 霧の構造を解析
Fog = {
  type: "magical",
  density: 0.7,
  manaSource: "unknown",
  behavior: function(target) {
    return disorient(target);
  }
}
</code></pre>

<p>カイトは無意識のうちに、霧の性質を頭の中でコード化していた。この世界に来てから強まった、彼だけの特殊能力だ。</p>

<blockquote><p>「何か分かる？」</p></blockquote>

<p>エリナの問いに、カイトは頷いた。</p>

<blockquote><p>「この霧は、魔力で作られている。何かが——誰かが、意図的に霧を発生させているんだ」</p></blockquote>

<blockquote><p>「誰かが？」</p></blockquote>

<p>リアムが剣の柄に手をかけた。</p>

<blockquote><p>「迷彩の魔法か、罠か……どっちにしろ、歓迎されていないことは確かだな」</p></blockquote>

<hr>

<p>森の入り口に立った三人は、慎重に足を進めた。霧はさらに濃くなり、五メートル先も見えなくなっている。</p>

<blockquote><p>「カイト、道案内できる？」</p></blockquote>

<p>エリナの声が緊張を帯びていた。</p>

<blockquote><p>「やってみる」</p></blockquote>

<p>カイトは目を閉じ、集中した。彼の意識が霧の中の魔力パターンをトレースしていく。複雑に入り組んだ光の糸——それが、この空間を覆う魔法の「プログラム」だった。</p>

<pre><code>
// 魔法の構造を可視化
function revealPath(fog) {
  let manaFlow = fog.traceMana();
  let weakPoints = manaFlow.findGaps();
  
  return weakPoints.map(point => {
    return createPath(point);
  });
}
</code></pre>

<p>カイトは右手を前に突き出した。</p>

<blockquote><p>「<code>navigate(fog).reveal()</code>」</p></blockquote>

<p>彼の声とともに、霧の中に淡い光の筋が浮かび上がった。三人の前方に、道しるべのように光が続いている。</p>

<blockquote><p>「すごい……」</p></blockquote>

<p>リアムが息を呑んだ。</p>

<blockquote><p>「霧の魔力の流れを利用して、安全な道を見つけたんだ。ついてきて」</p></blockquote>

<p>カイトを先頭に、三人は光の道を進んだ。霧は彼らの周りだけで薄く、外の景色が見える不思議な空間ができていた。</p>

<hr>

<p>森の中を進むこと三十分。突然、カイトが足を止めた。</p>

<blockquote><p>「どうした？」</p></blockquote>

<p>リアムが剣を抜き、周囲を警戒する。</p>

<blockquote><p>「ここだ。魔力の源は、この近くにある」</p></blockquote>

<p>カイトは霧の向こうを指差した。光の粒子が、一点に向かって収束している。</p>

<blockquote><p>「誰かが魔法を維持している。その制御点まで、あと少し」</p></blockquote>

<p>エリナが短剣を構え、カイトの横に立った。</p>

<blockquote><p>「準備はいい？」</p></blockquote>

<p>三人は頷き合い、霧の中へ踏み出した。</p>

<p>開けた場所に出た瞬間、カイトは息を呑んだ。</p>

<p>そこには、巨大な水晶が地面から生えるように鎮座していた。高さは五メートルほど。青白い光を放ち、周囲の霧を生み出しているのが分かる。</p>

<p>そして、水晶の根元に——一人の少女が座っていた。</p>

<blockquote><p>「誰だ？」</p></blockquote>

<p>リアムの鋭い声に、少女が顔を上げた。銀色の髪、透き通るような肌、そして——猫のような耳。彼女は亜人だった。</p>

<blockquote><p>「……人間」</p></blockquote>

<p>少女は警戒した目で三人を見た。</p>

<blockquote><p>「来ないで。殺すわよ」</p></blockquote>

<p>その瞬間、水晶の光が激しく明滅した。霧が濃くなり、三人の視界を奪おうとする。</p>

<blockquote><p>「カイト！」</p></blockquote>

<p>エリナの叫び声。しかしカイトは動じなかった。彼の目には、少女が水晶に流す魔力の「コード」が見えていた。</p>

<pre><code>
// 水晶の魔法を解析
Crystal = {
  power: 8000,
  output: function() {
    return generateFog(this.power);
  },
  controller: girl,
  emotion: "fear"
}
</code></pre>

<p>カイトは理解した。この少女は、自分を守るために霧を作り出しているのだ。</p>

<blockquote><p>「待って！僕たちは敵じゃない！」</p></blockquote>

<p>カイトは両手を挙げ、敵意がないことを示した。</p>

<blockquote><p>「その水晶——君を守るためのものだろう？襲ったりしない」</p></blockquote>

<p>少女は疑わしげにカイトを見つめた。</p>

<blockquote><p>「嘘。人間はいつも嘘をつく」</p></blockquote>

<blockquote><p>「嘘じゃない。僕は君の魔法を見た——必死に、誰かから逃れようとしている」</p></blockquote>

<p>カイトの言葉に、少女の目がわずかに揺れた。</p>

<blockquote><p>「……なんで分かるの？」</p></blockquote>

<blockquote><p>「僕にも、特別な力があるんだ。魔法を——読めるんだよ」</p></blockquote>

<p>カイトは一歩踏み出した。エリナとリアムが緊張する中、彼は少女に近づいていく。</p>

<blockquote><p>「名前、教えてくれる？僕はカイト」</p></blockquote>

<p>少女は長い沈黙の後、小さく口を開いた。</p>

<blockquote><p>「……ミウ」</p></blockquote>

<blockquote><p>「ミウ。きれいな名前だね」</p></blockquote>

<p>カイトが微笑むと、ミウの猫耳がぴくりと動いた。警戒心が、少しだけ和らいだように見えた。</p>

<hr>

<p>三人はミウから話を聞いた。彼女は森の奥にある亜人の集落の出身で、数日前に集落が何者かに襲われたという。</p>

<blockquote><p>「黒い鎧を着た人たち……集落を焼いて、みんなを連れて行った」</p></blockquote>

<p>ミウの声が震える。</p>

<blockquote><p>「私は逃げて、この水晶を見つけたの。魔力が強くて……霧を作れば、見つからないと思って」</p></blockquote>

<p>エリナが深刻な表情でカイトを見た。</p>

<blockquote><p>「黒い鎧……奴隷商人か、それとも——」</p></blockquote>

<blockquote><p>「王国軍の可能性もある」</p></blockquote>

<p>リアムが忌々しげに言った。</p>

<blockquote><p>「亜人は差別されている。不法に捕らえて、労働力として売る連中がいるんだ」</p></blockquote>

<p>カイトは拳を握った。この世界の暗部——彼はまだ何も知らないが、目の前の少女の悲しみは本物だった。</p>

<blockquote><p>「ミウ、君の集落の人たちを助けたいんだけど、場所は分かる？」</p></blockquote>

<p>ミウは頷いた。</p>

<blockquote><p>「北へ半日……でも、危険よ」</p></blockquote>

<blockquote><p>「大丈夫。僕たちがいる」</p></blockquote>

<p>カイトは水晶を見上げた。</p>

<blockquote><p>「それに、この水晶も使える。霧を作るだけじゃなく——もっと色々なことができるはずだ」</p></blockquote>

<p>カイトの目は、すでに水晶の「ソースコード」を解析していた。これは単なる霧発生装置ではない。魔力を増幅し、様々な形に変換できる——汎用性の高い魔法デバイスだ。</p>

<pre><code>
// 水晶の潜在的機能を探索
Crystal.capabilities = [
  "fog_generation",
  "mana_amplification", 
  "barrier_creation",
  "healing_aura",
  "summon_spirit" // 未実装？
]
</code></pre>

<p>カイトは微笑んだ。</p>

<blockquote><p>「ねえミウ。僕と一緒に、この水晶の『デバッグ』をしない？君の魔力と、僕のコードがあれば——すごいことができるかもしれない」</p></blockquote>

<p>ミウはきょとんとした表情でカイトを見た。「デバッグ」という言葉の意味は分からないようだが、カイトの目にある真剣さは伝わったらしい。</p>

<blockquote><p>「……信じて、いいの？」</p></blockquote>

<blockquote><p>「ああ。約束する」</p></blockquote>

<p>カイトは手を差し出した。ミウは少し躊躇った後、その手を握り返した。</p>

<p>彼女の手は冷たかったが、その瞳には微かな希望の光が宿っていた。</p>

<hr>

<p>四人は森の奥へと進み始めた。カイトの前に立つ新たな道——それは、この世界の魔法とコードが出会う、冒険の始まりだった。</p>

<p><strong>（第4章　了）</strong></p>
