第50章：起源の扉

破壊の余韻が、まだ空気中に漂っていた。

カイトは、目の前に広がる光景を呆然と見つめていた。かつて堅牢だったギルドの地下訓練場が、今は瓦礫の山と化している。壁には亀裂が走り、天井の一部が崩れ落ちて、光が差し込んでいた。

「……これ、僕がやったのか？」

自分の両手を見る。震えていた。魔力を使い果たした反動ではない。恐怖だ。

先ほどの戦い——収束者マルクスとの激突——で、カイトは自身の中に眠る「破壊の力」を解放した。だが、その力は彼が想像していたものを遥かに超えていた。制御不能の、暴走するエネルギー。

「正確には、あんたの中にある『古の因子』が暴走したんだ」

リアムが、瓦礫を乗り越えて近づいてきた。彼もまた、傷だらけだ。左腕を押さえているのが痛々しい。

「マルクスの収束能力——時空を歪める力——に刺激されたんだろう。あんたの体内にある古代の魔術回路が、共鳴しちまった」

カイトは記憶を辿った。マルクスが放った最後の一撃。時空を裂く、圧倒的な破壊光。それに対抗するために、カイトは無意識にコードを紡いだ。

class DestructiveResonance {
    source: "ancient_factor";
    trigger: "survival_instinct";
    power: Infinity;
    
    activate() {
        return this.source.unleash({
            target: "all_threats",
            radius: "expansion"
        });
    }
}

そのコードは、彼が意図して書いたものではなかった。体が勝手に動いた。魂の奥底から、何かが溢れ出したのだ。

「カイト、怪我はない？」

エリナが、心配そうに駆け寄ってくる。彼女の服は煤で汚れているが、大きな怪我はないようだ。安堵する。

「ああ、大丈夫。ただ……少し、自分が怖くなった」
「怖い？」
「あの中にある力が、僕の制御を超えてる。いつか、本当に暴走して、みんなを傷つけるかもしれない」

エリナは、カイトの手を握った。その手は温かく、少しだけ震えていた。

「カイトは暴走しないよ。だって、カイトはカイトだもん。古代の因子だって、カイトの一部でしょ？ きっと、ちゃんと制御できるようになる」

simplistic な励ましだが、不思議と心に響いた。カイトは、微かに笑った。

「……ありがとう、エリナ」
「ふふ、どういたしまして」

その時、瓦礫の奥から、うめき声が聞こえた。

「ぐっ……まだ、終わって、ない……」

マルクスだ。

収束者は、崩れた柱の下敷きになっていた。だが、まだ息がある。驚くべき生命力だ。

カイトは、エリナの手を離して、マルクスに近づいた。リアムとセシリアも続く。

「マルクス、まだ何か言いたいことがあるのか？」

カイトの問いに、マルクスは血を吐きながら笑った。

「お前……本当に、何も知らないんだな。この世界の、真実を」
「真実？」
「古代の魔術師……ヴェリディアンが、何のために力を求めたか。そして、彼が最後に何を残したか」

カイトの心臓が、早鐘を打った。ヴェリディアン——第34章で遭遇した、古代の魔術師の意志。彼の記憶の断片は、カイトの中にまだ残っている。

「……彼は、愛する者を守れなかった。その後悔から、力を求めた」
「半分だけ正解だ」

マルクスは、弱々しく首を振った。

「彼が守れなかったのは、たった一人の女性じゃない。……この世界、そのものだ」

カイトは息を呑んだ。

「どういう意味だ？」
「この世界は、滅びの危機に瀕している。古代から、ずっとだ。WorldCollapse——世界崩壊のプロセスは、すでに始まっている。ヴェリディアンは、それを止めるために力を求めた。だが、間に合わなかった」
class WorldCollapse {
    status: "in_progress";
    progress: 0.73; // 73%進行
    remainingTime: "unknown";
    
    cause: "dimensional_instability";
    
    // このプロセスは不可逆
    // 止めるには、起源に遡る必要がある
}

マルクスの言葉に伴って、カイトの脳内にコードが浮かんだ。これは彼が書いたものではない。ヴェリディアンから受け継いだ、古代の知識の一部だ。

「世界崩壊……？ 信じがたい話だな」

リアムが、懐疑的な表情で言った。

「この世界は、何百年も平和に続いてる。崩壊の兆候なんて、見たことも聞いたこともない」
「それは、守人たちが必死に隠してきたからだ」

マルクスは、苦しげに咳き込んだ。

「アルフレッド……守人の長……彼は知っている。真実を。そして、真実を隠し続けるために、多くのことを犠牲にしてきた」

アルフレッド。第7章で出会った、守人の長。カイトたちに古代の遺跡を案内してくれた人物だ。彼が、真実を隠している？

「何のために？」

セシリアが、恐る恐る尋ねた。

「パニックを避けるためだよ。世界が崩壊の危機にあると知れば、人々は絶望する。秩序が崩れる。だから、守人たちは真実を隠し、崩壊を遅らせるための小細工を続けてきた」
「小細工？」
「次元の裂け目を塞いだり、魔力の流入を調整したり……だが、対処療法に過ぎない。根本的な解決には、起源に遡る必要がある」

起源。

その言葉が、カイトの胸に突き刺さった。

「起源って……この世界が生まれた場所、ってことか？」
「そうだ。OriginGate——起源の扉。そこから、すべてが始まった。そして、そこから、すべてをやり直すことができる」

マルクスは、最後の力を振り絞って、懐から一枚の地図を取り出した。

「これが……起源の扉への道標だ。わたしは、それを開こうとして……力及ばなかった。だが、お前なら……」

マルクスの声が、弱々しくなっていく。

「お前の中にある古の因子は、起源の扉を開く鍵だ。ヴェリディアンの遺産……それを、正しく使ってくれ……」

そう言い残して、マルクスの意識が途切れた。彼は気絶したのか、それとも——。

「……死んだ、のか？」

セシリアが、震える声で尋ねた。リアムが、マルクスの首筋に指を当てる。

「いや、生きてる。ただ、かなり弱ってる。しばらくは動けないだろう」

カイトは、マルクスの手から地図を受け取った。古びた羊皮紙に、複雑な魔術陣と座標が記されている。

struct OriginGate {

    // カイトの古の因子が鍵の一部

カイトの脳内で、情報が整理されていく。起源の扉——そこは、この世界の理（ことわり）が集約された場所。すべての魔力の源であり、次元の安定を保つ要（かなめ）だ。

「マルクスの言ってることが本当なら……僕たちは、とんでもないことに巻き込まれてるってことだな」

カイトは、地図を見つめながら呟いた。

「世界崩壊を止めるために、起源の扉を開く。そのための鍵が、僕の中にある。でも、扉を開いたら何が起こる？ 世界を救えるのか、それとも……」
「それを確かめるのが、冒険でしょ？」

エリナが、明るい声で言った。

「カイト、悩んでも答えは出ないよ。まずは、アルフレッドさんに会おう。真実を聞いて、それから考えればいい」

カイトは、エリナの言葉にハッとした。そうだ。一人で抱え込む必要はない。仲間がいる。彼らと一緒に、進めばいい。

「……ありがとう、エリナ。君の言う通りだ」

カイトは、地図を慎重に畳んで、懐にしまった。

「リアム、セシリア、悪いけど、もう少し付き合ってくれるか？」
「もちろんよ。カイトさんを一人にするわけにはいかないもの」

セシリアが、微笑んだ。

「わたしも断るつもりはない。古代の魔術の謎、興味深いしな。それに、世界崩壊だなんて、放置できない」

リアムも、ニヤリと笑った。

「みんな……ありがとう」

カイトは、仲間たちに深く感謝した。そして、新たな決意を胸に、口を開いた。

「まずは、ここを出よう。それから、アルフレッドさんを探す。真実を聞き出して、起源の扉へ向かう。……それが、僕たちの次の目標だ」

四人は、瓦礫の山を後にした。

夕日が、空を茜色に染めていた。長い一日が、終わろうとしている。

だが、カイトの冒険は、まだ終わらない。むしろ、ここからが本当の始まりなのかもしれない。

起源の扉。世界の真実。そして、カイト自身の運命。

すべての鍵は、彼の中にある。
