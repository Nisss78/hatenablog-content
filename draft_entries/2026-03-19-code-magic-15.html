第15章：古の試練

契約の光が収まると、俺の目の前に透明なウィンドウが浮かんだ。


// Contract Status: ACTIVE
// Contractor: 海藤翔太（カイト）
// Contractee: ヴェリディアン（Ancient Guardian）
// Terms: 継承の試練を完了すること
// Reward: 古代魔法の完全なアクセス権
// Failure: 契約の破棄 ＋ 記憶の一部消去
「......失敗したら記憶消去って、どういうことだよ」俺は思わず呟いた。

ヴェリディアンの声が、頭の中に響いた。

『当然だろう。古代の知識は、選ばれし者にしか継承されない。資格なき者がその知識を持てば、世界を滅ぼすことになりかねん』
「それは......分かるけど」
『安心せい。お前なら通過できるはずだ。私の勘がそう告げている』

......頼りない勘だ。

試練の開始

俺の周りの風景が、一変した。

さっきまで遺跡の中にいたはずなのに、今は——

何もない空間。

白い霧が立ち込め、足元以外は何も見えない。

「ここは......？」
『精神世界だ』ヴェリディアンの声。「お前の魂の中にある、最も深い場所。ここで試練を行う」

精神世界。つまり、俺の心の中ってことか。

『第一の試練を開始する』

霧の中から、人影が現れた。

——俺自身だ。

「えっ......俺？」

もう一人の俺が、静かに立っていた。表情はなく、ただ俺を見つめている。

『これは「影」だ。お前の弱さ、恐れ、迷いが具現化した存在。これを倒せ』

影との戦い

俺は杖を構えた。


// Combat Initialization
const kaito = {
  hp: 100,
  mp: 80,
  spells: ["fireball", "shield", "light"]
};

const shadow = {
  hp: ???,
  mp: ???,
  abilities: "unknown"
};

// Strategy: Analyze first, then attack
analyze(shadow);

影が動いた。

——速い！

影は俺と同じ魔法を使ってきた。ファイアボール、シールド、ライト。

でも、俺より速い。正確だ。無駄がない。


// Analysis Result
// Shadow mirrors my abilities
// But faster and more efficient
// Conclusion: Cannot win in direct combat
「どうすれば......！」

影のファイアボールが俺を襲った。シールドで防ぐが、衝撃で後ろに飛ばされる。


kaito.hp -= 15;
// HP: 85/100

このままじゃ勝てない。

俺より強い俺に、どうやって勝つ？

答え

俺は考えた。

影は俺の「弱さ」の具現化だ。つまり——

俺が自分の弱さを受け入れれば、影は消えるんじゃないか？

「......分かった」

俺は杖を下ろした。

「お前は俺だ。俺の弱さ、俺の恐れ、俺の迷い。全部ひっくるめて、俺だ」

影が止まった。

「俺は、完璧じゃない。エンジニアとしても、魔法使いとしても。失敗するし、怖がるし、迷う」
「でも——それを含めて、俺なんだ」

俺は、影に近づいた。

影は攻撃してこなかった。

「だから、戦わない。お前を受け入れる」

俺は、影に手を差し伸べた。

影が、光に包まれた。

そして——

俺の中に溶けていった。


// Integration Complete
// Shadow absorbed into self
// New ability unlocked: Shadow Step
// Effect: Short-range teleportation
『......予想外だ』ヴェリディアンの声に、驚きが混じっていた。「戦って勝つのではなく、受け入れて統合するとは。面白い」

第二の試練

霧が晴れた。

次は、古びた図書館のような場所にいた。

『第二の試練は、知識だ』

目の前に、巨大な本が現れた。


// The Tome of Ancients
// Contains all forgotten spells
// Question: How do you use knowledge responsibly?

本が勝手に開いた。

ページには、強力な魔法がぎっしりと書かれている。

  • 都市を一瞬で消滅させる魔法
  • 死者を蘇らせる魔法
  • 時間を巻き戻す魔法

「これを......俺が使えるようにするのか？」

『そうだ。ただし——』

本が、さらにページをめくった。

そこには、悲惨な光景が描かれていた。

都市が燃え、人々が苦しみ、世界が壊れていく。

それら全てが、古代魔法の暴走によるものだった。

『古代の魔法使いは、強大すぎる力を手に入れた。そして、それを使いこなせなかった。結果——世界は一度滅びかけた』

俺は、本を見つめた。

力には、責任が伴う。

それは、元の世界でプログラマーとして働いていた時も同じだった。

コード一つで、システムを破壊できる。人の生活を狂わせられる。

だからこそ、倫理が大切だ。

「この魔法は、使わない」俺は言った。
『ほう？』
「少なくとも、正当な理由がない限りは。守るためなら使うかもしれない。でも、攻めるために使うつもりはない」

俺は、本を閉じた。

「知識は、管理して使うものだ。手放しで使えば、悲劇を生む」

一瞬の沈黙。

そして——


// Trial 2: PASSED
// Wisdom recognized
// Access granted to: Basic Ancient Spells
// Restricted: Destructive Spells (requires authorization)

第三の試練

再び、風景が変わった。

今度は、夕暮れの丘だった。

一人の少女が、丘の上に立っていた。

——エリナだ。

「カイトさん」エリナが、悲しそうに微笑んだ。「魔法、諦めて帰りましょうか」
「えっ......？」
「だって、危険すぎます。カイトさんが怪我したり、死んだりしたら......私は......」

エリナの目から、涙が溢れた。

「お願いです。私のそばにいて。危険なことはしないで」

これは——試練だ。

大切な人を守りたいという気持ちと、自分の使命の間で揺れる心。

俺は、エリナに近づいた。

「エリナ。君の気持ちは嬉しい。本当に」
「じゃあ......！」
「でも、俺は諦めない」

エリナの表情が、凍りついた。

「この世界には、解決しなきゃいけない問題がある。古代魔法の暴走、人々の苦しみ、世界の危機」
「俺には、その力がある。プログラミングの知識と、魔法の才能。それを使わないのは、逃げだ」
「だから——前に進むよ」

エリナが、泣きながら首を振った。

「でも......カイトさんがいなくなったら......」
「いなくならないよ」俺は言った。「君を守りながら、世界も守る。それが俺のやり方だ」

俺は、エリナの涙を拭った。

「君と一緒に、未来を作ろう」

エリナが——微笑んだ。

そして、光の中に消えていった。


// Trial 3: PASSED
// Determination confirmed
// Unwavering will recognized
// Full access granted

継承

目が覚めると、元の遺跡に戻っていた。

エリナ、リアム、セシリアが、心配そうに俺を見ていた。

「カイトさん！ 大丈夫ですか！？」エリナが叫んだ。
「......ああ。大丈夫だ」俺は体を起こした。

体の中に、新しい力が満ちているのを感じた。

古代の魔法が、俺のものになった。

『見事だ』ヴェリディアンの声が響いた。「三つの試練を全て突破するととは。お前は、私が予想していた以上の器だ』
「で、何ができるようになったんだ？」
『古代魔法の基礎全てだ。時空操作、魂の干渉、世界の構造把握。ただし——最強の魔法は封印されている。お前が真に覚醒した時、解除されるだろう』

最強の魔法、か。

今は必要ないけど、いつか必要になるかもしれない。

「分かった。ありがとう、ヴェリディアン」
『礼を言うのは私の方だ。長い時を経て、ついに継承者が現れたのだから』

ヴェリディアンの気配が、消えた。

俺は立ち上がり、仲間たちの方を向いた。

「さて——次に行こうか」

エリナが、涙目で俺に抱きついた。

「もう......心配させないでください......！」

俺は、エリナの頭を撫でた。

「ごめん。でも、約束する。君を置いて死んだりしない」

リアムが、苦笑した。

「相変わらず、女心を掌握するのが上手いな、カイトは」

セシリアが、静かに微笑んだ。

「古代魔法の継承者か......世界が、動き始めましたね」

俺は、空を見上げた。

青い空に、白い雲が流れている。

新しい力を手に入れた俺の、新しい物語が始まる。
