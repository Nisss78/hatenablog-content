第34章：共鳴の果て

カイトの視界が揺らいだ。

時間差の魔術——DelayedSync——が完了した瞬間、世界が二重に見えた。いや、二重ではない。もっと複雑だ。現在と過去、ふたつの時間軸が彼の意識の中で重なり合っている。

「カイト！ しっかりして！」

エリナの声が、遠くから聞こえるようで、すぐ耳元で響いているようでもあった。距離感が狂っている。時間感覚も狂っている。

カイトは膝をついた。両手で頭を抱える。意識の奥底で、何かがざわめいている。

「……見つけた」

その声は、彼自身のものではなかった。

低く、古びた響き。古代の魔術師ヴェリディアンの声だ。カイトが時間差の魔術で一時的にリンクした相手。だが、リンクは解かれたはずだ。なのに、なぜ——

class Resonance {
    constructor(source, target) {
        this.source = source;
        this.target = target;
        this.isActive = false;
    }
    
    trigger(delay) {
        setTimeout(() => {
            this.isActive = true;
            this.synchronize();
        }, delay);
    }
    
    synchronize() {
        // 意識の同期処理
        this.source.consciousness.merge(this.target.consciousness);
    }
}

カイトの脳内に、コードが浮かび上がる。これは彼が書いたものではない。ヴェリディアンが残した魔術の痕跡だ。時間差の魔術は、単にタイミングをずらすだけのものではなかった。もっと深い、意識レベルでの接続を引き起こすトリガーだったのだ。

「まずい……予想以上の反応だ」

リアムが駆け寄り、カイトの肩を掴んだ。その手が、少しだけ透けて見える。カイトの視覚が、過去の光景を拾い始めている証拠だ。

「カイト、聞こえるか？ DelayedSyncのコールバック関数を確認しろ。古代の魔術師が仕込んだリスナーが動いてる」

リアムの言葉は、専門用語を含んでいても、この状況では正しく伝わってくる。彼もまた、カイトの魔術コードを理解できる数少ない存在だ。

カイトは深呼吸をした。意識を集中させる。脳内のコードに干渉する。

// 緊急停止処理
function emergencyBreak() {
    const resonance = new Resonance(kaito, veridian);
    if (resonance.isActive) {
        resonance.source.consciousness.detach();
        resonance.target.consciousness.detach();
        resonance.isActive = false;
    }
}

コードを実行——しようとした瞬間、強烈な抵抗が返ってきた。

「無駄だ、若い魔術師よ」

ヴェリディアンの声が、今度ははっきりと脳内に響いた。ただの声ではない。意志だ。強固な、数百年の時を経た意志。

「わたしの魔術は、一度起動すれば自己保存機能が働く。SelfPreserve——聞いたことがあるか？」

カイトは歯を食いしばった。自己保存機能。予期せぬ停止を防ぐための安全装置。だが、それは本来、魔術師自身を守るためのものだ。ヴェリディアンにとって、カイトは「守るべき対象」ではなく「利用すべきリソース」になっている。

「お前の体……良いホストになりそうだ。現代の魔術知識と、わたしの経験が融合すれば、かつてない力を手に入れられる」

支配。それがヴェリディアンの狙いだ。

カイトは必死に思考を巡らせた。通常の魔法であれば、術者の意志が最優先される。だが、今の状況は違う。ヴェリディアンの魔術は、カイトの魔術コードを「バックドア」として利用している。カイト自身が書いたコードが、彼を逆に縛り付けているのだ。

「カイト！ 何か言って！」

エリナの叫びが、現実世界からのアンカーになる。彼女の手が、カイトの頬に触れた。温かい。リアルだ。この感覚を頼りに、カイトは意識を保とうとした。

——逆転できないか？

突然の発想。ヴェリディアンの魔術を無効化するのではなく、逆に利用する。バックドアは双方向だ。相手が侵入してくるなら、こちらからも相手の領域に入り込める。

class ReverseInfiltration {
    constructor(intruder, defender) {
        this.intruder = intruder;
        this.defender = defender;
    }
    
    execute() {
        // 侵入経路を逆方向に使用
        const channel = this.intruder.getChannel();
        return this.defender.infiltrate(channel);
    }
}

カイトは目を開けた。焦点が合っていない。だが、意識ははっきりとしている。

「セシリア、座標を！」

突然の言葉に、周囲が反応する。セシリアが、驚きながらも即座に杖を振った。

「え、ええ！ 魔力の流れを可視化します！」
spell ManaVisualization {

空中に、青白い光の筋が浮かび上がる。魔力の流れを視覚化したものだ。その中の一本が、カイトの胸から伸びて、虚空の彼方へと続いている。ヴェリディアンの意識と繋がっているチャンネルだ。

「リアム、このチャンネルの逆方向にPulseを送れるか？」
「逆方向？ ……なるほど、相手の侵入経路を使って、向こうの意識に干渉するつもりか」

リアムは素早く状況を理解した。彼は懐から小さな水晶を取り出す。

「理論上は可能だ。だが、タイミングがシビアだ。Pulseを送るのは、相手が最も油断している瞬間でなければならない」
「その瞬間を作る」

カイトは言った。そして、意識の奥底——ヴェリディアンと繋がっている領域——に、偽の思考を送り込んだ。

「わかった……お前の言う通りにする。体を明け渡す」

偽装降伏。ヴェリディアンが油断する瞬間を作るための、危険な賭けだ。

一瞬の沈黙。そして、ヴェリディアンの声に、喜びの色が混じった。

「賢明な判断だ。抵抗すれば、意識ごと消し去るつもりだったが……協力すれば、悪いようには扱わん」

来る。

ヴェリディアンの意識が、より強く押し寄せてくる。カイトの体を乗っ取ろうとする、強烈な意志の波。その瞬間——ヴェリディアンの注意が「乗っ取り」に集中した瞬間——カイトは叫んだ。

「今だ！」

リアムが水晶を砕いた。同時に、強力なPulseが、カイトとヴェリディアンを繋ぐチャンネルを逆流する。

ヴェリディアンの悲鳴が、カイトの脳内で響いた。

「なっ——貴様、騙したな！」

チャンネルが振動する。接続が不安定になる。カイトはこの機を逃さなかった。

実行。

強烈な光が視界を埋め尽くし——

そして、静寂。

カイトは荒い息を吐きながら、地面に倒れ込んだ。体が重い。鉛のように重い。だが、意識は自分だけのものだ。ヴェリディアンの声は、もう聞こえない。

「カイト！ 大丈夫！？」

エリナが彼を抱き起こす。その表情は、安堵と心配が混ざっている。

「……ああ。何とか、な」

カイトは弱々しく笑った。全身の魔力を使い果たした感覚だ。しばらくは、魔法どころか、立っているのもやっとだろう。

「古代の魔術師……予想以上に手強かった」

リアムが、砕けた水晶の欠片を見つめながら言った。

「時間差の魔術自体は成功したんだが、その中に仕込まれたリスナーが予想外だった。DelayedSyncは、単なるタイミング調整じゃない。意識レベルでの接続を引き起こすトリガーだったんだ」

カイトは頷いた。

「ああ。そして、その接続を利用して、彼は僕の体を乗っ取ろうとした。……現代の知識と、彼の経験を組み合わせれば、最強の魔術師になれる——そう考えてたみたいだ」
「恐ろしい話ね」

セシリアが身震いした。

「古代の魔術師の意志が、数百年経っても残っていたなんて。カイトさん、本当に大丈夫ですか？ 何か残っていないか、診断したほうが——」

「大丈夫だと思う」

カイトは自分の胸に手を当てた。

「接続は完全に断ち切った。彼の意識は、また深淵へと戻ったはずだ。……ただ」

彼は言葉を濁した。

「ただ、何ですか？」

エリナが促す。

「……彼の記憶の一部が、流れ込んできた気がする。断片的だけど。この世界の古代について、そして——彼がなぜそんな执念を抱くに至ったか」

カイトの脳裏に、ヴェリディアンの記憶の断片がよぎる。

炎に包まれた塔。悲鳴。そして、愛する者を守れなかった後悔。数百年の時を超えて、その痛みだけが残っていたのだ。

「……守りたかったんだよ、彼は」

カイトは独り言のように呟いた。

「力を求めたのは、誰かを守るため。その誰かを失ったからこそ、どんな手段を使ってでも、二度と同じ失敗を繰り返さないための力を求めた。……歪んでいたけれど、根っこは孤独と喪失だった」

エリナが、カイトの手を握った。

「カイトは違うよ。力を求めてるわけじゃない。ただ、みんなを守りたいだけ——そうでしょう？」

カイトは、エリナの瞳を見つめた。真っ直ぐで、何の曇りもない瞳。

「……ああ。そうだな」

彼は、微かに笑った。

「僕は、プログラマーだからな。コードを書くのは、バグを潰すためじゃない。良いものを作るためだ。この世界でも、それは変わらないよ」

リアムが、ため息をつきながら立ち上がった。

「感動的な話は置いておくとして……カイト、しばらくは魔力を使うなよ。DelayedSyncの副作用で、精神に負荷がかかってる。完全に回復するまでは、最低でも三日は安静だ」
「三日も？」
「文句言うな。古代の魔術師に乗っ取られそうになったんだぞ。三日で済むなら安いものだ」

もっともな意見だ。カイトは苦笑しながら、エリナに支えられて立ち上がった。

その時、彼の脳内に、微かなコードが浮かんだ。

// Recovery Protocol
function startRecovery(target) {
    target.status = "resting";
    target.mpRegenRate *= 0.1; // 回復速度低下
    target.consciousness.stability = "unstable";
    
    setTimeout(() => {
        target.status = "normal";
        target.mpRegenRate = 1.0;
        target.consciousness.stability = "stable";
    }, 3 * 24 * 60 * 60 * 1000); // 3日
    
    return "Recovery protocol initiated";
}

カイトは、自分の内側で回復プロセスが始まったのを感じた。リアムの診断通り、三日間は魔力が満足に使えないだろう。

だが、それは悪いことばかりではない。この三日間で、ヴェリディアンから流れ込んできた記憶の断片を整理できる。古代の魔術に関する知識は、これからの冒険で必ず役に立つはずだ。

「さて、拠点に戻ろうか」

カイトは言った。エリナが頷く。

「うん。カイトが休めるように、美味しい紅茶も淹れるね」
「頼む。甘めで」
「もう、甘党なんだから」

二人のやり取りを見ながら、リアムとセシリアは顔を見合わせた。

「……あの二人、本当にただのパートナーなのか？」
「さあ？ 少なくとも、エリナさんはカイトさんに……あ、いえ、何でもありません」

セシリアが慌てて口を閉じる。リアムは、意味深な笑みを浮かべた。

夕日が、四人の影を長く伸ばしていた。

古代の魔術師との遭遇。予想外の戦い。そして、新たな知識。

カイトの冒険は、まだ始まったばかりだ。
