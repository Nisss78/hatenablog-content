第51章：守人の里

夜明け前の森は、静寂に包まれていた。

カイトたちは、マルクスから託された地図を頼りに、守人の里を目指していた。ギルドの地下訓練場から脱出した後、彼らは一晩中歩き続けたのだ。

「里まで、あとどれくらい？」

エリナが、眠そうな目で尋ねた。無理もない。昨夜の激戦の疲れが、まだ残っている。

「地図によると、この森を抜けた先にある峡谷の向こう側だ」

カイトが、羊皮紙を広げて答えた。魔術陣が淡い光を放ち、方角を示している。

class GuardianVillage {
    location: {
        terrain: "hidden_valley",
        barrier: "ancient_protection",
        accessibility: "restricted"
    };
    
    inhabitants: "Guardians";
    population: "unknown";
    
    // 里への侵入には、守人の許可が必要
    // 認証プロセスが存在する
}

カイトの脳内に、情報が浮かんだ。これは地図に込められた古代の魔術が、彼に知識を与えているのだ。

「守人の里か……伝説の場所だな」

リアムが、興味深そうに呟いた。

「一般には存在すら知られていない。守人たちは、表の世界から隠れて、世界の秩序を守り続けてきた」
「それなのに、アルフレッドさんは第7章で僕たちに遺跡を案内してくれた。どうして？」

カイトの疑問に、セシリアが考え込むように答えた。

「おそらく、カイトさんの中にある古の因子を感じ取ったからでしょう。アルフレッドさんは、カイトさんが特別な存在だと気づいていたはずよ」

特別な存在。

その言葉が、カイトの胸に重くのしかかった。彼は、自分が普通の人間ではないことを受け入れつつあった。だが、それが意味する責任の重さに、まだ戸惑いを感じていた。

「……とにかく、進もう。話は、アルフレッドさんに会ってからだ」

カイトは、前を向いた。悩んでいても始まらない。行動あるのみだ。

一行は、霧の深い森の中を進んでいった。

峡谷の関門

森を抜けると、目の前に壮大な峡谷が広がっていた。切り立った岩壁が、空を切り裂くように聳え立っている。

「すごい……こんな場所に、里があるの？」

エリナが、感嘆の声を上げた。

「地図の指示では、この峡谷を渡る必要がある。でも、橋なんて見当たらないぞ」

リアムが、周囲を見渡した。確かに、峡谷を渡る手段が見当たらない。幅は百メートル以上、底は深く、見えない。

「待って、何かあるわ」

セシリアが、峡谷の縁に近づいた。彼女の鋭い観察眼が、見逃しそうなものを見つけたようだ。

「ここに、古代の魔術陣が刻まれてる。起動すれば、何か起こるはず」

カイトも近づいて確認した。確かに、岩肌に微かな刻印がある。風化してほとんど見えないが、魔力を感じ取ることができた。

「僕がやってみる」

カイトは、深呼吸をして、魔術陣に手をかざした。体内の魔力を意識し、古代の因子と共鳴させる。

刹那、峡谷の底から光が溢れた。

光は、複雑な幾何学模様を描きながら、峡谷を跨ぐように浮かび上がる。そして、それが凝縮されて、透明な橋となった。

「すげえ……光の橋だ」

リアムが、目を見開いた。

「古代の守人が作った移動手段よ。認証された者だけが、使える」

セシリアが説明した。カイトの古の因子が、認証をパスしたのだ。

「カイト、すごいじゃん！ これで渡れるね」

エリナが、カイトの背中を叩いた。その明るさに、カイトも少し励まされた。

「みんな、気をつけて。橋は透明だけど、落ちたら終わりだ」

一行は、慎重に光の橋を渡り始めた。

守人の里

峡谷を渡りきると、目の前に隠された谷が広がっていた。

そこは、まるで別世界だった。

穏やかな丘に、点在する木造の家々。中心には、大きな神殿のような建物が建っている。空は、外の世界よりも青く、空気が澄んでいた。

「ここが……守人の里」

カイトは、息を呑んだ。伝説の場所が、実在したのだ。

「誰か来るぞ」

リアムが、剣の柄に手をかけた。丘の向こうから、数人の影が近づいてきた。守人たちだ。皆、紺色の法衣を身に纏っている。

「止まれ！ 異邦人よ、ここは守人の聖域。立ち入ることは許されていない」

先頭の守人が、厳しい声で告げた。年齢は四十代ほど、鋭い目つきの男性だ。

「僕たちは、アルフレッドさんに会いに来ました。彼は、守人の長ですよね？」

カイトが、落ち着いて答えた。

「アルフレッド様に？ 何用だ？」
「マルクス……収束者から、この地図を託されました。世界崩壊の真実を、聞きに来たんです」

「マルクス」という名前に、守人たちの間に動揺が走った。

「収束者から……？ ならば、お前が、古代の因子を持つ者か」

守人は、カイトをまじまじと見た。

「わかった。アルフレッド様に連絡する。ここで待て」

守人の一人が、神殿の方へ走っていった。残りの守人たちは、カイトたちを警戒しながら、周囲を囲んだ。

「緊張するね」

エリナが、小声でカイトに囁いた。

「大丈夫。僕たちは、敵じゃないってこと、ちゃんと伝えるから」

カイトは、エリナの手を握った。彼女の手は、少し冷たかった。

アルフレッドとの再会

十分後、神殿から一人の老人が現れた。

白髪に長い髭、穏やかな目。第7章で出会った、守人の長——アルフレッドだ。

「カイト殿、久しぶりじゃな」

アルフレッドは、柔和な笑みを浮かべて近づいてきた。

「アルフレッドさん……覚えていてくれたんですね」

「当然じゃ。そなたの中にある古の因子、忘れるはずがない」

アルフレッドは、カイトの前に立った。そして、真剣な表情になった。

「マルクスから、地図を託されたと聞いた。彼は、どうしておる？」
「戦いになりました。……彼は、生きていますが、深手を負っています」

カイトは、正直に答えた。

「そうか……彼もまた、過酷な運命を背負っておった」

アルフレッドは、寂しげに目を伏せた。

「さて、そなたたちは、真実を聞きに来たと言ったな。よかろう。神殿へ案内しよう。だが、その前に……」

アルフレッドは、カイトの仲間たちを見渡した。

「そなたの仲間たちも、真実を知る覚悟はあるか？ 一度聞けば、後戻りはできぬぞ」

カイトは、仲間たちを見た。

エリナが、力強く頷いた。

「私は、カイトと一緒に行くって決めたの。どんな真実でも、怖くない」

リアムが、肩をすくめた。

「俺は、古代の魔術に興味があるだけだ。真実だろうが何だろうが、知りたいね」

セシリアが、静かに微笑んだ。

「カイトさんを支えるのが、わたしの役目です。どこへでもついていきます」

カイトは、仲間たちの言葉に胸が熱くなった。

「みんな……ありがとう。アルフレッドさん、彼らも一緒に、真実を聞かせてください」

アルフレッドは、満足げに頷いた。

「よい仲間を持ったな、カイト殿。では、参ろう」

神殿の地下

アルフレッドに案内されて、カイトたちは神殿の地下へと降りていった。

石段を下りると、広大な空間が現れた。壁には、無数の文献が収められている。古代の書物、魔術の記録、歴史の断片——守人が何百年もかけて蓄積した知識の宝庫だ。

「ここは、守人の図書室じゃ。そして、ここに世界の真実が記されておる」

アルフレッドは、部屋の中央にある巨大な石版を指差した。

石版には、古代の文字が刻まれている。カイトは、その文字を見た瞬間、理解できた。ヴェリディアンから受け継いだ知識が、翻訳しているのだ。

class WorldTruth {

カイトは、石版の内容を読み上げた。

「この世界は、かつて次元の収束点から生まれた。複数の次元が交差する場所、そこがこの世界の起源だ。だが、古代の戦争によって、次元のバランスが崩れた。dimensional_instability——次元の不安定化が始まり、世界崩壊のプロセスが起動した」

アルフレッドが、続けた。

「そうじゃ。古代の魔術師たちは、次元を操る力を手に入れ、それを使って戦った。その結果、世界の基盤が揺らいだのじゃ。ヴェリディアンは、その戦争の生き残りじゃった」
「ヴェリディアンが……戦争の当事者？」
「左様。彼は、愛する者を守るために力を求めたが、その力が世界を崩壊させる一因となった。彼は、最後に起源の扉を封印し、未来の誰かがそれを開いて世界を救えるように、自分の因子を残した」

カイトは、自分の胸に手を当てた。ヴェリディアンの因子——それが、自分の中にある。

「じゃあ、僕は……ヴェリディアンの遺志を継いで、起源の扉を開かなきゃいけないのか？」
「それは、そなたが決めることじゃ」

アルフレッドは、カイトの目を真っ直ぐに見た。

「起源の扉を開けば、世界を救えるかもしれん。だが、扉を開くことにはリスクもある。何が起こるか、誰にも予測できぬ」
「リスク……？」
「起源の扉は、次元の収束点じゃ。扉を開けば、この世界と他の次元が繋がる。善い存在だけでなく、悪しき存在も侵入してくるかもしれぬ」

カイトは、息を呑んだ。世界を救うために、新たな危険を招く可能性があるのか。

「それでも、やるしかないんだろ？」

リアムが、静かに言った。

「世界崩壊が73%も進んでるなら、何もしない方が危険だ。リスクを背負ってでも、賭けるしかない」
「リアム……」

エリナが、カイトの手を握り直した。

「カイト、私はカイトを信じてる。カイトなら、きっと最善の選択ができるよ」

セシリアも、頷いた。

「わたしもです。カイトさんは、これまで多くの困難を乗り越えてきました。今回も、きっと大丈夫です」

仲間たちの信頼が、カイトの背中を押した。

「……わかった。僕は、起源の扉を開く。世界を救うために」

カイトは、決意を固めた。

「アルフレッドさん、起源の扉への道を教えてください」

アルフレッドは、少し寂しげに、しかし誇らしげに微笑んだ。

「よかろう。じゃが、その前に、準備が必要じゃ。起源の扉へは、特定の条件を満たさねば近づけぬ」
「条件？」
「三つの試練じゃ。身も心も、覚悟を問う試練。それを乗り越えてこそ、扉を開く資格が与えられる」

三つの試練。

カイトの前に、新たな壁が立ちはだかった。だが、彼はもう迷わない。

「受けて立ちます。僕たちは、必ず乗り越えてみせます」

カイトの言葉に、アルフレッドは深く頷いた。
