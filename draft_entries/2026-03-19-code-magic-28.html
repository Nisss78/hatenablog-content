第28章：条件分岐の岐路

ゴーレムを倒した後、カイトたちは古代図書館のさらに奥へと進んだ。石段を下りるにつれて、空気は冷たく、湿り気を帯びてきた。壁に刻まれた古代の文字が、淡い青い光を放っている。

「カイト、さっきの魔法……本当にすごかったわね」

エリナが隣を歩きながら言った。その声には、純粋な感嘆と少しの不安が混じっていた。

「変数を使えば、魔法が生き物みたいに振る舞える。でも、それには限界があることもわかった」

カイトは冷静に答えた。

「限界？」
「ああ。変数は値を動的に変えられる。でも、魔法の行動パターン自体は変わらないんだ」

リアムが眉をひそめた。

「どういうことだ？」

カイトは足を止め、振り返った。

「例えば、さっきのゴーレム戦。俺は変数を使って火球の威力を調整した。でも、火球であることには変わりなかった。もしゴーレムが火に耐性を持っていたら？ その場合、自動で別の属性に切り替えられたら便利だと思わないか？」

セシリアが頷いた。

「確かに。敵に合わせて属性を変えられる魔法があれば、どんな状況にも対応できるわね」
「それが、条件分岐だ」

カイトは指を空中に走らせ、光の文字を描いた。


CONDITIONAL_SPELL {
    if (@target.resistance == FIRE) {
        element: ICE
        power: @target.defense * 2.0
    } else if (@target.resistance == ICE) {
        element: THUNDER

    }
}
「if、else if、else。これが条件分岐の基本構造だ。状況に応じて、全く異なる魔法を展開できる」

エリナが目を輝かせた。

「すごい……まるで、魔法が自分で判断してるみたい」
「厳密には、判断してるのは魔法じゃない。条件分岐のロジックだ。でも、結果として見れば、魔法が自律的に動いているように見える」

四人は再び歩き出した。通路はやがて広い空間へと開けた。そこは、巨大な円形の広間だった。天井は見えないほど高く、壁には無数の扉が並んでいた。

「これは……」

リアムが息を呑んだ。

「試練の間だ。古代の魔法使いが、自分の実力を試すために作った場所」

セシリアが説明した。

「各扉の先には、異なる試練が待っている。一度入ると、試練をクリアするまで出られないわ」

カイトは扉を観察した。それぞれの扉には、異なるシンボルが刻まれていた。剣、盾、杖、本、宝石……計十二の扉があった。

「どの扉を選ぶ？」

エリナが尋ねた。カイトは少し考え、指差した。

「杖の扉だ。魔法の試練なら、俺の新しい技術をテストするのにちょうどいい」

カイトは杖の扉に近づき、手を置いた。すると、扉が音もなく開いた。中から、冷たい風が吹き出してきた。

「行くぞ」

四人は、扉の中へと足を踏み入れた。

扉の先は、薄暗い森だった。木々は歪み、枝はまるで手のように伸びていた。地面は苔で覆われ、歩くたびに湿った音がした。

「雰囲気のある場所ね」

エリナが皮肉っぽく言った。その時、森の奥から低い唸り声が聞こえた。

「何か来る」

リアムが剣を抜いた。カイトもまた、魔法の構築を開始した。条件分岐を組み込んだ、新しい魔法を。

森の奥から現れたのは、三匹の魔獣だった。一匹は炎を纏った狼。一匹は氷の結晶でできた蛇。最後の一匹は、雷光を放つ鳥だった。

「属性魔獣の群れか！」

セシリアが叫んだ。

「それぞれ異なる属性を持ってる。一つの魔法じゃ対処できないわ！」
「それが狙いだ」

カイトは笑った。

「条件分岐の魔法をテストするには、最高の相手だ」

カイトは、魔導言語のネットワークに接続した。変数と条件分岐を組み合わせた、複雑な魔法を構築する。


class AdaptiveElemental {
    constructor() {
        this.variables = {};
    }
    
    analyze(target) {
        // 敵の属性を解析
        if (target.aura.includes('FIRE')) {
            return { weakness: 'ICE', resistance: 'FIRE' };
        } else if (target.aura.includes('ICE')) {
            return { weakness: 'FIRE', resistance: 'ICE' };
        } else if (target.aura.includes('THUNDER')) {
            return { weakness: 'EARTH', resistance: 'THUNDER' };
        } else {
            return { weakness: 'LIGHT', resistance: 'DARK' };
        }
    }
    
    generateSpell(targets) {
        const spells = [];
        
        for (let target of targets) {
            const analysis = this.analyze(target);
            
            spells.push({
                target: target.id,
                element: analysis.weakness,
                power: target.defense * 1.5,
                priority: target.threat
            });
        }
        
        // 脅威度でソート
        spells.sort((a, b) => b.priority - a.priority);
        
        return spells;
    }
}
「解析開始」

カイトの目が光った。彼の脳内で、魔導ネットワークが高速で処理を行っていた。

「最も危険なのは、雷鳥だ。速度が高い。先に処理する」

カイトは右手を前に出した。

「条件分岐・属性適応魔法、展開！」

カイトの手から、三つの光弾が放たれた。それぞれが異なる色をしていた。土色、赤色、青色。

「目標、雷鳥！ アース・スパイク！」

土色の光弾が雷鳥に向かって飛んだ。地面から岩の槍が突き出し、雷鳥の翼を貫いた。

「ぎゃあああ！」

雷鳥は悲鳴を上げ、墜落した。

「次、炎狼！ アイス・ランス！」

青色の光弾が炎狼に向かった。空中で無数の氷の槍に変化し、炎狼を串刺しにした。

「最後、氷蛇！ ファイア・ウェーブ！」

赤色の光弾が氷蛇に直撃し、炎の波となって包み込んだ。氷蛇の体が溶け、蒸発していった。

三匹の魔獣は、わずか数秒で全滅した。

「す、すごい……」

エリナが呆然としていた。

「一つの魔法で、三匹の属性魔獣を同時に倒すなんて……」
「条件分岐のおかげだ」

カイトは肩で息をした。

「それぞれの敵に最も効果的な属性を自動選択できる。これが、if文の力だ」

リアムが感心して頷いた。

「戦場でこれが使えたら、戦略の幅が劇的に広がるな」
「でも、完璧じゃない」

カイトは冷静だった。

「条件分岐は、事前に定義した条件にしか対応できない。想定外の敵が出たら、どうする？ その場合、else句でデフォルトの行動を取ることになるが、それが最適解とは限らない」

セシリアが首を傾げた。

「つまり、すべての可能性を想定しておく必要があるってこと？」
「そうだ。それが、条件分岐の限界であり、プログラミングの永遠の課題だ」

カイトは、森の奥を見つめた。試練はまだ続いているはずだ。

その時、地面が揺れた。森の中心から、巨大な影が立ち上がった。それは、三匹の魔獣の融合体だった。炎、氷、雷、三つの属性が浑然一体となった、異形の怪物。

「まさか、融合タイプ！？」

セシリアが絶叫した。

「三属性同時持ちなんて、聞いたことないわ！」

カイトは眉をひそめた。条件分岐の魔法は、単一属性の敵を想定していた。三属性同時持ちの敵には、どう対応すればいいのか？

「想定外の事態だ……」

カイトは焦った。しかし、その焦りの中で、新しいアイデアが浮かんだ。

「待てよ……条件分岐は、単一の条件に対して単一の行動を取る。でも、複数の条件を組み合わせたらどうなる？」

カイトは、魔法を急いで書き換えた。


COMPLEX_CONDITIONAL {
    if (@target.hasFire && @target.hasIce && @target.hasThunder) {
        // 三属性持ちの場合
        element: HOLLOW // 属性なし

        power: @target.defense * 3.0
        effect: DISPELL // 属性打ち消し
    } else if (@target.hasFire && @target.hasIce) {
        // 炎と氷の複合
        element: WATER
        power: @target.defense * 2.5
    } else if (@target.hasFire && @target.hasThunder) {
        // 炎と雷の複合
        element: EARTH
        power: @target.defense * 2.5
    }
    // ...他の組み合わせ
}
「複合条件！ AND演算子で、複数の条件を同時に評価する！」

カイトは、新しい魔法を展開した。

「属性打ち消し魔法、ホロウ・インパクト！」

無色透明の衝撃波が、融合怪物に向かって飛んだ。それは属性を持たない純粋な物理エネルギーだった。

衝撃波が怪物に直撃した瞬間、怪物の体から炎、氷、雷の光が消えた。属性が打ち消されたのだ。

「今だ！ みんな、攻撃して！」

カイトが叫んだ。エリナ、リアム、セシリアが同時に動いた。

エリナの風魔法、リアムの剣技、セシリアの光魔法。三者の攻撃が、属性を失った怪物に降り注いだ。

「うおおおお！」

怪物は咆哮し、倒れた。その体は、灰色の塵となって散っていった。

戦闘が終わると、森の景色が変わり始めた。歪んだ木々が消え、美しい庭園が現れた。試練のクリアを示していた。

「やった……」

エリナが座り込んだ。

「本当に、勝てたわ」
「条件分岐の応用、成功だ」

カイトもまた、疲労を感じていた。複雑な条件分岐は、精神力を大きく消耗した。

「でも、まだ改良の余地がある。条件分岐だけでなく、もっと柔軟なロジックが必要だ」

リアムが尋ねた。

「例えば？」
「繰り返し処理……ループだ。あるいは、例外処理。 try-catchみたいな」

カイトは庭園の中央にある噴水に近づいた。その水面に、古代の文字が浮かんでいた。

「これは……？」

カイトは文字を読んだ。それは、次の試練へのヒントだった。


The next challenge awaits:
LOOP OF ETERNITY
Where beginning meets end,
And end meets beginning.
「Loop of Eternity……永遠のループ」

カイトは呟いた。

「どうやら、次はループの概念を学ぶことになりそうだ」

セシリアが興味深そうに覗き込んだ。

「ループって何？」
「同じ処理を繰り返すことだ。特定の条件が満たされるまで、何度でも」

カイトは立ち上がった。

「例えば、敵が死ぬまで攻撃を繰り返すとか。あるいは、魔力が尽きるまで魔法を撃ち続けるとか」
「それは強力ね」

エリナが言った。

「でも、危険でもある。終了条件を間違えたら、無限ループに陥る。魔力が尽きるまで止まらない……最悪の場合、自分自身を滅ぼすことになる」

カイトの言葉に、全員が静かになった。プログラミングの概念は強力だが、その分、リスクも大きかった。

「でも、怖がっていては前に進めない」

カイトは笑った。

「次の試練、行こうか」

四人は、庭園の奥にある次の扉へと向かった。そこには、「∞」のシンボルが刻まれていた。無限大のマーク。永遠のループの象徴。

カイトは深呼吸をした。そして、扉を開いた。

異世界での冒険は、まだ終わらない。条件分岐を習得したカイトの次の挑戦は、ループの概念。そしてその先には、まだ見ぬプログラミングの概念が待っていた。

「行くぞ、みんな」

カイトの声に、三人は頷いた。彼らは、無限の可能性を求めて、一歩を踏み出した。
