第65章：次元の境界線

カイトは目を開けた。そこには、かつて見たこともない風景が広がっていた。

空には二つの月が浮かび、地面からは淡い光の筋が幾筋も天に向かって伸びている。世界そのものが、プログラムのコードとして可視化されているかのような光景だった。

「ここは……どこなんだ？」

カイトの声は、空虚な空間に吸い込まれていくようだった。

「次元の境界線よ」

背後から声がした。振り返ると、銀色の髪をした女性が立っていた。その姿は半透明で、向こう側の風景が透けて見える。

「君は……前にも会ったことがある？」

女性は微かに微笑んだ。

「私の名はエーテリア。この世界の管理者の一人です。カイト、あなたが起こした『世界のシフト』によって、次元の境界が不安定になっています」

カイトは息を呑んだ。第64章での戦いで、彼は限界を超えて魔法を行使した。世界そのものを書き換えるような、強大なコードを実行したのだ。

「それって……まずいことなのか？」
「状況によります」

エーテリアは指を鳴らした。空中にホログラムのような映像が浮かび上がる。それは、この世界の地図のように見えたが、あちこちに赤い点滅が表示されていた。

「これらは、次元の裂け目です。世界のシフトによって生じた歪み。放っておけば、異世界からの侵入者を招くことになるでしょう」

カイトは眉をひそめた。

「侵入者って……どんな？」
「想像もつかないほど、恐ろしい存在です」

エーテリアの表情が曇った。

「虚空の向こう側には、我々が理解できない領域があります。そこに住む者たちは、論理も法則も持たない。ただ、すべてを飲み込もうとする渇望だけを持っている」

カイトは拳を握りしめた。

「じゃあ、その裂け目を塞げばいいんだな？」
「ええ。ただし、簡単ではありません。裂け目は世界中に散らばっていますし、塞ぐには膨大な魔力が必要です」

カイトは考え込んだ。プログラマーとしての彼の頭脳が、問題を解析し始める。

「裂け目の構造を解析して、パッチを当てるようなものか……」

彼は右手を掲げ、指先から光の文字を空中に浮かべ始めた。


class RiftSealer:
    def __init__(self, coordinates):
        self.coords = coordinates
        self.stability = 0
    
    def analyze(self):
        # 裂け目の構造を解析
        return self.scan_structure()
    
    def seal(self, mana_source):
        # 魔力を使って裂け目を塞ぐ
        required_mana = self.calculate_cost()
        if mana_source.available >= required_mana:
            self.patch_reality()
            return True
        return False

エーテリアが感心したように頷いた。

「なるほど。あなたの魔法は、世界をコードとして認識しているのですね」
「ああ。俺にとって、魔法はプログラミングと同じなんだ」

カイトは目を閉じ、集中した。彼の意識が、世界の構造へと潜っていく。

  • 地上の裂け目：23カ所
  • 空中の裂け目：7カ所
  • 深海の裂け目：3カ所
  • 計：33カ所の異常ポイント

膨大な数だった。一つずつ塞いでいては、何日かかるかわからない。

「一つずつやってたら日が暮れるな……」

カイトは別のアプローチを考えた。

「エーテリア、質問があるんだけど」
「何でしょう？」
「この裂け目って、何が原因でできてるんだ？ 俺が世界のシフトを起こしたから、連鎖反応みたいに広がったのか？」

エーテリアは首を横に振った。

「原因は一つではありません。あなたの行動は引き金になりましたが、元々、この世界には亀裂があったのです」

彼女は手を振ると、新しい映像を表示した。それは、古代の都市の遺跡のように見えた。

「千年以上前、この世界は『大崩壊』を経験しました。当時の文明が、無謀な魔法実験を行い、次元の壁に大穴を開けてしまったのです」

カイトは息を呑んだ。

「それって……俺と同じようなことを？」
「似ていますが、彼らにはあなたのような制御能力がありませんでした。結果として、世界の半分が虚空に飲み込まれました」

恐ろしい話だった。カイトは自分の行動が、どれほど危険だったのかを悟った。

「じゃあ、俺がやったのは……」

「ですが、あなたは成功しました」

エーテリアがカイトの目を見つめた。

「あなたのコードは完璧だった。世界を壊すのではなく、書き換えた。その違いは大きいです」

カイトは少しだけ安堵した。しかし、問題は解決していない。

「で、どうすればいい？ この33カ所の裂け目を塞ぐには？」

「協力が必要です」

エーテリアが指を鳴らすと、四人の姿がホログラムとして浮かび上がった。

「あなたの仲間たち。エリナ、リアム、セシリア、そして……もう一人」

最後の一人は、見覚えのない人物だった。黒いローブを着た、背の高い男。

「こいつは誰だ？」
「マルクス。収束者と呼ばれる存在です。彼は、次元の境界線を監視する役割を持っています」

カイトは記憶を辿った。確か、どこかで聞いたことがある名前だった。

「ギルドマスターのマグナスが言ってたな……収束者って言葉」
「彼もまた、あなたの協力者となるでしょう」

エーテリアが近づいてきて、カイトの額に触れた。

「私はあなたに、管理者権限の一部を付与します。これで、裂け目の詳細なデータにアクセスできるようになります」

温かい光がカイトを包んだ。彼の脳内に、膨大な情報が流れ込んでくる。


// 管理者権限付与完了
// アクセス可能領域：次元境界データベース
// 権限レベル：Lv.3（制限付き）

class AdminAccess {
    constructor(user) {
        this.user = user;
        this.level = 3;
        this.restrictions = ['void_access', 'reality_rewrite'];
    }
    
    getRiftData() {
        return database.query('SELECT * FROM dimensional_rifts');
    }
    
    patchRift(coords, mana) {
        if (this.level >= 3 && !this.restrictions.includes('reality_rewrite')) {
            return rifts.seal(coords, mana);
        }
        return 'Permission denied';
    }
}

カイトは目を開けた。彼の視界には、常に次元の境界線のデータが表示されるようになっていた。

「すごいな……まるで、デバッグモードに入ったみたいだ」
「そのような表現で正しいでしょう」

エーテリアが微笑んだ。

「あなたは世界をコードとして見ている。それは、この世界の本質に近い理解です」

カイトは立ち上がった。

「まずは、一番近い裂け目から始めよう。場所はどこだ？」

エーテリアが指差した方向には、遠くに山脈が見えた。

「北方の山脈、標高3000メートル地点。そこに、最も大きな裂け目があります」

カイトは頷いた。

「よし、行こう」

その時、空気が震えた。次元の境界線が、再び揺らいだのだ。

「何だ！？」

エーテリアの表情が変わった。

「来ましたか……早すぎる」

彼女が見上げた先には、空に亀裂が走っていた。そして、その亀裂から、何かが覗いていた。

巨大な、黒い影。無数の目が、こちらを見つめている。

「虚空からの侵入者です」

エーテリアが警告した。

「カイト、戦う準備を。彼らは……世界を食べる者たち」

カイトは右手を掲げ、戦闘態勢に入った。彼の指先から、コードが流れ始める。

カイトの周囲に、光の障壁が展開された。

「来いよ、虚空の化け物ども。俺が相手だ」

黒い影が、不気味な音を立てて笑った。それは、人の声ではない。世界そのものが悲鳴を上げているような音だった。

戦いの幕が上がった。

カイトは、次元の境界線で、異世界からの脅威と対峙することになった。彼のコードが、世界を守る最後の希望となる。
