第52章：試練の儀式

朝霧が立ち込める守人の里。カイトは宿の窓から、静かに佇む集落を眺めていた。昨日の出会いから、多くの情報を得た。アルフレッドという守人の長老、そして彼が語った「侵食」の真実。

世界の境界が薄れ、異質な存在が侵入し始めている。その源を突き止め、封印すること——それが「構築者」としてのカイトの使命だった。

「起きたのか？」

振り向くと、リアムが既に身支度を整えて立っていた。

「ああ。少し考え事をしていて」
「アルフレッドさんの話のことか？」

カイトは頷いた。

「『侵食』の源を見つけるには、この里にある『継承の間』で試練を受けろって言われた。その準備をしてくるよう、朝食後に集合だ」
「試練、か。どんな内容なんだろう」

リアムは肩をすくめた。

「古代の儀式らしいから、生半可な覚悟じゃ通れないだろうな」

二人は一緒に食堂へ向かった。エリナとセシリアは既に席についており、朝食をとっていた。

「おはよう、二人とも」

エリナが手招きした。

「ちょうど話していたのよ。『継承の間』について」
「何かわかったのか？」

カイトが尋ねると、セシリアが口を開いた。

「文献を調べた限りでは、『継承の間』は『構築者』としての能力を完全に覚醒させる場所のようです。そこで試練を乗り越えると、先代の知識や技術の一部を受け継げるとか」
「知識の継承……」

カイトは静寂城で手に入れた端末のことを思い出した。あれにも膨大な知識が記録されていた。もしさらに多くを得られるなら、戦いは有利になるだろう。

「でも、危険も大きいはずよ」

エリナが真剣な表情で言った。

「精神に直接干渉する儀式らしいから。失敗したら……」

彼女は言葉を濁したが、意味は明白だった。失敗すれば、自我を失う可能性もある。

「やるしかないよ」

カイトは決意を込めて言った。

「リスクがあるのはわかってる。でも、この世界を救うために召喚されたんだ。逃げるわけにはいかない」

仲間たちが頷く。彼らもまた、覚悟を決めていた。

朝食を終えた一行は、アルフレッドに案内されて里の奥へと進んだ。石畳の道を抜け、古木が生い茂る森の中へ。その奥に、苔むした石造りの神殿があった。

「ここが『継承の間』じゃ」

アルフレッドが足を止めた。

「中に入れるのは『構築者』一人だけ。付き添いは許されん」

エリナがカイトの腕を掴んだ。

「一人で大丈夫なの？」
「大丈夫だ」

カイトは彼女の手を優しく握り返した。

「みんなを信じてる。そして、自分の力も」

カイトは神殿の入り口に立った。重厚な扉には、静寂城で見たものと同じ古代文字が刻まれている。

真の構築者のみ、試練を超えよ

カイトは深呼吸をして、扉を押した。静かに、しかし重々しく扉が開いた。

神殿の内部は、外見とは異なり広大だった。天井は見えないほど高く、壁には無数の光る文字が浮かんでいる。

中央に、円形の台座があった。その上には、古代の「構築者」たちが使っていたと思われる端末が置かれている。

カイトが台座に近づくと、端末が勝手に起動した。

SYSTEM: Constructor detected.
SYSTEM: Initiating Inheritance Trial...
SYSTEM: Preparing mental synchronization...

突然、カイトの視界が白く染まった。

目を開けると、カイトは見知らぬ場所に立っていた。何もない白い空間。無限に広がる虚空。

「ようこそ、新たなる構築者よ」

声が響いた。振り向くと、一人の人物が立っていた。長い白髪、賢者のローブ、そして静かな眼差し。それは以前、静寂城で見たヴェリディアンのホログラムと同じ人物だった。

「ヴェリディアン……？」
「正確には、私の精神の残響じゃ」

彼女は微笑んだ。

「『継承の間』は、先代の記憶と直接繋がる場所。ここで試練を受け、成功すれば、私の知識の一部を受け継ぐことができる」
「試練というのは？」
「二つある」

ヴェリディアンは指を二本立てた。

「一つ目は、魔力の制御。あなたの力を、最大限に引き出せ」

彼女が手を振ると、カイトの目の前に巨大な岩が現れた。

「この岩を、あなたの『コード魔法』で消滅させなさい。ただし、単純な破壊魔法ではだめ。岩を構成する要素を分析し、その構造を書き換えるのじゃ」

カイトは岩に近づいた。表面を触り、魔力を流し込む。

AnalysisResult = scanTarget({
    object: "rock",
    depth: "molecular",
    output: "structure"
});

脳内に情報が流れ込んでくる。岩の成分、分子構造、魔力の通り道。すべてがコードとして理解できた。

function disintegrateRock() {
    let rock = getTarget("rock");
    rock.components.forEach(atom => {
        atom.bonds = null;
        atom.stability = 0;
    });
    return rock.dissolve();
}

カイトがコードを実行すると、岩は瞬く間に粒子となって消滅した。

「ほう、見事じゃ」

ヴェリディアンが満足そうに頷いた。

「破壊ではなく、分解。構造を理解し、書き換える。それが『構築者』の真髄じゃ」

第二の試練。

「二つ目は、精神の強さ」

ヴェリディアンの表情が変わった。

「あなたの中にある恐怖、迷い、弱さ。それらと向き合い、乗り越えよ」

再び視界が変わり、カイトは自分の記憶の中に立っていた。

それは、まだ地球にいた頃の記憶。プログラマーとして働いていた日々。プロジェクトの失敗。上司からの厳しい言葉。自分には才能がないかもしれないという疑念。

「お前には無理だ」

幻聴が響く。

「異世界で英雄になれるわけがない。お前はただのプログラマーだ」

カイトは膝をついた。恐怖と不安が胸を締め付ける。

「逃げるな」

自分自身の声が聞こえた。

「お前は逃げてきたんじゃない。新しい世界で、自分を証明しようとしているんだ」

カイトは顔を上げた。目の前に、もう一人の自分が立っていた。だが、その姿は敵意を持っていない。むしろ、励ましているようだった。

「過去の失敗は、今のお前を形作っている。それを認め、前に進め」

カイトは立ち上がった。恐怖は消えていない。でも、それを受け入れる強さが生まれていた。

function acceptWeakness() {
    let self = getSelf();
    self.fears.forEach(fear => {
        fear.acknowledged = true;
        fear.controlled = true;
    });
    return self.empower();

カイトの体が光り輝いた。精神的な負荷を乗り越え、より強い自分へと進化した瞬間だった。

白い空間に戻ると、ヴェリディアンが満足そうに立っていた。

「見事じゃ。二つの試練を乗り越えた」

彼女はカイトに手をかざした。

「約束通り、私の知識の一部を受け継がせよう」

光がカイトの体に吸い込まれていく。膨大な情報が脳内に流れ込んできた。古代の魔法理論。世界の構造。「侵食」の本質。そして、それを止めるための方法。

InheritedKnowledge = {
    magicTheory: "ancient_constructor_techniques",
    worldStructure: "dimensional_barrier_maintenance",
    erosionSource: "located_at_void_boundary",
    sealingMethod: "code_barrier_reconstruction",
    warnings: ["erosion_is_spreading", "time_is_limited"]
};

カイトは膝をついた。情報量が多すぎて、少しめまいがした。

「大丈夫か？」

ヴェリディアンが心配そうに覗き込んだ。

「はい……少し、情報量が多くて」
「時間が経てば慣れるじゃろう。今は、仲間のもとへ戻りなさい」

彼女の姿が少しずつ薄れていく。

「最後に一つだけ教えてくれ」

カイトが言った。

「俺をこの世界に召喚したのは、誰なんだ？」

ヴェリディアンは静かに微笑んだ。

「召喚を行ったのは、今は亡き最後の『構築者』の弟子……そして、今の『侵食』の源に最も近い存在じゃ」
「え？」

カイトが聞き返そうとした瞬間、視界が再び白く染まった。

カイトは神殿の台座の前で目を覚ました。全身に疲労感があったが、同時に力も漲っていた。

神殿の入り口から、仲間たちが駆け寄ってくる。

「カイト！大丈夫！？」

エリナが最初に駆けつけ、カイトの肩を支えた。

「ああ、大丈夫だ。試練を乗り越えた」

カイトは立ち上がり、仲間たちを見回した。

「『侵食』の源の場所がわかった。そして、それを止める方法も」

アルフレッドが頷いた。

「儀式は成功したようじゃな。継承、おめでとう」
「ありがとうございます。でも、同時に新たな疑問も生まれました」

カイトは真剣な表情で言った。

「俺をこの世界に召喚した人物について。その人物は、今の『侵食』の源に近い場所にいるらしい」

全員が沈黙した。カイトの言葉の意味を理解するのに、少し時間がかかった。

「つまり……召喚者自身が、何かに関わっているってこと？」

リアムが慎重に尋ねた。

「わからない。でも、真相を知るには『侵食』の源に行くしかない」

カイトは拳を握った。

「次の目的地は、世界の境界が最も薄い場所——『虚空の境界』だ」

セシリアが地図を取り出した。

「『虚空の境界』……ここから北東、約一週間の距離ですね。途中、いくつかの危険地帯を通過する必要がありますが」
「構わない。進もう」

エリナがカイトの隣に立ち、肩を叩いた。

「一人で背負い込むなよ。私たちがいるんだから」

カイトは微笑んだ。

「ああ。頼りにしてる」

一行は神殿を後にした。里の人々が、静かに彼らを見送っていた。

新たな知識、新たな力、そして新たな謎。カイトの戦いは、まだ始まったばかりだった。
