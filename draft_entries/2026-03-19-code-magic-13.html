第13章：インターフェースの誓い

継承の塔での激戦から一夜が明けた。

カイトたちは、塔の近くにある小さな村の宿屋に身を寄せていた。ヴォイド・ライブラリアンとの戦いは辛くも勝利したが、代償は大きかった。塔の守護者たちの多くは、まだ半透明の状態から回復していない。復元魔法は一部しか効果を発揮しなかったのだ。

カイトは窓際に座り、遠くの塔を眺めていた。朝日に照らされた黒い塔は、奇妙なほど静かだ。

「眠れないの？」

背後から声がした。エリナだ。彼女は寝間着のまま、カイトの隣に座った。

「ああ……いろいろ考えてて」

カイトは答えた。

「ヴォイド・ライブラリアンは倒したのに、守護者たちは元に戻らない。俺の復元魔法じゃ、不完全なんだ」
「自分を責めないで。あなたはできる限りのことをしたわ」

エリナはカイトの手を握った。温かい。

「でも、足りなかった。もっと根本的な解決策が必要なんだ」

カイトは立ち上がった。決意が固まった顔だ。

「明日、王立魔法研究所に行く。エレノアさんが言ってた『古代のアーカイブ』を調べたい」
「古代のアーカイブ？」
「ああ。この世界の魔法システムの原点があるはずだ。そこに行けば、守護者たちを完全に復元する方法が見つかるかもしれない」

エリナは頷いた。

「わかった。みんなを起こすわ」

王立魔法研究所は、王都の中央に位置する巨大な白亜の建物だった。尖塔が空を突き刺し、壁には無数の魔法陣が刻まれている。

「ここが、王国の魔法研究の中心地です」

案内してくれたのはエレノアだ。彼女は研究所の一室へカイトたちを案内した。

「古代アーカイブへのアクセス権限は、通常、所長しか持っていません。しかし……今回の件で、あなたたちの功績は十分に認められました」
「感謝する」

カイトは礼を言った。部屋の中央には、巨大な水晶球が置かれている。水晶球の中には、無数の文字が浮遊していた。

「これが『アーカイブ・コア』です。古代の魔法使いヴェリディアンが残した、あらゆる魔法の設計図が保存されています」

エレノアが説明した。

「ヴェリディアン……」

カイトはその名前に聞き覚えがあった。アルゴリズム遺跡で出会った、古代の知性。彼女の名前だ。

「カイト、何か知ってるの？」

リアムが尋ねた。

「ああ……遺跡で会ったんだ。彼女の意識の残り香に」

カイトは水晶球に手を触れた。瞬間、視界が白く染まった。

——意識が、深淵に落ちる。

カイトが目を開けると、そこは無限の図書館だった。天井も壁も見えない。ただ、本棚だけが無限に続いている。本棚には本ではなく、光る文字列が並んでいた。

「ようこそ、コード・メイジ」

声がした。振り向くと、一人の女性が立っていた。透き通るような肌に、銀色の髪。目は星のように輝いている。

「ヴェリディアン……？」

カイトが尋ねた。

「ええ。私の意識の一部が、このアーカイブに保存されています」

彼女は微笑んだ。

「あなたの問いは聞こえています。守護者たちを復元する方法を知りたいのですね？」
「はい。俺の復元魔法じゃ不完全で……」
「復元魔法には限界があります。削除されたデータは、元の設計図がなければ戻せない」

ヴェリディアンは指を鳴らした。空中にコードが浮かび上がる。

function restore_guardian(corrupted) {
  // 復元には「インターフェース」が必要
  // インターフェースとは、クラスが必ず実装すべき契約
  interface GuardianContract {
    name: string;
    protection_level: number;
    inherit(master: Guardian): void;
  }
  
  // この契約に基づいて、新しいインスタンスを生成
  return new GuardianClass(corrupted.original_blueprint);
}
「インターフェース……？」

カイトは首を傾げた。

「オブジェクト指向における重要な概念です」

ヴェリディアンは説明した。

「クラス継承は、親から子へ能力を受け継ぐ仕組み。しかしインターフェースは違います。それは『契約』——あるクラスが必ず備えるべき機能を定義するものです」
「契約……」
「そうです。例えば、『守護者』というインターフェースがあれば、すべての守護者は必ず『守る』という機能を持たなければならない。継承関係に関係なく、契約に基づいて機能が保証されるのです」

カイトは理解した。クラス継承は「血筋」のようなもの。インターフェースは「誓い」のようなものだ。

「それを守護者たちに適用すれば……？」
「元の能力を復元できます。インターフェースは、削除されても残る『契約の刻印』だからです」

ヴェリディアンは空中に新たなコードを描いた。

// すべての守護者が持つインターフェース
interface Guardian {
  readonly name: string;
  readonly oath: string;  // 誓いの言葉
  
  protect(target: any): void;
  inherit(master: Guardian): void;
  restore(): Guardian;
}

// インターフェースから復元
class GuardianRestorer {
  static restore(corrupted: any): Guardian {
    // 契約から再構築
    const contract = corrupted.__interface__;
    return contract.restore();
  }
}
「ただし、一つ条件があります」

ヴェリディアンの表情が曇った。

「インターフェースによる復元には、その守護者の『誓い』が必要です。彼らが生前、何を守ると誓ったのか——その言葉が鍵になります」
「誓い……？ どうやって見つけるんだ？」
「彼らの心に残っています。腐敗した体の中に、まだ微かに残る意志。それを引き出すのです」

カイトは頷いた。難しいが、不可能ではない。

「もう一つ、教えてくれ。ヴォイド・ライブラリアンは、なぜこんなことをしたんだ？」

カイトが尋ねた。ヴェリディアンは静かに首を振った。

「彼はかつて、私の弟子でした」
「えっ？」
「本名はヴィクター・ライブラリアン。才能ある魔法使いでしたが、『効率』に囚われすぎたのです」

ヴェリディアンは遠い目をした。

「彼は『継承の無駄』を嫌いました。古い魔法、古い概念、古い命——すべてを最適化して、新しいものに置き換えるべきだと信じていた」
「それで、Voidクラスへの強制キャストを……」
「ええ。彼が倒された後も、その遺産は残っています。塔の地下深くに、彼が作った『最適化の釜』があるはずです。そこで彼は、次々と守護者たちをVoidに変えていた」
「その釜を壊せば、守護者たちは戻るのか？」
「壊すだけでは不十分です。インターフェースによる復元も必要です。両方を行って初めて、彼らは元に戻れる」

カイトは拳を握った。

「わかった。必ずやってみせる」
「信じていますよ、カイト」

ヴェリディアンは微笑んだ。その表情は、どこか寂しげだった。

「この世界の魔法は、あなたの世界のプログラミングと共鳴しています。それは偶然ではありません。いつか、その意味を知る日が来るでしょう」
「どういうことだ？」

カイトが尋ねたが、ヴェリディアンの姿は既に薄れていた。

「今は、約束を果たしてください。守護者たちの誓いを、取り戻すのです」

そして、カイトの意識は現実に戻った。

「カイト！ カイト！」

目を開けると、エリナが心配そうに覗き込んでいた。

「大丈夫？ 五分くらい意識がなかったのよ」
「……五分？ ずっと話してた気がしたけど」

カイトは頭を振った。アーカイブの中での時間は、現実とは違うようだ。

「みんな、集まってくれ。方法がわかった」

カイトは仲間たちに説明した。インターフェース、誓い、そして最適化の釜。

「つまり、塔に戻って、地下の釜を壊しつつ、守護者たちの誓いを集める必要があるってことだな」

リアムがまとめた。

「その通り」
「誓いはどうやって集めるの？」

セシリアが尋ねた。

「彼らの心に残ってる。腐敗した守護者に触れて、その意志を引き出すんだ」

カイトは答えた。

「危険なことよ。彼らはまだ敵対的かもしれない」

エレノアが懸念を示した。

「わかってる。でも、やるしかない」

カイトは決意を新たにした。

「明日、塔に戻る。準備を頼む」

その夜、カイトは一人で街を歩いていた。

ヴェリディアンの言葉が頭から離れない。「この世界の魔法は、あなたの世界のプログラミングと共鳴しています。それは偶然ではありません」

何か大きな秘密がある。カイトはそう感じていた。なぜ自分はこの世界に召喚されたのか。なぜプログラミングで魔法が使えるのか。そして——元の世界に戻る方法はあるのか。

「考えても答えは出ないか」

カイトは独り言を言った。今は目の前の任務に集中する。守護者たちを救う。それが先だ。

ふと、カイトは空を見上げた。星が美しい。元の世界では見られない、色とりどりの星々。

「……綺麗だな」

背後から声がした。エリナだ。

「また眠れないの？」

彼女が隣に来た。

「ああ。考え事が多くて」
「ヴェリディアンのこと？」
「……色々」

カイトは答えを濁した。エリナはそれ以上聞かなかった。ただ、静かに星を眺めていた。

「ねえ、カイト」

エリナが言った。

「あなたがこの世界に来てから、ずっと一人で抱え込んでる気がする」
「そんなことない」
「あるわ」

エリナはカイトの目を見た。

「私たちは仲間よ。あなたの悩みも、あなたの秘密も、分かち合えるはず」
「……」

カイトは言葉を失った。確かに、ずっと一人で抱え込んでいた。元の世界のこと、戻りたいという気持ち、そして心のどこかにある「戻りたくない」という矛盾した感情。

「エリナ……」
「無理に話さなくていいの。ただ、覚えておいて」

エリナはカイトの手を握った。

「あなたは一人じゃない。私たちがいる。その『インターフェース』は、絶対に破られないわ」
「インターフェース……？」
「誓いよ。私たちの」

エリナは笑った。カイトもつられて笑った。

「……ああ。頼りにしてる」

二人は並んで、静かに星を眺めた。明日はまた戦いだ。でも今は、この温もりを感じていたかった。

——インターフェースの誓い。それは、仲間との絆の契約。

翌朝、カイトたちは再び継承の塔へ向かった。

今度は、守護者たちを救うために。彼らの誓いを取り戻すために。そして——ヴォイド・ライブラリアンの遺産を完全に断ち切るために。

「行くぞ、みんな」

カイトが言った。

「誓いのリストア、開始だ」

塔の扉が開く。カイトたちは中へと足を踏み入れた。

——新たなる戦いが、幕を開ける。
