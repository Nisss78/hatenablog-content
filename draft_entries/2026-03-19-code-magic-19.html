第19章：適格者の条件

王立魔法図書館の巨大な扉が、軋んだ音を立てて開いた。

カイト、エリナ、リアムの三人は、埃っぽい空気の中へと足を踏み入れた。高い天井にはシャンデリアが吊るされ、無数の本棚が迷路のように並んでいる。

「相変わらず、静かすぎるな」

リアムが声を潜めた。図書館は静寂に包まれていたが、その静けさにはどこか不気味なものがあった。

「セシリアさんは奥の書庫にいるはずだ」

カイトは案内板を確認し、三人は「禁書庫」への階段を下りていった。

禁書庫の守人

地下への階段は長く、壁には古代のルーン文字が刻まれている。カイトはそれらを解読しながら歩いた。

// 古代ルーン文字の翻訳
const runes = {

  "ᚨᚱᚲ": "POWER",          // 力
  "ᚷᛖᛒ": "SACRIFICE",      // 犠牲
  "ᚹᛟᚱ": "TRUTH"           // 真実

「知識、力、犠牲、真実……」

カイトは呟いた。どれも重い言葉だ。

禁書庫の入り口に、一人の女性が立っていた。薄紫色のローブを纏い、手には巨大な杖を持っている。長い黒髪が、微かな風になびいていた。

「カイト、来たわね」

セシリア・クロフォード。王立魔法図書館の館長にして、古代魔法の第一人者。彼女の紫の瞳が、カイトを真っ直ぐに見据えていた。

「セシリアさん、『適格者』について聞きたいんです」

カイトは単刀直入に切り出した。セシリアは静かに頷き、三人を禁書庫の奥へと招き入れた。

適格者の記録

禁書庫の中央には、一冊の古びた書物が台座に置かれていた。表紙には何も書かれていない。ただ、微かに光を放っているだけだ。

「これは『適格者の書』。ヴェリディアンが遺した、最も重要な記録よ」

セシリアは本に手を置いた。

「適格者とは、並列世界を統合できる唯一の存在。ただし、誰でもなれるわけではない」
「条件があるんですか？」

エリナが尋ねた。

「三つの条件があるわ」

セシリアは指を立てた。

  • 条件その一：世界を見通す力——並列世界の存在を認識し、それぞれの因果を理解できること
  • 条件その二：魂を分かつ勇気——自らの魂を複数の世界に分割しても耐えられる精神力を持つこと
  • 条件その三：罪を受け入れる覚悟——全ての世界で犯した罪を、自分のものとして背負うこと

カイトは息を呑んだ。三つ目の条件が、最も重い。

「罪を受け入れる……つまり、左の世界の僕が禁断魔法を使った罪も、右の世界の僕が遺跡を破壊した罪も、全部僕が背負うってことですか」
「その通りよ」

セシリアの声は静かだったが、その重みは計り知れなかった。

class EligibleCandidate {

「カイト、あなたは既に条件の一つ目をクリアしているわ。並列世界を見通す力、それはあなたが魔法をコードとして書ける能力そのものよ」
「じゃあ、残り二つは？」

セシリアは少し躊躇った後、口を開いた。

「魂を分かつ勇気——これは、あなたが既に実践している。三つの世界に意識を飛ばせる時点で、魂を部分的に分割できているのよ」
「ということは……」
「残るは三つ目だけ。罪を受け入れる覚悟」

セシリアはカイトの目を真っ直ぐに見た。

「あなたは、自分が犯した罪を全て受け入れる覚悟はある？」

罪の試練

カイトが答えようとした瞬間、地下空間が揺れた。本棚がガタガタと音を立て、天井から埃が落ちてくる。

「何だ！？」

リアムが剣を抜いた。セシリアは冷静に状況を分析した。

「並列世界からの干渉よ。カイト、あなたの試練が始まったわ」

空中に三つの窓が浮かび上がった。左、中央、右——三つの世界の光景が同時に映し出される。

左の世界：禁断魔法の代償

左の窓には、古代の図書館で膝をつくカイトの姿があった。隣にはエリナが寄り添っているが、その顔色は蒼白だ。

「エリナ、大丈夫か？」

左のカイトが尋ねた。しかし、左のエリナは弱々しく首を振った。

「魂をリンクしたせいで……禁断魔法の影響が、私にも流れてきてる」

左のカイトは絶望した。魂のリンクは、自分の罪をエリナにも背負わせてしまったのだ。

class SoulLink {
  // 魂のリンクには副作用がある
  // パートナーにも罪の重荷が伝播する
  transferSin(from: MagicUser, to: MagicUser): void {
    const sinWeight = from.sin.load();
    to.sin.add(sinWeight * 0.5); // 50%がパートナーに転送
  }
}
「僕が……僕が彼女を巻き込んだ」

左のカイトは拳を握りしめた。禁断魔法を使った罪が、最も大切な人を苦しめている。

中央の世界：選択の重圧

カイトは自分の世界——中央——を見た。そこには、倒れているセシリアの姿があった。

「セシリアさん！」

カイトが駆け寄ると、セシリアは薄目を開けた。

「大丈夫……ただ、魔力を使いすぎただけよ」

セシリアはカイトの手を握った。

「カイト、聞いて。適格者になるということは、全ての世界の因果を自分一人で背負うということ。逃げることはできないのよ」
「分かってます。でも、僕にできるんでしょうか」
「できるわ。あなたは——」

セシリアの言葉は、突然の爆発音にかき消された。

右の世界：破壊の連鎖

右の窓には、崩れゆく遺跡の光景が映っていた。アルフレッド——古代の守人——が、巨大な石像の中からカイトを睨んでいる。

「遺跡を破壊した罪、償ってもらうぞ」

右のカイトは逃げ場を失っていた。崩れた壁が道を塞ぎ、天井が今にも落ちてきそうだ。

「待ってくれ！ 話を聞いてくれ！」
「話など聞かん。行いが全てを語る」

アルフレッドが石の腕を振り上げた。その一撃は、右のカイトを粉砕するのに十分な威力だ。

class AncientGuardian {

「くそっ、回避できない！」

右のカイトは死を覚悟した。

三つの罪

カイトは三つの窓を見つめた。左では禁断魔法の代償がエリナを苦しめ、右では破壊の罪が自分自身を追い詰めている。

そして中央——

「カイト、選んで」

セシリアの声が響いた。彼女はまだ床に倒れたままだ。

「どれか一つの世界を救うか、全てを受け入れるか」

選択肢が目の前に浮かび上がった。

「全部だ。全部受け入れる」

カイトの声は静かだったが、揺るぎない決意が込められていた。

罪の統合

カイトは目を閉じ、三つの世界に同時に意識を飛ばした。左、右、中央——三つのカイトの魂が、一つの場所に集まる。

そこは、何もない純白の空間だった。三人のカイトが向き合い、互いを見つめる。

「君が……中央の僕か」

左のカイトが言った。その顔には疲労と絶望が張り付いている。

「ああ。左の僕、右の僕——二人の罪を、僕が背負う」
「バカか？ 重すぎるぞ」

右のカイトが反論した。

「遺跡を破壊した罪、禁断魔法を使った罪——そんなものを一人で背負ったら、魂が崩壊するかもしれない」
「それでもやる」

カイトは二人に手を差し出した。

「僕たちは、同じ存在だ。別々の選択をしたけれど、根っこは同じ。だから、一人の罪は三人の罪——違うか、一人が三人の罪を背負うのが筋だ」

左のカイトと右のカイトは顔を見合わせた。やがて、二人も手を差し出した。

「……分かった。任せる」

三つの手が重なり合った瞬間——

警告：魂の統合を開始します

カイトの脳内に警告が響いた。同時に、激痛が全身を襲う。

class SoulIntegration {
  async integrate(souls: Soul[]): Promise<UnifiedSoul> {
    // 全ての罪を一つに統合
    const totalSin = souls.reduce(
      (sum, soul) => sum + soul.sin.load(),
      0
    );

    // 魂の容量を超える場合、崩壊のリスクあり
    const capacity = souls[0].capacity;
    if (totalSin > capacity) {
      throw new SoulCollapseError(
        `罪の重さ ${totalSin} が容量 ${capacity} を超えています`
      );
    }

    // 統合実行
    return new UnifiedSoul(souls, totalSin);
  }
}

激痛の中で、カイトは三つの人生の記憶を同時に体験した。

禁断魔法を唱えた時の恐怖。エリナが苦しむ姿を見た時の絶望。遺跡を破壊した時の罪悪感。アルフレッドの怒りを目の当たりにした時の後悔。

全てが、カイトの魂に刻み込まれていく。

「うああああああああああ！！！」

カイトの叫び声が、三つの世界に同時に響き渡った。

試練の終わり

痛みが収まった時、カイトは中央の世界——王立魔法図書館の禁書庫——に立っていた。

エリナとリアムが、心配そうに駆け寄ってくる。

「カイト！ 大丈夫！？」

カイトはゆっくりと目を開けた。視界は歪み、手足には感覚がない。それでも、彼は微笑んだ。

「大丈夫……たぶん」

セシリアが、驚いた表情でカイトを見ていた。

「まさか、本当に罪を統合したの？」
「はい。三つの世界の僕が犯した罪を、全部背負いました」

カイトは自分の胸に手を当てた。そこには、重い重い何かが沈殿している感覚があった。

新たな旅立ち

カイトはゆっくりと立ち上がった。体は鉛のように重いが、心は奇妙なほど軽い。

「適格者になったからといって、終わりじゃないわ」

セシリアが言った。

「世界統合を実行するには、まだ条件が残っている。三つの世界が安定していること、そして統合の合意を得ること」
「安定させるにはどうすれば？」
「それぞれの世界で、未解決の問題を片付けるのよ」

セシリアは三つの指を立てた。

  • 左の世界：禁断魔法の影響を完全に浄化する
  • 右の世界：アルフレッドとの和解を果たす
  • 中央の世界：世界統合の賛同者を集める
「三つの課題……全部クリアしないと」

カイトは覚悟を決めた。

「分かった。やるよ」

エリナがカイトの手を握った。

「私も手伝うわ」
「俺もだ」

リアムが剣を肩に担いだ。

「三人で、三つの世界を救う。悪くない旅だな」

カイトは二人に向かって頷いた。三つの太陽が、窓から差し込んでいる。

夕暮れの約束

図書館を出た三人は、夕暮れの街を歩いた。空はオレンジと紫が混ざり合い、美しいグラデーションを描いている。

「カイト、体は大丈夫なの？」

エリナが心配そうに聞いた。

「正直、きついよ。でも、やり遂げないと」

カイトは空を見上げた。そこには、薄く光る三重の太陽が見える。並列世界の象徴だ。

「適格者になったからには、責任がある。三つの世界を一つにする——それが、ヴェリディアンが遺した最後の願いだから」
「ヴェリディアンって、誰？」

リアムが聞いた。カイトは少し考えてから答えた。

「古代の魔法使い。僕と同じように、魔法をコードとして書ける人だったらしい。彼女は、世界が分かたれる前の時代に生きていた」
「彼女？」
「ああ。女性だったらしい。詳しいことはまだ分からないけど、彼女の遺した魔法書には、世界統合の方法が書かれてる」

エリナは何かを考えているようだった。

「カイト、あなたとヴェリディアンって、何か関係があるのかも」
「どういうこと？」
「魔法をコードで書ける能力。並列世界を認識できる力。それって、普通の魔法使いにはないものよ」

カイトはエリナの言葉を反芻した。確かに、そうかもしれない。

「いつか、真実を知る時が来るかもしれないな」

カイトは再び空を見た。三つの太陽が、ゆっくりと沈んでいく。

「でも、今は目の前の課題に集中しよう。明日から、旅が始まる」

【第19章 あらすじ】

王立魔法図書館でセシリアに会ったカイトは、「適格者」の三つの条件を告げられる。世界を見通す力、魂を分かつ勇気、そして罪を受け入れる覚悟。三つ目の条件を試される中、左の世界では禁断魔法の影響がエリナにも及び、右の世界ではアルフレッドが破壊の罪を問い詰めていた。カイトは三つの世界の自分と魂を統合し、全ての罪を背負うことを決意。激痛に耐え抜き、適格者として認められる。しかし、世界統合にはまだ条件が残っていた。
