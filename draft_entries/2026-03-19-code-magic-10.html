第10章：クラス継承の試練

デバッグ・クエストから三日が経った。カイトは宿屋の一室で、『コード・メイジ・マニュアル』の新しいページを読み進めていた。

「クラス……継承……オーバーライド……」

カイトは独り言を呟きながら、メモを取っていた。マニュアルには、魔法を体系的に管理するための概念が書かれている。それによれば、すべての魔法は「クラス」という単位で定義され、似た魔法同士は「継承」という仕組みで関連付けられるらしい。

「要するに、親の魔法を引き継いで、新しい魔法を作れるってことか」

カイトは納得した。オブジェクト指向プログラミングの概念と同じだ。しかし、理屈はわかっても実践するのは別問題だ。

コンコン。

ドアがノックされた。

「カイト、いる？ 王都から使いが来てるわよ」

エリナの声だ。カイトは本を閉じて、ドアを開けた。

「王都？ 何の用だ？」
「詳しくは下で。急ぎらしいわ」

ダイニングには、王都の紋章が入った制服を着た女性が待っていた。年齢は二十代後半。鋭い目つきと、整った顔立ちをしている。

「初めまして。私は王立魔法研究所のエージェント、エレノア・ヴァレンタインです」

女性は立ち上がり、優雅にお辞儀をした。

「王立魔法研究所……？」

カイトが首を傾げる。

「はい。王国で魔法の研究と開発を行っている機関です。あなたの活躍、アルゴリズム遺跡でのデバッグの件ですが……噂になっておりますわ」

エレノアは微笑んだ。しかしその笑顔には、何か含みがあるように見えた。

「それで、用件は？」
「単刀直入に申し上げます。あなたに、ある任務をお願いしたいのです」

エレノアはテーブルに地図を広げた。

「ここにある『継承の塔』で、異常が起きています。塔を守護していた魔法使いたちが、次々と姿を消しているのです」
「姿を消す？ 死亡したわけじゃないのか？」
「ええ。遺体も、痕跡も、何も残っていない。ただ、『存在が上書きされた』という奇妙な報告だけが残されています」

カイトの背筋が凍った。存在が上書きされる？ それはまるで——

「オーバーライド……」

カイトが呟いた。エレノアが目を細める。

「ご存知ですか？ その言葉」
「ああ。プログラミング用語だ。親クラスの機能を、子クラスが上書きすることを指す」

カイトは説明した。エリナ、リアム、セシリアも真剣に聞いている。

「もし、誰かがこの世界の『魔法の継承』を悪用していたら……？」

カイトは可能性を口にした。

「既存の魔法使いの能力を上書きして、新しい能力に置き換える。そうすれば、元の魔法使いの存在自体が消滅する」
「恐ろしい話だな」

リアムが顔を曇らせた。

「ですから、私たちも急いでいるのです。あなたのデバッグ能力があれば、この異常を解決できるかもしれない」

エレノアはカイトを見つめた。

「引き受けてくださいますか？」

カイトは仲間たちを見た。エリナは頷き、リアムは剣の柄に手を置き、セシリアは静かに微笑んだ。

「わかった。俺たちに任せてくれ」

継承の塔は、王都から馬車で半日の距離にあった。黒い石で作られた荘厳な塔で、高さは優に五十メートルを超える。塔の周りには、古代のルーン文字が刻まれた石碑が並んでいた。

「これらの石碑は、代々の魔法使いたちが魔法を継承するためのものですわ」

エレノアが説明した。

「弟子が師匠の魔法を受け継ぐとき、この石碑の前で儀式を行います。そうすることで、師匠の魔法の『クラス』を引き継ぐことができるのです」
「なるほど……これが継承の仕組みか」

カイトは石碑に近づき、ルーン文字を観察した。文字は複雑に絡み合っているが、何とか読み取れる。

class FireMage extends Mage {
  constructor(master) {
    super(master);
    this.fireAffinity = master.fireAffinity;
  }
  
  cast_fireball() {
    return this.inherit() + "Fireball activated";
  }
}
「これが継承の魔法か……」

カイトは感嘆した。この世界の魔法は、本当にプログラミングで構築されている。

「カイト、何か感じる？」

エリナが尋ねた。

「ああ……塔の中から、強烈なエラーの気配がする。誰かが、継承のシステムを乱用している」

カイトは塔の入り口を見上げた。扉は開いており、中から冷たい風が吹き出している。

「行こう。真相を確かめる」

塔の中は螺旋階段になっていた。壁には無数の魔法陣が刻まれている。階段を登るにつれて、カイトは奇妙な感覚を覚えた。

「何か……重い」

カイトが呟いた。

「どうしたの？」

エリナが心配そうに尋ねた。

「ここ全体が……巨大なクラス定義の中にいるみたいだ。壁の魔法陣が、継承チェーンを形成している」

カイトは壁に手を触れた。手から情報が流れ込んでくる。

class TowerGuardian extends ArchMage {
  constructor() {
    super(ArchMage);
    this.protection_level = 10;
  }
}

class ArchMage extends HighMage {
  constructor() {
    super(HighMage);
    this.spell_slots = 100;
  }
}

class HighMage extends Mage {
  constructor() {
    super(Mage);
    this.mana_pool = 1000;
  }
}
「なるほど……これが塔の守護者たちの継承ツリーか」

カイトは理解した。塔の守護者たちは、世代を重ねるごとに継承によって能力を蓄積してきた。しかし——

「！ 上から何か来る！」

リアムが叫んだ。階段の上から、人影が降りてきた。いや——人だったもの、と言うべきか。

影は半透明で、顔が歪んでいる。口からは意味不明な言葉が漏れている。

「オーバーライド……オーバーライド……」
「あれは……！」

エレノアが息を呑んだ。

「失踪した守護者の一人です……！」

カイトは目を凝らした。影の中に、奇妙な文字が見える。

class CorruptedMage extends Void {
  constructor(original) {
    super(Void);
    this.original_name = original.name;
    delete original;
  }
}
「まさか……Voidクラスへの強制キャスト！？」

カイトは戦慄した。誰かが、塔の守護者たちを「Void」という空のクラスに強制的に継承させているのだ。元の能力はすべて削除され、空の殻だけが残る。

「これは殺人じゃない……もっと酷いことだ」

カイトは震えた声で言った。

「存在そのものを、nullにしているんだ」
「 null……？」

エリナが首を傾げる。

「何もないってことだ。空白。虚無。この人たちは、存在を消されたんじゃない。存在する意味を消されたんだ」

カイトは拳を握りしめた。こんな残酷な魔法があるなんて。

「カイト、来るわよ！」

エリナが叫んだ。腐敗した守護者が襲いかかってくる。

「みんな、下がって！ 俺がやる！」

カイトは前に出た。通常の魔法では勝てない。これはコードレベルの問題だ。バグを直すのではなく、壊されたコードを復元しなければならない。

「リストア・スペル！ 元のクラス定義に戻れ！」

カイトは詠唱した。

function restore_class(corrupted, original_blueprint) {
  let restored = new original_blueprint();
  restored.name = corrupted.original_name;
  restored.state = "recovering";
  return restored;
}

光が腐敗した守護者を包み込む。影が沸騰し、悲鳴のような音が響いた。

「ウウウウウ……！」

守護者の形が少しずつ戻っていく。しかし——

「くっ、完全には戻らない！？」

カイトは歯を食いしばった。復元魔法は部分的に成功したが、守護者の本体はまだ半透明のままだ。根本的な原因——この異常な継承を行っている黒幕を倒さなければ、元には戻せない。

「とりあえず、ここから撤退だ！ 上まで行くぞ！」

カイトたちは走り出した。守護者は追ってこなかった。復元魔法で弱っているようだ。

塔の最上階に着いた。そこには巨大な魔法陣が描かれ、その中心に一人の男が立っていた。

男は長い黒髪を持ち、青白い肌をしている。目は赤く光り、不気味な笑みを浮かべていた。

「ようやく到着したか。コード・メイジよ」

男が声を発した。

「お前は誰だ！？」

カイトが問い詰めた。

「私はヴォイド・ライブラリアン。この塔の継承システムを管理していた者だ」

男は名乗った。

「しかし、管理者はもっと自由であるべきだと思わないか？ 制限なく、新しいクラスを定義し、古いクラスを削除する自由が！」
「削除だと……？ 守護者たちを消したのはお前か！？」
「消した？ 違うな。私は『最適化』したのだ」

ヴォイド・ライブラリアンは笑った。

「この世界は継承の無駄が多すぎる。古い魔法、古い概念、古い命——すべてを削除し、新しいものに置き換える。それが私の理想だ」
「狂ってる……！」

カイトは呟いた。

「継承は、過去の叡智を未来に繋ぐためのものだ！ 削除するためじゃない！」
「ほう？ では、証明してみるがいい」

ヴォイド・ライブラリアンは手を広げた。魔法陣が激しく光り出す。

「お前のクラスを、私のVoidクラスで上書きしてやろう。さあ、新しい継承の時代の始まりだ！」

戦いの火蓋が切られた。

——クラス継承の試練、開幕。
