<h3>第21章：世界を書き換えるコード</h3>

<p>継承の儀式が終わった後、カイトは膝をついていた。体の中に流れ込んできたのは、計り知れない量の魔力と知識だった。それはまるで、膨大なデータベースが一気に脳内に展開されたような感覚。</p>

<p>「大丈夫か、カイト？」</p>

<p>エリナが心配そうに駆け寄ってくる。カイトはゆっくりと顔を上げ、微かに笑った。</p>

<blockquote>「ああ……大丈夫だ。ただ、少し情報量が多すぎてな」</blockquote>

<p>彼が目を閉じると、脳内に無数のコードが浮かび上がる。ヴェリディアンが千年の間に蓄積した世界の構造、魔法の基盤、そして「歪み」の正体がすべてそこにあった。</p>

<pre><code>WorldCore world = new WorldCore();
world.distortionLevel = 0.73; // 臨界点に近い
world.stabilityThreshold = 0.80;</code></pre>

<p>「これが……世界の現状か」</p>

<p>カイトは呟いた。歪みのレベルは七三パーセント。臨界点の八〇パーセントまで、もうわずかしかない。</p>

<p>リアムが剣を抱えながら近づいてきた。</p>

<blockquote>「で、どうなんだ？　継承者になったお前なら、世界を救えるのか？」</blockquote>

<p>「理論上は可能だ。だが、実行には時間がかかる。今の歪みの進行速度を考えると、おそらく三日以内に修正を完了しないと……世界は崩壊する」</p>

<p>セシリアが眉をひそめた。</p>

<blockquote>「三日？　それだけあれば十分ではありませんか？」</blockquote>

<blockquote>「甘いな」</blockquote>

<p>カイトは首を振った。</p>

<blockquote>「世界のコアにアクセスして修正を行うには、膨大な量のコードを書く必要がある。それに、一回の実行で終わるわけじゃない。テスト、デバッグ、最適化……それらすべてを三日でやらなきゃならない」</blockquote>

<p>エリナがカイトの手を握った。</p>

<blockquote>「一人じゃ無理よ。私たちが手伝うわ」</blockquote>

<blockquote>「エリナ……」</blockquote>

<blockquote>「あなたはコードを書くことに集中して。私たちはあなたを守るわ。邪魔が入らないようにね」</blockquote>

<p>リアムがニヤリと笑った。</p>

<blockquote>「そういうことだ。俺たちはお前のファイアウォールだと思え」</blockquote>

<p>セシリアも頷いた。</p>

<blockquote>「私も援助します。癒やしの魔法であなたの疲労を回復させますわ」</blockquote>

<p>カイトは深呼吸をした。仲間の言葉が、重荷とともに勇気も与えてくれた。</p>

<blockquote>「ありがとう。皆……行こう。世界のコアがある場所へ」</blockquote>

<hr />

<p>世界のコアがあるのは、大陸の中央に位置する「始原の塔」だった。千年以上前、ヴェリディアンが世界の基盤システムを構築した聖地。</p>

<p>四人は馬を駆って塔へ向かった。道中、カイトは脳内のコード解析を続けていた。</p>

<pre><code>// 世界の基盤構造
class WorldFoundation {
    ManaFlow manaNetwork;
    ElementalBalance elements;
    SpaceTimeContinuum spacetime;
    
    void repair() {
        // 歪みの原因を特定して修正
        // 予想コード量: 10,000行以上
    }
}</code></pre>

<p>「十万行……いや、もっとかかるかもしれない」</p>

<p>カイトは独り言のように呟いた。隣を走るエリナが聞きとがめた。</p>

<blockquote>「何か言った？」</blockquote>

<blockquote>「いや、ただ独り言だ。世界の修正には膨大なコードが必要になりそうだと」</blockquote>

<blockquote>「……あなた、本当にプログラマーだったのね」</blockquote>

<p>エリナが呆れたように言った。カイトは苦笑いする。</p>

<blockquote>「ああ。元の世界じゃ、深夜までバグ取りしてたもんだ」</blockquote>

<blockquote>「バグ取り？」</blockquote>

<blockquote>「エラーを修正することだ。世界にだってバグはある。今はそれを修正するだけの話さ」</blockquote>

<p>リアムが前を向いたまま口を挟んだ。</p>

<blockquote>「相変わらず、よく分からんことを言う奴だな」</blockquote>

<blockquote>「まあ、待てよ。シンプルに言えば……世界は一つの巨大なプログラムで動いている。そのプログラムにエラーが発生して、今暴走しかけている。俺の仕事は、そのエラーを修正することだ」</blockquote>

<blockquote>「なるほど。シンプルで分かりやすいな」</blockquote>

<p>セシリアが馬を並べた。</p>

<blockquote>「ですが、何故カイト様だけがその『プログラム』を操作できるのですか？　他の魔道士にはできないことでしょう？」</blockquote>

<blockquote>「それが、ヴェリディアンが遺した継承者の力だ。普通の魔道士は、世界のシステムにアクセスできない。彼らが使えるのは、システム上で動くアプリケーション……つまり個別の魔法だけだ。だが、継承者は違う。俺たちはシステムそのものに介入できる」</blockquote>

<p>カイトは空を見上げた。青空の向こうに、微かに歪んだ空間が見える気がした。</p>

<blockquote>「ヴェリディアンは、世界を一つの巨大なソフトウェアとして設計した。それを保守・管理するための権限を持つのが、継承者……つまり、今の俺ってわけだ」</blockquote>

<hr />

<p>始原の塔は、黄昏時に彼らの目の前に現れた。</p>

<p>塔は想像していたものとは違った。石造りの古びた建物ではなく、半透明の結晶でできた、天を突くような構造物だった。その表面には、絶えず光の模様が流れている。</p>

<blockquote>「これが……始原の塔」</blockquote>

<p>エリナが息を呑んだ。</p>

<blockquote>「美しい……」</blockquote>

<blockquote>「これが世界のコアへの入り口だ」</blockquote>

<p>カイトは馬から降りた。塔の入り口に向かって歩き出す。三人もそれに続いた。</p>

<p>入り口には何もなかった。扉も、守人也いない。ただ、透明な壁が立ちはだかっているだけだ。</p>

<blockquote>「どうやって入るんだ？」</blockquote>

<p>リアムが剣に手をかけた。</p>

<blockquote>「破るのか？　斬るのか？」</blockquote>

<blockquote>「いや、もっと簡単だ」</blockquote>

<p>カイトは壁に手を当てた。そして、脳内のコードを実行する。</p>

<pre><code>TowerEntrance entrance = new TowerEntrance();
if (entrance.authenticate(inheritor)) {
    entrance.open();
}</code></pre>

<p>その瞬間、壁が波打った。そして、スライドするように左右に開いていく。</p>

<blockquote>「通れ。認証は完了した」</blockquote>

<p>四人は塔の中へと足を踏み入れた。</p>

<hr />

<p>内部は広大だった。壁も床も天井も、すべてが半透明の結晶でできている。そして、そこかしこで光が脈動していた。</p>

<blockquote>「これは……」</blockquote>

<p>セシリアが驚嘆の声を上げた。</p>

<blockquote>「魔力が、目に見える形で流れていますわ」</blockquote>

<blockquote>「ああ。これが世界のマナネットワークの可視化だ」</blockquote>

<p>カイトは説明した。</p>

<blockquote>「この塔は、世界の基盤システムへのインターフェースなんだ。ヴェリディアンが作った」</blockquote>

<p>彼らは塔の中心へと進んだ。そこにあったのは、巨大な水晶球だった。直径五メートルはあるだろうか。その中で、無数の光の粒子が渦を巻いている。</p>

<blockquote>「これが世界のコアへのターミナルか」</blockquote>

<p>カイトは水晶球に近づいた。手を伸ばし、その表面に触れる。</p>

<p>その瞬間、彼の視界が一変した。</p>

<p>無数のコードが視界を埋め尽くす。世界の構造、魔法の基盤、空間と時間の定義、生命の循環……すべてがデータとして彼の脳内に流れ込んできた。</p>

<pre><code>// 世界システム ログ
// 歪み検出: Zone-7, Zone-12, Zone-23...
// 修正必要箇所: 847カ所
// 推定作業時間: 72時間
// 警告: 歪み進行中。至急の対応が必要</code></pre>

<p>「八四七カ所……」</p>

<p>カイトは息を呑んだ。修正が必要な箇所は、予想を遥かに上回っていた。</p>

<blockquote>「カイト？　どうしたの？」</blockquote>

<p>エリナの声で、彼は現実に引き戻された。</p>

<blockquote>「状況は最悪だ。修正が必要な箇所は八四七カ所ある。三日で全部直さなきゃならない」</blockquote>

<blockquote>「無理なの？」</blockquote>

<blockquote>「……いや、やるしかない。手伝ってくれ」</blockquote>

<p>カイトは水晶球の前に座り込んだ。そして、本格的な「コーディング」を開始した。</p>

<hr />

<p>時間が経つにつれて、カイトの集中は深まっていった。</p>

<p>彼が行っているのは、世界の歪みを修復するコードの記述だった。脳内で構築したコードを、魔力を通じて世界のコアに送り込む。</p>

<pre><code>// 歪み修正コード - Zone-7
public void repairDistortion_Zone7() {
    ManaFlow flow = getManaFlow("Zone-7");
    flow.redirect(blockedChannels);
    flow.balance(elementalRatios);
    stabilize(flow);
}</code></pre>

<p>一つ一つは単純な処理だが、それを八四七カ所分行わなければならない。しかも、各ゾーンごとに異なる処理が必要だ。</p>

<p>エリナたちは、その間ずっと彼のそばにいた。食事を用意し、水を飲ませ、必要な時には癒やしの魔法をかける。</p>

<blockquote>「休憩しなさい、カイト。もう十二時間は経っているわ」</blockquote>

<p>エリナが彼の肩に手を置いた。</p>

<blockquote>「……あと少し。今、Zone-45を処理している」</blockquote>

<blockquote>「今の進捗は？」</blockquote>

<blockquote>「三十二カ所だ。八百十五カ所残っている」</blockquote>

<p>エリナは顔をしかめた。</p>

<blockquote>「このペースじゃ、三日じゃ終わらないわ」</blockquote>

<blockquote>「分かっている。だが、これ以上速くはできない。各ゾーンの特性を理解して、最適なコードを書かなきゃならないんだ。雑にやれば、逆に歪みを広げることになる」</blockquote>

<p>リアムが剣を研ぎながら口を開いた。</p>

<blockquote>「何か手伝えることはないのか？」</blockquote>

<blockquote>「……ある」</blockquote>

<p>カイトはふと思いついたように顔を上げた。</p>

<blockquote>「お前たちが使える魔法、それを俺に教えてくれ。具体的に、どのような処理を行っているか」</blockquote>

<blockquote>「何のために？」</blockquote>

<blockquote>「世界の歪みは、魔法の基盤にも影響している。お前たちが普段使っている魔法の処理を理解できれば、修正のヒントになるかもしれない」</blockquote>

<p>三人は顔を見合わせた。</p>

<blockquote>「分かった。私の炎の魔法から話しましょうか」</blockquote>

<p>エリナが答えた。</p>

<hr />

<p>次の十二時間は、カイトにとって貴重な時間となった。</p>

<p>エリナからは炎の魔法の構造、リアムからは剣技に付与される風の魔法、セシリアからは癒やしの魔法の原理。それぞれの処理を聞き出し、それをコードとして解析していく。</p>

<pre><code>// 炎の魔法の基盤構造（エリナ版）
class FireMagic {
    Element fire = Element.FIRE;
    int temperature = 1500; // 摂氏
    
    void cast(Target target) {
        thermalEnergy.increase(target, temperature);
        ignite(target);
    }
}</code></pre>

<p>「なるほど……。炎の魔法は、対象の熱エネルギーを急激に上昇させているのか」</p>

<blockquote>「そうよ。だから、火を点火するイメージで唱えるの」</blockquote>

<p>カイトは頷いた。この情報は、世界の歪み修正に役立った。特定のゾーンでは、熱エネルギーの制御が暴走していたからだ。</p>

<p>同様に、リアムの剣技とセシリアの癒やしの魔法も解析した。剣技は風の流れを制御し、癒やしは細胞の再生を促進する処理を行っている。</p>

<p>それらを組み合わせることで、カイトは修正速度を上げることができた。</p>

<blockquote>「二百カ所まで進んだ。残り六百四十七カ所」</blockquote>

<p>カイトが報告したのは、開始から二十四時間が経った頃だった。</p>

<p>「三分の一……まだ足りない」</p>

<p>エリナが不安そうに呟いた。</p>

<blockquote>「大丈夫だ。ペースは上がっている」</blockquote>

<p>カイトは答えたが、その声には疲労の色が混じっていた。二十四時間、ほぼ不眠不休でコードを書き続けているのだ。無理もない。</p>

<p>セシリアが彼の額に手を当てた。</p>

<blockquote>「癒やしの魔法をかけておきますわ。少しは楽になるはずです」</blockquote>

<blockquote>「ありがとう、セシリア」</blockquote>

<p>温かい光がカイトを包み込んだ。疲労の一部が消え、集中力が戻ってくる。</p>

<blockquote>「……よし、続けるぞ」</blockquote>

<hr />

<p>四十八時間が経った時、カイトはついに声を上げた。</p>

<blockquote>「六百カ所まで進んだ！　残り二百四十七カ所！」</blockquote>

<p>しかし、その喜びは長く続かなかった。</p>

<p>突然、塔全体が揺れた。</p>

<blockquote>「何！？」</blockquote>

<p>リアムが剣を抜いた。エリナも炎を掌に灯す。</p>

<p>塔の入り口の方から、何かが近づいてくる気配がした。</p>

<blockquote>「侵入者か？　いや、違う……」</blockquote>

<p>カイトは水晶球を見た。そこに表示されたログは、彼に危機を告げていた。</p>

<pre><code>警告: 歪みが臨界点に到達
      - Zone-100, Zone-101, Zone-102 で空間亀裂発生
      - 亀裂より「歪みの化身」出現
      - 推定脅威レベル: S</code></pre>

<blockquote>「歪みの化身……！　ついに現れたか」</blockquote>

<p>カイトは立ち上がった。まだ修正は終わっていない。だが、目の前の脅威に対処しなければ、すべてが水の泡になる。</p>

<blockquote>「エリナ、リアム、セシリア！　奴が来る。迎撃するぞ！」</blockquote>

<p>三人が頷いた瞬間、塔の入り口から巨大な影が現れた。</p>

<p>それは、歪んだ空間そのものが形を成した存在だった。不定形の体に、無数の目が浮かび上がっている。そして、その口からは、世界を飲み込むような絶叫が漏れた。</p>

<blockquote>「これが……歪みの化身」</blockquote>

<p>エリナが震える声で呟いた。</p>

<p>カイトは水晶球に向き直った。修正を続けながら、同時に戦わなければならない。</p>

<blockquote>「俺はコアの修正を続ける。三人は、あの化身を足止めしてくれ！」</blockquote>

<blockquote>「分かった！」</blockquote>

<p>三人が化身に向かって突撃した。</p>

<p>カイトは再び水晶球に手を置いた。残り二百四十七カ所。時間は残り二十四時間。そして、目の前では仲間が命を賭して戦っている。</p>

<p>彼は覚悟を決めた。</p>

<blockquote>「やるしかない……世界を、守り抜くんだ！」</blockquote>

<p>カイトの指が、空中に仮想のコードを打ち込み始めた。それは、世界を救うための、最後のコーディングセッションの始まりだった。</p>

<p>（つづく）</p>
