第41章：静寂の罠

王都への道中、森は不気味なほど静かだった。いつもなら聞こえる鳥のさえずりも、風が木々を揺らす音もない。まるで世界全体が息を止めているかのような、重苦しい沈黙。

「……おかしいわね」

エリナが警戒心を露わにして周囲を見渡す。彼女の魔法使いとしての勘が、危険を察知していた。

「魔力の流れが遮断されている。自然な現象じゃない」

カイトも同意した。彼の〈解析眼（アナライズ・アイ）〉が見ているのは、空中に張り巡らされた不可視の結界——魔力を封じるジャミングフィールドだった。

class JammingField {
    constructor() {
        this.type = "anti_magic";
        this.range = 500; // メートル
        this.intensity = 0.8;
        this.source = null; // 発生源不明
    }
    
    scan() {
        // 結界の発生源を探る
        const trails = this.traceManaFlow();
        return trails.filter(t => t.intensity > 0.7);
    }
}

カイトは目を細めた。発生源が見つからない。これだけ大規模な結界を張れる者となれば、相当の実力者だ。しかも、彼らの進行方向——王都へと続く道を正確に封鎖している。

「罠ですね」

セシリアが静かに言った。

「誰かが私たちを待っている。おそらく、古代図書館で何を手に入れたかを知っている者です」

リアムが剣の柄に手をかける。

「誰だ？帝国の手先か？それとも———」

彼の言葉は途中で途切れた。

森の奥から、人影が現れたからだ。

それは一人の女性だった。腰まで届く銀髪、冷徹な青灰色の瞳。身につけているのは、古代魔導協会の紋章が刻まれた純白の法衣。

「久しぶりね、カイト」

女性の声は、氷のように冷たかった。

「……誰だ？」

カイトは彼女を知らなかった。だが、彼女は自分を知っている。それが不気味だった。

「覚えていないのは当然でしょう。あなたがこの世界に来た時、私はまだ存在していなかったのだから」

女性は優雅に一礼した。

「自己紹介が遅れたわ。私はイレーネ・フォン・クロイツェル。古代魔導協会、審判局の執行官よ」

その名を聞いた瞬間、エリナの顔色が変わった。

「『銀氷のイレーネ』……！まさか、伝説の審判官が！」
「伝説とは大げさね。ただ、任務を全うしているだけ」

イレーネの視線がカイトに戻る。

「カイト。あなたには投降していただくわ。古代の遺産——ヴェリディアンの継承を不当に受けたとして、協会があなたを指名手配しているの」

カイトは眉をひそめた。

「不当？ヴェリディアン様自身が選んだんだぞ。それがなぜ不当なんだ？」
「選ばれたからといって、継承してよいわけではないの。古代の力は封印されるべきもの。それが協会の方釾よ」

イレーネの手のひらに、冷気が集まっていく。

「おとなしく来てくれれば、危害は加えない。ですが———抵抗するなら、力尽くで連行することになる」

リアムが前に出る。

「カイトは何も悪いことをしていない！彼は———」
「黙りなさい」

イレーネが指を鳴らすと、リアムの体が氷の檻に閉じ込められた。

「なっ……！」

ゼロ距離からの不意打ち。反応すらできなかった。

「リアム！」

エリナが魔法を放とうとするが、結界のせいで魔力が満足に使えない。

「無駄よ。この結界の中では、あなたたちの魔法は半減する。逆に、私は強化される」

セシリアが神聖魔法で氷を溶かそうとするが、効果がない。

「浄化できない……これ、ただの氷じゃないわ！」
「正解。これは『絶対零度の概念体（アブソリュート・ゼロ・コンセプト）』——物理法則を無視した永遠の氷よ。あなたの浄化魔法では、概念レベルの攻撃じゃないと破壊できない」

カイトは状況を分析した。

イレーネの能力——氷を具現化する魔法。だが、それは普通の氷ではない。概念そのものを凍らせる能力だ。魔法を凍らせ、感情を凍らせ、思考さえも凍らせる。だからこそ『銀氷』と呼ばれる。

そしてこの結界。魔力を阻害するだけでなく、特定の属性——おそらく氷属性——を強化するチューニングがされている。

カイトは結論を出した。正面から戦えば勝ち目はない。このフィールドの恩恵を受けられるのはイレーネだけ。イレーネの能力を無効化するか、結界そのものを破壊しなければならない。

「……わかった。投降しよう」

カイトが両手を挙げた。

「カイト！？」

エリナが驚く。

「待って。本気なの？」
「ああ。無駄な争いは避けたい」

カイトはゆっくりとイレーネに近づいた。完全に無防備な状態。

イレーネが冷徹な笑みを浮かべる。

「賢明な判断ね。やはり、あなたは賢い———」

その瞬間。

カイトの体が、無数の光の粒子となって弾けた。

「！？」

イレーネが反応する間もなく、カイトの声がどこからか響く。

「残念だったな。俺の得意分野は『デバッグ』———欠陥を見つけて、書き換えることだ」

カイトが姿を現した場所は、イレーネの背後。

そして、彼の手には一本の矢があった。結界の要（コア）を射抜くために編み出した、対結界用の魔導矢だ。

class PiercingArrow extends Spell {
    constructor() {
        super("concentrated_mana");
        this.property = "anti_barrier";
        this.homing = true;
    }
    
    async execute() {
        // 結界の脆弱点を探す
        const weakPoint = await this.findFracture();
        
        // そこを貫く矢を生成
        const arrow = new ConceptualProjectile({
            target: weakPoint,
            penetration: "infinite",
            speed: "light"
        });
        
        return arrow.fire();
    }
}

カイトが矢を放つ。

光の矢は一直線に結界の要へと飛び、音もなく貫いた。

結界が音を立てて崩壊する。

「っ……！」

イレーネが結界を維持しようとするが、もう遅い。要を破壊された以上、復元は不可能だ。

魔力が正常に戻るのを感じる。エリナとセシリアが即座に体勢を立て直す。

「リアム！」

セシリアが浄化魔法を放つ。今度は成功した。概念体の氷が聖なる光に溶かされていく。

リアムが檻から解放され、剣を構える。

「カイト、ナイスだ！今のうちに———」
「待て」

カイトがリアムを制止した。

イレーネはまだ倒れていない。結界を破壊された程度で、彼女の実力は削がれていない。

「……予想以上ね」

イレーネが立ち上がる。その表情には、わずかながら興味が混じっていた。

「結界のコアを一瞬で見抜くなんて。協会の報告には、あなたが『コードで魔法を書く』とあったけれど……なるほど、解析能力が異常なレベルね」
「褒め言葉として受け取っておくよ」

カイトは油断なく距離を取る。

「だが、これで条件は五分だ。結界なき状態で戦うなら、勝負はわからない」

「勝負？可笑しいわね」

イレーネが冷笑する。

「私は『任務』で来ているの。遊びじゃない。結界を破られたなら———別の手を使うまで」

彼女が懐から一つの水晶を取り出す。

「これは古代の召喚具（サモニング・クリスタル）。中に封じられているのは、かつて一国を滅ぼした氷竜よ」

エリナが顔を蒼白にする。

「氷竜……！？そんなものを———」
「逃げ場はないわ。降伏するか、氷漬けにされるか。選びなさい」

カイトは考えた。

氷竜。伝説級の魔獣だ。普通の魔法では傷一つつかない。だが———彼には、普通の魔法以外の武器がある。

「……一つ、聞かせてくれ」

カイトが言った。

「協会は、ヴェリディアン様の継承を『不当』だと言った。だが、ヴェリディアン様が選んだ事実は変えられない。協会の方針と、本人の意志———どちらが優先されるべきなんだ？」

イレーネは一瞬、言葉に詰まった。

「それは———」
「ヴェリディアン様は、千年の時間を超えて継承者を探していた。やっと見つけた相手を、協会が『不当』と判断する権利があるのか？」

カイトの言葉に、イレーネの瞳が揺れた。

「……あなたは、知らないのね」

彼女が水晶を握りしめる。

「ヴェリディアンがなぜ、継承者を探していたのか。その真の理由を」
「真の理由？」
「古代の図書館。その奥に眠る『禁忌の書』———それが何を記しているか、あなたはまだ知らないでしょう」

カイトは警戒した。

古代図書館。ヴェリディアンが「重大な秘密」と言っていたもの。まだ奥まで探索できていない。

「……何が言いたい？」
「ヴェリディアンが継承者を求めたのは、自分の過ちを正すためよ。彼が千年前に挑み、敗れた『時空を歪める実験』———その失敗が、今も世界を蝕み続けている」

イレーネの声に、わずかな悲しみが混じった。

「協会は知っているの。その失敗を。そして、継承者が現れれば———その失敗を引き継ぐことになると」

カイトは息を呑んだ。

「……どういうことだ？」
「ヴェリディアンの継承とは、力だけでなく『負債』も受け継ぐということ。彼が残した時空の歪み———それを修正する義務が、継承者には課されるのよ」

エリナが口を挟む。

「待って！じゃあ、カイトは———」
「ええ。世界を救うか、自分を犠牲にするか。究極の選択を迫られることになる」

イレーネが水晶を収めた。

「協会は、あなたを守るために指名手配したの。継承を無効にすれば、負債からも解放される。それが私たちの———私の判断よ」

沈黙が流れた。

カイトは考えた。ヴェリディアンの言葉を思い出す。知識は力であり、力は責任を伴う

彼は既に覚悟を決めていたはずだ。だが、まさか自分が犠牲になる可能性があるとは——。

いや。

カイトは首を横に振った。

「……感謝する。教えてくれて」

彼はイレーネを見据えた。

「だが、俺は逃げない。ヴェリディアン様が選んでくれた。その期待に応えるのが、継承者の務めだ」
「……愚かね」

イレーネが目を伏せる。

「でも、それがあなたの選択なら———尊重するわ。協会には、あなたを見失ったと報告しておく」

彼女は背を向けた。

「次に会う時は、敵としてかもしれない。その時は———容赦しない」

イレーネの姿が、氷の粒子となって消えていった。

森に静寂が戻る。

「……カイト」

エリナが心配そうに彼を見た。

「今の話、本当なの？」

カイトは小さく頷いた。

「多分、な。だが———」

彼は空を見上げた。王都の方角。

「問題を回避するために逃げるのは、プログラマーらしくない。バグがあるなら、修正するだけだ」

リアムがカイトの肩を叩く。

「俺たちはついてるぜ。何があってもな」

セシリアも微笑んだ。

「私もです。神聖魔法で、何かお役に立てるはず」

エリナは何も言わず、ただカイトの手を握った。その温もりが、何よりの励ましだった。

「……行こう。王都へ」

カイトは歩き出した。

待ち受ける真実が何であれ、彼は前に進むと決めた。

それが継承者の——コードを紡ぐ者の、選んだ道だった。
